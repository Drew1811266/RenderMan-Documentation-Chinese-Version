# RenderMan 27 Translation Glossary

Terminology control for Codex-authored Chinese translation of RenderMan HTML documentation.

## Translation Rules

- RenderMan product names, renderer names, node names, API identifiers, file paths, environment variables, and UI labels with exact spelling must be preserved unless the glossary gives a Chinese rendering.
- First occurrence of an important English technical acronym may use Chinese explanation plus acronym, e.g. '任意输出变量（AOV）'; subsequent occurrences may use the acronym.
- Do not translate identifiers beginning with Pxr, Rix, Ri:, RfM, RfH, RfK, RfB, PRMan, RMAN, RMANTREE, or node/operator names from DCC applications.
- Translate explanatory prose naturally, but keep command-line examples, code blocks, attribute names, parameter names, JSON keys, and file extensions unchanged.
- Prefer established CG/VFX Chinese terminology: 渲染, 着色, 材质, 纹理, 采样, 积分器, 光路, 合成, 置换, 体积, 次表面散射, 法线, 反照率, 噪声, 降噪.
