Export to RIB

Source: https://rmanwiki-27.pixar.com/space/RFK27/628719654/Export+to+RIB
Downloaded: 2026-06-15T03:49:12+00:00

你可以在 Node Graph 面板中，将 Katana scene 的一部分或全部导出为 RenderMan RIB 文件。右键单击任意 node 打开 context menu，选择 Debug menu，然后选择 “Open prman .rib Output in External Editor”。RfK 会构建截至该 node 的 scene，将其写入 RIB 文件，并让 Katana 在你的 偏好设置中设置的外部文本编辑器中打开该文件。

这个 Debug RIB Output 会与该 scene 的 disk render 版本匹配。

要检查 live renders 的 scene state，可以使用一个 PrmanGlobalStatements option，让它在任何 render 前写出 RIB。该 option 位于 “Debug” 下的 plugin section：plugin->Debug->debugRibFile。
