Activate Interactive Denoising（启用交互式降噪）

interactive denoising option 可以在 PrmanGlobalStatements node 上找到。

Preferences（首选项）

Cheap First Pass（低成本首次 Pass）
如果勾选，first pass 会使用成本更低的 heuristic（稍快，但 quality 较低）。

如果正在 render 的内容 convergence 很快，并且你想优先更快获得 denoised result，这会很有用。

Interval（间隔）
denoise runs 之间的 time interval（以 seconds 为单位）

LogInfo
Output logging info（输出日志信息）

Min Samples（最小 Samples）
interactive denoiser 第一次运行前，每个 bucket 的 average samples 最小数量。更改此 preference 后，需要 restart render，此 option 才会生效。
