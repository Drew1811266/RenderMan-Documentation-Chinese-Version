User Attributes

Source: https://rmanwiki-27.pixar.com/space/RFK27/546177856/User+Attributes
Downloaded: 2026-06-15T03:49:12+00:00

User attributes（用户属性） 是一种便捷方式，可在不更改 shading network（着色网络）的情况下，让多个对象之间的着色产生变化。它们特别有用，因为与 primitive variables（图元变量）不同，它们可以随 instance（实例）变化。

Setting User Attributes（设置用户属性）

下面是一个示例，展示如何使用 OpScript 设置 color user attribute（颜色用户属性）：

gb = GroupBuilder()
gb:set("value", FloatAttribute({0.0, 1.0, 0.0}, 3))
gb:set("type", StringAttribute("color"))
Interface.SetAttr("prmanStatements.attributes.user.myColor", gb:build())

虽然所有 user attributes（用户属性）都可以设置为 GroupAttribute，但 color attributes（颜色属性）是唯一需要这样做的类型。Int、float 和 string user attributes（字符串用户属性）可以直接设置，其类型会从 Katana attribute type（Katana 属性类型）推断出来：

Interface.SetAttr("prmanStatements.attributes.user.myFloat", FloatAttribute(3))

Using User Attributes（使用用户属性）

RenderMan 随附了几个使用 user attributes（用户属性）的插件。  PxrAttribute 或 PxrVary 是读取 user attributes（用户属性）的 patterns（模式）。 PxrCryptomatte sample filter（采样过滤器）可以基于 string user attributes（字符串用户属性）创建不同的输出。使用这些插件时，attribute（属性）参数应包含前缀 user:，后接 attribute name（属性名称）。
