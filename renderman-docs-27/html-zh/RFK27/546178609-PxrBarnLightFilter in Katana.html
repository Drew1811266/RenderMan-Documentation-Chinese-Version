PxrBarnLightFilter 有两种模式：Physical 和 Analytic。Physical Barn 会产生物理准确的阴影，Analytic Barn 则提供更精细的用户控制。

Physical Barn

概要

Physical Barn 可用于模拟片场中的实际做法。可以把它看作会投射阴影的物理图元。Physical Barn 的作用就像一扇窗口，只允许灯光通过开口形状进入。

用途

  • 让光线穿过空间中的开口，从而产生物理准确的阴影。

  • 在大型场景中约束光线反弹。

Shaping（塑形）

灯光的尺寸以及挡光板（barn）相对于灯光的位置，都会改变灯光在片场上“投射阴影”的方式。光线穿过挡光板后，会根据这个尺寸和位置形成本影与半影区域。Viewer Modifier 引导线会在你调整灯光尺寸或移动挡光板时显示本影和半影区域。理解其作用时，请观察挡光板起始处的形状，也就是光线穿过的窗口。该形状边缘在空间中相对于灯光尺寸和距离的位置，决定了本影和半影的大小。

影响灯光本影和半影质量的两个主要因素：

  1. 灯光尺寸

    1. 小尺寸灯光穿过挡光板形状时会投射清晰阴影（小半影），并产生更大的本影。

    2. 大尺寸灯光穿过挡光板形状时会投射柔和阴影（大半影），并缩小本影。

  2. 挡光板到灯光的距离：

    1. 靠近灯光的挡光板会产生较大的本影。

    2. 远离灯光的挡光板会产生较小的本影。

增大 edge 参数

默认情况下，基于物理的挡光板将 edge 参数设为 0。你会在表面上看到来自灯光的衰减，因为它是真正的阴影。这种阴影柔和度由上文说明的灯光尺寸/挡光板位置决定，而不是由 edge 参数设置决定。回到“挡光板开口只是一个物理图元”这一概念：edge 参数就像放在挡光板边缘外侧的一张透明片。它允许光线通过，但不会改变半影大小（不会改变阴影的锐利/柔和程度）。

开始增大 edge 值时，你会看到 Viewer 中出现一个新的引导环。这是 edge 区域的半影终点。该环表示灯光不再产生影响的位置。 

关于 edge 参数最重要的几点：

  • 从 0 开始：先看看能否通过改变挡光板位置或灯光尺寸得到想要的柔和度

  • 用它让更多光线穿过挡光板，从而创建更柔和、更大的衰减

  • 它不会改变阴影质量（阴影有多柔和或多紧）

Viewer Modifier 引导线

  • 内环内部是本影

  • 内侧引导环是本影的结束和半影的开始

  • 中间引导环是半影的结束

  • 外侧引导环是 edge 区域的结束，也就是灯光不再产生影响的位置

  • 本影引导环与半影引导环之间的距离表示灯光会投射出的阴影大小。

 preBarn

  • no light：灯光只从挡光板的起始处开始发射。这是默认行为。

  • no effect：灯光从光源位置开始发射；如果挡光板被放在偏离灯光的位置（很可能如此），那么光线会在挡光板起始处受到挡光板形状影响

  • cone：灯光从光源位置开始发射。如果挡光板位于灯光前方，所发射的光会受到约束，就像挡光板的边缘向后延伸到灯光位置一样。

偏斜

  • 向上偏斜：在 Viewer 中选择挡光板（barn），然后向上移动

  • 向下偏斜：在 Viewer 中选择挡光板（barn），然后向下移动

注意事项

  • 如果 Physical Barn 位置与灯光位置完全对齐，引导线只会变成一条围绕灯光位置的线。把它稍微移离灯光一点，引导线就应该会显示出来。

  • 没有引导线时，很难判断 Physical Barn 正在做什么。请在 PrmanLightFilterCreate 的查看器区段中设置 Guide Size 参数，以控制引导线长度。

Analytic Barn

概要

Analytic Barn 使用起来非常直接，但不会提供 Physical Barn 那样的物理阴影。请参阅 PxrBarnLightFilter 文档，查看 Analytic Barn 参数如何影响灯光形状的示例。
