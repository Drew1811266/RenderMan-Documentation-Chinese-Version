Volumes in Katana

Source: https://rmanwiki-27.pixar.com/space/RFK27/546177488/Volumes+in+Katana
Downloaded: 2026-06-15T03:49:12+00:00

在 Katana 中渲染体积与渲染其他几何体类似：你需要一个体积图元和一个体积着色器。RenderMan Volume 图元是在 Katana 中使用 PrmanVolume 节点创建的。对于雾这类简单均匀体积，只需把 PxrVolume 着色器指定给 PrmanVolume 图元。更复杂的着色可以建立在 PxrPrimvar pattern 之上，将体积数据连接到体积着色器中。RenderMan 的体积着色器 PxrVolume 与 PxrPrimvar 以及其他 pattern 一起，为 OpenVDB 体积数据或几何体积图元提供完整的体积工作流。有关 RenderMan 体积图元的更多详情，请参阅 RenderMan Volume 文档.

当前 PrmanVolume 支持两种体积类型：

  • 有形几何体积 ('box') - 这最常用于雾等效果的均匀体积。PrmanVolume 着色器可以附加到任何几何图元；不过，使用 PrmanVolume box 图元可实现体积内灯光等效果。

  • OpenVDB 隐式场插件 ('vdb') - 外部生成的体积数据，使用 OpenVDB 文件格式导入 Katana 和 RenderMan。

PrmanVolume 节点

PrmanVolume 节点包含用于配置体积图元的参数，以及用于定位和缩放体积的变换属性。UI 中公开的参数取决于所选体积类型：

Box Parameters（Box 参数）

Voxel Resolution（体素分辨率）

体积每个维度中的体素数量。

PrimVars（图元变量）

体积上的图元变量（primvars）动态列表。对于每个 primvar，必须指定名称和类型。如果变量是 uniform（均匀）变量，请勾选 'uniform' 复选框并输入常量值。随后 RfK 会自动为该体积生成 uniform 数据并发送给 RenderMan。如果你的数据是 non-uniform（非均匀）的，则必须自行生成数据，并通过 OpScript 或其他方式将其连接到 primvar。 

VDB Parameters（VDB 参数）

Filename（文件名）

OpenVDB 数据文件的完整路径。

Density Grid（密度网格）

VDB 网格是体素数据的稀疏树表示。例如，float 网格会为每个体素存储一个单精度浮点值。加载 OpenVDB 文件后，下拉菜单中会出现网格名称列表。选择你想要采样的网格名称。 所有网格都会作为 primvars 发送到渲染器，但只有连接到 PxrVolume 着色器的那些网格会被使用。

Density Multiplier（密度倍增）

缩放体积密度网格中的值。这应该比在 PxrVolume 着色器中缩放密度更高效。

Density Rolloff（密度衰减）

大于 0 的值会在密度值中产生对数衰减。较小的 rolloff 值会产生更大的衰减。

Filter Width（过滤宽度）

 如果设为 0，会禁用 mipmapping。如果大于 0，小于 1 的值会偏向 mipmap 的更精细层级，大于 1 的值会偏向更粗糙层级。

PrmanVolume 节点目前不支持 Blobbies。

OpenVDB 体积

PrmanVolume 还支持通过 OpenVDB 文件生成的体积数据。OpenVDB 是用于体积的开源层级数据结构。它已经成为应用程序之间交换体积数据的标准。有关 OpenVDB 的更多信息，请参阅  OpenVDB FAQ（OpenVDB 常见问题） .

你在 PxrSeExpr 节点中使用的表达式会取决于具体外观。创建文档用文件时，下面是用于 incandescence（白炽/自发光）和 density bias（密度偏置）的表达式。

Incandescence（白炽/自发光）:

$temperature = floatInput1;
$incandescence = (1.0 - smoothstep(bias($temperature, 0.8), 0.143, 0.857)) * [20, 7.776, 3.702];
$incandescence

Density bias（密度偏置）:

$density = bias($density,0.815);
$density1 = 0.9 * smoothstep($density, 0.136, 0.15);
$density2= 0.9 * (1 - smoothstep($density, 0.15, 0.857));

if ($density < 0.15) {
    $density = $density1;
} else {
     $density = $density2;
}
$density
