Using（使用） PxrCryptomatte workflow，可以在 Katana 中 output IDs，方便在 compositing application 中选择。随后可以将该 selection 用作 mask，在 post packages 中有选择地进行修改。

PxrCryptomatte 作为 Sample Filter plugin 提供。因此，Cryptomatte 的 output 会作为独立 EXR file 存储，与 beauty 或 denoise outputs 分开。请注意，需要 OpenEXR output（file.exr）才能正确存储这些 values。

只有在使用 disk render 时才会 render 该输出，以避免在 interactive session 中覆盖 files


附带的 example file 用于 Katana workflow。

此 file 包含两个 PxrCryptomatte nodes，用于同时 output object 和 material IDs。

Example Workflow（示例工作流）：

  1. 创建 PxrCryptomatte node。指定 file name 和 path。example file 使用 script 写入 Temp directory。

  2. Layer：选择 layer type，Name（object）或 Material。

  3. 将这些连接到 ShadingNodeArrayConnector

  4. 创建 PrmanShadingNode 并选择 PxrSampleFilterCombiner。将 ShadingNodeArrayConnector.out 连接到此 shading node 的 input。

  5. 创建 NetworkMaterial node，right-click 并选择 "Add terminal"，在 Prman dropdown 中选择 "samplefilter"。

  6. 将 prmanSamplefilter.out 连接到 NetworkMaterial。

  7. 将上述 network（易于 reuse）merge 到 trunk 中。

  8. 创建 MaterialAssign node，并将 Cryptomatte NetworkMaterial assign 到 /root

render scene，然后在上面定义的 path 中查找 output EXR。example file 会 output 两个 EXRs，一个包含 Object IDs，另一个包含 Material IDs，如下所示。
