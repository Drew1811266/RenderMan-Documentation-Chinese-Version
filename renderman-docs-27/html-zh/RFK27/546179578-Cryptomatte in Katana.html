Katana 中的 Cryptomatte

Source: https://rmanwiki-27.pixar.com/space/RFK27/546179578/Cryptomatte+in+Katana
Downloaded: 2026-06-15T03:49:12+00:00

使用 PxrCryptomatte 工作流，可以在 Katana 中输出 ID，方便在合成应用中选择。随后可以将该选择用作遮罩，在后期软件中有选择地修改。

PxrCryptomatte 作为 Sample Filter 插件提供。因此，Cryptomatte 的输出会作为独立 EXR 文件存储，与 beauty 或降噪输出分开。请注意，需要 OpenEXR 输出（file.exr）才能正确存储这些值。

只有在使用磁盘渲染时才会渲染该输出，以避免在交互式会话中覆盖文件


附带的示例文件用于 Katana 工作流。

此文件包含两个 PxrCryptomatte 节点，用于同时输出对象和材质 ID。

示例工作流：

  1. 创建 PxrCryptomatte 节点。指定文件名和路径。示例文件使用脚本写入 Temp 目录。

  2. Layer：选择图层类型，Name（object）或 Material。

  3. 将这些连接到 ShadingNodeArrayConnector

  4. 创建 PrmanShadingNode 并选择 PxrSampleFilterCombiner。将 ShadingNodeArrayConnector.out 连接到此着色节点的输入。

  5. 创建 NetworkMaterial 节点，右键单击并选择 "Add terminal"，在 Prman 下拉菜单中选择 "samplefilter"。

  6. 将 prmanSamplefilter.out 连接到 NetworkMaterial。

  7. 将上述网络（易于复用）合并到主干中。

  8. 创建 MaterialAssign 节点，并将 Cryptomatte NetworkMaterial 指派到 /root

渲染场景，然后在上面定义的路径中查找输出 EXR。示例文件会输出两个 EXR，一个包含对象 ID，另一个包含材质 ID，如下所示。
