Katana 中的材质层

Source: https://rmanwiki-27.pixar.com/space/RFK27/546178139/Material+Layers+in+Katana
Downloaded: 2026-06-15T03:49:12+00:00

RenderMan 支持通过材质分层创建复杂的外观效果。这意味着车漆、瓶身标签、污垢或泥浆等效果，都可以高效地以分层方式组合并用于渲染。

下面是一个逐步示例，演示如何在青铜雕像上制作铜绿。在本教程底部，可以下载一个 Katana 场景，其中包含复现此图像所需的部分。


基本场景准备

加载 layeredMaterial_prep.katana 教程场景，其中包含用于加载雕像几何体、地面平面和聚光灯的框架。此时场景已经可以渲染；但是我们还没有给雕像分配任何材质，因此它会使用 RenderMan 的默认 Bxdf 着色器：PxrDiffuse 进行渲染。


步骤 1：PxrLayerSurface

创建一个着色网络，并为对象分配一个 PxrLayerSurface Bxdf。这一框架会告知 RenderMan 要为已分配的对象渲染哪些材质（新节点以红色显示）：

此时你可以使用新的着色网络进行渲染，但这个材质单独使用时可调整的内容并不多；该节点的属性集有限。原因是 PxrLayerSurface 被设计为顶层 Bxdf，用于组合多层图案。PxrLayerSurface 只包含会应用到所有层的全局属性。颜色和纹理等逐层属性来自连接到 inputMaterial 属性的上游材质。

从这里开始，为避免重复工作，并让材质网络尽可能简单实用，规划就非常重要。研究如何在铜或其他金属上制作铜绿时，你会发现这通常是一个多阶段项目，可能涉及多种方法。根据目标外观不同，材质的布局也会变化。例如，通过加热金属产生不同颜色，可以使用 iridescence 属性在 PxrLayerSurface 中复现。制作铜绿的另一种方法是使用醋和盐在金属上形成结壳，这也是我们在这里要试验的效果。

需要两个基本材质层：

  1. 类似青铜或铜的金属层
  2. 铜绿层，一个略带结壳感、颜色偏蓝绿色的漫反射层

步骤 2：PxrLayerMixer

创建一个  PxrLayerMixer 并将它的 out 端口连接到 inputMaterial 端口，这是 PxrLayerSurface 上的输入端口。这样会建立一个节点，我们将在其中连接所有层。我们来看一下 PxrLayerMixer 着色节点的默认状态：

其中有一个 BaseLayer 以及另外四个编号为 1-4 的层。请注意 Layer 1 默认是启用的，因为通常假定至少会有两层输入到 PxrLayerMixer。如果某个层已启用但未连接，整个渲染会变黑。我们先专注于构建基础层，因此先禁用 Layer 1，直到我们接入铜绿层。请注意，你可以把其他 PxrLayerMixers 连接到这些层中，用于组合外观集合并一起使用遮罩。这在你想在其他位置复用网络时非常有用。本例只使用两个层： Base 和 Layer 1。

步骤 3：基础层

铜/青铜将作为我们的“基础”层。创建一个 PxrLayer 并将它的 pxrMaterialOut 端口连接到 Base Layer 输入端口，也就是 PxrLayerMixer 的对应输入。

为了近似铜材质，我们设置漫反射和镜面反射，然后加入一些程序化凹凸，让铜表面带有合适的质感。这些设置有一定任意性，因为它们是按艺术方向调整的；我们的目标是较陈旧、暗哑的铜或青铜外观：

 

步骤 4：铜绿层

铜绿将作为我们的“Layer 1”，位于基础层之上。创建另一个 PxrLayer 节点，并将它的 pxrMaterialOut 输出端口连接到 Layer 1 输入端口（在你的 PxrLayerMixer 上）。请注意，如果你在处理基础层时禁用了 Layer 1，可能需要重新启用它。铜绿着色的漫反射颜色取样自真实图像，并加入了一些漫反射粗糙度，以帮助呈现铜绿结壳的“粉状”外观。我们添加了 PxrFractal 和一个凹凸图案来提高真实感。前面的凹凸图案在这一层中再次使用以保持一致，不过你也可以在这里使用不同的图案，进一步提高真实感。

这一层会完全覆盖 Base 层，直到我们给 Layer 1 Mask添加输入。

步骤 5：铜绿遮罩

我们可以用纹理完成这个遮罩，但这里改用 PxrDirt 节点来控制哪些位置显示铜绿、哪些位置显示金属。PxrDirt 是一种遮挡着色器，可让着色效果受几何体局部拓扑影响（也就是凹处和缝隙）。创建一个新的 PrmanShadingNode，并将节点类型设置为 PxrDirt。它的输出是颜色，但因为这里是在创建遮罩，可以把 resultR连接到 Layer 1 Mask（位于 PxrLayerMixer 上；也可以使用 PxrToFloat 将颜色转换为标量值）。

 

为了产生最终图像中的风化图案，我们提高 Bias Normal 属性在 PxrDirt 中的 Y 分量，使结果沿垂直方向拉伸，就像雨水和风化痕迹顺着雕像向下流过一样。PxrDirt 的 Unoccluded 属性由 PxrFractal 和 PxrGamma 驱动，分别用于打散结果并控制强度。请注意 PxrDirt 下方额外放置的 PSN_PxrConstant 节点。通过将 PxrDirt 输出 resultRGB 连接到 emitColor（位于 PxrConstant 上），我们就可以把 PxrLayerSurface Bxdf 替换为 PxrConstant Bxdf，从而单独渲染污垢层遮罩。
