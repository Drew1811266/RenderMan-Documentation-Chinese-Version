欢迎使用 RenderMan for Katana！

RenderMan for Katana (RfK) 利用了新版 Katana 中的改动，并继续完整支持最新的 RenderMan。有关更多细节，请参阅 RenderMan Release Notes（RenderMan 发布说明）。

当前版本支持：

  • Katana 8.0

  • Katana 7.5

  • Katana 7.0

Katana 限制

  • 如果使用 ROI 与 overscan 或 cropWindow，在编辑分辨率后，需要稍微调整 ROI，才能在 live render 期间获得正确值

  • 使用 "Light Filter" coordsys 的带 filter instanced lights 会产生不正确的 transform。解决方法是使用 light filter reference 将该 light filter 提升为 shared light filter。

  • 对于添加在 node graph 末尾、正好位于 Render node 上方的 nodes，我们不会从 Katana 接收到 live render edits。如果在 Render node 上方插入一个 no-op node（例如 Merge），然后把该 node 添加在它上方，则可以接收到 edit。

  • 任何 live updates 都会导致 interactive motion blur 被禁用。必须重新启动 render 才能看到 motion blur。
