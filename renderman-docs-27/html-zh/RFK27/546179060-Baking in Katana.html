Baking in Katana

Baking（烘焙） 允许用户将 Pattern 网络预先计算或“烘焙”到 2D 纹理图像或 3D 点云中。 

Bake Patterns（烘焙图案）

添加 PxrBakeTexture 或 PxrBakePointCloud 将这些 Pattern 节点插入到着色网络中希望把该网络状态烘焙到纹理或点云的位置。根据节点和 hider 设置，此节点会烘焙纹理、读取纹理，或作为直通节点。

Starting a Bake Render（启动烘焙渲染）

要烘焙纹理，必须将 PrmanGlobalStatements 节点上的 hider 设置设为“bake”。烘焙渲染必须是 Disk Renders（磁盘渲染），而不是 Live/Preview Renders，因为渲染器会将纹理写入磁盘，没有可交互显示的内容。烘焙后的纹理会写入各个 PxrBakeTexture 或 PxrBakePointCloud Pattern 中指定的位置。
