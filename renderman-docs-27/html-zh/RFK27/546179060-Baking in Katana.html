Baking（烘焙） 允许 users 将 pattern networks 预先计算或 "bake" 到 2D texture images 或 3D point clouds 中。 

Bake Patterns（烘焙图案）

添加 PxrBakeTexture 或 PxrBakePointCloud patterns 到 shading network 中希望将该 network 状态 bake 到 texture 或 point cloud 的位置。根据 node 和 hider settings，此 node 会 bake textures、read textures，或作为 pass through node。

Starting a Bake Render（启动烘焙渲染）

要 bake texture，必须将 PrmanGlobalStatements node 上的 hider setting 设为 "bake"。Bake renders 必须是 Disk Renders（不是 Live/Preview Renders），因为 renderer 会将 textures 写入磁盘，没有可交互显示的内容。baked textures 会写入各个 PxrBakeTexture 或 PxrBakePointCloud patterns 中指定的位置。
