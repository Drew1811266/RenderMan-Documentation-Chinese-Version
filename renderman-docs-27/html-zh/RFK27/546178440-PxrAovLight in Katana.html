PxrAovLight 是一个实用的 utility light（工具灯），允许 lighting artist 输出 shot-specific AOV mask（镜头专用 AOV 遮罩），而无需向 shading artists 请求。

使用 PxrAovLight 进行 Light Linking（灯光链接）

使用 light linking 将 geometry 片段包含在 AOV 中或从中排除。在此示例中，PxrAovLight 在全局为 "off"，并在场景中的 tree 上为 "on"。此 linking setup 会为 tree 创建一个 mask。

Light Filters 和 PxrAovLight

因为 PxrAovLight 是一个 "light"，light linking 和 light filters 可以修改 output AOV。在此示例中，PxrIntMultLightFilter 附加到 PxrAovLight。Light Filter 只链接到 tree trunk，因此 tree trunk mask 会得到一个 gray value。

定义 Output AOV（输出 AOV）

使用 PrmanOutputChannelDefine 和 RenderOutputDefine 节点创建 PxrAovLight 将写入的 AOV。这三个参数应全部相同：
