PrmanSignalVisualizer

Source: https://rmanwiki-27.pixar.com/space/RFK27/546178051/PrmanSignalVisualizer
Downloaded: 2026-06-15T03:49:12+00:00

信号输入模式不适用于在 Katana 3.2+ 中使用 NetworkMaterialCreate 创建的材质。无法将着色节点的输出连接到 NetworkMaterialCreate 组外部的节点。

PrmanSignalVisualizer 是一个调试节点，可用于检查场景中 Pattern 节点对材质的影响。该节点会隔离选定的 Pattern 输出，并在分配了该材质的对象上将其可视化为常量颜色。它还可用于隔离来自 PxrLayer 或 PxrLayerMixer 的 vstruct 输出。这类似于其他桥接产品中的 "solo" 操作。

Parameters（参数）

Mode（模式）

PrmanSignalVisualizer 有两种模式 - 选择端口 和 信号输入。

Material Location（材质位置） 

你希望检查的材质的场景图位置。

Keep Displacement（保留置换）

启用时，会保留正在可视化材质的对象上的置换。禁用时，该对象会在没有置换的情况下渲染。

"Select Port" 模式专用参数

Set from Scenegraph Selection（从场景图选择设置）

在 Scene Graph 中选中一个位置后，按下此按钮来填充 Material Location（材质位置）, Shading Node（着色节点）以及 Output Port（输出端口） 参数。

Shading Node（着色节点）

所选 Pattern 节点的名称。当你在 Set from Scenegraph Selection（从场景图选择设置） 弹出窗口中选择输出端口时，此项会自动填充。

Output Port（输出端口） 

所选输出的名称。当你在 Set from Scenegraph Selection（从场景图选择设置） 弹出窗口中选择输出端口时，此项会自动填充。

Clear Node and Port（清除节点和端口） 

此按钮会重置只读的 Shading Node（着色节点） 和 Output Port（输出端口） 参数

使用节点

选择端口模式

  1. 在 Node Graph 中，将该节点添加到创建材质的位置之后

  2. 在 Scene Graph 中选择一个位置 - 可以是材质位置，也可以是分配了本地或继承材质的位置

  3. 按下 Set from Scenegraph Selection（从场景图选择设置） 按钮（位于 PrmanSignalVisualizer 上）

  1. 会弹出一个窗口，显示所选位置上材质的着色网络。这些节点由 Katana 自动布局，因此该布局可能与原始材质的布局不一致。

  2. 在此窗口中，选择你希望可视化的着色节点输出。这会填充 Material Location（材质位置）, Shading Node（着色节点）以及 Output Port（输出端口） 等 PrmanSignalVisualizer 参数。

  1. 所有分配了该材质的位置现在都会使用隔离信号进行渲染。

信号输入模式

  1. 在 Node Graph 中，将该节点添加到创建材质的位置之后。对于此模式，将它放在 NetworkMaterial 节点后面效果很好。

  2. 将一个着色节点的输出连线到 signal（信号） 输入（PrmanSignalVisualizer） 

  1. 将材质的场景图位置添加到 Material Location（材质位置） 参数

  1. 所有分配了该材质的位置现在都会使用隔离信号进行渲染
