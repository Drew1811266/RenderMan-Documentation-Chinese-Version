Light Filters in Katana

Downloaded: 2026-06-15T03:49:12+00:00

Light filters 可以在 GafferThree 中创建。 

Shared Light Filters（共享灯光滤镜）

要共享 light filter，请先选择该 light filter，然后选择想要共享它的 lights 并右键单击，再从菜单中选择 Share Light Filters，如下所示。生成的 light filter 随后会被共享，并显示在每个 light 下方。

Parented Light Filters（作为子级的灯光滤镜）

在选中某个 light 时创建的 light filters，会自动成为所选 light 的子级。

 

Lock Size Cone Angle

只有将 Cone Angle 连接到 Cookie 和 Barn filters 的 lights，才可以选择让 light filter 效果匹配该 light 的 cone angle 参数。

 

Coordinate System（坐标系统）

  • Coordinate

  • Light + Light Filter

  • Light Filter

 

使用 Light Filters

所有 light filters 都带有 Viewer Modifiers，用于在 Viewer 中显示这些 light filters。在 Viewer 中设置 light filters 时，这些 Viewer Modifiers 有助于近似 light 的形状和颜色。对于 Barn、Cookie 和 Gobo light filters，可以使用 Guide Size 参数；它位于 viewer 组中，也就是 PrmanLightFilterCreate macro 底部，用于调整 light filter projection guides 的长度。

提供的 light filters 还带有 Viewer Manipulators，可在 Viewer 中交互式调整 light filter 参数。

 

其他说明

如果物理 Barn 或 Cookie 的位置与 light 位置完全对齐，或位于 light 后方，则不会显示 barn 或 cookie guide lines。当你将 filter 移离 light 时，guides 会出现并变得更加聚焦。
