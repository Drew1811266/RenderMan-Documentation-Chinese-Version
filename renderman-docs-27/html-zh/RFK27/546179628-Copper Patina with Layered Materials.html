使用分层材质制作铜绿

Source: https://rmanwiki-27.pixar.com/space/RFK27/546179628/Copper+Patina+with+Layered+Materials
Downloaded: 2026-06-15T03:49:12+00:00

RenderMan 允许通过分层材质创建复杂外观。这意味着汽车漆、瓶身标签、污垢或泥浆等效果都可以高效分层后用于渲染。

下面是一个分步示例，演示如何在青铜雕像上创建铜绿。在本教程底部，可以下载一个 Katana 场景，其中包含重现此图像所需的组件。


基础场景准备

加载 layeredMaterial_prep.katana 教程场景，其中包含用于加载雕像几何体、地面平面和聚光灯的框架。此时该场景已可渲染；不过我们还没有为雕像指派材质，因此它会使用 RenderMan 的默认 Bxdf 着色器：PxrDiffuse 来渲染。


步骤 1：PxrLayerSurface

创建着色网络，并为对象指派一个 PxrLayerSurface Bxdf。这是一个框架，会告诉 RenderMan 为已指派的对象渲染哪些材质（新节点以红色显示）：

此时可以使用新的着色网络进行渲染，但这个材质本身能做的事情不多；该节点的属性集合有限。原因是 PxrLayerSurface 被设计为一个 Bxdf，用于在其上组合多层 pattern。PxrLayerSurface 只包含会应用到所有层的全局属性。颜色和纹理等层专用属性来自连接到 inputMaterial 属性的上游材质。

从这里开始，规划非常重要，这样可以避免重复工作，并让材质网络尽可能简单且实用。在研究如何在铜或其他金属上创建铜绿时，会发现这是一个多阶段项目，可能涉及多种方法。根据目标外观，材质的布局会有所不同。例如，加热金属产生不同颜色的效果，可以使用 PxrLayerSurface 中的 iridescence PxrLayerSurface 属性来复现。创建铜绿的另一种方法是使用醋和盐在金属上产生结壳，这也是我们将在这里实验的外观。

需要两个基础材质层：

  1. 类似青铜或铜的金属层
  2. 铜绿层，也就是有些结壳、颜色为蓝绿色的漫反射层

步骤 2：PxrLayerMixer

创建一个  PxrLayerMixer 并连接其 out 端口连接到 inputMaterial PxrLayerSurface 的端口。这样就设置了用于连接所有层的节点。来看一下 PxrLayerMixer 着色节点的默认状态：

这里有一个 BaseLayer 以及另外四个层，编号为 1-4。注意 Layer 1 默认启用，因为通常假设至少会有两个层进入 PxrLayerMixer。如果某个层已启用但未连接，整个渲染会变黑。我们会先专注于构建基础层，所以先禁用 Layer 1 直到把铜绿层接入进来。注意，你可以将其他 PxrLayerMixer 连接到层中，以便使用外观集合并将它们遮罩在一起。如果想在其他位置复用网络，这很有用。本示例只使用两个层，即 Base 和 Layer 1。

步骤 3：基础层

Copper/Bronze 将作为我们的“基础”层。创建一个 PxrLayer 并连接其 pxrMaterialOut 端口连接到 Base Layer PxrLayerMixer 的输入端口。

为了近似铜材质，我们设置漫反射和镜面反射，然后添加一些程序化凹凸，让铜具有不错的纹理。这些设置有些任意，因为它们受艺术指导控制；但我们的目标是较旧、偏暗的铜或青铜外观：

 

步骤 4：铜绿层

铜绿将作为我们的 "Layer 1"，位于基础层上方。创建另一个 PxrLayer 节点，并连接其 pxrMaterialOut output 端口连接到 Layer 1 PxrLayerMixer 的输入端口。注意，如果你在处理 基础层 时 禁用了 Layer 1，可能需要重新 启用它。铜绿着色使用从真实图像采样的漫反射颜色，并带有一些漫反射粗糙度，以帮助表现铜绿结壳的“粉状”外观。我们添加了 PxrFractal 和凹凸 pattern 来提高真实感。前一个凹凸 pattern 在此层中再次使用以保持一致，但你也可以在这里使用不同 pattern 来进一步提升真实感。

此层会完全覆盖 Base 层，直到我们为 Layer 1 Mask添加输入。

步骤 5：铜绿遮罩

我们可以用纹理来完成这一点，但这里改用 PxrDirt 节点来控制铜绿和金属出现的位置。PxrDirt 是一个遮挡着色器，可根据几何体中的局部拓扑（即凹槽和缝隙）产生着色效果。创建新的 PrmanShadingNode，并将节点类型设置为 PxrDirt。它的输出是颜色，但由于我们要创建遮罩，可以将 resultR连接到 Layer 1 Mask PxrLayerMixer 的输入（也可以使用 PxrToFloat 将颜色转换为标量值）。

 

为了产生最终图像中看到的风化图案，我们提高 Bias Normal 在 PxrDirt 中的 Y 分量属性，使结果纵向拉伸，仿佛雨水和风化痕迹沿雕像流下。PxrDirt 的 Unoccluded 属性由 PxrFractal 和 PxrGamma 分别驱动，用于打散结果并控制强度。注意位于 PxrDirt 下方的额外 PSN_PxrConstant 节点。通过将 PxrDirt 输出 resultRGB 连接到 emitColor PxrConstant 的输入，我们可以将 PxrLayerSurface Bxdf 替换为 PxrConstant Bxdf，从而单独查看污垢层遮罩的渲染结果。
