Holdouts in Katana

Source: https://rmanwiki-27.pixar.com/space/RFK27/546179038/Holdouts+in+Katana
Downloaded: 2026-06-15T03:49:12+00:00

Holdouts（遮罩保留） 可以使用 PrmanHoldoutCreate macro 创建。

 PrmanHoldoutCreate macro 有多个 parameters，可用于控制 render 的行为：

holdoutGeo：在这里添加所有要作为 holdout 的 geometry。

Mode（模式）：

  • Off：关闭 holdout workflow。忽略此 macro。

  • Composite：将 holdout image composite 到 imagePlaneSettings 中指定的 background image 之上。

  • No Composite：创建 holdout image，但不 composite background image。

outputAOVs：将 shadow 和 occluded AOVs Write（写入）到 renderLocation。image names 将是 holdout_shadowAOV.exr 和 holdout_occludedAOV.exr。

renderLocation：shadow 和 occluded AOVs 的 render directory。除非另有指定，否则 renderLocation 会设为 $KATANA_TMPDIR。勾选 outputAOVs 时此 parameter 可用。

imagePlaneSettings：Filename parameter 用于指定 background image。此 group 中的所有 settings 都与 PxrImageDisplayFilter上的相同 parameters 一致。该 macro 使用此 sample filter，将 holdout image 和 shadow AOV composite 到 background image 之上。
