Basic Setup in Katana（Katana 基础设置）

Step 1（步骤 1）：

添加一个 PrmanDenoiseChannelDefine node，用于创建 denoiser 所需的全部 DisplayChannels。

Step 2（步骤 2）：

添加一个 PrmanDenoiseRenderOutputDefine node，用于为 beauty pass 输出 multichannel OpenEXR file。该 file 包含所有必需的 denoise channels。

将 renderLocation parameter 设为 output image 的位置。denoiser 期望 images 采用以下形式 <NAME>.<FRAME_NUMBER>.exr（例如 image.1001.exr）。在 Katana 中，可以通过 filename 中 # 的数量指定 frame number 的 digits 数（例如 image.##.exr 会得到 image.01.exr).

Step 3（步骤 3）：

将 pixelfiltermode 设置为 importance（如果你已将其从默认值改掉）。此设置在 PrmanGlobalStatements 中完成。

Custom setup in Katana（Katana 自定义设置）

这两个 PrmanDenoiseChannelDefine 和 PrmanDenoiseRenderOutputDefine nodes 都是 macros。可以 command-click 查看这些 macros 内部，并在必要时修改 settings。

如果你的 pipeline 不支持 multichannel EXRs，可以使用 separate RenderOutputDefine nodes，为以下 node 中定义的每个 channel 创建一个 output： PrmanDenoiseChannelDefine。然后使用 $RMANTREE/bin/exrmerge 将 AOVs 合并成 multichannel EXR。将这个 merged multichannel EXR 用于 denoiser。使用 Katana 的 RenderOutputDefine merge 创建的 combined EXRs 无法用于 denoiser。

在任何为 denoising 创建的 RenderOutputDefine nodes 上，type 都应设为 "raw"，以避免 Katana 进行任何 post-processing。然后确保 asrgba 处于 off 状态，使 channel names 与 denoiser 期望的一致。 

请参阅 denoise_teapots.katana 这个位于 RfK 的 Examples directory 中的 example file，了解如何设置 denoising。 

Running the Denoiser（运行 Denoiser）：

有几种 options 可用于对 Katana renders 执行 denoise。RenderMan 同时提供名为 denoise_batch 的 command line denoise tool 和 Denoise UI。

Option 1：RenderOutputDefine Script（Single Frame）

  1. 创建一个 RenderOutputDefine。

  2. 将此 node 连接到前一个 RenderOutputDefine node。

  3. 输入 denoise_primary，将 outputName 设为 denoise_primary。

  4. 通过选择 script option，将 type 设为 script。这会导致 scriptInput parameter 显示 error。忽略它并继续下一步。

  5. 通过选择 primary option，将 scriptInput 设为 primary。

  6. 将 scriptCommand 设为 expression。

  7. 然后输入 expression

    '"' + path.join(getenv("RMANTREE", ''), 'bin', 'denoise_batch') + '" -o ' + path.join(katanaTmpDir, 'denoised') + ' $INPUT'

此时，后续的 Render node 应该有两个 outputs：primary（raw）和 denoise_primary（script - 在 primary 上运行）。

Disk Render 完成后，original image 会位于 /tmp/katana/image.1.exr，denoised image 会位于 /tmp/katana/denoised/image.1.exr。

Option 2：command line（Crossframe 或 Single Frame）

使用 katana --batch 或 katana queue render 出 images。然后运行以下 command： 

$RMANTREE/bin/denoise_batch -o /path/to/directory/for/denoised/images imagename.*.exr

请参阅 Denoiser Command Line documentation，了解可用 settings 的更多信息。

Option 3：Denoise UI（Crossframe 或 Single Frame）

  1. 将 images 从 Katana 作为 disk 或 batch render 输出。

  2. PRMan 25 shelf 中有一个名为 RenderMan Denoiser 的 script。运行此 script 可启动 Denoise UI。也可以直接从以下位置访问 UI： $RMANTREE/bin/denoise 

  3. 在 UI 中，选择 "Add Directory..."，并选择包含已从 Katana render 出来的 images 的 directory。

  4. 接下来选择 folder icon，选择 denoised images 要保存到的 directory。

  5. 点击 "start" 开始 denoising。

请参阅 Denoiser UI documentation，了解可用 settings 的更多信息。
