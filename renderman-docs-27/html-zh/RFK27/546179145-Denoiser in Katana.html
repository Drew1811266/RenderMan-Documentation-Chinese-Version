Katana 中的 Denoiser

Katana 基础设置

步骤 1：

添加一个 PrmanDenoiseChannelDefine 节点，用于创建 Denoiser 所需的全部 DisplayChannels。

步骤 2：

添加一个 PrmanDenoiseRenderOutputDefine 节点，用于为 beauty pass 输出多通道 OpenEXR 文件。该文件包含所有必需的降噪通道。

将 renderLocation 参数设为输出图像的位置。Denoiser 期望图像采用以下形式 <NAME>.<FRAME_NUMBER>.exr（例如 image.1001.exr）。在 Katana 中，可以通过文件名中 # 的数量指定帧编号的位数（例如 image.##.exr 会得到 image.01.exr).

步骤 3：

将 pixelfiltermode 设置为 importance（如果你已将其从默认值改掉）。此设置在 PrmanGlobalStatements 中完成。

Katana 自定义设置

这两个 PrmanDenoiseChannelDefine 和 PrmanDenoiseRenderOutputDefine 节点都是宏。可以按住 Command 点击查看这些宏内部，并在必要时修改设置。

如果你的管线不支持多通道 EXR，可以使用单独的 RenderOutputDefine 节点，为以下节点中定义的每个通道创建一个输出： PrmanDenoiseChannelDefine。然后使用 $RMANTREE/bin/exrmerge 将 AOV 合并成多通道 EXR。将合并后的多通道 EXR 用于 Denoiser。使用 Katana 的 RenderOutputDefine merge 创建的合并 EXR 无法用于 Denoiser。

在任何为降噪创建的 RenderOutputDefine 节点上，type 都应设为 "raw"，以避免 Katana 进行任何后处理。然后确保 asrgba 处于 off 状态，使通道名称与 Denoiser 期望的一致。 

请参阅 denoise_teapots.katana 这个位于 RfK 的 Examples 目录中的示例文件，了解如何设置降噪。 

运行 Denoiser：

有几种选项可用于对 Katana 渲染结果执行降噪。RenderMan 同时提供名为 denoise_batch 的命令行降噪工具和 Denoise UI。

选项 1：RenderOutputDefine Script（单帧）

  1. 创建一个 RenderOutputDefine。

  2. 将此节点连接到前一个 RenderOutputDefine 节点。

  3. 输入 denoise_primary，将 outputName 设为 denoise_primary。

  4. 通过选择 script 选项，将 type 设为 script。这会导致 scriptInput 参数显示错误。忽略它并继续下一步。

  5. 通过选择 primary 选项，将 scriptInput 设为 primary。

  6. 将 scriptCommand 设为表达式。

  7. 然后输入表达式

    '"' + path.join(getenv("RMANTREE", ''), 'bin', 'denoise_batch') + '" -o ' + path.join(katanaTmpDir, 'denoised') + ' $INPUT'

此时，后续的 Render 节点应该有两个输出：primary（raw）和 denoise_primary（script - 在 primary 上运行）。

Disk Render 完成后，原始图像会位于 /tmp/katana/image.1.exr，降噪后的图像会位于 /tmp/katana/denoised/image.1.exr。

选项 2：命令行（跨帧或单帧）

使用 katana --batch 或 katana queue 渲染图像。然后运行以下命令： 

$RMANTREE/bin/denoise_batch -o /path/to/directory/for/denoised/images imagename.*.exr

请参阅 Denoiser Command Line 文档，了解可用设置的更多信息。

选项 3：Denoise UI（跨帧或单帧）

  1. 将图像从 Katana 输出为磁盘渲染或批渲染。

  2. PRMan 25 shelf 中有一个名为 RenderMan Denoiser 的脚本。运行此脚本可启动 Denoise UI。也可以直接从以下位置访问 UI： $RMANTREE/bin/denoise 

  3. 在 UI 中，选择 "Add Directory..."，并选择包含已从 Katana 渲染出的图像的目录。

  4. 接下来选择文件夹图标，选择降噪后图像要保存到的目录。

  5. 点击 "start" 开始降噪。

请参阅 Denoiser UI 文档，了解可用设置的更多信息。
