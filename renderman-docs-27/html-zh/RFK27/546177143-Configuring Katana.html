Configuring Katana

Source: https://rmanwiki-27.pixar.com/space/RFK27/546177143/Configuring+Katana
Downloaded: 2026-06-15T03:49:12+00:00

安装 RenderMan for Katana (RfK) 后，只需少量设置就可以开始生成渲染！在运行 Katana 本身之前，你需要正确设置环境，之后 Katana 会自动加载 RfK 插件。进入 Katana 后，你需要用摄影机和几何体设置场景，添加 RenderMan (PRMan) 灯光和材质，然后就可以开始渲染。  

如果安装了多个渲染器，要使用 RenderMan 渲染，请将 DEFAULT_RENDERER 环境变量设置为 "prman"

Set Up Environment（设置环境）

要使用 RenderMan for Katana，必须将 RfK 插件路径添加到你的 KATANA_RESOURCES 搜索路径：

setenv KATANA_RESOURCES ${KATANA_RESOURCES}:${RFKTREE}/plugins/katana8.0

其中 RFKTREE 是该软件包的安装位置，Katana 版本后缀设置为 major.minor 受支持 Katana 版本之一的字符串。这会告诉 Katana 在哪里查找 RfK 插件，以及渲染和管理场景所需的支持模块。

RenderMan Nodes and Macros（RenderMan 节点和宏）

任何 Katana 渲染器插件的目的，都是让艺术家使用指定渲染器的全部能力生成图像。若要开始利用 Katana 的 RenderMan 插件能力，我们需要先讨论 PRMan 专用节点和参数：

PrmanGlobalStatements

此节点包含大多数 PRMan 世界状态选项。这里可以找到诸如 pixelVariance, maxsamples。

PrmanObjectStatements

此处用于设置对象专属的 PRMan 属性。可在这里查找 visibility（可见性） 和 trace（追踪） 等属性。

PrmanIntegratorSettings

此节点控制与每个积分器相关的设置。将这些控制项分离出来，可以更轻松地集成你自己的自定义积分器。

PrmanCameraSettings

此节点控制每个摄影机的设置，例如景深和视场角。

PrmanDenoiseChannelDefine

此宏会创建 降噪 工具所需的所有 DisplayChannels。

PrmanDenoiseRenderOutputDefine

此宏用于定义降噪器所使用的多通道 OpenEXR 所需的输出。

PrmanHoldoutCreate

此宏会自动完成遮罩保留所需 AOV 和 Sample Filters（采样过滤器）的设置流程。

PrmanOutputChannelDefine

此节点用于定义任意输出变量（AOV），这些变量用于在渲染计算期间从着色器或渲染器输出额外数据。

PrmanShadingNode

此着色节点用于将 PRMan 着色器（Bxdf 和图案节点）添加到更大的着色网络中（与 NetworkMaterial 节点一起）。

PrmanVolume

此节点提供一个简单接口，用于创建体积盒或导入 VDB 文件。

PrmanOpDebug

此节点提供一种方式，用于检查某些 Ops 对属性状态的影响。

PrmanSignalVisualizer

此宏可让你隔离并可视化着色图案输出。
