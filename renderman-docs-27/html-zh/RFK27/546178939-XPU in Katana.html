XPU in Katana

Source: https://rmanwiki-27.pixar.com/space/RFK27/546178939/XPU+in+Katana
Downloaded: 2026-06-15T03:49:12+00:00

使用 XPU 渲染时，还有一些额外 options 可供配置。为了支持 studios 希望采用的不同工作方式，RfK 提供了多种设置和 override 这些 settings 的方法。

PrmanGlobalStatements 有一个 renderVariant setting，可用于选择使用 RIS 还是 XPU 渲染。还有额外的 XPU settings，可限制它只使用 GPU 或只使用 CPU。XPU - CPU 在需要在没有受支持 GPU 的 machine 上渲染时尤其有用。

如果使用 XPU 或 XPU-GPU 渲染，还会有一个 GPU Selection option；如果你的 machine 有多个 GPU，可以在这里选择要使用的 GPU。

对于 interactive renders（Live/Preview renders），Preferences menu 中的 settings 会优先于 renderVariant selection，除非 "interactiveXPUSettingsOverride" preference 被禁用。这样可以让每个人根据自己的 machine 自定义 settings，而不是被 show settings 或 batch renders 的默认值约束。仍然需要将 PrmanGlobalStatements 的 renderVariant 设为某个 XPU option，才能在 scene 中启用 XPU；但会使用 enableGPU/enableCPU 和 gpuSelection preferences，而不是 PGS settings。

对于 batch renders，RfK 提供了一种方式，可在 katana batch command 中 override renderVariant 和 GPU Selection，例如：katana --batch --katana-file=myfile.katana --render-node=Render -t 1 -- --prmanRenderVariant=xpugpu --prmanGpuSelection=0。请注意，-- 必须位于 prman command line arguments 之前。prmanRenderVariant 的 options 为 xpugpu、xpucpu、xpu 和 prman。prmanGpuSelection option 接受一个整数值，表示 gpu number。
