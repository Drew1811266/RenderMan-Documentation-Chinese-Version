RenderMan 27 for Katana

Katana 强大而灵活的 look development（外观开发）环境与 RenderMan 最先进的渲染技术相结合，为用户提供面向高端照明和渲染的出色开箱即用体验。

RenderMan for Katana (RfK) 为 Katana 用户提供对 RenderMan 强大渲染系统的无缝访问；用户可以利用该 plugin（插件）的灵活性，创建最佳的自定义管线。

Installation（安装）

RenderMan for Katana 需要安装 RenderMan Pro Server 以及 RfK plugin（插件）才能正常运行。RenderMan 安装程序会自动完成这些工作。

About the Plugin（关于插件）

用于 Katana 的 RenderMan plugin（插件）在生产管线中扮演双重角色；它既是用户从 Katana 场景生成渲染图像的桥梁，也提供访问工具以构建场景中的 3D 元素。  

虽然 RfK 通常被称为 “plugin”（插件），但它实际上是一组 plugins（插件）、nodes（节点）、macros（宏）和其他工具；这些组件共同让 Katana 用户能够完整访问 PRMan 渲染系统。

RfK plugin（插件）包中包含：

  • Rendering plugin（渲染插件）

  • Renderer introspection plugin（渲染器自省插件）

  • 用于 interactive renders（交互式渲染）的设备驱动程序

  • 用于 PRMan lights（PRMan 灯光）的 GafferThree 扩展

  • 用于 PRMan lights（PRMan 灯光）的 Viewer plugins（查看器插件）和 manipulators（操纵器）

  • 用于 Denoise 和 Holdouts 等功能的 macros（宏）

  • 用于 PRMan render configuration（PRMan 渲染配置）的 nodes（节点）

  • 教程场景

无需“加载” RfK plugin（插件）。一旦它已经 安装 并 配置 Katana 会自动找到该软件包，并按需加载各个模块。

Live Rendering（实时渲染）

大幅 改进后的 Live Rendering（实时渲染）是一种 iterative rendering（迭代渲染）：你可以修改场景的某些部分，并看到它们在渲染中更新。实时渲染提升了照明和着色迭代效率。RfK 支持以下功能：

  • 编辑材质参数，以及添加、删除和编辑 shading nodes（着色节点）

  • 编辑灯光和 light filters（灯光滤镜）的属性与变换，包括 mesh lights（网格灯）。

  • 编辑对象的形状和变换

  • 在 Live Rendering（实时渲染）期间导入和导出资产

  • 在 Monitor（监视器）中使用 pixel probe（像素探针）进行 Object picking（对象拾取）。

  • 在实时渲染期间添加灯光（analytic lights 和 mesh lights）并创建对象。

  • renderSettings（渲染设置）的实时更新（ROI、cropWindow、resolution）

  • integrators（积分器）和 options（选项）的实时更新（在 PRMan 支持的范围内）

  • AOVs 的实时创作

Lighting and Shading（照明与着色）

RenderMan 随附内置的生产级 shaders（着色器）和 lights（灯光），这些 shaders（着色器） 和 lights（灯光） 正是 Pixar Animation Studios 在故事长片中使用的着色器和灯光。Pixar shaders（Pixar 着色器）会在启动时自动加载，并可通过标准 Katana material workflows（材质工作流）使用。Pixar lights（Pixar 灯光）可以在 GafferThree 中通过 single key-stroke（单次按键） 创建，也可以直接通过菜单创建。还可以只需选择几何体并创建 PxrMeshLight 来创建 Mesh lights（网格灯）。还包含用于内置灯光和 light filters（灯光滤镜）的 Viewer Plugins（查看器插件）和 Manipulators（操纵器）。

Plugin Customization（插件自定义）

RenderMan for Katana 支持通过 custom Ops（自定义 Ops）进行工作室自定义。 

Image Tool (it)（图像工具 it）

RenderMan 包含 “it”，这是一个稳健的 framebuffer/render view window（帧缓冲/渲染视图窗口），提供完整的 floating point support（浮点支持）和强大灵活的 catalog（目录）。“it” 是快速强大的成像工具，具备通常只有高端独立产品才有的生产级图像处理和合成功能。得益于它与 plugin（插件）的集成以及全新的 Python 脚本接口，“it” 让用户能够把复杂的渲染后图像操作纳入自己的渲染管线。

RenderMan Features（RenderMan 功能）

Photo Realistic RenderMan (PRMan) 设计目标是在快速、易用的同时生成生产级渲染；global illumination（全局照明）开箱即用，live rendering（实时渲染）为艺术家提供快速迭代。RenderMan for Katana 通过标准节点和 PRMan 专用节点支持 PRMan 的各个方面。

Interactive Rendering（交互式渲染）

RenderMan 支持在渲染期间更新场景。支持许多功能，包括即时导入和导出资产、动画、变换、灯光和材质参数更改等！

Fast Ray Tracing Core（快速光线追踪核心）

为配合 interactive rendering（交互式渲染）的改进，RenderMan 的 ray tracing core（光线追踪核心）也得到改进。速度和质量均有提升，可在艺术家工作时提供快速、无缝的体验。

Immediate Rendering（即时渲染）

Stop（停止等待）渲染开始。RenderMan 在交互式会话中渲染时会跳过 RIB 生成过程，让用户尽快获得控制权和结果。

Analytic Area Lights（解析区域光）

Area Lights（区域光）是创建 physically accurate lighting（物理准确照明）的强大方式，同时优化 sampling budgets（采样预算），进而提升性能。PRMan 的 area lights（区域光） 支持 light filters（灯光滤镜），以及 light-linking（灯光链接）、groups（组）、per-group AOVs（逐组 AOV）、IES profiles（IES 配置文件）和 portals（门户）。

Global Illumination（全局照明）

RenderMan for Katana 支持 Global Illumination（全局照明）效果，有时也称为 “Indirect Illumination”（间接照明）。这是一类微妙的 diffuse shading effects（漫反射着色效果），通过从给定点多次采样场景照明来创建，可产生高度真实的结果。Occlusion（遮蔽）仅采样某点的 “coverage”（覆盖率）（成本低于 Indirect Illumination），也会自动受支持，或可通过 PxrOcclusion Integrator（积分器）支持。

Volume Rendering（体积渲染）

RfK 支持 PRMan 的一流 volumetric（体积） 渲染能力，可以创建包括 motion blur（运动模糊）、multi-scattering（多重散射）、non-homogenous（非均匀）等在内的体积效果。

True Curved Surfaces（真实曲面）

使用 RenderMan 渲染 NURBS 和 subdivision（细分） surfaces（曲面）时，tessellation settings（曲面细分设置）从来不是问题。由于 RenderMan 渲染真实曲面，NURBS 和 subdivision surfaces（细分曲面）永远不会出现 faceting artifacts（平面化伪影）。

Subsurface Scattering（次表面散射）

RenderMan for Katana 支持 path-traced subsurface scattering（路径追踪次表面散射），这是逼真渲染半透明材质（如皮肤、肉、脂肪、水果、牛奶、大理石等）的重要效果。Subsurface scattering（次表面散射）负责材质内部的 color bleeding（颜色渗透），或光线跨越阴影边界的扩散等效果。RfK 通过 RenderMan 随附的材质支持 ray-traced subsurface scattering（光线追踪次表面散射）。

Secondary Outputs（次级输出）

RenderMan for Katana 支持 secondary outputs（次级输出，即 AOVs/Arbitrary Output Variables，任意输出变量），它们可以是 shader（着色器）中包含的任何任意信息。用户还可以利用 Light Path Expressions（光路径表达式） (LPE) 对输出进行强大的控制。

Down the Pipe(line)（进入管线）

RenderMan for Katana 是潜在复杂生产管线中的一环。为了把工作从 script-to-screen（脚本到银幕）完成，你可能还需要更多工具，这正是 Tractor 发挥作用的地方。将 RfK 与 Tractor 和 RenderMan Pro Server 配合使用，用户可以实现从桌面到 renderfarm（渲染农场）的无缝集成渲染工作流。

Tractor

Tractor 是一个用于把排队作业中的任务分发到执行服务器农场的系统，可管理来自众多用户的大量并发作业队列，并强制执行依赖关系和调度策略。Tractor 软件包包含中央排队机制、远程执行服务器、监控子系统和基于浏览器的用户界面。

Tractor 作为单独的独立产品提供，并拥有自己的文档。

RenderMan Pro Server（RenderMan Pro 服务器）

RenderMan Pro Server 是高性能渲染管线的基础。它提供渲染器及其相应渲染实用程序的 standalone implementations（独立实现），并提供 API 访问，让有经验的用户扩展功能。作为核心技术，RenderMan Pro Server 是所有像素流出的源泉。
