hdPrman in Katana

Source: https://rmanwiki-27.pixar.com/space/RFK27/546177441/hdPrman+in+Katana
Downloaded: 2026-06-15T03:49:12+00:00

Katana 5.0 或更高版本，以及 RenderMan 26.0 或更高版本，在 Katana Viewer 中支持我们的 Hydra render delegate（Hydra 渲染委托）hdPrman。   

Environment Variable Configuration（环境变量配置）

运行该 render delegate（渲染委托）需要进行一些额外的环境变量设置。

Environment Variable（环境变量）

Purpose（用途）

Example with Default Install（默认安装示例）

FNPXR_PLUGINPATH

Katana 需要它来查找随 RfK 提供的 hdPrman 插件

/opt/pixar/RenderManForKatana-26.0/plugins/katana6.0/usd

RMAN_SHADERPATH

RenderMan 需要它来查找随 hdPrman 提供的着色器。如果尚未设置 $RMAN_SHADERPATH，应追加 $RMANTREE/lib/shaders 中的默认着色器路径。

$RMAN_SHADERPATH:/opt/pixar/RenderManForKatana-26.0/plugins/katana6.0/usd/resources/shaders

RMAN_RIXPLUGINPATH

hdPrman 需要它来加载纹理。如果尚未设置 $RMAN_RIXPLUGINPATH，应追加 $RMANTREE/lib/plugins 中的默认插件路径。

$RMAN_RIXPLUGINPATH:/opt/pixar/RenderManForKatana-26.0/plugins/katana6.0/usd

Running hdPrman in the Viewer（在 Viewer 中运行 hdPrman）

使用 Viewer 选项卡中的 Display menu（显示菜单），可以将 Render Delegate 从 GL 切换为某个 RenderMan delegate。目前 RenderMan 有 4 个可用选项，用于通过 RIS 或某个 XPU 变体进行渲染：RIS、XPU-CPU、XPU-GPU 和 XPU。  

如果想在 Viewer 中看到 RenderMan 材质和灯光，需要在 Display menu 中禁用 Basic Material 和 Basic Lighting 复选框。如果不禁用这些设置，渲染将使用一个 dome light（穹顶灯）和一个灰色漫反射材质。将 Selection 设置为 "outline" 而不是 "fill" 也很有用，这样可以更容易看到所选对象的材质。这些设置可以作为布局的一部分保存，以便将来使用。

Known Limitations（已知限制）

目前 Viewer render delegate 可渲染的内容存在一些限制。以下 RenderMan 功能尚不受支持。请注意，此列表仅表示最新 Katana 版本所支持的内容。

Lighting（照明）

  • Mute/solo（静音/独显）

  • Mesh lights（网格灯）

  • Portal lights（门户灯）

  • Camera Visibility on lights（灯光上的摄影机可见性）

Render Settings（渲染设置）

  • RenderMan options（RenderMan 选项）

  • Integrators（积分器）

  • Display/Sample Filters（显示/采样过滤器）

  • Projections and Camera settings（投影和摄影机设置）

Geometry（几何体）

  • NURBS

  • Volumes（体积）

Limitations specific to rendering through Katana attributes（通过 Katana 属性渲染时的特定限制）

  • Light linking（灯光链接）

  • Light filters（灯光过滤器）

  • PrmanObjectStatements

  • Displacement（置换）
