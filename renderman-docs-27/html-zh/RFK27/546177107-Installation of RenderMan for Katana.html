Installation and Licensing（安装和许可）

RenderMan for Katana (RfK) 可通过以下两种方式之一安装：

  1. 使用 RenderMan Installer（RenderMan 安装程序），它会管理 RenderMan Pro Server（实际渲染器）以及 RenderMan for Katana 的下载、安装和许可。 

  2. 从论坛 Product Downloads（产品下载）页面下载 RenderMan for Katana 和 RenderMan Pro Server 软件包。RPMs 可以手动安装。

参见 RenderMan Installation and Licensing（安装和许可） 文档以了解更详细的说明。

如果安装了 3Delight，则需要将 DEFAULT_RENDERER 环境变量设置为 prman，或将 RenderSettings 中的 renderer 参数改为 prman。
