采样

Source: https://rmanwiki-27.pixar.com/space/RFM27/544542378/Sampling
Downloaded: 2026-06-15T03:49:12+00:00

Sampling 选项卡会通过以下设置 Max Samples 和 Pixel Variance、所选择的 Integrator 及其相关设置、 Default Ray Depths, Clamping, Russian Roulette（可用于缩短路径长度）以及 Filtering来让用户控制基础质量。这主要是控制渲染质量和渲染时间的位置。

在窗口顶部的 Presets 菜单中，有 draft、production 等采样预设。此外，你也可以在那里保存自己的采样预设。

或者，你可以使用参数名称旁三横线图标标出的参数预设。

参数预设可以像这样访问：

IPR Samples

IPR 采样为 IPR 渲染提供独立的 Max Samples 和 Pixel Variance。默认情况下，它设得低于正式渲染设置，以便快速反馈。

交互式细化使用一种技术对图像进行稀疏采样，然后随时间逐步细化，以便在更新场景时提供更快反馈。级别为 0 - 6，其中 0 表示关闭，6 会让渲染器在决定每次只更新多少像素时最激进。更新的像素越少，重新采样帧时越可能先看到一些模糊或重影伪影，但也能更快提供更新提示。

Progressive Pixels

注意：在 XPU 模式下运行 IPR 时，如果将 Interactive Refinement 设为高于 1，会激活 Progressive Pixels。

积分器

让你选择用于渲染的 Integrator。默认的 PxrPathTracer 适用于大多数渲染情况。

有关各积分器设置的信息，请参阅各积分器页面：

  • PxrDefault
  • PxrDirectLighting
  • PxrOcclusion
  • PxrPathTracer
  • PxrVCM
  • PxrValidateBxdf
  • PxrVisualizer

Default Ray Depths

Max Specular Depth: 反射和折射的反弹次数。除非需要多次反弹效果，或有玻璃片/窗户，否则值 1 或 2 已经足够。此属性可以在单个对象上覆盖。

Max Diffuse Depth:  间接照明的漫反射反弹数量。值 1 通常足够；更大的值会更慢。此属性可以在单个对象上覆盖。

Filter

Filter: 用于控制已渲染像素的 过滤器。box、triangle、disk、gaussian 和 blackman-harris 是柔化过滤器；catmull-rom、sinc、mitchell、separable-catmull-rom、lanczos 和 bessel 是锐化过滤器。Separable-catmull-rom 是一个不错的折中选择。

Filter Size: 像素过滤器的宽度。通常为 1 到 6。设为 0 会使用所选像素过滤器的标准宽度。


Integrator Overrides

当需要覆盖某个渲染层的设置时，按以下步骤操作：

  1. 在 Render Setup 中创建所需渲染层，并确认当前处于这个渲染层（visible 已勾选，即眼睛图标）
  2. 在 Render Settings 中为此渲染层选择要使用的积分器。这会创建所需节点。
  3. 右键单击积分器，然后选择 "Create Connection Override for Visible Layer"

4. 要连接这个新的积分器，只需在菜单中键入你选择的积分器名称，并在名称后添加 .msg。例如输入：PxrOcclusion.msg，然后按 Enter/Return。这会连接积分器，并且覆盖会在 Render Settings 中以橙色高亮显示。
