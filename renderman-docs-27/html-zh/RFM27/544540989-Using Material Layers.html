Using Material Layers

Source: https://rmanwiki-27.pixar.com/space/RFM27/544540989/Using+Material+Layers
Downloaded: 2026-06-15T03:49:12+00:00

RenderMan 允许通过 material layering（材质分层）创建复杂 look（外观）。这意味着 car paint（车漆）、瓶身 labels（标签）、dirt（污垢）或 mud（泥）等 effects 都能以高效分层方式进行 rendering（渲染）。

下面是一个 step by step（分步）示例：在青铜雕像上创建 patina（铜绿），然后将其 composited（合成）到 plate（背景板）中。页面末尾有可供尝试的文件包！



Step 1：首先应为对象指定一个 PxrLayerSurface material（材质）。这会告诉 RenderMan 为该对象渲染哪些 materials。除非你偏好不同 specular model（镜面模型），否则保持默认值。此项必须在已指定的 PxrLayerSurface 中更改。本示例选择 GGX 作为 specular model。在 Maya 中，我们会自动创建一个 base 和 upper layer。

在 Maya 中选择 PxrLayerSurface 时，我们会为用户创建 upper layer；因此应在 PxrLayerMixer 中禁用它，以便你在工作时能看到 base layer。


从这里开始，planning（规划）非常重要，可以避免重复工作，并让 material network（材质网络）尽可能简单且有用。研究如何为铜和其他金属添加真实 patina（铜绿）时，你会发现这是一个包含不同方法的多阶段项目。从加热金属以产生不同颜色（这种 look 很容易通过 PxrLayer Iridescence 实现），到使用醋和盐在金属上形成 crust（硬壳），这些都是可行方法；这里我们将试验后一种 look。

需要两个基本 material layers（材质层）：

  1. 一个类似 bronze（青铜）或 copper（铜）的 metal layer（金属层）

  2. 一个 patina（铜绿）层，也就是带一点 crusty（硬壳感）、呈蓝绿色的 diffuse layer（漫反射层）

Step 2：使用 PxrLayerMixer 并确保它连接到 PxrLayerSurface 的 Input Material。技术上：PxrLayerMixer.pxrMaterialOut -> PxrLayerSurface.inputMaterial。如果手动操作，在 Maya 中简单地用鼠标中键拖拽即可。 

这样会设置一个 node（节点），后续所有内容都会连接到它。使用 Base 非常简单：然后按 1 到 4 的顺序将每个额外 PxrLayer 分层叠加在其上。还要注意，可以将其他 PxrLayerMixers 连接到 Layers，以便使用 look 集合并将它们一起 masking（遮罩）。如果想在别处复用 networks，这很有用。本示例只使用两个 layers：Base 和 Layer 1。


Step 3：Copper/Bronze（铜/青铜）通过使用 PxrLayer 并将其连接到 PxrLayerMixer，作为 Base来实现。你可以开始调整它以获得 base look（基础外观）。为了制作 copper（铜），我们会覆盖 specular 和 diffuse properties。这里添加 bump 也很有用，可让铜具有不错的 texture（纹理）。

设置说明：这些设置有些 arbitrary（任意），因为它们受 art direction（艺术指导）影响，但我们想要一种较旧、暗淡的 copper 或 bronze look，并且在 layering 后仍能保持效果。这里可以看到 Diffuse 和 Primary Specular Enabled。请记住，此 PxrLayer 现在是我们的 base，因此不需要更改指定给对象的 PxrLayerSurface material。这使得日后需求变化时，可以灵活重新排列并修改 materials 和 layers。

material look 很大程度上由 diffuse 和 face color 参数中的 colors 控制。每个参数都使用了修改过的 Maya Crater pattern，为颜色提供一些变化。Maya Noise pattern 用于控制 roughness，并通过 PxrClamp，让 roughness 落在有用范围内。同样，你可以选择任何 pattern 或手绘 texture 来制作几乎任何东西！

Step 5：你的 patina layer 是 PxrLayerMixer 的 Layer 1。请注意，Layer 1 在创建时默认启用，但其他 layers 不会启用。在调校上面的 base look 时，我们已将其设为 disabled，以免遮挡 base。


没有 mask，因此此 override 会完全覆盖 Base layer。我们需要创建某种形式的 mask pattern。


Step 6：使用 textures 可以很轻松地做到这一点，不过也可以使用 PxrDirt node 来控制 patina 和 metal 的可见区域。开头可以看到由 PxrDirt node 和 noise pattern 产生的 weathering pattern。PxrDirt node 的 Bias Normal 在 Y 方向为 50.0，用于垂直拉伸结果，就像雨水和风化沿雕像向下流动一样。下面是调整后的 PxrDirt 单独渲染出的 resulting mask。unoccluded color 分别由 PxrFractal 和 PxrGamma 驱动，用于打散结果并控制强度。


Step 7：PxrDirt 输出 color，你可以将 Red result 连接到 Layer 1 mask，也可以在中间使用 PxrToFloat node。然后 render（渲染）！
