在 Maya 中，可以通过 RenderMan Shelf、RenderMan Menu 或 Hypershade Window 创建 RenderMan lights。 

  • 不支持原生 Maya lights。请使用 "Pxr" lights。

  • 在 Maya node editor 中，你可能会看到 lights 上有一个可用的 "Light Data" output connection。尽管 Maya 将其显示为有效 output，但它并不是可连接的 parameter。

RenderMan 中的 Lighting 很灵活，对用户来说也非常直接。

RenderMan 内置了一组基础 geometric lights。这些 specialized shapes - DomeLight、DistantLight、RectLight， 以及 SphereLight （尤其是后两者）- 与任意 geometry（即附加 light Bxdf 的标准 geometric primitives）相比，可以提供更优的 sampling behavior。此外还有一个 PortalLight 用于改善通过窗户、天窗等开口间接照明的 interior scenes 的 lighting，以及一个 Daylight simulation（模拟）。

Light Filters（灯光滤镜） 也可以修改和定向 lighting，提供更多 control 与艺术选项。你可以找到真实世界中的可能性，例如 Gobos 以及 Cookies，也可以找到高级方式，用于从单个 light 链接并控制 object-specific intensities 等结果。


  • 请注意，由于这些 optimizations，spherelight 要求所有 scale transforms 都是 uniform；例如 X、Y 和 Z scale 必须完全相同，以免 sphere shape 被扭曲成 ellipse 或其他形状。

  • 默认情况下，Lights 在 refraction/transmission 中不可见。可以在 light visibility settings 中启用这一点。



请注意，lighting scene 主要有三种方式：

  • Analytic Lights - 这是首选。这类灯光提供更好的 memory usage、flexibility 和 speed。对于 camera visibility，它们的 visibility 只是简单的 on/off。

  • Mesh Lights - 这类灯光非常适合在 lighting 中使用任意 shapes。由于它们是 geometry，因此会消耗更多 memory，复杂 shapes 可能会增加 noise。它们可以像其他 object 一样使用 visibility：camera、shadow/transmission 和 indirect visibility。

  • Emissive Surfaces - 这类表面通常使用 constant bxdf 或 "glow" parameter 来间接照亮 scene。这种方式效率较低，通常应避免使用；但在未来感 objects 和 panels 上进行 texture mapping "lights" 时可能会有帮助！


Analytic Light Types（解析灯光类型）

  • PxrDomeLight - domelight 充当 scene 的 environment light，并且通常会映射 high dynamic range image（HDRI）。

  • PxrRectLight - 此 light 常用于矩形 area lights。

  • PxrDistantLight - 此 light 非常适合作为 Sunlight 或其他 distant source 使用，此时 light rays 近乎平行并覆盖整个 scene。

  • PxrDiskLight - 很适合作为 area light 使用，也可以与 filters 结合来制作 spotlight。

  • PxrSphereLight - 对怀念 point light 的用户来说，此 light 可用于向所有方向照明。

  • PxrEnvDaylight - 一种 daylight simulation shader，你可以提供地球上任意 time 和 location，或者直接把 sunlight 放在想要的位置。

  • PxrPortalLight - 对于间接照明室内等困难 lighting situations，此 light 可与 PxrDomeLight 一起使用，提高 renders 的 quality 和 speed。

Geometric Light Types（几何灯光类型）

  • PxrMeshLight - mesh light 使用一块 geometry 并将其转化为 light source。这比使用 bxdf 的 glow 或 emission 高效得多。它非常适合 neon lights 之类的对象！

Utility Light Types（实用灯光类型）

  • PxrAOVLight - 这是一个 utility，被识别为 light type，但它 不会 照亮任何东西。它是一个有用工具，可通过 light linking 为 objects 快速创建 masks。



Lights 还有一些与 shadows 和 bidirectional path tracing 相关的特殊 parameters，这些由 PxrVCM 和 PxrUnified integrators 实现。可以参考 technical memo：
