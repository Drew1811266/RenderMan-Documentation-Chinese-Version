User Interface in Maya

Source: https://rmanwiki-27.pixar.com/space/RFM27/544538919/User+Interface+in+Maya
Downloaded: 2026-06-15T03:49:12+00:00

RenderMan for Maya 的新用户界面由多个部分组成。

  • RenderMan Shelf
  • RenderMan Menu
  • RenderMan Preferences
  • Supported Maya Nodes（受支持的 Maya 节点）
  • RfM 中的字符串标记
  • Preset Browser in Maya


请注意，某些节点可能提供可用预设；你可以通过选择参数旁边的三横线 “sandwich” 菜单来使用它们
