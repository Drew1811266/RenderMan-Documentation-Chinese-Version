Maya 中的预览渲染

Downloaded: 2026-06-15T03:49:12+00:00

启动预览渲染

Preview Rendering（也称为 Render Current Frame）可以通过点击 RenderMan shelf 中的渲染按钮启动。默认会弹出 Image Tool (it) 图像工具窗口。

Maya 中用于渲染当前帧的常规方法也可使用。唯一不同是，RenderMan shelf 中的按钮在开始渲染前总会先切换到 RenderMan。

停止预览渲染

预览渲染会一直渲染到图像收敛。这可能需要一些时间，因此你通常会希望在预览渲染仍显得有噪声时停止它。对于非排队渲染，请在 Maya 中按 Esc 键在 Maya 中停止渲染。

'It' 中的取消按钮也可以停止渲染，无论是否已排队。

预览线程

在 RenderMan Preferences的线程部分中，Rendering Threads 设置同时用于 Preview 和 IPR 渲染。默认值 -1 表示除一个 CPU 外其余都将用于渲染。建议至少保留一个线程，以便渲染进行时 Maya 仍保持交互性。线程数为 0 表示所有处理器都将用于渲染。你也可以选择具体数量。

预览显示

预览渲染的像素默认显示在 "It" 中。在 Maya Preferences 窗口的 RenderMan 部分中，可以在 "It"、"Render View" 或 "Use Global Image Format" 之间选择。请注意，Render View 只有在 spooling 未启用时才可用。

预览队列

默认情况下，预览渲染会直接从 Maya 启动。也可以对预览渲染进行排队，也就是将作业传递给队列系统：LocalQueue 或 Tractor。这些应用程序会管理渲染作业列表。你可以在 Maya Preferences 窗口的 RenderMan Preview Queuing 部分中启用 spooling。
