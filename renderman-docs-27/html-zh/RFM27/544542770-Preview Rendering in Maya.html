Preview Rendering in Maya

Starting Preview Rendering

Preview Rendering（也称为 Render Current Frame）可以通过点击 RenderMan shelf 中的 render button 启动。默认会弹出 Image Tool (it) 图像工具窗口。

Maya 中用于 rendering current frame 的常规方法也可使用。唯一不同是，RenderMan shelf 中的 button 在开始 render 前总会先切换到 RenderMan。

Stopping Preview Rendering

Preview renders 会一直 render 到 image converged。这可能需要一些时间，因此你通常会希望在 preview render 仍显得 noisy 时停止它。对于 non-spooled renders，请在 Maya 中按 Esc key 在 Maya 中停止 render。

'It' 中的 cancel button 也可以停止 renders，无论是否 spooled。

Preview Threads

在 RenderMan Preferences的 threads section 中，Rendering Threads setting 同时用于 Preview 和 IPR renders。默认值 -1 表示除一个 CPU 外其余都将用于 rendering。建议至少保留一个 thread，以便 rendering 进行时 Maya 仍保持 interactivity。Zero threads 表示所有 processors 都将用于 rendering。你也可以选择 specific number。

Preview Display

preview renders 的 pixels 默认显示在 "It" 中。在 Maya Preferences window 的 RenderMan section 中，可以在 "It"、"Render View" 或 "Use Global Image Format" 之间选择。请注意，Render View 只有在 spooling 未启用时才可用。

Preview Queuing

默认情况下，Preview renders 会直接从 Maya 启动。也可以 spool preview renders，也就是将 job 传递给 queuing system：LocalQueue 或 Tractor。这些 applications 会管理 rendering jobs 列表。你可以在 RenderMan Preview Queuing section of the Maya Preferences window 中启用 spooling。
