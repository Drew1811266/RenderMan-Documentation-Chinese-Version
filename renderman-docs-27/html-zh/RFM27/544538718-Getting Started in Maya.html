加载插件

在 Maya 中打开 Plug-in Manager（插件管理器），向下滚动并找到 RenderMan for Maya 插件。如果你找不到列出的 RenderMan for Maya， 不要 不要点击浏览按钮。请参阅 故障排查 来弄清楚安装期间可能出了什么问题。

Windows > Settings/Preferences > Plug-In Manager




RenderMan for Maya 插件成功加载后，你首先应该注意到的是 RenderMan Shelf（RenderMan 工具架）；它会在每次 Maya 启动时可用，甚至在插件加载之前也可用。



工具架包含一个用于加载插件的按钮。

注意，卸载再重新加载 RenderMan for Maya 插件预计不会正常工作。如果想重新加载插件，需要重启 Maya。

首选渲染器

第一次加载 RenderMan for Maya 时，会出现一个对话框，询问你是否希望 RenderMan 成为首选渲染器。这只是设置 Maya 的首选渲染器偏好。每次在 Maya 中创建新场景时，首选渲染器都会变为活动状态。



如果 RenderMan 未设为你的首选渲染器，任何 RenderMan 工具架按钮都会在执行其动作前切回 RenderMan；你也可以在 Render Settings（渲染设置）中切换。

来渲染点东西！

RenderMan for Maya 需要 RenderMan 灯光和材质。Maya 的大多数纹理节点都 supported（受支持），但 Maya 灯光和材质会被忽略。

设置灯光

RenderMan 灯光 可以通过 RenderMan Shelf（RenderMan 工具架）、Hypershade 或 RenderMan Menu创建。

继续创建一个 PxrEnvDayLight，使用工具架按钮的弹出菜单。

你会在场景中看到一个新的灯光操纵器：

选中操纵器时按下 't' 键，可交互式定位太阳。其他 RenderMan 灯光在你按下 't' 时会显示兴趣中心操纵器。

RenderMan 灯光上几个常调控件是曝光和可见性。灯光具有物理衰减。如果场景中的灯光看起来较暗，可以把曝光从 0 调高。每增加 1，光源发出的能量就会翻倍。

默认情况下，灯光对相机不可见，因此如果想在背景中看到日光颜色，需要让它对相机可见。

即使你没有在场景中创建任何 RenderMan 灯光或材质，只要 Render Settings（渲染设置）中启用了 Maya 的默认灯光设置，渲染也不会是黑的。任何 Maya 材质都会使用默认 RenderMan PxrDiffuse 着色器渲染，它看起来很像 Maya 的默认 lambert。

创建几何体

从 Maya 的 Polygons 工具架创建一个多边形球体。

设置材质

RenderMan 工具架有一个按钮，可创建 PxrSurface 或 PxrLayerSurface 并将其附加到当前选择。Legacy materials（旧版材质，例如 PxrDisney）可通过 Hypershade 窗口中的 RenderMan 部分获取。

在场景中选择球体，然后点击 PxrSurface 按钮。

Patterns

PxrSurface 的属性可以连接到大多数 Maya 纹理节点 或 RenderMan 丰富的 pattern（图案） 节点。

Preset Browser

PxrSurface 是可以满足几乎所有着色需求的单一材质。不过我们刚创建的那个看起来其实相当乏味，而且属性列表看起来有点吓人。该从哪里开始？RenderMan 包含一个基于 PxrSurface 的预设材质网络库。

来看一下！点击 RenderMan 工具架中的这个按钮来打开 Preset Browser。

第一次打开 Preset Browser（预设浏览器）时，会出现一个文件对话框，要求你导航到预设库的位置。RenderMan 随附的库位于安装目录下。

例如  C:/Program Files/Pixar/RenderManProServer-24.0/lib/RenderManAssetLibrary

可以将该目录复制到自己的工作区并指向那里，也可以直接让预设浏览器指向安装目录中的那个目录。

如果以后需要更改此位置，请访问 Preset Browser（预设浏览器）部分，位置在 RenderMan Preferences中的 Preset Browser（预设浏览器）部分。

Preset Browser（预设浏览器）找到预设后，并且你点击其中一个类别时，它会像这样显示。

右键单击一个材质，可将其导入并附加到选中的几何体。

渲染！

RenderMan 工具架包含用于预览渲染和 interactive preview rendering (IPR) 的按钮。Maya 中用于这些动作的按钮也同样可用。

IPR 会持续渲染，直到被停止；对材质或灯光的更改会显示在渲染中。预览渲染会持续到图像收敛或被取消。

第一次渲染时， Image Tool “It” 会打开。

Image Tool 有一个按钮，可取消预览或 IPR 渲染。

你也可以使用 "escape" 热键，从 Image Tool 窗口轻松取消渲染。

如果你更喜欢使用 Maya 的 Render View，请访问 RenderMan Preferences。

Render Settings

无需调整 Render Settings，你就能获得合理的开箱即用渲染。不过，我们来看看其中可以访问的一些有趣内容。


  • Path Tracer 是默认积分器，但即使 IPR 正在运行，你也可以试用其他积分器。  

  • 启用诸如 运动模糊 和降噪等功能。
