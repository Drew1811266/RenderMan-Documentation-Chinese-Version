加载 Plug-in

在 Maya 中打开 Plug-in Manager，向下滚动并找到 RenderMan for Maya plug-in。如果你找不到列出的 RenderMan for Maya， 不要 去点 browse button。请参阅 troubleshooting（故障排查） 来弄清楚 installation 期间可能出了什么问题。

Windows > Settings/Preferences > Plug-In Manager




RenderMan for Maya plug-in 成功 loaded 后，你首先应该注意到的是 RenderMan Shelf；它会在每次 Maya 启动时可用，甚至在 plug-in 加载之前也可用。



shelf 包含一个用于加载 plug-in 的 button。

注意，卸载再重新加载 RenderMan for Maya plug-in 预计不会正常工作。如果想重新加载 plugin，需要重启 Maya。

Preferred Renderer（首选渲染器）

第一次加载 RenderMan for Maya 时，会出现一个 dialog，询问你是否希望 RenderMan 成为 preferred renderer。这只是设置 Maya 的 preferred renderer preference。每次在 Maya 中创建 new scene 时，preferred renderer 都会变为 active。



如果 RenderMan 未设为你的 preferred renderer，任何 RenderMan shelf buttons 都会在执行其 action 前切回 RenderMan；你也可以在 Render Settings 中切换。

Let's Render Something!（来渲染点东西！）

RenderMan for Maya 需要 RenderMan lights 和 materials。Maya 的大多数 texture nodes 都 supported（受支持），但 Maya lights 和 materials 会被忽略。

Set Up Lights（设置灯光）

RenderMan lights 可以通过 RenderMan Shelf、Hypershade 或 RenderMan Menu创建。

继续创建一个 PxrEnvDayLight，使用 shelf button 的 popup menu。

你会在 scene 中看到一个新的 light manipulator：

选中 manipulator 时按下 't' key，可 interactively position the sun。其他 RenderMan lights 在你按下 't' 时会显示 center of interest manipulator。

RenderMan lights 上几个常调 controls 是 exposure 和 visibility。Lights 具有 physical falloff。如果 scene 中的 light 看起来较暗，可以把 exposure 从 0 调高。每增加 1，light source 发出的 energy 就会翻倍。

默认情况下，lights 对 camera 不可见，因此如果想在 background 中看到 daylight 的 color，需要让它对 camera 可见。

即使你没有在 scene 中创建任何 RenderMan lights 或 materials，只要 Render Settings 中启用了 Maya 的 default light setting，render 也不会是黑的。任何 Maya materials 都会使用 default RenderMan PxrDiffuse shader 渲染，它看起来很像 Maya 的 default lambert。

Create Geometry（创建几何体）

从 Maya 的 Polygons shelf 创建一个 polygon sphere。

Set Up Materials（设置材质）

RenderMan shelf 有一个 button，可创建 PxrSurface 或 PxrLayerSurface 并将其附加到当前 selection。Legacy materials（例如 PxrDisney）可通过 Hypershade window 中的 RenderMan section 获取。

在 scene 中选择 sphere，然后点击 PxrSurface button。

Patterns

PxrSurface 的 attributes 可以连接到大多数 Maya texture nodes 或 RenderMan 丰富的 pattern nodes。

Preset Browser

PxrSurface 是可以满足几乎所有 shading needs 的单一 material。不过我们刚创建的那个看起来其实相当 dull，而且 attributes 列表看起来有点 intimidating。该从哪里开始？RenderMan 包含一个基于 PxrSurface 的 preset material networks library。

来看一下！点击 RenderMan shelf 中的这个 button 来打开 Preset Browser。

第一次打开 Preset Browser 时，会出现一个 file dialog，要求你 navigate 到 preset library 的 location。RenderMan 随附的 library 位于 install 下。

例如  C:/Program Files/Pixar/RenderManProServer-24.0/lib/RenderManAssetLibrary

可以将该 directory 复制到自己的 work area 并指向那里，也可以直接让 preset browser 指向 install 中的那个 directory。

如果以后需要更改此 location，请访问 Preset Browser section，位置在 RenderMan Preferences中的 Preset Browser section。

Preset Browser 找到 presets 后，并且你点击其中一个 categories 时，它会像这样显示。

Right-click 一个 material，可将其 import 并 attach 到 selected geometry。

Render!（渲染！）

RenderMan shelf 包含用于 preview rendering 和 interactive preview rendering (IPR) 的 buttons。Maya 中用于这些 actions 的 buttons 也同样可用。

IPR 会持续 render，直到被停止；对 materials 或 lights 的更改会显示在 render 中。Preview rendering 会持续到 image converges 或被取消。

第一次 render 时， Image Tool “It” 会打开。

Image Tool 有一个 button，可取消 preview 或 IPR renders。

你也可以使用 "escape" hot-key，从 Image Tool window 轻松取消 renders。

如果你更喜欢使用 Maya 的 Render View，请访问 RenderMan Preferences。

Render Settings

无需调整 Render Settings，你就能获得合理的 out-of-the-box render。不过，我们来看看其中可以访问的一些有趣内容。


  • Path Tracer 是 default integrator，但即使 IPR 正在运行，你也可以试用其他 integrators。  

  • 启用诸如 motion blur 和 denoise 等 features。

  • 添加额外 Outputs 到 render。这些 outputs 会与 main render 同时写出，并可在 Image Tool 中查看
