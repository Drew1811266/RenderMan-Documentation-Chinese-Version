Cameras in Maya

Source: https://rmanwiki-27.pixar.com/space/RFM27/544541707/Cameras+in+Maya
Downloaded: 2026-06-15T03:49:12+00:00

摄影机在 RenderMan for Maya 中提供多种有趣的效果。

  • Motion Blur（运动模糊）
  • Bokeh（散景）
  • Tilt-Shift（移轴）
  • Projections（投影）（我们支持 Maya 原生投影工作流）
  • Pixel and Sample Filtering（像素与采样过滤）

关于 camera projection 插件（tilt-shift 和 vignetting 等高级功能需要该插件），你可以在 PxrCamera 页面了解更多信息。


Image Planes（图像平面）

RenderMan 目前将图像平面实现为 PxrImageDisplayFilter SampleFilter 插件。在这种情况下，重要的是 txmake 从非方形输入图像（.png 示例）生成未失真的纹理（.tex），以便采样滤镜能正确投射它。txmake 命令大致如下：

$ txmake -resize up imageplane.png imageplane.ip.tex

此行为会导致旧版 RenderMan 生成的图像平面纹理不兼容。如果遇到失真问题，很可能是因为你在不同 RenderMan 版本之间使用了相同的图像平面纹理（.tex）。也可能是因为你使用错误的 txmake 参数手动生成了图像平面纹理（.tex）（注意 “up-” 与 “up” 的区别）。

旧版行为可以通过禁用 Image Plane As Sample Filter 选项来获得旧版行为；该选项位于 Advanced render settings 下。


为了使用 Pattern 插件作为图像平面颜色，必须禁用 Image Plane As Sample Filter 选项。
