RenderMan 27 for Maya

RenderMan for Maya (RfM) 可通过以 Maya 为中心的 workflow（包括 Maya-style pattern nodes 等），或结合其他 tools，轻松访问 RenderMan features（功能）。

该 plugin 还以新的方式连接 Maya 与 RenderMan Pro Server，无需编写 RIB 即可获得快速的 interactive results（交互式结果）。RIB-out functionality 会继续支持现有 pipelines 和 farm rendering。RfM 为 Maya users 提供对 RenderMan 速度、能力和稳定性的真正无缝访问；他们既可以保持简单的 Maya-centric workflow，也可以利用 plugin 的灵活性创建 optimal、customized pipeline。


安装

RenderMan for Maya 需要同时安装 RenderMan Pro Server 和 RfM 插件才能正确运行。RenderMan 安装程序会自动完成这一点。


"it"

"it" 是一个 robust framebuffer/render view window，提供完整的 floating-point support 和强大、灵活的 catalog，同时也是一个快速强大的 imaging tool，能够执行通常只有高端 standalone products 才具备的 production-quality image manipulation 和 compositing。得益于它与 plugin 的集成以及全新的 Python scripting interface，"it" 让用户能够把 sophisticated post-render image operations 纳入自己的 rendering pipeline。

RenderMan Features

RenderMan for Maya 支持 Maya 原生不具备的 advanced rendering features。由于 Maya 不直接支持这些 advanced features，RfM 提供了自己的 techniques 来创建和控制这些 effects。这些 techniques 通常很简单，只需把 RenderMan-specific attributes 附加并配置到 geometry、lights 和/或 materials 上。Photo Realistic RenderMan (PRMan) 被设计为快速且易用，同时生成 production-quality renders；global illumination 开箱即用，interactive rendering 可为 artists 提供快速迭代。

True Interactive Rendering（真正的交互式渲染）

无需启动新的 render，就能完成更多工作。你的更改会立即从 Maya 进行渲染，不需要中间阶段，也不需要 export 到 RIB。你可以更改 parameters、import 或 update assets、update options，甚至在查看结果更新的同时更改 LPE。

Render to Viewport 2.0

选择 RenderMan 作为 renderer，即可直接向 Maya viewport 进行 interactive render。无需重新启动 render，就可以修改 scene、编辑 parameters 等。

Improved Interactive Creation of AOVs and LPEs（改进的 AOV 与 LPE 交互式创建）

用于创建 AOVs 和 custom LPEs 的 streamlined interface 是定义所需 outputs 的单一位置；这些 outputs 可用于 live rendering 期间更新的 post-processing workflows。

Analytic Area Lights（解析 Area Lights）

Area Lights 是创建 physically accurate lighting 的强大方式，同时可以优化 sampling budgets，并进而优化 performance。PRMan 的 area lights 支持 light filters，以及 light-linking、groups、per-group AOVs、IES profiles 和 portals。

Global Illumination（全局照明）

有时也称为 “Indirect Illumination”，RenderMan for Maya 支持 Global Illumination effects。这些 subtle diffuse shading effects 是从给定 point 多次 sample scene lighting 后产生的。该 technique 可以得到高度真实的结果。Occlusion 只是 sample 一个 point 的 “coverage”（并且比 Indirect Illumination 更便宜），也会自动支持，或通过 PxrOcclusion Integrator 支持。

Volume Rendering（体积渲染）

RfM 包含 PRMan 的一流 volumetric rendering capabilities，可使用 OpenVDB 5.0 创建 volumetric effects，包括 Maya 的 Fluid Effects，并提供完整 PRMan support：motion blur、multi-scattering、non-homogenous 等。

True Curved Surfaces（真正的曲面）

使用 RenderMan render NURBS 和 subdivision surfaces 时，tessellation settings 从来不是问题。由于 RenderMan 会 render true curved surfaces，NURBS 和 subdivision surfaces 永远不会出现 faceting artifacts。Maya 的 tessellation mode settings 会被直接忽略，RenderMan 的优势会立即体现出来。

Subsurface Scattering

RenderMan for Maya 支持 Subsurface Scattering，这是 realistic rendering translucent materials 的重要 effect，例如 skin、flesh、fat、fruits、milk、marble 以及许多其他 materials。Subsurface scattering 负责 material 内部的 color bleeding、或 shadow boundaries 之间的 light diffusion 等 effects。RfM 通过 RenderMan 随附的 materials 支持 path-traced subsurface scattering。

Secondary Outputs（二级输出）

RenderMan for Maya 支持 secondary outputs（AOVs，即 Arbitrary Output Variables），它们可以是 shader 中包含的任意信息。用户还可以利用 Light Path Expressions（LPE）对 outputs 进行强力控制。

Studio Quality RenderMan Shaders（电影工作室品质的 RenderMan Shaders）

RenderMan 随附 Pixar 在 feature films 中使用的同一套 shaders。这让 artists 可以把 RenderMan 的全部能力带入自己的 projects，并提供比 Maya built-in materials 更高层级的 control。

Ri Procedurals

作为 RIB-based pipeline 的 foundation，RenderMan for Maya 支持各种 procedurals。此外，RenderMan for Maya 还为 procedurals 提供简单支持。


Down the Pipe(line)

RenderMan for Maya 是潜在 sophisticated production pipeline 中的一部分。为了把 work 从 script 带到 screen，你可能还需要更多工具，这时 Tractor 和 RenderMan Pro Server 就会登场。通过将 RfM 与 Tractor 和 RenderMan Pro Server (RPS) 配合实施，用户可以实现从 desktop 到 renderfarm 的 seamless integrated rendering workflow。

Tractor

Tractor 是一个 system，用于把 queued jobs 中的 tasks 分发到网络中的 compute servers。它包括 central queueing mechanism、remote execution server、monitoring subsystem 和 browser-based user interface。Tractor 被设计为 Alfred 的 high-performance replacement，可用于任何 work distribution capacity，但尤其适合管理 RenderMan for Maya 到 RenderMan Pro Server 的 pipeline。

Tractor 作为单独的 standalone product 提供，并拥有自己的 documentation。

RenderMan Pro Server

RenderMan Pro Server 是 high-performance rendering pipeline 的 foundation。它提供 renderer 及其对应 rendering utilities 的 standalone implementations，并提供 APIs，让熟练用户能够扩展 functionality。作为 core technology，RenderMan Pro Server 是所有 pixels 流出的源泉。
