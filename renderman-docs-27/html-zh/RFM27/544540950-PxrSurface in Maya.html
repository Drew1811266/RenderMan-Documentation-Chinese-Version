PxrSurface in Maya

Source: https://rmanwiki-27.pixar.com/space/RFM27/544540950/PxrSurface+in+Maya
Downloaded: 2026-06-15T03:49:12+00:00

在 Maya 中，可通过选择对象并点击按钮栏中的 PxrSurface 图标，将 PxrSurface BXDF 应用于该对象。也可以在 RenderMan Menu 的 Materials 标题下选择场景中的对象来应用 PxrSurface。

下图展示了可通过此 BXDF 修改的一小部分 attributes（属性）。

Globals 中描述的 Pattern Utility 参数不会显示在 Attribute Editor 中，因为 Maya 对 arrays（数组）的处理方式如此。你可以改为在 Hypershade 内的 Node Editor 中建立连接来使用它。示例包括使用 Maya 中的 PxrSurface 为每个 material instance（材质实例）创建 ID pass，或将 PxrSurface 与另一个 textured pattern（带纹理的 pattern）组合，创建用于 compositing（合成）的其他有用 AOV，例如 noise 等。请注意，这样做可能会增加已指定 material（材质）的 evaluation time（求值时间）。

在 RenderMan 中渲染 surfaces（表面）依赖一种名为 PxrSurface 的可分层 material（材质）（见下方 Input Material）。 

这是 Pixar Animation Studios 长片动画中使用的同一套 material model（材质模型）。因此，该材质有一些重要优势：

  • 单一 material（材质）即可处理所有 look（外观），例如：

    • 木材

    • 皮肤

    • 塑料

    • 玻璃

    • 车漆

    • 以及更多更多！

  • 提供 physically based（基于物理）的 presets（预设），帮助你开始。

  • 无需牺牲灵活 controls（控制项），也能得到 physically plausible（物理可信）的结果。

    • 艺术家可以有选择地 "break" energy conservation（能量守恒）。

    • Art direction（艺术指导）可以自然完成，无需 workaround（变通方案）。

    • Controls 会方便地标记为 "Artistic" 和 "Physical"

  • Layering（分层）提供无限可能。

作为 artist（艺术家），能够为所有 shading（着色）选择单一 material（材质），可以消除猜测，并让 RenderMan 按需优化。我们会讨论所有不同设置，以及它们与制作特定 materials（材质）的关系。请特别注意关于 performance considerations（性能考量）的说明，以及哪些组合对特定 effects（效果）有用的提示。

下面会看到 material（材质）的每种 effect（效果）或 lobe 都被拆分到独立章节，以帮助用户定义希望学习的区域。不过，随着不同 lobes 被启用和调整，material 的 look（外观）会发生变化。正是通过组合这些 lobe 设置，你几乎可以创建任何想象中的 material。


  • Diffuse Parameters（漫反射参数）
  • Specular Parameters（镜面参数）
  • Clear Coat Parameters（清漆参数）
  • Glass Parameters（玻璃参数）
  • Iridescence Parameters（虹彩参数）
  • Fuzz Parameters（绒毛参数）
  • Subsurface Scattering Parameters（次表面散射参数）
  • Single Scatter Parameters（单次散射参数）
  • Glow Parameters（发光参数）



Input Material

连接到一个 layer pattern（层 pattern），用于对 Bxdf 的参数进行分层。这让用户能够通过 layered effects（分层效果）创建更复杂的 look。示例可能包括 dust（灰尘）、scratches（划痕），甚至 labels（标签）。

在下面的参数中，有些可以由 PxrLayer 覆盖；当它连接到 Input Material 或通过 PxrLayerMixer连接时即可生效。 PxrLayerSurface 的设计意图是只包含 global（全局）参数，从而更清楚地显示哪些参数无法在 layer（层）中被覆盖。当你确定会使用 layering（分层）时，我们建议使用此 material（材质）。这些设置的结果不会改变。

  • Bold Face（粗体） 参数是 layerable（可分层）的，可以按 layer 覆盖。

  • Italicized（斜体） 参数无法分层或覆盖。这些参数会在所有 layers 中作为 global（全局）设置生效。例如：选择 GGX 作为 specular model（镜面模型）意味着所有 layers 的该参数都会使用 GGX。


Properties Parameters（属性参数）

本节提供整个 material（材质）的 global controls（全局控制）。例如，如果只在此处提供 Bump，则它会应用到上方所有 lobes。Presence 也是 global parameter（全局参数），会遮罩或 "cutout" 所有 lobes，就像在 mask 值为 0.0 的位置对象不存在一样。

Bump

除非被单个 lobe 的 bump normal 覆盖，否则用于所有 lobes 的 Normal/Bump。

Presence

在此连接 mask（遮罩），可将 cutout pattern（镂空图案）应用到对象。这对创建树叶和其他薄而复杂形状等 cutouts 很有用。对于灰度值，当直接由 camera（相机）看到时，它也可以作为 opacity（不透明度）用于半透明结果。

  • 对于 non-thin glass refraction（非薄玻璃折射）和 "thick" style objects（厚式对象），此项会被忽略；在这些对象中，color（颜色）是穿过内部的 attenuation（衰减）

Presence Cached

指定 presence 是否缓存。启用 cached 时，presence 将按 per micropolygon 求值，而不是在关闭 cached 时按 per shading point 求值。 

  • 由于 presence 计算可能很昂贵，如果存在 heavy shading network（重型着色网络），例如 texturing（纹理），则 caching 有利于提升 rendering speed（渲染速度），但会增加 memory（内存）。

  • 因为 caching 本质上存储在 tessellation level（细分镶嵌级别），如果需要更精细的 render details（渲染细节），建议使用 uncached presence，因为它按 per shading point 求值。

  • memory 与 speed 之间存在 trade off（权衡）。如果 presence 非常简单，例如 constant value（常量值），cached 和 uncached 可能几乎同样快，并能节省一些 memory。请注意，如果 presence 为 constant 1.0（默认值），PxrSurface 会优化所有这些处理。

  • 启用 subsurface（次表面）或 single scattering（单次散射）时，为避免 artifacts（伪影）（与 volume tracking 相关），PxrSurface 会在内部强制使用 uncached presence（忽略 Presence Cached 的参数设置）。 

  • 出于类似原因，对于 non-thin glass refraction，PxrSurface 会忽略 presence，无论 cached 还是 not cached。

  • 该 integrator 可以控制 opacity（不透明度）不再是 stochastic（随机/有噪声）的最大 depth（深度）。

Shadow Mode

Shadow opacity（阴影不透明度）计算模式：

  • Shader and shadow color（Shader 和阴影颜色）

  • Shadow color only

Shadow Color

指定 shadow color（阴影颜色）。此参数可用于出于 art direction（艺术指导）目的伪造阴影颜色。请注意，更改 shadow color 可能会影响 performance（性能）和 global illumination（全局照明）结果，因为它会被视为 transparent shadow（透明阴影）（可理解为 colored glass（彩色玻璃）），即使对象是不透明的。

Utility Pattern

指定用于输出的 pattern node（图案节点）。例如：可以将 PxrMatteID 连接到此参数以输出 ID AOV。也可以输出另一个 pattern，例如 texture（纹理）。

Enable Manifold Exploration/Walk（启用 Manifold Exploration/Walk）

仅在使用 PxrUnified Integrator 且 Enable Manifold Walk（流形行走）选项设为 ON 时可用。对于要 捕获 caustic light paths（焦散光路）的对象，此设置应开启。例如 specular object（镜面对象）内部的 diffuse surface（漫反射表面）。

Specular Curvature Clamp（镜面曲率钳制）

此 control（控制项）用于覆盖对应的 integrator 设置，即 Specular Curvature Filter 参数，该参数位于 PxrVCM 和 PxrUnified integrators 中。

User Color

指定要渲染到 userColor Lobe 的 color（颜色）或 pattern（图案），它可以 作为 utility（实用输出）通过 LPE 输出。此连接或颜色不会渲染到 beauty 中，目的是输出对 compositing（合成）或后期调整有用的 AOV。
