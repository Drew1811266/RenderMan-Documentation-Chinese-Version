Preset Browser in Maya

Source: https://rmanwiki-27.pixar.com/space/RFM27/544539564/Preset+Browser+in+Maya
Downloaded: 2026-06-15T03:49:12+00:00

简介

  • 可以使用工具架按钮或 RenderMan 菜单打开 Preset Browser（预设浏览器）。

  • Presets（预设）包含灯光套件、环境灯光和材质等类别。

  • 可以按名称、用途类别或元数据搜索资产。

  • 可以选择两种预设图像尺寸。

保存和使用

  • 保存/导出材质很简单！

    • 选择 BxDF（导出会识别已连接的节点）

    • 从 Preset Browser（预设浏览器）中选择所需类别

    • 选择保存图标（预览图像下方的材质球加号图标）

    • 为它指定一个有用的名称，并添加 pipeline（流程）所需的 metadata（元数据）

    • RenderMan 会生成一个预览色样渲染，并自动填充到 Preset Browser（预设浏览器）中

  • 导入/使用已保存资产

    • 只需右键单击并选择 "Import and Assign"（导入并指定），即可应用到选中的对象

    • 根据对象需要进行任何调整！

环境贴图

  • 可以使用 Preset Browser（预设浏览器）中大预览色样下方的图像加号图标保存环境贴图

    • 在 Preset Browser（预设浏览器）中选择环境类别

    • 从文件浏览器窗口中选择所需的 OpenEXR 或 .HDR 文件

    • 可以在对话框中添加/更改 metadata（元数据），或更改名称。

    • 随后我们会将图像转换为兼容的 .tex 格式。根据原始图像尺寸，这可能需要一些时间。

    • 接着会执行预览渲染，并自动填充到浏览器类别中。

  • 使用已保存的贴图很简单。

    • 右键单击资产并导入。

    • 更换贴图就像双击另一个环境贴图资产一样简单！

资产依赖项

将资产保存到磁盘时，它的所有依赖项也会保存到磁盘。 

这些包括：

  • 纹理：以 tex 格式存储，支持 UDIM。

  • HDRI 环境贴图：以 tex 格式存储。我们会使用一组特定标志尽可能高效地压缩纹理。

  • OSL 着色器：以 oso 格式存储以保护你的 IP；如果想共享源代码，也可以手动将 osl 文件添加到资产目录中。

这些依赖项会增加预设在磁盘上的大小；如果想通过互联网分发它们，应牢记这一点。

跨应用材质

RenderManAssets 被设计为可以轻松从一个应用程序（如 Maya）转移到另一个应用程序（Katana、Houdini、Blender 等），也可以反向转移。

兼容性问题

不过，如果材质包含 Maya 原生节点，仍可能遇到兼容性问题。RenderMan for Maya 会自动转换若干 Maya 原生节点，而这些节点在其他应用程序中不可用，例如文件节点或 Maya 的 Cloud pattern。为缓解这些问题，每个 RenderManAsset 都会维护一个已用节点列表，并使用它评估预设与当前宿主应用程序的兼容性；如果不兼容，则显示警告。

系统会检查：

  • 资产是否包含宿主专用节点

  • 资产的渲染器版本是否高于或等于当前版本

  • 如果资产包含宿主专用节点，则确保当前宿主版本高于或等于该资产的宿主版本。

最佳实践

如果想确保跨应用兼容性，请避免使用非 Pixar 的 BxDFs 和 Patterns。

库配置

RenderMan 随附一个标准库，可在以下位置找到： $RMANTREE/lib/RenderManAssetLibrary

我们建议将标准库复制到你选择的位置，并保持默认库不变，作为备份。新的副本应可读可写。如果资产浏览器指向不可写位置，它将不允许创建新资产。

资产库的位置可以通过环境变量或 Maya 首选项设置。

Maya Preferences（Maya 首选项）

资产库的路径可以通过 Maya 的首选项对话框设置：

  1. 打开 Maya 的首选项对话框

  2. 选择 资产浏览器 偏好设置，位于 RenderMan

  3. 单击 选择 按钮以打开目录选择器。

  4. 导航到你选择的 RenderManAssetLibrary 文件夹，然后单击 选择库 按钮。

  5. 资产浏览器会刷新并显示所有可用预设。

可以按住 Shift 并单击预设浏览器的工具架图标，直接打开库选择对话框！

环境变量

资产库的位置可以通过 RMAN_ASSET_LIBRARY 环境变量来设置。

可使用不同方法进行设置：

Maya.env

Maya.env 是 Maya 启动时读取的配置文件，用于配置若干环境变量。如果此文件不存在，可以在 一个标准位置中创建新的文件。此方法适用于所有平台：Linux、OSX 和 Windows。

在 Maya.env 末尾输入以下行；下次启动时，Maya 应该就能找到标准库：

Linux / OS X
RMAN_ASSET_LIBRARY = $RMANTREE/lib/RenderManAssetLibrary

Windows
RMAN_ASSET_LIBRARY = %RMANTREE%/lib/RenderManAssetLibrary

系统命令

Linux

在所有 POSIX 平台（Linux 和 OSX）上，可以在启动 Maya 的 shell 中定义环境变量，或将其追加到 shell 配置文件（.cshrc 用于 csh/tcsh， .profile 或 .bashrc 用于 bash）。

csh / tcsh
setenv RMAN_ASSET_LIBRARY $RMANTREE/lib/RenderManAssetLibrary

bash
export RMAN_ASSET_LIBRARY=$RMANTREE/lib/RenderManAssetLibrary
OSX

  • 如果从 shell 运行 Maya，可以像 Linux 一样设置环境变量。

  • 如果从 dock 运行 Maya，建议改用  Maya.env  方法。

Windows

  • 右键单击 My Computer（我的电脑）并选择 Properties（属性）（或在 control panel（控制面板）中双击 System（系统）），然后单击 Advanced（高级）选项卡，再单击 Environment Variables（环境变量）。
