从 ZBrush 导出 OBJ 和 OpenEXR

  •  按 Shift+d 下移到 最低 subdivision（细分级别）。 Lion 由 Olivier Drion 雕刻。

  •  点击 Switch 按钮，在 Morph Target 中切换到原始模型。如果尚未设置 Morph Target，请查阅 ZBrush 文档，了解如何设置 Morph Target。

点击 Switch 按钮，在 Morph Target 中切换到原始模型。如果尚未设置 Morph Target，请查阅 ZBrush 文档，了解如何设置 Morph Target。

  • 在 Preferences、Import/Export、Vector Displacement Map 中，将 Tangent FlipAndSwitch 设置为 43。  共有 48 种不同的 vector displacement（矢量置换）设置，因为在某个 renderer（渲染器）中红色通道可能朝上，而在另一个 renderer 中可能朝下。RenderMan 使用 43。

近期变更意味着原先的 Tangent FlipAndSwitch 41 已调整；如果 displacement（位移）不正确，你可能需要使用 41 而不是 43。

  • 在 Tools、Vector Displacement Map 中，将 vd Tangent 设为 on，将 vd 32 Bit 设为 on 以导出 32-bit map（贴图），否则将导出 16-bit；将 vd SUV 设为 on 以启用 smooth UV（平滑 UV），将 vd Normals 设为 on。 

  • 点击 CreateAndExport VDMap 导出 maps（贴图）和 OBJ。这些 maps 将导出为 OpenEXR。

指定 Displacement Shader（位移着色器）

  • 创建一个 PxrTexture 节点。将 Filename 设置为导出的 ZBrush EXR map（贴图）。其他参数保持默认值。

  • 创建一个 PxrDispTransform 节点。将 Displacement Type 设置为 ZBrush Vector 并将 Vector Space 设置为 Tangent。

  • 创建一个 PxrDisplace 节点（这是实际对 surface（表面）执行 displacement（位移）的 displacement shader）。

  • 将 PxrTexture 的输出 Result RGB 连接到 PxrDispTransform 的 Disp Vector。
