Common Tab 提供 Maya 中所有 renderer 共享的设置。请注意，此配置会在将 RenderMan 设为 renderer 后，或从 RenderMan menu 打开设置后显示。

你会发现此类功能的大部分已经移到 Workspace Tab 以简化选择。下面说明的是我们不再支持的旧选项；其中许多已经从 RenderMan Common Tab 中移除。


Path

不适用；如果仍能看到此项，请将 renderer 选择切换到其他 renderer，然后再切回 RenderMan 以刷新。

File Name

不适用；请使用 Workspace Tab 通过便捷 tokens 进行命名。

Image Size

显示 resolution（通常以 pixels 和 inches 表示）。

Color Management

RenderMan 不支持 Output Transform 设置。不过，"it" 中提供了 view remapping 功能。

File Output

渲染图像文件名最多可由三个元素组成：file name、frame number 和 file format。

File Name Prefix

决定渲染图像的 base name。

Image Format

不适用； AOVs tab 会设置显示用的 display driver。

Compression
不适用；AOVs tab 会在 Display Driver 中设置 compression。

Frame/Animation Ext

用于命名 output images 的格式。

Frame Padding

不适用；请使用 Workspace Tab 按文档页说明的 tokens 设置此项。

Framebuffer/Naming Scheme

不适用于 RfM；这些会在 AOVs tab 中设置。

Use Custom Extension

不适用于 RfM；请使用上文提到的 Workspace tab。

Frame Range

Start Frame, End Frame

指定要 render 的第一帧和最后一帧。Frame/Animation Ext 必须设为包含 # 的选项才能启用这些参数，否则只会使用单帧。

By Frame

rendered frames 的 output names 之间的增量。只有当 Frame/Animation Ext 设为包含 # 的选项时，By Frame 才可用。

Renderable Cameras

Renderable Camera

从此列表中选择你的 render camera。  如果场景只有一个 renderable camera（例如 perspective camera），无论在这个 drop-down list 中选择哪个 camera，它都会成为 renderable camera。换句话说，新选择的 camera 会成为场景中的 renderable camera（默认 perspective camera 会变为不可渲染）。

Alpha Channel (Mask)

确定 rendered images 是否包含 mask channel。

Depth Channel (Z Depth)

确定 rendered images 是否包含 depth channel。建议使用 AOVs tab 指定 __depth 或 Z。

Image Size

Presets

根据行业标准预设选择 output images 的 resolution。

Maintain Width/Height Ratio

如果勾选此框，custom resolutions 会保持锁定的 height/width ratio。禁用此参数后，可以为 image resolution 输入任意数值。

Width/Height

对于 custom resolutions，请在此处输入所需的 resolutions。

Size Units

选择合适的 measurement unit。

Resolution

设置图像的 rendered resolution。

Resolution Units

选择 resolution 的定义单位，可以是 pixels per inch 或 pixels per centimeter。

Device Aspect Ratio

target display（例如 "it" 或 Maya's Render View）的 aspect ratio。

Pixel Aspect Ratio

显示 device 上单个 pixels 的 aspect ratio，也就是你查看 rendered image 时所用设备的像素纵横比。

Render Options

Enable Default Light

在 rendering 期间开启或关闭 default lighting。请注意，如果场景中有 RenderMan lights，我们不会 render default light。

Pre/Post MEL scripts

这些字段用于输入 MEL scripts/commands，以便在指定时机运行。
