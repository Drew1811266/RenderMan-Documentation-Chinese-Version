  • 最佳实践简介 介绍使用 Cookies and Milk 示例完成的一些基础内容。
  • Using PxrMatteID（使用 PxrMatteID） - 本教程将帮助你创建一组 RGB 或纹理化通道，它们有时称为遮罩、遮罩层或 clown pass，可供后续合成使用。
  • Cryptomatte 是为后续合成操作创建遮罩层的优秀方式。它很简单，并涵盖自定义遮罩层以及自动的全场景遮罩层设置。
  • Using Trace Sets in Maya（在 Maya 中使用 Trace Sets） 说明如何将对象分组，并选择性地追踪它们的阴影、反射或透射（transmit）可见性。
