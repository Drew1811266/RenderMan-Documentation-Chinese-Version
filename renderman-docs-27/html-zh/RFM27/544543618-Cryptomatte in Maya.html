Cryptomatte in Maya

Downloaded: 2026-06-15T03:49:12+00:00

使用 PxrCryptomatte 工作流，你可以在 Maya 中输出 ID，以便在合成应用程序中轻松选择。随后可以将该选择用作遮罩，在后期软件包中有选择地进行修改。

PxrCryptomatte 作为 Sample Filter 插件提供。因此，Cryptomatte 的输出会作为一个独立的 EXR 文件保存，与 beauty 或降噪输出分开。请注意，需要 OpenEXR 才能正确存储这些值。

在 Features Tab 的 Render Settings下，导航到 Images / Sample Filters 选项卡，并在空槽位上右键单击以选择 创建 PxrCryptomatte。

上方选项为默认值。你可以访问 PxrCryptomatte 页面了解更多关于这些选项的信息。

请注意，创建 Sample Filter 后，会再创建一个空连接，使你可以使用不同条件输出 Cryptomatte 文件。例如，如果你想要一个按对象输出和另一个按材质输出，可以再创建一个用于输出的 PxrCryptomatte，并为该文件指定另一个名称。

目前，每个插件都需要输出一个单独的 OpenEXR，并与降噪或 beauty EXR 分开。你可以使用正确的字符串标记创建序列，例如：cryptomatte.<f4>.exr 会创建一个 4 位补零的渲染序列，即 cryptomatte.0001.exr。
