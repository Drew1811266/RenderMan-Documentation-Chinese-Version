Hypershade Nodes（Hypershade 节点）

Surface（表面）

仅 RenderMan shading（RenderMan 着色） 节点受支持。

Volumetric（体积）

仅 RenderMan volume（RenderMan 体积） 节点受支持。

Displacement（位移）

仅 RenderMan displacement（位移） 节点受支持。
