Stylized Looks in Maya Overview

Source: https://rmanwiki-27.pixar.com/space/RFM27/544541301/Stylized+Looks+in+Maya+Overview
Downloaded: 2026-06-15T03:49:12+00:00

开始使用 RenderMan 中的 Stylized Looks（风格化外观）非常直接，可以通过 2 种方式完成。


第一种方式是使用自动设置流程：只需选择对象，然后点击 Stylized 工具架按钮。这将会：

  • 在 Render Settings（渲染设置）中启用 Stylized Looks，并添加所需的全部 Stylized AOV。
  • 为所选对象的每个表面着色器添加并连接一个 PxrStylizedControl 节点
  • 添加一个 Hatching 显示滤镜，这样就可以开始风格化渲染了！



或者，如果你更喜欢手动设置，可以按照以下步骤操作 

  1. Enable Stylized Looks（启用 Stylized Looks）
  2. Attach Stylized Pattern（附加 Stylized Pattern）
  3. Attach Stylized Display Filters（附加 Stylized Display Filters，风格化显示滤镜）


1.   Enable Stylized Looks（启用 Stylized Looks）

RenderMan 中 Stylized Looks（风格化外观）系统的核心能力来自 自定义 AOV 集，系统需要这些 AOV 来确定排线、线稿和卡通着色等特征。 

幸运的是，在 Maya 中设置这些 AOV 是一键流程。只需点击 "Enable Stylized Looks"，RenderMan for Maya 就会为你创建所有必要的 AOV。

View AOVs（查看 AOV）

不要忘记启用 "Output AOVs" 选项；该选项位于 RenderMan preferences（RenderMan 首选项）中。




2.  Attach Stylized Pattern - PxrStylizedControl（附加 Stylized Pattern - PxrStylizedControl）

加载所有 AOV 后，现在需要逐一检查连接到模型的每个 shader（着色器），并将 AOV Output 从 PxrStylizedControl 节点连接到 Materials 的 Utility Pattern input（输入）。 


3.  Attach Stylized Display Filters（附加 Stylized Display Filters，风格化显示滤镜）

真正有趣的部分从这里开始！现在可以开始添加 Stylized Looks Display Filters（Stylized Looks 显示滤镜）。 

  • PxrStylizedHatching - 此滤镜会添加由艺术驱动的插画式排线概括效果。

  • PxrStylizedLines - 添加曲率线稿、轮廓线以及其他很酷且独特的外观
  • PxrStylizedToon - 这是用于添加阶梯式卡通着色效果的理想 Display Filter（显示滤镜）

Stylized Looks Tip（Stylized Looks 提示）

为避免空背景，请使用 PxrBackgroundDisplayFilter 模拟纸张或其他背景。 

4. Add lighting（添加灯光）

从 RenderMan 灯光中发挥你的照明创意。请记住，灯光越小，阴影看起来越锐利。

5. Interactive Rendering（交互式渲染） 

Stylized Looks（风格化外观）系统的主要优势之一，是艺术家在移动灯光、改变几何体、调整动画和着色参数时，可以直接在创作应用的渲染视口中看到这些新的处理效果。同一应用中的这种“机内预览”能力可以显著加快艺术探索和决策。

6. Combine Display Filters to really get creative（组合 Display Filters（显示滤镜），真正发挥创意）

由于许多处理效果可以由手绘贴图或程序化贴图驱动，因此可以进行大量自定义。此外，应用这些处理效果的专用显示滤镜由艺术家选择的图案信号驱动。为提供的 pattern AOV 找到新颖用法，或向组合中添加其他 AOV，都很容易（也很有趣！）。这些效果也可以由渲染量驱动，例如光照强度、灯光滤镜域、朝向比、到相机的距离等。 

创建 PxrStylizedToon 作为第一个风格化滤镜，然后添加一个应用半调预设的 PxrStylizedHatching filter，接着添加 PxrStylizedLines filter 来创建漂亮且细致的轮廓线，从而形成独特外观。

Stylized Looks 显示滤镜节点也可以 daisy-chained（菊链连接），产生外观的分层合成，而这些排列组合几乎无穷无尽。
