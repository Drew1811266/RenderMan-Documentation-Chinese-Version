Real Time（实时）

Default Lights Shading（默认灯光着色）

创建灯光时的默认模式：线框（wireframe）或着色（shaded）。

Texture File Extensions（纹理文件扩展名）

用于识别显示纹理文件的扩展名。默认使用符合 RenderMan 规范的文件类型。

Display R in lights（在灯光上显示 R）

切换是否在 RenderMan 灯光上显示 R 标志。

Lights Wire Color（灯光线框颜色）

用于在视口中显示这些对象的线条颜色

灯光滤镜线框颜色

用于在视口中显示这些对象的线条颜色

Max Texture Resolution（最大纹理分辨率）

视口中显示纹理的最大分辨率。更高分辨率会影响性能，尤其是在需要显示许多纹理时。此设置不会影响视口 IPR 渲染。

Viewport Controls（Viewport 控件）

在面板图标栏中显示用于渲染到 Viewport 2.0 的控件。

Playblast

Playblast Viewer Command（Playblast 查看器命令）

选择一个可执行文件，并使用 # 表示要传递的帧编号和填充位数。

Show Command Output（显示命令输出）

将命令打印到脚本编辑器。

RenderMan

Viewport Controls（Viewport 控件）

在视口面板中显示 RenderMan 控件图标。

Crop Window Color（裁剪窗口颜色）

选择绘制裁剪窗口时偏好的颜色。

Max FPS（最大 FPS）

限制帧率，以避免延迟或鼠标虚影等问题。

Progress Bar in Viewport（Viewport 中的进度条）

一个小进度条会在视口底部显示帧进度，可在此处禁用。

Viewport Info String（Viewport 信息字符串）

你可以在视口中显示有用信息，并可根据这里的选择，使用 Workspace Render Tab 提供的 token（令牌）进行配置。默认会根据可容纳输出的空间显示尽可能多的信息。
