可以在 Render Settings 窗口的 Features 选项卡下启用运动模糊。默认情况下它未启用。下拉菜单还提供了输出 2D motion（二维运动）以用于合成操作的选项。

 

快门时序

Frame Open（帧开启）

以下图像显示第 1、2 和 3 帧的 Frame Open（帧开启）模糊。快门会在帧开始时打开。默认快门角度为 180，表示快门会打开半个帧。请注意最后一帧没有模糊。这是因为该帧的快门打开之后没有动画。

Frame Center（帧居中）

以下图像显示 Frame Center（帧居中）模糊，也就是快门在帧之前打开，并在帧之后关闭，以该帧为中心。第一帧和最后一帧的模糊看起来较短，因为动画没有延伸到这些帧之外。

运动采样

在第二帧使用居中模糊时，我们的红球看起来有些奇怪。模糊是水平的，但球实际上应该在这条条纹的中间到达顶点。默认情况下只有两个运动采样，因为它们成本较高，而且两个通常已经足够。你可以在 Render Settings 的 Motion Blur 部分中更改全局运动采样数量。或者，如果你知道只有某一块几何体需要额外采样，也可以只为该几何体变换添加它们。  

在变换的 Attribute Editor 中，选中球体变换的选项卡后，你会看到运动采样的设置。-1 是默认值，表示继承全局设置。

两个采样表示球的位置只会在快门开启时间和快门关闭处采样。超过两个采样时，它们会均匀分布在快门开启和关闭之间。这有时称为 Multi-Segment 运动模糊。

例如，下面展示了从 2 个采样增加到 7 个采样时图像如何变化。这是 Frame Center（帧居中）模糊。当达到 7 个采样时，看起来已经相当平滑！你可以按需要继续增加，没有采样数量上限。

快门角度

快门角度是一个角度（以 360 度为总量），它决定相机的快门为了曝光 “film” 而打开一个旋转周期的多大比例。默认值为 180 度，因此快门会打开半个帧。快门角度越高，运动对象看起来就越模糊。
