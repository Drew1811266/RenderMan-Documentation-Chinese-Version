Workspace

Source: https://rmanwiki-27.pixar.com/space/RFM27/544542704/Workspace
Downloaded: 2026-06-15T03:49:12+00:00


Workspace 选项卡主要是一个查看器，用于显示与你的资产管理策略相关的偏好设置和变量的设置与值。

你可以在这里使用标记来定义输出命名，甚至通过管线以及 User Names and Values 自动化这些命名。这是一种便捷的简写方式，可用于定义命名约定；无需把内容输入成长字符串，也能让标记根据项目设置自动解析，而不需要用户交互。

RenderMan for Maya 可以在字符串参数中替换若干预定义标记。它们可用于动画纹理序列、引用特定存储位置等。

 

标记含义
<scene>场景名称
<layer>Maya 渲染层名称
<camera>渲染摄影机名称
<aov>AOV 或 LPE 名称
<aovdir>会将 AOV 写入具名子目录
<f4>帧号填充，此处为 4；可省略表示不填充，或使用较小数字
<ext>文件类型扩展名
<ws>Maya 当前工作区，即项目中的文件路径
<version>插入上方指定的版本号
<take>插入上方指定的 take 编号
<date>当前日期
<assetlib>指向标准 RenderMan Asset Library 的路径
<file>场景文件名
<frame>十进制帧号
<jobid>唯一作业标识符
<shape>短形状名称（所有实例具有相同形状名称）
<shapepath>长形状名称（所有实例都有唯一的 形状路径名称）
<time>当前时间
<udim>UDIM 标识符（仅在渲染时替换）
<imagedir>包含完整图像输出目录
_MAPID_通用纹理图集标识符 （仅在渲染时替换）

环境变量

环境变量必须以 $ 符号开头，并且只能使用大写字母、数字和 "_" 字符：

  • $RMANTREE/bin/txmake：返回 txmake 工具的路径。
  • 大括号也可以用于组合： 
    • ${SHOW}_assets/<shape.assetName>/<shape.assetVersion>/tex/<shape>_diff.<udim>.tex

获取属性值

RfM 不再支持 TCL 表达式，但允许属性求值：

  • <shape.time> 返回 shape 节点上 'time' 属性的值（指 'shape' 节点，而不是 'transform' 节点）。
  • 也可以通过指定格式化字符串来格式化数值属性值：
    • <shape.time:%04d>：如果 time 属性的当前值是 12.2，则会返回 0012。
    • 这里使用的是 Python 标准格式化说明符。

 

 

由于这些标记是简写，你可能希望在渲染前查看它们会解析成什么。你会看到眼睛图标。选择它会显示你的标记选择得到的结果字符串。
