Workspace

Source: https://rmanwiki-27.pixar.com/space/RFM27/544542704/Workspace
Downloaded: 2026-06-15T03:49:12+00:00


Workspace tab 主要是一个 viewer，用于显示与你的 asset management policies 相关的 preferences 和 variables 的 settings 与 values。

你可以在这里使用 tokens 定义 output naming，甚至通过 pipeline 以及 User Names and Values 来自动化它们。这是一种便捷的 shorthand，可用于定义 naming conventions，无需把内容输入成长字符串，并允许 token 根据 project settings 自动 resolve，而不需要 user interaction。

RenderMan for Maya 可以在 string parameters 中替换若干 predefined tokens。它们可用于 animated texture sequences、引用特定 storage locations 等。

 

TokenMeaning
<scene>Scene name
<layer>Maya Render Layer name
<camera>Render camera name
<aov>AOV 或 LPE name
<aovdir>会将 AOVs 写入 named sub-directories
<f4>Frame padding，此处为 4；可省略表示 no padding，或使用较小数字
<ext>File type extension
<ws>Maya 当前 workspace，即 project 中的 file path
<version>插入上方指定的 version number
<take>插入上方指定的 take number
<date>Current date
<assetlib>指向 standard RenderMan Asset Library 的 Path
<file>Scene file name
<frame>Decimal frame number
<jobid>Unique job identifier
<shape>Short shape name（所有 instances 具有相同 shape name）
<shapepath>Long shape name（所有 instances 都有唯一的 shapepath names，用于完整路径标识）
<time>Current time
<udim>UDIM identifier（仅在 render time 替换）
<imagedir>包含完整的 image output directory 路径
_MAPID_Generic texture atlas identifier （仅在 render time 渲染时替换）

Environment Variables

Environment variables 必须以 $ sign 开头，并且只能使用 capitals、numbers 和 "_" characters：

  • $RMANTREE/bin/txmake：返回 txmake utility 的 path。
  • Curly braces 也可以用于 composition： 
    • ${SHOW}_assets/<shape.assetName>/<shape.assetVersion>/tex/<shape>_diff.<udim>.tex

Retrieving attribute values

RfM 不再支持 TCL expression，但允许 attribute evaluation：

  • <shape.time> 返回 shape node 上 'time' attribute 的 value（'shape'，而不是 'transform' node）。
  • 也可以通过指定 formatting string 来 format numerical attribute values：
    • <shape.time:%04d>：如果 time attribute 的当前 value 是 12.2，则会返回 0012。
    • 这里使用的是 Python 标准格式化说明符（python's standard format specifiers）。

 

 

由于 tokens 是 shorthand，你可能希望在 rendering 前查看它们会 resolve 成什么。你会看到 eye icon。选择它会显示 token choices 得到的 resulting strings。
