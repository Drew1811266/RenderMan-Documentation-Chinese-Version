Instances in Maya

Downloaded: 2026-06-15T03:49:12+00:00

通过 实例化 通过实例化复用几何体，可以在 RenderMan 渲染复杂场景时显著降低内存占用。即使是在包含多个相似对象的较简单场景中，把实例化纳入工作流程也会很方便。

在 Maya 中，设置实例有几种方法，包括：

Maya Instances（Maya 实例）

Maya 实例大概是 Maya 中最常见的实例化形式。Maya 实例会在 RenderMan 中渲染为 "object instances"（对象实例），这是 RenderMan 最高效的实例化形式。

在此示例中，假设需要对一个茶杯进行实例化。

选择要实例化的层级结构顶层组节点，在本例中就是茶杯节点。

如果要实例化的对象非常复杂，可以先创建一个 RenderMan Archive（RenderMan 归档），然后对它进行实例化，这样原始几何体甚至不需要出现在场景中。

执行 Edit->Duplicate Special，打开 option box（选项框）。

确保在选项中启用了实例化。

一个新实例会出现在 Outliner（大纲视图）中。

打开 HyperGraph Hierarchy 窗口，可以看到原始节点和实例是两个独立的变换节点，但共享同一个几何体。（Windows->HyperGraph: Hierarchy）

新的实例可以独立放置。  

Multiple Materials（多个材质）

可以把新材质分配给已实例化的形状节点。在视口中选择要分配新材质的形状节点，然后按平常的方式分配即可。新材质只会分配给你点击的那个形状节点。  

Particle Instancer（粒子实例器）

当实例数量很大时，创建 Maya 实例并逐个独立摆放会变得不切实际。Maya 的 particle instancer（粒子实例器）是在粒子上进行几何体实例化的一种方法。

首先创建一些粒子。这可以使用 emitter（发射器，用于动态放置粒子）完成，也可以使用 Maya 的 nParticle Tool 来放置单个粒子。nParticles 和 Maya 的 legacy particles（旧版粒子）都受支持。

在下面的示例中，粒子是使用 nParticle Tool 逐个放置的。

注意，进行粒子实例化时，有时会在 Maya 中得到意外的偏移。为避免这种情况，请把将要被实例化的对象（例如茶杯）移到 0,0,0。执行 Modify->Freeze Transformations。为了让每个实例都能使用 per-particle attributes（逐粒子属性），可以把原始对象从视图中移开。

通过 "top" orthographic camera（top 正交相机）查看。点击来在新实例应该放置的位置创建粒子，完成后按 Enter。

先选择应该被实例化的节点（例如茶杯），再选择粒子形状节点的变换节点（例如 nParticle1），然后执行 nParticles->Instancer。如果选择顺序错误，Maya 会指出这一点。

你会看到每个粒子位置都出现一个实例。

 

XGen Archives（XGen 归档）

Nested Instancing（嵌套实例化）

特别是在处理大型数据集时，例如用 XGen 对许多对象进行实例化，我们会尝试 flatten（展平）实例，以便用户能够灵活设置 overrides（覆盖）并提高速度。但有些时候这可能会消耗过多内存。原因是展平后的实例在渲染时会相乘，因此 1,000 个由 1,000 个实例组成的实例会变成 1,000,000 个实例。

启用嵌套实例后，1,000 个由 1,000 个实例组成的实例在内存中会变成 2,000 个。不过，实例的一些选项（例如某些变化选项等）将不再提供给渲染器。因此，我们允许用户通过给变换节点打标签来决定何时嵌套实例化。


创建 XGen Description（XGen 描述）

XGen 是一个强大的工具，可用于放置任意几何体的实例。首先需要创建一个 XGen 描述，其中保存实例将被放置到的点集合。然后可以创建归档，并在这些点上对它进行实例化。

选择要在其上放置实例的形状节点（例如地面平面）。然后在 XGen 工具栏中，点击最左侧带有 X 的图标。

在弹出的 XGen 窗口中，选择标有 "Create New Description" 的按钮。

在弹出的 Create XGen Description 对话框中，选择名为 "Custom Geometry/Archives (use for any model you have created)" 的单选按钮。

接着点击 Create 按钮。这会在平面上随机分布点。现在不用担心这个。我们先创建一个归档。

创建 XGen Archive（XGen 归档）

在 XGen Window（XGen 窗口）中的 Primitives 选项卡下，向下滚动到名为 Archive Files 的部分。

选择要归档的层级结构顶层变换节点（例如茶杯节点），然后点击名为 "Create From Selection..." 的按钮。

在出现的 "Export Selection as Archives" 对话框中，为 XGen 归档选择位置和名称，然后执行 Export。系统也可能提示你重新保存场景。

接下来，点击 Archive Files 列表下方的 "Add..." 按钮。浏览到之前保存的归档。文件名应以 .xarc 结尾。

你应该会看到归档的实例密集分布在地面平面上。

Specify Points（指定点）

如果不想使用随机定位，可以在特定位置指定点。

滚动到 XGen Primitives 选项卡顶部。将 Generate Primitives 设置从 "Randomly across the surface" 改为 "At specified locations"。然后点击标有 "Specify Points..." 的按钮。

视口中会出现一个用于放置点的工具。

放置点之后，你应该会看到归档在这些点上被实例化。

Add a Material（添加材质）

在 Preview/Output 选项卡下，向下滚动到 RenderMan Settings 部分。这里可以为归档应用新的材质。
