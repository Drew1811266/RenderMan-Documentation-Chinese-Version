Render Settings in Maya

Source: https://rmanwiki-27.pixar.com/space/RFM27/544542282/Render+Settings+in+Maya
Downloaded: 2026-06-15T03:49:12+00:00

RenderMan for Maya (RfM) 的 Render Settings 通过 Maya Render Settings dialog 控制。可以从多个位置访问：

  • 主要是在 Maya Shelf 中，有一个 Render Settings 图标： 
  • 通过菜单：Windows -> Rendering Editors -> Render Settings
  • 通过 RenderMan menu：RenderMan -> Render Settings
  • 当 Maya 处于 Rendering mode 时，通过菜单：Render -> Render Settings

 

如果你没有在场景中创建 light，RfM 会在渲染时为你创建一个默认 directional light。这是为了支持标准 Maya Default Light 工作流。如果你需要一个没有 lights 的场景（例如场景仅由 emissive objects 照亮），必须在 Common Tab 下关闭此项。

 

 

所有这些选项都会打开同一个 dialog；用过 Maya 原生 Render Settings 的人会觉得它很熟悉。 

当 RenderMan 被选为 Maya renderer 时，会显示上面的 dialog。顶部有 6 个 tabs，用于控制 renderer 的不同部分。

  • The Common Settings tab 用于控制 image resolution、frame range 和其他选项。
  • The Sampling tab 用于访问 RenderMan 的 sampling controls，这些控件会影响渲染时间和质量。
  • The AOVs tab 用于设置额外输出和有用的数据 passes。
  • Features tab 可用于开启 motion blur，并添加 Cryptomatte 等其他 render features。 
  • The Advanced Settings Tab，artist 可以在这里编辑更高级的选项，例如 cache sizes 或 bucket order。
  • 最后是 Workspace Settings 它会显示 output locations 的 job settings，以及更多 pipeline/file system settings
