Maya 渲染设置

Source: https://rmanwiki-27.pixar.com/space/RFM27/544542282/Render+Settings+in+Maya
Downloaded: 2026-06-15T03:49:12+00:00

RenderMan for Maya (RfM) 的 Render Settings 通过 Maya 的 Render Settings 对话框控制。可以从多个位置访问：

  • 主要是在 Maya 工具架中，有一个 Render Settings 图标： 
  • 通过菜单路径：Windows -> Rendering Editors -> Render Settings
  • 通过 RenderMan 菜单：RenderMan -> Render Settings
  • 当 Maya 处于 Rendering 模式时，通过菜单：Render -> Render Settings

 

如果你没有在场景中创建灯光，RfM 会在渲染时为你创建一个默认方向光。这是为了支持标准的 Maya Default Light 工作流。如果你需要一个没有灯光的场景（例如场景只由自发光对象照亮），必须在 Common 选项卡下关闭此项。

 

 

所有这些选项都会打开同一个对话框；用过 Maya 原生 Render Settings 的人会觉得它很熟悉。 

当 RenderMan 被选为 Maya 渲染器时，会显示上面的对话框。顶部有 6 个选项卡，用于控制渲染器的不同部分。

  • Common Settings 选项卡 用于控制图像分辨率、帧范围和其他选项。
  • Sampling 选项卡 用于访问 RenderMan 的采样控制，这些控件会影响渲染时间和质量。
  • AOVs 选项卡 用于设置额外输出和有用的数据通道。
  • Features 选项卡 可用于开启运动模糊，并添加 Cryptomatte 等其他渲染功能。 
  • Advanced Settings 选项卡，艺术家可以在这里编辑更高级的选项，例如缓存大小或渲染块顺序。
  • 最后是 Workspace Settings，它会显示输出位置的作业设置，以及更多管线/文件系统设置。
