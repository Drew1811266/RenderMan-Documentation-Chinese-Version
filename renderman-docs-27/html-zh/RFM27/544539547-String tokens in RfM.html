String tokens in RfM

Source: https://rmanwiki-27.pixar.com/space/RFM27/544539547/String+tokens+in+RfM
Downloaded: 2026-06-15T03:49:12+00:00

RenderMan for Maya 可以在字符串参数中替换若干预定义 tokens（标记）。它们可用于动画纹理序列、引用特定存储位置等。

 

Token（标记）Meaning（含义）
<scene>场景名称
<layer>Maya Render Layer 名称
<camera>渲染 camera（相机）名称
<aov>AOV 或 LPE 名称
<aovdir>这会将 AOVs 写入具名子目录
<f4>帧号 padding（填充位数），此例为 4；若不需要填充可省略，或使用更小的数字
<ext>文件类型扩展名
<ws>Maya 当前 workspace（工作区），即项目中的文件路径
<version>插入上方指定的 version（版本）编号
<take>插入上方指定的 take（拍摄/分次）编号
<date>Current date（当前日期）
<assetlib>标准 RenderMan Asset Library 的 Path（路径）
<file>场景文件名
<frame>十进制帧号
<jobid>唯一 job（作业）identifier（标识符）
<shape>短 shape 名称（所有实例具有相同的 shape 名称）
<shapepath>长 shape 名称（所有实例都有唯一的 shapepath 名称）
<time>Current time（当前时间）
<udim>UDIM identifier（标识符，仅在渲染时替换）
<imagedir>此项包含完整的图像输出目录
_MAPID_Generic texture atlas identifier（通用纹理图集标识符） （仅在渲染时替换）

Environment Variables（环境变量）

环境变量必须以 $ 符号开头，并且只能使用大写字母、数字和 "_" 字符：

  • $RMANTREE/bin/txmake：返回 txmake 工具的路径。
  • 也可以使用 curly braces（花括号）进行组合： 
    • ${SHOW}_assets/<shape.assetName>/<shape.assetVersion>/tex/<shape>_diff.<udim>.tex

Retrieving attribute values（检索属性值）

RfM 不再支持 TCL 表达式，但允许进行 attribute evaluation（属性求值）：

  • <shape.time> 返回 shape 节点上 'time' 属性的值（这里是 'shape' 节点，而不是 'transform' 节点）。
  • 也可以通过指定 formatting string（格式化字符串）来格式化数值属性值：
    • <shape.time:%04d>：如果 time 的值为 12.2，则会返回 0012。
    • 这里使用 Python 的标准格式说明符。
