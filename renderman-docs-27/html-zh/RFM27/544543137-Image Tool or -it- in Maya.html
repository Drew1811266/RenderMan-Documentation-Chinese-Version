Image Tool or "it" in Maya

Source: https://rmanwiki-27.pixar.com/space/RFM27/544543137/Image+Tool+or+%22it%22+in+Maya
Downloaded: 2026-06-15T03:49:12+00:00

""it" 功能强大、稳定、实用，并且与 Maya 紧密集成。"it" 可以以独立模式运行，也可以通过 User Interface（用户界面）或 command line（命令行）启动。"it" 是默认的预览渲染显示窗口，可在 RenderMan Preferences中配置。

Image Tool ("it") 是一个稳健的 framebuffer/render view window（帧缓冲/渲染视图窗口），提供完整的浮点支持和强大灵活的 catalog（目录）；它同时也是快速而强大的成像工具，可执行通常只有高端独立产品才具备的生产级图像处理和合成。

User Interface（用户界面）

"it" 窗口可通过选择 "Window" 菜单前两个分区中的选项来自定义。这些可选窗口和工具栏可以从菜单中显示或隐藏，也可以用菜单中标出的对应快捷键开关。它们还可以停靠到主窗口中，或保留为浮动窗口。双击这些子窗口的标题栏，可在停靠和取消停靠之间切换。像 "Pixel Readout" 这样的工具栏很小，没有标题区域，而是在末端或顶部提供用于拖放的手柄。双击该手柄也可以切换停靠状态。

对于高 DPI 屏幕，你可能会发现文本难以阅读。可以手动更改它。请查找此文件：

$RMANTREE/lib/Resources/stylesheets/it.qss

它的作用类似于 "it" 的 Cascading Style Sheet（层叠样式表）。你可以使用此分区更改文本大小：

QLabel#PixelReadout {
    font-family: Courier;
    font-size:   20px;
    margin-left: 10;
}

由于我们不建议修改随产品发布的文件，你可以将新的 .qss 文件副本放在由环境变量 RMS_SCRIPT_PATHS 定义的 custom directory（自定义目录）中。

Catalog（目录）

此窗口显示当前 "it" 会话中存储的所有图像和渲染结果。可以使用多个 catalog，每个 catalog 都存储自己的图像。若要创建新 catalog，请转到 File->New Catalog。

Image Window（图像窗口）

此窗口显示从 catalog 中选择的当前图像或渲染结果。

Inspector（检查器）

此窗口显示与所选图像或渲染结果相关的信息。大的文本区域是 Notes 字段，可用于向图像添加任意 metadata。Custom View 选项将在后面的 Views 分区，也就是 Special "it" Features（特殊 "it" 功能） 分区下解释。


Tools（工具）


tools 分区可让你访问 "it" 中最常用的工具。从左到右分别是：

Pan Tool（平移工具）

在 image window 上点击并按住鼠标左键，即可围绕图像平移。（默认快捷键: 1)

Zoom Tool（缩放工具）

在 image window 上点击并按住鼠标左键，即可放大或缩小图像。（默认快捷键: 2)

Crop Tool（裁剪工具）

允许你在图像上绘制裁剪窗口，从而重新渲染图像的一部分。若要移除裁剪窗口，请在裁剪窗口内单击左键，并点击右上角的 X。（默认快捷键: 3)

Scrub Tool（拖动浏览工具）

允许你快速拖动浏览 catalog 中的所有图像。按住鼠标左键并向左移动会在 catalog 中向后浏览；向右移动会向前浏览。（默认快捷键: 5)

Select Object Tool（选择对象工具）

将在你的内容创作应用程序中选择对象（可能并非适用于所有桥接产品）。（默认快捷键: 6)

Select Surface Tool（选择表面工具）

将在你的内容创作应用程序中选择对象的 shader（可能并非适用于所有桥接产品）。请注意，渲染必须包含 Float id AOV 才能正常工作。某些 IPR 渲染会从 DCC 应用程序输出此内容。（默认快捷键: 7)

Reset Zoom（重置缩放）

重置使用 Zoom Tool 时应用的任何缩放。（默认快捷键: X)

Wipe Tool（擦拭对比工具）

允许你在 catalog 中的两张图像之间进行擦拭对比。（默认快捷键: 4）

要使用它，请选择 catalog 中的第一张图像，然后转到 Image->Toggle Background。然后选择 catalog 中的第二张图像，并在 Wipe Tool 打开时，在 image window 中按住鼠标左键拖动。

Background（背景）

将当前图像切换为 background。参见 Background。

Wipe Direction（擦拭方向）

更改 wipe tool 的方向（垂直或水平）

Denoise（降噪）

为当前 IPR / Image 激活 Denoiser（降噪器）

Cancel Render（取消渲染）

取消当前传入渲染

Sequence Controls（序列控制）

sequence controls 分区可让你访问图像序列播放的控制项。从左到右分别是：

Step Back Frame（后退一帧）

在序列中后退一帧。（默认快捷键: Left)

Play Sequence（播放序列）

播放/暂停序列播放。（默认快捷键: Space)

Step Forward Frame（前进一帧）

在序列中前进一帧。（默认快捷键: Right)

Loop Mode（循环模式）

打开或关闭序列的循环播放。

关于 Sequences 的进一步说明，请参见 Sequences，位于  Special "it" Features（特殊 "it" 功能） 分区。

Monitor Controls（监视器控制）

monitor controls 可让你访问 Gamma 和 Exposure 等设置。最右侧按钮会将 Gamma 和 Exposure 控制项重置为默认值。还有一些有用的 monitor control 快捷键，包括：

Exposure Up 1 F-Stop（曝光增加 1 档）

（默认快捷键: Ctrl/Cmd + ] )

Exposure Down 1 F-Stop（曝光降低 1 档）

（默认快捷键: Ctrl/Cmd + [ )

Gamma Up 0.1（Gamma 增加 0.1）

（默认快捷键: Ctrl/Cmd + } )

Gamma Down 0.1（Gamma 降低 0.1）

（默认快捷键: Ctrl/Cmd + { )

Pixel Readout（像素读数）


Pixel Readout 窗口可控制查看用色彩空间、各种 heat maps / look、像素重映射，并提供当前鼠标光标下像素的相关信息，包括像素坐标和像素值。你可以通过转到 View->Lock Pixel Readout 锁定当前 pixel readout（默认快捷键: M）。remap 菜单允许在图像显示前对其执行一些后期处理（重映射）。这不会改变图像中的像素值。参见 Remapping 了解更多信息。

channel view 显示当前可见的图像通道，选项包括 RGB（普通颜色）、Red、Green、Blue、Alpha（如果存在）以及 "Luminance"。

Other（其他）

在窗口的图像区域中右键单击也可访问菜单。"Window" 菜单上可访问的其他窗口（例如提供对 "it" 的 Python 脚本访问的 "Console"）不可停靠，会作为顶层窗口出现。

Preferences Window（首选项窗口）

General（常规）

Save Sessions（保存 Sessions）

控制是否应将 "it" 会话数据视为需要保留。如果打开，当 image catalogs 关闭时会显示保存对话框。

Image Cache Size（图像缓存大小）

"it" 应在内存中保留的图像数量。超过后，"it" 会将最旧的图像移动到 Scratch Image Location 中的临时文件。

Scratch Image Location（暂存图像位置）

当所有 catalogs 中的图像总数超过 Image Cache Size 首选项时，用于保存图像的位置。

Maximum Zoom（最大缩放）

限制 "it" 放大或缩小的最大程度。

Short Window Titles（短窗口标题）

打开后，窗口标题栏中只会显示图像文件名。



保存到 jpg 或其他 8Bit 图像格式时，IT 现在会自动 burn in color space（烘焙色彩空间）。


Appearance（外观）

New bucket indicator（新 Bucket 指示器）

允许你编辑 recent bucket marker 的颜色。参见 Recent Bucket Markers。

Events（事件）

Focus Incoming Render（聚焦传入渲染）

如果打开，image window 会自动显示当前传入渲染。

Display Mapping（显示映射）

设置从 remap 菜单选择 Automatic view mapping 时要使用的 view mapping，并按不同 bit depths（8-bit、16-bit、Float）以及图像来自传入渲染还是磁盘分别设置。

Shortcuts（快捷键）

允许你编辑映射到 "it" 中特定命令的任何快捷键。

Special "it" Features（特殊 "it" 功能）


AI Denoise（AI 降噪）

为预览目的，"it" 可以使用以下方案对 beauty renders 进行交互式降噪： Nvidia's AI Denoising solution 可对 beauty renders 进行交互式降噪。你可以在 View 菜单中选择它，或按键盘上的 N 来开关它。这需要最新 Nvidia drivers（本文撰写时为 391.89）以及系统中较新的 Nvidia GPU（最低为 Kepler，更新的卡性能会显著更好）。收敛程度更高（samples 更多）的图像会得到更好的质量。收敛不足时，图像会呈现类似水彩或 "painterly" 的风格。

此功能仅用于交互式预览渲染，它 不是 为最终渲染或动画而设计。AI Denoise 只在 beauty（Ci）上正确运行，不适用于 AOVs。若要正确查看 AOVs，请关闭 denoiser，否则会看到伪影。最终 EXR 输出不受 denoiser 影响。请改用 "it" 的 Snapshot 功能保存降噪后的 Beauty（见此条目下方）。

Denoising options 需要统计独立的像素，这意味着你要么手动为像素提供 rendering option "Importance Sample Filtering"，要么在相应桥接产品中选择 Denoise 选项，由其强制对图像使用 Importance Sample Filter。如果未启用此项，使用 Optix Denoiser 时将看不到变化。
必须遵守 Optix 7 的要求，即此版本所需硬件必须是 Nvidia Maxwell generation 或更新。


Snapshot（快照）

在 Commands 窗口中，可以选择对某个图像执行 Snapshot。这会快速复制已完成的渲染或正在进行的交互式会话。若只需捕获 beauty render，也有 No AOVs 选项。对于交互式会话，这很有用：你可以在更改当前场景前保存其外观，然后进行比较。snapshot 会被放到 Catalog 列表的下一项。若要保存使用 Nvidia AI Denoiser 的图像，必须使用 Snapshot，因为 export 只会保存底层原始图像。 

Views（视图）

"it" 中的 Views 是缩放、平移、像素长宽比和 view mappings 的组合。默认情况下，catalog 中所有图像共享一个 view。当你想逐帧查看渲染到 "it" 的镜头时，这很方便（快捷键 PageUp/PageDown）。当你放大后切换到镜头中下一帧时，缩放和平移会保持不变。不过，有时你可能希望大幅放大一张图像来查看某个特征，再在它与缩小后的图像之间切换。"it" 通过 Custom View来支持这一点。打开 Custom View 后，该图像就拥有自己的缩放和平移，独立于 Catalog 中的其他图像。如果/当 Session（或 Catalog）被保存时，Custom Views 也会被保存。

Pixel aspect ratio 在处理会生成非方形图像的项目时很有用。Cinemascope 胶片格式就是一个例子。在某些 pipeline 中，你可能会决定让数字文件使用一种格式，其中像素在水平方向上的尺寸是垂直方向的一半。"it" 允许你在 view 中指定这一点，使图像在显示器上看起来正常。此菜单可从 it.ini 文件完全配置，包括菜单上的名称和像素长宽比。即使自定义此菜单，也可以为不同长宽比绑定快捷键。

Recent Bucket Markers（最近 Bucket 标记）

当 "it" 用作 render destination 时，它会在每个 bucket 到达时显示一个矩形。该指示器会在几秒钟内淡出。图像正在逐步细化时，有时很难看出 renderer 正在处理图像的哪一部分，此功能很有用。可开关此功能，并可在 preference dialog 中更改矩形的颜色。

Background（背景）

Catalog 中的任意单张图像都可以设置为 Background image。之后当你选择该 Catalog 中的其他条目时，显示的图像是所选图像 叠加在 指定 background 上的合成结果。若要将当前图像切换为 background image，请使用 Image->Toggle Background。Catalog 中该图像旁边的图标随后会亮起。参见 Working With Background Images（使用背景图像）。

Remapping（重映射）

remap 菜单会显示图像所在的 color space。对于已渲染图像，通常是 linear；对于 8-bit 图像文件，通常是 sRGB。默认情况下，"it" 会选择图像的 image space；这样做时，image space 前面会有一个 *。你可以使用菜单覆盖 "it" 的选择（* 指示器会消失）。

"it" 支持使用 OpenColorIO system 对 output images 进行 view mapping，以适配你选择的 workflow。你的 view mapping defaults 可以分别为渲染到 "it" 作为 frame buffer 的图像（无论 bit depth）或 imported images 设置（如有需要，也可针对不同类型使用不同 mapping）。

关于 remapping，需要记住几件重要事项：

  • Remapping 只应用于查看文件，不会被 burn in。参见 Burn Mapping On Save 首选项。

  • 通过 RenderMan rendering 得到的图像默认位于 linear space，除非对 output 应用了 exposure（这可以在 RfM 中通过 RenderMan controls 的 Output Settings 选项卡完成）。

    如果确实对 output 应用了 exposure，请确保对 rendered images 的 default image mapping 进行适当调整。

  • OpenColorIO setup 可以通过 it.ini 更改；不过一如既往，我们强烈建议在 site-specific surrogate ini 文件中进行更改，并通过 RMS_SCRIPT_PATHS 引用。若设置了环境变量 OCIO，"it" 也会使用它来查找 OpenColorIO config 文件。

remap 菜单上有一个特殊条目：Shadow Map 选项，用于查看 shadow map 文件。这些文件可能具有远超 0 到 1 的巨大范围，因此传统显示方法是将最小值和最大值重新缩放到 0 到约 0.9，然后用 (1 - val) 对它们取反。


Sequences（序列）

Sequences 适用于这种情况：你已经渲染了一个镜头，或可能对 shader 的一组参数做了 wedge，现在希望将它们作为序列播放。Sequence 工具栏包含开始/停止播放和帧范围控制项。"it" 可以播放整个 catalog，也可以将磁盘上的一组图像作为一个元素（称为 "sequence"）打开。还有一个 "scrub" 鼠标工具，激活后可按住鼠标左键并沿水平方向拖动来浏览序列。

可以通过打开某个目录中磁盘上的多个文件来创建图像序列，请使用 File->Open Sequence 而不是 File->Open Image。"it" 会识别哪些文件属于该序列，帧范围会出现在 Sequence 工具栏中。

Sequences 是较简单的技术，因为它们不提供保存或加载影片的方式，整个 "movie" 会保存在 RAM 中。"it" 会限制其在 RAM 中保留的图像数量；为了平滑播放，需要将该上限（Image Cache Size）设置为至少等于序列中的帧数。



Color chart generator（色卡生成器）

摄影和电影行业已经使用 color rendition charts 数十年，以确保高质量结果。你可以阅读关于此主题的 many articles 来了解更多。

它提供一个标准色卡生成器（使用已发布的 sRGB 值），可在你的 look-dev renders 中作为稳定的色彩参考。

  • 色卡以 1024x768 生成。

  • 图像应保存为 8-bits image（tiff、png），并启用 Burn Mapping。

  • 在 PxrTexture 中引用它，并启用 Linearize。

Analysis Tools（分析工具）

Diff（差异）

此命令会计算 catalog 中 2 张图像之间的差异。通常，你可能想从 diff image 生成 heat map（见下文）。

Find Variance（查找方差）

此命令会计算图像相对于参考图像的方差。

Heat Map（热力图）

通常，从 data image 创建 heat-map 来可视化其内容很有用。此功能提供一些控制项，可从单通道图像或彩色图像的 luminance 生成 heat maps。

  • 可以一次选择多个图像进行处理。

  • 按住 Shift 键可跳过对话框，并使用上次设置处理图像。

 

Palette（调色板）

可以从 3 种不同 palette 中选择：Classic、Fire 和 Hue。

Custom Display Range（自定义显示范围）

未勾选时，heat map 会使用像素的最小值-最大值显示；否则会使用你选择的固定值。当比较不同图像（例如不同渲染的 sampleCount 或 cpuTime AOV）时，使用固定范围很有用。

  • Normalization Min Value（归一化最小值）：可视化的最低值。

  • Normalization Max Value（归一化最大值）：可视化的最高值。

Text（文本）

可以在图像底部添加一些文本，包括最小值、最大值和平均值，以及几个可选字段：

  • Font：选择你喜欢的字体

  • Font size：调整字体大小，同时记住所有内容都必须放在同一行。

  • Display Range：显示 custom display range values（自定义显示范围值）。

  • Notes：显示 source image 可能包含的任何注释。

Isolate Values（隔离数值）

这个简单命令会将未通过测试的任何值替换为黑色。你可以使用 <、<=、==、>= 和 > 将 pixel value 或 luminance 与固定值比较。

这对于在图像中查找特定值很有用。

Dump Metadata to message log（将元数据转储到消息日志）

检索图像的 metadata，并在 message log window 中显示。请注意，只有已加载的图像才有 metadata。
