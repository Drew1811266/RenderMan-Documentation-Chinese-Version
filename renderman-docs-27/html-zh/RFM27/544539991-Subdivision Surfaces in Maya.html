Maya 中的细分曲面

Source: https://rmanwiki-27.pixar.com/space/RFM27/544539991/Subdivision+Surfaces+in+Maya
Downloaded: 2026-06-15T03:49:12+00:00

任何多边形网格都可以渲染为 细分曲面，用于 RenderMan for Maya。

有两种方法可以指示某个网格应渲染为细分曲面：

  • 启用 Maya 的 Subdiv Proxy 功能。

  • 在 Shape Node 中选择 RenderMan 的 "Subdivision Scheme" 设置，通常使用 Catmull-Clark。

Read on（继续阅读），了解这两种方法各自涉及的步骤。

创建细分代理

若要在 Maya 中创建细分代理，请选择一个网格，然后执行以下任一操作：

  • Mesh > Smooth Proxy > Subdiv Proxy（网格 > 平滑代理 > 细分代理）

  • 按 Ctrl + Shift + ~ (tilde)（波浪号）

在视口中，你会看到细分曲面的预览。

默认情况下，网格会被渲染为 Catmull-Clark 细分曲面。也可以渲染为 Loop 细分曲面，方法是添加 Subdivision Scheme 属性并更改其值。

Subdivision Scheme（细分方案）

在 Maya 中选择 Subdivision Scheme 属性后，它会按照所选细分类型进行渲染。可选项为 Catmull-Clark、Loop 或 None（用于禁用）。

无论是否启用 Maya 的 Subdiv Proxy 模式，都会遵循该属性；但在 Subdiv Proxy 模式中并不必须设置它，除非你想切换到 Loop。

该属性会出现在 Attribute Editor 的 RenderMan 折叠栏中。

折痕

处理细分曲面时，有时需要给控制网格的部分区域应用折痕，以调整最终细分网格的形状。Maya 的 Subdiv Proxy 允许这样做。启用折痕工具：

Mesh > Smooth Proxy > Crease Tool（网格 > 平滑代理 > 折痕工具）

然后选择控制网格的一条边，并拖动鼠标中键来应用折痕值。

在此示例中，折痕已应用到立方体最近角点处相交的三条边上。
