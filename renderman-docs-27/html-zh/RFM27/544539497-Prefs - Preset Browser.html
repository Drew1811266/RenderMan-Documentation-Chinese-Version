Configuration File（配置文件）

Current File（当前文件）

这是一个格式为 JSON 的 *.rac RenderMan Asset Configuration 文件。它包含用户指定的其他库位置，甚至可以包含其他 *.rac 文件。

Libraries From File（来自文件的库）

此项显示从上方 *.rac 文件加载的库。

Save Current Configuration（保存当前配置）

保存一个格式为 JSON 的 *.rac RenderMan Asset Configuration 文件，其中包含用户指定的已加载库位置。

User Libraries（用户库）

Add a Library（添加库）

向库文件夹添加新路径

Forget Selected Library（忘记所选库）

从可选项中移除一个库

Render all HDRs（渲染所有 HDR）

导出材质时，除了默认的多云色样外，还会渲染一个日光色样。

Hide local Factory Library（隐藏本地 Factory Library）

从浏览器选项中移除默认随附的库
