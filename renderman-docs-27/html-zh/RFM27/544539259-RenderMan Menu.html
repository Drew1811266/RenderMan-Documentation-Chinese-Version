RenderMan Menu

Source: https://rmanwiki-27.pixar.com/space/RFM27/544539259/RenderMan+Menu
Downloaded: 2026-06-15T03:49:12+00:00

RenderMan for Maya 会在 Maya 主菜单栏中提供一个 RenderMan 菜单，便于访问该插件的各项功能。

IPR Render（IPR 渲染）

启动交互式预览渲染会话。

Preview Render（预览渲染） 

启动将当前帧渲染到 "it" 或 Maya 的 Render View。

Batch Preview Render（批量预览渲染）

启动一次批量渲染，输出到显示设备或 "it"。

Batch Render（批量渲染）

启动批量渲染。默认使用 Mayabatch，而不是 RIB。你可以在 Maya Preferences 的 RenderMan 部分更改此设置。

Bake Render（烘焙渲染）

启动一次烘焙渲染；这是一种批量渲染，用于在没有照明的情况下烘焙纹理。

Render Settings...（渲染设置...）

打开 Maya 的 Render Settings 窗口，并以 RenderMan 作为当前渲染器。

Preferences（首选项）

打开 Maya 首选项中的 RenderMan 区域。

Tools（工具）

打开 "it"、Texture Manager 等常用工具，而无需使用工具架图标。

Archive >（归档 >）

导出、导入或选择 RenderMan Archive 节点

Diagnose >（诊断 >）

View Statistics（查看统计信息）

在网页浏览器中打开上次渲染生成的统计文件

View RIB（查看 RIB）

在文本编辑器中打开新生成的 RIB 文件（注意，在 Maya 内部渲染时我们不会创建 RIB）

View RIB for selection（查看所选对象的 RIB）

生成一个 RIB 文件并在文本编辑器中打开，其中包含 Maya 当前所选对象的几何体调用

Script Editor Logging Level（Script Editor 日志级别）

选择要输出到 Maya Script Editor 的详细程度。Debug 输出最多，便于排查问题；Info 输出最少，只显示进度信息。

Disk Logging Level（磁盘日志级别）

输出日志会写入磁盘，并提供与上方 Script Editor 相同的选项，但还包含 Error、Critical 和 Warning 消息级别。

Utilities >（实用工具 >）

更新 Render Globals 预设 

        重建渲染设置质量选项使用的预设。

Convert to PxrLayerSurface（转换为 PxrLayerSurface）

        尝试将 PxrSurface 转换为 PxrLayerSurface 节点

License（许可证）

用于检查该插件许可状态的选项。你可以重试从服务器签出许可证，或打开许可应用程序查看可用许可证

Help（帮助）

指向文档和论坛的有用链接。它会在默认浏览器中打开。

About RfM...（关于 RfM...）

打开一个显示版本和构建号信息的对话框
