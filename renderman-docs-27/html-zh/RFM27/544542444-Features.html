Features tab 可用于启用 RenderMan renderer 的特殊 features，例如 renderer type、OCIO、Motion Blur 等。



Renderer

定义场景使用的 RenderMan renderer。

  • RIS：使用 RIS render 场景。这是 CPU-only renderer。

  • XPU:  使用 XPU render 场景。这是 CPU + GPU hybrid renderer。

    • XPU Mode - CPU：使用 XPU render，但只使用 CPU。

    • XPU Mode - GPU：使用 XPU render，但只使用 GPU。


Motion Blur

RenderMan 提供了许多控件，用于创建快速而高效的 motion blur。

Motion Blur:  启用 motion blur 会让移动 objects 沿其运动 path 产生 blur，以防止 temporal aliasing 和 strobing。使用 motion blur 时，应增加 pixel samples 的数量（Sampling tab 中的 max samples），并预期 rendering times 会更长。 

  • 3D：完整 render 且准确的 motion blur

  • 2D：会计算 motion vectors，但 motion blur 此时 不会 被 render；在输出 dPdtime 和 dPdcameratime 等 AOVs 后用于 post processes 时很有用。

Camera Blur:  当此 feature 被禁用时，renderer 在计算 motion blur 时会忽略 camera 的移动。如果你移动 camera 并希望场景中的所有内容都真实地产生 blur，则需要开启此 feature；但应预期这可能是开销较高的操作，因为场景中的所有内容都会 blur，并产生必须 converge 的 noise。

Shutter Angle:  Shutter Angle 只有在启用 Motion Blur 时才有意义。它控制 shutter 在一个 frame 中打开多久。以 degrees 为单位，设置为 360 表示 shutter 在整个 frame 内保持打开。angle 越大，blur 越强，可能会增加 render times，因为需要更多 samples 才能平滑 blur。

Shutter Opening:  默认情况下，shutter 实际上会瞬时打开和关闭。 Shutter Opening 设置由两个值组成： a 和 b。它们是 shutter interval 的 fractions，可用于调整 shutter 打开和关闭所需时间。从 0 到 a 时，shutter 打开；从 b 到 1 时则关闭。  

Shutter Timing:  Shutter Timing 允许你控制 shutter opening 应如何与 frame 对齐。它可以在 frame 开始时打开，或在 frame 之前打开以便以 frame 为中心，也可以在 frame 开始时关闭。


OpenColorIO

Configuration

  • $OCIO：使用系统 environment variable 定义的 color space

  • Off :  Scene Linear 被定义为 scene-wide color space

  • ACES 1.2：使用 RenderMan 24 随附的 ACES 1.2 color space

  • Custom：允许指定 custom color space file

Interactive Denoiser

此 Interactive Denoiser 在 Maya viewport 中启用，并提供 RenderMan Machine Learning Denoiser 的交互式版本。

Cheap First Pass：首次 denoise pass 会使用更快（更低成本）但准确度较低的 heuristic。

Min Samples：Denoiser 首次运行前所需的最低 average samples 数。

Interval：Denoise 运行之间的 time interval，单位为 seconds。

Stylized Looks

此 checkbox 会启用以下 Stylized Looks toolset，包括向 beauty pass 添加所需的 LPEs 和 AOVs。


Filters

这些选项允许使用 Display or Sample Filters，例如 Stylized Looks、Cryptomatte 和其他 effects。
