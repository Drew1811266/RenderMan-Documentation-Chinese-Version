PxrMarschner 是一种材质，专为渲染真实的头发、毛发和纤维而设计。

这是 Pixar 许多动画长片中使用的同一种材质和设计，现在提供给用户渲染自己的场景。控制项很少，并且对艺术家友好。该材质也具有物理可信特性，因此艺术家可以从真实结果开始，再根据艺术指导细化选项。

如果你想深入了解该材质的技术细节，可以查看  Pixar Technical Memo 15-02 "A Data-Driven Light Scattering Model for Hair"，作者为 Leonid Pekelis、Christophe Hery、Ryusuke Villemin、Junyi Ling。本文档将通过示例说明这些参数，让艺术家可以立即开始创作！

下面的示例会说明更改参数如何影响头发外观。




漫反射参数

漫反射模型有两种选择：Zinke 和 Kajiya。默认设置为 Zinke 漫反射模型，用于双向散射。参见 Importance Sampling for Physically-Based Hair Fiber Models。

Diffuse Gain

此参数是 Diffuse Color 的乘数。0.0 表示没有漫反射颜色，而 1.0 表示完整效果。默认值为 0.0，可提供非常真实的深色到中色头发，但渲染成本可能更高。对于铂金金发或灰发等较浅头发，你可能需要使用更高值。

Diffuse Color

这是头发本身的漫反射颜色。即使是金发，对于大多数真实头发来说，它也应比其他颜色参数更暗。为了获得最佳真实感，建议它与 Secondary Specular Color 相同，并只调整增益。默认值为中灰色。


镜面参数

大部分控制项都位于 Specular Parameters 下。每个控制项都以某种方式与下面的说明相关。

Marschner 镜面模型由三个镜面波瓣组成，使用上方方向箭头所示的多个镜面传输路径：

  • Primary Specular: R 用于 "reflect" 路径，即光线击中头发表面并反射。这是强烈的主要无色镜面反射。

  • Secondary Specular: TRT 用于 "transmit", "reflect", and "transmit" 路径，即光线穿过头发表面，然后在头发内部反射，并从头发前侧射出。这是次级有色镜面反射。

  • Transmit Specular: TT 用于 "transmit" and "transmit" 路径，即光线进入头发表面并从另一侧射出。这是会响应边缘光的有色透射。


这些 Specular Gain 控制项允许艺术家出于艺术目的打破能量守恒。如果输入值相加大于 1.0，则头发不再保持能量守恒。只有当你对外观不满意（太“热”）或希望从物理真实着色开始时，这一点才重要。

默认增益现在为 1.0，旧版本可能设置为 0.33。



Primary Specular Gain

Gain 用于 Marschner 镜面反射的 R lobe。这个 Marschner 镜面波瓣类似清漆，其中镜面反射相当锐利且有光泽，通常不带颜色。此镜面反射是直接反射，不会被任何体积衰减着色。对 R 波瓣的修改也有助于让头发看起来“湿润”。

Secondary Specular Gain

Gain 用于 Marschner 镜面反射的 TRT 波瓣。这是更粗糙且有色的 Marschner 镜面反射。它会两次穿过头发纤维体积。


Transmit Specular Gain

Gain 用于 Marschner 镜面反射的 TT 波瓣。这是一种透射型（折射）Marschner 镜面反射，并带有一些体积衰减。它会一次穿过纤维体积。较高值可能增加噪声。


Glint Gain

闪烁高光的 Gain。这是 Glints 效果的乘数，其工作方式类似其他增益控制项。



Primary Specular Color

用于给 Primary Specular（即 R 波瓣）着色的镜面颜色。对于自然头发和毛发，它应始终为白色。

Secondary Specular Color

Specular color 用于 TRT 和闪烁高光波瓣的镜面颜色。

由于它表示头发纤维内部的两条路径，因此它应比 Transmit Specular Color 更暗、更饱和；从技术上说，Transmit Specular Color 是 Secondary Specular Color 的平方根。为了获得更真实的头发，这应与 Diffuse Color匹配。

Transmit Specular Color

Specular color 用于 TT 波瓣的镜面颜色。以下示例设置为 Transmit Specular Gain 为 1.0，所有其他增益为 0.0

Primary Cone Angle

用于 R 波瓣的锥角（theta），单位为度。角度越宽，高光越柔和、越宽。

Secondary Cone Angle

用于 TRT 波瓣的锥角（theta），单位为度。角度越宽，高光越柔和、越宽。

Transmit Cone Angle

用于 TT 波瓣的锥角（theta），单位为度。角度越宽，高光越柔和、越宽。请注意，此参数的高增益会产生柔和且浅色的头发，但会生成相当多的噪声。这可能会在较高角度下增加渲染时间。

Specular Offset

锥偏移，单位为度；可用于沿头发移动镜面结果。

Refractive Index

头发的折射率（IOR）。较高数值会增加镜面性，类似其在不透明表面上的作用方式。这可以让头发纤维看起来几乎像金属。

Fresnel Mix

此控制项用于平衡沿头发的 Fresnel 能量衰减。 

Glint Width

此参数会平滑 Secondary Specular（TRT）和 Glints 分布。对于真实头发，此值通常应保持在 10 到 25 之间。

Eccentricity

这是头发的横截面宽高比，对于真实头发通常在 0.85 到 1.0 之间。渲染卷发时最常使用它。（请注意，这取决于头发法线；你应通过 PrimVar 提供用户法线，而不是指定 N。）


Glow

Glow 可用于创建看起来在场景中发光的头发纤维。 

Gain

Glow 增益或权重。下面我们从典型黑发开始，将增益（使用白色）增加到 0.5，然后增加到 1.0。

Color

控制材质的自发光颜色或 Glow。下面是 Color 参数的各种纹理输入示例。

Globals

本节提供整个材质的全局控制。Presence 也是全局参数，会遮罩或“镂空”所有波瓣，就像在遮罩值为 0.0 的位置对象不存在一样。


Specular Energy Compensation

此控制项允许在镜面响应占用能量时让漫反射结果变暗，并用剩余能量照亮漫反射波瓣。将其设为 1.0 对应物理效果。默认值为 off（艺术控制），且不保持能量守恒。


Presence

在此连接遮罩，将镂空图案应用到对象。Presence 被定义为二值（0 或 1）函数，也可取连续值以对形状进行抗锯齿。这对创建树叶和其他薄而复杂形状等镂空很有用。

此值应为 0 或 1。介于 0 和 1 之间的值会产生不需要的噪声！


Eccentricity Direction

此参数可用于覆盖几何法线，以控制 Eccentricity 参数。


Shadow Color

指定阴影颜色。此参数可用于出于艺术指导目的伪造阴影颜色。请注意，这会创建“透明阴影”，并会随着头发之间追踪更多光线而增加渲染时间。它也可能影响对象之间的全局照明。
