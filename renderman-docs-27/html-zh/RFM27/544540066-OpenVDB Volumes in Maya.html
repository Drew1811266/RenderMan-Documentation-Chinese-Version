OpenVDB Volumes in Maya

Source: https://rmanwiki-27.pixar.com/space/RFM27/544540066/OpenVDB+Volumes+in+Maya
Downloaded: 2026-06-15T03:49:12+00:00

OpenVDB 是面向体积数据的开源层级数据结构。RenderMan for Maya 支持 OpenVDB Maya 插件（$RMSTREE/plug-ins/OpenVDB.so）。


Maya 工具架中的 OpenVDB 图标

只需点击一次该图标，RfM 就会自动创建用于读取、可视化和渲染 OpenVDB 体积数据的默认设置。

有关 OpenVDB 的更多信息，请参阅 OpenVDB FAQ（OpenVDB 常见问题）。


  • OpenVDB 节点默认会将间接可见性设为 "off"；这是因为 Maya 早期默认设置会在 Attribute Editor（属性编辑器）的 Render Stats 展开栏中，将导入对象设为对反射/折射不可见。请打开这些选项以获得正确渲染。


如何读取 OpenVDB

在 RenderMan for Maya 中，若要读取 OpenVDB 文件，只需点击 OpenVDB 图标。这会创建一个 OpenVDB Read（OpenVDB 读取），用户可以通过文件选择器选择 OpenVDB 文件，并设置 VDB File Path（VDB 文件路径）。




如何可视化 OpenVDB

加载 OpenVDB 文件后， OpenVDB Visualize（OpenVDB 可视化） 选项卡会出现在 Attribute Editor（属性编辑器）中。

OpenVDB Visualizer（OpenVDB 可视化器）会在 Maya 中显示 OpenVDB 体积数据，并提供选项，用于更改 OpenVDB 数据输出到 RIB 的方式。

默认情况下，RenderMan for Maya 会输出 RiVolume。如果勾选 OpenVDBVisualize 节点中的 Surface 复选框，或者 OpenVDB 文件只包含水平集数据，则会改为输出 RiBlobby。在这种情况下，用户需要连接一个不同于默认 PxrVolume的着色器。可以添加 RenderMan for Maya 的 Custom Shading Group 属性来连接着色组。


RIB 选项

以下选项可用于更改 OpenVDB 体积数据输出到 RIB 的方式。

VDB Grid Name（VDB 网格名称）

VDB 网格是体素数据的稀疏树表示。例如，float 网格会为每个体素存储一个单精度浮点值。加载 OpenVDB 文件后，下拉菜单中会出现网格名称列表。选择你想采样的网格名称。

Surface（曲面）

这会创建 RiBlobby，而不是 RiVolume。请注意，开启此选项可能会减慢渲染。

Isovalue（等值）

此项设置体积的阈值。

可视化器选项

以下属性仅用于可视化。更改它们不会改变体积的渲染结果。

Active Value Bounding Box（活动值包围盒）

显示/隐藏活动值的包围盒。

Internal Nodes（内部节点）

显示/隐藏。具有动态分支因子的根节点包含固定分支的内部节点。

Leaf Nodes（叶节点）

显示/隐藏 存储在叶节点中的值对应于单个体素；所有其他值都对应于 tile（区块）

Active Tiles（活动区块）

显示/隐藏 存储在类型为  RootNode  或  InternalNode  的节点中的值，对应于索引空间中具有常量值和活动状态、且尺寸为 2 的幂的区域。 网格中的每个值都有一个二进制状态，称为 "active state"（活动状态）。对此开/关状态的解释取决于具体应用，但通常 active (on) 值在某种意义上是“有意义的”，而 inactive (off) 值则不那么有意义。

Active Voxels（活动体素）

显示/隐藏 同上，但适用于体素。 存储在类型为  LeafNode  的节点中的值（如果存在，这类节点具有固定深度）对应于单个体素。这些是索引空间中最小的可寻址单位。

OpenVDB 着色网络

OpenVDB Visualizer（OpenVDB 可视化器）图标会自动设置体积着色组，并连接到相应的 OpenVDB 可视化器。
