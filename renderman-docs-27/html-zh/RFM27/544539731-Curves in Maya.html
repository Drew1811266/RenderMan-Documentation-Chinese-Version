Maya Curves（Maya 曲线）

Maya curves（Maya 曲线）可以被标记为可渲染。使用以下位置的某个 curve tools（曲线工具）创建曲线：

Create > Curve Tools（创建 > 曲线工具）

在 Attribute Editor（属性编辑器）中，务必为曲线指定 material（材质）：

就是这样！你应该能在渲染中看到这条 curve（曲线）。

对于 hair/fur curves（毛发/毛皮曲线）：若要将 IGS/Interactive Xgen surface UVs 用于 color/texture（颜色/纹理），可以在颜色 placement（放置）的 PxrManifold2D 中将其引用为 scalpS 和 scalpT Primvars。

Ribbons（带状曲线）

默认情况下，RenderMan 会将 curves（曲线）渲染为 tubes（管状），这非常适合 hair（头发）和 fur（毛皮）。

对于 flat、ribbon style curves（扁平带状曲线），curve（曲线）需要提供 normals（法线）。Maya 默认没有此功能，因此可以通过一种以 RenderMan 为中心的方法借助 primvars（基元变量）。

  • 选择 curve（曲线）的 shape node（形状节点）（在 outliner 中选择该曲线并按向下箭头）。

  • 添加一个名为 rmanNN 的 RenderMan primvar

    • main menu > Modify（主菜单 > 修改）

    • Add Attribute...（添加属性...）

    • Long Name = rmanNN（长名称 = rmanNN）

    • data Type = Vector（数据类型 = Vector）

  • 将 curve shape node（曲线形状节点）中的 rmanNN 改为非 0 的任意值，例如 1, 0, 0。

  • 这会为 curve（曲线）提供一条 normal（法线），并扭转该曲线。
