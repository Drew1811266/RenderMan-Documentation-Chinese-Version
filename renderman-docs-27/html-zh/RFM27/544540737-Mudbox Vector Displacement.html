从 Mudbox 导出 FBX

开始雕刻之前，请先从 Mudbox 导出 FBX。

从 Mudbox 导出 OpenEXR

从 Mudbox 导出 PTex


导入 FBX 并添加 Subdiv Scheme

  • 导入刚从 Mudbox 导出的 FBX。对于它的 shape 节点，添加 Attributes|RenderMan|Subdiv Scheme:

指定位移着色器（UV 纹理）

  • 创建一个 PxrTexture 节点。将 Filename 设置为导出的 Mudbox EXR 贴图。其他参数保持默认值。

  • 创建  PxrDispTransform 节点。将 Displacement Type 设置为 Mudbox Vector 并将 Vector Space 设置为 Tangent。

  • 创建一个 PxrDisplace 节点（这是实际对表面执行位移的位移着色器）。

  • 将 PxrTexture 的输出 Result RGB 连接到 PxrDispTransform 的 Disp Vector。

  • 将 PxrDispTransform 的 Result XYZ 连接到 PxrDisplace 的 Disp Vector。你的图应如下所示：

指定位移着色器（PTex）

  • 创建一个 PxrPtexture 节点。将 Filename 设置为导出的 Mudbox PTex 贴图。其他参数保持默认值。

  • 创建  PxrDispTransform 节点。将 Displacement Type 设置为 Mudbox Vector 并将 Vector Space 设置为 World。

  • 创建一个 PxrDisplace 节点（这是实际对表面执行位移的位移着色器）。

  • 将 PxrTexture 的输出 Result RGB 连接到 PxrDispTransform 的 Disp Vector。

  • 将 PxrDispTransform 的 Result XYZ 连接到 PxrDisplace 的 Disp Vector。你的图应如下所示：

渲染

  • 添加 PxrSurface，将它指定给模型，并添加一些 灯光，适当设置置换边界，然后渲染！
