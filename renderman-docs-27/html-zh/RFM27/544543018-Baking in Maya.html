Baking in Maya

Baking（烘焙） 允许用户将 Pattern 网络预计算或“烘焙”到 2D 纹理图像或 3D 点云中。 

Bake Patterns（烘焙 Patterns）

添加 PxrBakeTexture 或 PxrBakePointCloud 将这些 Pattern 节点插入到你的着色网络中，也就是你希望将网络的该状态烘焙到纹理或点云的位置。执行常规渲染时，这些 Pattern 会使用已烘焙纹理；执行烘焙渲染时，纹理会被烘焙，也就是写入磁盘。

 

Starting a Bake Render（启动烘焙渲染）

烘焙渲染本质上是一个只有一项设置不同的批渲染。它使用 bake Hider，而不是 raytrace Hider。要启动烘焙渲染，只需在 RenderMan menu 中选择 Bake Render，而不是 Batch Render。所有常规的 批渲染首选项 都会应用到烘焙渲染。烘焙后的纹理会写入各个 PxrBakeTexture 或 PxrBakePointCloud Pattern 中指定的位置。
