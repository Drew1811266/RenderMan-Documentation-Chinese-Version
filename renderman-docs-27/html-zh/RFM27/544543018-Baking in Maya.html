Baking（烘焙） 允许用户将 pattern networks 预计算或 "bake" 到 2D texture images 或 3D point clouds 中。 

Bake Patterns（烘焙 Patterns）

添加 PxrBakeTexture 或 PxrBakePointCloud patterns 到你的 shading network 中，也就是你希望将 network 的该状态 bake 到 texture 或 point cloud 的位置。执行 regular renders 时，这些 patterns 会使用 baked textures；执行 bake renders 时，textures 会被 baked，也就是写入 disk。

 

Starting a Bake Render（启动 Bake Render）

bake render 本质上是一个只有一项 setting 不同的 batch render。它使用 bake Hider，而不是 raytrace Hider。要启动 bake render，只需在 RenderMan menu 中选择 Bake Render，而不是 Batch Render。所有常规的 batch render preferences 批量渲染首选项 都会应用到 bake render。baked textures 会写入各个 PxrBakeTexture 或 PxrBakePointCloud patterns 中指定的位置。
