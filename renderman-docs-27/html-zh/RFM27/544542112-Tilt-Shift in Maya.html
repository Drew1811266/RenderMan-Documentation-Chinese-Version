移轴选项实际上是摄影机中的两组不同设置。Tilt（倾斜）和 Roll（滚转）控件用于让用户将镜头指向与摄影机机身不同的方向。Shift Options（平移选项）允许用户在镜头仍指向与摄影机相同方向的同时，将镜头沿 X（横向）和 Y（升降）移动。Focus 1、2 和 3 允许用户用三角形定义任意对焦平面（与 Tilt 控件相关）。下面会逐项进一步说明。

要访问这些控件，请打开 Camera Attribute Editor（摄影机属性编辑器），并在 RenderMan 展开项下选择“Enable Camera Projection”。

你会注意到 Camera Projection（摄影机投影）字段为空。右键单击会出现两个选项：Omnidirectionalstereo（用于 VR 和类似应用）和 PxrCamera，请选择后者。

务必将你的 视场（Field of View） 与 Maya 摄影机匹配，否则可能无法获得与你在视口中看到的相同构图。也可以在摄影机编辑器上选择“Use Camera Settings”。

倾斜角（Tilt Angle）： 此项允许将镜头指向不同于摄影机机身的方向。如果你在 Maya 中的景深 一节中注意到，对焦平面与成像平面平行。但如果主体并不平行于图像平面怎么办？在建筑场景中，你可能有一面墙以一定角度远离摄影机，并希望在不改变摄影机构图的情况下让整面墙保持焦内。Tilt（倾斜）可以通过倾斜镜头来做到这一点，使镜头不再垂直于图像平面。

滚转角（Roll Angle）： 此选项允许用户顺时针旋转镜头，并且只有在应用了一定程度的倾斜时才有效。结合倾斜和滚转意味着镜头可以从摄影机机身指向一个有一定限制的半球范围。

对焦点（Focus Points）： 你可能还记得几何课上讲过：三个点定义一个平面或三角形。这个三角形可以在 RenderMan 中由用户定义，用来创建任意对焦平面。下面展示如何在 Maya 中使用定位器以更交互的方式完成此操作，从而获得精确的选择性对焦。

这些点都位于场景的世界空间中。手动调整它们可能非常繁琐，因此建议使用下面的工作流程，以可视化方式定义对焦平面：

  1. 创建一个多边形三角形。

  2. 使用 Animation Deformer（动画变形器）菜单，在每个顶点上创建一个簇控制柄（cluster handle）。先选择顶点，然后选择 Animation Menu > Anim Deform > Open Full Deform Menu > Cluster

  3. 在 Create Menu > Locator 下创建一个定位器，并将它吸附到该顶点。将簇控制柄作为定位器变换的子级。现在移动定位器就会移动簇和它的顶点。

现在你应该得到一个多边形三角形，它会在移动定位器时改变位置。注意，定位器是额外步骤；你也可以只使用簇控制柄。不过，定位器在视口中很容易选择，也可以使用 Show Menu 将它们从视图中排除。记得在渲染前使用预渲染脚本隐藏该三角形，否则它会在场景中渲染出来；或者你也可以选择三角形形状，并关闭所有 Render Stats 使其不被渲染。RenderMan 会在渲染时报告一条警告，说明它不可见。

现在你已经有办法看到对焦平面，需要将定位器（或簇控制柄）的变换输出连接到 Focus Parameters（对焦参数）（用于 PxrCamera）。

  1. 从 Outliner 中选择 PxrCamera（可以通过在 Outliner Window 中 取消选择 Display > DAG Object Only 使它可见），也可以使用这个方便的脚本： 

    select -r PxrCamera ;

  2. 将它和一个定位器变换加载到 Connection Editor（连接编辑器）中。将 Locator 的 Translate（平移） 连接到每个定位器对应的 PxrCamera Focus（对焦）。

X/Y 平移（Shift X/Y）： 此控件会将镜头上下或前后平移。在建筑领域，此控件用于在构图中保持平行线。例如：在某些情况下，高楼可能看起来像向后倒，因为楼顶似乎收敛到一个点。平移可以校正这种透视视图。

在下面的图像中，想象你正在用摄影机构图一栋建筑，但建筑太高，无法放入视图中。于是你将整个摄影机向上倾斜并构图包含整栋建筑。但这时会产生透视畸变。在最后一帧中，镜头被平移 以校正畸变。
