PxrVary

该 PxrVary Pattern 节点允许随机变化。我们来试着根据 Particle ID 改变茶杯的颜色。

选择 nParticleShape1 节点。查看 Instancer Attributes（实例化器属性）。将 Object Index 从 None 改为 Particle ID。

这会使一个名为 particleId 的 primvar 可用于这些实例。

接下来，把 PxrVary Pattern 节点连接到每个杯子和碟子所用材质的颜色。

设置 PxrVary，使其使用 particleId 作为随机化种子。  

  • 将 Vary Source 改为 Primvar。

  • 输入 particleId 作为 Variable Name。

  • 将 Hue 滑块值移到 1，以获得最大的色相变化。
