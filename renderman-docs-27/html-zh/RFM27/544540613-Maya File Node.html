Maya File Node

Source: https://rmanwiki-27.pixar.com/space/RFM27/544540613/Maya+File+Node
Downloaded: 2026-06-15T03:49:12+00:00

Maya file node（Maya 文件节点）提供了一种方便方式，可将纹理应用到曲面并驱动参数。不过，RenderMan 只支持基于首选工作流的一部分控制项。

 

Supported Features（支持的功能）：

File Attributes（文件属性）

Image Name（图像名称） - 你要使用的纹理路径

UV Tiling Mode（UV 平铺模式） - 除 Explicit Tiles（显式贴片）外，我们支持所有模式

Use Image Sequence（使用图像序列） - 这可能比在其他节点中使用表达式更方便

 

Color Balance（颜色平衡）

Default Color（默认颜色） - 当没有纹理覆盖时，此颜色会填充这些区域

Color Gain（颜色增益） - 这是纹理的颜色乘数

Color Offset（颜色偏移） - 将此颜色值添加到纹理中

Alpha Gain（Alpha 增益） - 纹理 alpha 值的乘数（你的纹理需要 alpha 通道，除非你已选择 Alpha is Luminance）

Alpha Offset（Alpha 偏移） - 将此值添加到纹理的 alpha 值中（你的纹理需要 alpha 通道，除非你已选择 Alpha is Luminance）

 

Effects（效果）

Invert（反相） - 反转纹理的值，颜色会变为色轮上的相反值

Color Remap（颜色重映射） - 使用渐变来改变色相和对比度值

 

RenderMan Controls（RenderMan 控制项）： 

我们使用 RenderMan rollout（RenderMan 卷展栏）中包含的自有文件过滤选项。我们推荐 Box，它也是默认值。
