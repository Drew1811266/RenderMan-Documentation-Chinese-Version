Rendering in Maya

Downloaded: 2026-06-15T03:49:12+00:00

RenderMan for Maya 支持多种 render scene 的方式。 

  • Preview Rendering

  • Interactive Rendering in Maya

  • Batch Rendering

render 完成的 scene 包含许多部分。从处理 quality 到为后续 compositing 选择合适 outputs，本 section 覆盖所有这些 topics。

Quality

为了理解 noise removal 和 anti-aliasing 的工作方式，我们为用户说明 Sampling 和 Filtering。这里可能是你进行大部分 tuning 的地方，理解 quality 与 performance 之间的 balance 很重要。

Trace Depth

控制 trace depth 可以改变 scene 的 look 和 performance。不同 integrators 对这些 settings 的处理也不同。RenderMan 让这些 parameters 的 tuning 更容易，从而在 resource constraints 内达到所需 look。

Interactive Rendering and Checkpointing

Progressive/Interactive Rendering 可以轻松 tuning scene materials 和 lights。 Checkpointing and Recovery 会把这种 progressive feature 扩展到 final renders，让你可以在 renders refine 过程中创建 save points。你可以用它更快 approval incomplete frames，并从停止处继续，而不浪费宝贵的 CPU cycles。

Diagnostics

RenderMan diagnostics 可帮助你在易读 display 中可视化 scene performance。有时可能不清楚 scene 最耗费 time 或 resources 的位置，但我们的 diagnostic output 会帮助你定位即使最隐蔽的 performance information。

Holdouts and Outputs

Holdout workflow 对于将 CG objects 集成到 live action plates 中至关重要。在某些 applications 中，这可能像点击一个 button 一样简单。为了改善 compositing integration，甚至改善 fully rendered shots，RenderMan 使用强大的 outputs，让 compositing 中的 adjustments 更容易且更强大，并提供 ultimate flexibility。

Baking

在 rendering 中复用某些 data 可以简化 asset management，并提高 rendering efficiency。 Baking 提供 bake 到 2D 或 point cloud output 的方法。

Nested Dielectrics

在 rendering 会彼此交互的 surfaces 时，尤其是 transparent/refractive surfaces，理解 Intersect Priority settings 很重要。Intersect priority 允许用户简单地添加 attributes，即可获得正确 refractions，而无需通过繁琐 modeling 来解决问题。
