避免多边形网格裂缝

对具有硬法线的网格进行置换时，面可能会分离并产生裂缝。置换会沿顶点法线方向移动顶点，但硬边两侧的相邻顶点具有不同法线，会向不同方向移动，从而产生间隙或裂缝。

RenderMan 有一个 "polygon:smoothdisplacement" 属性，它会创建一份完全平滑的法线副本，并使用它代替原始法线。

在 RenderMan for Maya 中，此属性通过名为 "Prevent polygon cracking" 的复选框启用；其内部属性名为 rman_preventPolyCracks。 

"Prevent polygon cracking" 只应在必要时启用，因为它会使用稍多一些内存。使用量与网格中的顶点数量成正比。


请注意，使用平滑法线会如何扭曲立方体的置换图案并改变其外观：

你可以通过在尖锐边附近插入额外边来避免这种效果。这会将法线插值限制在边缘附近区域，并通常保留原始外观。
