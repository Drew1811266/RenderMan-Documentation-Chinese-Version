Starting IPR

Interactive Preview Rendering 可以通过点击 RenderMan shelf 中的 IPR button 启动。

Maya 中启动 IPR 的常规方法也同样可用。唯一不同是，RenderMan shelf 中的 button 在开始 render 前总会先切换到 RenderMan。

你也可以使用 Render Settings 中的 Interactive Refinement control，让 RenderMan 决定像素更新频率，从而提供更快的 preview；它会在后续几次迭代中更新或“refine”并填充 pixels。

IPR button 提供一个 menu，其中有一些便捷选项，可用于从特定 camera 或 resolution 启动。

  • Stop - 结束 interactive session

  • Flush Texture Cache - 如果你修改了 textures 并希望看到变化，请 flush cache，以便在 session 中更新它们

  • Render Selected Object Only - 只渲染 viewport 中选中的 objects。注意不要意外保持此选项开启，否则你可能会误以为这是错误

  • Camera - 指定用于 render 的 camera（如果不是来自最后一个 active viewport）

  • Resolution - 指定 resolution（测试百分比等）

Stopping IPR

IPR 运行时，可以从 shelf button 的 popup menu 停止它。

"It" 中也有一个 cancel button，可用于停止 IPR。

Viewport Rendering

性能注意事项：viewport rendering 会按 monitor 中窗口的 resolution 进行渲染。例如，native resolution 为 4k 的 monitor 会触发匹配该 resolution 的 render，即使你的 render settings 指定了较低的设置。如果你不习惯按屏幕 resolution setting 进行渲染，这可能会让人意外。

交互式 rendering 时，我们会在 viewport 底部显示 progress bar。如果你分心了，或因为画面已经高度 converged 而看不出视觉进度，窗口底部的 progress indicator 可以根据 quality settings 告诉你 rendering 距离完成还有多远。


Viewport 左上角有一个 button，可用于 start/stop IPR、playblasting、调整 resolution、创建 crop window、快速切换到 PxrVizualizer Integrator、restart（用于不会触发 update 的变化），或拍摄 snapshot。

你可以通过右键点击 IPR button 来切换 viewport 底部的 progress bar。此显示的相关 settings 可在 RenderMan Preferences中找到。

此外，你可以右键点击 camera icon，将 snapshot 发送到 Image Tool（"it"），可选择带或不带 geometry outlines。默认使用 anti-aliased version，但取消选择 "smooth geometry" 后可关闭。

crop window tool 可以通过在窗口中绘制或选择 corners 来设定/重新设定大小。从 center 拖动 mouse 会移动该 window；点击 window 外部会将其重置为 viewport size，从而取消 crop。

使用 PxrVisualizer Integrator 非常适合快速浏览 scenes；取消选择后，它会恢复到先前选择的 integrator。

IPR Threads

在 RenderMan Preferences的 threads section 中，Rendering Threads setting 同时用于 Preview 和 IPR renders。默认值 -2 表示除两个 CPU 外其余都将用于 rendering。建议保留 至少 一个 thread，以便 rendering 进行时 Maya 仍保持 interactivity。Zero threads 表示所有 processors 都将用于 rendering。你也可以选择 specific number。

IPR Display

在 Maya Preferences Window 的 RenderMan section 中，你可以选择 IPR 使用 Image Tool (It) 图像工具窗口 还是 Maya 的 Render View。

Select Object or Material from Render（从渲染中选择对象或材质）

在 "It" 中运行 IPR 时，可以很方便地从 render 中选择 objects 或其 materials。点击 "It" 中的这些 icons 之一来激活 object 或 material picking，然后在 render 中点击；对应的 object 或 material 将在 Maya 中被选中。

Hidden Objects

渲染到 IPR 时，templated nodes 会从 export 中裁剪，但 hidden objects 不会裁剪（以便你可以交互式取消隐藏它们）。在 batch render 中，hidden 和 templated nodes 都会从 export 中裁剪

名为 "Prune Invisible Nodes" 的 preference 会在 IPR 期间生效，可在 RenderMan Preferences 中找到。默认情况下，IPR 期间 invisible nodes 不会被裁剪，以便轻松交互式更改 visibility。该设置位于 Preferences > RenderMan > IPR。

Crop Rendering in It

要在 IPR 期间于 "It" 中进行 crop render，请点击 crop button。

在 "It" window 中绘制一个 crop region。然后按 'r' key 触发 IPR update。按下 'r' key 后，pixels 只会在 crop window 内更新。

Remove or Resize Crop Window（移除或调整裁剪窗口）

点击 "It" 中 crop window 的内部，使其可编辑。当 crop window 可编辑时，右上角会出现一个 'x'。拖动任意 edge 可 resize 调整大小；也可以点击右上角的 'x' 移除 crop window。点击 crop window 外部可使其再次不可编辑。

如果你 resize 调整了 crop window 的大小或将其移除，请再次按 'r' key，表示 IPR 应开始使用该 region。

Crop Rendering in Render View

要在 Render View 中进行 crop render，只需在 image 中点击并拖动以绘制 crop region。IPR 会立即使用新 region 更新。要绘制新 region，请再次点击并拖动。只需在 image 中点击一次即可移除 crop region。

Rendering Selected Objects

若要阻止 objects 被 export，可以选择这些 objects，并将它们放入一个 Maya Display Layer。然后将该 layer 切换为 Template。作为其成员的 objects 将不会被 export。否则，无论 visibility 如何，objects 都会被 export，这样在 IPR rendering 期间你就能快速让 objects 可见/不可见，而不必重新 export 或重启 render。

你也可以从 Maya Render View 中选择该 option 来 render selected objects：Windows > Rendering Editors > Render View > Render > Render Selected Objects Only；或者用这个 MEL command 切换 mel shelf button：switchSelectObj;

或者，你可以使用以下命令指定 on/off buttons：optionVar -iv "renderViewRenderSelectedObj" 0; 其中 0 表示 off，1 表示 on。

Playblasting

在 RenderMan Preferences 中，现在有一个 section 可用于指定从 viewport 中进行 RenderMan Playblasting 的 controls。

你 必须 使用随附的 RenderMan Playblast menu，因为 Maya menu 不会调用我们的 tools 来 render playblast。

选择 Playblast RenderMan viewport icon 时，会弹出上面的 menu。它会采用 Maya Playblast window 中的 settings，同时也为 Quality 和 "it" viewing 添加 controls。按下 "OK" 会开始 Playblast process。请注意，每张 image 都会达到 quality 下所选的 values 后，才会进入下一帧。更高 quality 可能会使 playblast 的 render 时间显著增加。对于 final quality renders，请使用标准的 Batch Rendering options。
