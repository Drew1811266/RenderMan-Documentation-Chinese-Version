Starting IPR

交互式预览渲染可以通过点击 RenderMan 工具架中的 IPR 按钮启动。

Maya 中启动 IPR 的常规方法也同样可用。唯一不同是，RenderMan 工具架中的按钮在开始渲染前总会先切换到 RenderMan。

你也可以使用 Render Settings 中的 Interactive Refinement 控件，让 RenderMan 决定像素更新频率，从而提供更快的预览；它会在后续几次迭代中更新或“细化”并填充像素。

IPR 按钮提供一个菜单，其中有一些便捷选项，可用于从特定相机或分辨率启动。

  • Stop - 结束交互式会话

  • Flush Texture Cache - 如果你修改了纹理并希望看到变化，请刷新缓存，以便在会话中更新它们

  • Render Selected Object Only - 只渲染视口中选中的对象。注意不要意外保持此选项开启，否则你可能会误以为这是错误

  • Camera - 指定用于渲染的相机（如果不是来自最后一个活动视口）

  • Resolution - 指定分辨率（测试百分比等）

Stopping IPR

IPR 运行时，可以从工具架按钮的弹出菜单停止它。

"It" 中也有一个取消按钮，可用于停止 IPR。

视口渲染

性能注意事项：视口渲染会按显示器中窗口的分辨率进行渲染。例如，原生分辨率为 4k 的显示器会触发匹配该分辨率的渲染，即使你的渲染设置指定了较低的设置。如果你不习惯按屏幕分辨率设置进行渲染，这可能会让人意外。

交互式渲染时，我们会在视口底部显示进度条。如果你分心了，或因为画面已经高度收敛而看不出视觉进度，窗口底部的进度指示器可以根据质量设置告诉你渲染距离完成还有多远。


视口左上角有一个按钮，可用于启动/停止 IPR、playblast、调整分辨率、创建裁剪窗口、快速切换到 PxrVisualizer 积分器、重新启动（用于不会触发更新的变化），或拍摄快照。

你可以通过右键点击 IPR 按钮来切换视口底部的进度条。此显示的相关设置可在 RenderMan Preferences中找到。

此外，你可以右键点击相机图标，将快照发送到 Image Tool（“it”），可选择带或不带几何轮廓。默认使用抗锯齿版本，但取消选择“smooth geometry”后可关闭。

裁剪窗口工具可以通过在窗口中绘制或选择角点来设定/重新设定大小。从中心拖动鼠标会移动该窗口；点击窗口外部会将其重置为视口大小，从而取消裁剪。

使用 PxrVisualizer 积分器非常适合快速浏览场景；取消选择后，它会恢复到先前选择的积分器。

IPR Threads

在 RenderMan Preferences的 Threads 部分中，Rendering Threads 设置同时用于预览和 IPR 渲染。默认值 -2 表示除两个 CPU 外其余都将用于渲染。建议保留 至少 一个线程，以便渲染进行时 Maya 仍保持交互性。0 个线程表示所有处理器都将用于渲染。你也可以选择具体数量。

IPR Display

在 Maya 偏好设置窗口的 RenderMan 部分中，你可以选择 IPR 使用 Image Tool (It) 图像工具窗口 还是 Maya 的 Render View。

从渲染中选择对象或材质

在“it”中运行 IPR 时，可以很方便地从渲染图像中选择对象或其材质。点击“it”中的这些图标之一来激活对象或材质拾取，然后在渲染图像中点击；对应的对象或材质将在 Maya 中被选中。

隐藏对象

渲染到 IPR 时，templated 节点会从导出中裁剪，但 hidden 对象不会裁剪（以便你可以交互式取消隐藏它们）。在批渲染中，hidden 和 templated 节点都会从导出中裁剪

名为“Prune Invisible Nodes”的偏好设置会在 IPR 期间生效，可在 RenderMan 偏好设置中找到。默认情况下，IPR 期间 invisible 节点不会被裁剪，以便轻松交互式更改可见性。该设置位于 Preferences > RenderMan > IPR。

在 It 中裁剪渲染

要在 IPR 期间于 "It" 中进行裁剪渲染，请点击裁剪按钮。

在 "It" 窗口中绘制一个裁剪区域。然后按 'r' 键触发 IPR 更新。按下 'r' 键后，像素只会在裁剪窗口内更新。

移除或调整裁剪窗口

点击 "It" 中裁剪窗口的内部，使其可编辑。当裁剪窗口可编辑时，右上角会出现一个 'x'。拖动任意边可调整大小；也可以点击右上角的 'x' 移除裁剪窗口。点击裁剪窗口外部可使其再次不可编辑。

如果你调整了裁剪窗口的大小或将其移除，请再次按 'r' 键，表示 IPR 应开始使用该区域。

在 Render View 中裁剪渲染

要在 Render View 中进行裁剪渲染，只需在图像中点击并拖动以绘制裁剪区域。IPR 会立即使用新区域更新。要绘制新区域，请再次点击并拖动。只需在图像中点击一次即可移除裁剪区域。

渲染所选对象

若要阻止对象被导出，可以选择这些对象，并将它们放入一个 Maya Display Layer。然后将该层切换为 Template。作为其成员的对象将不会被导出。否则，无论可见性如何，对象都会被导出，这样在 IPR 渲染期间你就能快速让对象可见/不可见，而不必重新导出或重启渲染。

你也可以从 Maya Render View 中选择该选项来渲染所选对象：Windows > Rendering Editors > Render View > Render > Render Selected Objects Only；或者用这个 MEL 命令切换 MEL 工具架按钮：switchSelectObj;

或者，你可以使用以下命令指定开/关按钮：optionVar -iv "renderViewRenderSelectedObj" 0; 其中 0 表示 off，1 表示 on。

Playblasting

在 RenderMan Preferences 中，现在有一个部分可用于指定从视口中进行 RenderMan Playblast 的控件。

你 必须 使用随附的 RenderMan Playblast 菜单，因为 Maya 菜单不会调用我们的工具来渲染 playblast。

选择 RenderMan Playblast 视口图标时，会弹出上面的菜单。它会采用 Maya Playblast 窗口中的设置，同时也为质量和“it”查看添加控件。按下“OK”会开始 Playblast 过程。请注意，每张图像都会达到质量下所选的值后，才会进入下一帧。更高质量可能会使 playblast 渲染时间显著增加。对于最终质量渲染，请使用标准的批渲染选项。
