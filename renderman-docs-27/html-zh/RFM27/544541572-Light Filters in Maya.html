Light Filters in Maya

Source: https://rmanwiki-27.pixar.com/space/RFM27/544541572/Light+Filters+in+Maya
Downloaded: 2026-06-15T03:49:12+00:00

Light Filters 用于操控 lighting results。在真实世界中，这通常通过使用 stencils 和 gels 来改变镜头中 light 的 color、quality 和 amount。RenderMan 的 light filters 是一种强大且灵活的方式，可以在为 scenes 布光时实现同样的效果，甚至做得更多。 

Light filters 必须 始终 属于 Maya 的 Default Light Set，才能正确工作。你可以在 Maya Outliner 中展开 defaultLightSet group 来检查这一点。此 set 之外的任何内容都可以拖入其中以生效。

Creating Light Filters（创建灯光滤镜）

要在 Maya 中创建 light filter，先选择一个或多个 RenderMan lights。然后右键单击 RenderMan shelf 中的 light filter 按钮，并选择一种 light filter 类型。light filter 会被创建并连接到所选 light。为方便起见，light filter 会创建在与所选 light 相同的位置。通常你会希望把 light filter 从 light 稍微向前移动。默认情况下，light filter 不会 parent 到 light 之下；但如果你希望 light filter 始终随其 parent light 一起移动，也可以这样做。

对于所有 RenderMan light types，在 Attribute Editor 中，你会在靠近底部的位置看到一个名为 "Light Filters" 的 section。shelf 按钮会自动连接 light filters，但也可以通过右键单击该 section 中的某个 control 来创建并连接。可以连接多个 light filter。每次连接一个 filter，都会出现一个新的 control，以便再连接另一个。

Light Filter Types（灯光滤镜类型）

有关 parameters 的详细信息，请进入每种 light filter type 的链接。

  • PxrBarnLightFilter

  • PxrBlockerLightFilter

  • PxrCookieLightFilter

  • PxrGoboLightFilter

  • PxrIntMultLightFilter

  • PxrRampLightFilter

  • PxrRodLightFilter

Barn Door Light Filter

PxrBarnLightFilter 允许你创建物理准确的 window barns，用正确的 shadowing 模拟真实片场 lighting。它的其他用途包括在大型 scene 中控制 bounce lights。除 physical mode 外，它还提供 analytic mode。

Blocker Light Filter

PxrBlockerLightFilter 使用类似 "rod" 的 object 来阻挡 light。blocker 可以塑造成不规则形状，然后放置在你希望阻挡 light 的 object 旁边。

Rod Light Filter

PxrRodLightFilter 与上面的 PxrBlockerLightFilter 类似，但提供更多 controls 和 flexibility。

Gobo Light Filter

PxrGoboLightFilter 会在 light 前方投射 painted texture。

Cookie Light Filter

PxrCookieLightFilter 与 PxrGoboLightFilter 类似，但具有更多 options 和 flexibility。

Intensity Light Filter

PxrIntMultLightFilter 是一种 light filter，可让你倍增 light 的 intensity/exposure。当你想从 scene 其余部分中隔离出某个特定 asset，并让它具有不同的 intensity/exposure 时，这非常有用。

PxrIntMultLightFilter 的位置并不重要。你会希望使用 Maya 的 light linking，将此 light filter 链接到需要让特定 lighting effect 更亮或更暗的特定 objects。

Ramp Light Filter

PxrRampLightFilter 使用 ramp 来控制 light。这允许对 color、intensity 等进行复杂 control。
