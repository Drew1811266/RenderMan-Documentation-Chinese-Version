PxrAOVLight in Maya

PxrAovLight 是一种方便的 utility light，允许 lighting artist 输出 shot-specific AOV mask，而不需要向 shading artists 请求。在这个示例中，我们会拆解一个常见需求：为角色的眼睛获取 refraction AOV（cornea 内部的 sclera 或 iris）。

为了进一步简化，我们会用 glass sphere 内部的 red sphere 来演示。

Maya and RIB file（Maya 与 RIB 文件）

Light Linking with PxrAOVLight（使用 PxrAOVLight 进行灯光链接）

使用 light linking 在 AOV 中 include/exclude 几何体部分。在此示例中，PxrAovLight 在全局上为 "off"，并在 scene 中对眼睛为 "on"。此 linking setup 会为 refractive glass 内部的 red sphere 创建一个 mask。 

Defining the Output AOV（定义输出 AOV）

在 PxrAOVLight attributes 中设置 AOV Name 后，使用 Render Settings AOVs 选项卡定义一个新的 display，PxrAovLight 会写入该 display。
