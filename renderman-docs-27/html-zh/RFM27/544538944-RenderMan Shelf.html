RenderMan Shelf

Downloaded: 2026-06-15T03:49:12+00:00

RenderMan for Maya 提供自己的 Shelf，其中包含该插件许多基础且常用功能的快捷方式。

Rendering（渲染）

按钮菜单说明

Start 一次预览渲染。

Start 交互式预览渲染。选择要从哪台摄像机进行渲染。

也可以只渲染选中的对象（此选择同样适用于

预览渲染）


打开 Image Tool，即 'It'


打开 Localqueue 渲染管理器


Lighting（灯光）

按钮菜单说明

创建新的 灯光。

Auto-place lights 可在选中的对象处创建灯光

即在视口中的对象位置创建，而不是在原点创建。

创建新的 穹顶灯, 门户灯 或 日光灯。


将选中的几何体转换为 网格灯光。

创建一个 灯光滤镜 并将其附加到选中的灯光


Shading（着色）

按钮菜单说明

创建一个表面并将其附加到选中的几何体

创建一个 PxrVolume 或 OpenVDB Visualizer。

从选中的几何体创建遮挡保留

Stylized Looks



Archive / Texture / Statistics（归档 / 纹理 / 统计）

按钮菜单说明

创建 RenderMan Archive


打开 Dynamic Shader Binding editor（动态着色器绑定编辑器）


打开 Texture Manager


打开 Preset Browser


Live Statistics


查看上次渲染的统计信息



Plugin / Help（插件 / 帮助）

按钮菜单说明


加载 RenderMan for Maya plug-in


在网页浏览器中打开 RenderMan 文档
