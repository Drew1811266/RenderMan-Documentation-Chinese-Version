External Editors（外部编辑器）

Text Editor（文本编辑器）

设置用于打开 ASCII RIB 文件的默认编辑器。

Node Editor（节点编辑器）

Enable Solo（启用 Solo）

允许在 Node Editor 中单独显示所选节点，方式与在 Hypershade 中选择 Solo 按钮相同。

Solo Key（Solo 快捷键）

选择用于单独显示节点的热键，默认为反引号键。

Channel Box（通道盒）

Attributes in Channel Box（Channel Box 中的属性）

切换 Channel Box 是否显示 RenderMan 控件。

Script Editor Widget（Script Editor 控件）

Font Size（字体大小）

更改显示字体的大小。

Font（字体） （字体族）

更改编辑器中使用的字体。

Tab Width（Tab 宽度）

按下 Tab 时，在编辑器中移动指定数量的空格。
