PxrVoronoise

Source: https://rmanwiki-27.pixar.com/space/REN27/542225517/PxrVoronoise
Downloaded: 2026-06-15T03:49:12+00:00


与所有 texture style nodes（纹理样式节点）一样，此节点接收一个 manifold，用来描述应用 voronoise 纹理的 2D 或 3D 域。如果未连接 manifold，默认行为是在 3D 中沿 P 应用。此节点会计算 voronoise 函数，也就是 noise 与 Voronoi 的混合，如 Inigo Quilez 所描述。

jitter 和 smoothness 参数允许你创建不同类型的图案。

  • jitter=0,smoothness=0 会生成最小距离、未抖动的数值网格，类似 RSL 的 cellnoise。
  • jitter=0,smoothness=1 会得到一个 noise 函数，类似 Perlin 噪声。
  • jitter=1,smoothness=0 会生成抖动后的最小距离 Voronoi 图案。
  • jitter=1,smoothness=1 会生成抖动 Voronoi 与 noise 的组合：“voronoise”！

与 Inigo 的代码相比，此版本还添加了 fractal octaves（分形倍频层）和 turbulence（湍流）


 

Manifold


用于应用 noise 的 manifold。（默认值为 P）。

Global

Surface Position

Frequency

设置 noise layers（噪声层）的最低（起始）频率。



Octaves

使用的 noise octaves（噪声倍频层）数量。更多 octaves 会逐级添加更多 noise。


Inner Loop

Gain

也称为 persistence（持续度）。用于缩小或放大每个后续 noise octave 的振幅。当 gain = 1/lacunarity 时，可得到 1/f noise。



Lacunarity

用于决定每个后续 noise octave 频率的缩放因子。较小值会使各层在频率上更紧密；较大值会使它们间隔更远。



Jitter

控制应用的 jitter（抖动）量。0 给出规则网格，1 给出类似 Voronoi 的网格。



Smoothness

使用 0 会产生 Voronoi/cellnoise 图案，1 会得到平滑 noise。



Turbulent

是否让 noise 呈现 turbulence（湍流）

Better Contrast

在 smoothness 增加时提升对比度

Octaves

Smoothness Offset

按 octave 增加/减少 smoothness


Smoothness Offset Wrap

对 smoothness 范围（0-1）进行循环，或仅进行 clamp

Advanced

Filter

更改 filter size（滤波尺寸）


Adjust Output

Color Scale

纹理颜色值的乘数，可用于调整亮度或操控单独的颜色通道


Color Offset

对结果应用 offset，使结果的颜色发生偏移

Float Scale

标量 float 值 

Float Offset

Float Offset 值 


Output Parameters


resultF

voronoise 纹理的结果。
