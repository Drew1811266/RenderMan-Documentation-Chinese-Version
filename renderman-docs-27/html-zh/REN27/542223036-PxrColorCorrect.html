PxrColorCorrect

PxrColorCorrect 结合了多种经典重映射和颜色校正方法。

Input Parameters（输入参数）

Input Color

在此连接输入颜色 pattern。

Mask

Input Mask

用于定义经过颜色校正的区域的遮罩。

Invert Mask

反转遮罩的影响。

Mix Mask

Blend 进遮罩中。设置为 0.0 时，完全不会进行颜色校正。

Input Range

Input Min

Input Min 会将 0.0 值重映射为你选择的值。设置为 0.0 时，不会发生变化。这等同于 Photoshop 的 Levels 对话框中的黑场点。

Input Max

Input Max 会将 1.0 值重映射为你选择的值。设置为 1.0 时，不会发生变化。

Color Correct

Gamma

应用逐通道 gamma 校正。低于 0.0 的值会被忽略，1.0 为中性。

Contrast

应用逐通道 Contrast。有效范围为 -1 到 +1，0.0 为中性。

Contrast Pivot

指定对比度曲线的逐通道枢轴。默认居中于 0.5，以模拟经典 Photoshop 风格操作。

RGB Gain

使用 RGB Gain 为输入着色。此颜色会直接乘以输入颜色。1.0 为中性。

HSV

应用 Hue、Saturation、Value 颜色校正。Hue 是偏移量，Saturation 和 Value 是乘数。

Exposure

按给定的档数调整输入颜色的曝光。每个正档会使输入强度加倍；每个负档会使输入强度减半。通常更适合使用 Exposure 而不是直接乘法（例如 RGB Gain），因为它在感知上线性。

Output Range

Output Min

Output Min 会将最终颜色的 0.0 值重映射为你选择的值。

Output Max

Output Max 会将最终颜色的 1.0 值重映射为你选择的值。

Clamp Output

Clamp Output

最终颜色可以选择进行钳制，以确保它位于用户定义范围内。

Output Parameters

resultRGB

由加热到给定温度的黑体发出的颜色。

resultR

来自 resultRGB 输出的 R 通道。

resultG

来自 resultRGB 输出的 G 通道。

resultB

来自 resultRGB 输出的 B 通道。
