Pixar 的 RenderMan

RenderMan 27 是 Pixar 旗舰渲染器的最新版本，三十多年来广泛用于动画和视觉特效制作。

RenderMan 27 是十多年来最重要的一次版本发布，带来了突破性的性能、交互性和创作控制能力。凭借 XPU 的最终帧能力、视觉特效工作流、深度数据支持、扩展的风格化功能，以及改进的艺术家工作流，Pixar 正在继续重新定义高端长片动画和 VFX 的渲染技术。

RenderMan 27 - 主要功能

最终帧渲染 — XPU 架构现在支持最终帧渲染，使艺术家能够同时利用 CPU 和 GPU 资源，高效生成高质量图像。这一根本性转变提升了渲染的交互性和可扩展性，也让 XPU 成为未来 RenderMan 开发的核心。

交互式降噪 — 交互式降噪器显著缩短了渲染会话中的迭代时间。对于处理复杂灯光和着色设置的艺术家来说，这项功能尤其有用，因为它能在保持最终图像保真度的同时提供快速反馈。

检查点 — 引入检查点后，艺术家可以将部分帧输出为检查点。这项功能减少了重新渲染所需的时间，并提高了制作效率。

合成 – 在现有 LPE（光路表达式）支持的基础上，XPU 现在加入了视觉特效所需的更多合成功能，例如遮罩、遮挡保留以及扩展的 AOV 选项；Cryptomatte 将在 27.3 中提供。这显著改进了合成管线，让图像组装具备更精细的控制能力。

深度工作流 — XPU 现在支持深度数据工作流，可以对复杂体积元素和透明元素进行精确合成。OpenEXR Deep IDs 已完整实现，从而提供更顺畅的数据提取和更灵活的合成能力。

Stylized Looks — Stylized Looks™ 工具集不仅已在 XPU 中完整实现，而且得到了大幅扩展，可实现一系列非照片级真实效果，例如绘画风格、卡通着色和素描式渲染。这些能力为希望突破传统照片级真实美学的艺术家扩展了创作可能性。

OSL 显示滤镜和采样滤镜 — 将开放着色语言（OSL）的显示滤镜和采样滤镜集成到 XPU 中，提供了更强的着色自定义能力、更好的艺术控制，并提升了生产工作流的效率。

网格灯光 — XPU 现在完整支持网格灯光，能够使用物理灯光类型，并显著提高灯光效果在几何形态上的灵活性。

MaterialX Lama – MaterialX Lama 现已作为 XPU 中的 Early Access 功能提供，借助 Industrial Light & Magic 的着色工具集强化艺术家的工作流。它将在 27 版本周期内获得完整支持。

更多 lookdev 功能 – 嵌套实例化、OSL 改进（trace、getattribute）、内部体积聚合，以及玻璃和单次散射增强，让 XPU 与 RIS 的匹配更加接近，扩展了外观开发能力，并确保不同渲染架构之间的一致性。

多 GPU 支持 — 借助多 GPU 支持，XPU 现在可以在 CPU 之外同时利用多个 GPU，优化硬件利用率，并充分调动所有渲染资源。
