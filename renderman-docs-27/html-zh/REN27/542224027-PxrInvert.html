PxrInvert

Source: https://rmanwiki-27.pixar.com/space/REN27/542224027/PxrInvert
Downloaded: 2026-06-15T03:49:12+00:00

对传入颜色的一个或多个分量进行反相。输入颜色可以使用 RGB、HSL 或 HSV 颜色模型。注意：输出是一个 RGB 颜色，并被钳制到 [0 0 0] 到 [1 1 1] 范围。

Input Parameters（输入参数）

Input Color

要反相的颜色。

Color Model

inputColor 使用此颜色模型。

  • 0: RGB
  • 1: HSL
  • 2: HSV

 

Invert Channel 0

启用时（设为 1），第一个颜色通道会被反相。例如，如果 Color Model 设置为 RGB，则 R 通道会被反相。

Invert Channel 1

启用时（设为 1），第二个颜色通道会被反相。例如，如果 Color Model 设置为 RGB，则 G 通道会被反相。

Invert Channel 2

启用时（设为 1），第三个颜色通道会被反相。例如，如果 Color Model 设置为 RGB，则 B 通道会被反相。

Output Parameters（输出参数）

resultRGB

钳制后的颜色结果。

resultR

来自 resultRGB 输出的 R 通道。

resultG

来自 resultRGB 输出的 G 通道。

resultB

来自 resultRGB 输出的 B 通道。
