此节点会垂直堆叠两个材质，其中基底材质位于堆栈底部，顶层材质位于其上方。

Material Top

用作顶层的材质。   如果未设置顶层，则只使用基底材质本身。

Material Base

用作基底层的材质 

Top Mix

定义基底层与顶层之间的混合。   此值可通过滑块、黑白贴图、PxrPattern 或 float 值设置。

Top Thickness

此值定义顶层的厚度。  此值可通过滑块、黑白贴图、PxrPattern 或 float 值设置。

它只与顶层材质关联的内部效果有关，例如吸收。如果顶层材质本身是层节点，则此值会传递给它的基底组件。如果顶层材质是 mix 或 add，则此值会传递给两个子材质。

Layer Mode

切换分层模型。

  • Fresnel Blend 会使用顶层的 transmission 来简单衰减基底层。

  • Smooth Coating 在此模型基础上，还会让光线通过顶层发生折射。

  • Rough Coating 在 Smooth Coating 模型基础上，将基底层的粗糙度钳制到顶层的粗糙度。

  • Auto 会根据正在分层的材质选择模型。

在 Auto 模式中，当基底层为漫反射（diffuse）时使用菲涅耳混合；当基底层为镜面（specular）时使用粗糙涂层。

Layering Modes

Fresnel Blend

菲涅耳混合层模式会对两个材质执行加权相加，其中基底材质的权重由顶层材质的 transmission 决定。这确保了能量守恒。

这是最简单的层模式，在材质模型中非常常用。它与 MaterialX 层节点的行为匹配。

粗糙金色基底层和电介质顶层，IOR 从 1.0 到 3.0 变化

Rough Coating

菲涅耳混合模式的简单性意味着它会忽略若干物理效果。

从物理角度看，此层模式会在入射光线上忽略顶层，并忽略出射光线的折射。基底层也会忽略顶层的 IOR，这意味着除非手动设置，否则 Relative IOR 是不正确的。

粗糙涂层层模式通过近似它所忽略的物理效果来扩展这个简单模型。

粗糙金色基底层和电介质顶层，IOR 从 1.0 到 3.0 变化

Attenuation

LamaDielectric（absorptionColor 设置为 red）分层在白色 LamaDiffuse 之上，层厚度从 1 到 3.2 变化

顶层会同时作用于出射和入射光线，这意味着光线会被顶层衰减两次：离开时一次，退出时一次。这既适用于界面处的 Fresnel 衰减，也适用于穿过层厚度时的吸收。

Refraction

折射会通过让入射和出射光线都穿过顶层发生折射来计算，而不是像菲涅耳混合模式那样让它们直接穿过。这会改变基底层的外观，因为它接收到的是这些折射后的光线。作为近似，折射计算会认为顶层的粗糙度为 0。这意味着随着顶层粗糙度增加，模型精度会降低。

Scattering

为了补偿粗糙顶层，会将基底层的粗糙度钳制到顶层的粗糙度；因此如果顶层粗糙度更高，基底层粗糙度会提高到与之匹配。这近似了光线穿过粗糙顶层时的散射。

Relative IOR

同时还会跟踪顶层的 IOR，以确保基底层的 Relative IOR 正确。例如，如果皮肤层上方有一层水层，皮肤层的 Relative IOR 会降低，从而导致反射率降低。

Smooth Coating

平滑涂层层模式与粗糙涂层模式非常相似，但不进行粗糙度钳制。这允许更精细地控制基底层粗糙度。

Auto

Auto 模式在大多数情况下与粗糙涂层相同，但对于漫反射层会切换到菲涅耳混合。这是为了适应常见的 Diffuse < Specular < Clearcoat 材质配置，其中 diffuse 和 specular 材质被视为单一基底材质，而 Clearcoat 位于其上方。

带粗糙/凹凸紫色涂层的光滑金材质

这些图像展示了光滑金色基底层，其上覆盖带吸收的凹凸贴图电介质。

在菲涅耳混合图像中，凹凸贴图的效果被限制在顶层电介质上，因为到达基底层的光线只穿过顶层，而不会发生折射。在粗糙涂层图像中，可以看到由于光线被顶层折射，凹凸贴图对基底层的影响大得多。

吸收的工作方式也不同，在粗糙涂层情况下，表面上的颜色变化更少。这是因为折射会把光线在顶层与基底层之间行进的距离压缩到更小范围内。

还可以看到，在粗糙涂层图像中，基底层反射扩散得多，这是由于光线穿过顶层时发生散射，而该效果由粗糙度近似实现。
