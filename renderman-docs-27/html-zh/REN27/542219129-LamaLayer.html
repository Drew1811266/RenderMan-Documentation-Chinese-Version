此节点会垂直堆叠 2 个材质，其中 base 位于堆栈底部，Top layer 位于其上方。

Material Top

用作 top layer 的材质。   如果未设置 top layer，则只使用 base material 本身。

Material Base

用作 base layer 的材质 

Top Mix

定义 base layer 与 top layer 之间的混合。   此值可通过滑块、黑白贴图、PxrPattern 或 float 值设置。

Top Thickness

此值定义 top layer 的厚度。  此值可通过滑块、黑白贴图、PxrPattern 或 float 值设置。

它只与 top material 关联的内部效果有关，例如 absorption。如果 top material 本身是 layer node，则此值会传递给它的 base component。如果 top material 是 mix 或 add，则此值会传递给两个 child materials。

Layer Mode

切换分层模型。

  • Fresnel Blend 会使用 top layer 的 transmission 来简单衰减 base layer。

  • Smooth Coating 在此模型基础上，还会让光线通过 top layer 发生折射。

  • Rough Coating 在 Smooth Coating 模型基础上，将 base layer 的 roughness 钳制到 top layer 的 roughness。

  • Auto 会根据正在分层的材质选择模型。

在 Auto mode 中，当 base layer 为 diffuse 时使用 Fresnel Blend；当 base layer 为 specular 时使用 Rough Coating。

Layering Modes

Fresnel Blend

Fresnel Blend layer mode 会对两个材质执行加权相加，其中 base material 的权重由 top material 的 transmission 决定。这确保了能量守恒。

这是最简单的 layer mode，在材质模型中非常常用。它与 MaterialX layer node 的行为匹配。

粗糙金色 base layer 和电介质 top layer，IOR 从 1.0 到 3.0 变化

Rough Coating

Fresnel Blend mode 的简单性意味着它会忽略若干物理效果。

从物理角度看，此 layer mode 会在入射光线上忽略 top layer，并忽略出射光线的折射。base layer 也会忽略 top layer 的 IOR，这意味着除非手动设置，否则 Relative IOR 是不正确的。

Rough Coating layer mode 通过近似它所忽略的物理效果来扩展这个简单模型。

粗糙金色 base layer 和电介质 top layer，IOR 从 1.0 到 3.0 变化

Attenuation

LamaDielectric（absorptionColor 设置为 red）分层在白色 LamaDiffuse 之上，layer thickness 从 1 到 3.2 变化

top layer 会同时作用于出射和入射光线，这意味着光线会被 top layer 衰减两次：离开时一次，退出时一次。这既适用于界面处的 Fresnel 衰减，也适用于穿过 layer 厚度时的 absorption。

Refraction

Refraction 会通过让入射和出射光线都穿过 top layer 发生折射来计算，而不是像 Fresnel Blend mode 那样让它们直接穿过。这会改变 base layer 的外观，因为它接收到的是这些折射后的光线。作为近似，refraction 计算会认为 top layer 的 roughness 为 0。这意味着随着 top layer roughness 增加，模型精度会降低。

Scattering

为了补偿粗糙 top layer，会将 base layer 的 roughness 钳制到 top layer 的 roughness；因此如果 top layer roughness 更高，base layer roughness 会提高到与之匹配。这近似了光线穿过粗糙 top layer 时的散射。

Relative IOR

同时还会跟踪 top layer 的 IOR，以确保 base layer 的 Relative IOR 正确。例如，如果 skin layer 上方有一层 water layer，skin layer 的 Relative IOR 会降低，从而导致反射率降低。

Smooth Coating

Smooth Coating layer mode 与 Rough Coating mode 非常相似，但不进行 roughness clamping。这允许更精细地控制 base layer roughness。

Auto

Auto mode 在大多数情况下与 Rough Coating 相同，但对于 diffuse layers 会切换到 Fresnel Blend。这是为了适应常见的 Diffuse < Specular < Clearcoat 材质配置，其中 diffuse 和 specular materials 被视为单一 base material，而 Clearcoat 位于其上方。

这些图像展示了光滑金色 base layer，其上覆盖带 absorption 的 bump mapped 电介质。

在 Fresnel Blend 图像中，bump map 的效果被限制在 top dielectric layer 上，因为到达 base layer 的光线只穿过 top layer，而不会发生折射。在 Rough Coating 图像中，可以看到由于光线被 top layer 折射，bump map 对 base layer 的影响大得多。

absorption 的工作方式也不同，在 Rough Coating 情况下，表面上的颜色变化更少。这是因为 refraction 会把光线在 top layer 与 base layer 之间行进的距离压缩到更小范围内。

还可以看到，在 Rough Coating 图像中，base layer reflection 扩散得多，这是由于光线穿过 top layer 时发生 scattering，而该效果由 roughness approximation 实现。
