毛发梳理与渲染：Paul Sheehy


此图案节点让艺术家能够创建易于控制的写实或艺术指导风格毛发和羽毛。只需少量控制项，就可以实现从人类赤褐色头发到幻想外星羽毛的效果。

通过对毛发纤维中的黑色素浓度建模，可以计算写实的人类毛发颜色。黑色素是一种深褐到黑色的色素，存在于人和动物的毛发、皮肤以及眼睛虹膜中。

这是基于 "An Energy-Conserving Hair Reflectance Model"（EGSR 2011，Eugene d’Eon、Guillaume Francois、Martin Hill、Joe Letteri、Jean-Marie Aubry）。

用法

PxrHairColor 输出 3 个颜色，用于配合 PxrMarschnerHair BxDF 提供物理上可信的结果。应按如下方式连接：

为了获得写实结果，PxrMarschnerHair 的 Primary Specular Color 应始终设为 白色 ，同时使用 Physical 控制项。

使用 Artistic Mode 控制项时，建议在 PxrMarschnerHair 材质上设置以下参数：

diffuseGain 设为 1.0（Zinke 模型）

specularGainR、specularGainTT、specularGainTRT、specularMixFresnel 和 specularEnergyCompensation 设为 1.0

我们始终建议 至少 为漂亮的毛发设置 2 次间接反弹。设置位置为所选 积分器。

Parameters（参数）

Mode

Physical Mode: 这是默认控制集，会基于自然毛发颜色和响应提供写实结果。结合 Dye 参数，用户可以创建任何可信的效果。这里期望颜色输入位于 sRGB 色彩空间。

Artistic Mode: 此模式用 Color 选项替代 Melanin 控制项。虽然这可能产生不写实的结果，但能提供更大的艺术指导自由度。

Melanin Randomize

如果定义了 hairIndexPrimvar，则随机化黑色素浓度。

Artistic Mode

Color

可以选择 RGB 颜色，或使用另一个图案节点为毛发提供任意颜色。渲染时会尽可能准确地复现所选颜色。


Color Randomize

使用 RGB 颜色或所选图案节点，可以提供随机颜色，与上方选择的颜色混合。



Darken Roots

通过逐步增加黑色素，让毛发纤维根部逐渐变暗。该值对应根部额外增加的黑色素量。可使用 Darken Slope 和 Darken End Position， 下面我们通过 Darken End Position 设为 1.0 来让整根毛发变暗。

Darken Slope

调整变暗的斜率。这会控制从深色到浅色的过渡。

Darken End Position

大多数情况下，变暗应发生在接近根部的位置，并保持其余毛发不变。结束位置默认值为 0.5，因此只有前半段毛发会变暗。正确范围是 0. 到 1.0

Stray Hair Color

Stray Density

如果定义了 hairIndexPrimvar，则此项控制离散毛发出现的频率。白色离散毛发通常会在人开始变老或承受非常高压力时出现。

Physical Mode

Stray Melanin

离散毛发的黑色素浓度。它的工作方式类似主 Melanin 参数，可用于控制结果颜色。

Stray Random Melanin

如果定义了 hairIndexPrimvar，则随机化离散毛发的黑色素级别。


Artistic Mode

Stray Color

可为离散毛发颜色选择任意 RGB 颜色或图案节点。下面 Color 为白色，离散毛发使用纯色或渐变着色。Stray Density 设置为 0.5


Stray Random Color

可选择任意 RGB 颜色或图案节点，用于随机化并组合离散毛发颜色。下面颜色设置为黑色/无、深红色、红色。Hair Color 选择为蓝色。Stray Density 设置为 0.5


Dye Color

Dye

毛发可以用颜色染色。该颜色会以可信方式与毛发颜色结合。第一个示例中，毛发非常深，因为红发上的深蓝色染剂会产生这种颜色。更纯的饱和颜色，或在黑色素设置为 0.0 的毛发上选择的颜色，会得到更鲜艳的颜色，例如红色示例。

Dye Random Hue

如果定义了 hairIndexPrimvar，则随机化染剂色相。下面使用无黑色素的毛发和绿色染剂，然后逐步增加随机化。注意，此结果可以是任何类型的随机颜色，而随机黑色素控制项只会产生某种形式的可信毛发颜色。

Dye Random Saturation

如果定义了 hairIndexPrimvar，则随机化染剂饱和度。较低的饱和度会显露设置好的毛发黑色素颜色。你可能会发现需要高于 1.0 的值，才能让更多毛发降低饱和度；也可以将 Random Pivot 参数提高到 0.5 以上，当此 Random 参数设得很高时，1.0 可能会完全消除 Dye Color。
