PxrDispTransform

Source: https://rmanwiki-27.pixar.com/space/REN27/542223359/PxrDispTransform
Downloaded: 2026-06-15T03:49:12+00:00

此 pattern 会将输入置换值变换到自定义数值范围，用作 PxrDisplace 的输入。这包括重映射置换中心，并将其变换到不同的置换尺度和空间。

需要将此 pattern 的结果连接到 PxrDisplace 的 Scalar 或 Vector Displacement 输入，才能置换几何体。

 

Input Parameters（输入参数）

Displacement Type

指定输入的 Displacement Type（置换类型）。

  • 1: Scalar.  
    这是浮点类型。  
  • 2: Generic Vector.  
    这可以是任意程序化创建或贴图生成的向量，不需要像 Mudbox 或 ZBrush 矢量置换贴图那样进行特殊解码。
  • 3: Mudbox Vector.  
    这是从 Mudbox 提取的矢量置换贴图。 
  • 4: ZBrush Vector. 
    这是从 ZBrush 提取的矢量置换贴图。 

Scalar Displacement

Scalar displacement（标量置换）输入。这是 float displacement（浮点置换）。连接一个 PxrTexture 或任意 float pattern 节点到此参数。

仅当 Displacement Type 设置为 Scalar 时有效。

Vector Displacement

Vector displacement 输入。Vector displacement data 是三通道 XYZ 数据。将 PxrTexture 或任意 RIS vector pattern 节点的 resultXYZ 或 resultRGB 连接到此参数。 

仅当 Displacement Type 为 Vector 时有效。

Vector Space

指定向量创建时所在的空间。对于 Mudbox 和 ZBrush vector type，这会是导出矢量置换时使用的空间。  

  • 1: World
  • 2: Object
  • 3: Tangent
  • 4: Current

设置正确的 Vector Space 非常重要，否则生成的矢量置换会渲染错误。 

大多数情况下，如果出现意外结果，原因都是 Vector Space 设置不正确。 

对于 Mudbox，矢量置换需要从 Mudbox 以 Absolute Tangent 导出，而不是 Relative Tangent。

Displacement Height

Displacement Height 作用于向外置换的置换值。

Displacement Depth

Displacement Depth 作用于向内置换的置换值。

Remapping Mode

指定如何重映射置换值：

  • 1: None.  
    按原样读取置换值，不进行任何重映射。使用 Displacement Height 缩放置换值。
  •  2: Centered: 
    置换值会以指定的 Displacement Center 为中心。如果该值等于中心，则不会产生置换。  
    低于中心的任何值都会向内置换，高于中心的任何值都会向外置换。

  • 3: Interpolate Depth and Height:

    通过 catmull-rom spline（spline("catmullrom", -depth, -depth, 0, height, height)）拟合指定的 Displacement Depth 和 Displacement Height。此模式不使用 Displacement Center。

Displacement Center

用于 "Centered" Remapping Mode 的 Displacement Center。

Displacement Scale Space

用于测量置换单位的空间。例如 "world"、"object" 或用户定义的坐标系统。

我们提供两个用于指定置换的渲染空间：world 和 object space。

默认情况下，RenderMan 会在 Object space 中置换。这通常适合雕刻模型。但对某些对象来说，这可能导致不连续。例如，一组一起置换的对象，像由一组平面组成的环境部件。在 object space 中，这些对象可能会像下方示例一样，在面片之间产生缝隙或不连续。

要修正这个特定场景，可以选择 World Space，用相同的八个面片获得下方结果。这会解决海面平面、悬崖和岩石等问题，在这些情况下置换会跨多个形状共享。

Use Displacement Direction

仅当 Displacement Type 设置为 Scalar 时有效。

开启时，置换会沿下方提供的置换方向进行，而不是沿表面法线。

Displacement Direction

仅当 Displacement Type 设置为 Scalar 时有效。

置换会沿此项所指向的方向进行，而不是沿表面法线。请注意，置换方向向量在使用前会被归一化。

Displacement Direction Space

置换方向向量所在的空间。例如 "world"、"object" 或用户定义的坐标系统。

Output Parameters

resultXYZ

矢量结果。
