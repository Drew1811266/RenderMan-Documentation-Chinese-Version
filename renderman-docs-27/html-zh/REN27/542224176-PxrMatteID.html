PxrMatteID

Source: https://rmanwiki-27.pixar.com/space/REN27/542224176/PxrMatteID
Downloaded: 2026-06-15T03:49:12+00:00

三个 MatteID AOV 设置，用于反映材质分配

此节点输出用于合成的 AOV。应将它连接到 Utility Pattern 参数，所属节点为 PxrSurface ；它会输出 MatteID AOV。

它最多输出八个 RGB 遮罩（24 个遮罩），命名为 MatteID0, MatteID1 ... MatteID7。

MatteID AOV 可用作逐对象、逐对象组、逐材质的合成遮罩。它们会正确带有运动模糊或失焦效果，让你能在合成软件中隔离对象。

要将对象或着色器关联到 MatteIDx，需要向该对象或着色器添加一个特殊属性：

Attribute "user" "color MatteID0" [1 0 0]


在此示例中 MatteID0 AOV 会让你的对象显示为红色，但你可以使用绿色和蓝色通道处理另外两组对象。

在 RenderMan for Maya (RfM) 中，可以通过 transform 上的 User Attributes 菜单添加这些属性；同时 MatteID 选项卡位于 Outputs 部分（RenderMan globals）。

Input Parameters

Input AOV

此输入只能用于串接其他不产生任何输出的节点。

Enable

控制 MatteID AOV 输出。

Matte Texture(s)

这些参数允许为每个遮罩指定要输出的颜色或图案。该结果会乘以上述 user attribute 选择的颜色。如果将 user attribute 设为白色，此参数纹理会按原样渲染。

使用方法
  1. 将 MatteID 节点连接到 PxrSurface 的 utilityPattern[0] 插槽。

Output Parameters（输出参数）

resultAOV

一个整数。此输出仅用于触发图中该图案节点的执行。
