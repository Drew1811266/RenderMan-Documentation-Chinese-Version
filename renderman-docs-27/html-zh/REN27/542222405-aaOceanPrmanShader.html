aaOceanPrmanShader

Source: https://rmanwiki-27.pixar.com/space/REN27/542222405/aaOceanPrmanShader
Downloaded: 2026-06-15T03:49:12+00:00


一个 Tessendorf 海洋生成器。代码由 Amaan Akram 贡献（www.amaanakram.com）。

它会生成一个带凹凸扰动的法线，可输入到置换着色器中，用来创建类似海洋的波浪。

输入参数

Resolution（分辨率）

指定生成的纹理分辨率。有效设置为 3 到 7。每个值会生成以下分辨率：

  • 值为 3 时生成 128x128 的分辨率

  • 4：256x256

  • 5：512x512

  • 6：1024x1024

  • 7：2048x2048。

Ocean Scale（海洋尺度）

海面区块的尺寸，单位为米。较大的值会把更大范围的海面“装入”被渲染的几何体中。

Seed（种子）

随机数生成器的种子。不同值会生成不同的海面区块。

Current Time（当前时间）

当前场景时间，单位为秒。使用表达式驱动此参数。

Repeat Time（重复时间）

经过指定秒数后，波浪会循环（重复其形状）。

Fade（淡化）

淡化（缩放）海洋矢量置换强度。

Chop Amount（浪尖锐化量）

此参数控制波峰的尖锐程度。也会影响泡沫值。

Velocity（速度）

控制波浪的尺寸。较高的速度值会产生数量更少但更大的波浪；较低的速度值会产生平静海面。

Wave Speed（波速）

波浪的速度乘数。

Cutoff（截断）

定义整体海面的平滑因子。会截断（移除）波长小于指定值的波浪。

Wave Height（波高）

波浪的高度乘数。

Wind Direction（风向）

波浪传播的方向。

Damp（阻尼）

让波浪偏向沿风向传播。值为 0 时，波浪会朝风向相反方向传播；1.0 时，波浪只沿风向传播。

Wind Align（风向对齐）

将波浪对齐为垂直于风向传播，类似浅水中观察到的情况。

Invert Foam（反转泡沫）

让波峰显示为白色，而不是让波谷显示为白色。

Gamma（伽马）

应用到特征值输出的伽马因子。

Brightness（亮度）

特征值的乘数。

Normalize（归一化）

是否对特征值进行归一化。请参见 fMin 和 fMax。

fMin

fMin 和 fMax 用于归一化特征值；这些值可能进入负值和正值范围。请查看渲染日志，了解某一帧上的典型值。

fMax

fMin 和 fMax 用于归一化特征值；这些值可能进入负值和正值范围。请查看渲染日志，了解某一帧上的典型值。

Write File（写入文件）

写出对象空间中的全浮点 OpenEXR 矢量置换贴图序列。

Output Folder（输出文件夹）

输出图像所在文件夹的路径。

Postfix（后缀）

提供要应用到生成文件上的任意后缀，用于唯一标识该文件。

Current Frame（当前帧）

为当前帧设置表达式，用于带帧号填充的序列名称。

Ocean Depth（海洋深度）

随着深度减小而减慢波浪速度。

Surface Tension（表面张力）

在海面顶部传播的毛细波。仅用于小尺度效果。详情请参见原始 Tessendorf 论文。

Invert T（反转 T）

反转流形的 t 参数。

Manifold

提供应用置换的域。默认为 s,t。

输出参数

outputDisplacementRGB

带凹凸扰动的法线。

outEigenvalueFloat

特征值。

用法示例

为了快速得到可信的海洋渲染，可以使用 PxrSurface（使用 Glass 参数）作为 BxDF。唯一需要调整的是 Refractive Index （eta），将其改为 1.33，以匹配水的数值。

对于置换，可以使用 PxrDisplace 着色器。连接 aaOceanPrmanShader 的 outputDisplacementRGB 输出。

下面是使用 aaOceanPrmanShader 默认设置的示例渲染。海洋由环境光照亮，该环境光使用 Luxo-Jr_4000x2000.tex 这张 RenderMan 随附的纹理。

平静的海洋。

根据对象大小，可能需要提高 Resolution（分辨率） 参数。在下一张渲染中，由于平面尺寸较大， Resolution（分辨率） 已提高到 7。此外， Chop Amount（浪尖锐化量） 提高到 10.0。

海面开始变得颠簸。

要添加一些简单的白色泡沫，可以向我们的 PxrSurface 添加漫反射，从而为海洋添加一个漫反射层。

创建一个 PxrLayerSurface 着色器，并将 outputDisplacementRGB 输出连接到（漫反射） Bump 参数，并将（漫反射） Color 设为更接近白色。确保镜面反射和清漆涂层瓣已禁用。

对于层遮罩，可以使用 outEigenvalueFloat 输出，该输出来自 aaOceanPrmanShader 图案节点。其值必须落在 0-1 范围内。我们可以使用一个 PxrRemap 节点轻松强制实现这一点。将 PxrRemap 连接到层遮罩参数。由于 PxrRemap 只处理颜色，可以使用一个 PxrToFloat3 来把 outEigenvalueFloat 转换为三元组。将 Input Min 和 Input Max 分别设为 -5 和 5（在 PxrRemap 节点上）。

最后，在 aaOceanPrmanShader 节点中开启 invertFoam 这一 aaOceanPrmanShader 参数，使白色出现在波峰处。随后你应该会得到类似下图的渲染结果。
