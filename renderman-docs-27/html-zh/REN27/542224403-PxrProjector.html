PxrProjector

Source: https://rmanwiki-27.pixar.com/space/REN27/542224403/PxrProjector
Downloaded: 2026-06-15T03:49:12+00:00

此节点用于与 PxrProjectionLayer 配合使用，但也可以作为通用 投影流形 使用。此流形支持相机、球面和圆柱映射。它为 pattern 生成器 封装 2D 参数化，并允许进行 变换 以及选择绑定到 基元 的任意 变量。


输入参数

Projection

选择投影：
  • 0: Camera
  • 1: Spherical
  • 2: Cylindrical
  • 3: Planar


Coordinate System

坐标系变换的名称（例如 Maya 中的 place3dTexture 节点或相机）。


你 必须 设置坐标系。此字段不能留空。


Use

选择要投射到哪些位置上。
  • 0: P - N  P 是当前位置，N 是当前法线
  • 1: Pref - Nref  如果对象正在变形，并且你希望投影纹理能够“贴附”，请使用此选项。它依赖两个 primvars： __Pref 和 __Nref ，它们应存在于 变形几何体 上。在 Maya 中，使用 RenderMan > Primvars > Freeze 菜单将它们添加到所选几何体。
  • 2: WPref - WNref  如果对象正在变形并且经过变换，此选项可让投影纹理仍然贴附。它依赖 __WPref 和 __WNref primvars。在 Maya 中，使用 RenderMan > Primvars > Freeze 菜单将它们添加到所选几何体。


Occlusion frontOnly
将投影限制到朝向投影方向的点。

此设置会更新 resultMask 输出，因此你可以用它在投影与底层表面材质之间进行混合。


Front Falloff
使用 frontOnly 时引入平滑过渡。适合在不同投影之间混合过渡。默认值 0.0 表示没有衰减。

此设置会更新 resultMask 输出，因此你可以用它在投影与底层表面材质之间进行混合。


Trace Occlusion
发射光线，以避免从投影视角投射到被几何体遮挡的区域。

此设置会更新 resultMask 输出，因此你可以用它在投影与底层表面材质之间进行混合。


Trace Max Distance
当 traceOcclusion 开启时，可以限制光线长度。此数值以场景单位表示，无论你的创作软件包中使用什么单位。

此设置会更新 resultMask 输出，因此你可以用它在投影与底层表面材质之间进行混合。


Trace Set
trace set 的名称，用于将光线命中限制到特定对象组。默认会考虑场景中的所有对象。


Camera


Horizontal Resolution
水平图像分辨率。它用于指定图像在 txmake-ing 之前的原始分辨率。


Vertical resolution
垂直图像分辨率。它用于指定图像在 txmake-ing 之前的原始分辨率。

Horizontal Overscan

以 pixels 表示的 Horizontal overscan

Vertical Overscan

以 pixels 表示的 Vertical Overscan

Camera Type

Is Orthographic


Focal Length
相机焦距以毫米 (mm) 为单位。要将视野 (fov) 转换为焦距，请使用此公式


fov = atan((0.5 * aperture) / (focalLength * 0.0393701))


Horizontal Aperture

相机胶片背板的宽度，以英寸为单位。


Vertical Aperture

相机胶片背板的高度，以英寸为单位。


Near Clip Plane

比此距离更靠近坐标系的位置会被遮罩。此数值以场景单位表示，无论你的创作软件包中使用什么单位。
此设置会更新 resultMask 输出，因此你可以用它在投影与底层表面材质之间进行混合。

Far Clip Plane

比此距离更远离坐标系的位置会被遮罩。此数值以场景单位表示，无论你的创作软件包中使用什么单位。
此设置会更新 resultMask 输出，因此你可以用它在投影与底层表面材质之间进行混合。

Film Fit

当图像与胶片背板具有不同宽高比时，图像适配到胶片背板的方式。它模拟 Maya 的相机设置。
  • 0: Fill
  • 1: Horizontal
  • 2: Vertical
  • 3: overscan
此设置会更新 resultMask 输出，因此你可以用它在投影与底层表面材质之间进行混合。


Clamp To

激活遮罩输出。你可以遮罩不同区域，但这主要对相机投影有用。
  • 0: Off  投影纹理会出现在所有位置
  • 1: Frustum  投影纹理只出现在相机视锥内部
  • 2: Texture  纹理会完整投射，不受相机视锥限制。
此设置会更新 resultMask 输出，因此你可以用它在投影与底层表面材质之间进行混合。

Planar

Clamp To

钳制平面投影，假设坐标系表示为一个立方体。

  • 0: Off  投影纹理会出现在所有位置。
  • 1: XY  投影纹理会出现在坐标系的 XY 平面上。
  • 2: XZ  投影纹理会出现在坐标系的 XZ 平面上。
  • 3: YZ  投影纹理会出现在坐标系的 YZ 平面上。
  • 4: XYZ  投影纹理会出现在坐标系的体积内部。


CoordSys Type

有 2 种不同类型，用于匹配不同的 视口表示。

  • 0: Generic  坐标系是宽 1 个单位的立方体。
  • 1: Maya place3dTexture  place3dTexture 宽 2 个单位。

2D 参数

Angle

围绕原点的旋转角度。


Scale S

S 方向上特征的频率。


Scale T

T 方向上特征的频率。


Offset S

S 方向上相对于原点的偏移。


Offset T

T 方向上相对于原点的偏移。


Invert T

在 T 方向上翻转流形。


Advanced

Verbose

输出各种信息，帮助你调试场景。


Front Only Tolerance

此参数有时可用于避免投射到接近掠射角的表面 上。

输出参数


result

2D 流形。

resultS

流形 S 分量的 float 表示。

resultT

流形 T 分量的 float 表示。

resultMask

定义 受限投影区域的二值遮罩。
应将其接入 PxrProjectionLayer 的 遮罩输入，或用于在 投影与底层纹理 之间进行混合。
