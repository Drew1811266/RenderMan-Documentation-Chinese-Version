PxrProjector

Source: https://rmanwiki-27.pixar.com/space/REN27/542224403/PxrProjector
Downloaded: 2026-06-15T03:49:12+00:00

此节点用于与 PxrProjectionLayer 配合使用，但也可以作为通用 projection manifold 使用。此 manifold 支持 camera、spherical 和 cylindrical mapping。它为 pattern generators 封装 2D parameterization，并允许进行 transformations 以及选择绑定到 primitives 的任意 variables。


Input Parameters

Projection

选择 projection：
  • 0: Camera
  • 1: Spherical
  • 2: Cylindrical
  • 3: Planar


Coordinate System

coordinate system transform 的名称（例如 Maya 中的 place3dTexture 节点或 camera）。


你 必须 设置 coordinate system。此字段不能留空。


Use

选择要投射到哪些 positions 上。
  • 0: P - N  P 是 current position，N 是 current normal
  • 1: Pref - Nref  如果 object 正在变形，并且你希望 projected texture 能够 'stick'，请使用此选项。它依赖两个 primvars： __Pref 和 __Nref ，它们应存在于 deforming geometry 上。在 Maya 中，使用 RenderMan > Primvars > Freeze 菜单将它们添加到所选 geometry。
  • 2: WPref - WNref  如果 object 正在变形并且经过 transform，此选项可让 projected texture 仍然 stick。它依赖 __WPref 和 __WNref primvars。在 Maya 中，使用 RenderMan > Primvars > Freeze 菜单将它们添加到所选 geometry。


Occlusion frontOnly
将 projection 限制到朝向 projection direction 的点。

此设置会更新 resultMask output，因此你可以用它在 projection 与 underlying surface material 之间进行混合。


Front Falloff
使用 frontOnly 时引入平滑过渡。适合在不同 projections 之间混合过渡。默认值 0.0 表示没有 falloff。

此设置会更新 resultMask output，因此你可以用它在 projection 与 underlying surface material 之间进行混合。


Trace Occlusion
发射 rays，以避免从 projection 的视角投射到被 geometry 遮挡的区域。

此设置会更新 resultMask output，因此你可以用它在 projection 与 underlying surface material 之间进行混合。


Trace Max Distance
当 traceOcclusion 开启时，可以限制 ray length。此数值以 scene units 表示，无论你的 authoring package 中使用什么单位。

此设置会更新 resultMask output，因此你可以用它在 projection 与 underlying surface material 之间进行混合。


Trace Set
trace set 的名称，用于将 ray hits 限制到特定对象组。默认会考虑场景中的所有对象。


Camera


Horizontal Resolution
水平图像分辨率。它用于指定图像在 txmake-ing 之前的原始分辨率。


Vertical resolution
垂直图像分辨率。它用于指定图像在 txmake-ing 之前的原始分辨率。

Horizontal Overscan

以 pixels 表示的 Horizontal overscan

Vertical Overscan

以 pixels 表示的 Vertical Overscan

Camera Type

Is Orthographic


Focal Length
camera 的 focal length 以 millimeters (mm) 为单位。要将 Field of View (fov) 转换为 Focal Length，请使用此公式


fov = atan((0.5 * aperture) / (focalLength * 0.0393701))


Horizontal Aperture

camera film back 的 width（宽度），以 inches 为单位。


Vertical Aperture

camera film back 的高度，以 inches 为单位。


Near Clip Plane

比此距离更靠近 coordinate system 的 positions 会被 masked。此数值以 scene units 表示，无论你的 authoring package 中使用什么单位。
此设置会更新 resultMask output，因此你可以用它在 projection 与 underlying surface material 之间进行混合。

Far Clip Plane

比此距离更远离 coordinate system 的 positions 会被 masked。此数值以 scene units 表示，无论你的 authoring package 中使用什么单位。
此设置会更新 resultMask output，因此你可以用它在 projection 与 underlying surface material 之间进行混合。

Film Fit

当 image 与 film back 具有不同 aspect ratios 时，image 适配到 film back 的方式。它模拟 Maya 的 camera settings。
  • 0: Fill
  • 1: Horizontal
  • 2: Vertical
  • 3: overscan
此设置会更新 resultMask output，因此你可以用它在 projection 与 underlying surface material 之间进行混合。


Clamp To

激活 mask output。你可以 mask 不同区域，但这主要对 camera projection 有用。
  • 0: Off  projected texture 会出现在所有位置
  • 1: Frustum  projected texture 只出现在 camera frustum 内部
  • 2: Texture  texture 会完整投射，不受 camera frustum 限制。
此设置会更新 resultMask output，因此你可以用它在 projection 与 underlying surface material 之间进行混合。

Planar

Clamp To

Clamp planar projection，假设 coordinate system 表示为一个 cube。

  • 0: Off  projected texture 会出现在所有位置。
  • 1: XY  projected texture 会出现在 coordinate system 的 XY plane 上。
  • 2: XZ  projected texture 会出现在 coordinate system 的 XZ plane 上。
  • 3: YZ  projected texture 会出现在 coordinate system 的 YZ plane 上。
  • 4: XYZ  projected texture 会出现在 coordinate system 的 volume 内部。


CoordSys Type

有 2 种不同类型，用于匹配不同的 viewport representations。

  • 0: Generic  coordinate system 是宽 1 unit 的 cube。
  • 1: Maya place3dTexture  place3dTexture 宽 2 units。

2D Parameters

Angle

围绕 origin 的 Rotation angle。


Scale S

S direction 上 feature 的 Frequency。


Scale T

T direction 上 feature 的 Frequency。


Offset S

S direction 上相对于 origin 的 Offset。


Offset T

T direction 上相对于 origin 的 Offset。


Invert T

在 T direction 上翻转 manifold。


Advanced

Verbose

Outputs（输出）各种信息，帮助你 debug scenes。


Front Only Tolerance

此参数有时可用于避免投射到接近 grazing angle 的 surfaces 上。

Output Parameters


result

2D manifold。

resultS

manifold 的 S component 的 float representation。

resultT

manifold 的 T component 的 float representation。

resultMask

定义 restricted projection area 的 binary mask。
应将其接入 PxrProjectionLayer 的 mask input，或用于在 projection 与 underlying texture 之间进行混合。
