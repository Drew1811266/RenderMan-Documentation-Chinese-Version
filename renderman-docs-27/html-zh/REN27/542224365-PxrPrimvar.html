PxrPrimvar

Source: https://rmanwiki-27.pixar.com/space/REN27/542224365/PxrPrimvar
Downloaded: 2026-06-15T03:49:12+00:00

允许 GPrim 图元变量（primvars）通过 Pattern 图传递。

Input Parameters

Variable

定义变量的样式

Variable Name

图元变量的名称。

Variable Type

图元变量的类型：

  • float
  • float2
  • color
  • point
  • vector
  • normal

Default Float

如果检索类型为 'float' 的变量时出错，将使用此值。

Verbosity

用于场景调试的诊断输出级别

Output Parameters

result

以颜色形式输出的结果。
