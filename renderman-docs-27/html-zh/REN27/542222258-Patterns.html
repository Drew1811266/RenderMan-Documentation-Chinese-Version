Patterns

Source: https://rmanwiki-27.pixar.com/space/REN27/542222258/Patterns
Downloaded: 2026-06-15T03:49:12+00:00


Materials 控制对象的外观，而 pattern 节点通过在表面上改变材质 Bxdf 的参数来控制细节。这类似于纹理艺术家使用纹理来控制材质外观的方式。RenderMan 随附一组 pattern 节点，可用于纹理化、颜色校正、凹凸处理以及各种实用着色系统。

这些 pattern 节点可分为以下类别：

Image Texture（图像纹理）

使用纹理图像驱动 pattern。

  • PxrMultiTexture
  • PxrProjectionLayer
  • PxrPtexture
  • PxrTexture


Procedural Texture（程序化纹理）

以程序方式创建 noise（噪声）和 pattern。

  • PxrChecker
  • PxrCurvature
  • PxrFractal
  • PxrVoronoise
  • PxrWorley
  • PxrPhasorNoise


Manifolds（映射流形）

Manifolds 会告诉渲染器如何在表面上应用 pattern。这会控制重复、放置和尺寸等特征。

  • PxrManifold2D
  • PxrBumpManifold2D
  • PxrManifold3D
  • PxrProjector
  • PxrRandomTextureManifold
  • PxrRoundCube
  • PxrTileManifold
  • PxrHexTileManifold


Color（颜色）

在 shading network 中创建或修改颜色。

  • PxrBlackBody
  • PxrBlend
  • PxrClamp
  • PxrColorCorrect
  • PxrColorSpace
  • PxrExposure
  • PxrGamma
  • PxrHairColor
  • PxrHSL
  • PxrInvert
  • PxrLayeredBlend
  • PxrMix
  • PxrProjectionStack
  • PxrRamp
  • PxrRemap
  • PxrThinFilm
  • PxrThreshold
  • PxrVary


Bump（凹凸）

创建低幅度的浮雕效果。

  • PxrBump
  • PxrBumpMixer
  • PxrNormalMap
  • PxrFlakes
  • aaOceanPrmanShader


Geometry Nodes（几何节点）

可以使用这些节点计算几何属性和向量。

  • PxrDirt
  • PxrDispScalarLayer
  • PxrDispTransform
  • PxrDispVectorLayer
  • PxrDot
  • PxrCross
  • PxrFacingRatio
  • PxrTangentField


Script（脚本）

  • OSL Patterns


Utility（实用工具）

  • PxrAttribute
  • PxrMatteID
  • PxrPrimvar
  • PxrVariable
  • PxrToFloat
  • PxrToFloat3
  • PxrTee
  • PxrShadedSide


Baking（烘焙）

这些直通 pattern 旨在允许用户烘焙任意 pattern 网络。

  • PxrBakeTexture
  • PxrBakePointCloud


Guidance for Writing Your Own Patterns（编写自定义 Patterns 指南）

你不必局限于 RenderMan 提供的 pattern。如果遵循 OSL Patterns 中的指导，就可以构建自己的 pattern。XPU 只运行 OSL patterns，而 RIS 同时运行 C++ 和 OSL patterns。
