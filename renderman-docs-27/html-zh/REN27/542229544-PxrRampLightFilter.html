PxrRampLightFilter

Source: https://rmanwiki-27.pixar.com/space/REN27/542229544/PxrRampLightFilter
Downloaded: 2026-06-15T03:49:12+00:00

PxrRampLightFilter 使用 ramp 来控制灯光。它也可用于以人工和艺术化方式控制灯光衰减。

Parameters（参数）

Ramp Direction

Ramp Type

指定 ramp 应用的方向。

  • distToLight(0) - 到灯光的距离。

  • linear(1) - 线性 ramp。

  • spherical(2) - 球形 ramp。

  • radial(3) - 径向 ramp。

Begin Distance

ramp 开始的位置距离。

End Distance

ramp 结束的位置距离。

Ramp

Ramp

定义 knot 数量。

Ramp Knots

knot 值数组。

Ramp Floats

float 值数组。

Ramp Interpolation

ramp 插值的 Type（类型）：

  • linear

  • catmull-rom

  • bspline

  • constant

Multiplier

Density

控制投射效果的强度。

Invert

指定应用前是否反转投射纹理。

Intensity

下方 diffuse 和 specular 贡献的全局 Multiplier。

Diffuse

此 light filter 结果用于 diffuse contribution 的 multiplier。

Specular

此 light filter 结果用于 specular contribution 的 multiplier。

Color Ramp

Color Ramp

定义 knot 数量。

Color Ramp Knots

knot 值数组。

Ramp Colors

color 值数组。

Color Ramp Interpolation

color ramp 插值的 Type（类型）：

  • linear

  • catmull-rom

  • bspline

  • constant

Combine Mode

PxrRampLightFilter 的 combine mode 始终为相乘。
