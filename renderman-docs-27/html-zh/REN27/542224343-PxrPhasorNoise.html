Dimensions

噪声的维度决定我们如何对发射相量波的核网格进行采样。对于地面平面或屏幕空间效果，选择 2D Flat（2D 平面）。

Align Mode

使用 phaseAlign 或 directionAlign 参数时，alignMode 会设置相量对齐的取向方式。例如，Cylindrical Z（Z 轴圆柱）会围绕 Z 轴创建环。

Frequency

相量波的主频率。此输入可连接。

Frequency Flatten

将频率相对于曲面法线展平。在 3D 模式下，相量波取向有时可能会以斜角穿过表面，导致感知到的噪声频率降低。此滑块会在噪声方向与曲面法线对齐时提高频率，以尝试补偿这种情况。

Shaping

Shaping Mode

Shaping Mode 选择应用到相量结果的轮廓。Sine 和 cosine 会生成平滑的周期波。Pulse 和 pulse centered 会分别按输入宽度在周期边缘或中心生成实心阶跃函数。Gabor 模式表示此噪声的 Gabor 版本，在波干涉区域的对比度低得多。Spline 模式允许使用样条曲线控制整形。

Shaping Soften

使用此控制项尝试柔化相量波干涉很强时出现的尖锐区域。提高柔化量会混入所选整形模式的解析平均值。

Direction

Direction

设置相量波的初始方向。此输入可连接。如果方向在一个相量波核的尺寸范围内快速变化，可能会引入扭曲伪影。可尝试让方向变化更慢，或减小核之间的间距。

Direction Align

按 alignMode 设置的方式对齐相量波方向。可用它在相量波噪声中创建线性、圆柱或球形图案。

Direction Jitter

对相量波方向加入抖动。此抖动是内置的，因此可以在相量波核和脉冲中心处评估噪声，从而防止扭曲伪影。

Direction Jitter Frequency

directionJitter 噪声的频率。

Direction Jitter Scale

directionJitter 噪声在 XYZ 上的缩放。

Direction Rotate

用它围绕曲面法线旋转相量噪声方向。相量噪声的方向有些反直觉：它定义的是波运动方向，也就是与波峰顶部正交的方向。此控制项允许以任意角度围绕法线摆动该方向。

Direction Flatten

将方向相对于曲面法线展平。在 3D 模式下，相量波取向有时可能会以斜角穿过表面，导致感知到的噪声频率降低。此滑块会将方向推向切线方向，以尝试补偿这种情况。

Phase Offset

偏移相量波的相位。接入时间值可以制作流动效果动画。

Phase Align

按 alignMode 设置的方式对齐相量波的相位。可用它尝试减少噪声结果中的扰动量。如果波方向也在变化，或与 alignMode 差异很大，对齐相位可能没有帮助。

Kernel

Kernel Frequency

相量波核位置的基础频率。

Kernel Scale

相量波核位置在 XYZ 上的缩放。

Kernel Neighbors

为卷积相量波而采样的近邻核数量。在 2D 中，会访问 (2n+1)^2 个相邻核。在 3D 中，会访问 (2n+1)^3 个相邻核。默认值为 2 时，2D 是 25 个核，3D 是 125 个核！更多核成本更高，但可能给出更平滑的结果。

Kernel Impulses

每个相量波核要采样的脉冲数量。它们会随机散布在每个相量波核网格单元内，但相位偏移采用均匀分布。更多脉冲成本更高。

Kernel Falloff

每个相量波核都有从中心开始的余弦形衰减。此控制项是作用在衰减上的幂函数；减小会展平影响区域，增大会锐化影响区域。

Kernel Error

设置采样核时允许的误差和不连续程度。为零时使用余弦衰减，以确保没有不连续；但当近邻核较少且核缩放被拉伸时，可能出现填满灰色的区域。增大到零以上会切换到高斯衰减来填补这些区域，但也可能沿核单元网格引入不连续。

Octaves

Phasor Octaves

增加相量倍频程会在不同频率上添加相量波脉冲。使用此控制项可以在不影响对比度的情况下向噪声结果添加纹理。

Phasor Octave Scale

每个后续相量倍频程的频率缩放。

Phasor Octave Weight

每个后续相量倍频程的权重。

Phasor Octave Offset

每个后续相量倍频程的相位偏移。

Phasor Octave Rotate

每个后续相量倍频程围绕曲面法线的旋转。

Fractal Harmonic Mode

每个分形倍频程和谐波倍频程的组合模式。这些倍频程会在相量结果评估完成后于振幅空间中组合。

Fractal Octaves

增加分形倍频程会在不同频率上计算相量结果，然后在振幅空间中组合。这些倍频程很可能不会彼此对齐。

Fractal Octave Scale

每个后续分形倍频程的频率缩放。

Fractal Octave Weight

每个后续分形倍频程的权重。

Fractal Octave Offset

每个后续分形倍频程的相位偏移。

Fractal Octave Rotate

每个后续分形倍频程围绕曲面法线的旋转。

Harmonic Octaves

增加谐波倍频程会以精确的 2x、4x 等频率计算相量结果，然后在振幅空间中组合。这些倍频程会彼此对齐。

Harmonic Octave Weight

每个后续谐波倍频程的权重。

Harmonic Octave Offset

每个后续谐波倍频程的相位偏移。

Advanced

Filter Scale

此噪声中的过滤基于每个相量波脉冲的最终频率和方向，并会计算后应用到每个分形与谐波倍频程。我们不能跳过核循环来节省计算时间，但过滤对收敛很有帮助。降低 filterScale 可以恢复细节，代价是需要更多迭代。
