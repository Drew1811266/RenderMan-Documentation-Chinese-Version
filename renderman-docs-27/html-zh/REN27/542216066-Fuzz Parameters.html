Fuzz Parameters

在下方参数中，有些参数可在连接到 PxrLayer 后，通过 Input Material 被覆盖，或通过 PxrLayerMixer覆盖。 PxrLayerSurface 设计用于更清楚地展示哪些参数不能在层中被覆盖：它只包含全局参数。当你确定要进行分层时，建议使用这种材质。这些设置的结果保持不变。

  • Bold Face 参数是可分层的，可以按层覆盖。

  • Italicized 参数不能分层或覆盖。所有层都会全局遵循这些参数。例如：选择 GGX 作为 Specular 模型时，所有层在该参数上都会使用 GGX。

Fuzz Parameters

该参数会引入少量逆反射，并有助于模拟织物、绒毛和细粉末。

Gain

Fuzz 权重。数值越高，该效果越强。下面的 Cone Angle 设为 16。



Color

Fuzz 颜色。它会模拟柔软的天鹅绒感效果。该效果会叠加在之前的 Specular 波瓣“之上”，可能看起来像污垢或细灰尘。下面的 Cone Angle 设为 16。



Cone Angle

Fuzz Roughness（对应 Marschner R cone angle）。请注意，这里使用了 Marschner 毛发模型中的一部分。这有助于模拟类似毛发/皮毛的响应。数值越高，面向角处的效果越强。下面的值为 8、16 和 20。



Bump

用于 Fuzz 照明的法线。如果未设置，它会使用本页底部 Properties 中指定的全局 bump normal（凹凸法线）。



Double Sided

如果开启，该 Fuzz 波瓣会照亮表面的两侧，也就是说，法线背向相机的表面也会被照亮。
