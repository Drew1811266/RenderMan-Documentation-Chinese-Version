Signal AOV String

用于 toon shading Signal 的 color AOV 名称。通常为 diffuse，但也可以在这里放入任何 light AOV。此 AOV 必须在 Maya/Katana/Houdini 中设置，否则 DisplayFilter 会退出。

Diffuse Energy from 

在底层，会基于 Signal AOV 创建一个白色 shader 近似来驱动 Toon response（卡通响应）。默认使用 Signal AOV 的 RGB 分量平均值，但也可以用 NPRalbedo AOV 或 albedo AOV 中的纯 albedo，去除 Signal AOV 光照结果中的 albedo 分量。如果选择 NPRalbedo AOV，则必须在 Stylized Controller Pattern 节点中设置 Albedo Color。

Toon

Step Mode

Albedo：从 albedo AOV 阶梯化到对象 albedo。Light To Albedo：从白色或 light color 阶梯化到对象的 albedo color（如果 albedo 或纹理较暗，可使用 albedoLift 看到更多细节）。Darks Color：阶梯化到所选颜色，而不是默认黑色。

Toon Diffuse Steps

用于量化 diffuse lighting 的 step 数量

Albedo Lift

当 Step Mode = Albedo 或 Light to Albedo 时，可使用它抬升 albedo 值

Darks

要 step 到的 Color

Linstep Min

stepping 的 min-max remap 最小值。查看 NPRtoonDiffRamp AOV 以进行可视化。

Linstep Max

stepping 的 min-max remap 最大值。查看 NPRtoonDiffRamp AOV 以进行可视化。

Gamma

toon diffuse ramp 的 Gamma。查看 NPRtoonDiffRamp AOV 以进行可视化。

Step to Albedo

Step to Albedo：step 到对象的 Albedo Color，而不是 Blacks。仍然使用 Blacks，但会把它作为 Albedo Color 上的 multiplier，因此请将 Blacks 切换为灰度值。

Toon Tips

  • 将 integrator 改为 PxrDirectLighting 会得到更清晰的 toon 图像

  • 若要获得传统且更清晰的 toon 渲染，可尝试使用 PxrDistanceLight 和 PxrSphereLight 等灯光

  • 若要减少 Toon banding 中的噪点，请使用 PxrDistantLight，并关闭 bounces（可使用 PxrDirectLighting integrator，或在 PxrPathTracer integrator 中将 Max Indirect Bounces 设为 0）
