PxrEnvGround

Source: https://rmanwiki-27.pixar.com/space/REN27/542223428/PxrEnvGround
Downloaded: 2026-06-15T03:49:12+00:00

为外观开发创建转台时，渲染带有环境和地面的对象会很有用，可用于照明和反射。

此 pattern 旨在与高动态范围图像（HDRI）配合使用，通常是摄影捕获环境的 OpenEXR 文件；该环境也作为光源，并使用一个 PxrDomeLight。然后将此 pattern 连接到你的地面几何体材质。

Offset 和 Rotation 这类放置控制用于让它与你的 PxrDomeLight 最终放置位置相匹配。

此 pattern 适用于外观开发工作流：将你的资产放在场景原点/中心，以保持正确投射。


Input Parameters（输入参数）

Filename

在此提供用作地面的 HDRI 纹理。请注意，离原点越远，细节可能会逐渐丢失。

Environment Radius

更改所提供投射的距离；不正确的值会导致可见拉伸。

Offset X

沿 X 方向移动结果。负值会向相反方向移动。

Offset Z

沿 Y 方向移动结果。负值会向相反方向移动。

Rotation

以度为单位，围绕 Y 轴旋转结果。

Filter Width

可用于模糊纹理结果。

Output

ResultRGB

通常连接到材质的 Diffuse color 输入。

Result Presence

连接到材质的 Presence 参数。

ResultNG

这会连接到一个 PxrBumpMixer （必需），然后再连接到材质的 Normal/Bump 参数，通常是 global bump（全局 bump，如果存在）。
