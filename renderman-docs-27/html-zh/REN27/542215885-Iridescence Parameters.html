在下方参数中，有些参数可在连接到 PxrLayer 后，通过 Input Material 被覆盖，或通过 PxrLayerMixer覆盖。 PxrLayerSurface 设计用于更清楚地展示哪些参数不能在层中被覆盖：它只包含全局参数。当你确定要进行分层时，建议使用这种材质。这些设置的结果保持不变。

  • Bold Face 参数是可分层的，可以按层覆盖。

  • Italicized 参数不能分层或覆盖。所有层都会全局遵循这些参数。例如：选择 GGX 作为 Specular 模型时，所有层在该参数上都会使用 GGX。

Iridescence

Iridescence 是一种视角相关的光散射，会造成颜色偏移。肥皂泡、孔雀羽毛或发亮甲虫上的颜色旋涡都来自同一种效果。“Holographic”或变色涂料也会使用这种效果。

Iridescence Mode

选择要使用的 Iridescence 模式： Artistic 或 Physical。

在 Artistic mode下，我们只设置 2 种颜色。根据 Iridescence 的缩放因子，会看到 N 条“彩虹”。默认的红色和蓝色适合获得最大的颜色跨度，但你可以通过修改这些默认值减少渲染出的颜色数量。除非另有说明，或正在演示某个 Artistic 参数，否则示例使用 Physical mode。

在 Physical mode下，我们传入薄膜厚度（纳米）。当物理厚度接近可见光谱时，就会产生 Iridescence 效果。可以从约 800nm 开始增加数值来观察效果。这个选项很好用，因为它减少了需要调节的参数，但代价是灵活性降低。除非另有说明，或正在演示某个 Artistic 参数，否则示例使用 Physical mode。

下方为 Artistic （左）和 Physical （右）模式。



Face Gain

面向角（0 度入射）处的 Iridescence Gain。



Edge Gain

掠射角（90 度入射）处的 Iridescence Gain。



Fresnel Exponent

该参数控制 Face 与 Edge 之间的衰减，类似于 Fresnel Exponent 在 Specular Lobes 中的作用。通常保留默认值 5.0


Primary Color

这仅适用于 Artistic mode 。

Iridescence 在色相环上的起始主颜色。从这里开始，颜色会在 Primary Color 和 Secondary Color 之间经过其他可用色相而偏移。你在色相环上选择得越近，渲染出的颜色就越少。下面三个示例分别从红色、黄色，最后从绿色开始。色条显示这些选择之间可用的颜色。


Secondary Color

这仅适用于 Artistic mode 。

Iridescence 在色相环上的结束次颜色。如上所示，可以用它限制渲染出的颜色。下面三个示例中，Secondary Color 从紫色变为蓝色，最后变为绿色。色相条以 ramp 展示这种变化。


Falloff Speed

这仅适用于 Artistic mode 。

从 Primary Color 到 Secondary Color 的衰减速度。数值越大，衰减越慢。下面使用 Artistic Mode 的默认值，但将 Falloff Speed 从 0.1 改为 0.5，再改为 1.0。



Falloff Scale

这仅适用于 Artistic mode 。

此参数设置 Iridescence “彩虹”颜色重复的次数。下面从 0.5 到 1.0，最后到 3.0。请注意，较高的值会开始重复彩虹效果。这对于模拟油膜图案很有用，例如水面上的油或肥皂泡。



Flip Hue Direction

这仅适用于 Artistic mode 。

在 Primary Color 和 Secondary Color 之间翻转色相环方向。默认情况下，色相环方向为逆时针。左侧为 off，右侧为 on。



Thin Film Thickness

这仅适用于 Physical mode 。

薄膜厚度，单位为纳米。从左到右，我们从 400 开始，然后是 800，最后是 1600 纳米。请注意，在 1600 时开始看到彩虹效果重复。这类似于在 Falloff Scale 中使用 Artistic Mode的效果。



Roughness

Iridescence Roughness 与其他 Roughness 参数类似，可以从 0.0 的镜面般反射过渡到 1.0 的 Diffuse 反射。下面示例从左到右为 0.0、0.5 和 1.0。较柔和的外观会让人联想到变色妆容和类似粉末。



Double Sided

如果开启，会为该 Iridescence 波瓣照亮表面的两侧。这对于羽毛、叶片等无厚度建模的薄开口表面很有用。
