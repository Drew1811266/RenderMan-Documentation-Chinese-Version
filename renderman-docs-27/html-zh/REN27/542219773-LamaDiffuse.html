Diffuse shading 是渲染中最常见的外观之一；diffuse 对象通常干燥且未抛光，例如报纸、泥土和岩石，并且常常是许多材质的 base。

这些对象都会从表面向广泛方向反射光线。这意味着它们不会显得闪亮或具有反射性；如果需要添加光泽和反射性，则需要类似 LamaConductor 这样的节点。


Diffuse Parameters

Color

Color 通常用于连接 diffuse/albedo textures 或 patterns，为不透明对象创建颜色。


Roughness

Diffuse roughness 用于模拟干黏土或灰尘那样的粉末状表面。 
当 roughness 为 0.0 时，LamaDiffuse 使用 Lambertian model 计算 diffuse response。 
当 roughness > 0.0 时，则改用 Oren Nayar model。


Normal / Bump

在这里提供一个 signal（texture 或 procedural pattern），用于在表面上创建 bump，以“伪造”小凸起或划痕等表面细节。这意味着美术师无需对模型中这些繁琐且经常重复的部分进行建模。
