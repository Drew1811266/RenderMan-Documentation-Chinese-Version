此 pattern 用于根据可从不同节点读取的 index，评估不同的 pattern 网络。


Switch

Index

这个 float 值指示要评估网络的哪一部分，并且通常从 attribute 读取。值的读取方式如下：

-0.5 到 0.49 为 input 0

  0.5 到 1.49 为 input 1

  1.5 到 2.49 为 input 2，依此类推。


Inputs

Inputs RGB

要评估的输入数组，评估依据是返回的索引值。


下面是一个简单示例：每个球体 transform 都被分配一个 user attribute 作为 index，值为 0.0、1.0、2.0。可用颜色及其 index 是 Red[0]、Green[1] 和 Blue[2]（向 Inputs RGB 数组中添加的三个输入）。

同一个 PxrSurface 会附加到这三个球体上，并且一个 PxrAttribute 节点连接到材质的 Diffuse Color 输入。

该 attribute 可以使用你想要的任意名称，并采用 Float data type（Float 数据类型）。
