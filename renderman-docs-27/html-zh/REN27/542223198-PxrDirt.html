PxrDirt

Source: https://rmanwiki-27.pixar.com/space/REN27/542223198/PxrDirt
Downloaded: 2026-06-15T03:49:12+00:00




PxrDirt 会渲染局部几何遮挡。这可用于将着色效果应用到凹角和缝隙。

传统的环境光遮蔽效果可以通过将被遮挡颜色设为黑色、将未遮挡颜色设为白色来实现。

Parameters（参数）

Occluded and Unoccluded

被遮挡和未遮挡着色的结果颜色。

Num Samples

控制每个摄像机命中点的遮挡采样数量。更多采样可以降低噪声，但代价是增加渲染时间。

Sample Distribution

可用值为 uniform 和 cosine (Lambertian)。


说明

Uniform

光线在着色点上方的半球内不会在任何特定方向上加权。

Cosine

Cosine 分布通常用于渲染环境光遮蔽。

Cosine Spread

使用 cosine 分布时的光线采样扩散。值 1.0 会给出完美的 Lambertian 分布。

Falloff

Falloff 可用于控制遮挡效果的柔和度。

Max Distance

物体可能产生遮挡的最大距离。零表示无限（所有物体都会遮挡）。

Direction

沿表面法线方向追踪光线（outside），沿法线反方向（inside），或沿 both 方向。

Bias Direction

向法线向量添加方向偏置。值必须以世界空间指定。

Bias Co-ordinate System

偏置方向坐标系统变换的名称，例如 object、world 或 camera

Trace Set

将光线追踪限制到由一个或多个追踪组定义的物体子集
