PxrLayer

Source: https://rmanwiki-27.pixar.com/space/REN27/542217837/PxrLayer
Downloaded: 2026-06-15T09:18:58+00:00

PxrLayer 是一种 pattern 节点，用于对 PxrSurface的着色器参数进行分层。


该 pattern 节点由 Pixar Animation Studios 使用 OSL 编写。它只会分层匹配的着色器参数，并跳过不匹配的参数。建议将 PxrLayer 与 PxrLayerMixer 以及 PxrLayerSurface 材质配合使用。PxrLayer 负责你的结果（最终外观），而 PxrLayerSurface 负责整个分层材质波瓣设置的全局控制，并且必须分配给你的对象。

参数从下方开始。

下方参数可由另一个 PxrLayer 覆盖：当其连接到 Input Material （属于 PxrSurface ）时，或通过 PxrLayerMixer覆盖。 PxrLayerSurface 的设计目标是通过只包含全局参数，更清楚地展示哪些参数不能在层中被覆盖。我们建议使用 PxrLayerSurface 材质，适用于明确需要分层的情况。

Diffuse Parameters

Diffuse Parameters

Diffuse 参数控制基础漫反射的外观。它们通常用于定义不透明对象的主要颜色属性；木纹、标签文字、圆点图案等等，都可以连接在这里。该波瓣默认开启。

Enable

默认开启；如果关闭，则使用下一层更低的层（或基础层）波瓣。

Gain

Gain 是应用到 Diffuse 参数上的权重。你也可以用另一个 pattern 节点驱动它，用于表现褪色或潮湿之类的效果（液体会让表面变暗）。下方示例展示 50% 灰色材质在 0.0、0.5 和 1.0 Gain 下的效果。


Color

Color 通常用于连接纹理或 pattern，以创建不透明对象的颜色。例如，木材颜色纹理就会放在这里。


Roughness

Diffuse 粗糙度可用于模拟干黏土或灰尘这类粉状表面。

Bump

凹凸贴图是一种用着色来伪造物理细节外观的好方法。如果未设置，它将使用本页底部附近 Properties 部分中指定的全局凹凸法线。

Back Color

这仅在 Double Sided 开启时适用。当 Use Diffuse Color 关闭时，此项设置背面颜色。默认情况下，它使用 Diffuse Color，但选择不同颜色可以让 2D 对象的背面呈现不同外观。

Transmit Gain

这仅在 Double Sided 开启时适用。这里不是使用 Diffuse Gain 来控制漫透射的强度，而是使用 Transmit Gain。如果它为 0.0，则该效果关闭。下方的 Transmit Color 是亮蓝色。

Transmit Color

这仅在 Double Sided 开启时适用。此项设置透射颜色，它可以不同于 Diffuse 或 Back Color。如果 Transmit Gain 为零，则会忽略此项。该效果适合叶片或纸张这类薄物体。下方对象内部放置了一盏灯，随着光线透过表面，可以看到一些内部文字。

Specular Parameters

Specular, Rough Specular, and Clear Coat Parameters

Specular 参数控制镜面反射。你可以在这里定义对象有多闪亮或多具反射性：它是塑料、抛光大理石桌面，还是一面镜子？下面的 Rough Specular 波瓣具有相同的设置和效果。

还可以选择 Artistic 和 Physical 控制项。Artistic 控制项允许你手动改变反射属性以符合你的偏好。使用 Physical（配合提供的预设）可以为希望复制真实世界、又不想无休止调整参数的用户提供匹配的真实响应。

Enable

默认关闭；如果开启，除非被遮罩掉，否则这些设置会覆盖其他层。

Face Color (Artistic Mode)

面向角度（0 度入射）处的 Specular 颜色。请注意，这里没有单独的 Gain 控制。要控制 Specular“增益”，只需调整颜色值，或将其连接到 PxrExposure 节点。下方是在 Roughness 0.25 下的不同选择，包括纹理输入。

Edge Color

掠射角（90 度入射）处的 Specular 颜色。要控制边缘镜面反射“增益”，只需调整颜色值，或将其连接到 PxrExposure 节点。下方是在 Roughness 0.25 下的不同选择，包括纹理输入。这里还降低了 Fresnel Exponent，使效果更明显。

请理解，该控制项在 Physical Mode 中的含义会发生变化，并作为 Physical 模式中反射结果的乘数。你可以在 Physical Mode 中使用该参数控制反射强度，甚至为结果着色。下方使用了黑色、50% 灰色和白色。

Refractive Index (Physical Mode)

这是一个用于描述物理折射率的参数；即材质的 电介质 折射率。该参数的通道值通常位于 1 到 3 范围内。由于我们支持 3 个颜色值来捕捉光谱效果，使用预设可能比颜色选择器更合适，可以避免大量微调。

Extinction Coefficient (Physical Mode, not available for Clearcoat)

Extinction Coefficient 是材质的第二个折射率，用于表征金属行为。该参数的通道值通常位于 1 到 3 范围内。由于我们支持 3 个颜色值来捕捉光谱效果，预设可能比颜色选择器更合适。为 0 时，材质会像电介质（玻璃）一样响应。非零时，材质会像导体一样响应。由于它基于物理值，你会发现预设比手动调参更有帮助。下方是 Copper、Gold 和 Nickel 的预设。


Roughness

Specular 粗糙度。较大的值会产生更粗糙或“模糊”的镜面反射。1.0 时类似漫反射表面，0.0 时是完全清晰的反射。多数对象在这些值之间会显得真实。为该值添加纹理可能产生有趣效果，例如污迹、油腻指纹和磨损表面。下方示例从 0.0 到 0.5，最后到 1.0（漫反射）。

Anisotropy

控制镜面高光和反射的形状。0 表示各向同性，会产生常规圆形镜面高光。从 -1.0 到 1.0 的值会产生从横向到纵向的一系列椭圆（拉伸）。

默认情况下，Anisotropy 的方向由模型纹理参数控制。如果指定了 Shading Tangent，则改用它。下方示例为 -1.0、0.0 和 1.0。

Bump

用于 Specular 照明的法线。如果未设置，它将使用本页底部附近 Properties 中指定的全局凹凸法线。

Rough Specular

与 Specular 参数相同，只是默认 Roughness 更大，为 0.6。该层旨在用于比上方 Specular 波瓣更高的 Roughness 设置。下方从左到右：Rough Specular、Specular，以及两个波瓣组合。

Layer Thickness (Physical Mode)

该控制项模拟材质上的厚涂层或釉层。它也可以通过纹理改变颜色。下方的着色球只在 Diffuse 波瓣上添加了 Bump，使 Thickness 的使用效果更明显。前两张图使用相同的 Absorption Tint，Thickness 分别为 1 和 3，以展示更厚的设置会在物理上吸收更多光。最后一张图是使用棋盘格 pattern 节点的纹理化 Thickness。

Absorption Tint (Physical Mode)

该参数控制最终衰减或涂层的颜色。请注意，上方 Thickness 参数可能改变外观：更厚的设置会更饱和或吸收更多光。下方是两种纯色以及一个渐变，Thickness 保持恒定为 1.0。底层 Diffuse 波瓣应用了 Bump，以帮助模拟“厚度”。

Iridescence Parameters

Iridescence

Iridescence 是一种依赖视角的光散射，会导致颜色偏移。这与肥皂泡、孔雀羽毛或闪亮甲虫上的色彩旋涡是同一种效果。“Holographic” 或变色漆也会使用这种效果。

Enable

默认开启；如果关闭，则使用下一层更低的层（或基础层）波瓣。

Face Gain

面向角度（0 度入射）处的 Iridescence Gain。

Edge Gain

掠射角（90 度入射）处的 Iridescence Gain。

Primary Color

这仅适用于 Artistic mode 。

Iridescence 在色轮上开始的 Primary Color。从这里开始，颜色会在 Primary Color 和 Secondary Color 之间穿过其他可用色相。你在色轮上选择的颜色越接近，渲染出的颜色越少。下方三个示例分别从红色、黄色，最后从绿色开始。色条显示这些选择之间可用的颜色。

Secondary Color

这仅适用于 Artistic mode 。

Iridescence 在色轮上结束的 Secondary Color。如上所示，你可以用它限制渲染出的颜色。下方三个示例中，Secondary Color 从紫色到蓝色，最后到绿色。色相条在一个渐变上显示了这种变化。


Roughness

Iridescence 粗糙度，与其他 Roughness 参数类似，可从 0.0 的镜面式反射过渡到 1.0 的漫反射。下方示例从左到右为 0.0、0.5 和 1.0。更柔和的外观会让人联想到变色妆容和类似粉末。

Fuzz Parameters

Fuzz Parameters

此参数会引入一点逆向反射，并帮助模拟织物、Fuzz 和细粉末。

Enable

默认开启；如果关闭，则使用下一层更低的层（或基础层）波瓣。

Gain

Fuzz 权重。数值越高，此效果越强。下方的 Cone Angle 设置为 16。

Color

Fuzz 颜色。它模拟柔软、类似天鹅绒的效果。该效果应用在之前的 Specular 波瓣“之上”，可能类似污垢或细尘。下方的 Cone Angle 设置为 16。

Cone Angle

Fuzz 粗糙度（对应 Marschner R cone angle）。请注意，这里使用了 Marschner 毛发模型中的内容，有助于模拟类似毛发/皮毛的响应。数值越高，正对角度处的效果越强。下方为 8、16 和 32。

Bump

用于 Fuzz 照明的法线。如果未设置，它将使用本页底部附近 Properties 中指定的全局凹凸法线。

Subsurface Scattering Parameters

Subsurface Scattering Parameters

Enable

默认开启；如果关闭，则使用下一层更低的层（或基础层）波瓣。

Gain

次表面散射权重。数值越高，次表面散射的可见性越强。

Color

次表面散射颜色。

Mean Free Path Distance

次表面散射的 Mean Free Path Distance（mfpd）。它指定光线在对象内部传播多远，并因此决定次表面散射有多平滑。该值会乘以 Properties 部分中设置的 unit length。较大的值会让对象显得不那么不透明、更半透明，同时也会增加噪声。较小的值会让表面看起来像漫反射，此时如果视觉上不重要，更高效的做法可能是关闭该效果（0.0 Gain）。下面将数值从 8 改为 16，再改为 32。注意球体和基座会变得“更柔和”且更半透明。高数值时，半透明对象产生的噪声可能会增加渲染时间。

Mean Free Path Color

光线在红、绿、蓝光谱中传播的距离。此值会按 Mean Free Path Distance 缩放。不同颜色扩散程度可能不同，并可产生有趣效果，例如皮肤阴影边缘中的红色渗透。RGB 值对应光线在该色带中传播多远。例如，RGB 值 0.8 0.65 0.5 表示 红色在 0.8 时扩散最远，然后是绿色，最后蓝色在对象散射结果中传播距离最短。下面我们移除了中心球，并在外球背后用一个明亮的圆盘灯替换。请注意，球体 和 基座共享同一材质，灯光会对结果产生重要影响。

使用 Multiple Mean Free Paths 时该选项不可用，因为它改用 Short 到 Long Color 选项。

Post Tint

在次表面计算末尾应用的 Tint。下面左侧是普通渲染，右侧添加了非常浅的蓝色 Tint。如果想在次表面计算之前应用 Tint，请在 Properties 部分设置 Irradiance Tint。

Multiple Mean Free Paths Description


Multiple Mean Free Paths 在几个重要方面与其他模型的工作方式不同。大多数情况下，其他模型的结果已经足够令人满意。不过有些场景中，用户可能希望用非物理方式控制散射颜色。许多模型中的 Mean Free Path Color 会按参数说明使用提供的 RGB 值来决定颜色散射。这意味着结果可能产生不需要的颜色偏移。假设你提供了 RGB 0.51 0.28 0.31 的纹理颜色，也就是一种粉色。散射结果也会包含绿色（0.28）和蓝色（0.31）响应。Multiple Mean Free Paths 模型会保持这种粉色。下面示例使用这一场景来展示差异。纹理连接到 Jensen Dipole 的 Color 和 Mean Free Path Color，同时也连接到 Multiple Mean Free Paths 模型的 Short 和 Long Colors。注意 Jensen 模型（左侧）上的颜色偏移，而 Multiple Mean Free Paths 版本会在所有连接深度上保持纹理颜色。

Short Gain

短程次表面增益或权重。仅对 Multiple Mean Free Paths 次表面模型有效。


Short Color

短程次表面颜色。仅对 Multiple Mean Free Paths 次表面模型有效。上方使用红色作为所选颜色。


Long Gain

长程次表面增益或权重。仅对 Multiple Mean Free Paths 次表面模型有效。其行为与 Short Gain相同。


Long Color

长程次表面颜色。仅对 Multiple Mean Free Paths 次表面模型有效。其行为与 Short Color相同。

Short MFP Distance

短程次表面平均自由程标量距离。下面从 16 到 24，最后到 32。

Long MFP Distance

长程次表面平均自由程标量距离。下面从 16 到 24，最后到 32。

Transmit Gain

这仅在 Double Sided 开启时适用。这里不是使用 Subsurface Gain 来控制次表面透射的强度，而是使用 Transmit Gain。如果它为 0.0，则该效果关闭。

Single Scatter Parameters

Single Scattering Parameters

单次散射是一种简单且开销较低的散射效果。下方有一个 圆盘灯被放置在模型内部 （所有这些示例都如此），我们先在没有漫反射颜色的情况下渲染，然后再使用默认中灰漫反射颜色渲染。

Enable

默认开启；如果关闭，则使用下一层更低的层（或基础层）波瓣。

Gain

单次散射的增益或权重。

Color

单次散射颜色。

Mean Free Path

单次散射的 Mean Free Path 标量距离。它指定光线在对象内部传播多远，并因此决定单次散射有多平滑。该值会乘以 Properties 部分中设置的 Unit Length。较大的值会更半透明。此项类似于次表面控制项。

Mean Free Path Color

光线在 R、G 和 B 光谱中传播的距离。此值会按 Mean Free Path Distance 缩放。

Directionality

单次散射的 Directionality：

0：各向同性（无效果）。

1：前向散射，背面获得更多光照。

-1：后向散射，正面获得更多光照。

Refractive Index

单次散射的折射率。下方依次为 1.3、5.0 和 10.0。

Blur

单次散射的 Blur 强度。这个多边形平面后面有一些文字。增大 Blur 会让文字显得更模糊（并使结果稍微变暗）。你可以把它看作散射透射的粗糙度。

Backside Direct Illum Gain

来自另一侧直接照明的 Gain。这会提高背面照明的影响。

Direction Tint

Backside Direct Illum Gain 的着色颜色。下方使用不同颜色为第一张图（白色）的结果着色。

Glow Parameters

Glow Parameters

Glow 可让对象 看起来像是 在发光。这在你需要受纹理控制的效果时很有用，例如发光面板、电路、熔岩，或其他在照明中具有局部影响的复杂效果。

这不是给场景打光的高效方式，并且需要间接反弹至少为 2 才会有效。我们建议将其用作纹理效果，而不是实际照明。

Enable

默认开启；如果关闭，则使用下一层更低的层（或基础层）波瓣。

Gain

Glow 的增益或权重。下方从默认灰色材质开始，并将 Gain（使用白色 Color）增加到 0.5，然后增加到 1.0。

Color

控制材质的自发光颜色，即 Glow。下方展示了输入到 Color 参数的各种纹理示例。

Glass Parameters

Glass Parameters

Glass 参数旨在提供一个单一位置，用于创建和操作基于物理的玻璃结果。从清澈玻璃到有色玻璃，以及其间各种粗糙度和雾化程度都可以实现。

Enable

默认开启；如果关闭，则使用下一层更低的层（或基础层）波瓣。

Refraction Gain 和 Reflection Gain 控制项旨在配合使用。单独使用其中之一可能产生有趣效果，但这并非设计意图。还要注意，第一次创建 Glass 时，你会发现 Diffuse 的默认 Gain 1.0 可能干扰你的外观。要制作纯玻璃，请将 Diffuse 的 Gain 调为 0.0。

Refraction Gain

Refraction Gain 是一个乘数。较低数值会让更少光线穿过对象，可能看起来像深色染色玻璃，或在 0.0 时直接变黑。

Reflection Gain

Reflection Gain 是一个乘数。较低数值会降低可见反射结果。设置为 0.0 会产生不真实的外观，如下方第一张图所示。

Refraction Color

Refraction Color。它允许你创建具有真实着色、并像真实透射对象一样投射彩色阴影的对象（请注意，灯光上应启用 Thin Shadows）。

Roughness

折射和反射的 Roughness。你可以在这里模拟磨砂或蚀刻玻璃。

Refractive Index

折射率。默认值为 1.5，这是玻璃的典型值。在线 IOR 值资源可以提供更有用的设置。较高的 IOR 值会比低值更强地弯折光线。如果想避免弯折光线，请使用 Thin 。

Global Properties Parameters

Properties Parameters

本节为整个材质提供全局控制。例如，如果只在这里提供 Bump，它会应用到上面的所有波瓣。Presence 也是一个全局参数，它会遮罩掉或“镂空”所有波瓣，就像遮罩值为 0.0 的位置对象不存在一样。

Bump

用于所有波瓣的 Normal/Bump，除非被单个波瓣的凹凸法线覆盖。
