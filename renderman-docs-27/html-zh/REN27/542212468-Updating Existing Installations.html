Updating Existing Installations

Source: https://rmanwiki-27.pixar.com/space/REN27/542212468/Updating+Existing+Installations
Downloaded: 2026-06-15T03:49:12+00:00

在新的 RenderMan 安装完成后，如果需要新的许可证，现在可以使用新的 Pixar License App 下载并轻松升级到新的许可证文件。

更新浮动许可证

要更新现有浮动许可证，请在运行许可证服务器的主机上运行 LicenseApp。

你应该会看到当前正在提供服务的现有许可证：

然后可以选择： Get License 随后会弹出一个对话框，要求你输入论坛 ID 和密码，以及要下载的许可证序列号。

许可证下载后，正在运行的许可证服务器实际上不会自动加载它。你需要停止并重启许可证服务器，才能使此更改生效。你可以通过重启机器来完成；前提是许可证服务器已设置为在机器启动时自动启动。或者，你也可以手动启动和停止许可证服务器。此过程按操作系统分别说明如下：

Linux

macOS

Windows

如果在合理较短的时间内重启许可证服务器，正在运行的渲染任务不应失败。你会看到如下类型的警告：

R50004 {WARNING} License warning - code 10054: Connection lost to peer

R50004 {CONTINUED} license source: 9010@MYLICSERVER

R50004 {WARNING} License warning - license server connection re-established

 

更新节点锁定许可证

要更新现有节点锁定许可证，请在拥有该节点锁定许可证的主机上运行 LicenseApp。

你应该会看到当前锁定到该机器的现有许可证：

然后可以选择： Get License 随后会弹出一个对话框，要求你输入论坛 ID 和密码，以及要下载的许可证序列号。

新许可证下载完成后，可以直接关闭 LicenseApp；你的应用程序随后应会使用新的许可证文件运行。
