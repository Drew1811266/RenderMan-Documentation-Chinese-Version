PxrThreshold

Source: https://rmanwiki-27.pixar.com/space/REN27/542225099/PxrThreshold
Downloaded: 2026-06-15T03:49:12+00:00

对输入颜色 pattern 进行阈值化，以创建黑白 mask。衰减会定义从黑到白的渐变过渡。输出始终位于 0 到 1 之间。

Input Parameters

Input Color

要进行阈值化的颜色 pattern。

Channel

此节点将作用于的输入通道：

  • 0: Red
  • 1: Green
  • 2: Blue
  • 3: Luminance
  • 4: Average

Threshold

发生从黑到白过渡时所使用的值。

Transition Width

此控件允许创建锐利或渐进的过渡。过渡宽度以阈值为中心。

Transition Profile

从黑到白的插值可用三种不同曲线计算：

  • 0: Linear
  • 1: Smooth
  • 2: Gaussian

 

Output Parameters

resultRGB

阈值化后的 pattern，以单色颜色输出。

resultR

R 通道结果

resultG

G 通道结果

resultB

B 通道结果

resultA

如果存在 alpha，并且 resultRGB 已连接，则返回 RGB 之后的通道。如果 RGB 未连接，则返回第一个通道。
