PxrVary

Downloaded: 2026-06-15T03:49:12+00:00



此节点会创建受控随机变化。变化源可以基于 attribute 或 primvar。primvar 应具有 constant detail（每个 object 一个单一值）。只有非零 id/primvar/attribute 值才会创建变化。如果无法读取源，则不会产生变化。能够查找任意 float 或 int attribute，是调试场景的好方法。例如，如果你想知道哪些 object 是 single-sided（单面），可以在 varName 中输入 'Ri:Sides'。

Input Parameters

Input Color

你想要施加变化的颜色。

Vary Source

变化源，可为以下之一：

  • 0: Identifier Id
  • 1: Identifier Name
  • 2: Primvar
  • 3: Attribute

Variable Name

如果 varySource 为 1 或 2，则这里是 primvar 或 user attribute 的名称。如果使用 primvar，它应为 constant；不支持 per-vertex/face primvars。对于 user attributes，请输入 "user:myattribute"。

Hue

限制随机 hue（色相）偏移。范围从 0（无变化）到 1（完整变化）。

Saturation

限制随机 saturation scale（饱和度缩放）。范围从 0（无变化）到 1（完整变化）。

Luminance

限制随机 luminance scale（亮度缩放）。范围从 0（无变化）到 1（完整变化）。

Gamma

限制随机 gamma 变化。范围从 0（无变化）到 1（完整变化）。

Advanced

Probability

0 到 1 的范围指定应用变化的可能性。

Hue Mode

颜色 hue 将如何变化：

  • 0: Centered -- 变化会以当前值为中心。
  • 1: Additive -- 变化会添加到当前值。
  • 2: Subtractive -- 变化会从当前值中减去。

Saturation Mode

颜色 saturation 将如何变化：

  • 0: Centered -- 变化会以当前值为中心。
  • 1: Additive -- 变化会添加到当前值。
  • 2: Subtractive -- 变化会从当前值中减去。

Luminance Mode

颜色 luminance 将如何变化：

  • 0: Centered -- 变化会以当前值为中心。
  • 1: Additive -- 变化会添加到当前值。
  • 2: Subtractive -- 变化会从当前值中减去。

Gamma Mode

颜色 gamma 将如何变化：

  • 0: Centered -- 变化会以当前值为中心。
  • 1: Additive -- 变化会添加到当前值。
  • 2: Subtractive -- 变化会从当前值中减去。

Seed

允许在不改变已设置 primvar 的情况下改变变化结果

Scale Normalized Primvar

对于 primvar scale 超出 normalized range 的项目，会取已设置的 primvars，并将它们缩放到 pattern 使用的预期 0-1 范围

Verbosity

输出各种信息以帮助你调试场景。

  • 0: Silent -- 仅输出错误消息。
  • 1: Warnings -- 报告错误和缺失的 attributes。
  • 2: Infos -- 与上面相同，并额外输出 attributes 的值。

Output Parameters

resultRGB

clamp 后的颜色结果。

resultR

R 通道来自 resultRGB 输出。

resultG

G 通道来自 resultRGB 输出。

resultB

B 通道来自 resultRGB 输出。
