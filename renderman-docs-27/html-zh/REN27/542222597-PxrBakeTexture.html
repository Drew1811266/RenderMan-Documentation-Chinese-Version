PxrBakeTexture

Source: https://rmanwiki-27.pixar.com/space/REN27/542222597/PxrBakeTexture
Downloaded: 2026-06-15T03:49:12+00:00

PxrBakeTexture 节点 可作为 pass-through node 使用，用于将 pattern networks 烘焙到 2D 纹理；这是 baking（烘焙） workflow 的一部分。

Parameters（参数）

Input Color（输入颜色）

这是要烘焙的 pattern network 的传入结果。此连接需要 color (RGB)。此输入优先于 Input Float（输入 Float）。

Input Float（输入 Float）

这是 pattern network 的传入结果，其中输出为 float 或 scalar 值。

Filename（文件名）

生成的已烘焙纹理的输入/输出文件名。

此参数是必需的。

Atlas Style（图集样式）

atlas textures 的格式化样式。启用后，文件名必须包含 _MAPID_ 占位符 它会根据表面参数化自动替换。

Value（值）

Example（示例：texture_MAPID_.tex）

None（无）

texture.tex（示例）

UDIM (MARI)

texture1001.tex（示例）

UV Tile Base-1 (Mudbox)

texture_u1_v1.tex（示例）

UV Tile Base-0 (ZBrush)

texture_u0_v0.tex（示例）

UDIM 只针对 U 值在 0.0 到 10.0 范围内、且 V 值大于 0.0 的纹理定义。用户应避免在所有 atlas texture workflows 中使用负纹理坐标。

例如，如果要跨所有 UDIMS 烘焙资产的 PxrCurvature，文件路径应如下所示 outputpath/HouseOne_Curvature._MAPID_.tex（示例路径）

Filter（过滤器）

选择可在纹理查找期间使用的不同重建过滤器。可用 filters 包括：

  • Nearest（最近邻）

  • Box（盒式）

  • Bilinear（双线性）

  • Bspline（B 样条）

  • Gaussian（高斯）

  • Lagrangian（拉格朗日）

Mip Interpolate（Mip 插值）

在 mipmap 级别之间插值，使级别之间的过渡更平滑。

Invert T

为纹理查找反转 T 参数。

Missing Color（缺失颜色）

打开纹理文件出错时得到的颜色。

Bake Mode（烘焙模式）

此模式启用或禁用纹理 baking。

Render Mode（渲染模式）

控制常规渲染期间是从输入 patterns 读取，还是从已烘焙纹理读取。

Value（值）

Description（描述）

Automatic（自动）

尝试先读取先前已烘焙纹理，否则读取输入 pattern。

Input（输入）

始终从输入 pattern 读取。

Texture（纹理）

始终从先前已烘焙纹理读取，否则使用 missing color。

File Type（文件类型）

输出文件类型 display driver，例如 txmake (.tex), OpenEXR (.exr)，以及 TIFF (.tif)。

File Format（文件格式）

以下是 txmake (.tex) 文件类型的底层格式：OpenEXR 或 TIFF

Data Type（数据类型）

OpenEXR data type：half 或 float。

Compression（压缩）

Compression 用于 OpenEXR 文件格式的 Compression：None、RLE、ZIP 或 PIZ

Resolution X and Resolution Y（X 与 Y 分辨率）

已烘焙纹理的分辨率。较高数值可能保留更多细节，但会增加内存使用。为获得最佳结果，请使用 2 的幂。

PrimVar ST/S

此参数选择用于定义 texture manifold 的 2D 或 1D primvar。对于 NURBS 对象，应使用 uvw。

PrimVar T

此可选参数选择第二个 1D primvar，用于定义 texture manifold。

Active UDIM（活动 UDIM）

设置后，只烘焙活动 UDIM tiles。
