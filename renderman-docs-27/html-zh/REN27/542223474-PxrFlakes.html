PxrFlakes

Source: https://rmanwiki-27.pixar.com/space/REN27/542223474/PxrFlakes
Downloaded: 2026-06-15T03:49:12+00:00

带有金属闪片和涂层的车漆。

生成一个凹凸法线，用于在高光或反射材质效果中模拟类似闪片的结果。将它连接到 PxrSurface 的 Bump Normal 参数，可以创建金属闪片车漆。

Input Parameters（输入参数）

Input Normal

如果希望在已有凹凸图案之上叠加闪片，请在此连接。未连接时会忽略。

Flake Randomness

增加闪片法线的分布范围，或者说增加它们相对于基础法线的偏离程度。

Flake Frequency

改变金属闪片特征的全局尺寸。

Flake Density

改变给定面积内金属闪片的数量。有效范围为 0（无闪片）到 1（全部为闪片）。

Flake Size

改变金属闪片的尺寸。将数值推到 1 以上会产生类似镀锌金属的效果。

Octaves

用于计算闪片的噪声倍频程数。大多数情况下，1 就足够。

Jitter

抖动闪片的位置。0.0 会产生网格状排列，1.0 看起来非常随机。

Validate Normals

确保闪片法线始终与原始法线位于同一半球。

Match C++ patterns

匹配 RenderMan 23 C++ 场景，但可能会拖慢渲染。仅在必要时使用。

Manifold

应用闪片所使用的流形。

默认情况下（没有连接流形时）为 P 在 object 空间中。如果对象没有变形，这样就可以正常工作。
如果对象会变形：

  • 添加一个 __Pref primvar 到几何体的未变形状态。

  • 创建一个 PxrManifold3D 节点，并将其连接到 PxrFlakes 节点。

  • 在流形的 Pref 字段中输入： __Pref

  • 在流形的 Coordinate System 字段中输入： object

Output Parameters（输出参数）

resultN

凹凸法线。

resultA

闪片的遮罩。这是一个黑白遮罩，例如可用于给闪片着色或改变其粗糙度。
