PxrPtexture

Source: https://rmanwiki-27.pixar.com/space/REN27/542224432/PxrPtexture
Downloaded: 2026-06-15T03:49:12+00:00

提供对逐面纹理 Ptex 文件的访问。与其他纹理节点不同，此纹理始终在表面的默认参数化上工作。不需要提供流形，而是使用名为 __faceindex 的特殊图元变量（primvar）。


PxrPtexture

Filename

纹理的文件名。必须提供此参数。

可以在文件名中引用一个常量字符串 primvar，以便在不同几何体片段之间复用纹理；该 primvar 会决定文件名约定。

  • <primstr:primvarname> - 会替换为指定常量字符串图元变量的值。例如，在带有 "const string model" ["apple"] 的苹果模型上，"/assets/<primstr:model>/diffuse.tex" 中的资产名称会展开为 "/assets/apple/diffuse.tex"。

First Channel Offset

要查找的第一个通道偏移。


Face Index Offset

如果你在一个 Ptex 文件中放入了多组数据，可以按已知数量偏移索引。

Invert Winding Order

如果面被翻转或出现瑕疵，可能是你的桥接应用反转了绕序；此选项会再次反转绕序，可能缓解这些瑕疵。


Filter

选择纹理查找期间可使用的不同重建过滤器。PxrPtexture 可用的过滤器包括：
  • 0: Nearest
  • 1: Box
  • 2: Bilinear
  • 3: Bspline
  • 4: Mitchell
  • 5: Catmullrom
  • 6: Gaussian
  • 7: Lagrangian


Blur

指定从 Ptex 文件取回的图像要模糊多少。

Mip Interpolate


选择是否在多分辨率纹理的相邻分辨率之间插值，从而让层级之间的过渡更平滑。

Missing Color


如果打开纹理时出错，则使用此颜色。

Missing Alpha

如果打开纹理时出错，则使用此 alpha 值。

Linearize

对纹理应用反向 sRGB 变换。如果你在 sRGB 空间中绘制纹理（大多数绘制软件包的默认设置），但在数据线性空间中查看数据，纹理会显得发灰。此选项会对纹理应用 sRGB 变换，使其再次呈现为视觉上线性。

Adjust Output

Color Scale

纹理中颜色值的乘数，可用于调整亮度或操控单个颜色通道。


Color Offset

对结果应用偏移，使结果颜色发生偏移。


Saturation

提高（大于 1）或降低（小于 1）纹理结果的饱和度。


Alpha Scale

alpha 通道的乘数；当 alpha 用于驱动某个参数时很有用。


Alpha Offset

alpha 通道的偏移；当 alpha 用于驱动某个参数时很有用。


Output Parameters

resultRGB

经过过滤的颜色结果。注意，所有结果都会从 startChannel offset 开始查找。

resultR

R 通道结果。

resultG

G 通道结果。

resultB

B 通道结果。

resultA

如果 alpha 存在且 resultRGB 已连接，则返回 RGB 后面的通道。如果 RGB 未连接，则返回第一个通道。
