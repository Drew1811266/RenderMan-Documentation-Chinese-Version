PxrCookieLightFilter

Source: https://rmanwiki-27.pixar.com/space/REN27/542228829/PxrCookieLightFilter
Downloaded: 2026-06-15T03:49:12+00:00

PxrCookieLightFilter 会在灯光前方投射绘制纹理。

这个灯光滤镜是以下节点的更完整版本： PxrGoboLightFilter



参数

Cookie Mode

指定 cookie 模式：

Physical (0)：cookie 的行为类似一扇彩色玻璃窗，灯光从中穿过。衰减和模糊由灯光尺寸、到灯光的距离以及与 cookie 的距离决定。

Analytic (1)：cookie 具有固定投影，并提供手动模糊和衰减控制。

Map

投射纹理的文件名。如果想控制投射边缘的模糊，请将遮罩包含在此贴图的 alpha 通道中。否则，当 Blur > 0.0 时，它会使用安装在 $RMANTREE/lib/textures/white.tex 的白色不透明 alpha 来进行模糊。

Fill Color

如果纹理的平铺模式不重复，此项指定投射矩形外部以及后方区域的颜色。当下面指定 Blur 时，投射矩形边缘会与填充颜色混合。

Width

灯光穿过的矩形宽度。

Height

灯光穿过的矩形高度。

Multiplier

Density

控制投射效果的强度。

Invert

指定在应用之前是否反转投射纹理。

Intensity

下方漫反射和镜面反射贡献的全局 Multiplier。

Diffuse

此灯光滤镜结果用于漫反射贡献的倍增值。

Specular

此灯光滤镜结果用于镜面反射贡献的倍增值。

Projection (For Analytic Mode Only)

Directional

当开启此项时，纹理会使用正交投影沿某个方向投射。关闭时，纹理会使用由 Apex 指定的焦点进行投射。

Shear X

沿 X 轴对投影进行剪切。

Shear Y

沿 Y 轴对投影进行剪切。

Apex

cookie 中心与投影中心之间的距离。仅当 Directional 关闭时适用。

Use Light Direction

如果开启此项，投影方向由光源中心的位置决定。否则，它只会跟随 cookie 的方向。警告：这不适用于 dome light 和 mesh light。


Texture Mapping

Tile Mode

指定纹理如何重复。

  • No Repeat (0)

  • Edge Extent (1)

  • Tile (2)

Invert U

仅当你想左右翻转纹理时才开启此项。默认情况下，纹理方向会与在图像查看器中看到的方向一致。


Invert V

仅当你想从下到上翻转纹理时才开启此项。默认情况下，纹理方向会与在图像查看器中看到的方向一致。请注意，灯光滤镜已经考虑了 V 方向翻转，以得到正确的默认方向。


Scale U

投射纹理在 U 方向上的缩放。

Scale V

投射纹理在 V 方向上的缩放。

Offset U

投射纹理在 U 方向上的偏移。


Offset V

投射纹理在 V 方向上的偏移。

Use Alpha

如果关闭此项，结果可能不够平滑。


Premultiplied Alpha

如果使用未预乘 Alpha 的纹理，请将此项设为 0 或关闭，以获得正确行为。

 

Blur (for Analytic Mode Only)

Blur

指定投射纹理的模糊量，范围为 0-1。该值会乘以 blurNear/blurFar 插值。当纹理不重复时，这会在投射颜色与填充颜色之间进行模糊。

S Blur Mult

S 方向上的模糊倍增值。

T Blur Mult

T 方向上的模糊倍增值。

Blur Near Dist

从 cookie 起算，模糊插值开始的位置距离。

Blur Midpoint

近端与远端之间、midValue 所在位置的距离。

Blur Far Dist

从 cookie 起算，模糊插值结束的位置距离。

Blur Near Value

模糊插值开始处的模糊倍增值。

Blur Mid Value

模糊插值中间位置的模糊倍增值。

Blur Far Value

模糊插值结束处的模糊倍增值。

Blur Exponent

模糊插值的幂指数。


Density Falloff (for Analytic Mode Only)

Density Near Dist

从 cookie 起算，密度插值开始的位置距离。

Density Midpoint

近端与远端之间、midValue 所在位置的距离。

Density Far Dist

从 cookie 起算，密度插值结束的位置距离。

Density Near Value

密度插值开始处的密度倍增值。

Density Mid Value

密度插值中间位置的密度倍增值。

Density Far Value

密度插值结束处的密度倍增值。

Density Exponent

密度插值的幂指数。


Saturation, Contrast, Tint

Saturation

结果的饱和度（0=灰度，1=正常，>1=增强颜色）。

Midpoint

对比度控制的中点。

Contrast

对比度控制（小于 1 = 降低对比度，大于 1 = 提高对比度）。

White Point

如果 (contrast > 1.0)，用于对比度控制的白点。

Tint

经过饱和度、对比度和钳制后所得颜色的色调。

Combine Mode

Combine Mode

mult：所有滤镜的结果会相乘。

max：使用所有滤镜中的最大结果。这最适合灰度灯光滤镜。

min：使用所有滤镜中的最小结果。这最适合灰度灯光滤镜。

screen：类似于 max 操作，但会以更平滑的方式组合渐变。这最适合灰度灯光。

附加在灯光上的灯光滤镜会按各自的组合模式分组。同一组中的灯光滤镜会一起执行，并按组合模式进行组合。各组按以下顺序执行（max、min、screen，然后 mult），并且会相互相乘；这意味着 mult 组中把结果变黑的滤镜会将所有其他滤镜清零。
