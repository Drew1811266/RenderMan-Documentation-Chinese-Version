PxrMarschner 是一种用于渲染真实毛发、皮毛和纤维的材质。

这是 Pixar 多部动画长片中使用的同一材质和设计，现在提供给用户用于渲染自己的场景。其控制项精简且对艺术家友好。该材质也具备物理合理性，因此艺术家可以先得到真实可信的结果，再根据艺术指导细化选项。

如果想深入了解该材质的技术细节，可以查看  Pixar Technical Memo 15-02 "A Data-Driven Light Scattering Model for Hair"  ，作者为 Leonid Pekelis、Christophe Hery、Ryusuke Villemin、Junyi Ling，可获得更多信息。本文档会结合示例说明各参数，方便艺术家立即开始创作！

下方示例会说明更改参数如何影响毛发的外观。




Diffuse Parameters（漫反射参数）

漫反射模型有两种选择：Zinke 和 Kajiya。默认设置为 Zinke 漫反射模型，用于双向散射。参见 Importance Sampling for Physically-Based Hair Fiber Models。

Diffuse Gain

此参数是 Diffuse Color 的乘数。0.0 表示没有漫反射颜色，1.0 表示完整效果。默认值为 0.0，可为深色到中等色度毛发提供非常真实的效果，但渲染成本可能更高。对于铂金金发或灰发等较浅色毛发，可能需要使用更高数值。

Diffuse Color

这是毛发本身的漫反射颜色。即使对于金发，它也比大多数真实毛发的其他颜色参数更暗。为了获得最佳真实感，建议它与 Secondary Specular Color 使用相同颜色，只调整 Gain。默认值为中灰。


Specular Parameters（高光参数）

大部分控制项都位于 Specular Parameters 下。它们都以某种方式对应下面的说明。

Marschner 镜面模型由三个镜面波瓣组成，使用上图中方向箭头所示的多个镜面传输路径：

  • Primary Specular（主高光）： R 表示 “reflect（反射）”，即光照射到毛发表面并反射。这是强烈的 primary、无色镜面反射。

  • Secondary Specular（二级高光）： TRT 表示 “transmit（透射）”、“reflect（反射）”和“transmit（透射）”，即光穿过毛发表面，然后从毛发内部反射，并从毛发正面射出。这是带颜色的 secondary specular。

  • Transmit Specular（透射高光）： TT 表示 “transmit（透射）”和“transmit（透射）”，即光透入毛发表面，然后从另一侧射出。这是会响应边缘光的带色透射。


该 Specular Gain 控制项允许艺术家出于艺术目的打破能量守恒。如果输入的值相加大于 1.0，则毛发不再能量守恒。只有在对外观不满意（太“hot”）或希望从物理真实着色起步时，这一点才重要。

默认 Gain 现在为 1.0；之前版本可能设为 0.33。



Primary Specular Gain

Marschner 镜面的 Gain，对应 R 波瓣（Marschner 镜面）。它类似 clearcoat，其中镜面反射相当锐利、有光泽，并且通常不带色。该镜面反射是直接反射，不会被任何体积衰减着色。修改 R 波瓣也有助于让毛发看起来“湿润”。

Secondary Specular Gain

Marschner 镜面的 Gain，对应 TRT 波瓣（Marschner 镜面）。它是更粗糙且带颜色的镜面反射，会两次穿过毛发纤维体积。


Transmit Specular Gain

Marschner 镜面的 Gain，对应 TT 波瓣（Marschner 镜面）。它是一种透射型（折射）镜面反射，带有一定体积衰减。它会穿过纤维体积一次。较高数值可能增加噪点。


Glint Gain

glints 的 Gain。这是 Glints 效果的乘数，工作方式与其他 Gain 控件类似。



Primary Specular Color

用于给 Primary Specular（ R 波瓣）着色的镜面颜色。对于自然毛发和皮毛，这应始终为白色。

Secondary Specular Color

镜面颜色，用于 TRT 和 glints 波瓣的镜面颜色。

由于它表示毛发纤维内部的两条路径，因此应比 Transmit Specular Color 更暗且饱和度更高；严格来说，Transmit Specular Color 是 Secondary Specular Color 的平方根。为了获得更真实的毛发，它应与 Diffuse Color匹配。

Transmit Specular Color

镜面颜色，用于 TT 波瓣。下方示例设置为 Transmit Specular Gain 1.0，其他所有 Gain 为 0.0

Primary Cone Angle

用于 R 波瓣的锥角（theta，单位为度）。角度越宽，高光越柔和、越宽。

Secondary Cone Angle

用于 TRT 波瓣的锥角（theta，单位为度）。角度越宽，高光越柔和、越宽。

Transmit Cone Angle

用于 TT 波瓣的锥角（单位为度）。角度越宽，高光越柔和、越宽。注意，此参数使用高 Gain 会产生柔和且浅色的毛发，但也会生成相当多噪点。在较大角度下，这可能增加渲染时间。

Specular Offset

锥角偏移，单位为度，可用于沿毛发方向移动镜面结果。

Refractive Index

毛发的折射率（IOR）。较高数值会增加镜面性，类似于它在不透明表面上的工作方式。这可能使毛发纤维看起来近似金属。

Fresnel Mix

此控制项用于平衡沿毛发方向的 Fresnel 能量衰减。 

Glint Width

此参数会平滑 Secondary Specular（TRT）和 Glints 分布。对于真实毛发，该值通常应保持在 10 到 25 之间。

Eccentricity

这是毛发的横截面长宽比，对于真实毛发通常在 0.85 到 1.0 之间。它最常用于渲染卷发。（注意，这取决于毛发法线；应通过 PrimVar 提供用户法线，而不是指定 N。）


Glow

Glow 可用于创建看起来会在场景中发光或“glow”的毛发纤维。 

Gain

Glow gain 或 weight。下方从典型黑发开始，将 Gain（使用白色）增加到 0.5，然后增加到 1.0。

Color

控制材质的自发光颜色或 glow。下方是 Color 参数接入各种带纹理输入的示例。

Globals（全局参数）

本节提供整个材质的全局控制。Presence 也是一个全局参数，会遮罩或“镂空”所有波瓣，就像遮罩值为 0.0 的位置对象并不存在一样。


Specular Energy Compensation

此控制项允许漫反射结果变暗，因为能量被镜面响应占用，剩余能量用于照亮漫反射波瓣。设为 1.0 对应物理效果。默认值为 off（艺术控制），且不守恒能量。


Presence

在此处连接遮罩，可将镂空 pattern 应用于对象。Presence 定义为二值（0 或 1）函数，但可以接受连续值来对形状进行抗锯齿。适用于创建树叶和其他薄而复杂形状等镂空。

此值应为 0 或 1。介于 0 和 1 之间的值会产生不需要的噪点！


Eccentricity Direction

此参数可用于覆盖几何法线，以控制 Eccentricity 参数。


Shadow Color

指定阴影颜色。此参数可用于出于艺术指导目的伪造阴影颜色。注意，这会创建“透明阴影”，并随着毛发之间追踪更多光线而增加渲染时间。它也可能影响对象之间的全局照明。
