PxrProjectionStack

Source: https://rmanwiki-27.pixar.com/space/REN27/542224392/PxrProjectionStack
Downloaded: 2026-06-15T03:49:12+00:00

此节点用于合成多个 PxrProjectionLayer 节点，但也接受来自其他 Pattern 节点的连接。

Input Parameters

Layers Mode

控制哪些图层处于启用状态。使用 Solo layer 会停用所有 On layers。

  • 0: Off
  • 1: On
  • 2: Solo


Layers RGB

将 PxrProjectionLayer Pattern 的 resultRGB 输出连接到这里。

Layers A

将 PxrProjectionLayer Pattern 的 resultA 输出连接到这里。

Channels [0-16] RGB

这些参数都必须连接到 PxrProjectionLayer.outChannelsRGB 插口。参数编号必须匹配 PxrProjectionLayer 的 layer index。

Channels Aov Names

此数组包含一个（可选）AOV 名称数组，用于指定通道应输出到哪些 AOV。无效的 AOV 名称会被安全忽略。AOV 名称顺序必须与通道顺序匹配，也就是说，如果通道 2 包含镜面反射颜色数据，则 channelsAovNames[2] 应包含 "color projectedSpecularColor"。最多可以引用 32 个 AOV。

Output Channels Idx

此数组包含索引，用于将完全合成后的通道路由到某个 channelOutX 输出。这允许你路由全部通道，或只路由其中一个子集，以驱动其他下游节点。默认情况下，每个 outputChanIdx member 的值为 -1，且不会路由任何通道。

Output Parameters

resultRGB

钳制后的颜色结果。

resultR

来自 resultRGB 输出的 R 通道。

resultG

来自 resultRGB 输出的 G 通道。

resultB

来自 resultRGB 输出的 B 通道。

channelOut[0-7]

最多 8 个图层的可分配输出。参见 Output Channels Idx 进行配置。
