PxrColorGrade

Source: https://rmanwiki-27.pixar.com/space/REN27/542223051/PxrColorGrade
Downloaded: 2026-06-15T03:49:12+00:00

此 pattern 支持精细调校的调色，包括分别控制阴影、中间调和高光。


PxrColorGrade

Input Color

用作调色源的输入。 

Global

Enable（启用）后可控制整个源输入上的 global（全局）颜色校正

Saturation

控制输入源的 global（全局）饱和度

Contrast

控制输入源的 global（全局）对比度

Gamma

控制输入源的 global（全局）gamma

Gain

控制输入源的 global（全局）增益

Offset

控制输入源的 global（全局）偏移

Shadows

Shadows

Enable（启用）输入源中较暗区域/阴影的颜色校正

Saturation

控制输入源的阴影饱和度

Contrast

控制输入源的阴影对比度

Gamma

控制输入源的阴影 gamma

Gain

控制输入源的阴影增益

Offset

控制输入源的阴影偏移


Mid-Tones

Mid-Tones

Enable（启用）输入源中间调的颜色校正

Saturation

控制输入源的中间调饱和度

Contrast

控制输入源的中间调对比度

Gamma

控制输入源的中间调 gamma

Gain

控制输入源的中间调增益

Offset

控制输入源的中间调偏移



Highlights

Highlights

Enable（启用）输入源中高光的颜色校正

Saturation

控制输入源的高光饱和度

Contrast

控制输入源的高光对比度

Gamma

控制输入源的高光 gamma

Gain

控制输入源的高光增益

Offset

控制输入源的高光偏移


Clamp

Clamp Black

Enable（启用）黑色钳制

Clamp White

Enable（启用）白色钳制

Advanced

Contrast Pivot

调整对比度枢轴值

Shadows Range Min

调整 Shadows 的最小值

Shadows Range Max

调整 Shadows 的最大值

Highlights Range Min

调整高光的最小值

Highlights Range Max

调整高光的最大值
