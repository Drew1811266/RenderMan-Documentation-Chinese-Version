LamaConductor

Source: https://rmanwiki-27.pixar.com/space/REN27/542219348/LamaConductor
Downloaded: 2026-06-15T03:49:12+00:00

此材质常用于创建闪亮或金属表面。

大多数金属属性仅使用此材质即可创建；当它与 Lama 分层工具节点组合时，也可与其他涂层配合，用于创建金属车漆等材质以及类似的分层表面。

Conductor Properties

Conductor Tint

以物理方式为材质着色的正确做法，是使用 Fresnel Mode 选择项下的控制。

真正的金属材质会保留白色。这里只应做细微调整，以避免失去全部物理性，因为此控制的结果会与反射光线相乘。 


Fresnel Mode

这允许美术师在两种方式之间选择：依赖所选颜色的 Artistic 控制，或依赖物理 Index of Refraction（折射率）设置的 Scientific 方法。结果可以相同，只是方法不同。

Artistic

  • Reflectivity - 表面法线或正对视角方向上的颜色。

  • Edge color - 表面边缘或掠射角视角中的颜色。


Scientific

  • IOR - 使用 Eta 或金属表面的 IOR 测量值定义的、表面法线或正对视角方向上的颜色。

  • Extinction - 使用称为 Kappa 或消光系数的测量值，描述表面颜色如何从正对视角变化到掠射视角。白色表示无变化。


Roughness

微面元或细小粗糙度会以产生模糊结果的方式散射光线。粗糙度越低，材质越像镜面并且更光滑/顺滑。较高值会直接呈现漫反射外观。

Roughness 示例：

1.0


Normal/Bump

在这里提供一个信号（纹理或程序图案），用于在表面上创建凹凸，以“伪造”小凸起或划痕等表面细节。这意味着美术师无需对模型中这些繁琐且经常重复的部分进行建模。

Rotation : 0

Direction

使用来自图案节点的信号覆盖内置表面方向；对于拉丝或带纹理的金属，这也可称为“拉丝方向”。

Rotation

围绕表面法线将方向（可能由上方定义）旋转 0 到 360 度。


Rotation Example : 0 - 1

Advanced Parameters

Exterior IOR

定义外部介质 IOR，可以是空气（默认）、水，也可以是使用 Stack 节点在上方形成的清漆等分层材质。

Energy Compensation

允许向结果添加能量，因为微面元或非常粗糙的材质可能会损失能量，并使结果人为变暗。

Surface Mollification

根据 curvature 定义 roughness，并可能有助于减少渲染中的亮点噪声或亮像素/伪影。

Motion Mollification

根据对象运动定义 roughness，并可能有助于减少渲染中的亮点噪声或亮像素/伪影。

Lobe Name

为生成的 Light Path Expression (LPE) 定义 Arbitrary Output Variable (AOV) 名称

Matte

定义最终 lobe weight 的 lobe output 输出到 Arbitrary Output Variable (AOV) 时使用的名称
