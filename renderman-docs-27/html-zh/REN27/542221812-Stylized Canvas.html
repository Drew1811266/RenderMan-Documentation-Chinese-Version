Stylized Canvas

Source: https://rmanwiki-27.pixar.com/space/REN27/542221812/Stylized+Canvas
Downloaded: 2026-06-15T03:49:12+00:00

Canvas 是一个 Display Filter，用于在其他 Stylized Looks layers（Lines、Toon、Hatching）之间对纹理、颜色或渲染中任何其他现有 layer 进行 layering。

Result（结果）

  • Disable（禁用）

  • Canvas Final（Canvas 最终结果）

Canvas AOV

用于此 Layer 的 color AOV 名称。此 AOV 必须在 Maya/Katana/Houdini 中设置，否则这个 Canvas DisplayFilter 会退出。

参见 示例演示视频 以及 Canvas + Hatching：

Canvas Color

alpha 外部的背景颜色。

Canvas Texture

为 alpha 外部提供 BG texture；如果留空，则使用 BG Color

Ignore Alpha

忽略 alpha，将 BG Color 或 Texture 放到 Canvas 下方的所有内容上。
