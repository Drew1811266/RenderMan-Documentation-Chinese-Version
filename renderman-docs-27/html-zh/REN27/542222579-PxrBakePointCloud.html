PxrBakePointCloud

Source: https://rmanwiki-27.pixar.com/space/REN27/542222579/PxrBakePointCloud
Downloaded: 2026-06-15T03:49:12+00:00

PxrBakePointCloud 节点 可作为直通节点使用，用于将 Pattern 网络烘焙到 3D 点云；这是 烘焙 工作流的一部分。

Parameters（参数）

Input Color（输入颜色）

这是要烘焙的 Pattern 网络的传入结果。此连接需要 color (RGB)。

Input Float（输入 Float）

这是 Pattern 网络的传入结果，其中输出为 float 或 scalar 值。这对于烘焙某些程序化噪声（例如 PxrVoronoise）很有用。

Filename（文件名）

生成的已烘焙点云的输入/输出文件名。

此参数是必需的。

Bake Mode（烘焙模式）

启用或禁用点云烘焙。

File Type（文件类型）

输出文件类型显示驱动：PTC (.ptc)

Density（密度）

Density（密度）表示世界空间中每单位的烘焙点数量。较大的值会创建更多点。

Coordinate System（坐标系统）

输出点云坐标空间：

  • world（世界）
  • object（对象）
  • camera（相机）
  • ptex（输出与 Ptex 兼容的点云 (.ptc) 文件，用于 ptxmake 工具。

Ptex 要求几何体具有 “__faceindex” primvar。
