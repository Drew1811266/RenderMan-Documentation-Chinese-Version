PxrLayeredTexture

Source: https://rmanwiki-27.pixar.com/space/REN27/542224051/PxrLayeredTexture
Downloaded: 2026-06-15T03:49:12+00:00

此节点允许使用 mask 进行纹理分层。这适合向纹理结果中添加贴花、标签、损伤等效果。

与所有纹理类节点一样，此节点接受一个 manifold，用来描述应用纹理的 2D 或 3D domain。如果未连接 manifold，默认行为是在几何体上定义的 s,t domain 上应用纹理，这通常称为 "UV Layout"。

提供对纹理文件的访问。Texture atlas format（纹理图集格式）文件可以按 UDIM (Mari)、ZBrush 或 Mudbox 格式读取。此 plugin 也支持 Rtx procedural texture plugins。

Mute

在合成链中忽略此节点。

Texture Overlay

可在此连接一个 overlay（叠加）输入，从而串联多个 texture nodes。

Mask

Mask Value

用于控制此纹理在合成链中可见性的标量值、pattern 连接或另一个灰度纹理。建议在下面的参数中使用 Texture connections（纹理连接）。

Mask Texture

用于控制此节点可见性的 RGB 纹理。

Mask Channel

从上面的纹理输入中选择 mask channel 的 channel offset。

Mask Atlas Style

如果指定 atlas style，则 filename 必须包含 _MAPID_；它会根据 surface parameterization 和 atlasStyle 自动替换。对于第一个 UV tile，Mari UDIM 示例会是 Diffuse1001.tex，其中 "1001" 会替换纹理 filename 中的 _MAPID_。

Missing Mask

当以上某个连接或参数无法读取或使用时，使用此 value 或 pattern 输入。这是一个有用的 fallback。

Texture Input Parameters

Filename

纹理的 filename。必须提供此参数。

要使用图像序列，通常会通过字符串替换构造纹理贴图引用，在你偏好的表示方式中嵌入帧号。示例如下：

(Maya) $PROJPATH/sourceimages/flapping.<f4>.tex

(Katana) '/images/flapping.%04d.tex' % frame

(Houdini) 在文件名中使用 $F、$F3 或 Houdini expressions：`padzero(3, $F)`

最好避免在文件路径中使用特殊字符、变音符号和空格，因为它们可能无法正确解析，导致纹理渲染失败。

First Channel Offset

Offsets（偏移）要查找的第一个 color channel。

Atlas Style

如果指定 atlas style，则 filename 必须包含 _MAPID_；它会根据 surface parameterization 和 atlasStyle 自动替换。对于第一个 UV tile，Mari UDIM 示例会是 Diffuse1001.tex，其中 "1001" 会替换纹理 filename 中的 _MAPID_。

在所有 atlas texture workflows 中，用户应避免使用负的纹理坐标

Invert T

反转用于纹理查找的 t 参数。

Manifold

提供应用纹理的 domain。默认使用 s,t。可在此提供一个 manifold 来调整纹理放置，例如： PxrManifold2D

Texture Filtering（纹理过滤）

Filter

选择可在纹理查找期间使用的不同重建过滤器。PxrTexture 可用的过滤器包括：

  • 0: Nearest

  • 1: Box

  • 2: Bilinear

  • 3: Bspline

  • 6: Gaussian

  • 7: Lagrangian

Blur

指定从纹理文件取回的图像要模糊多少。

Mip Interpolate

选择是否在多分辨率纹理的相邻分辨率之间插值，使层级之间的过渡更平滑。

Missing Color

如果打开纹理时出错，则使用此颜色。

Missing Alpha

如果打开纹理时出错，则使用此 alpha。

Advanced Settings

MipBias

Bias（偏置）渲染时的 MIP 选择，可选择更高分辨率（负整数）或更低分辨率（正整数）。

MaxResolution

Clamp（钳制）MIP 选择到首选最大 level。根据你的选择，这可能阻止最高分辨率参与渲染。

OptimizeIndirect

在 indirect hits 上将 texture filter 设置为 Box（开销较低），而不是使用为其他可见效果选择的 filter。

Color Correct

Linearize

对纹理应用反向 sRGB transform。如果你在 sRGB space 中绘制纹理（大多数绘制软件包的默认设置），但在数据线性空间中查看数据，纹理会显得发白。此选项会将 sRGB transform 应用于纹理，使其再次呈现为视觉上线性。

Color Tint

纹理中颜色值的 multiplier，可用于调整亮度或操控单个 color channel

Color Offset

对结果应用 offset，使结果颜色发生偏移

Output Parameters（输出参数）

resultRGB

经过过滤的颜色结果。注意，所有结果都会从 startChannel offset 开始查找。

resultR

R channel 结果

resultG

G channel 结果

resultB

B channel 结果

resultA

如果 alpha 存在且 resultRGB 已连接，则返回 RGB 后面的 channel。如果 RGB 未连接，则返回第一个 channel。
