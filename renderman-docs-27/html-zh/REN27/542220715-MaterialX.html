The Open Chess Set
( 由 Moeen and Mujtaba Sayed 创作，由 SideFX 贡献给 MaterialX，并在 Solaris 中通过 RenderMan for Houdini 渲染 )


MaterialX 是一个 ASWF 标准，用于在渲染器之间定义和交换材质。它提供了一组 Physically Based Shading Nodes 的定义，以及一组 pattern nodes（如 texture nodes、procedural noise 等），可以用来搭建材质。每个支持 MaterialX 的渲染器都会接收这些材质定义，然后直接渲染它们，或将它们转换为内部表示。由于世界上有很多渲染器，并且各自从不同角度处理渲染这一挑战，MaterialX 并不保证不同渲染器之间能做到像素级匹配。 

MaterialX 只为每个渲染器提供指导，而每个渲染器都会尽力以自己最合适的方式匹配材质意图。

RenderMan 通过 USD 和 Hydra 支持 MaterialX 。USD 和 Hydra 会接收 MaterialX 材质定义，并将它们分配给相关几何体。当材质流经 RenderMan 的 Hydra Renderer Delegate 时，会经过 MaterialX ShaderGen 系统，转换为原生 RenderMan 材质。

RenderMan 支持 OSL patterns，因此你在 MaterialX networks 中使用的所有 pattern 都会通过 OSL 转换到 RenderMan，并且应当完全按预期运行。

你在 MaterialX graphs 中使用的 Physically Based Shading Nodes 都会转换为 PxrSurface ，然后传递给 RenderMan。PxrSurface 可以表示 MaterialX 定义的大部分内容。由于每个渲染器都不同，对于指定的 MaterialX network，RenderMan 得到的图像可能与另一个渲染器使用同一 network 得到的图像不同。目前存在的一个限制是，我们还不支持通过 MaterialX 进行材质分层。
