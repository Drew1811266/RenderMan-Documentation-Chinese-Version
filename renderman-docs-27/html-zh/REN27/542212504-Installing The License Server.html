Installing The License Server

Source: https://rmanwiki-27.pixar.com/space/REN27/542212504/Installing+The+License+Server
Downloaded: 2026-06-15T03:49:12+00:00

Pixar License Server 可以在 RenderMan Installer 中与 RenderMan Pro Server 和 RenderMan for Maya/Katana/Houdini 一起下载并安装。或者，也可以从 renderman.pixar.com 论坛网站的 Downloads 区域下载 License Server 的独立安装器，并手动运行。（你必须已登录）

用于获取许可证的机器需要能够与我们的许可证服务器通信。简单地说，如果它无法访问我们的论坛，就无法取得许可证。


要从集成安装器软件包安装许可证服务器，请先使用论坛名称和密码登录，并提供你已购买的浮动许可证序列号。

登录后，如果你从未在当前运行安装器的机器上设置过许可证服务器，安装器会给出警告。

这表示安装器即将把你的许可证服务器绑定到当前运行安装器的主机。

如果这台机器 不是 你希望运行许可证服务器的机器，请按 Cancel 按钮，并在你希望指定为许可证服务器的机器上重新运行安装器。

如果你决定把这台机器作为许可证服务器，安装器会联系 Pixar 来获取你的许可证文件。默认情况下，它会把许可证文件放在：

  • Windows: C:\Program Files\Pixar\pixar.license

  • macOS: /Applications/Pixar/pixar.license

  • Linux: /opt/pixar/pixar.license

注意：在 macOS 和 Windows 上，操作系统可能会要求输入管理员密码。在 Linux 上，安装器必须以 sudo 或 root 用户身份运行。

随后安装器会显示可供下载的软件包列表。

由于我们只关心安装许可证服务器，所以将 取消选中 安装器可能已经自动为我们选择的所有其他软件包。注意：你可能需要选中 "Show All" 复选框，才能列出 Pixar License Server 软件包。然后我们只选择：

Pixar License Server 23.0 (XXXXXXX)
