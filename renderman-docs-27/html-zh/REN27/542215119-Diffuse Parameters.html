在下方参数中，有些参数可在连接到 PxrLayer 后，通过 Input Material 被覆盖，或通过 PxrLayerMixer覆盖。 PxrLayerSurface 设计用于更清楚地展示哪些参数不能在层中被覆盖：它只包含全局参数。当你确定要进行分层时，建议使用这种材质。这些设置的结果保持不变。

  • Bold Face 参数是可分层的，可以按层覆盖。

  • Italicized 参数不能分层或覆盖。所有层都会全局遵循这些参数。例如：选择 GGX 作为 Specular 模型时，所有层在该参数上都会使用 GGX。

Diffuse Parameters 

Diffuse Parameters 控制基础漫反射的外观。它们常用于定义不透明对象的主要颜色属性；木纹纹理、标签文字、波点等，都可以在这里连接。

该波瓣默认开启。

Gain

Gain 是应用到 Diffuse 参数上的权重。你也可以用另一个 pattern 驱动它，用来表现褪色、潮湿等效果（液体会让表面变暗）。

Color

Color 通常用于连接纹理或 pattern，为不透明对象生成颜色。
例如，木材颜色纹理就会接到这里。


Roughness

Diffuse roughness（漫反射粗糙度）用于模拟干陶土或尘土这类粉末感表面。 
当 Roughness 为 0.0 时，PxrSurface 使用 Lambertian 模型计算 Diffuse 响应。 
当 Roughness > 0.0 时，则改用 Oren Nayar 模型。


Exponent

Diffuse exponent（漫反射指数）控制漫反射衰减。它是一个幂指数。数值越高，衰减越锐利。 
例如，在月球表面上，可以设置较高的 Exponent 来产生更锐利的衰减。

Diffuse exponent（漫反射指数）只适用于 Lambertian 漫反射，也就是 Roughness 为 0.0 的情况。将漫反射指数与 Oren Nayar Roughness 结合没有意义，因此当 Roughness > 0.0 时会被忽略。


Bump

Bump mapping（凹凸贴图）是一种很好的方式，可以用着色来伪造物理细节的外观。如果未设置，它会使用本页底部 Properties 部分指定的全局 bump normal（凹凸法线）。


Double Sided

如果开启，该 Diffuse 波瓣会照亮表面的两侧，也就是说，法线背向相机的表面（二维对象）也会被照亮。
这样背面会被着色，而不是显示为黑色。该选项默认关闭。


Use Diffuse Color

这仅在 Double-Sided 开启时适用。默认情况下，此选项开启，以便将 Diffuse Color 用作背面颜色。


Back Color

这仅在 Double-Sided 开启时适用。当 Diffuse Color 关闭时，此参数会设置背面颜色（背侧的颜色）。默认情况下，它使用 Diffuse Color；选择不同颜色则可以让二维对象的背侧呈现不同外观。



Transmit Gain

这仅在 Double-Sided 开启时适用。这里不是使用 Diffuse Gain 来控制漫透射的强度，而是使用 Transmit Gain。如果它为 0.0，则该效果关闭。下面示例中的 Transmit Color 为亮蓝色。


Transmit Color

这仅在 Double-Sided 开启时适用。此参数设置透射颜色，它可以不同于 Diffuse 颜色或背面颜色。如果 Transmit Gain 为零，则忽略此设置。该效果适用于树叶或纸张等薄对象。下面示例中，在对象内部放置了一盏灯，并且随着光线透过表面，可以看到一些内部文字。
