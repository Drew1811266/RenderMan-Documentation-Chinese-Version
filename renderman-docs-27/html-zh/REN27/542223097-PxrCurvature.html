PxrCurvature

Source: https://rmanwiki-27.pixar.com/space/REN27/542223097/PxrCurvature
Downloaded: 2026-06-15T03:49:12+00:00




这是一个简单而实用的 Pattern 节点，用于创建边缘效果，以及基于曲率的结果，例如边缘磨损和与其他表面接触的物体。

Parameters（参数）

Num Samples

控制用于计算曲率的采样数量。更多采样可以降低噪声，但代价是增加渲染时间。

Max Distance

采样在终止前会行进一定距离。较低值往往产生更锐利的变化；较大值会扩散效果，并可能丢失细节。随着距离增大，其他物体可能遮挡结果。内部空间是观察这种效果最明显的方式。0.0 表示 "infinite"（无限），本质上就是你的场景范围。

Trace Set

你可以使用 Trace Sets 将其他物体包含进被追踪的光线中。当你需要较大的 Max Distance，但不希望其他物体遮挡结果时，这很有用。注意，这不是排除，而是包含指定集合。

Sample Distribution

可用值依次为 uniform 和 cosine (Lambertian)。


说明

Uniform

光线不会在任何特定方向上加权。

Cosine

光线会根据下面的 Cosine Spread 参数，沿法线方向加权。


Cosine Spread

使用 cosine 分布时的光线采样扩散。值 1.0 会给出完美的 Lambertian 分布。较低值会沿曲线收缩或“收紧”效果；较大数值会扩散效果。

Bias

Bias 会影响曲率的衰减，并且这种衰减是线性的，会随着离着色点的距离增加而降低。Bias 可用于让更远的采样或多或少地计入结果，而附近点具有更高权重。

  • 越接近 0.0，只有最靠近着色点的点会产生贡献。

  • 使用默认值 0.5 时，靠近 Max Distance 的任何点贡献较少，靠近着色点的点贡献较多。 

  • 越接近 1.0，所有采样点贡献越多。正好为 1.0 时，所有距离上的采样点会等量贡献。

Convex Threshold

增大此参数会侵蚀结果，缩小可见效果。

Concave Threshold

增大此参数会侵蚀结果，缩小可见效果。

Output

Output Gain

结果曲率效果的乘数（没有曲率的区域自然不受影响）

Output Type

设为 "Concave" 时，只输出凹曲率。设为 "Convex" 时，只输出凸曲率。"Both" 会同时显示凸曲率和凹曲率。


Monochrome Output

对于凹和凸结果，输出都是简单的白色或灰度值。Off 会将 Convex 输出为红色，将 Concave 输出为绿色

Clamp Output

默认情况下，值会被钳制到 0-1 范围。如果需要扩展范围（也许配合增益控制，或用于驱动另一个参数连接），可以禁用此选项。
