PxrFractal

Downloaded: 2026-06-15T03:49:12+00:00

与所有纹理类节点一样，此节点接受一个流形，用于通过 UV 执行 2D 放置，或通过 3D 投影应用分形纹理。默认行为是：如果没有连接流形，就在 3D 中基于 P 进行应用。


输入参数

Surface Position

噪声可以基于 Current Position 或 Undisplaced Position （位移之前的表面位置）计算。

如果希望置换图案和着色图案匹配，请使用 Undisplaced Position。

Layers

使用的噪声层数量。更多层会逐层添加更多噪声，产生更细致的图案。

Frequency

设置噪声层的最低（起始）频率。

Lacunarity

用于确定每个后续噪声层频率的尺度。较小数值会使各层在频率上更接近；较大数值会使它们相距更远。

Dimension

频率指数。此值会使用公式 1/f^(3-2*Dimension) 缩放每个后续噪声层的幅度；其中 f 是给定层所用噪声的频率。它称为维度，因为该值与分形维度相关。数值越大越粗糙，数值越小越平滑。

Erosion

频率指数范围。负值会平滑低值区域，正值会平滑高值区域。由 Dimension 指定的频率指数本身会按噪声幅度发生偏移；此值用于调整该偏移。这决定了所谓的分形交叉尺度。0 会产生均匀外观。 

Variation

此值控制获得哪一种外观变化，而不会改变由其他控制项决定的外观属性。因此，例如可以通过连接到 Time 来为外观制作动画，或通过设置不同数值来创建多个独特的外观实例。

Mode

对噪声倍频程执行 modeOp 的不同方式

Manifold

应用噪声所使用的流形。默认为 P。


Adjust Output

Color Scale

用于纹理颜色值的乘数，可用来调整亮度或操控单个颜色通道

Color Offset

对结果应用偏移，从而移动结果的颜色

Float Scale

用于 Alpha 通道的数值乘数；在用该通道驱动参数时很有用

Float Offset

用于 Alpha 通道的偏移；在用该通道驱动参数时很有用

输出参数

resultF

分形纹理作为 float 时的结果。

resultRGB

作为颜色的分形纹理。
