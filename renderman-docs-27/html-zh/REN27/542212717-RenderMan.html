Pixar 的核心渲染技术

30 多年来，RenderMan 一直站在视觉特效革命的前沿；今天的 RenderMan 是一款高性能渲染器，专为处理可以想象到的最复杂 3D 场景而构建。

借助针对基于物理的渲染优化的新一代先进光线追踪框架，RenderMan 能为任何制作管线提供无与伦比的灵活性。

关于 RenderMan Pro Server

PhotoRealistic RenderMan (PRMan) 是一个渲染系统，可根据 3D 场景描述信息生成高质量 2D 图像；这些信息通常由 Autodesk Maya 等内容创作应用程序的插件生成。场景描述通过 RenderMan 接口（RenderMan Interface）创建，通常由以下内容组成：

  • 对场景中存在的几何体的描述，例如茶壶、外星人和球体；

  • 对函数的引用，这些函数描述几何体应如何着色，从而创建玻璃、木材和皮肤等材质效果；

  • 光源的规格说明（类型、位置和朝向）；以及

  • 用于渲染最终画面的虚拟摄影机规格说明。

场景描述还可以指定输出图像的目标位置（例如文件或帧缓冲区），以及控制渲染系统运行的各种参数：例如，为避免 走样（aliasing）而投入多少计算量。

场景不必由用户使用编辑器手动创建，尽管它们可以具备人类可读性。RenderMan 为流行的内容创作应用程序提供插件，以支持对艺术家友好且高效的工作流。RenderMan 的插件套件让艺术家能够专注于创作，而不是技术细节。

RenderMan 文档以教程和说明的形式涵盖了其中许多功能和工作流，并且还在不断扩展。
