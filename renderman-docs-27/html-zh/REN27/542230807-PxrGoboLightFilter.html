PxrGoboLightFilter

Source: https://rmanwiki-27.pixar.com/space/REN27/542230807/PxrGoboLightFilter
Downloaded: 2026-06-15T03:49:12+00:00

PxrGoboLightFilter 会在灯光前方投射绘制纹理。

这个 light filter 是以下节点的简化版本： PxrCookieLightFilter。



Parameters（参数）


Map

投射纹理的文件名。如果想控制投射边缘的 Blur（模糊），请将 mask 包含在此贴图的 alpha channel（alpha 通道）中。否则，当 blur > 0 时，它会使用安装在 $RMANTREE/lib/textures/white.tex 的白色不透明 alpha 来进行模糊。

Fill Color

如果纹理的 tile mode（平铺模式）不重复，此项指定投射矩形外部以及后方区域的颜色。当下面指定 Blur 时，投射矩形边缘会与填充颜色混合。

Width

灯光穿过的矩形宽度。

Height

灯光穿过的矩形高度。


Multiplier

Density

控制投射效果的强度。

Invert

指定在应用之前是否反转投射纹理。

Intensity

下方 diffuse contribution（漫反射贡献）和 specular contribution（镜面贡献）的全局 Multiplier。

Diffuse

此 light filter 结果用于 diffuse contribution（漫反射贡献）的 multiplier。

Specular

此 light filter 结果用于 specular contribution（镜面贡献）的 multiplier。


Texture Mapping

Tile Mode

指定纹理如何重复。

  • No Repeat (0)

  • Edge Extent (1)

  • Tile (2)

Scale U

投射纹理在 U 方向上的缩放比例。

Scale V

投射纹理在 V 方向上的缩放比例。

Offset U

投射纹理在 U 方向上的 offset。这只是沿 U 方向移动图像。


Offset V

投射纹理在 V 方向上的 offset。这只是沿 V 方向移动图像。


Premultiplied Alpha

如果使用未预乘 Alpha 的纹理，请将此项设为 0 或关闭，以获得正确行为。


Combine Mode

mult：所有滤镜的结果会相乘。

max：使用所有滤镜中的最大结果。这最适合灰度灯光滤镜。

min：使用所有滤镜中的最小结果。这最适合灰度灯光滤镜。

screen：类似于 max 操作，但会以更平滑的方式组合渐变。这最适合灰度灯光。

附加在灯光上的 Light filters 会按各自的 combine mode 分组。同一组中的 Light filters 会一起执行，并按 combine mode 进行组合。各组按以下顺序执行（max、min、screen，然后 mult），并且会相互相乘；这意味着 mult 组中把结果变黑的 filter 会将所有其他 filter 清零。
