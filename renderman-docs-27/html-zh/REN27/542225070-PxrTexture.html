PxrTexture

Source: https://rmanwiki-27.pixar.com/space/REN27/542225070/PxrTexture
Downloaded: 2026-06-15T03:49:12+00:00

PxrTexture 是用于加载 textures 的主节点；与所有 texture style nodes 一样，此节点接受一个 manifold，该 manifold 描述用于应用 texture 的 2D 或 3D domain。

如果没有连接 manifold，默认行为是应用到 geometry 上定义的 s,t domain。

Texture atlas 格式文件可以读取 UDIM (Mari)、ZBrush 或 Mudbox 格式。此插件还支持 Rtx procedural texture plugins。

Input Parameters

Filename

texture 的 filename。必须提供此参数。

要使用 image sequence，通常会通过在你偏好的表示方式中嵌入帧号，使用 string substitutions 构建 texture map 引用。下面是一个示例：

（Maya）$PROJPATH/sourceimages/flapping.<f4>.tex

（Katana 示例）'/images/flapping.%04d.tex' % frame

（Houdini）在文件名中使用 $F、$F3 或 Houdini expressions，`padzero(3, $F)`

最好避免在 file path 中使用特殊字符、变音符号和空格，因为这些内容可能无法被正确解析，导致 texture 渲染失败。

除了我们支持的 Atlas substitution strings，你还可以在 filename 中引用 constant string primvar，以便在不同 geometry 片段之间复用 texture；此时 primvar 会指定 filename 约定。

  • <primstr:primvarname> - 将替换为具名 constant string primitive variable 的值，例如 asset 名称。例如，在一个带有 "const string model" ["apple"] 的 apple 上，"/assets/<primstr:model>/diffuse.tex" 会扩展为 "/assets/apple/diffuse.tex"

First Channel Offset

Offsets（偏移）要查找的第一个 channel。

Atlas Style

如果指定了 atlas style，则 filename 必须包含 <UDIM>，它会根据 surface parameterization 和 atlasStyle 自动替换。

第一个 UV tile 的 Mari UDIM 示例为 Diffuse.1001.tex 其中 "1001" 会替换 texture filename 中的 <UDIM>。

用户在所有 atlas texture workflows 中都应避免使用负 texture coordinates

Invert T

反转 texture lookup 的 t 参数。

Filter

选择 texture lookup 期间可使用的不同重建过滤器。PxrTexture 可用的过滤器为：

  • 0: Nearest

  • 1: Box

  • 2: Bilinear

  • 3: Bspline

  • 6: Gaussian

  • 7: Lagrangian

Blur

指定对从 texture file 取回的图像进行 Blur 的程度。

Mip Interpolate

选择是否在 multi-resolution texture 的相邻分辨率之间插值，从而在 level 之间获得更平滑的过渡。

Missing Color

如果打开 texture 时出错，使用此 color。

.0 0.0 1.0

Missing Alpha

如果打开 texture 时出错，使用此 alpha。

Linearize

对 texture 应用反向 sRGB transform。如果你在 sRGB space 中绘制 textures（大多数绘画软件的默认值），但在 data linear space 中查看数据，textures 会显得发白。此选项会对 texture 应用 sRGB transform，使其再次呈现视觉线性。

Manifold

提供应用 texture 的 domain。默认值为 s,t。



Adjust Output

Color Scale

texture 中 color values 的 multiplier，可用于调整亮度或操作单个 color channels

Color Offset

对 result 应用 offset，从而移动 result 的 colors

Saturation

提高（大于 1）或降低（小于 1）texture result 的 saturation

Alpha Scale

alpha channel 的 multiplier；当 alpha 用于驱动参数时很有用

Alpha Offset

alpha channel 的 offset；当 alpha 用于驱动参数时很有用

Advanced Settings

MipBias

Bias mip selection 会选择用于渲染的更高（负整数）或更低分辨率（正整数）选择。

 
MaxResolution

Clamp mip selection 到首选最大 level。根据你的选择，这可能会阻止最高分辨率被渲染。

 
OptimizeIndirect

在 indirect hits 上将 texture filter 设置为 Box（低开销），而不是用于其它可见效果的所选 filter。

Output Parameters

resultRGB

过滤后的 color result。请注意，所有结果都会从 startChannel offset 开始查找。

resultR

R channel result（R 通道结果）

resultG

G channel result（G 通道结果）

resultB

B channel result（B 通道结果）

resultA

如果存在 alpha 且连接了 resultRGB，则返回 RGB 之后的 channel。如果未连接 RGB，则返回第一个 channel。
