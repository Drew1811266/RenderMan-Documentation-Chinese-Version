PxrBumpManifold2D

Source: https://rmanwiki-27.pixar.com/space/REN27/542222866/PxrBumpManifold2D
Downloaded: 2026-06-15T03:49:12+00:00

此节点允许艺术家使用 2D 方案放置 patterns（图案），该方案通常依赖艺术家提供的 UVs。你可以对生成的 placement（放置）进行变换（平移、旋转等），也可以使用下面的控制项重复 pattern。此节点还会提供 bump mapping（凹凸贴图）所需的正确 derivatives（导数）。

Input Parameters（输入参数）

Center（中心）

Transformations（变换）以此处为中心。

Angle（角度）

围绕原点的 Rotation angle（旋转角度）。

Scale S（S 缩放）

特征在 S 方向上的 Frequency（频率）。

Scale T（T 缩放）

特征在 T 方向上的 Frequency（频率）。

Offset S（S 偏移）

相对于原点在 S 方向上的 Offset（偏移）。

Offset T（T 偏移）

相对于原点在 T 方向上的 Offset（偏移）。

Invert T（反转 T）

在 T 方向翻转 manifold（流形）。

PrimVar S/ST

自定义 1D S 或 2D ST primvar 的名称。

PrimVar T

自定义 1D T primvar 的名称。

uvSet

默认值为 ST


Output Parameters（输出参数）

result（结果）

2D manifold（2D 流形）。

resultS

manifold 的 S 分量的 float 表示。

resultT

manifold 的 T 分量的 float 表示。
