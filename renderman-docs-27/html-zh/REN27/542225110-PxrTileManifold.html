PxrTileManifold

Source: https://rmanwiki-27.pixar.com/space/REN27/542225110/PxrTileManifold
Downloaded: 2026-06-15T03:49:12+00:00

PxrTileManifold 会计算 st 映射坐标，以在 2D 表面上创建带纹理的平铺块。如有需要，此流形可以引入随机性来隐藏重复图案。

在上面的示例中，木地板是通过将 PxrTileManifold 接入 PxrMultiTexture 来把多个纹理组装成平铺木地板。 PxrMultiTexture 使用了 10 个不同的木板纹理，这些纹理会被随机选取并进行调色。

下方显示着色图和源纹理。

PxrTileManifold 只能与 PxrMultiTexture 配合使用。

Input Parameters

Number of Textures

你想使用的纹理数量。你应该至少有一个纹理，并且 PxrMultiTexture 最多支持 10 个纹理。

该值会传递给已连接的 PxrMultiTexture 节点，因此可以从单一位置控制变化数量。

Texture Order

纹理可以用不同方式分配。它们可以按照在 PxrMultiTexture 节点中出现的顺序枚举，也可以随机选择。

Value

Mode

0

Ordered

1

Pattern Settings

Grid offset

纹理会以常规网格排列组织，并可带有可选偏移。这对砖块或木地板图案等很有用。

Grout Width

灰缝是平铺块之间的间隙。设为 0.0 时，平铺块之间没有间隙。最大值为 1.0，此时平铺块会消失。

Tile Bevel Width

此参数控制从灰缝到平铺块的过渡宽度。它可用于塑造平铺块的置换振幅。

Swap Tile ST

有时你的纹理方向不正确，此控件会通过交换 s 和 t 来旋转平铺块的局部 st 坐标。

Randomize

Random Source

为了让每个对象获得不同变化，你需要选择它们的某个唯一内容来创建唯一种子。可以在 “Object id” 和 “Object Name” 之间选择。
这些属性由输出 RIB 的软件创建，并且取决于你的应用程序（Maya、Katana 等），其中一种可能比另一种更合适。

Extra Seed

如果希望这些图案中的某一个计算出不同变化，请将 randomExtraSeed 设置为非零值。此参数只会影响随机化参数（randomOrientation、randomFlipS 等）。

Random Orientation

通过随机交换 s 和 t 来随机化纹理方向。

Random S Flip

随机翻转 s 的方向。

Random T Flip

随机翻转 t 的方向。

Random S Offset

随机偏移 s。如果纹理不能无缝平铺，通常应避免使用此项。

Random T Offset

随机偏移 t。如果纹理不能无缝平铺，通常应避免使用此项。
