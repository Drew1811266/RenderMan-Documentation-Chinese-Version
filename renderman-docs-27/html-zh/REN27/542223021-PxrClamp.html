PxrClamp

Source: https://rmanwiki-27.pixar.com/space/REN27/542223021/PxrClamp
Downloaded: 2026-06-15T03:49:12+00:00

将颜色的各个独立通道钳制在指定的最小值和最大值之间。如果某个 RGB 通道值小于最小值，就会被设为最小值；如果某个 RGB 通道大于最大值，就会被设为最大值。

Input Parameters（输入参数）

Input Color

此颜色的 R、G、B 各通道都会被钳制在指定的最小值和最大值之间。例如，如果颜色值为 [.8 .5 .2] ，并指定最小值为 [.3 .3 .3] 、最大值为 [.6 .6 .6] ，则得到的颜色将为 [.6 .5 .3]。

Min

每个颜色通道允许的最小值。

Max

每个颜色通道允许的最大值。

Output Parameters

resultRGB

钳制后的颜色结果。

resultR

来自 resultRGB 输出的 R 通道。

resultG

来自 resultRGB 输出的 G 通道。

resultB

来自 resultRGB 输出的 B 通道。
