PxrBumpRoughness

Downloaded: 2026-06-15T03:49:12+00:00

Bump Roughness 用于高效采样微细节。使用此 Pattern 节点时，RenderMan 会根据特征在相机中的尺寸，自动从凹凸贴图切换为粗糙度值，从而降低渲染这些细节所需的采样率，并有效改善渲染时间。 

© Disney/ Pixar

这样的一个例子是 Cars 3 中由 Yeon Kang 负责着色的 Louise Nash。这里可以看到车漆清漆涂层的大面积磨损，它形成微划痕，并以有趣的方式打散镜面反射，同时还驱动各向异性，使其更准确地反映预期的外观开发效果。


© Disney/ Pixar

一个关键特性是 Bump Roughness 能在物体远离相机时仍呈现微细节。在上面的示例中可以看到，只用 1 个采样，而不必发射数百条光线，就能非常接近预期的外观开发效果，并可视化纹理中的目标细节层级。

在上面的示例中可以看到，如何用一张纹理同时驱动粗糙度、各向异性和凹凸。放大 4X 时可以看到，Bump Roughness 能在凹凸通常会因纹理过滤而丢失的地方保留细节。

还有许多其他类型的材质也能受益于 Bump Roughness 贴图带来的额外细节，例如碳纤维、金属、塑料和织物。


© Disney/ Pixar

网络连接

为了让 Bump Roughness 正常工作，需要将它连接到以下材质输入：

  • Roughness
  • Bump Normal
  • Anisotropy
  • Anisotropy Direction


Input Parameters（输入参数）

b2r Texture

只需使用传统的凹凸贴图或法线贴图，Texture Manager 就会在 RFM、RFH 或 RFB 中自动将它们转换为 Bump Roughness 纹理。你会注意到新创建的 tex 文件现在会在文件名后带有 b2r 后缀，例如 myTexture_b2r.exr.tex 

请参阅 txmake 文档 了解详细信息。

Texture Type

要用作 Bump Roughness Map 的贴图类型

TxMake Icon

在 Texture Manager 中显示设置

Manifold

在此连接一个流形节点，用来控制 Bump Roughness 贴图的位置、旋转和平铺

Multi Manifold

在此连接 PxrHexTileManifold 或 PxrRoundCube Manifold。

Bump Roughness

Base Roughness

这是完全放大到细节时基础表面的粗糙度。

Texture Roughness

设置在完全远离细节时由输入纹理提供的纹理粗糙度。它会与基础粗糙度结合。默认值 -1.0 允许纹理提供其自然粗糙度。

Gain

这是聚合粗糙度的乘数

Bump Normal Gain

法线贴图幅度的乘数

Anisotropy Gain

这是聚合各向异性的乘数

Advanced

Adjust Amount

Amount，用于在法线背向相机时调整法线

Surface Mix

将结果法线与表面法线混合的量

Blur

Amount，用于控制应用到纹理查找的模糊量。

Filter

要使用的过滤器。（不建议将 unfiltered/closest 用于置换。）

Filter Scale

乘以纹理过滤器大小；小于 1 会让纹理更锐利。
