LamaDielectric

Source: https://rmanwiki-27.pixar.com/space/REN27/542219603/LamaDielectric
Downloaded: 2026-06-15T03:49:12+00:00

此材质用于渲染玻璃、亚克力和宝石等透明表面。
它们可以是无色或有色的，并具有正确着色的阴影以及相应的吸收/雾化程度。 


Dielectric Parameters

Reflection Tint

真正的电介质材质会保留白色。这里只应做细微调整，以避免失去全部物理性，因为此控制的结果会与反射光线相乘。


Transmission Tint

与上方类似，真正的电介质材质会保留白色。这里只应做细微调整，以避免失去全部物理性，因为此控制的结果会与内部介质以及从内部反射的光线相乘。为了真实感，最好使用下方的 Interior > Absorption 参数。

Reflection / Transmisson：白 / 白 - 蓝 / 白 - 白 / 蓝


Fresnel Mode

  • Artistic - Reflectivity 此结果控制 normal 方向的 reflection 量，以及光线在内部的弯折。这里提供了常见材质的 presets 供你使用。

  • Scientific - IOR 此结果使用折射率来控制光线弯折量，以及作为物理效果的 reflection 结果。你可能记得，冕牌玻璃和水等材质的 IOR 分别为 1.5 和 1.33。


Roughness

微面元或细小粗糙度会以产生模糊结果的方式散射光线。roughness 越低，材质越像镜面并且更光滑/顺滑。较高值会直接呈现 diffuse 外观。

Roughness 示例

Normal/Bump

在这里提供一个 signal（texture 或 procedural pattern），用于在表面上创建 bump，以“伪造”小凸起或划痕等表面细节。这意味着美术师无需对模型中这些繁琐且经常重复的部分进行建模。

Tail Parameters

Mix

tail 效果应以线性方式达到的强度。

Tail Mix 示例

Length

定义 tail 的 roughness；1.0 为完全 roughness，会拉伸得最远，0.0 表示无 roughness 或无 length。

Anisotropy Parameters

Anisotropy

此控制决定光线穿过对象时的行进方向：更向前（正值）或更向后（负值）朝向光源。


Direction

允许使用 tangent map/pattern 改变 anisotropy 的方向。


Rotation

可以围绕 normal 将指定方向完整旋转 360 度。

Dispersion Parameters

Mode

在使用 spread（IOR 的 multiplier）和使用 Abbe value 指定色散效果强度之间切换

Spread

作为 IOR 值的 multiplier，定义色散效果让折射光线扩散的距离。值为 0 会关闭 dispersion。

Saturation

控制 dispersion 的饱和度。

Interior Parameters

这些参数控制电介质材质的内部体积。当你思考玻璃对象为何呈现那样的外观，也就是它的颜色和清晰度时，关键就在 absorption。

Absorption Color

这是体积颜色。它允许对象和阴影被正确着色。白色表示没有吸收、没有着色或阴影。黑色值表示完全不透明、黑色且实心。如果提供了颜色，并且这是 Lama Stack 的最上层，它将成为涂层颜色。

在 XPU 中，从版本 27.3 开始，此参数可以是一个连接。这会创建体积内部散射，其消光颜色会在光线进入玻璃时计算，并且不会随着光线在对象内部延伸而变化。这不是物理精确的着色模型，但易于控制，并且在艺术上可能很有趣。RIS 不支持此功能。

Absorption Radius

光被吸收前，在 scene units 中可到达的最大距离。

在 XPU 中，从版本 27.3 开始，此参数可以是一个连接。RIS 不支持此功能。

Absorption Radius 示例


Scatter Color

定义由上方 absorption 控制项吸收之外、被散射的光线颜色。

Scatter Anisotropy

定义光线的散射方向，可以是前向（正值）或后向（负值）散射。值为 0 表示 isotropic，即向所有方向散射。

Shadow Parameters

Density

在考虑上方控制 Fresnel、coloring、absorption 等的参数后，此项控制最终 shadow 的不透明程度。值为 0 表示完全透明或不可见的 shadow。

Coloring

控制最终 shadow 的着色，其中 0 为黑色，1 与 Transmission Tint 设置的颜色相同。

Fresnel Front

设置光线穿过 front-facing geometry 后，Fresnel 设置对 shadow 的影响。

Fresnel Back

设置光线穿过 back-facing geometry 后，Fresnel 设置对 shadow 的影响。

Advanced Parameters

Exterior IOR

定义外部介质 IOR，可以是空气（默认）、水，也可以是使用 Stack node 在上方形成的清漆等分层材质。

Energy Compensation

允许向结果添加能量，因为 microfacet 或非常粗糙的材质可能会损失能量，并使结果人为变暗。

Surface Mollification

根据 curvature 定义 roughness，并可能有助于减少渲染中的亮点噪声或亮像素/伪影。

Motion Mollification

根据对象运动定义 roughness，并可能有助于减少渲染中的亮点噪声或亮像素/伪影。

Lobe Name

为生成的 Light Path Expression (LPE) 定义 Arbitrary Output Variable (AOV) 名称

Interior Id

如果几个堆叠的带消光的电介质 lobes 使用相同的 Lobe Name，它们需要定义不同的 Interior Ids。如果它们的 Lobe Names 不同，则 Id 无关紧要。

Matte

定义最终 lobe weight 的 lobe output 输出到 Arbitrary Output Variable (AOV) 时使用的名称
