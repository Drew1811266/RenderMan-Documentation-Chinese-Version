PxrDispVectorLayer

Source: https://rmanwiki-27.pixar.com/space/REN27/542223392/PxrDispVectorLayer
Downloaded: 2026-06-15T03:49:12+00:00

此 pattern 会对输入的矢量置换图案进行分层组合。

每个 PxrDispVectorLayer 最多支持 5 个层。若要使用超过 5 个层，可以将一个 PxrDispVectorLayer 节点的结果连接到另一个 PxrDispVectorLayer 节点的输入。 

如果需要超过 5 个层，请将一个 PxrDispVectorLayer 节点的结果连接到另一个 PxrDispVectorLayer 节点的输入。

需要将结果连接到 PxrDisplace 的 Vector Displacement 输入，才能置换几何体。


Input Parameters（输入参数）

Overall Gain

控制最终分层后的矢量置换结果的增益。当需要全局调整所有分层置换值的强度时，这很方便。每个层的增益都会与此参数相乘。

Base Layer

Enabled

默认情况下，基础层处于启用状态。但如果需要调试，可以取消勾选此参数来禁用基础层。取消勾选后，基础层会在分层时被跳过。

Base Layer Gain

基础层的 Gain（增益）。

Base Layer Vector

连接到矢量置换 pattern。它可以是任意 pattern 的矢量或颜色输出结果，例如 resultXYZ 或 resultRGB。

Layer1

Enabled

勾选后，启用第 1 层。

Layer 1 Gain

第 1 层的 Gain（增益）。

Layer 1 Vector

连接到矢量置换。它可以是任意 pattern 的矢量或颜色输出结果，例如 resultXYZ 或 resultRGB。

Layer 1 Mask

第 1 层的 Mask。Values（值）越接近零，此层应用得越少。

Layer 1 Operation

选择此层的置换如何与基础层组合。

  • 1: Add - 将第 1 层的每个矢量分量加到基础层。
  • 2: Over - 将第 1 层的每个矢量分量叠合到基础层之上。
  • 3: Multiply - 将第 1 层的每个矢量分量与基础层相乘。
  • 4: Average - 返回第 1 层与基础层之间每个矢量分量的平均值。
  • 5: Min - 返回第 1 层与基础层之间每个矢量分量的最小值。
  • 6: Max - 返回第 1 层与基础层之间每个矢量分量的最大值。

Layer2

Enabled

勾选后，启用第 2 层。

Layer 2 Gain

第 2 层的 Gain（增益）。

Layer 2 Vector

连接到矢量置换。它可以是任意输出矢量或颜色结果的 pattern，例如 resultXYZ 或 resultRGB。

Layer 2 Mask

第 2 层的 Mask。Values（值）越接近零，此层应用得越少。

Layer 2 Operation

选择此层的置换如何与前一层组合。如果第 1 层未启用，它会与基础层的结果组合。

  • 1: Add - 将第 2 层的每个矢量分量加到第 1 层。
  • 2: Over - 将第 2 层的每个矢量分量叠合到第 1 层之上。
  • 3: Multiply - 将第 2 层的每个矢量分量与第 1 层相乘。
  • 4: Average - 返回第 2 层与第 1 层之间每个矢量分量的平均值。
  • 5: Min - 返回第 2 层与第 1 层之间每个矢量分量的最小值。
  • 6: Max - 返回第 2 层与第 1 层之间每个矢量分量的最大值。

Layer 3 to 4

参数与 Layer 2 类似。

Output Parameters

resultXYZ

矢量结果。

resultX

来自 resultXYZ 输出的 X 分量。

resultY

来自 resultXYZ 输出的 Y 分量。

resultZ

来自 resultXYZ 输出的 Z 分量。
