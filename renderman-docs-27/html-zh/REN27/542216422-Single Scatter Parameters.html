Single Scattering Parameters

Single scatter 是一种简单且开销较低的散射效果。下方有一个 圆盘灯被放置在模型内部 （所有这些示例都如此），我们先在没有 diffuse color 的情况下渲染，然后再使用默认中灰 diffuse color 渲染。



Gain

Single scatter 的 Gain 或权重。



Color

Single scatter 颜色。



Mean Free Path

Single scattering 的 Mean Free Path 标量距离。它指定光线在对象内部传播多远，并因此决定 single scattering 有多平滑。该值会乘以 Properties 部分中设置的 Unit Length。较大的值会更半透明。


Mean Free Path Color

光线在 R、G 和 B 光谱中传播的距离。此值会按 Mean Free Path Distance 缩放。



Directionality

Single scatter 的 Directionality：

0：各向同性（无效果）。

1：forward scatter，背面获得更多光照。

-1：backward scatter，正面获得更多光照。

Refractive Index

Single scatter 的折射率。下方依次为 1.3、5.0 和 10.0。



Blur

single scatter 的 Blur 强度。这个多边形平面后面有一些文字。增大 Blur 会让文字显得更模糊（并使结果稍微变暗）。你可以把它看作 scatter transmission 的 roughness。



Backside Direct Illum Gain

来自另一侧直接照明的 Gain。这会提高背面照明的影响。



Direction Tint

Backside Direct Illum Gain 的 Tinting color。下方使用不同颜色为第一张图（白色）的结果着色。



Double Sided

如果开启，该 single scatter 波瓣会照亮表面的两侧，也就是说，它也会照亮法线背向摄像机的表面。



Trace Control Parameters（操作方式与说明中展示的 Subsurface Controls 类似）：

Consider Backside

控制 single scatter 是否考虑另一侧的表面。这针对命中侧，而不是照明侧（照明侧由 singlescatterDoubleSided 控制）：

  • “Off” - 完全忽略另一侧的表面。这有助于让对象看起来比实际更厚。

  • “On” - 正常模式，扩散会发生在正面与其后的第一个表面之间。


Continuation Ray Mode

控制 Continuation Ray Mode：

“Off” - 只从对象中追踪出去（默认）。

“Last Hit” - 忽略内部几何体，并跳到最后一个表面。

“All Hits” - 当光线离开对象时，在所有命中处进行散射（收集光照）。这可以带来额外亮度，代价是增加噪声。



Max Continuation Hits

在 All Hits 模式中要测试的最大命中数。仅当 Continuation Ray Mode 等于 All Hits 时有效。



Direct Gain Mode

控制 Continuation Ray Mode：

“First Hit” - 只追踪到下一个表面（这与 considerBackside 绑定）。

“Last Hit” - 忽略内部几何体，并跳到最后一个表面。

“All Hits” - 当光线离开对象时，在所有命中处进行散射（收集光照）。这可以带来额外亮度，代价是增加噪声。



Trace Subset

指定在被光线间接击中时用于包含/排除的 trace subset。


Scattering Globals

Irradiance Tint

在由 subsurface 或 single scatter 散射之前应用到照明上的 Tint。



Irradiance Roughness

应用于 subsurface 或 single scatter 的 Diffuse roughness。值为 0 表示经典 Lambertian shading model。非零值会提高 Oren-Nayar shading model 的 microfacet roughness。更大的值会产生更粗糙的 Diffuse，并可能显得略暗。


Unit Length

Subsurface 和 single scatter 的 Unit Length。它是 Mean Free Path Distance 的乘数。Mean Free Path Distance 通常以毫米为单位。如果场景以其他尺度建模，应相应设置 Unit Length。默认值 0.1 适合以厘米建模的场景（Maya 默认）和以毫米测量的 Mean Free Path Distance。较小数值会让对象更 Diffuse（半透明程度更低），较大数值会增强 scatter 效果，使其更半透明。
