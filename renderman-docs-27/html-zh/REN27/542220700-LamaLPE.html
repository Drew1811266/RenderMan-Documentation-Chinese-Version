LamaLPE

Source: https://rmanwiki-27.pixar.com/space/REN27/542220700/LamaLPE
Downloaded: 2026-06-15T03:49:12+00:00

这是一个实用工具节点，主要用于将特定着色信号路由到 Lama Material 中的 User lobes，以输出到由光路表达式定义的 AOV。如果不需要 LPE 的完整能力，也可以改用 PxrTee shading node，或使用 OSL debug() closure 的自定义节点。 

不过，如果希望 AOV 只包含通过特定光路看到的着色信号，就需要定义custom LPE（自定义 LPE），在这些光路中收集特定 User lobe，并使用 LamaLPE 节点将信号写入该 User lobe。 

LamaLPE Parameters

Color

要在指定的 LPE accessible lobe 中累积的颜色

LPE Lobe Name

用于指定哪些 LPE 会累积此 lobe 的 lobe 名称


Example Usage（用法示例）

使用 Option "lpe" "string <lobeslot>" 将名称应用到这些 lobe ["Comma,Separated,Lobe,Names"]

然后定义一个 LPE，使其从该 <lobeslot> 获取值；可以使用custom LPE（自定义 LPE），也可以使用 displaychannel 行，然后用 display line 输出。
 
使用 LamaLPE 节点将输入 Color 写入这些 slot 之一，例如 Bxdf "LamaLPE" "ucolor" "reference color LPEColor" "voronoise:resultRGB" "string LPELobeName" ["Comma"]
