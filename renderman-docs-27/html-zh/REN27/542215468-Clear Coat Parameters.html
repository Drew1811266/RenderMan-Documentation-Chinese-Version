Clear Coat Parameters

在下方参数中，有些参数可在连接到 PxrLayer 后，通过 Input Material 被覆盖，或通过 PxrLayerMixer覆盖。 PxrLayerSurface 设计用于更清楚地展示哪些参数不能在层中被覆盖：它只包含全局参数。当你确定要进行分层时，建议使用这种材质。这些设置的结果保持不变。

  • Bold Face 参数是可分层的，可以按层覆盖。

  • Italicized 参数不能分层或覆盖。所有层都会全局遵循这些参数。例如：选择 GGX 作为镜面模型时，所有层在该参数上都会使用 GGX。

Clear Coat

透明涂层适合制作镀膜对象或车漆、碳纤维等涂料中的顶部釉面层。你甚至可以为该层使用专属凹凸，以制作可信的涂层瑕疵。虽然粗糙度可用，但该层适合较低的粗糙度。你会注意到参数示例中基础漫反射为 50% 灰色，以说明它如何作为涂层工作。如果需要金属表面，请使用上面的镜面波瓣。

透明涂层也可通过 Layer Thickness 参数伪造表面上类似釉面的一定厚度。

还可以选择 Artistic 和 Physical 控制项。Artistic 控制项允许你手动调整反射属性，以匹配想要的效果。使用 Physical（带有提供的预设）可以为希望复现实物、又不想反复微调的人提供匹配真实世界的响应。

Specular Model

选择要使用的镜面模型：Beckmann 或 GGX。同样，Beckmann 适用于完美镜面和类似铬的材质。GGX 可能因其“尾部”或从反射光源中心高光向外衰减的方式而更合适。


Specular Fresnel Mode

在 Artistic 模式下，镜面 Fresnel 响应由 Face Color、Edge Color 和 Fresnel Exponent 控制。

在 Physical 模式下，镜面 Fresnel 响应由 Refractive Index 和 Edge Color 控制。


Face Color (Artistic Mode)

面向角（0 度入射）处的镜面颜色。请注意这里没有单独的 Gain 控制。要控制镜面的“Gain”，只需调整颜色值，或连接到 PxrExposure 节点。


Edge Color

掠射角（90 度入射）处的镜面颜色。要控制边缘镜面的“Gain”，只需调整颜色值，或连接到 PxrExposure 节点。

请注意，此控制项在 Physical Mode 中含义会改变，并作为 Physical Mode 下反射结果的乘数。你可以用这个参数控制反射强度，甚至给结果着色。

Fresnel Exponent (Artistic Mode)

Specular Fresnel 曲线的指数。较低的数值会减弱 Face Color 的效果，同时增强 Edge Color 的效果。较高的数值则相反。如果 Face Color 和 Edge Color 相同，就不会有可见效果。下面我们使用 红色 Face Color 和 绿色 Edge Color，并在较小粗糙度下将 Fresnel Exponent 从 0.1 增加到 1.5，最后增加到 5.0。

Refractive Index (Physical Mode)

这是用于描述物理折射率的参数；即材质的 电介质 折射率。此参数的通道值通常在 1 到 3 范围内。由于我们支持 3 个颜色值来捕捉光谱效应，使用预设可能比拾色器更适合，以避免大量调试。

Layer Thickness (Physical Mode)

此控制项模拟材质上的厚涂层或釉面。它也可以通过纹理改变颜色。下面的着色球只在漫反射波瓣上添加了凹凸，使厚度效果在使用时更明显。前两张图使用相同的 Absorption Tint，厚度分别为 1 和 3，用来展示更厚的设置会按物理方式吸收更多光线。最后一张图使用棋盘格图案为厚度添加纹理。


Absorption Tint (Physical Mode)

此参数控制最终衰减或涂层的颜色。注意上方的 Layer Thickness 参数可能改变外观，较厚的设置会更饱和或吸收更多光线。下面是两个纯色，然后是一个渐变，厚度保持恒定为 1.0。底层漫反射波瓣应用了凹凸，以帮助模拟“厚度”。

Roughness

镜面粗糙度。值越大，镜面反射越粗糙或“模糊”。1.0 时类似漫反射表面，0.0 时则是完全清晰的反射。大多数对象在这些值之间会更真实。为该值添加纹理可能产生有趣的效果，例如污迹、油腻指纹和磨损表面。下面是 0.0、0.5 和 1.0 的值。


Anisotropy

控制镜面高光和反射的形状。0 表示各向同性，会产生常规的圆形镜面高光。从 -1.0 到 1.0 的值会产生从宽到高的一系列椭圆（拉伸）。

默认情况下，各向异性方向由模型纹理参数控制。如果指定了 Shading Tangent，则改用它。你甚至可以通过超过 -1.0 和 1.0 来“过驱”该参数。

Shading Tangent

控制各向异性方向。只有连接到图案时才有效。这对于制作拉丝金属很有用。

Bump

用于 Clear Coat 照明的法线。如果未设置，它会使用页面底部 Properties 中指定的全局凹凸法线。单独设置它可以产生“釉面”效果，即在平滑表面上方有一个凹凸的 Clearcoat。

Double Sided

如果开启，该 Clear Coat 波瓣会照亮表面的两侧，也就是说，法线背向相机的表面也会被照亮。

Specular Energy Compensation

将 Fresnel 能量补偿应用于漫反射和次表面照明波瓣。值为 1.0 时，会尝试通过相对于镜面和粗糙镜面照明响应压暗这些结果，来完全平衡它们。

镜面和粗糙镜面的粗糙度也会纳入考虑。当镜面的 Face Color 或 Edge Color 接近 1.0 时，效果会逐渐消失，因此金属可以添加漫反射基线颜色。可查看 Clearcoat Energy Compensation 以获得可视化示例。启用 Physical Fresnel 模式时，能量补偿会根据 ior 和 kappa 衰减。启用 Artistic 模式时，它会根据由 faceColor/edgeColor 比率推导出的曲线衰减，并非纯物理。


Clearcoat Energy Compensation

将 Fresnel 能量补偿应用于 Clearcoat 自身以外的所有波瓣。值为 1.0 时，会尝试通过相对于 Clearcoat 照明响应压暗这些结果，来完全平衡它们。

Clearcoat 的粗糙度也会纳入考虑。当 Clearcoat 的 Face Color 或 Edge Color 接近 1.0 时，效果会逐渐消失，因此金属可以添加漫反射基线颜色。左侧为 0.0（默认），右侧为 1.0。请注意发生的变暗（能量守恒变化）。
