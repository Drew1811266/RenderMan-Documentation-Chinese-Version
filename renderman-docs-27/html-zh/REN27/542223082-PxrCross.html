PxrCross

Source: https://rmanwiki-27.pixar.com/space/REN27/542223082/PxrCross
Downloaded: 2026-06-15T03:49:12+00:00

计算两个向量之间的叉积。叉积会生成一个结果向量，该向量垂直于两个输入向量。如果在多边形平面中使用两个输入向量，这在尝试查找多边形的法线向量时会很有用。

Input Parameters（输入参数）

Vector 1

叉积计算中的第一个向量。

Vector 2

叉积计算中的第二个向量。

Normalize Result

启用时（设为 1），叉积结果将是归一化向量。

Output Parameters

resultXYZ

叉积结果向量。

resultX

来自 resultXYZ 输出的 X 分量。

resultY

来自 resultXYZ 输出的 Y 分量。

resultZ

来自 resultXYZ 输出的 Z 分量。
