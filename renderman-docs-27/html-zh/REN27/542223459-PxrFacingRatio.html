PxrFacingRatio

Source: https://rmanwiki-27.pixar.com/space/REN27/542223459/PxrFacingRatio
Downloaded: 2026-06-15T03:49:12+00:00

计算几何体的朝向比：即摄像机向量或方向与表面法线之间的简单点积。请注意，某些场景实体没有已定义法线，例如体积。

Input Parameters（输入参数）

使用

选择渲染摄像机方向或一个预定义方向

Direction

不使用摄像机方向时，在此提供方向向量

Coordinate System

提供一个有用的坐标系统：object、world、camera 或其他已定义空间。

Face Forward

如果法线背向摄像机，朝向比可能为负。此选项会翻转法线，使结果始终为正。

Clamp

将结果 Clamp（钳制）到 0 到 1 范围

Invert

反转朝向比：黑色变为白色，白色反之变为黑色。

Mode

选择指定 Ior 还是 Gamma

Ior

使用上方设置的 Ior 模式时，选择一个折射率值，通常为 1.0 到 3.0，可用于模拟电介质表面。

Gamma

使用上方设置的 gamma 节点时，通过一个简单的 gamma 函数塑造朝向比。值 1.0 为中性

Bump Normal

如果表面应用了 bump mapping，请在此输入凹凸法线。如果未连接，节点会使用此着色法线。

Output Parameters（输出参数）

resultF

由朝向比计算产生的单分量（float）值。
