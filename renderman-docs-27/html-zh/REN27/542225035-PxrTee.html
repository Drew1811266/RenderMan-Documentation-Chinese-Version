PxrTee

Source: https://rmanwiki-27.pixar.com/space/REN27/542225035/PxrTee
Downloaded: 2026-06-15T03:49:12+00:00

PxrTee 接收 inputRGB 或 inputF 的值，并将原值输出到 AOV。其 resultRGB 或 resultF 需要连接到着色网络中的某个位置。当着色摄像机可见主射线时，它会把这些值发送到 AOV。

这是将着色网络的任意部分导出为磁盘上任意 AOV 文件的最佳方式，可帮助外观开发艺术家或合成师在查看器中调试，或在合成中迭代工作。

请注意，inputRGB 或 inputF 二者只能连接其一，不能同时连接。如果 inputRGB 和 inputF 都有连接，PxrTee 将输出错误消息。另外，由于让 inputRGB 或 inputF 使用均一值没有实际意义，这些参数只能接受连接。

顾名思义，它就像一个三通分流器：

这也有助于调试着色网络。例如，如果某个图案节点的工作结果不太符合预期，可以在网络中插入一个 PxrTee 节点，查看该位置产生的值。

请注意，PxrTee 节点必须接入着色网络的活动部分才能工作。如果其结果最终没有传入下游活动 Bxdf，则 PxrTee 节点不会被调用，也不会向其 AOV 写入任何内容。
输入参数

Input RGB

要直通到 resultRGB 并写入 color AOV 的三元值。它可以是以下类型：color、vector、normal 和 point。它只会评估一个连接。

Input Float

要直通到 resultF 并写入 float AOV 的浮点值。它只会评估一个连接。

AOV Name

要写入的 AOV 名称。

Verbosity

输出各种信息，帮助你调试场景。

  • 0: Silent -- 只会输出错误消息。

  • 1: Warnings -- 会报告错误和缺失属性。

  • 2: Infos -- 与上面相同，另外还会输出属性值。

输出参数

resultRGB

输出三元值（color、vector、normal 和 point）。

resultF

输出 float。
