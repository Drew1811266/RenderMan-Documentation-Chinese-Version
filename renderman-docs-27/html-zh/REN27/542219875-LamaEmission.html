Emission，或 glow，可以让对象 看起来 在发光。

当你需要发光面板、电路、熔岩等纹理化效果，或其他在照明中具有局部影响的复杂效果时，这很有用。由于这些材质并不真正发光，因此会使用间接照明来实现此材质影响周围环境的效果。这里没有 bump 或 normal mapping 可用，因为无论在这个效果中还是现实中，都看不到这种影响。


Emission Parameters

Color

这是用于驱动效果的纯色，或来自 texture/procedural pattern 的 signal 结果。黑色本质上表示“关闭”或无效果。

在此示例中，一个 texture 插入到 Color input。TV 线条使用 PxrPhasorNoise 创建，并且 texture 与 PhasorNoise offsets 都随时间动画化，从而创建发光的全息广告牌。
