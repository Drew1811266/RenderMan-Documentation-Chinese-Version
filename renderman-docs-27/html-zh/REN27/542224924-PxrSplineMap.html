PxrSplineMap

Source: https://rmanwiki-27.pixar.com/space/REN27/542224924/PxrSplineMap
Downloaded: 2026-06-15T03:49:12+00:00

使用此 pattern 将一个输入同时重映射到 float 和 color 输出


Input Parameters

Input

输入颜色将使用指定的 Float Ramp 进行重映射

Input Range

Input Min

输入的下限范围。在应用 Float Ramp 之前，此值会被线性映射到 0.0

Input Max

输入的上限范围。在应用 Float Ramp 之前，此值会被线性映射到 1.0

Clamp Input

启用后，小于 inputMin 或大于 inputMax 的输入会在应用 Float Ramp 前被钳制。钳制会独立应用于每个颜色分量。如果禁用，可能会产生过大的结果。


Float Ramp

为你的 Float 输出调整 Float Ramp

Output Range

Output Min

输出的下限范围。应用 Float Ramp 后，值会从 0.0 线性映射到此值

Output Max

输出的上限范围。应用 Float Ramp 后，值会从 1.0 线性映射到此值

Clamp Output

启用后，最终输出会被钳制到 0.0 到 1.0 范围。此钳制会独立应用于每个颜色分量。


Color Ramp

为你的 Color 输出调整 Color Ramp


Output Parameters

OutColor

钳制后的结果。

OutColor R

来自 OutColor 输出的 R 通道。

Ou Color G

来自 OutColor 输出的 G 通道。

OutColor B

来自 OutColor 输出的 B 通道。
