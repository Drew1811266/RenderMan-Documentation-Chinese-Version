Light Filters

Source: https://rmanwiki-27.pixar.com/space/REN27/542228334/Light+Filters
Downloaded: 2026-06-15T03:49:12+00:00

Light Filters 用于调控灯光结果。在真实世界中，这通常通过使用遮片和滤色片来改变镜头中灯光的颜色、质感和强度。RenderMan 的灯光滤镜提供了一种强大而灵活的方式，可以在场景布光中实现同样的效果，甚至实现更多控制。 

Light Filters 不会影响 PxrVCM 或 PxrUnified integrators 中的光子发射。

Gobo 和 Blocker light filters 在 RenderMan 26 中已弃用。请改用 cookie 和 rod light filters。


Intensity Light Filter

PxrIntMultLightFilter 是一种 light filter，可用于乘以灯光的 intensity/exposure（强度/曝光）。当你想将某个特定资产（或多个资产）从场景其余部分中分离出来，并为它使用不同的 intensity/exposure 时，这非常有用。实现方式是把对象链接到 PxrIntMultLightFilter。


Barn Door Light Filter

PxrBarnLightFilter 允许我们创建物理准确的 window barns（窗形遮扉），以正确的阴影模拟真实片场灯光。它的其他用途还包括在大型场景中控制反弹光。除了 physical mode（物理模式）之外，它还提供 analytic mode（解析模式）。


Cookie Light Filter

PxrCookieLightFilter 会在灯光前方投射绘制纹理。它类似于 PxrGoboLightFilter，但提供更多选项和更高灵活性。


Rod Light Filter

PxrRodLightFilter 使用类似“rod”的对象来遮挡灯光。该 blocker 可以被塑造成不规则形状，然后放置在你希望遮挡灯光的对象旁边。它类似于上面提到的 PxrBlockerLightFilter，但提供更多控制和更高灵活性。


Ramp Light Filter

PxrRampLightFilter 使用 ramp 来控制灯光。这允许对颜色、强度等进行更复杂的控制。
