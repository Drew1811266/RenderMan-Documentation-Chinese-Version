PxrNormalMap

Source: https://rmanwiki-27.pixar.com/space/REN27/542224317/PxrNormalMap
Downloaded: 2026-06-15T03:49:12+00:00

从法线贴图（编码法线的颜色贴图）生成带凹凸扰动的法线。凹凸会应用在 对象 空间中。 对象 空间与 当前 空间之间的缩放变化会由节点自动应用。

和其他纹理节点一样，此节点接收一个流形，用它描述要将纹理应用到的 2D 或 3D 域。默认情况下，如果未连接流形，则会应用到几何体上定义的 s,t 域。

纹理图集格式文件可以读取为 UDIM（Mari）格式或 Mudbox 格式。

Input Parameters

Bump Scale

缩放凹凸效果。典型范围为 0.0 到 2.0。

Input Normal

指定法线贴图颜色（如果提供了 Filename ，此参数会被忽略）。

Filename

法线贴图文件名。着色器只会从下方 Mapping Controls 的 Channel 读取文件中的一个通道。请注意，它会按切线空间中的 Mudbox 法线贴图格式读取。

Bump Overlay

如果需要组合多个图案节点，请在这里连接 PxrBump 或 PxrNormalMap 节点。

Bump Orientation

Invert Bump

反转凹凸方向。凹面会变为凸面，反之亦然。

Orientation

某些纹理应用提供不同的方向约定，例如 OpenGL 或 DirectX。也可以手动处理（选择 "Custom"）来尝试修正法线贴图的外观。

Flip X

翻转法线贴图的 x 轴（红色通道）。法线贴图没有统一编码标准，每个应用都会以不同方式编码。此开关结合 Flip Y 和 Invert Bump 即可设置正确的方向。

此开关仅在 Orientation 设置为 "Custom" 时才可见。

Flip Y

翻转法线贴图的 y 轴（绿色通道）。法线贴图没有统一编码标准，每个应用都会以不同方式编码。此开关结合 Flip X 和 Invert Bump 即可设置正确的方向。

此开关仅在 Orientation 设置为 "Custom" 时才可见。

Mapping Controls

Manifold

提供应用纹理的范围。默认值为 s,t。

Advanced

Reverse Normal

反转生成的法线。

Adjust Amount

当法线背向摄像机时的调整量。

Surface Normal Mix

将生成的法线与表面法线混合的量

Disabled

使用几何法线


Output Parameters（输出参数）

resultN

输出的带凹凸扰动法线。

法线贴图方向

有时你可能会将法线贴图应用到几何体上，而结果看起来不正确。通常这是方向问题，你需要找到 Invert Bump, Flip X，以及 Flip Y的正确组合。最好的方法是启动 IPR/LiveRender 会话并尝试不同组合。一旦为你的创作应用（ZBrush、Mudbox、3DCoat 等）找到正确组合，同一应用以相同设置导出的所有贴图通常都可使用该组合。
