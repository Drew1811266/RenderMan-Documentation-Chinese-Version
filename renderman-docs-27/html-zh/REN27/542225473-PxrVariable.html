PxrVariable

Source: https://rmanwiki-27.pixar.com/space/REN27/542225473/PxrVariable
Downloaded: 2026-06-15T03:49:12+00:00

允许通过 pattern graphs（图案图）传递 GPrim primitive variables（primvars）。可用它代替 PxrPrimvar


Input Parameters

Variable Name

primitive variable（primvar）的名称。如果未选择任何项，应在 UI 中填写 "PrimVar" 字段来指定自己的选择。

可包含以下选项：

  • 着色位置 (P)
  • 未置换位置 (Po)
  • 着色法线 (Nn)
  • 几何法线 (Ngn)
  • 未置换法线 (Non)
  • 着色切线 (Tn)
  • 归一化视图向量 (Vn)
  • 视图向量长度 (VLen)
  • 表面平均曲率 (curvature)
  • 表面 U 方向曲率 (curvature_u)
  • 表面 V 方向曲率 (curvature_v)
  • Ray Spread（光线扩散） (incidentRaySpread)
  • 光线半径 (incidentRayRadius)
  • 微多边形半径 (PRadius)
  • 微多边形尺寸 (mpSize)
  • 反射偏置 (biasR)
  • 透射 (biasT)
  • 表面 U (u)
  • 表面 V (v)
  • 表面 W (w)
  • 表面 UV (uv)
  • 表面 UVW (uvw)
  • 光线 footprint U (du)
  • 光线 footprint V (dv)
  • 光线 footprint W (dw)
  • 光线 footprint UV (duv)
  • 光线 footprint UVW (duvw)
  • 表面导数 U (dPdu)
  • 表面导数 V (dPdv)
  • 表面导数 W (dPdw)
  • 速度 (dPdtime)
  • 时间 (time)
  • 外部 IOR (outsideIOR)
  • 不透明度 (Oi)
  • 前向运动 (motionFore)
  • 后向运动 ( motionBack )

PrimVar

添加 primitive variable（primvar）的名称

Type

primitive variable（primvar）的类型；这必须与你作为 Variable 选择的类型匹配：

  • float
  • float2
  • color
  • point
  • vector
  • normal

Coordinate System

默认情况下，shader 使用当前坐标系。可用的坐标系包括 world, object，或 user-defined 坐标系。

Output Parameters

result RGB

以 color 形式输出的结果。

result F

以 float 形式输出的结果。

result F3

以三个 float 形式输出的结果。
