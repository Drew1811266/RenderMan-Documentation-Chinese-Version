PxrSurface

Lightyear © Disney/Pixar

在 RenderMan 中渲染表面依赖一种可分层材质（见下方 Input Material），名为 PxrSurface。 

这是 Pixar Animation Studios 长片动画中使用的同一套材质模型。因此，这种材质有一些重要优势：

  • 一个材质即可处理各种外观，例如：

    • 木材

    • 皮肤

    • 塑料

    • 玻璃（Glass）

    • 车漆

    • 以及更多更多！

  • 提供基于物理的预设，帮助你快速开始。

  • 即使使用灵活的控制项，也不必牺牲物理可信性。

    • 艺术家可以有选择地“打破”能量守恒。

    • 艺术指导会变得自然，不需要额外变通。

    • 控制项被方便地标记为 “Artistic” 和 “Physical”。

  • 分层提供无限可能。

作为艺术家，能够为所有着色选择单一材质，可以消除猜测，并让 RenderMan 按需优化。我们会讨论所有不同设置，以及它们与制作特定材质之间的关系。请特别注意有关性能考量的说明，以及哪些组合适用于特定效果的提示。

下面将每种效果或 波瓣 分别放入自己的章节，以帮助用户定位想了解的区域。不过，随着不同波瓣被启用和调整，材质外观也会变化。正是通过组合这些波瓣设置，你几乎可以制作任何想象中的材质。

输入材质

连接到一个分层图案节点，用于对 Bxdf 参数进行分层。这让用户能够通过分层效果创建更复杂的外观。例如灰尘、划痕，甚至标签。

在下方参数中，有些参数可在连接到 PxrLayer 后，通过 输入材质 被覆盖，或通过 PxrLayerMixer覆盖。 PxrLayerSurface 设计用于更清楚地展示哪些参数不能在层中被覆盖：它只包含全局参数。当你确定要进行分层时，建议使用这种材质。这些设置的结果保持不变。

  • 粗体 参数是可分层的，可以按层覆盖。

  • 斜体 参数不能分层或覆盖。所有层都会全局遵循这些参数。例如：选择 GGX 作为镜面模型时，所有层在该参数上都会使用 GGX。


属性参数

本节为整个材质提供全局控制。例如，如果只在这里提供 Bump，它会应用到上面的所有波瓣。Presence 也是一个全局参数，它会遮罩掉或“镂空”所有波瓣，就像遮罩值为 0.0 的位置对象不存在一样。

Bump

用于所有波瓣的 Normal/Bump，除非被单个波瓣的凹凸法线覆盖。

Presence

在这里连接遮罩，可把镂空图案应用到对象上。这对于创建叶片和其他细薄、复杂形状等镂空效果很有用。当从相机直接看到时，它也可把灰度值用作不透明度，以得到半不透明结果。

  • 对于非薄玻璃折射，以及颜色通过内部衰减的“厚实”风格对象，此设置会被忽略。

Presence Cached

指定 Presence 是否缓存。启用缓存时，Presence 将按 每个微多边形 求值，而不是 每个着色点 ；关闭缓存时则按每个着色点求值。 

XPU 不支持 Presence 缓存。

  • 由于 Presence 计算可能很昂贵，如果存在纹理等较重的着色网络，缓存有利于提高渲染速度，不过会增加内存。

  • 由于缓存本质上存储在镶嵌细分级别，如果需要更细的渲染细节，建议使用非缓存 Presence，因为它会按每个着色点求值。

  • 内存和速度之间存在取舍。如果 Presence 非常简单，例如常量值，那么缓存和非缓存速度可能几乎一样快，而非缓存还能节省一些内存。请注意，如果 Presence 为常量 1.0（默认值），PxrSurface 会优化所有这些处理。

  • 启用次表面或单次散射时，为避免伪影（与体积跟踪有关），PxrSurface 会在内部强制使用非缓存 Presence（忽略 Presence Cached 的参数设置）。 

  • 出于类似原因，对于非薄玻璃折射，PxrSurface 会忽略 Presence，无论它是否缓存。

  • 该 积分器 可以控制不透明度不再随机（有噪声）的最大深度。

Shadow Mode

阴影不透明度计算模式：

  • 着色器和阴影颜色

  • 仅阴影颜色

Shadow Color

指定阴影颜色。该参数可用于为艺术指导目的伪造阴影颜色。请注意，修改阴影颜色可能影响性能和全局照明结果，因为它会被视为透明阴影（可以理解为彩色玻璃），即使对象本身是不透明的。

Utility Pattern

指定一个用于输出的图案节点。例如，可以把 PxrMatteID 连接到此参数来输出 ID AOV。也可以输出另一种图案，例如纹理。

Enable Manifold Exploration/Walk

仅在使用 PxrUnified Integrator（积分器），并且 Enable Manifold Walk 选项设置为 ON 时可用。对于需要 捕获 焦散光路的对象，应开启此设置。例如镜面对象内部的漫反射表面。

Specular Curvature Clamp

该控制项用于覆盖对应的 Specular Curvature Filter 参数，该参数位于 PxrVCM 和 PxrUnified 积分器中。

User Color

指定要渲染到 userColor 波瓣的颜色或图案，该波瓣可作为实用输出 通过 LPE 输出。此连接或颜色不会渲染在 beauty 输出中，而是用于输出对合成或后期调整有用的 AOV。
