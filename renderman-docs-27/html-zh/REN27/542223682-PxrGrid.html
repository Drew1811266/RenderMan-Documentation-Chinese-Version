PxrGrid

Source: https://rmanwiki-27.pixar.com/space/REN27/542223682/PxrGrid
Downloaded: 2026-06-15T03:49:12+00:00

此图案会程序化创建矩形或六边形网格，并且可以连接额外的图案来控制颜色和边框厚度。


Style

定义要渲染的图案形状。

  • Rectangular - 此图案会重复四边形形状
  • Hexagonal - 此图案会重复具有均匀外观的六边形形状


Tile Width

设置内部形状的宽度尺寸；小于 1.0 的值会产生额外重复。使用 Hexagonal 时，会采用 Width 或 Height 中的最小值。


Tile Height

设置内部形状的高度尺寸；小于 1.0 的值会产生额外重复。使用 Hexagonal 时，会采用 Width 或 Height 中的最小值。


Tile Shift

偏移隔行。默认值 0 会创建规则网格，而 0.5 会创建砖块状布局。


Border Thickness

设置每个 tile 的边框尺寸，并且在修改 tile 的宽度和高度时保持不变。


Color Tile

按宽度和高度定义，设置每个 tile 内部的颜色。


Color Border

按边框厚度定义，设置每个 tile 边框的颜色。


Manifold

可在此应用一个 manifold 工具，用来调整图案在表面上的位置以及其他属性。
