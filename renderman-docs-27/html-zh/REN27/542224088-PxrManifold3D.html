PxrManifold3D

Source: https://rmanwiki-27.pixar.com/space/REN27/542224088/PxrManifold3D
Downloaded: 2026-06-15T03:49:12+00:00

此节点允许艺术家使用 3D 投影放置图案，而不是使用通常用于依赖 UVs 的纹理的 2D 方案。它允许选择 Pref（用于变形网格），并指定要变换到的坐标系统。 

输入参数

Scale

在 3D 中统一缩放特征频率。

Use

选择要使用的位置类型。

用法

值

默认变量

Current position: P

使用当前（已置换的）曲面位置

0

P

Undisplaced position: Po

使用 置换前 的曲面位置

1

Po

Deform : __Pref

使用对象空间中的参考位置基元变量

2

__Pref

Deform & transform: __WPref

使用世界空间中的参考位置基元变量

3

__WPref

只有在  __Pref  和  __WPref  这些基元变量已通过桥接产品附加到你的几何体时，才可以使用它们。它们通常称为参考姿态或静止姿态。请注意，这些只是名称；底层桥接产品会按这些名称提供数据，而不是由图案节点提供。如果使用 Alembic 这类程序化数据源，请确保导出时已将 Pref 烘焙到对象中。

为什么使用未置换位置？

当在 BxDF 和置换中使用相同的 3D 噪声时，应使用 Po，以确保图案对齐。

Pref

几何体 Pref 的名称（Maya 使用 __Pref 和 __WPref）。 

仅当 Use 设置为 "Deform : __Pref" 或 "Deform & transform: __WPref" 时，才使用此字段。

如果留空，我们会假定使用 __Pref 或 __WPref ，具体取决于当前 Use 设置。

Coordinate System

要应用到流形的坐标系统变换名称。（Maya 称这些为 place3d 节点）。

如果留空，我们会使用 object-space中的位置，因为这是 非变形 对象所需的。

Warp

连接噪声或纹理来扭曲定义域

Warp amount

用于控制扭曲量的滑块。

输出参数

result

输出的 3D 流形。

resultX

流形的 X 分量的浮点表示。

resultY

流形的 Y 分量的浮点表示。

resultZ

流形的 Z 分量的浮点表示。
