用于近似多种真实世界物理效果的摄影机模型。

它支持所有传统 prman 透视摄影机设置，包括带形状控制的运动模糊和散景。

PxrCamera 可以实现的一些效果示例：



桶形畸变 / 色差 / 分割屈光镜 / 移轴 / 暗角



标准参数

fov 

以度为单位的视场角（FOV）。对于矩形图像，这通常是沿图像较窄维度的 FOV。

因此，要用 PxrCamera 模拟 50mm 定焦镜头，应在 FOV 参数中输入 39.6。

fov (end)

以度为单位的视场角（FOV）。用于对快速变焦进行运动模糊。0.0 表示它在渲染期间保持相同。

F-stop

光圈大小，控制景深的模糊程度。较高 F-stop 值会模拟针孔摄影机，并禁用景深。

Focal Length（焦距）

摄影机的焦距，以场景单位计量，从摄影机镜头开始测量。较高 F-stop 值会禁用此项。

对焦距离

到对焦平面的距离；在该平面上的对象处于焦内。该距离以场景单位计量，从摄影机到对象测量。较高 F-stop 值会禁用此项。
