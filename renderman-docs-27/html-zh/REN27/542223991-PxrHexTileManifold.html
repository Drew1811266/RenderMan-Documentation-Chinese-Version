PxrHexTileManifold

Source: https://rmanwiki-27.pixar.com/space/REN27/542223991/PxrHexTileManifold
Downloaded: 2026-06-15T03:49:12+00:00

示例设为规则频率，用于展示该图案


Hex Tiling（六边形平铺）通过以较不可预测的方式随机化图案放置，更好地可视化重复或可平铺纹理。

用法

PxrHexTileManifold 设计为连接到 PxrMultiTexture。


InputParameters

Grid Frequency

单元格网格的全局频率。

Texture Frequency

单元格中纹理的次级尺度。

Blend Width

Burley 描述的指数混合。指数从 8（blendWidth = 0.0）变化到 1（blendWidth = 1.0）。

Offsets

Translate Jitter

随机偏移量。

Rotate Jitter

随机旋转量。

Scale Jitter

随机缩放量

Advanced

Custom Rotation

Connectable Rotation

Custom Orientation

可连接朝向。
