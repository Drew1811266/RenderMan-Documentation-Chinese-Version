PxrVisualizer

Source: https://rmanwiki-27.pixar.com/space/REN27/542232400/PxrVisualizer
Downloaded: 2026-06-15T03:49:12+00:00


PxrVisualizer 是一种工具积分器，可在 Interactive re-rendering（交互式重渲染）期间用于导航大型场景并检查几何体。它允许使用不同的查看 style（样式），包括 shaded、flat、normals、st、wireframe。为了获得最佳结果，请 Disable（禁用） pretessellate（预细分）（在 Renderman for Maya、RfM 中会自动执行）。在 RfM 中将 PxrVisualizer 与 IPR 一起使用时，务必在 IPR Options 中将 Update Mode 设置为“Continuous”。对于复杂 Maya 场景，在 viewport（视口）中隐藏几何体可改善使用 PxrVisualizer 重渲染时的场景导航。


Parameters（参数）

参数说明

int wireframe

[1]：（默认）可视化几何体的线框。

color wireframeColor

线框颜色

float wireframeOpacity

线框不透明度

float wireframeWidth

线框宽度

string style

["shaded"]：（默认）简单的 N dot V shading（N 点乘 V 着色）。

["bxdf"]：使用虚拟灯光渲染场景，不计算阴影或间接照明。类似于 PxrDefault 积分器。

["flat"]：根据 object id（对象 ID）为每个对象渲染不同的 flat color（平面颜色）。

["st"]：渲染归一化的 st。

["normals"]：在 object space（对象空间）中渲染 normals（法线）。

int normalCheck

[1]：以亮橙色显示反转法线区域。此选项默认关闭。

int normalMap

将 normal 的坐标从 [-1;1] 映射到 [0;1]，以避免黑色。

string shadedPrimVar

"shaded" style 的 tint color（着色颜色）。

string matCap

style 为 "matcap" 时使用的 Material capture environment map（材质捕获环境贴图）。表面法线用于在此纹理中查找颜色。


标准 AOV

除常规基于 LPE 的 AOV 之外，此积分器还会输出若干合成师通常使用的标准 AOV。

声明内容Channels（通道）
color __Pworldworld-space（世界空间）中的 P
__Pworld.r：x 分量
__Pworld.g：y 分量
__Pworld.b：z 分量
color __Nworldworld-space（世界空间）中的 Nn
__Nworld.r：x 分量
__Nworld.g：y 分量
__Nworld.b：z 分量
color __depth多用途 AOV
__depth.r：world-space（世界空间）中从摄像机算起的深度
__depth.g：world-space（世界空间）中的高度
__depth.b：几何朝向比：abs(Nn.V)
color __st纹理坐标
__st.x：s
__st.y：t
__st.z：0.0
color __PrefReference Position primvar（参考位置 primvar，如可用）
__Pref.r：x 分量
__Pref.g：y 分量
__Pref.b：z 分量
color __NrefReference Normal primvar（参考法线 primvar，如可用）
__Nref.r：x 分量
__Nref.g：y 分量
__Nref.b：z 分量
color __WPrefReference World Position primvar（参考世界位置 primvar，如可用）
__WPref.r：x 分量
__WPref.g：y 分量
__WPref.b：z 分量
color __WNrefReference World Normal primvar（参考世界法线 primvar，如可用）
__WNref.r：x 分量
__WNref.g：y 分量
__WNref.b：z 分量
