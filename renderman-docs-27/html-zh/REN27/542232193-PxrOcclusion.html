PxrOcclusion

Source: https://rmanwiki-27.pixar.com/space/REN27/542232193/PxrOcclusion
Downloaded: 2026-06-15T03:49:12+00:00

PxrOcclusion 是一种非照片写实积分器，可用于渲染环境光遮蔽以及其他效果。

使用基于物理的积分器渲染时，需要注意遮挡天然就是照明计算的一部分。这是因为间接光或“ambient（环境）”光在角落、凹槽、裂缝等处被反射离开或变暗。不过在最终图像中，出于艺术指导，可能希望额外增强这种效果。你可以用它人为地对图像细节进行“knock back（压低）”或“deepen（加深）”。这个通道通常在合成应用中作为图像乘法或“mult”来应用。

Diffuse 和 specular 光线深度对该积分器没有影响。

Sample Distribution（采样分布）

可以使用不同的采样分布来创建不同的遮挡效果。

描述

Uniform（均匀）

光线在着色点上方的半球中，不会按任何特定方向加权。

Cosine（余弦）

Cosine 分布常用于渲染环境光遮蔽。

反射

根据指定材质的反射波瓣生成采样。仅包含 diffuse BRDF（漫反射 BRDF）的 Materials（材质）会得到类似于上方 cosine（余弦） 设置的结果。

Cosine Spread（余弦扩展）

使用 Cosine 分布时光线的采样扩散范围。值为 1.0 时会得到完美的 Lambertian 分布。

Max Distance（最大距离）

每个点被着色时，光线会伸入场景以击中附近对象。在某些情况下，例如内部空间，这可能产生很暗或黑色的结果。你可以使用这个控制项减少该距离。注意，它以 scene units（场景单位）计量。默认值为 0，即无限。

Falloff（衰减）

Falloff（衰减）可用于控制遮挡效果的柔和度。

不使用 Max Distance（最大距离）时，衰减会按指数衰减计算。数值取决于场景，值越大，外观越柔和。

与 Max Distance（最大距离）结合使用时，衰减会按归一化幂函数计算。

Use Albedo（使用反照率）

开启时，此设置会将 diffuse color（漫反射颜色）应用到生成的图像上。这可以是简单颜色，也可以是连接到材质的图案。

Num Samples（采样数）

此选项会在每个着色点生成固定采样。更多采样可能有助于用更少的摄像机采样消除噪声，但数值较高时可能降低性能。
