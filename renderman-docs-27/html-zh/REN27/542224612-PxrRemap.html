PxrRemap

Source: https://rmanwiki-27.pixar.com/space/REN27/542224612/PxrRemap
Downloaded: 2026-06-15T03:49:12+00:00

通过加深或提亮输入图像的中间调、高光和暗部来调整图像。哪些颜色会被重映射为更亮或更暗的颜色，取决于指定的 Bias 和 Gain 值。

Input Parameters

Input Color

输入颜色会使用指定的 Bias 和 Gain 进行重映射。

Input Range

Input Min

输入的下限范围。在应用 Bias 和 Gain 之前，此值会被线性映射到 0.0。

Input Max

输入的上限范围。在应用 Bias 和 Gain 之前，此值会被线性映射到 1.0。

Clamp Input

启用后，小于 inputMin 或大于 inputMax 的输入会在应用 Bias 和 Gain 之前被钳制。钳制会在每个颜色分量上独立执行。如果禁用，可能会产生过大的结果。

Remap

Bias

调整 inputRGB 颜色中的中间调。这是一个归一化 Gamma 校正因子。大于 0.5 的值会提亮 inputRGB 颜色中的中间调，小于 0.5 的值会使中间调变暗。

Gain

提亮或加深 inputRGB 颜色中的高光和暗部。当 Gain 小于 0.5 时，它会偏向暗部；当 Gain 大于 0.5 时，它会偏向亮部。

Output Range

Output Min

输出的下限范围。应用 Bias 和 Gain 之后，值会从 0.0 线性映射到此值。

Output Max

输出的上限范围。应用 Bias 和 Gain 之后，值会从 1.0 线性映射到此值。

Clamp Output

启用后，最终输出会被钳制到 0.0 到 1.0 的范围。此钳制会在每个颜色分量上独立执行。

Output Parameters

resultRGB

钳制后的颜色结果。

resultR

来自 resultRGB 输出的 R 通道。

resultG

来自 resultRGB 输出的 G 通道。

resultB

来自 resultRGB 输出的 B 通道。
