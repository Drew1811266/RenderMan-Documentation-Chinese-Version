PxrRandomTextureManifold

Source: https://rmanwiki-27.pixar.com/space/REN27/542224582/PxrRandomTextureManifold
Downloaded: 2026-06-15T03:49:12+00:00

PxrRandomTextureManifold 与 PxrMultiTexture 配合，用于向许多对象分配随机纹理：

  • 向一组对象随机分配最多 10 张纹理。
  • Randomize（随机化）纹理方向和纹理偏移。
  • 通过设置图元变量的值来分配纹理（请参见 Adding the textureID primvar in Maya)

PxrRandomTextureManifold 只能与 PxrMultiTexture 一起使用。

Input Parameters

Number of Textures

要使用的纹理数量。至少应有一张纹理，且 PxrMultiTexture 最多支持 10 张纹理。

Texture Order

PxrMultiTexture 将纹理分配给对象时使用的顺序。

使用 textureID PrimVar 时，纹理索引会匹配该 primvar。

Tiling Mode

Random Source

为了让每个对象获得不同变化，需要选择对象上某个唯一信息来创建唯一种子。可以在对象 id、对象名称，以及 "textureID" 图元变量之间选择。

  • 前两个属性由输出 RIB 的软件创建；根据应用程序（Maya、Katana 等）不同，其中一个可能比另一个更适用。
  • textureID 图元变量需要在场景形状上创建，并设置为 0 到 9 之间的值（PxrMultitexture 最多支持 10 张纹理）。如果选择此模式但 primvar 不存在，图案节点会假定值为 0，即第一张纹理。

Extra Seed

如果希望这些图案节点中的某一个计算出不同变化，请将 randomExtraSeed 设为非零值。此参数只会影响随机化参数（randomOrientation、randomFlipS 等）。

Random Orientation

通过随机交换 s 和 t 来随机化纹理方向。

Random S Flip

随机翻转 s 的方向。

Random T Flip

随机翻转 t 的方向。

Random S Offset

随机偏移 s。如果纹理不能无缝平铺，最好避免使用此项。

Random T Offset

随机偏移 t。如果纹理不能无缝平铺，最好避免使用此项。

Adding the textureID primvar in Maya

Using MEL

选择要添加 primvar 的对象，并运行下面的 MEL。

addAttr -at "float" -ln "rmanFtextureID" -k true -dv 0.0 `ls -sl -dag -shapes`;

属性 "rmanFtextureID" 会出现在 channel box 和 attribute editor 中，以便你设置它的值，并会由 RFM 作为预期的 "float textureID" primvar 导出。
