PxrMultiTexture

Source: https://rmanwiki-27.pixar.com/space/REN27/542224293/PxrMultiTexture
Downloaded: 2026-06-15T03:49:12+00:00

PxrMultiTexture 用于引用最多 10 张纹理，以多种方式混合它们，并支持多种贴图选择方法。

和所有 texture style nodes 一样，此节点接收一个 manifold，用它描述要将纹理应用到的 2D domain。默认行为是：如果没有连接 manifold，则应用到几何体上定义的 s,t domain。

PxrMultiTexture 仅兼容 PxrRoundCube, PxrTileManifold, PxrRandomTextureManifold，以及 PxrHexTileManifold。其他 manifolds 不能用于此 pattern。

PxrMultiTexture 不支持 Texture atlas 格式文件：这对它的用法没有意义。不过，此插件支持 RTX procedural texture plugins。


Input Parameters

Multi Manifold

通过以下节点提供应用纹理的 domain： PxrRoundCube 或 PxrTileManifold。如果未连接，则默认使用 s,t。

此输入仅兼容提供 "resultMulti" 输出的节点，例如 PxrRoundCube, PxrTileManifold, PxrRandomTextureManifold，以及 PxrHexTileManifold。

注意：使用 PxrHexTileManifold 时，不支持多张纹理。

Textures

Filename (0 to 9)

纹理的文件名。你至少应在 slot 0 中放置一张纹理。


Texture Parameters

这些参数对所有纹理（0 到 9）都有效。

First Channel Offset

Offsets（偏移）要查找的第一个通道。

通常你会从通道 0 开始（RGBA 纹理的 RGB），但如果使用多通道 OpenEXR 纹理，可能需要指向正确的通道。

假设有一张包含 9 个通道的纹理，并组织为 3 个 RGB 组（diff.r、diff.g、diff.b、spec.r、spec.g、spec.b、ior.r、ior.g、ior.b）。如果要读取 "spec" RGB 通道，就必须将 firstChannel 设置为 3。

Invert T

为纹理查找反转 t 参数。

Filter

选择纹理查找期间可使用的不同重建滤波器。PxrTexture 可用的滤波器包括：

0Nearest
1Box
2Bilinear
3Bspline

Blur

指定从纹理文件取回的图像要模糊多少。

Mip Interpolate

选择是否在多分辨率纹理的相邻分辨率之间插值，从而让层级之间的过渡更平滑。

Missing Color

如果打开纹理出错，则使用此颜色。

Missing Alpha

如果打开纹理出错，则使用此 alpha。

Linearize

对纹理应用反向 sRGB 变换。如果你在 sRGB space 中绘制纹理（大多数绘制软件的默认设置），但在 data linear space 中查看数据，纹理会显得发灰。此选项会对纹理应用 sRGB 变换，使其再次呈现为 visually linear。

Advanced Settings


MipBias

对渲染时选择的 mip selection 施加 Bias（偏置），可偏向更高分辨率（负整数）或更低分辨率（正整数）。

 
MaxResolution

将 mip selection Clamp 到首选的最高层级。根据你的选择，这可能会阻止最高分辨率被渲染。

 
OptimizeIndirect

在 indirect hits 上将纹理 filter 设置为 Box（开销较低），而不是使用其他可见效果所选的 filter。


Randomize

输出 multi manifold 的 Manifold patterns 会用 2D coordinate system 编码应使用哪张纹理。PxrMultiTexture 可以利用这些坐标计算 random seed，以驱动局部变化。

Random Source

为了让每个对象获得不同变化，需要选择对象上某个唯一的信息来创建唯一 seed。你可以在对象的 id 和对象名称之间选择。
这些属性由输出 RIB 的软件创建；根据宿主软件不同，其中一种可能比另一种更适用。

Seed

设为零时，所有使用同一 manifold 的 PxrMultiTexture patterns 都会计算出相同的 random seed。如果希望其中某个 pattern 计算不同变化，请将 randomSeed 设为非零值：

Hue

限制随机 hue shift。范围从 0（无变化）到 1（完整变化）。

Saturation

限制随机 saturation scale。范围从 0（无变化）到 1（完整变化）。

Luminance

限制随机 luminance scale。范围从 0（无变化）到 1（完整变化）。

Gamma

限制随机 gamma variation。范围从 0（无变化）到 1（完整变化）。

G 通道结果
