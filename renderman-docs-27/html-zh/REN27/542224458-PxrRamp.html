PxrRamp

Source: https://rmanwiki-27.pixar.com/space/REN27/542224458/PxrRamp
Downloaded: 2026-06-15T03:49:12+00:00

创建一个 color ramp（颜色渐变），在 colors 输入列表中指定的颜色之间逐渐过渡。

Colors 和 Positions 两个列表必须拥有相同数量的元素。它们共同定义用于 spline 插值的控制点列表。

注意：From now on（从现在开始），PxrRamp 会在内部向 spline 添加额外点，以输出预期结果。以前 ramp 至少需要 4 个点，并要求用户具备额外知识；现在不再需要。

Input Parameters

Ramp Type

控制 ramp 颜色变化的方向。

Tile

确定 ramp 纹理是否重复。

Reverse

启用后会翻转 ramp，使其方向反转。

Spline Type

控制 colors 列表中各颜色之间的中间颜色如何计算。也可以选择 "constant"（21 中为 "None"），让所选颜色之间不进行混合。

Spline Map

如果连接到一个 pattern，它就是控制 ramp 映射的 1D manifold。例如，如果将它连接到 noise pattern，则 ramp 中颜色变化的方向会跟随该 noise pattern。如果两者都连接，此 map 优先于 manifold 值。

下图显示一个 PxrVoronoise pattern（frequency 设为 10）通过 splineMap 参数连接到 ramp。noise pattern 的黑色值映射到 ramp 的红色，noise 的白色值映射到 ramp 的蓝色，介于其间的值会映射到 ramp 中部的绿色。

Random Source

当 Ramp Type 设置为 "Random Object Color" 时，PxrRamp 会使用对象 ID（Attribute "identifier:id"）或对象名称（Attribute "identifier:name"）计算 random seed。该 seed 会用于从 ramp 中选择一种颜色。

此示例使用 "None" Spline Type，但也支持其他 spline types。

Random Seed

更改此值可获得一组不同的随机变化。

Color Ramp (Manual)

Manifold

控制 ramp 的 2D 放置。默认使用表面几何体的 s 和 t 值，在表面上放置 ramp 颜色。

Match C++ patterns

匹配 RenderMan23 C++ 场景，但可能减慢渲染。仅在必要时使用。

Output Parameters

resultRGB

钳制后的颜色结果。

resultR

来自 resultRGB 输出的 R channel。

resultG

来自 resultRGB 输出的 G channel。

resultB

来自 resultRGB 输出的 B channel。
