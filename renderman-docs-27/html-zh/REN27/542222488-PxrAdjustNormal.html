PxrAdjustNormal

此图案可用于调整输入法线。有时，凹凸贴图中背向的法线会引入裂缝和黑色空隙等伪影。当使用 你自己的 凹凸节点时，此节点可以修正这个问题。 

我们在 PxrBump 和 PxrNormalMap 中也包含了处理此问题的参数，作为开箱即用的解决方案。


输入参数

Input Normal

连接要调整的法线图案或纹理。

Adjust Amount

当几何法线背向相机时，用于调整法线的量。看到渲染中出现伪影时，这很有用。下面的示例使用了极端凹凸设置，目的是夸大这种效果。请注意，边缘（掠射角）受影响最明显。对于更细微的凹凸，这种修正会进一步局限在边缘处。

将鼠标悬停在图像上以查看效果。

Surface Normal Mix

将结果法线与原始表面法线混合的量。趋近 1.0 时会混入更多表面法线。在下面示例中可以看到，原始对象没有定义其他法线，并且在设置为 1.0 时会变得平滑。
