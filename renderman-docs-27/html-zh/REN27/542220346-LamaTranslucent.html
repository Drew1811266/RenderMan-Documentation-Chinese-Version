半透明是一种漫透射形式，光线会穿过薄对象。此效果常用于模拟叶片、纸张和薄塑料等对象。


Translucency Parameters

Color

此参数是表面的最终颜色。你可以选择一种颜色，也可以连接 texture 或其他 procedural pattern。


Roughness

当光线穿过薄对象时，你可能会看到另一侧的阴影或剪影。
Roughness 可以让来自另一侧的光显得散射更多或更加模糊。

Roughness 示例：0.0 / 0.2 / 0.8 / 0.6 / 0.8 / 1.0


Normal/Bump

在这里提供一个 signal（texture 或 procedural pattern），用于在表面上创建 bump，以“伪造”小凸起或划痕等表面细节。这意味着美术师无需对模型中这些繁琐且经常重复的部分进行建模。
