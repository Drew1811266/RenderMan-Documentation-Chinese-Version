在下方参数中，有些参数可在连接到 PxrLayer 后，通过 Input Material 被覆盖，或通过 PxrLayerMixer覆盖。 PxrLayerSurface 设计用于更清楚地展示哪些参数不能在层中被覆盖：它只包含全局参数。当你确定要进行分层时，建议使用这种材质。这些设置的结果保持不变。

  • Bold Face 参数是可分层的，可以按层覆盖。

  • Italicized 参数不能分层或覆盖。所有层都会全局遵循这些参数。例如：选择 GGX 作为 Specular 模型时，所有层在该参数上都会使用 GGX。


Specular and Rough Specular Parameters 

Specular 参数控制镜面反射。这里可以定义对象有多么有光泽或反射性：它是塑料、抛光大理石桌，还是镜子？下面的 Rough Specular 波瓣具有相同的设置和效果。

还可以选择 Artistic 和 Physical 控制项。Artistic 控制项允许你手动调整反射属性，以匹配想要的效果。使用 Physical（带有提供的预设）可以为希望复现实物、又不想无休止微调的人提供匹配真实世界的响应。


Specular Model

选择要使用的 Specular 模型：Beckmann 或 Ggx。Beckmann 适用于完美镜面和类似铬的材质。Ggx 可能因其“尾部”而更合适，也就是高光从光源中心反射向外柔和衰减的方式。左侧为 Beckmann，右侧为 GGX，Roughness 为 0.25。



Specular Fresnel Mode

在 Artistic 模式下，Specular Fresnel 响应由 Face Color、Edge Color 和 Fresnel Exponent 控制。

在 Physical 模式下，Specular Fresnel 响应由 Refractive Index、Extinction Coefficient 和 Edge Color 控制。


Face Color (Artistic Mode)

面向角（0 度入射）处的 Specular 颜色。请注意这里没有单独的 Gain 控制。要控制 Specular 的“Gain”，只需调整颜色值，或连接到 PxrExposure 节点。下面是在 Roughness 0.25 下的不同选择，包括纹理驱动的选择。


Edge Color

掠射角（90 度入射）处的 Specular 颜色。要控制边缘 Specular 的“Gain”，只需调整颜色值，或连接到 PxrExposure 节点。下面是在 Roughness 0.25 下的不同选择，包括纹理驱动的选择。这里还降低了 Fresnel Exponent，使效果更明显。


请注意，此控制项在 Physical Mode 中含义会改变，并作为 Physical Mode 下反射结果的乘数。你可以用这个参数控制反射强度，甚至给结果着色。在下面示例中，我们使用了黑色、50% 灰色和白色。



Fresnel Exponent (Artistic Mode)

Specular Fresnel 曲线的指数。较低的数值会减弱 Face Color 的效果，同时增强 Edge Color的效果。较高的数值则相反。如果 Face Color 和 Edge Color 相同，就不会有可见效果。下面我们使用 红色 Face Color 和 绿色 Edge Color，并在较小 Roughness 下将 Fresnel Exponent 从 0.1 增加到 1.5，最后增加到 5.0。


Refractive Index (Physical Mode)

这是用于描述物理折射率的参数；即材质的 电介质 折射率。此参数的通道值通常在 1 到 3 范围内。由于我们支持 3 个颜色值来捕捉光谱效应，使用预设可能比拾色器更适合，以避免大量调试。


Extinction Coefficient (Physical Mode)

Extinction Coefficient 是材质的第二个折射率，用于表征金属行为。此参数的通道值通常在 1 到 3 范围内。由于我们支持 3 个颜色值来捕捉光谱效应，使用预设可能比拾色器更适合。当它为 0 时，材质会像电介质（玻璃、Clearcoat）一样响应；非零时，材质会像导体一样响应。由于这是基于物理值的控制，你通常会发现预设比手动微调设置更有用。下面是 Copper、Gold 和 Nickel 的预设。


Roughness

Specular roughness（镜面粗糙度）。值越大，镜面反射越粗糙或“模糊”。1.0 时类似 Diffuse 表面，0.0 时则是完全清晰的反射。大多数对象在这些值之间会更真实。为该值添加纹理可能产生有趣的效果，例如污迹、油腻指纹和磨损表面。下面示例从 0.0 到 0.5，最后到 1.0（Diffuse）。


Anisotropy

控制 Specular 高光和反射的形状。0 表示各向同性，会产生常规的圆形 Specular 高光。从 -1.0 到 1.0 的值会产生从宽到高的一系列椭圆（拉伸）。

默认情况下，各向异性的方向由模型纹理参数控制。如果指定了 Shading Tangent，则改用它。下面是 -1.0、0.0 和 1.0 的示例。

建议将 Anisotropy 值设得尽可能低，只要能达到所需外观即可。极端值（超过 -1.0 或 1.0）可能产生难以有效收敛的噪声。Roughness 值较低时，这种情况会更糟。


Shading Tangent

控制各向异性方向。只有连接到 pattern 时才有效。这对于制作拉丝金属很有用。下面三个示例使用纹理，并将 Anisotropy 设为 -10。


Bump

用于 Specular 照明的法线。如果未设置，它会使用本页底部 Properties 中指定的全局 bump normal（凹凸法线）。


Double Sided

如果开启，该 Specular 波瓣会照亮表面的两侧，也就是说，法线背向相机的表面也会被照亮。该选项默认关闭，以提升性能。



Rough Specular

与 Specular 参数相同，但默认 Roughness 更大，为 0.6。该层适用于比上方 Specular 波瓣更高的 Roughness 设置。下面从左到右依次为：Rough Specular、Specular、两个波瓣组合。
