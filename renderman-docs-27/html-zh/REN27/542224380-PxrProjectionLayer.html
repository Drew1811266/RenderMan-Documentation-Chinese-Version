PxrProjectionLayer

Source: https://rmanwiki-27.pixar.com/space/REN27/542224380/PxrProjectionLayer
Downloaded: 2026-06-15T03:49:12+00:00

PxrTexture 的专用版本，用于投射纹理；设计上与 PxrProjectionStack 配合使用。

它提供额外功能，用于优化 texturing，并一次应用多个 textures。


Input Parameters

Filename

texture 的 filename。必须提供此参数。

此参数不应留空，否则会改为投射 missing color 。

First Channel Offset

用于 filename 查找的 first channel offset。

Filter

选择 texture lookup 期间可使用的不同 reconstruction filters。PxrProjectionLayer 可用的 filters 为：

ValueFilter
0Closest
1Smart cubic
2Linear

Blur

指定从 texture file 读取的图像需要 blur 的程度。


Mip Interpolate

选择是否在 multi-resolution texture 的相邻分辨率之间插值，从而在 levels 之间获得更平滑的过渡。


Missing Color

如果打开 texture 时出错，则使用此 color。


Missing Alpha

如果打开 texture 时出错，则使用此 alpha。


Linearize

对 texture 应用反向 sRGB transform。如果你在 sRGB space 中绘制 textures（大多数 paint packages 的默认设置），但在 data linear space 中查看数据，textures 会显得发白。此选项会对 texture 应用 sRGB transform，使其再次呈现 visually linear 的效果。


PreMultiply

如果希望 final color 乘以 alpha，请 Enable（启用）此项。


Mask

可以使用黑白 mask 来避免在不可见点上读取 texture。PxrProjectionLayer 提供这样的输出（resultMask），用于避免读取被遮挡或位于 projection window 之外的 texels。


Manifold

提供应用 texture 的范围。在这里接入 PxrProjector，以向 texture function 提供 mapping coordinates。默认值为 s,t。

Channels 

Channels section 允许投射最多 16 个额外 textures，它们可以传递给 PxrProjectionStack 节点进行 compositing，并输出到单独的 AOVs 或 outputs。

Premultiply Channels

Multiply（相乘）：将 channels color 乘以 resultA。当 channels 尚未被主 texture 的 alpha 和 mask input 预乘时，这很有用。

Channels Filenames

最多 16 个额外 color textures 的 filename。

Channels Linearize

对 texture 应用 sRGB linearization。

Channels Missing Color

无法读取 texture 时使用的 color。

Channels Input Black

调整 texture 的 black point。类似于 Photoshop 的 "Levels"。

Channels Input Gamma

调整 texture 的 gamma。类似于 Photoshop 的 "Levels"。

Channels input White

调整 texture 的 white point。类似于 Photoshop 的 "Levels"。

Channels Output Black

设置 texture 的 maximum black value。类似于 Photoshop 的 "Levels"。

Channels Output White

设置 texture 的 minimum white value。类似于 Photoshop 的 "Levels"。



Advanced Texture Settings

MipBias

Bias mip selection：渲染时选择更高（负整数）或更低分辨率（正整数）的 mip。

MaxResolution

Clamp mip selection 到首选最大 level。根据你的 selection，这可能会阻止最高分辨率被渲染。

OptimizeIndirect

在 indirect hits 上将 texture filter 设置为 Box（低成本），而不是使用其他 visible effects 所选的 filter。


Output Parameters

resultRGB

经过过滤的 color result。请注意，所有 results 都会从 startChannel offset 开始查找。

resultR

R channel result

resultG

G channel result

resultB

B channel result

resultA

Alpha channel，会考虑通过 mask input 提供的 mask。

outChannelsRGB

RGB channel data 会打包到单个 struct output 中。PxrProjectionStack 会从该 struct 的 numChannels member 获取可用 layers 数量。
