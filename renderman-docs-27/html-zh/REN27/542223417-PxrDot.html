PxrDot

Source: https://rmanwiki-27.pixar.com/space/REN27/542223417/PxrDot
Downloaded: 2026-06-15T03:49:12+00:00

计算两个矢量之间的点积。点积会取两个矢量的大小和方向，并返回一个新的单一结果。彼此成 90 度的矢量点积为零。常用于 fresnel/曲率这类效果，或“电子显微镜”类型图像。

Input Parameters（输入参数）

Vector 1

点积计算中的第一个矢量。

Vector 2

点积计算中的第二个矢量。

Output Parameters

result

由两个输入矢量的点积产生的单分量（float）值。
