Stylized Looks 工具集依赖默认 AOV 与自定义 AOV 的组合来生成非照片写实效果。

桥接工具会通过 “Enable Stylized Looks” 复选框自动添加这些 AOV，但你也可以手动把它们添加到场景中以获得同样效果。

手动配置 AOV 时，请注意所有 AOV 都必须使用 color 类型，显示过滤器才能正确工作。

NPRtoonDiffRamp
Toon shader 重映射的灰度表示。

NPRfinal
Lines shader。最终线条已合成到 beauty 中

NPRallLines
Lines shader。如果串联线条，会将它们合成到此 AOV 中。使用每条线条的颜色。

NPRallLinesAlpha
所有线条的 Alpha，用于合成串联的 Lines shader。


NPRoutline

Lines shader。轮廓边缘检测结果。如果 Line Type 设为 outline，此 AOV 将包含边缘检测结果；否则会显示输入数据。


NPRlineNZ
Lines shader。表面法线的 X 与 Y 分量 + Z 数据的边缘检测结果。如果 Line Type 设为 lineNZ，此 AOV 将包含边缘检测结果；否则会显示输入数据。


NPRsections
Lines shader。每个对象形状的随机颜色。如果 Line Type 设为 “sections”，此 AOV 将包含边缘检测结果；否则会显示输入数据。

NPRlineAlbedo
Lines shader。由 PxrStylizedControl 设置，用于指定线条颜色，可逐对象控制。

NPRlineWidth
Lines shader。由 PxrStylizedControl 定义的逐对象线宽。如果将上游 fractal 接入 PxrStylizedControl 中的 line width min/max remap，这是一个很适合查看厚度的调试 AOV。

NPRlineMask
Lines shader。在 PxrStylizedControl 中设置。遮罩最终线条。

NPRcurvature
Lines shader。用于边缘检测的表面法线数据。如果 Line Type 设为 “curvature”，此 AOV 将包含边缘检测结果；否则会显示输入数据。

NPRtextureCoords
Hatching shader。在 PxrStylizedControl 中设置。用于 Hatching 投射的可选逐对象自定义纹理坐标。

NPRPtriplanar
Hatching shader。用于 Hatching 三平面投射的可选逐对象自定义 P（世界空间、对象空间等）坐标。

NPRoriginalBeauty
Toon/Hatching/Lines shader。保存原始 beauty；如果需要，可采用不写回 Primary 的可选工作流。

NPRlineCamdist
Lines shader。来自相机距离的线条数据。如果 Line Type 设为 “camdist”，此 AOV 将包含边缘检测结果；否则会显示输入数据。
