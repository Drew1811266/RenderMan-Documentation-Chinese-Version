SPECULAR

Reflection Tint

外部反射的颜色乘数。应谨慎使用，因为非白色值会破坏物理性。

Transmission Tint

进入介质的光线的颜色乘数（涵盖外部透射和内部反射）。应谨慎使用，因为非白色值会破坏物理性。
定义电介质颜色的首选方式，是使用正下方的 Interior 属性。

Fresnel Mode

在 Artistic 和 Scientific 参数之间切换菲涅耳设置。

Artistic 模式提供法线方向的反射率，以及掠射角处的 reflectivity90。

Scientific 模式为每个 RGB 分量提供一个折射率。请注意，IOR 只用于计算 Schlick 近似所使用的反射率值。

当 reflectivity90 为 1 时，两种模式会得到相同结果，因为二者可以相互映射。

Reflectivity

表面在法线方向反射的颜色。对应 specColor。

IOR

折射率（通常记作 “eta”），定义表面在法线方向的反射量，以及光线如何因折射而弯折。如有需要，可为每个颜色通道指定单独的值。

Reflectivity 90

表面在掠射角反射的颜色。对于多数材质，应保持为 1,1,1。对应 specColor90。

ReflectivityProfile

 用于 Schlick 的指数倒数

Roughness

微面元或细小粗糙度会以产生模糊结果的方式散射光线。粗糙度越低，材质越像镜面并且更光滑。较高值会直接呈现漫反射外观。

Normal/Bump

在这里提供一个信号（纹理或程序化图案），用于在表面上创建凹凸，以“伪造”小凸起或划痕等表面细节。这意味着美术师无需对模型中这些繁琐且经常重复的部分进行建模。

Tail Parameters

Mix

尾部效果应以线性方式达到的强度。

Length

定义尾部的粗糙度；1.0 为完全粗糙，会拉伸得最远，0.0 表示无粗糙度或无长度。

Anisotropy Parameters

Anisotropy

此控制决定光线穿过对象时的行进方向：更向前（正值）或更向后（负值）朝向光源。

Direction

允许使用切线贴图/图案改变各向异性的方向。

Rotation

可以围绕法线将指定方向完整旋转 360 度。

Interior Parameters

这些参数控制电介质材质的内部体积。当你思考玻璃对象为何呈现那样的外观，也就是它的颜色和清晰度时，关键就在吸收。

Absorption Color

内部体积的吸收颜色。它定义为消光系数的负对数，取值在 0 到 1 之间，使对象本身及其阴影按比例着色。设置为 1 时没有吸收；设置为 0 时，对象将完全不透明。请注意，要让单次散射生效，该值必须小于 1。当此节点作为堆栈中的顶层使用时，此值也会充当涂层颜色。

在 XPU 中，从版本 27.3 开始，此参数可以是一个连接。这会创建体积内部散射，其消光颜色会在光线进入玻璃时计算，并且不会随着光线在对象内部延伸而变化。这不是物理精确的着色模型，但易于控制，并且在艺术上可能很有趣。RIS 不支持此功能。

Absorption Radius

光被吸收前，在场景单位中可到达的最大距离。

在 XPU 中，从版本 27.3 开始，此参数可以是一个连接。RIS 不支持此功能。

Scatter Color

定义由上方吸收控制项吸收之外、被散射的光线颜色。

Scatter Anisotropy

定义光线的散射方向，可以是前向（正值）或后向（负值）散射。值为 0 表示各向同性，即向所有方向散射。

Shadow Parameters

Density

在考虑上方控制菲涅耳、着色、吸收等的参数后，此项控制最终阴影的不透明程度。值为 0 表示完全透明或不可见的阴影。

Coloring

控制最终阴影的着色，其中 0 为黑色，1 与 Transmission Tint 设置的颜色相同。

Fresnel Front

设置光线穿过正面朝向的几何体后，菲涅耳设置对阴影的影响。

Fresnel Back

设置光线穿过背面朝向的几何体后，菲涅耳设置对阴影的影响。

Advanced Parameters

Exterior IOR

定义外部介质 IOR，可以是空气（默认）、水，也可以是使用 Stack 节点在上方形成的清漆等分层材质。

Energy Compensation

允许向结果添加能量，因为微面元或非常粗糙的材质可能会损失能量，并使结果人为变暗。

Surface Mollification

根据曲率定义粗糙度，并可能有助于减少渲染中的亮点噪声或亮像素/伪影。

Motion Mollification

根据对象运动定义粗糙度，并可能有助于减少渲染中的亮点噪声或亮像素/伪影。

Lobe Name

为生成的光路表达式（LPE）定义任意输出变量（AOV）名称

Interior Id

如果几个堆叠的带消光的电介质波瓣使用相同的 Lobe Name，它们需要定义不同的 Interior Id。如果它们的 Lobe Name 不同，则 Id 无关紧要。

Matte

定义最终波瓣权重的波瓣输出到任意输出变量（AOV）时使用的名称
