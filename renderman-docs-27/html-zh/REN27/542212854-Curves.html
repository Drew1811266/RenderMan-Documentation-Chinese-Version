简介

曲线可用于非常高效地渲染大量细长的几何体线束，因此这种图元特别适合建模毛皮、头发和草叶。这种几何类型是一种近似表示；与等价的圆形横截面曲面相比，它在时间和内存上都被设计为更高效地渲染。代价是曲线不支持位移，并且在近距离查看曲线几何体时，有时会出现几何伪影；随着曲线远离摄影机，这些伪影通常会变得不可察觉。RenderMan 同时支持扁平（类似缎带）曲线和圆形（类似管状）曲线，并且可以从多种基函数中选择。

参数

宽度

曲线宽度可以沿曲线长度变化。可以指定一个适用于该图元中所有曲线线束的恒定宽度，也可以按每条曲线线束、每个分段或每个顶点分别指定不同宽度。

按分段或按顶点指定时，可以沿曲线使用负的 width 值：当 width 值沿曲线插值时，曲线会逐渐收细到尖端，而 width 为负的曲线部分会消失。这可用于在不改变控制点或其动画/模拟的情况下，轻微随机化一簇头发的长度。 

基函数

给定用户以顶点形式提供的一组控制点（它们构成曲线的外壳），基函数决定实际渲染出的曲线外观。例如，某些基函数（如 Catmull-Rom 基函数）保证渲染出的曲线会穿过所有内部的非端点控制顶点；而其他基函数（如 B-Spline）通常不会精确穿过任何控制顶点。每种基函数在平滑度、可控性和易用性方面都有不同取舍。

  • Bezier - Bezier 基函数会生成一条穿过每第三个控制顶点的 3D 曲线。其余控制顶点位置用于确定相邻、曲线确实穿过的顶点处曲线的入射和出射切线。对包含多个分段的曲线（也就是超过 4 个控制点）使用 Bezier 基函数时，应注意确保渲染曲线上的顶点处入射和出射切线对齐，否则曲线会在突然改变方向的位置出现平滑度不连续。

  • B-Spline - B-Spline 基函数生成的曲线通常只是近似靠近给定控制顶点，而不一定穿过它们，因此使用上可能略不直观；但它的优点是通常能产生非常平滑的曲线。如果多个控制顶点连续重复（也称为重数 multiplicity），曲线平滑度可能出现不连续。

  • Catmull-Rom - 这种基函数生成的曲线保证穿过每个内部的非端点控制顶点，并且通常会生成平滑曲线；不过选择这种基函数时，在高曲率区域有时会出现意外的“摆动”（示例见下方图片）。改用 B-Spline 基函数可以平滑这些摆动，但与 Catmull-Rom 基函数不同，B-Spline 基函数通常生成的曲线不会精确穿过控制顶点（对于非端点控制顶点而言）。

  • Linear - 指定 linear 曲线会产生直线段；曲线中连续的线性分段会用圆形端帽连接。

一种常见做法是在曲线端部对 B-Spline 和 Catmull-Rom 曲线使用重数，以便更好地控制曲线的起点和终点。请注意，为了让这类曲线基函数在第一个和最后一个曲线控制顶点的精确位置结束，Catmull-Rom 需要再重复一次这些顶点，而 B-Spline 需要再重复两次。

模式

RenderMan 同时支持扁平（类似缎带）曲线和圆形（类似管状）曲线，如下图所示。

  • 扁平曲线 - 通常推荐用于草叶，或其他曲线几何外观相对更像“缎带”而非管状的几何体。为扁平曲线指定用户法线后，曲线会扭转，使其朝向始终跟随用户指定的法线（对摄影机和其他光线类型都是如此）。不支持无法线曲线（始终面向摄影机）。扁平曲线不适合树叶或其他“扁平”表面，也就是几何体大致呈矩形且期望轮廓由裁剪纹理获得的情况。对于这些情况，使用多边形会好得多，也更高效。

  • 圆形曲线 - 通常推荐用于头发和毛皮，因为它通常能以更高保真度表示底层头发几何体（尽管并非所有情况都是如此）；直接结果是，与扁平曲线相比，圆形曲线往往能更准确地再现毛皮和头发的遮挡与阴影。如果提供了用户指定的法线，曲线将是扁平的（垂直于这些法线），因此会被渲染为一条跟随用户指定法线扭转的缎带状曲线（例如扁平曲线）。目前，与扁平曲线相比，圆形曲线通常具有更好的时间和内存性能。它高度依赖于 BxDF 和着色模型，但在某些情况下，使用圆形曲线时产生的管状变化法线可以得到更有趣的着色结果。

不透明度

与 RenderMan 中的所有几何体一样，如果附加到 curves 图元的 BxDF 计算 presence 或 opacity，这将控制曲线是否出现在屏幕上以及出现的百分比。为了提升光线追踪性能，曲线的 presence 和 opacity 信号总是会被缓存。圆形曲线还实现了一项优化：如果在 "Os" 图元变量中指定了 opacity，那么计算阴影时会采用更高效的代码路径，从而避免运行任何 BxDF 或查询 opacity 缓存。

最小宽度

可以使用一个选项为曲线指定最小宽度。指定后，它会人为加宽曲线，使其至少达到指定的最小宽度（以像素计），同时相应降低 presence。效果是渲染出的曲线看起来大致相同，但高空间频率几何现在被带限，从而最大限度减少 aliasing 伪影；这意味着对曲线进行抗锯齿所需的摄影机采样更少（因此需要追踪的光线更少，性能更快）。有用的最小头发宽度值范围为 0.0 到 1.0。

最佳实践

渲染头发或毛皮时，建议：

  • 在适当情况下，通常应启用圆形曲线。这样会通过与圆柱形头发几何体相交，而不是与扁平缎带相交，确保在任何照明配置中都能获得高质量的头发阴影，从而得到明显更真实的遮挡。

  • 许多头发或毛皮线束应打包到相对较少的 Curves 图元中（例如每个 Curves 图元 10,000 条或更多线束），而不是渲染数百万个 Curves 图元且每个只由一条或少数几条头发组成。

  • 如果模型由多个在体积上大量重叠的曲线对象组成（例如外层毛皮和底层绒毛），当这些对象共享相同的变换、可见性标志和组成员关系值时，可获得最佳性能。
