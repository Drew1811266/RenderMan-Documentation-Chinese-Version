PxrWorley

Source: https://rmanwiki-27.pixar.com/space/REN27/542225558/PxrWorley
Downloaded: 2026-06-15T03:49:12+00:00

与所有 texture style nodes（纹理样式节点）一样，此节点接收一个 manifold，用来描述应用 Worley noise 纹理的 2D 或 3D 域。如果未连接 manifold，默认行为是在 3D 中沿 P 应用。此节点会计算 Worley noise，如 Steven Worley 所描述。

Input Parameters

Surface Position

noise 可基于 Current Position 或 Undisplaced Position （表面置换之前的位置）来计算。

如果你希望 displacement 和 shading patterns 匹配，请使用 Undisplaced Position。

Frequency

控制 cells 的尺寸。频率越高，cells 越小。

Distance Metric

用于测量到相邻 cells 距离的方式。Manhattan distance 会产生更矩形的形状，而 Euclidean distance 会产生更球形的形状。

Euclidean

计算到最近点的 Euclidean distance。它看起来比 Squared Euclidean distance 更尖一些。

Euclidean Squared

计算到最近点的 squared Euclidean distance。它看起来比纯 Euclidean distance 更圆润，也更有机。

Manhattan

受 Manhattan 网格式组织启发，这是当你只能沿 cell 边界移动时，到最近点的距离。

Chebyshev

命名自 Pafnutty Chebyshev，也称为 Chessboard Distance（棋盘距离）。它有点类似 Manhattan distance，但旋转了 45 度。

Minkowski

Minkowski 是 Euclidean 和 Manhattan distance 的一种泛化。该 exponent 会使图案从 Euclidean 过渡到 Manhattan。

Minkowski 比其他 distance metrics 更昂贵，但用于 displacement 时是可以接受的，因为几何体被置换时你只会支付一次成本。


Jitter

控制 cells 的扭曲。

C1

到第一个 feature（特征点）距离的乘数。

C2

到第二个 feature（特征点）距离的乘数。

Minkowski Exponent

使 distance 从 Manhattan (1.0) 平滑过渡到 Euclidean (2.0)，再到一些奇特的未探索区域。

Shape

修改计算出的 distances 以创建不同形状。下面示例使用 c1 = 1.0 和 c2 = 0.0。

Clamp Output

使生成的 distances 被 clamp 到 0.0 到 1.0 范围内。

c1: 1.0    c2: -0.95    distancemetric: Euclidean



Invert

反转最终 pattern。


Random Scale

这会随机缩放 features（特征点）的振幅，并产生稍微更规则的外观。

Random Scale Center

这会在应用 random scale 之前对 signal 应用一个 offset。用它可以创建更多变化。


Manifold

用于应用 noise 的 manifold。（默认值为 P）。

可以连接 3D 或 2D manifold。


Adjust Output

Color Scale

纹理颜色值的乘数，可用于调整亮度或操控单独的颜色通道


Color Offset

对结果应用 offset，使结果的颜色发生偏移


Float Scale

标量 Float 值

Float Offset

Float Offset 值


Output Parameters

resultF

Worley noise 纹理的结果。

resultRGB

以单色颜色表示的纹理。
