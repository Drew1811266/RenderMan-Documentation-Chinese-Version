很多时候，仅使用单个 subsurface 波瓣不足以达到照片级真实感，正如这张来自 The Irishman (© Netflix) 的图像所示，其中 LamaSSS 与 LamaTriColorSSS 结合使用，以便在匹配实拍底板时获得最大灵活性。

图像由 Industrial Light & Magic 提供。

此 MaterialX/Lama 节点目前在 RenderMan XPU 中不可用。

Lama Tricolor Subsurface Scattering 在几个重要方面的运行方式不同于 LamaSSS 。

在大多数情况下，LamaSSS 的结果已经足够好。不过，有些情况下用户可能希望以非物理方式控制散射颜色。LamaSSS 中的 Color 会按参数说明使用提供的 RGB 值来确定颜色散射。这意味着结果可能产生不希望出现的颜色偏移。假设你提供了 RGB 0.51 0.28 0.31 的纹理颜色，也就是一种粉色。

散射结果还会包含绿色（0.28）和蓝色（0.31）的响应。LamaSSS 材质会保持粉色。下面示例使用这个场景展示差异。注意 Path-traced mode（左）上的颜色偏移，而 Tricolor 版本会在已连接的所有深度保持纹理颜色。

Tricolor Parameters

这些层根据深度指定。Near Layer 是最上层，而 Far Layer 最深。

Near Layer

Near Layer 是顶部或最浅的层。

Weight

此项指定结果强度，并可使用标量（灰度）信号进行纹理化，以改变强度或遮罩区域。 


Color

Color 通常用于连接纹理或 pattern，以创建最终颜色；也可以直接选择单一颜色。


Radius

当光进入表面时，它可以在停止（被吸收或离开）之前传播一定距离。此控制项指定以 scene units 计的传播距离。较大的值允许光更深入表面，并增强该效果。这也可能让对象看起来更柔和，并且随着光散射越过细节，导致一些建模出的细小细节丢失。

Mid Layer

Mid Layer 是 Near Layer 之后的下一层深度。

此项指定结果强度，并可使用标量（灰度）信号进行纹理化，以改变强度或遮罩区域。

Scattering Parameters

Scale

这是上方 Radius 效果的简单乘数。它适合快速调整结果，尤其是在用纹理驱动 Radius 且不想重新绘制时。当在不同尺度的场景之间转移已着色资产时，它也很有用。

Follow Topology

这会在计算散射时考虑表面法线（形状和隐含细节）。这可能有助于保留原本会丢失的细节。


Subset

在这里设置值可以控制哪些对象可随着光从一个表面传播到另一个表面而彼此间接贡献。有时对象彼此穿插时会产生光晕或变暗，而此控制项会根据它们的值指定哪些对象参与贡献、哪些不参与。


Continuation Rays

有时你可能有相互嵌入的对象，例如骨骼或牙齿。如果这些对象导致伪影或不想要的结果，将此项设置为“on”会忽略内部对象。
