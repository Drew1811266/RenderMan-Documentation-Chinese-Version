PxrLayeredBlend

Source: https://rmanwiki-27.pixar.com/space/REN27/542224038/PxrLayeredBlend
Downloaded: 2026-06-15T03:49:12+00:00

此图案节点提供与 PxrBlend 类似的功能，但可以组合最多 8 个层，而不是只使用顶层和底层。它还包含一个额外的背景或基底层。

Input Parameters（输入参数）

每个层暴露以下参数，其中 [x] 是 0 到 7 之间的数字。各层按如下顺序排列：layer 7 位于底部，layer 0 是最顶层。

Enable [x]

这会启用 layer [x] 以供使用。

Operation [x]

用于将当前层与前一个层混合的混合模式。此参数的行为与 PxrBlend 的 operation 相同。

Color [x]

图像 layer [x] 的 RGB 输入。

Alpha [x]

图像 layer [x] 的 alpha 通道。当 alpha < 1 时，此图像层下方会使用黑色背景。

Background Color

底部或基底图像的 RGB 输入。

Background Alpha

底部或基底图像的 alpha 通道或遮罩。

Clamp Output

所有层的结果颜色都会被钳制到 1.0

Output Parameters（输出参数）

resultRGB

钳制后的颜色结果。

resultR

来自 resultRGB 输出的 R 通道。

resultG

来自 resultRGB 输出的 G 通道。

resultB

来自 resultRGB 输出的 B 通道。
