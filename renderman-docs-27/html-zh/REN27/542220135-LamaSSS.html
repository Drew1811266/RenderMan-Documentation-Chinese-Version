次表面散射是一种创建类似皮肤、蜡以及其他表面外观的方法；这些表面允许光线进入对象内部并发生散射。例如，蜡烛点燃时，火焰的光也会照亮蜡本身，使光线能够穿过蜡。这也是为什么当耳朵或手指对着手电筒时，你能看到光透过它们。以微妙的方式来说，这正是让皮肤显得柔和且真实的原因。

这里提供了几种光散射模型，因此你可以选择最适合目标的模型。

Color Parameters

Color

Color 通常用于连接纹理或图案，以创建对象的散射颜色。你可以在这里创建蜡烛蜡的颜色，或雀斑、嘴唇皮肤等皮肤变化。RGB 值对应光在该颜色波段中传播的距离。例如，RGB 值 0.8 0.65 0.5 表示红色传播最远，为 0.8；然后是绿色，最后蓝色在对象散射结果中传播距离最短。下表可能会有帮助。

Normal/Bump

在这里提供一个信号（纹理或程序化图案），用于在表面上创建凹凸，以“伪造”小凸起或划痕等表面细节。这意味着美术师无需对模型中这些繁琐且经常重复的部分进行建模。


Subsurface Parameters

Radius (DMFP)

当光进入表面时，它可以在停止（被吸收或离开）之前传播一定距离。此控制项指定以场景单位计的传播距离。较大的值允许光更深入表面，并增强该效果。这也可能让对象看起来更柔和，并且随着光散射越过细节，导致一些建模出的细小细节丢失。

此值会像调制光路一样调制 RGB 值，并以场景单位表示。例如，值 <0.5,0.4,0.2> 会让红光散射 0.5 个场景单位。这些值会乘以 Unit Length 属性；如果设置过高，它们会从表面逸出并产生意外结果。 

注意：SSS 默认关闭（0,0,0），并会创建漫反射。增大 Radius 可添加 SSS。

Scale

这是上方 Radius 效果的简单乘数。它适合快速调整结果，尤其是在用纹理驱动 Radius 且不想重新绘制时。当在不同尺度的场景之间转移已着色资产时，它也很有用。

Scale 示例：0.0 / 0.5 / 1.0 / 1.5 / 2.0 / 5.0


Mode

此项会改变用于计算散射效果的技术。有两个路径追踪版本是现代选择，但可能需要更长时间才能解析；还有一个较旧的扩散模型，常用于真实皮肤。请注意，可用参数会根据所选 Mode 而不同。

  • 路径追踪 Davis（非指数） - 此方法提供稳健的光传输方案来产生散射。该技术适用于薄几何体和边缘，可替代某些需要组合多种方法的情况。请注意，过大的 Radius 设置可能会在最终渲染中产生亮点噪声或亮点。Davis 或非指数模式会更明亮，因为光会被带入对象更深处。

  • 路径追踪指数模式 - 与上方 Davis 模式相同，但光线衰减更快，因此结果可能更暗。除云之外，此模型更符合现实中的典型情况。

  • Burley 扩散 - 在保留细节的同时产生准确效果。由于它不是路径追踪方法，可能看起来解析更快，但不太适合非常薄的对象，并且可能显得“斑驳”。不过，它在大多数其他用途中非常常见。你可以在这里找到关于此模型技术细节的更多信息 此处


路径追踪模式参数

IOR - 折射率

在某些情况下，进入对象的光可能会向后弯折并再次传播到表面。如果它携带足够能量，某些细节可能会因返回的光而显得“发光”。你可以调整 IOR 来弯折这些光线，直到效果受控并符合你的需求。

IOR 示例：1.0 / 1.2 / 1.3 / 1.4 / 1.5 / 2.0


专业提示

皮肤的 IOR 约为 1.4。


Anisotropy

光可以在表面内部向前或向后散射。使用各向异性时，可以让光沿光线方向更强地向前散射。这可能会产生让对象显得更透明的效果。

此属性在不同的反照率反演模型下行为略有不同。对于人类皮肤，使用 Pixar 的反照率反演模型时建议值为 0.7，使用 Chiang 反照率反演模型时建议值为 0.2。


Bleed (Davis Mode Only)

这可以增加光传播的距离，就像 Radius 控制项一样。不过，此控制项会尝试在让光散射得更远的同时保留更细小的细节。

Blend 示例：0.0 / 0.2 / 0.8 / 0.6 / 0.8 / 1.0


Burley 模式参数

Follow Topology

这会在计算散射时考虑表面法线（形状和隐含细节）。这可能有助于保留原本会丢失的细节。


通用参数

Subset

在这里设置值可以控制哪些对象可随着光从一个表面传播到另一个表面而彼此间接贡献。有时对象彼此穿插时会产生光晕或变暗，而此控制项会根据它们的值指定哪些对象参与贡献、哪些不参与。


Unit Length

此值应与你的 3D 应用中的单位匹配。例如，Maya 单位是厘米，因此 Unit Length 应设置为 0.1。

默认值为 0.003，最适合英尺。


Continuation Rays

有时你可能有相互嵌入的对象，例如骨骼或牙齿。如果这些对象导致伪影或不想要的结果，将此项设置为“on”会忽略内部对象。


高级参数

Albedo Inversion Method

  • Pixar - Pixar 使用的反照率反演模型，优先将反照率颜色作为最终结果。

  • Chiang - 更接近传统随机游走模型的反照率反演模型 发表于 2016 年。
