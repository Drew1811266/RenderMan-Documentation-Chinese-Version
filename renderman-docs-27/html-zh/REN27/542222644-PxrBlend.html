生成一种颜色：使用一种标准 Photoshop blend modes（Photoshop 混合模式），将上方图像层混合到底部图像之上得到结果。

Input Parameters（输入参数）

Operation（操作）

用于混合上方层和底部层的混合模式。下表描述了每种混合模式，表中的示例图像使用以下上方层和底部层创建。

上方层底部层
数值名称说明示例
0Color（颜色）创建一个结果颜色，它使用底色的亮度，以及混合色的色相和饱和度。这会保留图像中的灰阶，适合为单色图像上色，也适合为彩色图像着色。
1Color Burn（颜色加深）查看每个通道中的颜色信息，并通过提高两者之间的对比度，使底色变暗以反映混合色。与白色混合不会产生变化。
公式：1-(1-A) / B
2Color Dodge（颜色减淡）查看每个通道中的颜色信息，并通过降低两者之间的对比度，使底色变亮以反映混合色。与黑色混合不会产生变化。
公式：A / (1-B)
3Darken（变暗）查看每个通道中的颜色信息，并选择底色或混合色中较暗的一个作为结果颜色。比混合色更亮的像素会被替换，比混合色更暗的像素保持不变。
公式：min (A, B)
4Darker Color（较暗颜色）比较混合色与底色所有通道值的总和，并显示总值较低的颜色。Darker Color（较暗颜色）不会生成第三种颜色；这不同于 Darken（变暗）混合可能产生的结果，因为它会从底色和混合色中选择最低的通道值来创建结果颜色。
公式：A + B - 1
5Difference（差值）查看每个通道中的颜色信息，并根据哪一个具有更高亮度值，从底色中减去混合色，或从混合色中减去底色。与白色混合会反转底色值；与黑色混合不会产生变化。
公式：| A - B |
6Dissolve（溶解）结果颜色会随机替换为底色或混合色中的像素，具体取决于任意像素位置的不透明度。
7Divide（划分）

查看每个通道中的颜色信息，并用混合色去除底色。颜色除以自身会变为白色。

公式：A / B

8Exclusion（排除）创建类似 Difference（差值）模式但对比度更低的效果。与白色混合会反转底色值。与黑色混合不会产生变化。
公式：0.5 - 2(A - 0.5)(B - 0.5)
9Hard Light（强光）根据混合色，对颜色执行 Multiply（正片叠底）或 Screen（滤色）。效果类似用强烈聚光灯照射图像。如果混合色（光源）比 50% 灰更亮，图像会变亮，就像经过 Screen（滤色）一样。这适合为图像添加高光。如果混合色比 50% 灰更暗，图像会变暗，就像经过 Multiply（正片叠底）一样。这适合为图像添加阴影。用纯黑或纯白绘制会得到纯黑或纯白。
公式： (B > 0.5) * (1 - (1-A) * (1-2*(B-0.5))) + (B <= 0.5) * (A * (2*B))
10Hard Mix（实色混合）将混合色的红、绿、蓝通道值加到底色的 RGB 值上。如果某个通道的结果总和大于或等于 1，该通道取值为 1；如果小于 1，则取值为 0。因此，所有混合后的像素，其红、绿、蓝通道值都只会是 0 或 1。这会把所有像素变为加色原色（红、绿或蓝）、白色或黑色。
11Hue（色相）创建一个结果颜色，它使用底色的亮度和饱和度，以及混合色的色相。
12Lighten（变亮）

查看每个通道中的颜色信息，并选择底色或混合色中较亮的一个作为结果颜色。比混合色更暗的像素会被替换，比混合色更亮的像素保持不变。
公式：max(A,B)

13Lighter Color（较亮颜色）比较混合色与底色所有通道值的总和，并显示总值较高的颜色。Lighter Color（较亮颜色）不会生成第三种颜色；这不同于 Lighten（变亮）混合可能产生的结果，因为它会从底色和混合色中选择最高的通道值来创建结果颜色。
14Linear Burn（线性加深）查看每个通道中的颜色信息，并通过降低亮度，使底色变暗以反映混合色。与白色混合不会产生变化。
公式：A + B -1
15Linear Dodge（线性减淡）查看每个通道中的颜色信息，并通过提高亮度，使底色变亮以反映混合色。与黑色混合不会产生变化。
公式：A + B
16Linear Light（线性光）

根据混合色，通过降低或提高亮度来执行加深或减淡。如果混合色（光源）比 50% 灰更亮，图像会通过提高亮度而变亮。如果混合色比 50% 灰更暗，图像会通过降低亮度而变暗。
公式：(B > 0.5) * (A + 2*(B-0.5)) + (B <= 0.5) * (A + 2*B - 1)

17Luminosity（明度）创建一个结果颜色，它使用底色的色相和饱和度，以及混合色的亮度。此模式会创建与 Color 模式相反的效果。
18Multiply（正片叠底）查看每个通道中的颜色信息，并将底色乘以混合色。结果颜色始终是更暗的颜色。任何颜色与黑色相乘都会得到黑色。任何颜色与白色相乘都会保持颜色不变。
公式：A * B
19Normal（正常）将上方层以 alpha 合成方式叠加到底部层之上。
20Overlay（叠加）根据底色，对颜色执行 Multiply（正片叠底）或 Screen（滤色）。Patterns（图案）或颜色会叠加到现有像素上，同时保留底色的高光和阴影。底色不会被替换，而是与混合色混合，以反映原始颜色的明暗。
公式：(A > 0.5) * (1 - (1-2*(A-0.5)) * (1-B)) + (A <= 0.5) * ((2*A) * B)
21Pin Light（点光）根据混合色替换颜色。如果混合色（光源）比 50% 灰更亮，比混合色更暗的像素会被替换，比混合色更亮的像素保持不变。如果混合色比 50% 灰更暗，比混合色更亮的像素会被替换，比混合色更暗的像素保持不变。这适合为图像添加特殊效果。
公式：(B > 0.5) * (max(A,2*(B-0.5))) + (B <= 0.5) * (min(A,2*B)))
22Saturation（饱和度）创建一个结果颜色，它使用底色的亮度和色相，以及混合色的饱和度。在没有（0）饱和度的区域（灰色）用此模式绘制不会产生变化。
23Screen（滤色）查看每个通道的颜色信息，并将混合色和底色的反相相乘。结果颜色始终是更亮的颜色。Screening（滤色）与黑色混合会保持颜色不变。Screening（滤色）与白色混合会产生白色。
公式：1 - (1 - A) * (1 - B)    
24Soft Light（柔光）根据混合色使颜色变暗或变亮。效果类似用漫射聚光灯照射图像。如果混合色（光源）比 50% 灰更亮，图像会变亮，就像被减淡一样。如果混合色比 50% 灰更暗，图像会变暗，就像被加深一样。用纯黑或纯白绘制会产生明显更暗或更亮的区域，但不会得到纯黑或纯白。
公式：(B > 0.5) * (1 - (1 - A) * (1-(B - 0.5))) + (B <= 0.5) * (A * (B + 0.5))
25Subtract（减去）查看每个通道中的颜色信息，并从底色中减去混合色。
公式：A - B
26Vivid Light（亮光）根据混合色，通过提高或降低对比度来执行加深或减淡。如果混合色（光源）比 50% 灰更亮，图像会通过降低对比度而变亮。如果混合色比 50% 灰更暗，图像会通过提高对比度而变暗。
公式：(B > 0.5) * (1 - (1 - A) / (2*(B - 0.5))) + (B <= 0.5) * (A / (1- 2 * B))


Top Color（上方颜色）

上方图像层的 RGB 输入。

Top Alpha（上方 Alpha）

上方图像层的 alpha channel（alpha 通道）。

Bottom Color（底部颜色）

底部图像层的 RGB 输入。

Bottom Alpha（底部 Alpha）

底部图像层的 alpha channel（alpha 通道）。

Output Parameters（输出参数）

resultRGB

将上方图像层混合到底部图像之上所得到的颜色。

resultR

来自 resultRGB 输出的 R 通道。

resultG

来自 resultRGB 输出的 G 通道。

resultB

来自 resultRGB 输出的 B 通道。
