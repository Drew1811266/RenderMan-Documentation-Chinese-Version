在下方参数中，有些参数可在连接到 PxrLayer 后，通过 Input Material 被覆盖，或通过 PxrLayerMixer覆盖。 PxrLayerSurface 设计用于更清楚地展示哪些参数不能在层中被覆盖：它只包含全局参数。当你确定要进行分层时，建议使用这种材质。这些设置的结果保持不变。

  • Bold Face 参数是可分层的，可以按层覆盖。

  • Italicized 参数不能分层或覆盖。所有层都会全局遵循这些参数。例如：选择 GGX 作为 Specular 模型时，所有层在该参数上都会使用 GGX。


Glass Parameters

Glass 参数用于在一个位置创建并调整基于物理的玻璃结果。从清澈玻璃到彩色玻璃，以及介于其间的各种 Roughness 和雾化程度，都可以在这里设置。

Refraction 和 Reflection Gain 控制项设计为配合使用。单独使用其中一个可能会产生有趣效果，但这不是设计意图。还要注意，第一次创建玻璃时，你会发现 Diffuse 默认 Gain 1.0 可能会干扰外观。要获得纯玻璃，请将 Diffuse 的 Gain 调为 0.0。

Refraction Gain

Refraction Gain 是一个乘数。较低的数值会让更少光线穿过对象，可能呈现为深色染色玻璃；设为 0.0 时则会直接变黑。



Reflection Gain

Reflection Gain 是一个乘数。较低的数值会降低可见反射结果。设为 0.0 会产生不真实的外观，如下方第一张图所示。


Refraction Color

Refraction Color。它允许你创建具有真实着色、并像真实透射对象一样投射彩色阴影的对象（请注意，灯光上应启用 Thin Shadows）。



Roughness

Refraction 和 Reflection 的 Roughness。这里可用于模拟磨砂或蚀刻玻璃。



Refraction Roughness

设置为非负数时，允许 Refraction Roughness 不同于 由上方 Roughness 参数控制的 Reflection。下面示例中，Reflection 不粗糙（0.0），而 Refraction Roughness 会逐渐增加。


Refraction 2 Roughness

用于玻璃折射的额外 Roughness 参数。设置为非负数时，允许 Refraction Roughness 不同于 上方的 Roughness 和 Refraction Roughness 参数。如果此数值为 0.0，则会使用 Refraction Roughness 进行折射。


Refraction 2 Roughness Blend Factor

控制 Refraction Roughness 与 Refraction Roughness 2 之间的混合权重。0.0 到 1.0 之间的值表示 Roughness 是 Refraction Roughness 和 Refraction Roughness 2 的混合。值为 0.0 表示 Roughness 等于 Refraction Roughness。值为 1.0 表示 Roughness 等于 Refraction Roughness 2。


Refraction 2 Tint

此参数为 Refraction 2 Roughness 设置颜色（作为使用 Refraction 2 Roughness 生成的样本上 Refraction Color 的乘数）。


Refractive Index

折射率。默认值为 1.5，这是玻璃的典型值。在线 Ior 值资源可以提供更有用的设置。较高的 Ior 值会比低值更强地弯折光线。如果想避免弯折光线，请启用 Thin 模式。



Enable Manifold Walk

该设置允许玻璃对象为焦散光路进行探索，需配合 PxrUnified 积分器以及 Manifold Walk 选项使用。


Manifold Walk IOR

该参数允许用户以艺术方式覆盖 Glass 波瓣中设置的折射率，用于结合 PxrUnified 积分器中的 Manifold Walk 控制项计算焦散光路。


Thin

如果开启，会根据 Refractive Index 在 Reflection 和 Refraction 之间正确分配能量，但不会在 Refraction 中弯折光线（用单层玻璃模拟双层玻璃，或模拟中空玻璃装饰物）。下方的 内部 参数不适用于该模式。



Ignore Accumulate Opacity

忽略积分器中基于玻璃不透明度/折射收集 alpha 值的设置。


Ignore Fresnel Weighting

关闭 Reflection，但允许渲染纯 Refraction/Transmission（折射/透射）。这对于合成很有用。


Blocks Volumes

当体积进入玻璃对象内部时，此设置会从渲染中省略或阻挡这些体积。它还会禁用 Glass 波瓣的内部设置。默认值为关闭（左侧），右侧显示开启效果。


Interior Parameters

Single Scatter Albedo

Single Scatter Albedo 颜色会在玻璃表面内部创建雾。这对于含有淤泥或脏污（均匀）水的容器很有用。 Extinction 不应为黑色或 0。当 Refraction Gain 非零时，这些设置会被使用/可用。纹理化需要 3D Manifold 才能工作。

Interior Parameters 设计为便于与 PxrPathTracer 一起使用。例如，使用 PxrVCM 以及灯光中的 Trace Light Paths 时，此效果在当前版本中无法渲染。


Extinction

Extinction 颜色。它应用于上方的 Single Scatter Albedo。要让 Single Scatter Albedo 生效，它不应为黑色。该参数选择最受消光影响的颜色或颜色组合。选择纯红色表示红色会被消光，但绿色和蓝色组合不会。下面有两个示例：左侧是从 0 增加到 4.0 的 RGB 单色 Extinction 颜色。注意这里超过了 1.0，因为比例基于场景尺度。右侧是为效果旋转颜色值；请记住，由于是消光，最终颜色会位于所选颜色的色轮对面。较高数值也会让效果更厚或更不透明，可用于模拟脏水，也许还可以模拟橙汁。同样，这些值基于场景尺度，所以你可能会发现自己会使用超过 1.0 的值。

Scattering Directionality

控制散射的方向性。在下面的动画中，我们将 Directionality 从 -0.9 旋转到 0.9。当光源大多位于我们身后时，负值会让外观看起来更实，因为光会向我们反向散射。

0：isotropic

1：forward

-1：backward

Multiscattering

允许多重散射效果；随着更多光线反弹在体积内部被计算，可能会让体积更亮。


Overlapping Volumes

允许内部和外部体积重叠，就像它们穿过对象一样。
