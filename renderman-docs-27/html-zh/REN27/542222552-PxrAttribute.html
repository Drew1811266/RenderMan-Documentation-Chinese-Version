PxrAttribute

Source: https://rmanwiki-27.pixar.com/space/REN27/542222552/PxrAttribute
Downloaded: 2026-06-15T03:49:12+00:00

PxrAttribute 允许用户读取附着（存储）在节点上的属性。例如，可以给一组对象添加颜色属性，稍后由材质读取。这样，材质可以根据正在渲染的对象改变结果，而无需为不同对象使用不同材质。下面材质球中的球体附着了颜色属性。单个 PxrSurface 材质 会根据每个形状所定义的属性值，以不同的漫反射颜色渲染。下面提供使用示例。

在场景中许多位置使用此节点可能会带来性能开销。若无必要，应避免过多求值用户属性；效率是关键。


输入参数

Variable name（变量名称）

此字段接受一个用于标识属性的字符串。该字符串应包含属性命名空间和属性名称，两者用冒号分隔。例如 trace:maxdiffusedepth 或 user:Ball。

Variable Type（变量类型）

这会指定要读取的变量类型，并且必须与前面在其他节点上指定的类型匹配。

  • Integer（整数）
  • Float（浮点）
  • Float 2（双浮点）
  • Color（颜色）
  • Point（点）
  • Vector（向量）
  • Normal（法线）

Default Float（默认浮点值）

如果检索 float 类型的属性时出错，将使用此值。

Verbosity（详细程度）

用于场景调试的诊断输出级别

  • Silent（静默）
  • Warnings（警告）
  • Info（信息）

输出参数

resultF

float 结果。

resultRGB

color 结果。


示例用法

DCC 应用程序可能使用不同机制来应用用户属性。下面是两个示例，演示如何把名为 “Ball” 的 color 类型用户属性应用到一个形状上：

Maya：

从 Maya Attribute Editor 的变换节点添加用户属性，Name: Ball，Type: color，Color: Light Pink。下面是生成的导出结果；请注意，我们会自动假定这是来自 Maya Editor 的用户属性，因此无需显式指定。

RiAttribute "user" "color Ball" 1 0.2 0.65

Katana：

下面是在 Katana 中设置相同属性的 OpScript 示例

gb = GroupBuilder()
gb:set("value", FloatAttribute({1.0, 0.2, 0.65}, 3))
gb:set("type", StringAttribute("color"))
Interface.SetAttr("prmanStatements.attributes.user.Ball", gb:build())

Houdini：

请参见 使用 PxrMatteID，作为如何在 Houdini 中添加用户属性的示例。
