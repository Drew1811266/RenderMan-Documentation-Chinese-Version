PxrBumpMixer

Source: https://rmanwiki-27.pixar.com/space/REN27/542222881/PxrBumpMixer
Downloaded: 2026-06-15T03:49:12+00:00

此节点提供一种现代方式来生成并分层 bump maps（凹凸贴图），用 shading（着色）而不是建模来添加细小表面细节，成本较低。

传统选项不允许 order-independent mixing（与顺序无关的混合），而此节点可在需要时轻松混合多个输入。你也可以把这些节点串接起来，构建更复杂的网络，通过 masking（遮罩）在表面上改变 bump maps，同时只使用一个 material assignment（材质指定）。



提供 resultNG 输出的节点可以连接到此节点，例如 PxrTexture 和 许多 procedural patterns（程序化图案）。 这种新的 NG vector type 表达 signal 的 gradient（梯度），比传统 bump map 技术计算起来更准确、更高效。

此技术实现了以下论文中的许多发展： Surface Gradient-Based Bump Mapping Framework。

Input Parameters（输入参数）

Mode（模式）

  • Add（相加） - 将结果相加，A + B

  • Over（叠加） - 将 A 置于 B 之上；mask（遮罩）在这里很有用，可用来显露下方图层，A+B(1-a)

  • Subtract（减去） - Subtract（减去）结果，A - B

Surface Gradient（表面梯度）

将你的 pattern 或 texture 节点的 resultNG 连接到此处。

Mask（遮罩）

一个简单的 grayscale（灰阶，scalar 标量）pattern 或 texture channel（纹理通道）会用于遮罩你希望此特定图层生效的位置。使用 Over 模式时，这尤其有用。

Amount（强度）

这表示你希望此图层的 bump effect（凹凸效果）有多强，可使用 pattern 或 texture channel 等 scalar input（标量输入）进行控制。

Advanced（高级）

Adjust Amount（调整量）

有时你可能会在表面上看到变暗现象，可能出现在 horizon（地平线/掠射边界）处，此时 normal（法线）可能背向摄像机。此控制项可以把这些法线推回朝向摄像机，以避免这种 artifact（伪影）。

Surface Normal Mix（表面法线混合）

此节点提供的 shading normals（着色法线）可以覆盖你的 surface geometry normals（表面几何法线）；如果你希望把这些 surface normals（表面法线）混入其中，可以使用此控制项。
