Using Manifold Walk

Source: https://rmanwiki-27.pixar.com/space/REN27/542232143/Using+Manifold+Walk
Downloaded: 2026-06-15T03:49:12+00:00

Manifold Walk（流形行走）选项在 PxrUnified 中使用一种技术来生成焦散，其开销可能低于在场景中生成光子。它在使用小型光源时也表现出色。你还可以细调哪些是接收焦散或生成焦散的 材质。这对于 VFX 和长片动画工作中的眼睛尤其有用。你可以选择性地让角膜把焦散追踪到眼睛的虹膜上，从而增加真实感和 beauty（美感）效果。你也可以更真实地照亮位于透射对象后方或内部的物体。

下面我们通过一个简单场景说明使用 Manifold Walk（流形行走）的步骤。

在下面的示例中，我们有一个简单的地面平面和一个玻璃圆柱体。地面平面是漫反射的，圆柱体是玻璃材质。使用 Thin Shadows（薄阴影） 的默认渲染会产生以下结果：

Thin Shadows（薄阴影，默认行为）

这个结果可以接受，但我们希望穿过圆柱体聚焦的光具有一些形状。因此按以下步骤操作：

  1. 在地面的 diffuse（漫反射） PxrSurface 材质的 Global Parameters（全局参数）下， 打开 Enable Manifold Exploration（启用 Manifold Exploration，mwStartable = 1；开启流形探索）

  2. 在圆柱体上的玻璃 PxrSurface 材质中，在 Glass > Advanced（玻璃 > 高级）里 打开 Enable Manifold Walk（启用 Manifold Walk（流形行走），mwWalkable = 1）

  3. 在光源中（这里仅有一个光源，所以很简单，但你也可以按每盏灯单独设置）， 打开 Manifold Walk（流形行走）（cheapCaustics = 1）

  4. 在 PxrUnified 中， 启用 全局 Manifold Walk（流形行走）选择项（manifoldWalk = 1）

  

Manifold Walk（流形行走）正确设置

灯光提供了一个位置，可用于为 manifold walk（流形行走）排除特定子集。当某些对象可能阻挡 manifold walk（流形行走）找到光源时，就可以使用这个设置。例如，角色眼前垂下的头发可能让这个行走过程难以透过头发找到光源。你可以排除头发，让这个过程更容易。
