PxrTangentField

Source: https://rmanwiki-27.pixar.com/space/REN27/542224996/PxrTangentField
Downloaded: 2026-06-15T03:49:12+00:00

定义一个切线场，通常用于控制各向异性方向。

  • InputRotation 采用单一灰度通道，其中黑色表示 0 度，白色表示 360 度。
  • InputVector 采用 2 通道贴图，其中红色通道为 x，绿色为 y。蓝色通道会被忽略。

向量方法通常通过减少纹理过滤伪影获得更好的结果。如果这些伪影过于明显，请使用 nearest 纹理过滤器。

nearest 过滤器会完全禁用 MIP 映射，因此只应将它用于低分辨率贴图，以免降低渲染器速度并消耗过多纹理内存。

Input Parameters

Input Rotation

用于旋转着色切线的单个 float 值。此输入优先于 inputVector。

Input Vector

红色和绿色通道会被读取为 2D 向量的 x 和 y 坐标。如果连接了 inputRotation，则会忽略此输入。

Rotation Offset

全局旋转偏移，以度为单位。

Rotation Range

缩放旋转范围。1.0 等同于 0 到 360 度。

Centered Vectors

将向量贴图的渐变值居中。使用 inputRotation 时忽略。

Output Parameters

resultXYZ

应连接到 Bxdf 的 Shading Tangent 输入的最终切线向量。

示例纹理

示例纹理的分辨率为 256x256 像素，并且应能无缝平铺（使用适当的 txmake 包裹模式，例如 txmake -mode periodic).

 

TextureDescription
用于创建各向异性拉丝金属图案的旋转贴图。
用于创建各向异性拉丝金属图案的向量贴图。
用于创建各向异性纤维网格图案的向量贴图。
用于创建各向异性碳纤维网格图案的向量贴图。
