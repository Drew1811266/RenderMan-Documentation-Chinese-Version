Sides（边数）
diaphragm blades（光阑叶片）的数量从技术上并不控制 bokeh（散景），但会控制 blur circle（模糊圆）的形状（而不是光在其中的分布方式）。改变边数会让原本完美圆形的高光按指定边数产生分面（例如三条边会形成三角形）。

Angle（角度）
此项会旋转 aperture（光圈），从而旋转失焦光点。

Roundness（圆润度）
Roundness 会弯曲 aperture blades（光圈叶片）的边缘，使它们看起来更圆润。因此高光也会呈现更圆润的形状。

Density
“good bokeh（优质散景）”在每个失焦光点中中心更亮，并向边缘衰减。不过并不是每个镜头都有 good bokeh。可以使用这个控制项模拟镜头中好的（或差的）光照衰减。默认值 0 会给出恒定亮度（neutral bokeh，中性散景）；-1 会让中心更亮并向边缘衰减（good bokeh）；1 会给出明亮边缘（bad bokeh，劣质散景）。
