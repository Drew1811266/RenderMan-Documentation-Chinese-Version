PxrWireframe

Source: https://rmanwiki-27.pixar.com/space/REN27/542225547/PxrWireframe
Downloaded: 2026-06-15T03:49:12+00:00

此 pattern 用于将几何线框作为 pattern assignment（图案分配）绘制出来。它可以与材质混合，也可以在 constant shader（常量 shader）中用作平面纹理。

Input Parameters

Wireframe Color

绘制的线的颜色

Background Color

线框下方表面的颜色

Wireframe Opacity

绘制线条的不透明度或可见性

Wireframe Width

绘制的线的宽度或粗细

Output Parameters

resultRGB

由该 pattern 生成的颜色分量（RGB）值。
