PxrExposure

Source: https://rmanwiki-27.pixar.com/space/REN27/542223444/PxrExposure
Downloaded: 2026-06-15T03:49:12+00:00

按给定的曝光档调整输入颜色的曝光。每个正曝光档会使输入强度加倍。每个负曝光档会使输入强度减半。通常最好使用 Exposure，而不是直接乘法（也称为增益），因为它在感知上呈线性。

Input Parameters（输入参数）

Input Color

要进行曝光调整的颜色。在此连接一个颜色 pattern。

Stops

用于调整 inputRGB 的曝光档数量。结果会乘以 2 的 stops 次方。

Output Parameters

resultRGB

钳制后的颜色结果。

resultR

来自 resultRGB 输出的 R 通道。

resultG

来自 resultRGB 输出的 G 通道。

resultB

来自 resultRGB 输出的 B 通道。
