PxrDisneyBsdf

Source: https://rmanwiki-27.pixar.com/space/REN27/542220743/PxrDisneyBsdf
Downloaded: 2026-06-15T03:49:12+00:00

这个 Bxdf 基于 Brent Burley 的扩展材质模型，该模型发表于“Extending the Disney BRDF to a BSDF with Integrated Subsurface Scattering”，SIGGRAPH 2015。  

PxrDisneyBsdf 是一个清晰直接的基于物理的模型，参数数量精简，同时可覆盖很宽范围的照片级真实和艺术化材质，包括漫反射与镜面反射和透射、清漆涂层、织物光泽，以及次表面散射。  

Walt Disney Animation Studios 制作的所有 CG 动画都使用这个材质模型；该材质的 Pixar 版本 PxrDisneyBsdf 的源代码可在 bxdf 代码示例中找到。

PxrDisneyBsdf 不同材质参数设置示例：金属反射、薄表面漫反射透射、薄表面镜面透射、次表面散射，以及玻璃折射和反射。  图像灵感来自 Burley BSDF 论文的图 1。



Basic

Base Color

主表面颜色，通常由纹理贴图提供。

Emit Color

发光能量。非黑色值表示辉光。请谨慎使用此参数。只将它用于低发光对象和材质。对于更亮的对象，始终使用 PxrMeshLight。

Specular Reflection

Metallic

定义材质的金属度。0 表示电介质，1 表示完全金属。PBR 金属度贴图可用作输入。
这是两个不同模型之间的线性混合。金属模型没有漫反射分量，并且其带色的入射镜面反射等于基础颜色。

Specular Tint

这是为了艺术控制而提供的折衷项，会将入射镜面反射朝基础颜色着色。掠射角镜面反射仍保持无色（即不会给入射光着色）。

Roughness

定义漫反射与镜面反射响应之间的表面粗糙度。同样，通常会使用 PBR 粗糙度贴图作为输入，以添加粗糙度变化。

Anisotropic

各向异性的程度控制镜面高光的纵横比。0：各向同性；1：最大各向异性。

Index of Refraction

材质的折射率，用于控制折射，也控制镜面反射量。（它还会影响次表面散射。）

Specular Reflection Scale

控制镜面反射强度；0 会移除它，1 会保持完整的镜面反射响应。通常不需要此参数，因为在物理正确的方式下，镜面响应由光的入射角以及 Roughness 和 IOR 参数决定，但此参数可以覆盖该结果。

Clearcoat

第二个专用镜面波瓣，适合为材质添加清漆或清漆涂层。

Clearcoat Gloss

控制清漆光泽度。0 产生模糊高光，1 产生紧致高光。注意：此参数与 Roughness 的含义方向相反。

Sheen

Sheen

额外的掠射角/前向散射漫反射分量，主要用于布料。 

Sheen Tint

将织物光泽朝基础颜色着色的量。为 0 时，织物光泽是无色的（即不会给入射光着色）。

Transmission

Diffuse Transmission

透射而不是反射的漫反射光量。0 表示 0% 透射、100% 反射；1 表示 50% 透射、50% 反射；2 表示 100% 透射、0% 反射。

Specular Transmission

入射（最大）镜面透射量。

Thin Surface 

表面应被视为薄表面，还是形成厚材质的界面。设置为 Thin 时，折射率为 1。

Transmission Color

用于体积吸收的透射光颜色（Beer 指数消光定律）。

Transmittance Distance

吸收达到指定透射率颜色时的距离。有时也标为 "atDistance"。距离越小，透射率颜色越饱和。

Subsurface Scatter

Subsurface

控制次表面散射量。

Subsurface Color

根据 Subsurface 参数与漫反射响应混合的颜色。

Scattering Distances

红、绿、蓝在次表面中发生散射的平均距离。如果为 0，则回退到漫反射 BRDF。

Subsurface Anisotropy

路径追踪次表面散射中的 Subsurface 体积各向异性（散射的平均余弦）。0 表示各向同性散射，0 到 1 之间的值表示以前向散射为主，-1 到 0 之间的值表示以后向散射为主。

Other parameters

Transmission Behavior

为阴影光线选择透射行为：opaque 或使用 baseColor。

Presence

在这里连接遮罩函数，为对象应用镂空 Pattern。Presence 定义为二值（0 或 1）函数，但可以采用连续值来对形状进行抗锯齿。它适合用于建模树叶和其他薄而复杂的形状。

重要提示：此值应为 0 或 1。介于 0 和 1 之间的值会产生不需要的噪声！！！

Bump Normal

用于带凹凸法线的可选连接点。

Shadow Bump Terminator

使用强凹凸或法线贴图时，隐藏突兀的阴影终止线。

Input AOV

在这里插入 MatteID 节点。
