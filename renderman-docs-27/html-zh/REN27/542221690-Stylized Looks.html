Stylized Looks（风格化外观） 仅面向 RenderMan 的商业客户提供。

创新的 Stylized Looks 工具集让 RenderMan 能够生成超越照片写实的图像。艺术家可以渲染出看起来像卡通、插画或水彩的图像。这个灵活系统可用于创建独特外观，唯一限制只是想象力。

Stylized Looks 亮点：

在 RenderMan 中渲染，再使用 Stylized Looks 后处理

Stylized Looks 是一种逐像素后处理，会在图像渲染完成后（或 IPR 会话期间）应用，由 3 种不同 Display Filter（PxrStylizedToon / PxrStylizedLines / PxrStylizedHatching）处理。
使用强大的 RenderMan 工具集中所有工具创建场景，包括 PxrSurface、Lama 和 RenderMan 灯光，然后将其转换为 Stylized Looks！

在一次渲染中 Mix（混合）照片写实与风格化！

使用 RenderMan Lights 控制照片写实与风格化之间的过渡，创建出色图像。 
例如，使用 PxrRectLight 的光束为照片写实渲染添加排线。光线照到哪里，哪里就会应用排线。

创建你自己的纹理集

Stylized Looks 不仅随附多个纹理预设，也允许你创建并使用自己独特的纹理集。
需要查看某种具体 NPR 效果？没问题，创建你自己的排线纹理集！

Toon + Illustrative/Hatching Lighting + Lines

Toon：按每盏灯量化任意信号（diffuse、spec、refraction 等），并可控制阶数
Illustrative Hatching Lighting（插画式排线照明）： 根据灯光或摄影机深度动态生成，用于插画式光影渲染
Lines：线稿功能，可按灯光控制线条粗细，按自定义逐对象设置控制线条粗细；线条颜色可由反照率、灯光或自定义逐对象设置控制

Mix & Match Styles（混合搭配样式）

混合搭配 PxrStylizedToon / PxrStylizedLines / PxrStylizedHatching Display Filter，创建独特而富有创意的组合
尝试叠加多个 PxrStylizedLines，每个使用不同厚度，以创建漂亮且具有艺术感的内部和外部线稿。

全局和逐对象控制

将 PxrStylizedControl 附加到 PxrSurface 和 LamaSurface 节点，以控制整个场景或逐对象控制。
控制特定对象的不同线条粗细和着色

导出 AOVs 以进行额外精修

与 Stylized Looks 关联的所有 AOVs 都可以导出，以便在 RenderMan 外部进一步进行创意精修和分层处理
