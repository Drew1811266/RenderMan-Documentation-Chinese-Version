PxrManifold2D

Source: https://rmanwiki-27.pixar.com/space/REN27/542224071/PxrManifold2D
Downloaded: 2026-06-15T03:49:12+00:00

此节点允许艺术家使用 2D 方案放置图案；该方案通常依赖艺术家提供的 UV。你可以变换生成的放置（translate、rotate 等），也可以使用下面的控制项重复图案。对于没有 UV 或需要 3D 空间投射的对象，请改用 PxrManifold3D 即可。

Input Parameters

Center

变换以此为中心。

Angle

围绕原点的旋转角度。

Frequency S

特征在 S 方向上的频率/重复。

Frequency T

特征在 T 方向上的频率/重复。

Offset S

从原点沿 S 方向移动。

Offset T

从原点沿 T 方向移动。

Invert S

在 S 方向上翻转流形。

Invert T

在 T 方向上翻转流形。

Warp

连接噪声或纹理来扭曲域

Warp Amount

用于控制 Warp Amount 的滑块。

Advanced

calcRadius

通过导数以较复杂方式计算半径，以绕过渲染器在半径计算中的错误。

enableJacobianfilter

启用基于屏幕空间中的 Jacobian 计算过滤半径。这应创建一个较少依赖几何体的参数化过滤器。

JacobianRadiusMult

将基于屏幕空间中的 Jacobian 计算出的半径相乘。

uvSet

特定的表面 UV 集。也有助于向毛发添加纹理 primvar。

Output Parameters（输出参数）

result

输出的 2D 流形。

resultS

流形的 S 分量的 float 表示。

resultT

流形的 T 分量的 float 表示。
