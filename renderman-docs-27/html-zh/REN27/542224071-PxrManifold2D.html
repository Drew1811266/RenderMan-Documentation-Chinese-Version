PxrManifold2D

Source: https://rmanwiki-27.pixar.com/space/REN27/542224071/PxrManifold2D
Downloaded: 2026-06-15T03:49:12+00:00

此节点允许艺术家使用 2D 方案放置 pattern；该方案通常依赖艺术家提供的 UVs。你可以变换生成的放置（translate、rotate 等），也可以使用下面的 controls 重复 pattern。对于没有 UVs 或需要 3D 空间投射的对象，请改用 PxrManifold3D 即可。

Input Parameters

Center

变换以此为中心。

Angle

围绕 origin 的 Rotation angle（旋转角度）。

Frequency S

特征在 S 方向上的 Frequency/repetition（频率/重复）。

Frequency T

特征在 T 方向上的 Frequency/repetition（频率/重复）。

Offset S

从 origin 沿 S 方向移动。

Offset T

从 origin 沿 T 方向移动。

Invert S

在 S 方向上翻转 manifold。

Invert T

在 T 方向上翻转 manifold。

Warp

连接 noise 或 texture 来 warp（扭曲）domain

Warp Amount

用于控制 Warp amount 的 slider。

Advanced

calcRadius

通过 derivatives 以较复杂方式计算 radius，以绕过 renderer 在 radius calculation（半径计算）中的 bug。

enableJacobianfilter

启用基于屏幕空间中的 Jacobian 计算 filter radius（过滤半径）。这应创建一个较少依赖几何体的参数化的 filter。

JacobianRadiusMult

将基于屏幕空间中的 Jacobian 计算出的 radius 相乘。

uvSet

特定的 surface UV set（表面 UV 集）。也有助于向毛发添加 texture primvars。

Output Parameters（输出参数）

result

输出的 2D manifold。

resultS

manifold 的 S 分量的 float 表示。

resultT

manifold 的 T 分量的 float 表示。
