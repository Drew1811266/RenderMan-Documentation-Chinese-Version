Windows License Server

Source: https://rmanwiki-27.pixar.com/space/REN27/542212629/Windows+License+Server
Downloaded: 2026-06-15T03:49:12+00:00

如果你使用 RenderMan Installer 安装了 License Server，那么所需的 pixar.license 文件已经位于

C:\Program Files\Pixar\pixar.license

如果你是通过下载并运行 .msi 安装器进行安装，那么安装完成后，请运行 Pixar LicenseApp；它通常位于这里：

C:\Program Files\Pixar\PixarLicense-LA-23.0\LicenseApp\bin\LicenseApp.exe

这会帮助你下载并安装许可证服务器的许可证文件。当 LicenseApp 发现没有许可证文件时，界面会如下所示：

点击 "Get License" 按钮，为许可证服务器下载新的许可证文件。它会把此许可证文件放在：

C:\Program Files\Pixar\pixar.license

请注意，Windows 上许可证服务器使用的默认端口是 9010。你的管理员应确保此端口未被阻止/防火墙拦截。

如果你确实有许可证，但许可证服务器尚未启动，RenderMan 将报告许可证错误：

R50005 {SEVERE}  License error - code 111:  connect Connection refused
R50005 {CONTINUED} license source: 9010@mylicserver

如果随后运行 LicenseApp，它同样会报告无法联系许可证服务器。

LicenseApp 是一个有用工具，可指示许可证服务器当前状态；它也可以在客户端机器上运行，并会远程查询服务器状态。

在 Windows 上安装时，服务会由安装器自动安装并启动。

启动和停止 License Server

要启动许可证服务，请打开 Start Menu，输入 Services 并选择 Services：

该 Services 控制面板会显示 Windows 上的所有服务，请找到 Pixar License Server 23.0 对应的服务：

 

在此窗口中，你可以启动、停止、暂停和重启许可证服务。

当 License Server 正常运行时，LicenseApp 程序会显示许可证功能和数量信息：

RenderMan 和其他插件随后应会按预期运行。

默认情况下，许可证服务器安装在 C:\Program Files\Pixar\PixarLicense-LA-<version>，并包含以下组件和脚本：

    • PixarLicenseServer - 许可证服务器程序

    • LicenseApp\bin\LicenseApp.exe - 许可证状态应用程序

    • queryPRMan.py - 用于查询许可证服务器的示例 Python 脚本
