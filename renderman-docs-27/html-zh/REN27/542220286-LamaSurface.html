所有 MaterialX Lama 材质都需要先将 LamaSurface node 附加到你的几何体上。 

LamaSurface 允许你创建材质，并为表面的正面和背面朝向定义不同材质。 

Material Properties

Material Sides

定义材质拥有的面数：1（默认）或 2；当为 2 时，背面可以拥有不同的材质外观。


上方示例展示了正面使用叶片材质、背面使用金色材质，从而创建出 ... GoldLeaf

Material Front

这是定义正面朝向结果的材质。

Material Back

如果 Material Sides 设置为 2，此连接可用，并定义表面背面的外观。

Presence

如果在下方 Features 中启用，此连接可用。此连接允许你使用 pattern 连接定义表面的可见性。

Features

Compute Presence

允许创建并连接 presence 设置或连接的 pattern，用于控制对象可见性。

Compute Opacity

这会启用或禁用透射材质（如玻璃）的 opacity 计算。关闭它可以加快可能的草稿或预览渲染。

Presence / Opacity Cached 

指定 renderer 是否应缓存 presence（以及 opacity）。缓存这些属性会加快 trace，尤其是 shadow queries，但相比未缓存的 presence/opacity，结果可能不那么锐利。（请注意，勾选 subsurface 或 interior 计算都会自动禁用 presence/opacity 缓存。）

Computer Subsurface

这会启用或禁用 subsurface scattering 计算（使用时）。关闭它可以加快草稿或预览渲染。

Compute Interior

这会启用或禁用透射材质（如玻璃）的 interior 计算。关闭它可以加快可能的草稿或预览渲染。如果启用，还需要启用 Compute Opacity 控制项。
