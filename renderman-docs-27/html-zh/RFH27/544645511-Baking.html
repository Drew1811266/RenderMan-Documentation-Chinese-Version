烘焙输出纹理只需两个简单步骤。

请参阅 烘焙 了解更多信息。

用户应为要进行烘焙的对象提供 UV（UV 映射）。

Set Bake Hider（设置 Bake Hider）

不要使用 raytrace，而是将其设为 bake（烘焙）。


Add PxrBakeTexture（添加 PxrBakeTexture）

在 RIS Network（RIS 网络）中，添加一个 PxrBakeTexture（纹理烘焙节点） VOP。将其输出连接到 PxrSurface color 参数，以便它会被求值。

在 PxrBakeTexture VOP 中，设置 output FileName（输出文件名）。请参阅 PxrBakeTexture（纹理烘焙节点） 了解每个参数的说明。
