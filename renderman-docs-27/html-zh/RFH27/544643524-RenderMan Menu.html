RenderMan 菜单

Source: https://rmanwiki-27.pixar.com/space/RFH27/544643524/RenderMan+Menu
Downloaded: 2026-06-15T03:49:12+00:00

加载 RenderMan for Houdini 插件后，Houdini 中会添加 RenderMan 菜单。

  • IPR Render（IPR 渲染）： 从所选 ROP 节点启动一个交互式渲染会话

  • Preview Render（预览渲染）： 触发一次预览渲染

  • Render in Background（后台渲染）： 从所选 ROP 节点开始在后台批量渲染到图像

  • Bake Render（烘焙渲染）： 使用 PxrBake 节点在后台执行烘焙操作

  • View Scene RIB（查看场景 RIB）： 查看当前场景的 RIB 文件

  • Stop Render（停止渲染）： 结束渲染进程

  • Flush Texture Cache（清空纹理缓存）： 从缓存中清空所有纹理

  • Create Light（创建灯光）： 快速创建 RenderMan 灯光类型

  • Create Light Filter（创建灯光过滤器）： 快速创建 RenderMan 灯光过滤器类型

  • Create Material（创建材质）： 快速创建 RenderMan 材质

  • Create Integrator（创建积分器）： 从列表中为 ROP 节点创建一个积分器

  • Create Display Filter（创建显示过滤器）： 从列表中创建一个 Display Filter

  • Create Sample Filter（创建采样过滤器）： 添加一个 Sample Filter

  • Subdivide（细分）： 为所选对象添加并启用细分属性

  • Stylize（风格化）： 选择此选项会将 PxrStylizedControl 节点连接到所选对象的材质

  • Aggregate Volume（聚合体积）：在场景中创建聚合体积设置

  • Holdout（遮挡）： 为所选 ROP 节点设置 AOV，以导出带阴影的遮挡通道

  • Open Image Tool（打开图像工具）： 打开 RenderMan Image Tool “it”，用于查看并编目渲染结果

  • Preferences（偏好设置）： 打开 RfH 偏好设置窗口

  • Update Lama Nodes（更新 Lama 节点）： 对于 RfH 版本早于 25 的场景，如果使用早于 H19.5.501 的 Houdini 版本，需要手动运行一次顶部菜单 “RenderMan” > “Update Lama Nodes”（否则会自动执行）。你还需要手动更新自定义锁定 HDA。

  • Help（帮助）： RenderMan 帮助文档

  • About RenderMan（关于 RenderMan）： RenderMan 版本信息和法律声明


如果自定义顶层菜单时移除了 Help Menu 或 RenderMan - RIS 工具，RenderMan 菜单工具将无法正确安装。需要在 MainMenuCommon.xml 配置文件中编辑 <insertBefore> 和 <insertAfter> 选项，指定插入位置，才能让这些工具在自定义菜单中正确定位。
