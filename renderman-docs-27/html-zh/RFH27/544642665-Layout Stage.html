RenderManStudio_Layout_v01.hip

这里是完成布局工作的场景。  

该 $HIP/usd/Assembly/RenderManStudio_Assembly_v01.usda 文件会被引用；然后同时使用 setvariant 和 edit 节点来摆放资产并形成场景。 

你也可以在这里放置一台相机，但如果不想保存这台布局相机，别忘了添加一个 layerbreak 节点。



对布局满意后，使用 usd_Layout USD ROP 保存布局文件 $HIP/usd/Layout/RenderManStudio_Layout_v1.usda - 以便在下一个也是最后一个阶段中加载 - 灯光、FX 与渲染!!
