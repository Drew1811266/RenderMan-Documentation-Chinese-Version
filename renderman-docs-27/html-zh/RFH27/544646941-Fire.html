决定是否渲染体积的火焰密度。如果烟雾和火焰都被禁用，则结果默认使用输入体积。这不会影响由 Scatter 生成的数据。

Diffuse Color（漫反射颜色）





Source Intensity Scale（源强度缩放）
这是 Bindings（绑定）选项卡 Flame（火焰）区域中 Intensity Volume（强度体积）字段的乘数，用于降低或提高生成漫反射颜色矢量所用的值。如果提高 Source Intensity Scale，则需要提高 Diffuse Color Ramp 的 Source Range 最大值，才能让完整范围适配到渐变。最大值为 Intensity Volume 字段 * Source Intensity Scale。

Source Range（源范围）
一种拟合操作，会把这些源值拟合到 0-1，以便使用 ramp（渐变）进行精确重映射。超出此范围的值会被钳制。取值应基于 SOPs 中为 Intensity Volume 字段生成的数据的最小值和最大值。

Color Intensity Scale（颜色强度缩放）
这是由源 Intensity Volume（强度体积）以及 Bindings（绑定）选项卡 Flame（火焰）区域中的 constant（常量）或 color ramp（颜色渐变）生成的颜色矢量乘数，用于降低或提高渲染中的颜色值强度。

Source Range（源范围）
一种拟合操作，会把这些源值拟合到 0-1，以便在 Scatter Color Ramp（散射颜色渐变）上进行精确映射。超出此范围的值会被钳制，并取得 ramp（渐变）任一端的颜色。


Diffuse Masking（漫反射遮罩）





Enable Mask（启用遮罩）
启用用于定义体积哪些区域获得火焰 Diffuse Color（漫反射颜色）的遮罩。遮罩值是漫反射颜色值的乘数。建议使用它为颜色字段添加更多细节和形状特征。该遮罩由 Flame Volume（火焰体积）生成。

Fire Volume Mask Scale（火焰体积遮罩缩放）
这是 Flame Volume（火焰体积）字段的乘数；如果启用了 Fire Volume Mask Intensity Ramp，则作用于其结果，用于提高遮罩值的对比度。

Source Range（源范围）
拟合操作的输入值，该操作会把这些源值拟合到 Destination Range（目标范围）。超出此范围的值会被钳制。

Destination Range（目标范围）
对 Source Range（源范围）值执行拟合操作后得到的输出值，用于 Fire Volume Mask Intensity Ramp。

Fire Volume Mask Intensity Ramp（火焰体积遮罩强度渐变）
用于重映射 Fire Volume Mask（火焰体积遮罩）源值的 Ramp（渐变）。此 Ramp 输出的值会先乘以 Fire Volume Mask Scale，然后再作为遮罩使用。
