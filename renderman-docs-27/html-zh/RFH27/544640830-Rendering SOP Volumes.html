Rendering SOP Volumes

Source: https://rmanwiki-27.pixar.com/space/RFH27/544640830/Rendering+SOP+Volumes
Downloaded: 2026-06-15T03:49:12+00:00

  • Set SOP Path parameter（设置 SOP Path 参数）在 SOP Import LOP 上。

    RenderMan 仅支持导入的 OpenVDB (VDB) volume SOPs（体积 SOP）。必须在渲染前导出 VDB simulation（VDB 模拟）。

  • 在 Material Library LOP 内使用 PxrVolume创建 volume material network（体积材质网络）。下面的网络是使用 density、temperature 和 heat grids 渲染 fireball（火球）的示例网络。

    目前只能渲染 float grids（浮点网格）。Volume motion blur（体积运动模糊）尚不受支持，因为 RenderMan 要求 velocity（速度）是单一 vector grid（向量网格），而不是 3 个 float grids（浮点网格）。

  • 照常使用该 path（路径）进行 Assign Material（分配材质）。
