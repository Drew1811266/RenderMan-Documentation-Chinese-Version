Using Displacement

Source: https://rmanwiki-27.pixar.com/space/RFH27/544644642/Using+Displacement
Downloaded: 2026-06-15T03:49:12+00:00

参见 PxrDisplace 了解如何添加 scalar displacement（标量置换）、vector displacement（矢量置换）以及 displacement layers（置换层）。

  • 将 PxrDisplace VOP 添加到 RIS Network，并连接到 PxrMaterialBuilder 的 output_collect 节点。连接你的 displacement map（置换贴图），例如创建一个 PxrTexture VOP 或 PxrPtexture VOP，分别用于 texture（纹理）或 ptex。也可以连接 shading network（着色网络）。我们推荐使用基于 vector（矢量）的 displacement，尤其是在分层时。

  • 将 PxrDisplace VOP 连接到 output_collect 节点的 shader 输入。

  • Displacement 设置位于被置换 geometry OBJ 上的 RenderMan tab > Primvars > Shading > Displacement。 

 

Trace displacements 默认开启。

设置 displacement bound（置换边界）以渲染置换；设为 0 会关闭对象上的 displacement（置换）。
