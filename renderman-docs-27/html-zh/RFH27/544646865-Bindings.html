Smoke（烟雾）

Smoke Volume（烟雾体积）
用于渲染预期密度的必需字符串字段。应始终指定一个浮点字段。该字段用于 PxrVolume 着色器的 Density Float 参数。

Scatter（散射）

Source Volume（源体积）
用于生成散射数据的字段。

Mask Volume（遮罩体积）
用于生成乘数的字段，以控制由 Scatter Source Volume 生成的散射数据的作用范围。

Scatter Volume（散射体积）
用于写入散射结果的必需字符串字段。

Fire（火焰）
Intensity Volume（强度体积）

用于为 Diffuse Color Volume 和 Emit Color Volume 生成基础颜色与强度值的字段。

Flame Volume（火焰体积）
配合 Fire 选项卡中的 Fire Mask 选项，用于定义来自 Intensity Volume 的颜色和强度数据的应用区域。当 Smoke 被禁用时，也会用于替代密度浮点值。

Color（颜色）

Diffuse Color Volume（漫反射颜色体积）
必需的字符串字段，用于让 pyro 使用用户设计的颜色进行渲染，而不是非默认颜色 (1,1,1)。该字段用于 PxrVolume 着色器的 Diffuse Color 参数。


Emit Color Volume（发射颜色体积）
必需的字符串字段，用于让 pyro 使用用户设计的颜色进行渲染，而不是非默认颜色 (0,0,0)。该字段用于 PxrVolume 着色器的 Emission Color 参数。

Render（渲染）

Velocity Volume（速度体积）
必需的字符串字段，用于使用模拟生成的速度进行运动，而不是默认值 (0,0,0)，或用户在 Render 选项卡中指定的统一方向。空字符串字段会在 Render 选项卡中显示统一方向速度参数。
该字段用于 PxrVolume 着色器的 Velocity 参数。
