请参见 曲线 以了解 RenderMan 曲线及其属性的更多信息；在 RenderMan 中，曲线默认渲染为管状曲线。不支持面向摄影机/公告板曲线，但如果提供了用户法线，则会渲染带状曲线。

要渲染曲线，至少需要添加宽度属性，除非你确实想得到巨大的曲线和毛发。 

可以在选中 Curve SOP 的情况下使用工具架按钮 AddSpareParameters 来创建 RenderMan 选项卡和 Geometry 选项卡，用于设置曲线属性：

  • Curve Basis（曲线基函数）： 选择曲线基函数的类型（用于影响形状和控制方式）

  • Opacity Samples（不透明度采样）： 在材质中使用 Presence（不透明度）时要进行的采样次数。注意，积分器可用于强制执行固定数量的不透明度采样，之后才改用随机采样；随机采样可能产生噪点。

Add Width Attribute（添加宽度属性）

  • 使用 Attribute Create 添加宽度属性。在底部设置宽度值（例如 0.01）并连接节点。系统会自动获取新值。

ROP 节点设置

在 Rendering > Dicing（渲染 > 切分） 下，可以找到全局控制项 Minimum Width（最小宽度），用于控制已渲染曲线的最小像素宽度。较小的值（或 0）可能导致渲染耗时更长，因为小于一个像素宽的曲线可能被漏掉，并需要从渲染质量设置中获取更多最小采样（min samples）。
