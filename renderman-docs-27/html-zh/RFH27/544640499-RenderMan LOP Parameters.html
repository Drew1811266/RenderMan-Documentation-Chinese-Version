RenderMan LOP Parameters

Source: https://rmanwiki-27.pixar.com/space/RFH27/544640499/RenderMan+LOP+Parameters
Downloaded: 2026-06-15T03:49:12+00:00

Solaris 提供多种 LOP 节点，用于创建和编辑 USD 图元。

在以下 LOP 节点子集上，除通用 USD 参数外，还会看到 “RenderMan” 选项卡，可用于指定 RenderMan 专用属性：

Camera、Light、Render Var、Render Product、Render Settings、Render Geometry Settings。

从 Houdini 20.5.486 开始，如果这些 “RenderMan” 选项卡缺失或已过期，SideFx 提供了一种重建它们的方法。

可以选择单个或多个节点，然后点击

Initialize Parameters -> Rebuild Renderer-Specific Parameters

也可以从顶部菜单点击 Render 选项来更新整个场景

Render -> Rebuild All Renderer-Specific LOP parameters
