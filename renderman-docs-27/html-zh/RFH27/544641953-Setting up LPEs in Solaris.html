Setting up LPEs in Solaris

Source: https://rmanwiki-27.pixar.com/space/RFH27/544641953/Setting+up+LPEs+in+Solaris
Downloaded: 2026-06-15T03:49:12+00:00

在制作中，经常需要为镜头创建非物理效果和艺术指导调整。Light Path Expressions（LPE，光路表达式）是一种强大的方式，可收集并输出特定光路，以便之后修改。

LPE 的能力包括：

  • 支持加法合成，保留图像的物理属性，并避免在后期处理中通过乘法或除法造成伪影。

  • 输出特定光线反弹（第 2、第 3 次等），甚至一段反弹范围（第 4 到第 8 次光线反弹）。你可以用它来隔离效果，甚至隔离带噪声的反弹以便做减法处理。

  • 输出特定对象到 LPE。虽然可以使用 Cryptomatte 通过 ID 隔离对象，但也可以分别输出对象或对象集合及其贡献。

  • 隔离某条带噪声的特定光路，用于在合成中做减法；或者输出一个不包含这些光路的 LPE 用于合成。

  • 创建非物理效果，例如让焦散按来源呈现不同颜色，改变对象的偏色，或缩放照明。

  • 输出逐灯光 AOV，以便后续调整。

在此示例中，我们会在摄影棚场景里，从单个灯光创建一个 LPE。

首先，选择要创建 Light Group 的灯光，然后在 RenderMan 选项卡 → Light Group 下为灯光组命名。注意，可以把多个灯光添加到一个灯光组中，以创建不同的隔离照明设置。


接下来，需要创建实际的 LPE 表达式，以便在 AOV 通道中隔离此灯光。  


使用 Render Var LOP，你可以在这里输入所需信息来创建 LPE。如图所示，该 LPE 将命名为 SideRoom_LPE，source name（源名称）定义 LPE 的结构；表达式中还包含 SideRoomLight，它指向我们之前创建的 Light Group。

有关 LPE 表达式的更多信息可见 此处。
