AOV Light

PxrAOVLight（AOV 灯），是一种方便的 utility light（实用灯光），允许 lighting artist（灯光艺术家）输出 shot-specific AOV mask（镜头专用 AOV 遮罩），无需向 shading artists（着色艺术家）请求创建，也无需生成 Light Path Expression（光路表达式）（LPE）。

使用 PxrAovLight 进行 Light Linking（灯光链接）

使用 Light Linking（灯光链接） 在 AOV 中包含/排除部分 geometry（几何体）。在此示例中，PxrAovLight 在全局范围为 "off"，在场景中的树上为 "on"。这个 linking setup（链接设置）会创建树的 mask（遮罩）。

Light Filters（灯光滤镜）与 PxrAovLight

由于 PxrAovLight 是一种 "light"（灯光），我们可以添加 light filters（灯光滤镜）来修改输出 AOV。在此示例中， PxrIntMultLightFilter（强度倍增灯光滤镜） 连接到 PxrAovLight。Light Filter（灯光滤镜）只链接到树干，因此树干 mask（遮罩）会得到一个灰度值。
