设置 Display Filters（显示过滤器）和 Sample Filters（采样过滤器）

在 RIS Network（RIS 网络）中，选择一个 Pxr Display 或 Sample Filter VOP。请注意，你可能会看到同一个 filter（过滤器）有 Display（显示）或 Sample（采样）两种形式。Display Filters 作用于 pixel level（像素级，即最终结果），而 Sample filters 在 samples（采样）被过滤为 pixels（像素）之前作用于它们。

将 VOP path（VOP 路径）拖到 render ROP（渲染 ROP）的 Filters 下的 Display Filter 或 Sample Filter parameter（参数）中。

向 multiparam（多参数）添加多个 display 或 sample filters（显示或采样过滤器），以组合 filters（过滤器）。
