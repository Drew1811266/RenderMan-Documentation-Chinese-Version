你可以添加 RenderMan IPR ROP（RenderMan 交互式渲染 ROP），可从 RenderMan Shelf（RenderMan 工具架） 或 /out pane（/out 面板）添加 RenderMan IPR ROP。

使用 RenderMan Shelf（RenderMan 工具架）

我们可以使用 ROP tool（ROP 工具），它位于 RenderMan Shelf（RenderMan 工具架） 中，用它创建一个 ROP，该 ROP 会自动向你的 RIS Network（RIS 网络）添加 integrator VOP（积分器 VOP）。

使用 out pane（手动配置）

添加 RenderMAN IPR ROP（交互式渲染 ROP）

 

添加 Integrator（积分器）

在你的 RIS Network（RIS 网络）中添加一个 integrator VOP（积分器 VOP），例如 PxrPathTracer

设置 Integrator（积分器）

只需将你的 integrator VOP（积分器 VOP）路径拖到 Integrator（积分器） 中，该参数位于 RenderMan RIS ROP 内。如果跳过此步骤，默认会使用 PxrPathTracer integrator（PxrPathTracer 积分器）及其默认参数。
