RenderMan for Houdini (RfH) 是 Houdini 内部面向 RenderMan 的集成接口。 

安装

RenderMan for Houdini 需要安装 RenderMan Pro Server 以及 RfH 插件才能正常工作。RenderMan 安装程序会自动完成此操作。


要完成设置，应遵循以下说明： IRenderMan for Houdini 安装。
