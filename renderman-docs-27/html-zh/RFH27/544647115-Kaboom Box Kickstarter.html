Kaboom Box HDA 是 RenderMan 版的标准 Houdini pyro lookdev node（火焰/烟雾外观开发节点）Pyro Bake Volume，它让 RenderMan artists（RenderMan 艺术家）能够快速且真实地渲染 pyro 和 volume sims（体积模拟），无论是大型爆炸、小型蜡烛火焰，还是介于两者之间的各种效果。


安装后，KaBoom Box 位于 OBJ context（OBJ 上下文）中，可以很容易地放入你的 node network（节点网络），位置在 RenderMan 选项卡下。

该 HDA 需要一个输入。此输入可以是 cached VDB file sequence（缓存的 VDB 文件序列），也可以位于你的网络中 pyrosolver 节点之后。 




在这个简单示例中，我们将使用 Houdini 中默认的 fireball preset（火球预设），并使用 Kaboom Box HDA 对火球进行 lookdev（外观开发）和渲染。继续之前，请确保你已经正确安装了 HDA。

首先，需要创建一个 geometry node（几何节点）；创建后进入其内部，然后放置一个 Pyro Configure Fireball 节点。






网络创建完成后，需要放置一个 RenderMan Kaboom Box 节点，并将它接入 pyrosolver 节点之后的网络。






最后，在 Kaboom Box properties panel（属性面板）的 presets（预设）中选择 Fireball，就完成设置了。




记得在场景中加入一些 lights（灯光）、camera（摄像机）和 RenderMan ROP；全部完成后，你就可以使用 Kaboom Box 渲染第一个 Fireball explosion（火球爆炸）了！
