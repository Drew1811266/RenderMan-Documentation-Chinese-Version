Preset Browser in Solaris

Source: https://rmanwiki-27.pixar.com/space/RFH27/544640927/Preset+Browser+in+Solaris
Downloaded: 2026-06-15T03:49:12+00:00

艺术家可以使用 RenderMan 的 Preset Browser，在不同 DCC 和部门之间保存并共享材质、穹顶灯和灯光装置。

Preset Browser 可在 Solaris 上下文中使用，并可在 RenderMan -> Preset Browser 下找到。


加载 HDRI

环境贴图可以很容易加载到现有 domelight 节点中：先选择要附加贴图的 domelight 节点，然后选择 Import and Assign to Selected（导入并分配给选中项）

如果没有选择 domelight 节点时选择 Import，会自动创建一个 domelight 节点，并分配选中的环境贴图。



加载材质

如果已有一个 Solaris Material Library 节点，并希望把预设材质加载进去，只需选择 materiallibrary 节点，然后选择 Import into Selected Material Library。 
这会将选中的材质预设导入到选中的 Material Library 中。

如果 materiallibrary 节点未被选择或不存在，可以选择 Import 或 Import and Assign to Selected，该预设会被加载到新创建的 Material Library 中。



保存材质 

目前 Solaris 中有一个限制：从选中的 materiallibrary中只会导出一个材质。 

只需选择 materiallibrary 节点，然后点击 Preset Browser 中的新建材质图标。
