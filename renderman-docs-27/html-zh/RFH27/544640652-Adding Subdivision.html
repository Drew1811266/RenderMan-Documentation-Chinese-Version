Adding Subdivision

Source: https://rmanwiki-27.pixar.com/space/RFH27/544640652/Adding+Subdivision
Downloaded: 2026-06-15T03:49:12+00:00

在 Solaris 中为几何添加渲染时细分，可通过 MeshEdit 节点完成。

创建一个 MeshEdit 节点，并将其连接到 Solaris 图中。

选择要细分的图元，然后使用 Set / Create（设置/创建） 设置 Subdivision Scheme（细分方案），并选择 Catmull Clark
