Grouping Membership

Source: https://rmanwiki-27.pixar.com/space/RFH27/544643653/Grouping+Membership
Downloaded: 2026-06-15T03:49:12+00:00

通过几何体上的 categories（类别） 参数为几何体设置 grouping membership（分组成员关系），以便在 trace set（追踪集合） 或 shadow subset（阴影子集）中使用这些分组。 

Simple Tokens（简单标记）：

  • Asterisk *（星号 *）- Everything（全部对象），默认值

  • 仅对象名称/路径 - 包含后面的对象，例如 sphere_object

  • Caret ^（插入符 ^）- 排除后面的对象，例如 ^sphere_object

  • The @ symbol（@ 符号）- 包含这组对象 bundle/grouping（捆绑/分组），例如 @carEngine

以下面的 PxrRectLight 为例，使用时只需在 Light Mask 中指定该 grouping membership（分组成员关系）名称，并添加 caret ^（插入符 ^）来排除指定的灯光路径。在下例中，网格对象不会被 PxrRectLight 照亮。注意，这可以通过 Light Bank 和 light Linking（灯光链接）的 UI 完成，也同样适用于 trace sets（追踪集合）。也可以只写 ^pxrrectlight，省略 /obj/ 路径。

另一个常见需求是忽略特定对象投射的 shadows（阴影）。为此，请使用 Edit the Parameter Interface（编辑参数界面）（Object Name 字段旁的齿轮图标）。

可以按下图所示复制 Reflection Mask 并重命名来添加 Shadow Mask。随后可用该字段选择哪些对象允许为该对象投射阴影，或哪些对象应被排除。
