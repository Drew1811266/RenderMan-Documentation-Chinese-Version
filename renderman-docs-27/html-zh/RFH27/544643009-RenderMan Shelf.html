RenderMan Shelf（RenderMan 工具架）

RenderMan for Houdini 提供了自己的工具架，其中包含许多最基本且实用功能的快捷方式。

Rendering / Setup（渲染/设置）

按钮

说明

启动 IPR 渲染会话

停止 IPR 渲染会话

打开 Image Tool，即 'It'

创建 RenderMan ROP，同时创建 PxrPathTracer Integrator，并设置到 /cam1 的路径（不会自动创建 camera）。

创建 RenderMan Stylized Looks（风格化外观）ROP，并添加用于风格化外观输出的 AOVs 和用于最终显示处理的 Display Filters；如果当前选中的 OBJ 带有 PxrSurface，也会自动为该对象分配相应的 PxrStylizedControl 控制节点。

为选中的 OBJ 添加参数，便于快速添加有效的 RenderMan 参数。

Shading（着色）

按钮

说明

将选中的对象设为 遮挡 对象

创建一个 PxrSurface VOP 并将其分配给 OBJ

创建一个 LamaSurface 并将其分配给 OBJ

Lighting（照明）

按钮

说明

创建一个 PxrRectLight OBJ

创建一个 PxrDiskLight OBJ

创建一个  PxrSphereLight OBJ

创建一个 PxrCylinderLight OBJ

创建一个 PxrMeshLight OBJ，用于所选 OBJ

为选中的 Volume OBJ 创建体积灯光设置

创建一个 PxrDomeLight OBJ

创建一个 PxrPortalLight OBJ

创建一个 PxrEnvDayLight OBJ

创建一个  PxrDistantLight OBJ

创建一个 PxrAOVLight OBJ
