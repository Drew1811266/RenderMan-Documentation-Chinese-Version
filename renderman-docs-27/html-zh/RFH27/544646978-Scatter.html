Scatter（散射）

Scatter Enable（启用散射）
启用此切换项只会计算数据。若要可视化并渲染这些数据，必须启用 Scatter to Diffuse 或 Scatter to Emission。如果希望让伪颜色和光照在模拟的烟雾中散射，建议使用此功能。





Intensity Scale（强度缩放）
应用在生成的散射结果上的整体乘数。


Scatter to Diffuse（散射写入漫反射）
将散射结果写入 Diffuse Color 体积。这可用于把来自散射源体积字段的伪颜色和光照贡献烘焙到密度体积上。对于带烟雾的火焰和爆炸，此选项可能很有用。

Contribution Scale（贡献缩放）
乘以漫反射颜色字段中的散射贡献。

Scatter to Emission（散射写入发射）
将散射结果写入发射颜色字段。这可用于把来自散射源体积字段的伪颜色和光照贡献烘焙到火焰体积上。对于爆炸和火球，此选项可能很有用。
注意：非灰度的散射颜色值可能在渲染中产生意外颜色。

Contribution Scale（贡献缩放）
乘以发射颜色字段中的散射贡献。

Color Mode（颜色模式） 
使用渐变或黑体颜色向量为生成的散射数据着色；这些数据可应用到漫反射颜色字段和/或发射颜色字段。


Blur（模糊）




Blur Scatter（模糊散射）
开启此选项后，会对计算得到的散射体积应用模糊。模糊会扩散发射，并模拟发射光的内部散射。建议启用模糊，以获得更自然的结果。

Use Blur Steps（使用模糊步数）
开启后，会以多个不同尺度对中间散射体积进行模糊，并组合这些结果得到最终散射场。使用 2 或 3 可进一步平滑散射区域。

Number of Blur Steps（模糊步数）
设置要组合在一起的单独模糊次数。第一步模糊会使用指定的 Filter Radius；之后每一步都会将前一步的过滤尺寸加倍。步数越多，散射体积会越柔和。

Blur Step Falloff（模糊步衰减）
控制每个模糊步对最终结果的贡献。该值等于后一步与前一步之间的权重比。值越高，使用更大 Filter Radius（过滤半径）模糊的步骤贡献越大。

Use World Space Radius Units（使用世界空间半径单位）
关闭时，Filter Voxel Radius 控制模糊半径。开启此选项可用世界单位指定半径。这意味着模糊结果对散射体积分辨率的依赖要小得多；该分辨率也会受 Downsample 参数影响。

Filter Voxel Radius（过滤体素半径）
控制执行平滑操作的体素数量。值越高，散射场会越模糊、越柔和。该值以世界单位计量。

Blur Iterations（模糊迭代）
重复平滑操作的次数。值越高，散射场会越模糊、越柔和。

Downsample（降采样）
在将 Source Volume 转换为 Scatter Field 之前对其进行降采样。降采样可以加快 cook 时间，代价是会丢失一些较锐利的细节。不过，此缺点通常并不明显，因为模糊也会削弱这些较锐利的特征。


Masking（遮罩）



Enable Mask（启用遮罩）
通过将 Source Volume 与 Bindings 选项卡 Scatter 区域中的 Mask Volume 相乘来对 Source Volume 进行遮罩。遮罩对带有较冷外壳的爆炸很有用。

Scatter Mask Scale（散射遮罩缩放）
这是作用于散射结果值的另一个乘数。更高强度会在渲染中产生更明亮、更炽热的颜色。建议将 Mode 设置为 "Use Ramp"，以获得更符合预期的结果，尤其是在为火球和爆炸生成核心时。
