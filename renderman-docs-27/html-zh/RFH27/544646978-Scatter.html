Scatter（散射）

Scatter Enable（启用散射）
启用此 toggle（切换项）只会计算数据。若要可视化并渲染这些数据，必须启用 Scatter to Diffuse 或 Scatter to Emission。如果希望让 faux color（伪颜色）和光照在模拟的烟雾中散射，建议使用此功能。





Intensity Scale（强度缩放）
应用在生成的 scatter 结果上的整体 multiplier（乘数）。


Scatter to Diffuse（散射写入漫反射）
Write（写入）scatter 结果到 Diffuse Color volume（漫反射颜色体积）。这可用于把来自 scatter source volume field（散射源体积字段）的 faux color（伪颜色）和光照贡献烘焙到 density volume（密度体积）上。对于带烟雾的火焰和爆炸，此选项可能很有用。

Contribution Scale（贡献缩放）
乘以 diffuse color field（漫反射颜色字段）中的 scatter 贡献。

Scatter to Emission（散射写入发射）
Write（写入）scatter 结果到 emitted color field（发射颜色字段）。这可用于把来自 scatter source volume field（散射源体积字段）的 faux color（伪颜色）和光照贡献烘焙到 fire volume（火焰体积）上。对于爆炸和火球，此选项可能很有用。
注意：非 grayscale（灰度）的 scatter color values（散射颜色值）可能在渲染中产生意外颜色。

Contribution Scale（贡献缩放）
乘以 emitted color field（发射颜色字段）中的 scatter 贡献。

Color Mode（颜色模式） 
使用 ramp（渐变）或 blackbody color vector（黑体颜色矢量）为生成的 scatter data（散射数据）着色；这些数据可应用到漫反射颜色字段和/或发射颜色字段。


Blur（模糊）




Blur Scatter（模糊散射）
开启此选项后，会对计算得到的 scatter volume（散射体积）应用 blurring（模糊）。模糊会扩散 emission（发射），并模拟发射光的内部散射。建议启用 blur，以获得更自然的结果。

Use Blur Steps（使用模糊步数）
开启后，会以多个不同尺度对中间 scatter volume（散射体积）进行模糊，并组合这些结果得到最终 scatter field（散射场）。使用 2 或 3 可进一步平滑散射区域。

Number of Blur Steps（模糊步数）
设置要组合在一起的单独 blur（模糊）次数。第一步 blur 会使用指定的 Filter Radius；之后每一步都会将前一步的过滤尺寸加倍。步数越多，scatter volume 会越柔和。

Blur Step Falloff（模糊步衰减）
控制每个 blur step（模糊步）对最终结果的贡献。该值等于后一步与前一步之间的权重比。值越高，使用更大 filter radius（过滤半径）模糊的步骤贡献越大。

Use World Space Radius Units（使用世界空间半径单位）
关闭时，Filter Voxel Radius 控制 blur radius（模糊半径）。开启此选项可用 world units（世界单位）指定半径。这意味着模糊结果对 scatter volume 分辨率的依赖要小得多；该分辨率也会受 Downsample 参数影响。

Filter Voxel Radius（过滤体素半径）
控制执行 smooth operation（平滑操作）的 voxel（体素）数量。值越高，scatter field 会越模糊、越柔和。该值以 world units（世界单位）计量。

Blur Iterations（模糊迭代）
重复平滑操作的次数。值越高，散射场会越模糊、越柔和。

Downsample（降采样）
在将 Source Volume 转换为 Scatter Field 之前对其进行 downsample（降采样）。降采样可以加快 cook times（cook 时间），代价是会丢失一些较锐利的细节。不过，此缺点通常并不明显，因为 blurring（模糊）也会削弱这些较锐利的特征。


Masking（遮罩）



Enable Mask（启用遮罩）
通过将 Source Volume 与 Bindings 选项卡 Scatter 区域中的 Mask Volume 相乘来对 Source Volume 进行遮罩。Masking 对带有较冷外壳的爆炸很有用。

Scatter Mask Scale（散射遮罩缩放）
这是作用于 scatter 结果值的另一个 multiplier（乘数）。更高强度会在渲染中产生更明亮、更炽热的颜色。建议将 Mode 设置为 "Use Ramp"，以获得更符合预期的结果，尤其是在为火球和爆炸生成 core（核心）时。
