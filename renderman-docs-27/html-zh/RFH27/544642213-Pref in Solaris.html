渲染时，通常需要为对象设置一个默认姿态或位置，作为后续渲染的参考，即使对象之后会变形或四处移动也一样。这样可以避免在动画过程中出现“swimming”（纹理游移），也就是纹理效果看起来在对象表面滑动。它通常以技术名称 Pref 来称呼。


在 Houdini / Solaris 中，先添加一个 Rest 节点来设置默认姿态

在你的 Solaris 网络中，添加一个 copyproperty 节点，并按下图所示进行设置。





添加 __Pref AOV 的位置是 rendermanrendersettings 节点下的 Render Vars (AOVs) → Integrator




启动一次 viewport（视口）或 disk render（磁盘渲染），然后在 AOV display（AOV 显示）中就能看到 _Pref AOV


从此之后，渲染应会使用 Rest 节点设置的默认位置和形状。
