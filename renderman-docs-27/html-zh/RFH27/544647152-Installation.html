Kaboom Box 以 Houdini HDA（Houdini Digital Asset，Houdini 数字资产）的形式提供，安装非常简单快捷。

下载 Kaboom Box HDA。下载完成后，打开 Houdini，并从 Asset 菜单中选择 Install Asset Library。 

选择已下载的 `rman_kaboom_box.hda`，然后按 Install。

现在，Kaboom Box HDA 已安装完成，可以使用了。
