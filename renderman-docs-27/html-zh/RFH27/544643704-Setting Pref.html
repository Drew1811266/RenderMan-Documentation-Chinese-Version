Setting Pref

Source: https://rmanwiki-27.pixar.com/space/RFH27/544643704/Setting+Pref
Downloaded: 2026-06-15T03:49:12+00:00

渲染时，通常需要为对象设置一个默认姿态或位置，使其即使发生变形或移动，也能作为后续渲染的参考。这可以防止 "swimming"（纹理游移），也就是纹理效果在对象动画过程中看起来沿对象表面滑动。它通常以技术名称 Pref 来指代。

在 Houdini 中，可以使用 rest 节点轻松设置这个参数。该节点会创建 rest 属性（以及可选的 rnml）。对于大多数 Houdini 几何体，RenderMan for Houdini 会自动将其重命名为 Pref（以及 Nref）。对于程序图元，不会发生这种重命名，因此需要自行重命名 rest 属性。

在 patterns（图案）中，应使用 Pref（以及 Nref），这就是 RenderMan 能看到的属性名称。 

此后，渲染应使用 Rest node 设置的默认位置和形状。
