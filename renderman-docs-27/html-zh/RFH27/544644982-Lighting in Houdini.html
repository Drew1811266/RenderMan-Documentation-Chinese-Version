Lighting in Houdini

Downloaded: 2026-06-15T03:49:12+00:00


Pixar Animation Studios 强大的灯光已经进入 Houdini；若要查看具体参数信息，请参阅 RenderMan 文档。


为场景布光主要有三种方式：

  • Analytic Lights（解析灯光） - 这是首选方式。它们在内存使用、灵活性和速度方面表现更好。它们的可见性只是针对摄影机可见性的简单开关。

  • Mesh Lights（网格灯） - 它们很适合使用任意形状进行照明。由于它们是几何体，会占用更多内存，复杂形状也可能增加噪声。它们可以像其他对象一样使用可见性：camera（摄影机）、shadow/transmission（阴影/透射）以及 indirect visibility（间接可见性）。

  • Emissive Surfaces（自发光表面） - 这类方式通常使用 constant Bxdf 或 “glow” 参数来间接照亮场景。它效率较低，通常应避免使用，但对于把纹理贴图形式的“灯”映射到未来风格对象和面板上，可能会有用！
