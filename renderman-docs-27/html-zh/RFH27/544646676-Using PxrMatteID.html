Using PxrMatteID

Downloaded: 2026-06-15T03:49:12+00:00

PxrMatteID 输出用于合成的遮罩 AOV。这需要添加一个用户属性（User Attribute），更多细节见 此处。

此后，PxrCryptomatte 已取代该工作流，成为一种有用且几乎自动化的替代方案。

为 MatteID0 添加一个用户属性。

复制以下字段：

  • Name

  • Label（这是 UI 名称）

  • Tags（交互式渲染期间查看和编辑 RenderMan 属性所必需）
    script_ritype color
    renderman attr

设置其类型和颜色值。

添加 PxrMatteID

创建一个 Houdini buildarray 节点，用于连接到 PxrSurface 材质上的 Utility Pattern。然后按下图所示，将 PxrMatteID Pattern 节点连接到 buildarray。

在 PxrMaterialBuilder Network 中添加 PxrMatteID VOP。将颜色设置为上面的用户属性。

将 PxrMatteID 的 resultAOV 连接到 PxrSurface 的 Utility Pattern。

为 Output 设置 AOV

从 RenderMan ROP 节点的 Displays 选项卡中选择正确/对应的 MatteID AOV 输出

你会注意到，我们在 OBJ 上通过属性设置了一种颜色，同时 MatteID Pattern 本身也有一种颜色。这样做是为了将属性的颜色与 Pattern 节点相乘。例如，你可以向 MatteID Pattern 节点提供一个纹理遮罩，并让它与 MatteID Attribute 中选择的颜色相乘。
