欢迎使用 RenderMan Kaboom Box HDA 

此 HDA 会大幅加速你的 pyro（火焰/爆炸模拟）和体积工作流，让艺术家快速轻松地对爆炸、火焰、蜡烛以及介于其间的各种效果进行外观开发和渲染。该 HDA 同时提供基础和高级 pyro 外观开发控制，以及一些预设，帮助你快速上手。  


下载



HDA 的核心是一个复杂且稳健的 PxrVolume 着色网络，它专为 RenderMan 开发，开发者为 Hope Schroers；她是 Pixar Animation Studios 众多才华出众的 VFX TD 之一。
