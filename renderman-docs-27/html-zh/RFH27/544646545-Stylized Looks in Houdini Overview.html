设置风格化渲染需要 3 个组件：

1. 添加风格化显示滤镜

2. 启用风格化 AOV

启用风格化渲染所需 AOV 的最简单方式，是从 RenderMan 菜单创建一个 R26 Stylized ROP。

这会自动创建一个 RenderMan ROP，并激活在风格化外观中渲染所需的所有 AOV。

如果已经有 RenderMan ROP 节点，也可以在 Images → Stylized 下手动启用风格化 AOV。

3. 将 PxrStylizedControl 连接到材质

场景中的所有对象并不都需要这个 Pattern 节点。需要使用动态数组将 PxrStylizedControl 的整数输出连接到 PxrSurface 的 utilityPattern 输入。若要把 PxrStylizedControl 添加到多个材质，请使用 RenderMan 菜单中的 Stylize 工具。

“R24 Stylized” 工具架按钮会设置上文所述的全部 3 个必要组件

  • 创建一个启用风格化外观的 RIS ROP。

  • 添加 Hatching 显示滤镜。

  • 将 PxrStylizedControl 添加到所有选定对象的材质。
