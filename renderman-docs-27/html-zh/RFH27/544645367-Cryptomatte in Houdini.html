Houdini 中的 Cryptomatte

Source: https://rmanwiki-27.pixar.com/space/RFH27/544645367/Cryptomatte+in+Houdini
Downloaded: 2026-06-15T03:49:12+00:00

PxrCryptomatte 作为  Sample Filter（采样过滤器）  插件提供。因此，Cryptomatte 的输出会作为独立的 EXR 文件存储，与 beauty（美图）或降噪输出分开。请注意，需要 OpenEXR 才能正确存储这些值。 请参阅 此处 了解如何附加采样过滤器以及如何组合采样过滤器。 

Cryptomatte 用于生成自动 ID 或用户提供的 ID，以便在合成中使用。通常，此通道用于在后期操作中识别并修改场景元素。它还省去了为场景实体手动分配有用 ID 的繁琐工作，尤其是在场景很复杂时。

此示例场景使用 Houdini 18.0 制作，可在页面底部获取。

内置选项

Layers（层）

  • Name（名称） 此项使用对象名称来指定 ID。更改对象名称会改变结果。

  • Path（路径） 类似于 Name（名称），但现在也会识别实例。

  • Material（材质） 此项基于分配的材质创建 ID。共享材质会具有相同 ID。


Attributes（属性）

  • user:__objectid（用户对象 ID） 基于 OBJ 节点分组。

  • user:cryptomatte（Cryptomatte 用户属性） 此项使用 RenderMan 备用参数中的参数。

  • user:myatttr（示例用户属性） 仅适用于打包几何体上的图元属性。请注意，在未打包几何体上，图元属性是 RenderMan Primvars（内建于几何体中的图元变量），而不是属性，因此 PxrCryptomatte 无法读取。

  • 可以指定任何属性，例如 visibility:camera、Ri:Matte 等，让 Cryptomatte 生成 ID。请参阅 PxrCryptomatte 文档了解更多详细信息。
