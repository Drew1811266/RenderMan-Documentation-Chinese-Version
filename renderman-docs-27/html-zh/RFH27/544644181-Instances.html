实例适合通过引用一份原始副本来复制几何体；在处理和渲染大量对象时，这可以节省内存和时间。随后可以为这些实例赋予不同着色，生成有趣且真实的变化，而不必让艺术家或计算机承担大量唯一副本的负担。

参见 实例化 了解有关实例化的更多信息。

Copy to Points SOP（复制到点 SOP）

创建一个 Copy to Points SOP，将要实例化的几何体连接到第一个输入，并将用于放置实例的位置连接到第二个输入。 

Instance OBJ（实例 OBJ）

创建一个 Instance OBJ，方式与平常创建 Houdini 实例。 

在 Instance 节点中，设置 Instance object（实例对象） 并选择 Fast point instancing（快速点实例化） 或 Full point instancing（完整点实例化）。目前这两个选项在渲染器中会产生相同的行为。

若要覆盖实例化对象的材质，可以在 Instance 内部使用 Material SOP。
