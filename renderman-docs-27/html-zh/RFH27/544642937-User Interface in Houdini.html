User Interface in Houdini

Downloaded: 2026-06-15T03:49:12+00:00



RenderMan 直接集成到 Houdini 熟悉的 options（选项）和 menus（菜单）系统中。

你可以在文档所述的 menu systems（菜单系统）中找到这些 nodes（节点），也可以通过 RenderMan Shelf 和 RenderMan Menu 中的实用 shortcuts（快捷方式）找到它们。


  • The RenderMan Menu（RenderMan 菜单）
  • The RenderMan Shelf（RenderMan 工具架）
  • Preset Browser（预设浏览器）
