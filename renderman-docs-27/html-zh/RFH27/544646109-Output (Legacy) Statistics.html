Output (Legacy) Statistics

Downloaded: 2026-06-15T03:49:12+00:00

从版本 24 开始，RenderMan 包含一个全新的 diagnostics sub-system（诊断子系统），该系统从头重写，用于纳入 live（实时）、configurable statistics（可配置统计信息）。请参见 RenderMan 页面 此处。

要继续使用 legacy statistics（旧版统计信息），请勾选 Render Statistics（渲染统计） 选项，位置在你的 RenderMan RIS ROP 的 Options | Statistics（选项 | 统计） Tab（选项卡）：

默认情况下，它使用 Disk File（磁盘文件）的路径和 base name（基本名称）。 

你可以用任何 Web browser（网页浏览器）查看输出的 XML file（XML 文件）。请参见 Legacy Diagnostics（旧版诊断） 了解更多信息。

  • File（文件）：生成 .stats 文件 

  • XML File（XML 文件）：生成可在 browser（浏览器）中查看的 XML 文件

  • IO Trace Filename（IO 跟踪文件名）：保存为 XML 的详细 input/output（输入/输出）报告

  • Stylesheet（样式表）：用于格式化 XML 文件的 stylesheet（样式表）

  • Shader Profile（着色器性能剖析）：用于 custom shader output（自定义着色器输出）的 profile（性能剖析）

  • Displace Ratios（位移比率）：当 displacement values（位移值）除以 max displacement（最大位移）后的结果落在此范围内时，抑制该报告

  • Maximum Displacement（最大位移）：停止前要打印的 displacement values（位移值）警告数量；0 表示报告所有值

  • Enable Texture Statistics（启用纹理统计）：允许对 texture usage（纹理使用情况）进行更细粒度的报告

  • Track Asset Objects（跟踪资产对象）：跟踪 scene entities（场景实体）
