Getting Started in Houdini

Downloaded: 2026-06-15T03:49:12+00:00


请注意，不同版本的 Houdini 有不同的工作流，请务必遵循与你所用版本对应的说明。


Adding a Light（添加灯光）

要添加灯光，请从 RenderMan Shelf 或 RenderMan Menu 中选择一种灯光工具，它会自动创建一个 Pxr 灯光 OBJ。 

参见 Lighting in Houdini（Houdini 中的照明） 了解更多信息。


Adding a Material（添加材质）

要添加材质，我们需要先创建着色网络。请注意，RIS 材质 not a SHOP（不是 SHOP） 而是 VOP ，位于 RIS Network中。

PxrMaterialBuilder 只是 RIS VOP 的容器。Houdini 要使用 RIS 材质，必须通过它。 

 Important（重要）：材质由带有 BxDF output_collect VOP 输入的 PxrMaterialBuilder VOP 表示。

要添加材质，可以从下面添加 PxrSurface： RenderMan Shelf这会自动创建相关节点；或者从下面选择另一个 BxDF： RenderMan Menu 并将其连接到 PxrMaterialBuilder 内部的 output_collect VOP。

如果 RIS Network 尚不存在，这些工具还会自动创建 RIS Network，并创建所选 VOP。


Assigning a Material（分配材质）

要将材质分配给对象，只需把刚创建的 PxrMaterialBuilder VOP 拖到对象的 Material 参数上。


Render（渲染）

要渲染，请从下面添加 RenderMan ROP： RenderMan Shelf。 

如果想使用 Stylized Looks 进行渲染，则需要从 RenderMan Shelf 添加一个 Stylized Looks ROP。

该工具还会创建 RIS Network，并将积分器设置为 PxrPathTracer。

确保 Camera 参数设置为渲染摄影机（必须在 Houdini 中创建一个摄影机，因为默认摄影机不能使用），然后按下 Render 按钮，即可在 Image Tool 中交互式查看渲染结果。

参见 Rendering in Houdini（Houdini 中的渲染） 了解更多信息。
