Frequently Asked Questions

  • RfH 由谁开发？ 
    这是由 SideFx 与 Pixar 的 RenderMan group（RenderMan 团队）共同开发的。

  • RfH 与 RenderMan release（RenderMan 版本发布）同步的频率如何？
    我们会尽量在 RenderMan release（RenderMan 版本）可用后尽快同步。对于大型版本，例如 major version change（主版本变更，而不是 point release 小版本发布），这需要时间，尤其是在 Houdini 尚未支持新功能 UI 时。

  • 如何报告 bug 和建议？ 
    请发布到 RenderMan for Houdini 论坛。

  • 当我不确定遇到的是 Houdini bug 还是 RenderMan bug 时，应该在哪里报告？ 
    请先查看 这里；然后，如果问题与 RenderMan shaders（RenderMan 着色器）和 output（输出）相关，请在 RenderMan for Houdini 论坛提交 bug。
