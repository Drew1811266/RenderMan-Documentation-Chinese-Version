Downloaded: 2026-06-15T03:49:12+00:00

若要让 RenderMan 控制几何设置，可以在 RenderGeometrySettings LOP 的 RenderMan 选项卡中找到这些设置

（需要安装 RfH）

选择 Set or Create（设置或创建） 或 Set if Exists（若存在则设置） 来创建或更改 geometry attribute（几何属性）上的 opinion（意见），或选择 Block（阻断） 来禁用该 attribute 上已有的 opinions（意见）。所有可用参数都会列出，但默认均未设置。

这些设置与 object nodes（对象节点）上的 RenderMan spare parameter 选项卡中可以找到的设置相同。
