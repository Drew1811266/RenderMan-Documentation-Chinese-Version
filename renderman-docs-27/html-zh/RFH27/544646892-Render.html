Render（渲染）


KaBoom Box HDA 此选项卡中的参数控制位于 HDA 内部的 PxrVolume 着色器的各项功能。
参见 Volume Material（体积材质） 了解此着色器节点的专门文档。





Light Source（光源）
开启此开关会向渲染器表明，该体积应被视为会参与下一事件估计的发光光源。只有当 emitColor 参数不是黑色时，这才有用。大多数情况下，对于被该体积照亮的场景部分（可能包括体积自身），这会降低噪声并加快收敛。

请注意，如果只是开启此开关，而不改变场景中的其他设置，通常会得到更亮的结果，因为它本质上增加了一次额外的光反弹：对于自发光体积，体积中的点会被同一体积中的其他点自照明。因此，在最大路径长度为 1 时渲染一个开启 __islight 的单次散射体积，实际上等价于在最大路径长度为 2 时渲染一个关闭 __islight 的多重散射体积。

Multiple Scattering（多重散射）
如果设置为 0，并且积分器遵循此提示，PxrVolume 将只执行单次散射：体积内部的点只会被光源直接照亮。如果设置为 1，体积内部的点也会被间接照明照亮（光会在体积内部散射多次）。此效果可能成本较高，因此默认值为 0（关闭）。请注意，某些积分器（例如 PxrVCM）会忽略此提示，并始终执行多重散射。此控制对聚合体积没有影响。


Velocity Multiplier（速度倍率）
应用在体积速度上的倍率。这可用于在按帧定义的速度和按秒定义的速度之间转换。

Anisotropy（各向异性）





Primary Anisotropy（主各向异性）
控制体积在主瓣中散射光线的方向。各向异性的范围为 -1 到 1，默认值为 0（各向同性）。各向异性为 0 表示体积为各向同性：光会以相同概率向所有方向散射。
大于 0 的值表示体积为前向散射：入射光更可能沿相同方向散射（即远离入射光的来向）。

各向异性小于 0 表示体积为后向散射：入射光更可能沿反方向散射（即朝向入射光来的方向返回）。

Secondary Anisotropy（次各向异性）
控制体积在次瓣中散射光线的方向。各向异性的范围为 -1 到 1，默认值为 0（各向同性）。各向异性为 0 表示体积为各向同性：光会以相同概率向所有方向散射。
大于 0 的值表示体积为前向散射：入射光更可能沿相同方向散射（即远离入射光的来向）。

各向异性小于 0 表示体积为后向散射：入射光更可能沿反方向散射（即朝向入射光来的方向返回）。

Lobe Blend Factor（瓣混合因子）
控制两个瓣之间的混合权重。值为 0 表示只有第一个瓣处于活动状态，值为 1 表示只有第二个瓣处于活动状态。介于两者之间的值表示两个瓣都处于活动状态。



Sampling（采样）





Equiangular weight（等角权重）
使用等角采样而不是密度采样的概率。等角采样可改善靠近光源的体积的收敛，而在处理密集或密度变化很大的体积时，密度采样可能是更好的技术。默认值 0.5 表示两种技术会等量使用，并通过多重重要性采样进行组合。

如果体积密集且远离光源，降低等角权重可能带来更好的收敛。
此控制对聚合体积没有影响。

Min samples（最小采样数）
控制每条射线在体积内部取得的最小采样数。仅在单次散射时使用。默认值为 4。
此控制对聚合体积没有影响。

Max samples（最大采样数）
控制每条射线在体积内部取得的最大采样数。默认值为 4。
此控制对聚合体积没有影响。



Multiscatter Optimization（多重散射优化）



MultiScatter Optimization（多重散射优化）
控制是否应用 MultiScatter Optimization（多重散射优化）。此控制对聚合体积没有影响。


Extinction Multiplier（消光倍率）
Extinction Multiplier（消光倍率）用于多重散射，可在多重散射事件中修改消光系数。该倍率可设置为任何大于 0 的值；默认值 1 表示不做修改。
小于 1 的值会降低消光系数，大于 1 的值会提高消光系数。此控制对聚合体积没有影响。


Contribution Multiplier（贡献倍率）
Contribution Multiplier（贡献倍率）用于多重散射，可在多重散射事件中修改直接照明贡献。该倍率可设置为任何大于 0 的值；默认值 1 表示不做修改。
小于 1 的值会降低直接照明贡献，大于 1 的值会增强直接照明贡献。

此控制对聚合体积没有影响。
