Installation of RenderMan for Houdini

Source: https://rmanwiki-27.pixar.com/space/RFH27/544640398/Installation+of+RenderMan+for+Houdini
Downloaded: 2026-06-15T03:49:12+00:00


有关支持的 Houdini 版本，请访问 安装与许可。
可以使用 RenderMan 安装器安装 RenderMan for Houdini；该安装器负责下载、安装和许可 RenderMan Pro Server（实际渲染器）以及所有桥接产品。

Package Installation（软件包安装）

RenderMan for Houdini 可以通过 packages 文件安装，该文件位于 RfH 安装目录中。

将 packages JSON 文件复制到下列位置之一，以便 Houdini 在启动时找到它。SideFX 的高级 package 说明在 此处。

  • $HOUDINI_USER_PREF_DIR/packages

  • $HFS/packages

  • $HSITE/houdinimajor.minor/packages (例如， $HSITE/houdini21.0/packages)

  • $HOUDINI_PACKAGE_DIR

Manual Installation（手动安装） 

RfH 的环境变量必须在 houdini.env 文件中排在第一位。（Windows 需要使用分号）例如：

HOUDINI_PATH=$RFHTREE/3.11/21.0.512:&


带 Solaris 的 RenderMan for Houdini 需要在配置文件中增加一行，才能渲染内存中的 VDB：

RMAN_PROCEDURALPATH = $RFHTREE/3.11/21.0.512/openvdb:&

从 Houdini 21 开始，为了支持在 RenderMan 中读取 COP 纹理，需要定义以下变量：

RMAN_TEXTUREPATH = $RFHTREE/3.11/21.0.512/rtxplugins:&

RenderMan for Houdini 在 Windows 上需要在配置文件中增加一行：

PATH=$RMANTREE\bin;&


本文档下方提供了更多说明和详细信息。


安装插件后，Houdini 需要修改主目录中的 Houdini 环境才能加载 RenderMan 插件。Houdini 专用环境变量也可以在 houdini.env 中设置；各操作系统对应位置如下：

Windows: %HOME%\Documents\houdiniXX.X\houdini.env
Mac: ~/Library/Preferences/houdini/YY.Y/houdini.env
Linux: ~/houdiniZZ.Z/houdini.env

注意： 首次运行新版 Houdini 时，可能需要运行两次，上述位置才会出现 houdini.env 文件；文件创建后，就可以添加下面提到的各行。


需要用文本编辑器编辑 houdini.env 文件，并在启动 Houdini 之前把下面几行放入该文件。每个正在使用的 Houdini 新主版本只需要执行一次。





示例

例如在 Linux 上，应编辑此文件：$HOME/houdini21.0/houdini.env：

RMANTREE=/opt/pixar/RenderManProServer-27.0
RFHTREE=/opt/pixar/RenderManForHoudini-27.0
RMAN_PROCEDURALPATH=$RFHTREE/3.11/21.0.512/openvdb:&
RMAN_TEXTUREPATH=$RFHTREE/3.11/21.0.512/rtxplugins:&
HOUDINI_PATH=$RFHTREE/3.11/21.0.512/:&

在 Windows 上，应编辑此文件：\Users\myself\Documents\houdini21.0\houdini.env：

注意：Windows 使用分号而不是冒号来分隔 HOUDINI_PATH！

RMANTREE="C:\Program Files\Pixar\RenderManProServer-27.0"
RFHTREE="C:\Program Files\Pixar\RenderManForHoudini-27.0"
RMAN_PROCEDURALPATH=$RFHTREE\3.11\21.0.512\openvdb;&
RMAN_TEXTUREPATH=$RFHTREE\3.11\21.0.512\rtxplugins:&
HOUDINI_PATH=$RFHTREE\3.11\21.0.512;&
PATH=$RMANTREE\bin;&

最后一个 macOS 示例，应编辑：/Users/myself/Library/Preferences/houdini/21.0/houdini.env：

RMANTREE=/Applications/Pixar/RenderManProServer-27.0
RFHTREE=/Applications/Pixar/RenderManForHoudini-27.0
RMAN_PROCEDURALPATH=$RFHTREE/3.11/21.0.512/openvdb:&
RMAN_TEXTUREPATH=$RFHTREE/3.11/21.0.512/rtxplugins:&
HOUDINI_PATH=$RFHTREE/3.11/21.0.512:&
