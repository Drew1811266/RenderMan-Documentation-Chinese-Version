Converting Textures

Source: https://rmanwiki-27.pixar.com/space/RFH27/544640896/Converting+Textures
Downloaded: 2026-06-15T03:49:12+00:00

在 Solaris context（Solaris 上下文）中处理纹理时，需要手动将纹理转换为所需的 .tex 文件格式，以便供 RenderMan 使用。

这可以通过 Texture Manager 轻松完成；在 RenderMan → Texture Manager 下创建一个新的 Solaris pane（Solaris 面板）即可找到它。
