Analytic Light（解析灯光）

要添加 light（灯光），请选择一种 light tool（灯光工具），例如 Dome 工具，位于 RenderMan Shelf（RenderMan 工具架）中，并会自动创建 Pxr light OBJ。   

也可以通过 OBJ pane（OBJ 面板）中的 TAB menu（TAB 菜单）添加 light。所有 Pxr lights 都列在 Render TAB Menu 下。

Light Transformation（灯光变换）

你会注意到某些 light 的 transformation components（变换组件）是隐藏的。这是因为它们不适用于该特定 light type（灯光类型）。

Light Name（灯光名称）

Transformation（变换）

PxrRectLight（矩形灯）

平移、缩放和旋转。

PxrDiskLight（圆盘灯）

平移、缩放和旋转。

PxrSphereLight（球形灯）

平移、 uniform（均匀） 缩放和旋转。即使进行非均匀缩放，渲染时仍会显示为 sphere（球体）。

PxrCylinderLight（圆柱灯）

平移、 uniform（均匀） 横截面缩放、长度缩放和旋转。即使进行非均匀缩放，渲染时仍会显示为 sphere（球体）。

PxrDistantLight（远距灯）

Rotation only（仅旋转）。

PxrDomeLight（穹顶灯）

Rotation only（仅旋转）。

PxrPortalLight（门户灯）

类似 PxrRectLight，但还需要一个外部 coordsys（坐标系）来指定 dome texture（穹顶纹理）本身的 orientation（朝向）。

PxrEnvDayLight（环境日光）

Rotation only（仅旋转）。

PxrAOVLight（AOV 灯）

由于它不是设计用于 illuminate（照明）的真实 light，因此没有平移、缩放和旋转。


Portal Light（门户灯）

这个 PxrPortalLight（门户灯） 是通过一个 “portal”（门户）查看 dome light（穹顶灯）的优化版本。要设置 PxrPortalLight OBJ，需要将 Dome Light 参数指向一个 PxrDomeLight OBJ。一个 dome light 可以拥有多个 portal light 子级。如果希望 portal 继承 dome 的 transform（变换），可以将 portal light parent 到 dome 下（可选）。

RenderMan for Houdini 将根据以下规则专门处理 portal light：

  • 如果某个参数不是 portal light 的本地参数，它会继承 parent dome light 上该参数的设置。

  • Intensity Multiplier 会与 parent dome 的 light Intensity 相乘。

  • Color Tint 会与 parent dome 的 Color 相乘。

  • 它会同时应用 dome 的所有 light filters 以及自身的 light filters。

Mesh Light（网格灯）

要添加 PxrMeshLight（使用所选 geometry 作为 light source），请选择 mesh light tool    工具，位于 RenderMan Shelf（RenderMan 工具架）。此工具会要求你为 mesh light 选择一个对象。这对于创建拉伸霓虹灯或使用圆柱体制作 fluorescent lighting（荧光照明）等效果很有用。shelf tool 会自动填充 Light Geometry（灯光几何） 参数，也可以手动设置。 

Light Attributes（灯光属性）

下面显示 light 的默认属性。Camera visibility（摄影机可见性）默认关闭，便于在 camera（摄影机）前方放置灯光时进行美术指导。


  • Camera Visibility（摄影机可见性） - 使 light 对 camera rays（摄影机光线）可见。

  • Transmission Visibility（透射可见性） - 使 light 在 transmission/refraction（透射/折射）中可见。

  • Indirect Visibility（间接可见性） - 使 light 对 indirect specular reflections（间接镜面反射）可见。
