Analytic Light（解析灯光）

要添加灯光，请选择一种灯光工具，例如 Dome，位置在 工具，位于 RenderMan Shelf（RenderMan 工具架），它会自动创建 Pxr 灯光 OBJ。   

也可以通过 OBJ 面板中的 TAB 菜单添加灯光。所有 Pxr 灯光都列在 Render TAB Menu 下。

Light Transformation（灯光变换）

你会注意到某些灯光的变换组件是隐藏的。这是因为它们不适用于该特定灯光类型。

Light Name（灯光名称）

Transformation（变换）

PxrRectLight（矩形灯）

平移、缩放和旋转。

PxrDiskLight（圆盘灯）

平移、缩放和旋转。

PxrSphereLight（球形灯）

平移、 uniform（均匀） 缩放和旋转。即使进行非均匀缩放，渲染时仍会显示为球体。

PxrCylinderLight（圆柱灯）

平移、 uniform（均匀） 横截面缩放、长度缩放和旋转。即使进行非均匀缩放，渲染时仍会显示为球体。

PxrDistantLight（远距灯）

Rotation only（仅旋转）。

PxrDomeLight（穹顶灯）

Rotation only（仅旋转）。

PxrPortalLight（门户灯）

类似 PxrRectLight，但还需要一个外部 coordsys（坐标系）来指定穹顶纹理本身的朝向。

PxrEnvDayLight（环境日光）

Rotation only（仅旋转）。

PxrAOVLight（AOV 灯）

由于它不是真正用于照明的灯光，因此没有平移、缩放和旋转。


Portal Light（门户灯）

这个 PxrPortalLight（门户灯） 是通过 “portal”（门户）查看穹顶灯的优化版本。要设置 PxrPortalLight OBJ，需要将 Dome Light 参数指向一个 PxrDomeLight OBJ。一个穹顶灯可以拥有多个门户灯作为子级。如果希望门户继承穹顶的变换，可以选择将门户灯作为子级挂到穹顶下。

RenderMan for Houdini 将根据以下规则专门处理门户灯：

  • 如果某个参数不是门户灯的本地参数，它会继承父级穹顶灯上该参数的设置。

  • Intensity Multiplier 会与父级穹顶灯的 Intensity 相乘。

  • Color Tint 会与父级穹顶灯的 Color 相乘。

  • 它会同时应用穹顶灯的所有灯光滤镜以及自身的灯光滤镜。

Mesh Light（网格灯）

要添加 PxrMeshLight（使用所选几何体作为光源），请选择网格灯工具，位置在    工具，位于 RenderMan Shelf（RenderMan 工具架）。此工具会要求你为网格灯选择一个对象。这对于创建拉伸霓虹灯或使用圆柱体制作荧光照明等效果很有用。工具架工具会自动填充 Light Geometry（灯光几何） 参数，也可以手动设置。 

Light Attributes（灯光属性）

下面显示灯光的默认属性。相机可见性默认关闭，便于在相机前方放置灯光时进行美术指导。


  • Camera Visibility（相机可见性） - 使灯光对相机光线可见。

  • Transmission Visibility（透射可见性） - 使灯光在透射/折射中可见。

  • Indirect Visibility（间接可见性） - 使灯光对间接镜面反射可见。
