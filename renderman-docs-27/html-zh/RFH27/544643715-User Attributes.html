User Attributes

Downloaded: 2026-06-15T03:49:12+00:00

要向对象添加用户属性，可通过 Edit Parameter Interface（编辑参数界面）添加。

在 RenderMan 选项卡中的 Attributes（属性）或 Primvars（图元变量）区段下，将参数添加到 User 选项卡。该参数必须位于 User 文件夹内，并且必须带有 "ri_user_" 前缀才能被读取。 

还必须为该参数提供 Tag Value（标签值），用来告知 Houdini 和 RenderMan 该参数类型，并允许交互式编辑。

注意，我们使用了靠近底部的 tags（标签）字段，这是为了告知 Houdini 和 RenderMan 该 RenderMan 属性。有效值类型如下。

有效的 Tag Values（标签值）列在这里。它们必须匹配左侧名为 "Create Parameters" 的窗格中所选 Parameter（参数）的预期数据类型：

  • int

  • float

  • color

  • point

  • vector

  • normal

  • hpoint

  • matrix

  • string

这会向 RIB 输出如下内容。

 Attribute "user" "color MatteID0" [0 1 0]
