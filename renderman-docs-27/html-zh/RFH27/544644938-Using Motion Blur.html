Using Motion Blur

Source: https://rmanwiki-27.pixar.com/space/RFH27/544644938/Using+Motion+Blur
Downloaded: 2026-06-15T03:49:12+00:00

参见 Houdini 的 Motion Blur（运动模糊） 页面了解更多信息。

要启用运动模糊，请启用 RenderMan ROP 的 Properties（属性） | Motion Blur（运动模糊）：Allow Motion Blur（允许运动模糊）。

Motion Blur Type（运动模糊类型）：此选项会切换 hider 中对带运动模糊的微多边形进行采样，因此只有场景使用运动模糊时才有意义。默认情况下，此 hider 选项已启用（设为 1），表示微多边形会在 hider 中被模糊并采样。禁用（设为 0）后，微多边形在采样前不再被模糊；不过仍会为所有对象计算 dPdtime 变量。该变量可以通过任意输出变量（AOV）输出到单独图像中，并用于以后处理方式执行运动模糊。

Xform Time Samples（变换时间采样）：此对象的变换模糊运动采样数量。设为 0 时使用全局参数。调高该值会为对象输出更多运动段，从而在渲染中创建平滑的模糊/运动路径。采样足够时，这有助于让弯曲运动保持弯曲，而不是在两点之间呈线性。

Geo Time Samples（几何时间采样）：此对象的变形模糊运动采样数量。设为 0 时使用全局参数。对于会发生变形的对象，调高该值会输出更多采样，从而在渲染中创建平滑运动路径。采样足够时，这有助于让弯曲运动保持弯曲，而不是在两点之间呈线性。

Shutter Offset（快门偏移）：要添加到全局运动模糊时间上的偏移。

如果需要为模拟对象（破碎、下落、弹跳、粒子等）渲染运动模糊，需要启用对象节点上的 Geometry Velocity Blur（几何速度模糊） 来为这些对象提供在 RenderMan 中渲染运动模糊和变形模糊所需的正确数据。这是因为拓扑变化需要特殊处理，运动模糊才能正常工作。
