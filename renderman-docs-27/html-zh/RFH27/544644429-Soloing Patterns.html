Soloing Patterns

Source: https://rmanwiki-27.pixar.com/space/RFH27/544644429/Soloing+Patterns
Downloaded: 2026-06-15T03:49:12+00:00

对 pattern 执行 solo，是一种只查看该 pattern 应用于对象后的渲染结果、而不包含着色效果的方式。


这可以帮助艺术家放置或微调 pattern 的外观，而不必为了测试手动建立连接。请注意，这适用于连接到 BxDF（材质）的 pattern，而不适用于 Displacement node（置换节点）。

要 solo 一个 pattern，只需选择节点上的 "Debug" 图标或蓝色条。受影响的材质将停止渲染其他图案，并关闭着色效果；实际效果相当于使用 PxrConstant 材质。
