Solaris 中的灯光滤镜

Source: https://rmanwiki-27.pixar.com/space/RFH27/544641289/Light+Filters+in+Solaris
Downloaded: 2026-06-15T03:49:12+00:00




灯光滤镜用于操控照明结果。在现实世界中，遮片和色片通常通过改变镜头中光线的颜色、质量和数量来实现这种效果。RenderMan 的灯光滤镜是以同样方式（甚至更多方式）为场景布光的强大且灵活的手段。 

在 Solaris 中使用灯光滤镜很简单，并能立即改变你的灯光和照明效果！

要添加灯光滤镜，只需添加一个专用 LOP 节点，以创建对应的 USD 灯光滤镜图元。

之前的 PxrLightFilter LOP 虽然仍可工作，但已弃用。

然后使用以下任一方法将其分配给灯光：

  • 使用 PxrAssign LOP（或 PxrAssignLightFilter 别名）。可以用单个 PxrAssign LOP 将多个灯光滤镜分配给多盏灯。

  • 使用 Light Edit LOP，并填写 灯光滤镜 参数，该参数位于 Base Properties（基础属性） 选项卡

  • 将 Light Edit LOP 放在灯光滤镜 LOP 之后，并填写 灯光滤镜 参数，该参数位于 Base Properties（基础属性） 选项卡

如果分配正确，在 Scene Graph 中，该灯光图元上应在 light:filters 属性中列出灯光滤镜图元。

要将灯光滤镜作为灯光的子级（如上面的 Scene Graph 所示），只需设置 Primitive Path（图元路径） 参数，并在灯光滤镜 LOP 上填入正确路径（例如使用 ‘/lights/light1/$OS’，而不是默认的 '/lightFilters/$OS’）。
