Solaris Limitations

Downloaded: 2026-06-15T03:49:12+00:00

Known Limitations（已知限制）

Geometry（几何）

  • 不支持 NURBS 和 Implicits（隐式体）

Shading（着色）

  • 为了获得最佳性能，应通过 txmake 手动预转换纹理，或使用 Texture Manager

  • dynamic 和 uniform array 节点预计无法在 Solaris 中工作

Image Tool

  • 使用 ACES 渲染到 IT 当前需要将环境变量 IT_OCIOV1 设置为你的 ACES config.ocio

  • Crop tool 和 image selector 不受支持

OSX

  • OSX 上的 M1 当前不受支持。 
    可以通过 Rosetta 运行，但请注意，在加入原生支持之前，性能不会同样好。
