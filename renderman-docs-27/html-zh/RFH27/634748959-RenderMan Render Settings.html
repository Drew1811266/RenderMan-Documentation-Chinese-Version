RenderMan Render Settings

Source: https://rmanwiki-27.pixar.com/space/RFH27/634748959/RenderMan+Render+Settings
Downloaded: 2026-06-15T03:49:12+00:00

在 USD 中，通常应针对 AOV 使用 Render Var（渲染变量） primitive（图元）进行描述，并将这些 primitive 连接到一个 Render Product（渲染产品） primitive，在其中设置输出图像文件路径以及 display driver（显示驱动）。该 Render Product（渲染产品） primitive 随后必须连接到一个 Render Settings（渲染设置） primitive。请注意， Render Settings（渲染设置） primitive 也可以连接 integrator（积分器）、display filter（显示过滤器）和 sample filter（采样过滤器）。为了快速上手，Pixar 提供 RenderMan Render Settings LOP 节点，用来完成上面提到的全部 primitive 创建和连接。该 RenderMan Render Settings LOP 在内部使用 RenderMan Render Vars LOP，用于快速定义最常见的 RenderMan AOV。 RenderMan Render Vars LOP 也可以单独使用：如果你决定设置 Render Product（渲染产品） 和 Render Settings（渲染设置） primitive，并直接使用 SideFX 的 LOP 节点，而不是使用 RenderMan Render Settings LOP。

Image Settings（图像设置）

输出图像的渲染设置，例如分辨率以及从哪台相机渲染。

Render Settings（渲染设置）

用于设置渲染采样的选项卡

Render Product（渲染产品）

用于指定使用哪个 USD Render Product、filename（Render Name）等内容的选项卡

Render Vars（渲染变量）

用于设置 AOV 的选项卡

PxrPathTracer

Integrator（积分器）设置

Denoiser（降噪器）

你可以启用 Interactive Denoiser（交互式降噪器），并配置降噪设置。
