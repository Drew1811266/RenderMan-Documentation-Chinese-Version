要了解 PxrSurface 的所有细节及其可分层参数，请访问 Pixar Surface Materials。

创建简单分层材质最容易的方法是打开 RenderMan 菜单，选择 Create Material（创建材质），然后选择 Pxr Layered Surface。这里会在 MAT 面板中创建并连接一个分层材质。


PxrLayer

可以覆盖 PxrSurface 中的参数，方法是使用 PxrLayer 选项。 

在这个示例中，我们在 PxrSurface 中设置绿色 diffuse color（漫反射颜色）。然后在其上方叠加一个带有青色 diffuse color 的 PxrLayer 节点。请注意，一种颜色会覆盖另一种颜色，因此需要调整 Mixer node。

PxrLayer Mixer

可以使用 PxrLayerMixer 来组合 layers（层）。Mixer 用于决定通过 mask 在哪里以及如何混合各个 layer。下面，patterns 会连接到 Mixer 中的 Mask。


可以对 specular（镜面）、iridescence（虹彩）和其他 illumination lobes（照明波瓣）进行分层。
