Solaris Lighting Workflow

Solaris 允许你创建灯光并将其导出到 USD。
这些照明设置可以共享并导入到其他场景中。有关照明工作流的更多信息，请参阅 Solaris 文档。

Creating Lights（创建灯光）

在舞台视图中，添加 Light 节点，并从 Type（类型） 参数列表中选择灯光类型。PxrDomeLight 可通过单独的 Dome Light 节点使用。 

属于 USDLuxLight schema 的参数位于该节点的 Base Properties（基础属性）, Shaping（塑形）和 Shadow（阴影） 选项卡中。这些参数可用于多渲染器工作流。可用参数会因灯光类型而异。

RenderMan 灯光专用的参数位于该节点的 RenderMan 选项卡中。

Light Mixer 节点是查看和编辑基础灯光属性的便捷位置，例如强度、曝光和颜色。你还可以通过创建灯光集合，同时控制多盏灯上的这些效果。Light Mixer 还提供了将灯光静音和独奏的方式。

灯光链接

可以使用 Light Linker 节点完成灯光链接。
只需将灯光或灯光集合拖到中央的 Rules（规则） 区域。然后将几何体拖到每盏灯的开启/关闭区域。类似地，也可以将几何体拖入 Rules 区域，并把灯光加入该几何体的开启/关闭区域。

与 RfH 的差异

灯光形状默认对相机可见。若要关闭所有灯光的可见性，请创建 Render Geometry Settings 节点，并将 Primitives 范围设置为 "All Light Primitives"。

然后在 RenderMan 选项卡中，使用 Set and Create 覆盖该设置，再取消勾选 Camera Visibility 设置。

Solaris 不会自动转换你的纹理。 
要在光源上使用纹理，必须先使用 Texture Manager 面板将文件转换为 .tex。
