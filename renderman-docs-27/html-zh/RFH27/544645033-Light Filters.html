Light Filters（灯光滤镜）可在 RenderMan 文档中查看完整说明。

Adding Light Filter（添加灯光滤镜）

要添加 light filter（灯光滤镜），请在 OBJ pane（OBJ 面板）中通过 TAB menu（TAB 菜单）选择该 light filter。  

如果希望 filter 从 light 继承 transformation（变换），请将 light filter OBJ parent（设为子级）到 light OBJ 下。

可以对 light filter 进行平移和缩放。最好沿 Z 方向将 light filter 从 light 移开，以避免看到 light filter 的边缘。

Set Light Filter（设置灯光滤镜）

将 light filter VOP 拖到 light 的 Light Filter shaders（灯光滤镜着色器）中。
