Solaris Render Settings

Downloaded: 2026-06-15T03:49:12+00:00

正确安装 RenderMan for Houdini 后，渲染设置 LOP 的 RenderMan 选项卡下会添加专用参数。这些参数与对应 ROP 上的参数相同，并带有一些补充项。（这些补充项是渲染器中存在的参数，RfH 会在内部处理它们，而 Solaris 中的 hdprman 不会。） 

你可以在 此处了解更多 renderer options（渲染器选项）。

默认情况下，除非由控制参数指定，否则不会在 USD 中设置任何参数；该控制参数的选项包括：

  • Set or Create（设置或创建）- 创建 attribute opinion（属性意见）或覆盖现有 opinion。

  • Set if Exists（存在时设置）- 覆盖该属性上的现有 opinion。 

  • Block（阻断）- 停用 attribute opinion（属性意见）。Read about Attribute Block functionality（阅读 Attribute Block 功能说明） 此处。

  • Do Nothing（不执行操作）- 不更改。

Solaris Render Stage 也可以控制 Display view（显示视图）。注意，如果存在节点，这些设置可能会被上面的设置覆盖。

你也可以通过以下位置选择要从哪个节点获取设置：RenderMan persp（位于相机选择旁边）> Render Settings > 选择包含首选设置的节点。render 选项卡包含正确的选项。
