要添加 OpenVDB 体积几何，请在 Add 菜单中使用 Import OpenVDB（导入 OpenVDB 体积文件的菜单项） ：

这会添加一个 Blender OpenVDB 节点，并附加一个 PxrVolume shader（着色器）。下面是使用来自 OpenVDB 网站的 bunny_cloud.vdb 示例得到的渲染。

OpenVDB 体积属性设置

Filter Width（过滤宽度，用于 mipmap 层级选择）：设为 0 会禁用 mipmapping。若大于 0，小于 1 的值会偏向 mipmap 的更精细层级，大于 1 的值会偏向更粗层级。

Density Multiplier（密度乘数，用于缩放密度网格）：缩放体积 density grid（密度网格）中的值。这应比在 shader 中缩放密度更高效。

Density Rolloff（密度衰减控制，用于密度值渐变）： 大于 0 的值会在密度值上产生 logarithmic falloff（对数衰减）。较小的 rolloff 值会产生更强衰减。

DSO Min/Max（DSO 最小/最大值，用于体积范围加速）： 目前仅用于 聚合体积，并且仅适用于使用 ImplicitField DSO（例如 OpenVDB）的体积。设为 1 时，DSO 可能可以直接使用文件中存储的信息来计算体积内部的最小值和最大值。这可能让渲染器避免预先对体积进行 shading（着色），从而大幅缩短 time to first pixel（到首像素时间）。只有在直接使用体积的密度信号，或密度信号仅由 DSO 本身调制时，才能启用此项。任何对密度信号的 shading 修改都需要将该参数设为 off。
