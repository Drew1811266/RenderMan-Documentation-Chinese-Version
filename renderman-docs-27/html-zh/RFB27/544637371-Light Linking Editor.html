Light Linking Editor

Downloaded: 2026-06-15T03:49:12+00:00

灯光链接编辑器可让你控制场景中特定灯光是否照亮某些对象。 

打开灯光链接编辑器

你可以进入场景属性来打开该编辑器 并点击 Light Linking Editor。

也可以从对象上下文菜单打开：在 Object Context Menu 中点击 Light Linking Editor 菜单项。

示例

下面是一个简单示例场景，我们希望在其中控制照明。在这个场景中，坦克女孩很“自私”，想把所有灯光都留给自己，让猴头处在黑暗中。场景里有三盏灯。

在编辑器中，我们先从下拉列表添加一盏灯。

接下来选择要与这盏灯关联的对象。可以使用下拉列表。

或者，使用过滤图标进行搜索。

最后，选择所需的照明行为。

共有三个选项：

  1. Default（默认）：表示只继承原有行为

  2. On（开启）：此对象会接受这盏灯的照明

  3. Off（关闭）：此对象不会接受这盏灯的照明

在我们的例子中，希望照明关闭。对场景中的另外两盏灯重复这些步骤。

快捷方式

你也可以使用右键菜单来把对象与灯光链接或取消链接，也可以更改照明选项。对于右键菜单，最后选中的对象必须是 RenderMan 灯光。

Invert（反转）

你也可以反转灯光链接行为，也就是说，只有链接到某盏灯的对象才会被照亮。如果某盏灯尚未加入灯光链接列表，它仍会照亮场景中的所有对象。要启用反转行为，请进入场景属性选项卡，在 Light Linking 下开启 Invert Light Linking。注意，要使用 Invert Light Linking，Workflow 必须设为 RenderMan。
