Workspace in Blender

Source: https://rmanwiki-27.pixar.com/space/RFB27/544637132/Workspace+in+Blender
Downloaded: 2026-06-15T03:49:12+00:00

Workspace 编辑器可让你控制 RenderMan for Blender 中资产管理的各个方面。

要打开该编辑器，请进入输出属性，然后点击 Open Workspace（打开工作区）。


随后会打开一个新窗口，外观应类似下图。



Output 路径（Path）

场景中所有生成文件的根输出路径。它可以通过 <OUT> token 在其他路径中引用。

RIB Output 路径（Path）

控制 RIB 文件的输出路径和命名方式。

Beauty 图像 Output 路径（Path）

控制 beauty 图像的输出路径和命名方式。

AOV 图像 Output 路径（Path）

控制辅助 AOV 图像的输出路径和命名方式。

烘焙 3D 照明 Output 路径（Path）

进行 3D 照明烘焙时，控制 pointcloud 文件的输出路径和命名方式。

烘焙 2D 照明 Output 路径（Path）

进行 2D 照明烘焙时，控制 2D 纹理文件的输出路径和命名方式。

Version（版本）与 Take

这些设置控制 <version> 和 <take> token 的取值。

用户 Tokens

此部分允许你添加自己的 token/value 对，这些对可使用上方的输出路径，也可用于各种文件路径和字符串设置。
这些用户 token 会作为场景的一部分保存。
