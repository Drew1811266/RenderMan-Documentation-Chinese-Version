Denoiser in Blender

Downloaded: 2026-06-15T03:49:12+00:00

RenderMan for Blender 包含两种不同的降噪器：来自 Disney Research 的 RenderMan AI Denoiser（参见 Denoiser（降噪器））以及 Nvidia AI Denoising（Nvidia AI 降噪） 方案。 

Nvidia AI 降噪不适用于 macOS。

Interactive Rendering（交互式渲染）

有关在 "it" 中使用 Nvidia 降噪器的更多信息，请参阅 Image Tool（图像工具）中的 “AI Denoise” 小节。

对于交互式渲染（视口渲染、IPR 到 "it"），可以进入视口首选项，并在 IPR Denoiser 下选择一种降噪器来启用。Optix 指的是 Nvidia 降噪器。

RenderMan AI Denoiser 还有一些额外控制项：

  • Cheap First Pass：如果勾选，第一次降噪遍次会使用成本更低的启发式方法（稍快但质量较低）。如果渲染内容收敛很快，并且你希望优先更快得到降噪结果，这会很有用。

  • Min Samples：交互式降噪首次运行前，每个 bucket（区块）所需的最小平均采样数。

  • Interval：两次降噪运行之间的时间间隔（秒）。

对于预览渲染（例如按 F12 或 Render -> Render Image），如果渲染到 Blender 渲染窗口，可以在 Render Properties 选项卡中选择要使用的降噪器。

对于 Nvidia AI 降噪器，应该会立即在渲染窗口中看到降噪后的结果。

对于 RenderMan Denoiser，降噪器会在初始渲染完成后启动。渲染窗口中应该会出现一条消息，提示降噪已开始。

Options（选项）

  • Use Color Pass：未勾选时，beauty 通道会通过合并已降噪的漫反射通道和降噪后的镜面通道来完成降噪。这可以得到稍好一些的结果。不过，如果场景使用了任何 Display Filters 或 Sample Filters，可能会出现问题。遇到这种情况时，请勾选此框。

  • Asymmetry（不对称性）：控制不对称性值，0 表示最佳质量；较高值会促使降噪器避免过度模糊，从而产生较弱的降噪效果。

Batch Rendering（批量渲染）

有关 RenderMan for Blender 中批量渲染的更多通用信息，请参见 Blender 中的批量渲染 。

请注意，如果使用 Blender Batch，并且启用了 Use Blender Compositor 选项，那么渲染属性选项卡中 Display 区段下的降噪器设置会被采用。

如果使用 RIB 批量渲染样式，或关闭了 Use Blender Compositor，可以在 View Layer Properties 选项卡中启用降噪器。

在此批量渲染工作流中，只能使用 RenderMan Denoiser。Nvidia Optix 降噪器不是可用选项

你需要切换到 RenderMan Displays（有关在 Blender 中使用 AOV 的更多详细信息， 链接在这里！)

  1. 前往 View Layer Properties 并点击 Switch to RenderMan Displays

  2. 在 RenderMan Display 中，选择 beauty AOV 并启用 Denoise（降噪）

  3. 设置所需的降噪器参数，就可以开始了。
