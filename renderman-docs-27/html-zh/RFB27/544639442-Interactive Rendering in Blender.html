Starting IPR（启动 IPR）

在 Blender 中启动 IPR session（IPR 会话）有很多不同方式。 

Viewport（视口）

若要在 Blender viewport（Blender 视口）内渲染，可以把 viewport shading（视口着色）切换为 render（渲染）


或者，也可以选择 RenderMan IPR 图标，并选择将 IPR session（IPR 会话）发送到 Blender 的 Viewport（视口）或 RenderMan 的图像工具 it


另一种开始 IPR session（IPR 会话）的方式是右键打开 context menu（上下文菜单），然后选择 Render → IPR 或 IPR to it

IPR Operators（IPR 操作项）

IPR 启动后，你会在 viewport（视口）中看到一排如下所示的 operators（操作项）：

请注意，如果启动的是 IPR to it，并非所有这些 operators（操作项）都可用。

 
Stop IPR（停止 IPR） /
此按钮会停止当前 IPR session（IPR 会话）。

Refresh IPR（刷新 IPR） /
此按钮会先停止再重新启动 IPR session（IPR 会话）。

 
Integrator /（积分器 /）
允许你快速更改当前 integrator（积分器）。注意，这并不会实际影响为 scene（场景）选择的 integrator（积分器）。



Interactive Refinement /（交互式细化 /）
更改当前 Interactive Refinement（交互式细化）值。

在 RIS 中，该值决定 interactive rendering（交互式渲染）期间会对图像应用多少 refinement（细化，采用 dither pattern 抖动图案）。0 表示完全细化，最高到 6；6 表示每次 iteration（迭代）的细化最少。

在 XPU 中， interactive refinement（交互式细化）模式称为 Progressive Pixels (PP)。在 IPR 设置中，其视觉影响类似 RIS，主要目标是为 artists（艺术家）提供流畅的交互体验，代价是每次只积分一部分 screen pixels（屏幕像素）。启用 PP 后，screen bucket（屏幕分块）内的像素结果会从较大的 screen region（屏幕区域）开始逐步 splat（泼溅/铺散），一直细化到单个 pixel（像素）。当解析到单个 pixel 后没有收到编辑，XPU 会继续以最精细层级处理像素。只有收到新的编辑时，XPU 才会退回到使用较大的 progressive splats（渐进式铺散）。

Scale Resolution /（缩放分辨率 /）
允许你降低 viewport（视口）中 render（渲染）的 resolution（分辨率）。降低 resolution（分辨率）有助于提升交互速度。

Channel Select /（通道选择 /）
如果正在渲染多个 AOVs（AOV），此功能允许你在 viewport（视口）中快速切换 AOVs（AOV）。目前这在 XPU 中不起作用。

Crop Window /（裁剪窗口 /）
允许你绘制一个 crop window（裁剪窗口），使后续编辑只在该 crop window 内可见。这等同于使用 Blender 的 Render Region（渲染区域）。 

Snapshot /（快照 /）
保存 viewport（视口）中当前 render（渲染）的 snapshot（快照），并将图像保存到 Blender 的 render window（渲染窗口）中。

Enhance /（增强 /）
这只适用于 perspective cameras（透视摄影机）或 PxrCamera projection plugin（PxrCamera 投影插件）。

按 zoom factor（缩放因子）放大 pixel (X,Y) 周围区域，用于 troubleshooting（故障排查）。  
zoom factor（缩放因子）可在 RenderMan for Blender preferences（RenderMan for Blender 首选项）中更改。
被放大的 pixel（像素）会相对于图像保持锚定位置。vignetting（暗角）等 Camera effects（摄影机效果）会相应缩放。  

它有意不影响 level-of-detail（细节层级）、dicing（切分）、displacement（位移）或 MIP map levels（MIP 贴图层级）。
左键单击想要放大的区域。右键单击可重置 zoom（缩放）。按 Enter 可直接退出 operator（操作项）并保留当前 zoom（缩放）。按 Esc 可退出并重置 zoom（缩放）。

Texture Cache /（纹理缓存 /）
这会清除 RenderMan texture cache（RenderMan 纹理缓存）。只有当磁盘上的 .tex 文件在 Texture Manager（纹理管理器）之外发生更改时，才应需要这样做。

IPR Sampling（IPR 采样）


在 live IPR session（实时 IPR 会话）运行期间，你可以调整 Min / Max Samples、Pixel Variance 和 Interactive Refinement 设置。

Progressive Pixels（渐进式像素）

注意：在 XPU mode（XPU 模式）下运行时，将 Interactive Refinement 设置为高于 1 会激活 Progressive Pixels（渐进式像素）。
