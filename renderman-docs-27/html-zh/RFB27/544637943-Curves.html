RenderMan Curve primitive（曲线图元）非常适合渲染 hair and fur（头发和毛发），因为它旨在高效渲染大量极细的线束。

参见 曲线 了解更多信息。

曲线类型

RfB 应能渲染 Blender 中的大多数曲线类型。不过，并非所有设置都一定受支持。

曲线宽度

如果想改变曲线宽度，可以选择对象并进入 Edit Mode（编辑模式）。选中要调整宽度的顶点，然后修改 radius（半径）设置。

曲线的渲染与着色

Curves（曲线）最好使用 PxrMarschnerHair 或 LamaHairChiang 材质进行着色。若要给曲线着色或添加纹理，请使用 PxrHairColor pattern（图案节点）；它提供 melanin（黑色素）、dye（染色）、strand variation（发束变化）等选项。

要为材质添加变化，请使用 index 这个 primvar；它位于 PxrhairColor > hairIndexPrimvar 属性中，用于按发束索引驱动材质变化。
