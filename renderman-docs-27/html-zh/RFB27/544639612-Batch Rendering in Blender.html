Batch Rendering in Blender

刚完成一段精彩的新动画，现在该把它渲染出来了吗？

Batch Rendering（批量渲染）就是用来把所有漂亮帧渲染到磁盘的地方。

使用 RenderMan for Blender 进行 batch rendering（批量渲染）可以在 render properties（渲染属性）中的 External Rendering panel（外部渲染面板）完成。
下面看一下可用的一些选项。



Batch Render（批量渲染）

启动并 spool（提交）一个 batch render job（批量渲染作业）

Batch Bake Render（批量烘焙渲染）

启动并 spool（提交）一个 bake render job（烘焙渲染作业）

Render Animation（渲染动画）

决定是渲染完整 animation（动画，即多帧），还是渲染指定的 in / out frame sequence（入点/出点帧序列）。
frame range（帧范围）应通过 Start（开始） 和 End（结束） 设置来指定。
将 By 保持默认值 1 会渲染每一帧；但如果想在序列中每隔 5 帧渲染一次，只需在这里输入 5。

Spool Style（提交方式）

在 Blender Batch 或 RIB 之间选择。
Blender Batch 会生成一个单独的 blender 进程来渲染。对于 RIB，RenderMan for Blender 会先把外部 RIB 文件写入磁盘，然后再渲染（Blender 不参与渲染）。每种模式都有优缺点。 

Spool To（提交到）

在 Local Queue（本地队列）或 Tractor。
Local Queue 是本地 render queue manager（渲染队列管理器）；所有渲染都在本机完成。Tractor 是我们的 render farm job queue（渲染农场作业队列）和 work distribution system（工作分发系统）。选择它要求你的本机安装了 Tractor，并且本地网络中还安装了 Tractor Engine 和 Tractor blades。

Frame Chunking（帧分块）

这只适用于 Blender Batch 样式。
这决定每个 Blender Batch 进程要渲染多少帧。例如，如果渲染 10 帧序列，frame chunk（帧块）设置为 1 会产生 10 个独立的 blender batch 进程；而 frame chunking（帧分块）设置为 5，则只会产生 2 个 blender batch 进程，每个进程渲染 5 帧。Frame Chunking 配合 Persistent Data（持久数据） 选项开启后，可以帮助加速 batch renders（批量渲染）。

Use Blender Compositor（使用 Blender 合成器）

这只适用于 Blender Batch 样式。如果启用，所有 RenderMan Displays 选项都会被忽略，并且只保存最终 composited image（合成图像）。如果关闭，则会忽略 Compositor（合成器）中的所有 compositing operators（合成操作项）。如果 composited image（合成图像）需要 denoise（降噪），请启用 Optix denoise 选项。

Blender Batch AOVs（Blender Batch AOV）

仅适用于 Blender Batch 样式，并且 Use Blender Compositor 已开启。启用后，AOVs（AOV）也会写入磁盘（如果使用 RenderMan Displays，AOVs 会作为 EXR 文件写出）。

Enable Recovery（启用恢复）

这只在 Spool Style 设置为 RIB 时适用。用于决定是否尝试 recovery render（恢复渲染）。更多信息请参见 Checkpointing and Recovery（检查点与恢复）。checkpoint 设置可在 Sampling panel（采样面板）下找到。

Custom Render Commands（自定义渲染命令）

允许你向 render command（渲染命令）添加任何其他所需参数

RIB Format（RIB 格式）

如果 Spool Style 设置为 RIB，此项控制 RIB 是以 ASCII 还是 Binary 格式写出。Binary 可以节省磁盘空间，但无法用文本编辑器检查。

RIB Compression（RIB 压缩）

如果 Spool Style 设置为 RIB，此项控制是否还要对 RIB 文件进行 GZip 压缩，以节省更多磁盘空间。
