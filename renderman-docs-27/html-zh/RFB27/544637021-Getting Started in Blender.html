加载插件

与 Blender 的其他插件一样，可以在偏好设置（Edit→Preferences）的插件标签页中加载/启用 RenderMan for Blender。 



查找 RenderMan Pro Server

如 安装 文档中所述，RenderMan for Blender 要求你的机器上已安装 RenderMan Pro Server（RPS）。可以在偏好设置中配置插件查找 RPS 的方式：

将 RenderMan Location 更改为 Choose From Installed 即可从安装在默认位置的版本中选择。你也可以将其设置为 Set Manually 并手动指定 RPS 的位置。默认情况下会查询 RMANTREE 环境变量。更改这项设置后需要重启 Blender。

如果某种方法失败，插件会尝试几个不同的回退方式。查找顺序如下：

  • 首先，根据你的偏好设置尝试查找。

  • 如果 Choose From Installed 或 Set Manually 失败，则改为查询 RMANTREE 环境变量

  • 如果偏好设置是查询 RMANTREE 而该方法失败，则回退为尝试检测默认安装路径中找到的最新版本

第一次渲染

要开始使用 RenderMan 渲染，请将当前渲染引擎设置改为 RenderMan Render。

可通过以下任一方式执行：

1. 使用对象上下文菜单中方便的 Use RenderMan 快捷方式。在视口中右键点击并选择 Use RenderMan。

2. 或者从渲染属性面板中更改。

要进行渲染，可以点击渲染属性面板中的 Render 按钮，或使用视口中对象上下文菜单里的 Render 快捷方式：

渲染结果应出现在 Blender 渲染窗口中。

另外，你也可以将渲染输出到 RenderMan 的 "it" 帧缓冲区，方法是将 Render To 设置从 "Blender" 更改为 "it"。
