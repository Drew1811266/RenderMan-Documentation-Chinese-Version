Material Layering with PxrLayerSurface in Blender

RenderMan 允许通过材质分层来创建复杂外观。这意味着车漆、瓶身标签、尘土或泥污等效果，都可以高效地分层并用于渲染。

下面是一个分步示例，展示如何使用 PxrLayerSurface在青铜猴头上创建铜绿效果。PxrLayerSurface 与 PxrSurface 相同，只是大多数不适用于分层的参数都已从 UI 中隐藏。

Attach Material（附加材质）

将 PxrLayerSurface 材质附加到选中的对象。可以通过 Object Context Menu（对象上下文菜单）完成。

对于 PxrLayerSurface，RenderMan for Blender 会自动添加一个 PxrLayerMixer 节点，以及两个 PxrLayer 节点；它们会分别作为 base layer（基础层）和 upper layer（上层）。起始时，你的网络应如下所示：

Masking（遮罩）

如果直接渲染当前场景，你会发现只显示 patina（铜绿）层，而看不到 copper（铜）层。为了将两者混合在一起，我们需要使用一个遮罩值。对于遮罩，我们将使用一个 PxrDirt pattern 节点。Start（开始）时先创建一个 PxrDirt 节点，并将 resultR 输出连接到 PxrLayerMixer 的 Layer 1 Mask 参数。可以在 Shader Editor（着色器编辑器）中这样操作，如下图所示：

如果你想在脱离整个网络的情况下调整遮罩值，可以使用 Solo Node 功能。点击材质面板上的连接控件，然后点击 Solo Input Node。
