Stylized Looks in Blender

Source: https://rmanwiki-27.pixar.com/space/RFB27/544638744/Stylized+Looks+in+Blender
Downloaded: 2026-06-15T03:49:12+00:00

在 Blender 中开始使用 Stylized Looks 渲染非常直接，只需要几个简单步骤。

RenderMan 材质

在使用 Stylized Looks 渲染前，请确认已向场景添加灯光，并已为对象附加 RenderMan 材质。


  1. 在视口中右键点击并选择 RenderMan → Stylized（风格化） Looks（外观） → Stylized Looks Editor（Stylized Looks 编辑器）



  2.  请先开启名为 Enable Stylized Looks 的开关。然后选择一个对象，并在 Attach Stylized Pattern（附加风格化图案）列表里选取 PxrStylizedControl 控制节点。 

    按下 OK 后，会创建所需的 PxrStylizedControl 节点，并将它们附加到机器人模型的 PxrSurface 上。 
    PxrStylizedControl 网络需要附加到场景中的每个对象，并允许对每个对象进行独立的着色控制。

    也可以通过选择一个对象，然后选择 RenderMan → Stylized Looks（风格化外观） → Attach Stylized Pattern（附加风格化图案） → PxrStylizedControl





  3. 使用 RenderMan Stylized Editor（RenderMan 风格化编辑器）后，现在可以开始添加风格化滤镜，以创建你独特的风格化外观。

    在此示例中，我们会添加排线滤镜 PxrStylizedHatching。



  4. 现在启动 IPR，场景就会以 Stylized Looks 开始渲染。 




    注意：在此示例中，默认设置已按如下方式调整：

    a) Signal Energy From（信号能量来源）： Albedo AOV（反照率 AOV）
    b) Blending Mode（混合模式）： Weighted（加权）
    c) Blend Black Darkest（黑色按最暗混合）： ON（开启）
    d) Projection（投影）： Screen（屏幕）
    e) Hatching Frequency（排线频率）参数： 4

    现在你已经可以使用 Stylized Looks 渲染了！
    你的 NPR 创作之旅已经开始。不要忘记，Stylized Looks 的强大之处在于可以组合多种风格化滤镜，创建你自己的独特风格和外观。

    Stylized Looks 还随附多个有趣的预设，帮助你快速开始。你可以在 RenderMan Preset Browser 中找到这些预设。
