Attaching a Material（附加材质）

可以使用 Object Context Menu（对象上下文菜单）中的快捷方式，把 Materials（材质，bxdf/Bxdf）附加到选中的对象：


这会将选中的材质附加到 Outliner（大纲视图）中选中的对象。

也可以在 Material Properties（材质属性）标签页中，按常规方式把材质附加到对象 。

点击 New（新建）：

然后点击 Add RenderMan Nodetree（添加 RenderMan 节点树） 

并选择要使用的 RenderMan 材质。

RenderMan shading networks（着色网络）始终会终止于一个 RenderMan Material Output（RenderMan 材质输出） 节点。

下面是一个 shading network（着色网络）示例：

Patterns（图案）

Pattern nodes（图案节点）可以像常规方式一样，使用 Shader Editor（着色器编辑器）连接到材质或其他 patterns（图案） ，也可以直接在 Material Properties（材质属性）标签页中完成。 。
对于可以接受 pattern connection（图案连接）的参数，参数值旁会显示一个  。这是连接控件。 

点击该控件会显示连接菜单。示例如下：

Patterns（图案）会按类别拆分，方便浏览。__EXISTING__ 部分会显示 shader graph（着色器图）中已有、可连接的节点。如果该参数已经有连接，还会出现更多选项。

顶部一行会显示当前连接到该参数的节点。Disconnect（断开连接）项会断开该参数与输入节点的连接。Remove（移除）项会断开连接，并从图中移除该输入节点。Solo Input Node（单独显示输入节点）可让你 solo（单独查看）输入节点。如果你想专注调整该节点，而不受整个材质影响，这会很有用。
