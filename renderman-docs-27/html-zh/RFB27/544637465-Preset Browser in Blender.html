Preset Browser in Blender

Source: https://rmanwiki-27.pixar.com/space/RFB27/544637465/Preset+Browser+in+Blender
Downloaded: 2026-06-15T03:49:12+00:00

Blender 中的 Preset Browser 可让你保存并复用材质、light rigs（灯光组）和环境贴图（HDR 图像）等资产。RenderMan 已随附一个预制资产库。还可以从 RenderMan 网站的 Resources（资源） 部分下载更多资产。

启动 Preset Browser

要调出 Preset Browser，请在视口中按 'n' 打开侧边面板，点击 Renderman 选项卡，此时应能看到 RenderMan Presets 面板。点击 Preset Browser 按钮会打开 UI。 

你也可以从视口中的右键菜单打开 Preset Browser：

UI

Info（信息）

默认情况下，preset browser 会使用 RenderMan 随附的 factory library（出厂库）。在窗口顶部，你会看到库名称，以及该库在磁盘上的完整路径：

锁形图标 表示该库是只读的。资产可以导入到场景中，但不能写入该库。

Top（顶部）

预览图像会显示当前选中 preset 的预览。右侧是当前选中 preset 的 meta information（元信息）。

预览图像下方的按钮可让你从当前选中的灯光保存 light rig，从当前选中的对象保存材质，或保存环境贴图。

顶部一行包括：

Add Library（添加库） - 此按钮可让你向 preset browser 添加一个新库。如果选择了空文件夹，或者在选中的文件夹中找不到现有库，factory library（出厂库）会被复制到该文件夹。 

Select Library（选择库） - 此按钮可让你从已添加到 preset browser 的现有库中进行选择。

Forget Library（忘记库） - 此按钮会让 preset browser 从已保存的库列表中忘记/移除当前库。

Edit Library Info（编辑库信息） - 此按钮可让你编辑当前库的 meta information（元信息），包括名称和 version。

Bottom（底部）

左侧显示资产分类。你可以使用 + 和 - 按钮添加或移除分类。注意，顶层分类（EnvironmentMaps、LightRigs、Materials）无法移除。

右侧显示当前选中分类可用的资产。其下方是 operations menu（操作菜单）：

Import and Assign to selected（导入当前资产并指定给当前选中的对象）：导入当前资产，并 Assign（指定）给当前选中的对象。此操作只适用于材质。

Import（导入）：这只是把当前资产导入到场景中。

Move to category（移动到分类）：可将当前资产移动到另一个分类。

Inspect json file（检查 json 文件）：可查看当前资产在磁盘上的 .json 文件。

Delete（删除）：删除当前资产。
