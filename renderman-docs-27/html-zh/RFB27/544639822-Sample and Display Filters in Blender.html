Sample and Display Filters in Blender

Source: https://rmanwiki-27.pixar.com/space/RFB27/544639822/Sample+and+Display+Filters+in+Blender
Downloaded: 2026-06-15T03:49:12+00:00

Sample filter 和 display filter 插件分别用于在 integrator（积分器） 生成着色采样和像素值之后，并在它们发送到 display driver（显示驱动器）之前，对这些值进行调整。

Sample Filters（采样滤镜）

Sample Filter（采样滤镜） 插件描述的是在像素过滤之前对采样值进行的颜色变换。它们作用于尚未应用像素过滤的原始 camera ray samples（相机光线采样）。一旦采样滤镜修改了某个采样并将其写入 framebuffer（帧缓冲），该修改就会永久成为所有渲染图像（包括深度图像）的一部分。请注意，如果单个 任意输出变量（AOV） 作为多个图像文件的一部分输出，则对某个采样的一次修改可能会影响所有这些图像。采样滤镜所做的修改也会影响 adaptive sampler（自适应采样器）。

  • PxrSampleFilterCombiner

  • PxrCryptomatte

Display Filters（显示滤镜）

Display Filter（显示滤镜） 插件描述的是在像素过滤之后、输出之前对最终像素值进行的颜色变换。多个图像共享的 AOV 在此时会被视为彼此独立，因此可以分别修改。目前 Display filters 不支持 deep images（深度图像），并且不会影响 adaptive sampler（自适应采样器）。

  • PxrLightSaturation

  • PxrDisplayFilterCombiner

  • PxrImageDisplayFilter

  • PxrShadowDisplayFilter

Blender 设置

可以创建 Display and Sample Filters（Display Filters 和 Sample Filters） ，位置在渲染属性中 。点击 Add RenderMan Display Filters Nodetree 按钮，或点击 Add RenderMan Sample Filters Nodetree。

你应该会看到 Display Filters 和 Sample Filters 面板展开，类似下图。

点击 Add RenderMan Display Filters Nodetree 会自动添加 PxrBackgroundDisplayFilter 节点，并继承 World color（世界颜色）作为背景色。


可以使用多个 Display and Sample Filters。点击 Add 会新增一个槽位，用于添加额外的 Display Filter 或 Sample Filter。若要移除某个滤镜，请点击名称旁边的 - 按钮。与 integrators（积分器）类似，显示滤镜和采样滤镜本质上都是 World 节点内部的着色节点。
