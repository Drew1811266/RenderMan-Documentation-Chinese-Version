RenderMan 27 for Blender

Source: https://rmanwiki-27.pixar.com/space/RFB27/544636929/RenderMan+27+for+Blender
Downloaded: 2026-06-15T03:49:12+00:00

RenderMan for Blender（RfB）通过 Blender 让用户轻松访问 RenderMan 功能。

该 addon（插件）还以新的方式将 Blender 与 RenderMan Pro Server 连接起来，无需编写 RIB 即可获得快速交互式结果。RIB-out 功能继续支持现有 pipelines（管线）和 farm rendering（农场渲染）。

RfB 为 Blender 用户提供对 RenderMan 速度、能力和稳定性的真正无缝访问；用户既可以保持简单的、以 Blender 为中心的 workflow（工作流），也可以利用 plugins（插件）的灵活性来创建最佳的自定义 pipeline（管线）。

Installation（安装）

RenderMan for Blender 要正常运行，需要同时安装 RenderMan Pro Server 和 RfB plugin（插件）。请参阅 Installation instructions（安装说明） 了解更多信息。
