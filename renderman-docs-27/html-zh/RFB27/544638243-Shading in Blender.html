Shading in Blender

Source: https://rmanwiki-27.pixar.com/space/RFB27/544638243/Shading+in+Blender
Downloaded: 2026-06-15T03:49:12+00:00



RenderMan for Blender 中的 Materials（材质）和 shading networks（着色网络），基本上与 Blender 内置渲染器一样，都是使用 Shader Editor（着色器编辑器）构建。不过，两者在一些快捷操作和 UI 控件上有所不同。


  • 附加材质
  • Blender 中的 Displacement（置换）
  • Blender 中的 Fur/Hair Shading（毛发着色）
  • Blender 中使用 PxrLayerSurface 进行 Material Layering（材质分层）
  • Blender 中的 UDIMs
  • Texture Reference Pose（纹理参考姿势）
