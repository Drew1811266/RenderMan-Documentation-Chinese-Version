导出为 RIB 归档

要把一个几何体或一组几何体导出为 RIB 归档，请选择对象，进入对象属性 ，在 RenderMan Geometry（RenderMan 几何）面板下选择 Export（导出），然后选择 Export Object as Archive（将对象导出为归档）。

还可以选择是否导出已指定给该几何体的材质，以及是否归档所有帧。

如果预计需要覆盖归档中对象上的材质，请取消勾选 Export Material（导出材质）。

导入 RIB 归档

可以从 Add 菜单导入 RIB 归档。

材质覆盖设置（Override Material 面板）

如果要给 RIB 归档附加材质，可以在对象属性中的 Material Override（材质覆盖）面板完成 。 注意： 如果归档本身的对象上已经附加了材质，则该材质优先。
