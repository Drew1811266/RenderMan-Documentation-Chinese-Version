String Tokens in Blender

Source: https://rmanwiki-27.pixar.com/space/RFB27/544637295/String+Tokens+in+Blender
Downloaded: 2026-06-15T03:49:12+00:00

RenderMan for Blender 可以在字符串参数中替换多种预定义 tokens。
它们可用于动画纹理序列、引用特定存储位置等。每个 token 都应包在一对尖括号内。

下面是默认 tokens 列表。


Token（令牌）含义
<TEMP>临时目录。在 Linux 和 macOS 上设置为 /tmp；在 Windows 上为 C:/tmp。
<OUT>场景中所有生成文件的主根输出目录。可在 Workspace Editor（工作区编辑器）中设置。
<blend>当前 blend 文件名。如果当前场景尚未保存，则设为 UNTITLED。
<blend_dir>当前 blend 文件所在的目录。
<scene>正在渲染的当前场景名称。
<layer>正在渲染的当前视图层
<aov>AOV 名称
<f4>帧号填充；本例中为 4。若不需要填充可以省略，也可以使用更小数字。大写或小写的 'f' 都可识别。
<ext>文件类型扩展名
<file>这与上面的 <blend> 相同
<version>插入在 Workspace Editor（工作区编辑器）中指定的 version 编号
<take>插入在 Workspace Editor（工作区编辑器）中指定的 take 编号
<date>Current（当前）日期
<frame>十进制帧号
<jobid>唯一 job identifier（作业标识符）
<time>Current（当前）时间
<ws>与上面的 <OUT> 相同

还可以添加 Custom（自定义）用户 tokens（请参见 Workspace Editor（工作区编辑器））。
