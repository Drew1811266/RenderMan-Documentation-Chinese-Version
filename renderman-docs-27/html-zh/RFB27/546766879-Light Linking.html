从 RenderMan for Blender 27.0 开始，灯光链接现在有两种模式。第一种模式默认启用，依赖 RenderMan Light Linking Editor（灯光链接编辑器）。第二种模式使用 Blender 的原生灯光链接工作流（请参阅 Blender 文档， 此处）。请注意，原生工作流要求 Blender 4.1 或更高版本。

Native Workflow（原生工作流）

要在 RenderMan for Blender 中使用 Blender 原生灯光链接，请在渲染属性选项卡的 Light Linking 下将 Workflow 设置改为 Blender，如下图所示：

Example（示例）

这里有一个简单场景，包含 Suzanne 模型、一个环面和一套三点布光设置。

可以从主光开始进行灯光链接（在本例中它名为 PxrDiskLightShape1）：进入该灯光的对象属性，并在 Shading → Light Linking 下查看。

创建该集合后，会立即关闭此灯光对该场景中所有对象的照明。

现在，如果想把这盏灯与 Suzanne 模型链接起来，可以将 Outliner（大纲视图）中的 Suzanna 对象拖放到灯光链接集合中。

这里可以看到，现在这盏灯只照亮 Suzanne 模型。
