Shading Variation（着色变化）

现实世界中没有什么东西看起来完全相同，那么实例化出来的储物柜为什么要一模一样呢？

先来看一下我们用于储物柜的 Lama 着色设置。它由多个层构成。

Layer 1（第 1 层）：Base Metal（基础金属）- LamaConductor

基础材质是一个 LamaConductor，IOR 设置为 Aluminium（铝）。

Layer 2（第 2 层）：Paint（油漆）- LamaDiffuse + LamaDielectric

为了在金属储物柜表面创建油漆层，我们使用一个 LamaDiffuse 设置提供基础颜色。在 LamaDiffuse 之上（Over），再添加一个 LamaDielectric 层来加入 Specular Roughness（镜面粗糙度）。

随后使用边缘遮罩设置，把这个油漆层叠加到基础金属之上；该设置会加入边缘破损和划痕，让下方的基础金属层显露出来。

Layer 3（第 3 层）：Dirt（污垢）- LamaDiffuse

储物柜顶部又添加了一层污垢，它由一个简单的污垢色 LamaDiffuse 节点提供，并通过 PxrDirt 节点控制，把污垢加入储物柜的遮蔽区域。

那么，如果希望每个储物柜都有不同颜色呢？

这很简单！

关键在于 PxrVary 节点。

在下面的示例中，我们把 PxrVary 节点连接到了 PxrHSL 节点的 input color（输入颜色）；PxrHSL 提供了原始的红色基础颜色。

为了展示材质变化，我们调大了 PxrVary 的色相值，以演示 PxrVary 产生的效果。

它们看起来很花哨，但现实中，每个储物柜通常会有相似的基础颜色，只带一点颜色变化。可以想象，每个储物柜最初都是同一种颜色（毕竟都来自同一桶漆），但随着时间推移，有些被阳光晒褪色了，有些则重新喷过漆。

下面的示例通过 Hue 参数设置基础红色，然后细微调整每个实例的饱和度、亮度和 gamma。每个储物柜之间的差异很轻微，而这正是我们想要的效果。

如果进一步强化这个储物柜着色器，细心观察会发现，每个储物柜都有不同的边缘破损、污渍和粗糙度。

不仅储物柜基础红色的变化由 PxrVary 随机化，所有已用纹理的放置也会通过 PxrRoundCube 节点内部的 Randomise（随机化）参数来随机化。
