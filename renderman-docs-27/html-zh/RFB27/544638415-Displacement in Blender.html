Displacement in Blender

Downloaded: 2026-06-15T03:49:12+00:00

要在 RenderMan for Blender 中为几何体渲染置换，请使用 PxrDisplace 节点。

添加 PxrDisplace

要添加 PxrDisplace，请确认已经为几何体添加了 RenderMan 材质。在材质属性  中，在 Displacement（置换）面板下，从 "Displacement" 旁边的下拉列表中选择 PxrDisplace。

你可能主要会使用以下参数：

  • Gain： 置换的 Gain。若为零，则不会产生置换。 

  • Scalar Displacement： Scalar Displacement（标量置换）输入。这是一个 float 置换（浮点置换）。

  • Vector Displacement： Vector Displacement（矢量置换）输入。矢量置换数据是一个三通道数据文件，即 XYZ。

还有一些用于置换的实用图案节点：

  • 对于与置换变换相关的功能，请使用 PxrDispTransform 图案节点。 

  • 对于置换分层，请使用 PxrDispScalarLayer 和 PxrDispVectorLayer 图案节点。 

避免多边形网格出现裂缝

对具有硬法线的网格进行置换时，面可能会分离并产生裂缝。置换会沿顶点自身法线的方向移动顶点，但硬边两侧的相邻顶点具有不同法线，因此会朝不同方向移动，从而产生间隙或裂缝。

RenderMan 提供了 "polygon:smoothdisplacement" 属性，它会创建一份完全平滑的法线副本，并用它代替原始法线。

在 RenderMan for Blender 中，可以通过对象数据属性里的 "Prevent polygon cracking"（防止多边形开裂）复选框启用该属性  ，该复选框位于网格上。
