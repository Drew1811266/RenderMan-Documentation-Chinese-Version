Installation of RenderMan for Blender

Source: https://rmanwiki-27.pixar.com/space/RFB27/544636971/Installation+of+RenderMan+for+Blender
Downloaded: 2026-06-15T03:49:12+00:00

RenderMan for Blender 要求同时安装并授权 RenderMan Pro Server。

如需查看支持的 Blender 版本列表以及安装 RenderMan Pro Server 的说明，请参阅 Installation and Licensing（安装与许可）。

RenderMan for Blender 仅支持 Intel 版本的 Blender，因为 RenderMan 目前尚不支持 Apple Silicon。

Downloading（下载）

Non-commercial Users（非商业用户）

下载 Non-Commercial RenderMan（NCR）安装程序： https://renderman.pixar.com/intro 

运行安装程序，输入 forum username（论坛用户名）和 password（密码）。点击 Next 后将下载 NCR license（许可证）。

接下来，请务必在列表中选择 RenderMan Pro Server 和 RenderMan for Blender，然后点击 Next。随后 RenderMan Pro Server 会自动安装。关于 RenderMan for Blender，请参阅下面的 Installing（安装） 小节。


Commercial Users（商业用户）

访问 RenderMan downloads page（下载页面）：

https://renderman.pixar.com/forum/download.php

你可以下载适用于当前运行平台的 RenderMan Installer（安装程序）。

运行安装程序时，它会要求输入 forum username（论坛用户名）、password（密码）和 serial number（序列号）。 

接下来，请务必在列表中选择 RenderMan Pro Server 和 RenderMan for Blender，然后点击 Next。随后 RenderMan Pro Server 会自动安装。关于 RenderMan for Blender，请参阅下面的 Installing（安装） 小节。

GitHub

如果你是开发者，也可以从 RenderMan GitHub repository（仓库）克隆 RenderMan for Blender： https://github.com/prman-pixar/RenderManForBlender

Installing（安装）

与 Blender 的任何其他 addon（插件）一样，RenderMan for Blender 可以从 Blender preferences（偏好设置）中的 Addons tab（插件标签页）使用 .zip 文件安装：

  1. 点击 Install 按钮

  2. 浏览并选择 RenderManForBlender_27.0_XXXXX.zip 文件。如果你使用了安装器，该文件应位于 Downloads 文件夹中。

  3. 点击 Install Add-On。

现在你已经可以使用 RenderMan for Blender。请参阅 Getting Started in Blender（Blender 入门） 开始使用 RenderMan for Blender。
