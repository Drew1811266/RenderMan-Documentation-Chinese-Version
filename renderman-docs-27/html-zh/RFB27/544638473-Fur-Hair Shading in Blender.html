Fur/Hair Shading in Blender

在 RenderMan for Blender 中进行皮毛和毛发渲染/着色时，可以沿用 Blender 中渲染毛发的常规方式。用于毛发的主要 RenderMan 材质是 PxrMarschnerHair。

创建毛发系统

若要创建毛发系统，请选择要添加毛发的网格对象，然后点击粒子属性。添加新的粒子系统，并选择 Hair（毛发）。

你的对象在 viewport（视口）中应大致如下。

毛发着色

如前所述，我们会使用 PxrMarschnerHair 材质为毛发着色。为此，请点击网格的材质属性 。在材质列表中，点击 + 按钮添加新的材质插槽，然后点击 New（新建）。

点击 Add RenderMan Nodetree（添加 RenderMan 节点树），然后从下拉菜单中选择 PxrMarschnerHair。

回到粒子属性 中，进入 Render（渲染）部分，并将 Material（材质）改为我们刚添加的 PxrMarschnerHair 新材质。

Artistic Control（艺术控制）

PxrMarschnerHair 材质应该足以渲染真实感毛发。不过，这些参数可能有点偏技术化。为了获得更适合艺术家的控制，可以在 PxrMarschnerHair 之外配合使用 PxrHairColor pattern 节点。PxrHairColor 有三个输出需要连接到 PxrMarschnerHair。最好使用 Shader Editor（着色器编辑器）  来完成这些连接。下图展示了它们应如何连接。

将 Mode（模式）改为 Artistic（艺术化）后，可以获得更适合艺术家的控制，帮助完成毛发着色。

Hair ID Primvar（毛发 ID Primvar）

PxrHairColor 包含用于为颜色添加随机性的控制项。这由一个 ID primitive variable 控制。在 RenderMan for Blender 中，我们会为每一根毛发添加一个 "index" primitive variable。这个 "index" primitive variable 可作为 ID primvar 的来源。
