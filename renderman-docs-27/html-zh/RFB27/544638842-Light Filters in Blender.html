Light Filters in Blender

Source: https://rmanwiki-27.pixar.com/space/RFB27/544638842/Light+Filters+in+Blender
Downloaded: 2026-06-15T03:49:12+00:00

Light Filters（灯光滤镜）用于操控 lighting results（照明结果）。在真实拍摄中，这通常通过 stencils（遮片）和 gels（滤色片）来改变镜头中光的颜色、质感和光量。RenderMan 的 light filters 是一种强大而灵活的方式，可在场景照明中完成同样的工作，甚至做得更多。 

Create Light Filters（创建灯光滤镜）

要为灯光添加 light filter（灯光滤镜），先从 Add 菜单创建一个 light filter：

接下来，选择想要添加滤镜的灯光，并查看 object properties panel（对象属性面板）。你应该会看到 Light Filters 面板。 

点击 + 以添加新条目，然后选择要链接的 light filter object（灯光滤镜对象）。

Shortcut（快捷方式）

也可以使用 object context menu（对象上下文菜单）中的快捷方式。选择想要过滤的灯光，并从 Attach New Light Filter 菜单中选择要添加的 light filter。新的 light filter 会自动添加到 light filters 列表中。
