Light Mixer Editor

灯光混合器编辑器是一个便捷窗口，可让你在同一界面中编辑多盏灯光。

打开 Light Mixer（灯光混合器）

你可以从场景属性中打开灯光混合器编辑器  然后点击 Open Light Mixer Editor（打开灯光混合器编辑器）。

或者，也可以从 Object Context Menu（对象上下文菜单）打开它。

创建 Mixer Group（混合器组）

要创建一个组，请点击右侧的 + 按钮。系统会为该组分配一个默认名称。

向组中添加灯光

创建组之后，你可以使用下拉列表从场景中添加灯光，也可以使用漏斗图标进行搜索。

使用 Light Mixer（灯光混合器）

把灯光添加到组中后，你应该会看到类似下面的内容。

可以看到，UI 会显示一些你通常会在灯光上编辑的常用参数。从左到右分别是：

  • “-” 按钮可将选中的灯光从选中的组中移除

  • 你添加的灯光名称

  • Temp：用于切换为按色温来指定灯光颜色。

  • Intensity：灯光的强度

  • Exposure：灯光的曝光

  • 灯泡图标可让你单独显示选中的灯光。如果单独显示某盏灯，场景中的所有其他灯光都会关闭

  • 眼睛图标可让你静音一盏灯，也就是在场景中关闭该灯光。
