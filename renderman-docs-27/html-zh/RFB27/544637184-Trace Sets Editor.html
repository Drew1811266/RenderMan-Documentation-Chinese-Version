定义组成员关系后，RenderMan 就可以为特殊效果把不同对象和灯光关联起来。例如，你可以把一盏灯与一组对象分到一起，以阻止这些对象投射来自这盏特定灯光的 shadows（阴影），甚至可以出于艺术目的避免反射某个对象。
你也可以在后期合成中通过 Light Path Expression（光路表达式） （LPE）来实现其中一些效果；不过，也可以在渲染期间以全局方式实现这些效果。这样一来，你的 LPE 会在全局范围内遵循这些选择。

为什么？

trace sets 对以艺术方式微调图像很有价值。你看到的许多图像都经过面向发布的调整：瑕疵会被移除，分散注意力的图案会被涂掉，就连不理想的高光和构图也会根据艺术指导被“修正”。

渲染时也适用同样的规则。你可能遇到过艺术总监要求你移除画面外某个分散注意力的阴影，或者擦掉某个过亮、遮挡画面的物体反射。有些情况很简单，但有时你需要只从单个对象，或者某个集合中移除这个造成问题的效果。

Trace Sets 正是完成这类处理的地方！

使用 Trace Sets 编辑器

要在 Blender 中打开 trace sets 编辑器，请进入场景属性面板，找到用于打开 Trace Sets Editor 的按钮并点击它。 

你也可以从视口中的对象上下文菜单打开它。

首次打开编辑器时，它应像这样为空：

要添加新的 trace set，请点击右侧的 + 按钮。系统会自动使用一个默认 trace set 名称。
点击 - 按钮会移除当前选中的 trace set。

接下来，从下拉菜单选择要添加到当前 trace set 的对象。你也可以点击左侧的漏斗过滤按钮来执行筛选搜索。

添加后的对象会出现在底部列表中。
点击对象名称旁边的 - 按钮，会将该对象从当前选中的 trace set 中移除。
