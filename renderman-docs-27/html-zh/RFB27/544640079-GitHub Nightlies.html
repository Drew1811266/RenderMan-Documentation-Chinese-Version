RenderMan for Blender 会持续获得更新和错误修复，因此请参考 RenderMan for Blender GitHub 页面获取最新的每夜构建版本。 
请注意，GitHub 页面上的发布包并不总是经过完整测试的版本，因此请自行承担使用风险。

GitHub 页面位于：
https://github.com/prman-pixar/RenderManForBlender

安装时，请使用安装 RenderMan for Blender 的常规方法。

  1. 下载 RenderManforBlender.zip 文件
  2. 启动 Blender，并从 File → User Preferences → Add-ons 安装已下载的 .zip 文件。
  3. 将渲染器切换为 RenderMan，然后开始使用。
