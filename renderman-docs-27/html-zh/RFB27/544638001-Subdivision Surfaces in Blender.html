Subdivision Surfaces in Blender

Source: https://rmanwiki-27.pixar.com/space/RFB27/544638001/Subdivision+Surfaces+in+Blender
Downloaded: 2026-06-15T03:49:12+00:00

RenderMan 细分曲面使用 Pixar 的 OpenSubdiv 来确保始终渲染出最平滑、最干净的几何体。这样可以避免因多边形镶嵌细分不佳而产生的渲染伪影，例如暗边和轮廓线。

使用 RenderMan 细分曲面是首选工作方式，不过 RfB 也支持 Blender 的 Subdivision Surface modifier（细分曲面修改器）。但请注意，对 RenderMan 而言，这是一种效率较低的细分曲面渲染方式，因为 Blender 会比 RenderMan 计算更多面。

使用以 RenderMan 为中心的工作流也能保持 viewport（视口）轻量，因为 SubD（细分曲面）成本只会在渲染时产生，从而释放原本会花在视口上的宝贵 GPU VRAM。

参见 细分曲面（Subdivision Surfaces） 了解更多信息。

也可以使用未定义 normals（法线）的多边形再进行细分；RenderMan 会为你计算 normals。

RenderMan 细分

渲染细分曲面的首选方式是使用 Convert to RenderMan Subdiv（转换为 RenderMan Subdiv） 操作器。

可以在视口的 RenderMan 菜单中找到它。任何选中的 mesh（网格）都会转换为 OpenSubdiv 曲面。

在这个渲染中可以看到，吉他效果器现在正以细分方式渲染。

也可以进入 mesh（网格）的 data properties（数据属性），修改 Subdivision Mesh 下的 Scheme 设置，以开启或关闭 RenderMan 细分曲面。选择 None 会将该网格作为 polygon（多边形）渲染。
