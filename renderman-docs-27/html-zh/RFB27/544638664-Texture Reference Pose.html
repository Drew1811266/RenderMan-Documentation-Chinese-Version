纹理参考姿态

Downloaded: 2026-06-15T03:49:12+00:00

渲染时，通常需要为对象设置一个默认姿势或位置，用作之后渲染的参考，即使对象发生变形或四处移动也一样。

这样可以防止纹理在对象动画过程中看起来在表面上“滑动”或“游动”。这种让纹理粘附在对象上的做法，通常以技术名称 Pref（piː rɛf）来称呼。其他 3D 应用可能称它为“纹理参考”。

Pref 设置

下面是我们设置好的一个基础场景。

在这个场景中，圆柱体被设为骨架系统的子对象。

我们有一台作为纹理投影器的摄像机。为了引用这个摄像机投影器并将流形连接到 PxrTexture 节点，我们把它用作流形或纹理放置。

下面更近地查看这个着色网络。

请注意，我们同时将 PxrTexture 和 PxrProjector 连接到一个 PxrMix 节点。我们使用 PxrProjector 的 resultMask 输出作为 Mixer 值。这样，投影范围之外的任何区域都会由 PxrMix 节点的 Color 1 着色。

我们把纹理投影摄像机的名称填入 PxrProjector 的 Coordinate System（坐标系统）（本例中名为 “TextureProjection”）

纹理没有随圆柱体弯曲，而是在其表面上游动。

Freeze Reference Pose（冻结参考姿态）

为了补偿这一点，需要冻结一个参考姿态。回到圆柱体尚未摆姿势的版本。

在圆柱体的数据属性选项卡中，在 Primitive Variables（基元变量）下，应该能看到 Freeze Reference Pose（冻结参考姿态）按钮。

这会打开如下所示的对话框。

在这里可以选择要冻结的数据。

  • __Pref - 是参考位置点。

  • __WPref - 与 __Pref 相同，但位于世界空间。

  • __Nref - 是参考法线。

  • __WNRef - 与 __Nref 相同，但位于世界空间。

注意，这些就是你会在着色网络中引用的名称。

回到示例场景的着色网络，如果进入 PxrProjector 节点，可以通过将 Use（使用） 改为 Deform（变形）(__Pref + __Nref)。

现在，如果重新为圆柱体摆姿势并渲染，可以看到纹理会正确地随圆柱体弯曲。

如果网格发生变化（例如添加了更多点），需要重新冻结参考姿态。可以回到数据属性选项卡并点击 Re-Freeze Reference Pose（重新冻结参考姿态）。
