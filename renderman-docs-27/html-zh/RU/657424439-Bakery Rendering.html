Bakery 烘焙坊场景渲染流程（Rendering）

在 render settings 中定义 frame range、aspect ratio 和 render camera 后，就该判断哪种 sampling 能在 calculation time 与 image quality 之间提供最佳 compromise。也要记住，RenderMan Machine Learning Denoiser 会帮助减少 sampling noise。

在 Render Settings > RenderMan > AOVs tab 中，我们可以选择 compositing 所需的 settings。对于这个 scene，我创建了五个 main outputs：

  • 美丽通道（Beauty） - 用于 Denoise，并恢复 image 中各 layers 的 alpha。

  • 光路表达式（LPEs） - 包含与 asset materials 相关的 information。

  • 灯光分组（Light Groups） - 分别对应主光 Key light、天空 dome light 和门后 door light 这三类灯光。

  • 数据输出（Data Outputs） - Utility passes，用于在 post 中创建轻微 depth of field。

  • 遮罩选择通道（Cryptomatte） - 帮助选择 assets，以便在 Nuke 中修改。这通过 Sample Filters 实现，而不是通过 AOVs tab。

Optimization 还涉及创建 render layers。对于这张 image，glass windows 需要更多 samples 和 Max Specular Depths，才能让 light 正确 travel 并 sample 它们后方的 assets。因此，我把两个 windows 从 main set 中分离出来，以便 override Max Samples、Pixel Variance 和 Max Specular Depth。

最后，把 volumes 隔离到自己的 “FOG” layer 中，并将 set 制作成 matte object，会让 compositing 中的 integration 和修改更容易。Sampling 应该向下 override，始终以 optimization 为目标。

三层都准备并检查完成后，就可以点击 batch render button 了！

Tip：阅读 XMLs（位于 “project name, renderman, rib”）可以指导 optimization search。
