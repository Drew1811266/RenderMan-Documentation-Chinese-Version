RazorBack Displacement

Downloaded: 2026-06-15T03:49:12+00:00

Lookdev 中第一个关键步骤，是设置从 ZBrush 到 RenderMan 的 displacement output，以最小化 ZBrush 与 RenderMan 之间的 frequency loss。细致的 displacement setup 会将每张 map 的 levels of detail 分类为 primary、secondary 和 tertiary。

每张 map 都以 average gray value 0.5 导出，然后在 RenderMan 的 PxrDispTransform node 中居中，以防止 model distortion。这确保 0.5 的 value 不会发生 displacement。

PxrDispScalarLayer 可以对每张 map 进行 precise control，并根据 PxrDisplace 的 intensity 做调整。

此外，使用低 value 的 PxrBump 会叠加 displacement maps，从而恢复 map exports 期间丢失的一些 volume frequencies。

primary displacement map 从 Kurtis model 的 ZBrush Subdiv 3 导出，在 topology 中保留必要 bake information，以准确转写所有 displacement data。secondary map 从 Mari 导出，通过 color variations 显示 volume 的变化。

最后，tertiary map 使用在 Substance Painter 中创建的 mask 生成，然后导入 Nuke 进行 blending，并创建额外 volume variations。这个 process 包括使用 High-pass filter 增强 map，并利用 expression nodes 引入 dead skin 和 minor breaks 等 small details。
