Texturing and UVs（贴图与 UVs）

对我来说，UV layout 最重要的是保持一致的 texel density，并确保 surfaces 之间有干净的 transitions。我根据四种主要 surface types 来组织 UDIMs，这有助于简化后面的 masking 和 cleanup 阶段。

我以尽量少的 fragmentation 打包 UV islands，确保每个区域都有足够的 texture space。事实证明，这种方法非常高效，既能保留 resolution，又能在 Mari 中绘制时减少视觉杂乱。它让我保持 scene 轻量，同时在 close-up shots 和宽阔视角中都获得足够细节。

贴图制作（Texturing）阶段

我使用 Mari 进行 texturing process，并且很喜欢这个 workflow，尤其是它的 node-based structure 带来的灵活性，帮助我保持所有内容干净有序。由于我在 PxrSurface material 中使用了 Physical Specular mode，因此相应地准备了 texture structure。

我首先把 base color data 投射到 model 上，这些数据是用 cross-polarization techniques 捕获的。通过同时给 camera 和 light source 使用 polarizing filters，我能够消除 surface reflections，从而获得更准确的 color 和更干净的 mask extraction。

projection 之后，我使用 Clone Tool 进行了大量 cleanup，以修正 imperfections 和 surface inconsistencies。在没有 scan data 的区域，我自己创建 textures，以保持 material continuity。

为了隔离 material regions，我依赖 Color to Mask method，它允许我直接从 base color information 生成精确 masks。这支持了一种 layered approach，让我逐步构建 chipped paint、dirt、filler material 和 subtle wear 等 surface detail。

Tip：整个项目中，我通过定义 OCIO environment variables 使用 ACES 1.2 作为 color management system。这有助于在 Mari、RenderMan 以及 production workflow 的每个阶段保持准确且一致的 color。
