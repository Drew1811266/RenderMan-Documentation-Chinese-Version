Lama Metallic Workflow

Source: https://rmanwiki-27.pixar.com/space/RU/606208122/Lama+Metallic+Workflow
Downloaded: 2026-06-15T03:49:12+00:00

无论你使用 online material library 中的 PBR maps，还是使用 Substance Painter 这类 texturing software，RenderMan 都附带了一个方便的 node，名为 PxrMetallicWorkflow，它可以很方便地与 PBR maps 和 Lama workflow 配合使用。

本课会带你完成这个 process。

本课使用的 maps 由我们的 rendering friends 提供，来自 Greyscalegorilla，示例使用的是 Copper Corroded Heavy 01。

由于 Eagle model 没有任何 UVs，我们采用 triplanar approach，通过以下节点把 textures 映射到 model 上： PxrMultiTexture node（节点）。

如果你的 model 有 UVs，可以使用 PxrTexture node，而不是 PxrMultiTexture node。

Note: 目前，从 Substance Painter export Lama material network 时，需要更新 normal connections。你可以在 DCC 的 RenderMan menu 中选择 Update Lama Nodes，轻松完成这一步。
