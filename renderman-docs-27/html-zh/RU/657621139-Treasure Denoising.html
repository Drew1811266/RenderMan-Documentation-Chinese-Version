Treasure 项目的降噪流程（Denoising）

我大量使用了 RenderMan machine learning denoiser。当我需要快速 turn-around 做 test 时，它帮了我很多；它也对缩短 final render times 至关重要，因为 image 不必在 denoiser 清理之前达到 full convergence。

在 Render Elements Node 内，有各种 AOVs 和 LPEs 会被 ingest 到 rendered EXR file 中。这会为 Denoise application 提供执行所需的数据。
