PxrSurface - Metallic Workflow

Source: https://rmanwiki-27.pixar.com/space/RU/619741326/PxrSurface+-+Metallic+Workflow
Downloaded: 2026-06-15T03:49:12+00:00

虽然 Pixar 不使用 PBR Metallic workflow，但无论你使用在线材质库中的 PBR maps，还是使用 Substance Painter 这类 texturing software，它仍然是艺术家中相当常见的工作流。幸运的是，RenderMan 附带了一个方便的节点，叫做 PxrMetallicWorkflow，可以很容易地与 PBR maps 和 PxrSurface 一起使用。

本课将引导你完成这个过程。

本课使用的 maps 由我们的渲染朋友 Greyscalegorilla慷慨提供，所用示例是 Copper Corroded Heavy 01。

由于 Eagle model 没有任何 UVs，我们采用 triplanar 方法，通过 PxrMultiTexture 节点将 textures 映射到模型上。

如果你的模型有 UVs，可以使用 PxrTexture 节点，而不是 PxrMultiTexture 节点。

设置如下：

  1. 创建一个 PxrSurface

  2. 创建一个 PxrMetallicWorkflow node

  3. 将 resultdiffuse 从 PxrMetallicWorkflow 连接到 PxrSurface 的 diffusecolor

  4. 为 diffuse map 使用 PxrTexture（或在本例中使用 PxrMultiTexture node）

  5. 将 diffuse texture 连接到 PxrMetallicWorkflow node 的 basecolor PxrMetallicWorkflow 节点的输入。

  6. 加载 Metallic texture map。

  7. 将 metallic map 中的 resultr 连接到 PxrMetallicWorkflow node 的 metallic PxrMetallicWorkflow 节点的输入。

  8. 将 PxrMetallicWorkflow 的 resultSpecularEdgeRGB 连接到 PxrSurface 的 specularEdgeColor。

  9. 将 PxrMetallicWorkflow 的 resultSpecularFaceRGB 连接到 PxrSurface 的 specularFaceColor。

  10. 加载 Roughness map。

  11. 将 roughness map（粗糙度贴图）中的 resultr 连接到 PxrSurface 的 specularRoughness PxrSurface 的输入。

  12. 使用 PxrNormalMap node 加载你的 Normal map。

  13. 将 PxrNormalMap node 中的 resultN 从 PxrNormalMap 节点连接到 PxrSurface 的 bumpNormal PxrSurface 的输入。

  14. 点击 Render，尽情沐浴在 PBR 的荣耀中！
