Overview（资源概览）

借助这个 Stylized Environment Light Library，你可以在 RenderMan 的 Preset Browser 中使用一些 realistic、psychedelic 和 stylized HDRI。


Enjoy!（祝你使用愉快！）

How-to（安装与使用步骤）

将 Stylized folder 放入你的 RenderMan Asset Library EnvironmentMaps folder。建议将安装默认的 RenderManAssetLibrary folder 转移到非安装目录中、且具有 full writing privileges 的位置。

当你转移 Materials folder 并把 Asset Library 移到其他位置后，必须通过 Maya preferences 告诉 Maya 使用新的 path。

你的 preset browser 会显示所有 new presets，包括我们的新 Stylized Env Lights，无需重启 Maya。你可能需要关闭并重新打开 Preset Browser 来 refresh。
