项目：RenderMan Digital Studio

由 Pixar 艺术家创建： Eugene Riecansky

功能特性：ACES、USD、PxrSurface

这个场景附带多种摄影棚弧形背景。另外，它还包含一个完整的资产库，内有大量受摄影棚启发的资产和套件，由 RenderMan 内部租赁公司 Point Reyes Electric 友情提供，并包含多种模型和着色变体。

你也可以在自己的项目中使用这些资产，而不必一定使用提供的摄影棚场景。

所有内容都已经准备好，方便你在 XPU 中创建摄影棚环境！

Maya（Maya 中的资产安装）

资产包位于 Maya 的 Content Browser 中。

要安装资产，请打开 Content Browser，选择 Favourites 标签页，点击 Add New，并选择 Maya 下载内容中的 Assets 文件夹。所有 Studio 资产随后都会加载到 Content Browser 中。

从这里，你可以把资产直接拖放到场景中。

为了确保所有纹理正确加载，请确认你的项目设置正确。

Solaris / USD / Katana（Solaris、USD 与 Katana 使用方式）

要在 Solaris 中使用，请把下载内容中的 Assets 文件夹添加到 Asset gallery，然后通过 layout 节点使用它们。或者，也可以使用 file 或 assetreference 节点单独加载。

如果你想在 Katana 中使用这些资产，可以通过 USDIN 节点加载它们，并创建你的摄影棚设置。

Blender（Blender 中的资产安装）

要在 Blender 中使用，只需把 Assets 文件夹添加到 Blender Preferences 中的文件路径结构，随后就可以在 Asset Library 面板中看到所有道具。
