Procedural Ground Dirt

Source: https://rmanwiki-27.pixar.com/space/RU/392265765/Procedural+Ground+Dirt
Downloaded: 2026-06-15T03:49:12+00:00

通常，当一个道具被放在地面或架子上，或者积累了带方向性的污垢/旧化痕迹时，道具的下半部分会比上半部分更脏。

想象一个在街角放了很多年的消防栓、一个留在脏地板上的盒子，或者像本例这样，一个展示盒长期放在凌乱的架子上，底部开始比顶部更脏。

当然，你可以使用 Mari 或 Substance Painter 绘制地面污垢遮罩，但在这个例子中，我们想看看如何使用 RenderMan 的着色工具集，以程序化方式实现同样的效果。

以程序化方式制作这个效果的优势在于，如果有许多这样的托盘散布在工作台、地板或展示架上，我们不必为每个托盘实例分别绘制遮罩，而且还能为每个托盘获得着色变化。

让我们看看如何在 Blender 中实现这一点。当然，这个效果也可以在其他 DCCs 中同样实现，但本例会使用 Blender。

目标

我们的目标是创建一个从道具底部开始并向上延伸的黑白渐变。

然后将它作为遮罩，用于叠加污垢漫反射层。

步骤

为了演示效果，这个着色器已通过使用 PxrConstant 节点简化，因此我们可以轻松预览正在创建的遮罩。

实现这个效果的关键在几个节点上。第一个是 PxrRamp，第二个使用 PxrManifold3D 节点。

  • 如果把 PxrRamp 连接到 PxrConstant 的 EmitColor，就会得到这样的结果。

目前它还不算很有用，也不太惊艳，但先等等……

下一步是创建一个 PxrManifold3D 节点，并将它连接到刚才创建的 PxrRamp 节点的 manifold 输入。

  • 接下来，在 PxrRamp 节点中，将 Ramp Type 改为 T Ramp。

为了让这个效果真正可控，我们需要在 Blender 中创建一个坐标系，以便控制渐变的方向。

  • 创建 Empty-> Plain Axis

  • 给 Plain Axis 节点起一个有意义的名字，例如 GroundDirt

  • 在 RenderMan Geometry 下，开启 Export as CoordSys

注意：在 Maya 中，可以使用 3D Placement 节点来创建坐标系

完成后，我们需要告诉 PxrManifold3D 节点：我们希望使用这个 Axis 控制器作为坐标系。

  • 在 PxrManifold3D 节点的 Coordinate System 下，输入你新建的 Axis Node 名称。在本例中，它叫 GroundDirt

现在，我们的渐变看起来有用多了。

我们可以通过调整 ramp 控制点，设置污垢在道具上开始和结束的位置，从而控制这个效果。

还可以通过上下移动 GroundDirt Axis 来进一步精调放置位置。

GroundDirt_MoveAxis.mp4

现在，我们有了一个更有用的地面污垢遮罩……但还可以通过用污垢纹理贴图打破 ramp 的平滑度，把它推得更远。

可以用很多不同方式使用这个纹理贴图来打散 ramp，其中一种方法是使用 PxrBlend 节点，将上方的污垢斑点纹理与地面污垢 ramp 进行混合。

由于我们希望尽可能保持程序化，因此会使用 PxrRoundCube 和 PxrMultiTexture 设置，把污垢斑点纹理以三平面方式投射到道具上。这个方法允许我们把污垢斑点随机偏移、旋转和平铺到其他实例上。

  • 使用 PxrBlend 节点后，我们用 Multiply ink mode，把三平面污垢斑点纹理叠在地面污垢 ramp 上进行混合。你也可以尝试不同的 ink mode，以获得不同结果。

  • 如果需要让遮罩效果更明显，可以再次通过多种方式调整遮罩；在本例中，我们会使用 PxrSplineMap 节点和曲线来调整遮罩。

很好！

现在，我们可以把这个遮罩用作程序化 Lama 或 PxrSurfaceLayer 着色设置的层混合。

请注意，一旦开始在实际着色器中使用这个遮罩，你可能会发现还需要进一步调整遮罩的亮度和对比度，才能调出想要的外观。
