Procedural Ground Dirt

Downloaded: 2026-06-15T03:49:12+00:00

通常，当一个 prop（道具）被放在地面或架子上，或者积累了带方向性的 grunge（污垢/旧化痕迹）时，prop 的下半部分会比上半部分更脏。

想象一个在街角放了很多年的消防栓、一个留在脏地板上的盒子，或者像本例这样，一个展示盒长期放在凌乱的架子上，底部开始比顶部更脏。

当然，你可以使用 Mari 或 Substance Painter 绘制 dirt ground mask，但在这个例子中，我们想看看如何使用 RenderMan 的 shading toolkit，以 procedural（程序化）方式实现同样的效果。

以 procedural 方式制作这个效果的优势在于，如果有许多这样的托盘散布在工作台、地板或展示架上，我们不必为每个 tray instance 分别绘制 mask，而且还能为每个 tray 获得 shading variation。

让我们看看如何在 Blender 中实现这一点。当然，这个效果也可以在其他 DCCs 中同样实现，但本例会使用 Blender。

Goal（目标）

我们的目标是创建一个从 prop 底部开始并向上延伸的黑白 gradient。

然后将它作为 mask，用于叠加 dirt diffuse layer。

Steps（步骤）

为了演示效果，这个 shader 已通过使用 PxrConstant node 简化，因此我们可以轻松预览正在创建的 mask。

实现这个效果的关键在几个 nodes 上。第一个是 PxrRamp，第二个使用 PxrManifold3D node。

下一步是创建一个 PxrManifold3D node，并将它连接到刚才创建的 PxrRamp node 的 manifold input。

  • 接下来，在 PxrRamp node 中，将 Ramp Type 改为 T Ramp。

为了让这个效果真正可控，我们需要在 Blender 中创建一个 coordinate system，以便控制 gradient 的方向。

  • 创建 Empty-> Plain Axis

  • 给 Plain Axis node 起一个有意义的名字，例如 GroundDirt

  • 在 RenderMan Geometry 下，开启 Export as CoordSys

注意：在 Maya 中，可以使用 3D Placement node 来创建 coordinate system（坐标系）

完成后，我们需要告诉 PxrManifold3D node：我们希望使用这个 Axis controller 作为 coordinate system。

  • 在 PxrManifold3D node 的 Coordinate System 下，输入你新建的 Axis Node 名称。在本例中，它叫 GroundDirt

现在，我们的 gradient 看起来有用多了。

我们可以通过调整 ramp knots，设置 dirt 在 prop 上开始和结束的位置，从而控制这个效果。

现在，我们有了一个更有用的 ground-dirt mask……但还可以通过用 dirt texture map 打破 ramp 的平滑度，把它推得更远。

可以用很多不同方式使用这个 texture map 来 breakup 我们的 ramp，其中一种方法是使用 PxrBlend node，将上方的 dirt spots texture 与 ground dirt ramp 进行 Blend。

由于我们希望尽可能保持 procedural，因此会使用 PxrRoundCube & PxrMultiTexture setup，把 dirt spot texture 以 tri-planar 方式投射到 prop 上。这个方法允许我们把 dirt spots 随机 offset、rotate 和 tile 到其他 instances 上。

  • 使用 PxrBlend node 后，我们用 Multiply ink mode，把 tri-planar dirt spot texture 叠在 ground dirt ramp 上进行 Blend。你也可以尝试不同的 ink modes，以获得不同结果。

  • 如果需要让 mask 效果更明显，可以再次通过多种方式调整 mask；在本例中，我们会使用 PxrSplineMap node 和 curves 来调整 mask。

很好！

现在，我们可以把这个 mask 用作 procedural Lama 或 PxrSurfaceLayer shading setup 的 layer mix。

请注意，一旦开始在实际 shader 中使用这个 mask，你可能会发现还需要进一步调整 mask 的 brightness 和 contrast，才能调出想要的 look。
