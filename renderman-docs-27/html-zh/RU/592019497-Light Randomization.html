上图是 Solaris 场景图，用于对我们场景中的所有灯光进行随机化。

所有关键操作都发生在 attributewrangle 节点中，它们控制招牌中每个字母的灯光随机化。场景中的每个字母我都使用一个 wrangle，因为我希望能单独控制每个字母。

我们先看看 meshlight 霓虹管：

字母 D/M/O 的霓虹管

创建一个 attributewrangle 节点，并在 Primitives 下选择我们希望该节点控制的霓虹管 meshlight。

下面是用于设置参数的 VEX 代码。

int off = chi("off");
int on = chi("on");
int transition = chi("transition");
int A = off;
int B = off + transition;
int C = off + transition + on;
int D = off + 2 * transition + on;

float val = 0;
int frame = int(@Frame - $FSTART + chi("frameOffset") * @elemnum) % (D + 1);

if(frame < A)
    val = 0;
else if (frame >= A && frame < B)
    val = 1 - (B - frame) / float(B - A);
else if(frame >= B && frame < C)
    val = 1;
else if (frame >= C && frame < D)
    val = (D - frame) / float(D - C);
val *= ch("intensity");

@inputs:defaultFloat = val;
//@inputs:intensity = val;

现在我们有了一组参数，可用于控制 prim 的随机化。

关闭帧数参数（OFF）

霓虹灯处于关闭状态的帧数。

开启帧数参数（ON）

霓虹灯处于开启状态的帧数。

过渡帧数参数（Transition）

在 OFF 和 ON 状态之间过渡所需的帧数。

帧偏移参数（FrameOffset）：

强度参数（Intensity）

设置并覆盖灯光强度。

字母 E 的球形灯泡

这个 wrangler 允许我们控制在 Globe Bulb USD 资产中设置的两个 primvar 值。

记住，它们名为 Bulb_COL 和 Bulb_INT。

下面是代码

float randomFloat(float min; float  max; string primpath) {
    return fit01(rand(random_shash(primpath), chi("seed")),min, max); 
}

//@inputs:intensity = fit01(rand(random_shash(@primpath), chi("seed")), ch('min'),ch('max')); 
@inputs:defaultFloat = randomFloat(ch('min'),ch('max'), @primpath); 
float randomHue = randomFloat(ch('Hue'), ch('Hue'), @primpath + "hue");
float randomSat = randomFloat(ch('minSat'), ch('maxSat'), @primpath + "sat");
float randomVal = randomFloat(ch('minVal'), ch('maxVal'), @primpath + "val");
v@inputs:defaultColor = hsvtorgb(randomHue,randomSat,randomVal);

，它会创建这一组参数。

强度最小值与最大值（Intensity Min + Max）

将为灯泡强度（Bulb_INT）随机创建的最小和最大强度值。

灯泡色相参数（Bulb Hue）

控制每个灯泡色相（Bulb_COL）的随机化程度。

饱和度最小值与最大值（Saturation Min + Max）

灯泡颜色（Bulb_COL）的最小和最大随机饱和度值。

明度最小值与最大值（Value Min + Max）

灯泡颜色（Bulb_COL）的最小和最大随机明度值。

随机种子参数（Seed）

允许你控制灯泡闪烁速度。

如果保留为单帧，灯泡将不会动画。不过，使用 $FF 表达式会让 seed 每帧改变，从而让灯泡产生动画。

朝向比率参数（Facing Ratio）

允许你控制灯泡的朝向比率。

字母 S 的游乐场灯泡

这个 wrangler 允许我们控制玻璃游乐场灯泡内部小球灯的强度和颜色值。

float randomFloat(float min; float  max; string primpath) {
    return fit01(rand(random_shash(primpath), chi("seed")),min, max); 
}

//@inputs:intensity = fit01(rand(random_shash(@primpath), chi("seed")), ch('min'),ch('max')); 
@inputs:intensity = randomFloat(ch('min'),ch('max'), @primpath); 
float randomHue = randomFloat(ch('Hue'), ch('Hue'), @primpath + "hue");
float randomSat = randomFloat(ch('minSat'), ch('maxSat'), @primpath + "sat");
float randomVal = randomFloat(ch('minVal'), ch('maxVal'), @primpath + "val");
v@inputs:color = hsvtorgb(randomHue,randomSat,randomVal);

总结

那么……总结一下。

通过这一组 wrangle，我们可以让 meshlight 随机闪烁，并控制 Glow 强度以及 PxrSurface 和解析型 USD 灯光的颜色。

如果你的场景需要创建灯光和着色器的随机化，可以自由使用这些代码片段。
