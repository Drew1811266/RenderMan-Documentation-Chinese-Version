上图是 Solaris scene graph，用于对我们场景中的所有 lights 进行 randomization。

所有关键操作都发生在 attributewrangle nodes 中，它们控制招牌中每个字母的 lighting randomization。场景中的每个字母我都使用一个 wrangle，因为我希望能单独控制每个字母。

我们先看看 meshlight Neon tubes：

Letters - D - M - O - Neon Tubes（字母 D/M/O 的霓虹管）

创建一个 attributewrangle node，并在 Primitives 下选择我们希望该 node 控制的 neon tube meshlights。

下面是用于设置 parameters 的 VEX code。

int off = chi("off");
int on = chi("on");
int transition = chi("transition");
int A = off;
int B = off + transition;
int C = off + transition + on;
int D = off + 2 * transition + on;

float val = 0;
int frame = int(@Frame - $FSTART + chi("frameOffset") * @elemnum) % (D + 1);

if(frame < A)
    val = 0;
else if (frame >= A && frame < B)
    val = 1 - (B - frame) / float(B - A);
else if(frame >= B && frame < C)
    val = 1;
else if (frame >= C && frame < D)
    val = (D - frame) / float(D - C);
val *= ch("intensity");

@inputs:defaultFloat = val;
//@inputs:intensity = val;

现在我们有了一组 parameters，可用于控制 prims 的 randomisation。

关闭帧数参数（OFF）

neons 处于 switched off 状态的 frames 数量。

开启帧数参数（ON）

neons 处于 switched on 状态的 frames 数量。

过渡帧数参数（Transition）

在 OFF 和 ON states 之间 transition 所需的 frames 数量。

帧偏移参数（FrameOffset）：

强度参数（Intensity）

设置并 override（覆盖）lights 的 intensity（强度）。

Letter - E - Globe Bulbs（字母 E 的球形灯泡）

这个 wrangler 允许我们控制在 Globe bulb USD asset 中设置的两个 primvars 的 value。

记住，它们名为 Bulb_COL 和 Bulb_INT。

下面是 code

float randomFloat(float min; float  max; string primpath) {
    return fit01(rand(random_shash(primpath), chi("seed")),min, max); 
}

//@inputs:intensity = fit01(rand(random_shash(@primpath), chi("seed")), ch('min'),ch('max')); 
@inputs:defaultFloat = randomFloat(ch('min'),ch('max'), @primpath); 
float randomHue = randomFloat(ch('Hue'), ch('Hue'), @primpath + "hue");
float randomSat = randomFloat(ch('minSat'), ch('maxSat'), @primpath + "sat");
float randomVal = randomFloat(ch('minVal'), ch('maxVal'), @primpath + "val");
v@inputs:defaultColor = hsvtorgb(randomHue,randomSat,randomVal);

，它会创建这一组 parameters。

强度最小值与最大值（Intensity Min + Max）

将为 bulb intensity (Bulb_INT) 随机创建的 min 和 max intensity values。

灯泡色相参数（Bulb Hue）

控制每个 bulb hue (Bulb_COL) 的 randomize 程度。

饱和度最小值与最大值（Saturation Min + Max）

bulb color (Bulb_COL) 的 minimum 和 maximum random saturation values。

明度最小值与最大值（Value Min + Max）

bulb color (Bulb_COL) 的 minimum 和 maximum random values。

随机种子参数（Seed）

允许你控制 bulbs flashing 的 speed。

如果保留为 single frame，bulbs 将不会 animate。不过，使用 $FF expression 会让 seed 每 frame 改变，从而让 bulbs 产生 animation。

朝向比率参数（Facing Ratio）

允许你控制 bulbs 的 facing ratio。

Letter - S - Fairground Bulbs（字母 S 的游乐场灯泡）

这个 wrangler 允许我们控制 glass fairground bulbs 内部 tiny sphere light 的 intensity 和 color values。

float randomFloat(float min; float  max; string primpath) {

v@inputs:color = hsvtorgb(randomHue,randomSat,randomVal);

Summary（总结）

那么……总结一下。

通过这一组 wrangles，我们可以随机闪烁并控制 meshlights、Glow intensity，以及 PxrSurface 和 analytic USD lights 的 color。

如果你的 scenes 需要创建 lights 和 shaders 的 randomisation，可以自由使用这些 code snippets。
