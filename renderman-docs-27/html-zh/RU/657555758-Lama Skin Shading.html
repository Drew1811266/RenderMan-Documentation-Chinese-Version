Mari 贴图制作（Mari Texturing）

第一步是使用 ZBrush 和 ZWrap plugin，从 TexturingXYZ 的 VFace scan 转移 textures。

完成后，将 textures 导入 Mari，并清理 scan 或 manual transfer process 中产生的任何 imperfections。

来自 scan 的 Diffuse、Displacement 和 Utility maps 被用来创建所有必要的 surfacing channels。

贴图制作提示（Texturing Tips）


粗糙度（Roughness）

创建 Roughness channel 时，要记住 oil 会在 pores 中堆积，因此这些区域会比 skin surface 更不 rough。这个差异应该在 Roughness map 中以细微方式表现出来。


清漆层（Clear Coat）

clear coat（这里具体指驱动 shader、用于创建 clear coat 的 mask）需要根据 reference，在 face 的不同部分拥有不同 values。不过通常 face 上的 T-zone 更油，因此 clear coat 需要更高 values。


清漆层粗糙度（Clear Coat Roughness）

与 Clear Coat 类似，Clear Coat Roughness 必须根据 references 调整。pores 内部的 Roughness value 应该高于 surface 上的值。这与 Roughness 使用的理论相反，但它有助于平衡 pores，避免角色看起来过于油亮。如果你的 character 非常 shiny 且 oily，也可以让 pores 更 shiny 来匹配。


DMFP

Diffuse Mean Free Path 是 light 在每个 wavelength 上传播的 distance，以 r、g、b channels 测量。这是一张对 humans 有用的 DMFP value chart，单位为 centimeters。

皮肤着色器（Skin Shader）

skin material 由 4 个主要部分组成：

  • SSS layer

  • primary spec layer

  • clear coat layer

  • displacement 和 Bump to Roughness layers

次表面散射（Sub Surface Scattering）

“LamaSSS” shader 驱动 skin material 的主体部分。

由于 character 有许多部分被 clothing、thick hair 或 armor 覆盖，我为 SSS 制作了一个 mask，用来在两个 Lama bsdfs 之间 mix：一个是用于可见部分的 “LamaSSS”，它是较重的 shader；另一个是用于被覆盖部分的 “LamaDiffuse”，它轻得多。

IOR 设置为 1.4，scale 设置为 1。

这样可以确保 DMFP map 中提供的 values 被尊重，而不会再被 scale 乘上。unit length 设置为 0.1，这会使 SSS calculation 按 cm scene 处理。

shader 使用 “Path-traced exponential” mode，因为虽然它是所有 modes 中计算成本最高的，但也是物理上最准确的。

human skin 是 forward scattering，所以将 anisotropy 设置为 0.9 可以提供 realistic behaviour。Light-scattering anisotropy 通常在 cartilage 或 thin skin 周围更明显，例如 ears。

你可能需要根据目标 photographic reference 调整这个 value。

我们需要设置 SSS shader，让它不要 trace meniscus，以避免 meniscus 与 skin 相交处出现 dark lines。做法是为 meniscus 创建一个 membership，并以减号方式把该 membership 添加到 shader。

我们还需要勾选 “Continuation Rays”。

在这里，也可以为 meniscus 设置 visibility 和 trace parameters，这有助于解决 eye intersection points 周围同样的 dark shadows 与 dark lines 问题。

主高光层（Primary specular）

primary specular layer 由一个 “LamaDielectric” shader 构成，它设置为 physical mode，IOR 为 1.45。roughness map 会输入其中，normals 来自 Bump to Roughness node，我们稍后会介绍这个节点。


清漆层（Clear Coat）

与 Primary specular layer 很相似，coat 使用一个 IOR 为 1.37 的 “LamaDielectric” shader。coat 不由 Bump to Roughness node 驱动，也不由任何 normals node 驱动；它只直接接收 Coat Roughness channel。


分层位移（Layered Displacement）

displacement 是大部分 skin detail 的来源。setup 相当基础。我使用从 TexturingXYZ 转移 scan 得到的 RGB displacement map，并调整每个 channel 的 values（分别代表 primary、secondary 和 tertiary displacement details），使它们匹配 reference。

这一步在 “PxrDispScalarLayer” 中完成。我还将 “MIP Bias” 设置为 -4，以确保在所有 distances 上都选择更高 resolution 的 MIP。

modeling 阶段还从 ZBrush 中提取了一个 detail displacement pass。我使用 groom 中的 follicle mask，通过 “PxrBlend” 给这个 ZDisp pass 添加 concave bump，然后把整套 setup 与 TexturingXYZ displacement 叠加，创建最终的 displacement setup。

凹凸转粗糙度（Bump to Roughness）

我使用 Bump to Roughness（BTR）来驱动 skin 中细小 pore bump details。这样即使通常由 bump shader 生成的 bump details 会因 aliasing 在远处被压掉，也能从远处保持 details 和 fidelity。

BTR node 会把 bump details 注入 roughness 和 anisotropy channels，以便当 bump 从远处不再可见时，仍然维持相同 look。

我在 advanced parameters 中把 filter scale 设置为 0.7，以确保 texture mips 中保留非常高的 detail level。

你可以把自己的 roughness map 接入 “Base Roughness” port 来 feed BTR node。这就是 BTR node 在完全 zoom into detail 时使用的 roughness，因此也是你最高 accuracy 的层级。

为了得到想要的 look，我必须 invert bump normal，并大幅提高 “Bump Normal Gain”。
