Overview（资源概览）

这是一个面向 RenderMan 的 production-ready lights library，包含 19 个 IES profiles。

IES profiles 可以帮助你获得 physical photometric data distribution。换句话说，它会基于原始 manufacturer specifications 提供 realistic light emission pattern，为你的 art 增添更多 realism。

关于 IES profiles 的更多信息，请参见 CGArena。

How-to（安装与使用步骤）

要安装到 RenderMan 的 Preset Browser 中，请将 IES folder 放入你的 RenderMan Asset Library 的 LightRigs folder。建议将安装默认的 RenderManAssetLibrary folder 转移到非安装目录中、且具有 full writing privileges 的位置。

当你转移 Materials folder 并把 Asset Library 移到其他位置后，必须通过 Maya preferences 告诉 Maya 使用新的 path。

或者，你可以像这样在 Maya.env file 中创建新的 environment variable：

RMAN_ASSET_LIBRARY = /My/Folder/Example/RenderManAssetLibrary

这会确保你始终指向正确的 folder。
