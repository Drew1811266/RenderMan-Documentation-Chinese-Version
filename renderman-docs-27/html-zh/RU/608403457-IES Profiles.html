概览

这是一个面向 RenderMan、可直接用于制作的灯光资源库，包含 19 个 IES 配光文件。

IES 配光文件可以帮助你获得符合物理规律的光度数据分布。换句话说，它会基于原始厂商规格提供逼真的灯光发射图案，为你的作品增添更多真实感。

关于 IES 配光文件的更多信息，请参见 CGArena。

操作步骤

要安装到 RenderMan 的 Preset Browser 中，请将 IES 文件夹放入你的 RenderMan Asset Library 的 LightRigs 文件夹。建议将安装时默认的 RenderManAssetLibrary 文件夹转移到非安装目录中，并放在具有完整写入权限的位置。

当你转移 Materials 文件夹并把 Asset Library 移到其他位置后，必须通过 Maya 偏好设置告诉 Maya 使用新的路径。

或者，你可以像这样在 Maya.env 文件中创建新的环境变量：

RMAN_ASSET_LIBRARY = /My/Folder/Example/RenderManAssetLibrary

这会确保你始终指向正确的文件夹。
