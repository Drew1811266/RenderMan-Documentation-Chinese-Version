在 Mari 中进行贴图制作（Texturing in Mari）

方法是使用 Substance Painter bake utility masks，这些 masks 会以 mask form 作为各种 maps 的 drivers。bake 完所有必要 maps 后，将它们连同来自 ZBrush 的 displacement 和 normal output maps 一起导入 Mari template。

每个 mask 都通过 teleport receiver 驱动，并且必须保留 scene 中 node paints 和 masks 的数量。

这意味着用于 color 的 mask 也可以驱动 displacement 或 roughness，为 workflow 提供灵活性和效率。

下一步是创建 ISO maps，它们在 Mari tree 中作为 masks 使用。生成并绘制完所有 ISO 和 bake maps 之后，就可以自由 experiment、modify 和 blend 这些 maps，以获得期望结果。

每个 mask 都会引入 color、hue 和 saturation 的 variations，从而产生自然、真实的外观。会调整 asset 的 hue/saturation/value（HSV）数值，以确保真实性。此外，每个 scale 都具有独特特征，creature 的姿态等因素会影响 wear and tear。

例如，creature 的 belly 可能因为频繁接触表面而显示更多 damage，而 tail 和 spikes 可能会因 combat 或 environmental interactions 出现磨损。

处理 shader previews 时，建议主要关注 displacement channel，通过把 preview 限制在单一 channel（例如 albedo）来尽量减少干扰。这种方式有助于保持清晰，避免可能影响决策的混乱。不过，仍应定期检查以确保一切正常。

使用 teleport receiver 和 bake points 可以精简 workflow，使调整能够减轻 scene 负担并提升 viewport 的响应性。这种 optimization 减少了花在 selections 上的时间，从而提高推进效率。

当所有 maps 和 ISOs 都同步后，建议按照 modeling section 中描述的 UV symmetry，将 model 左侧复制到右侧。

随后，通过 triplanar projecting rock、noise 和 oil maps 引入 procedural asymmetry，可以增强 ISO、color 和 bake maps 上的 variation，形成更 organic 的外观。
