procedural shading 中两个基础且常用的 geometry patterns 是 Ambient Occlusion 和 Curvature。RenderMan 可以渲染这两种特性；它们既可以用于 procedural shading network，也可以 bake 出来，再由 PxrTexture 节点重新读取。

PxrDirt（Ambient Occlusion）

该 PxrDirt 节点会渲染局部几何遮挡。

这很适合把 dirt、rust 和 dust 等 shading effects 加到模型的凹槽、缝隙和角落中。

PxrCurvature

该 PxrCurvature 节点是一种实用而直接的方法，可创建 edge effects 和基于 curvature 的结果，例如塑料和与其他 surface 接触的物体上的 edge wear、edge breakup 与 edge variation。

这两个节点的一个实际例子，是把 PxrCurvature 节点用作两个 layers 之间的 layer mask，例如 Midnight Rebel 场景中 base metal 与上方 paint layer 之间的遮罩。

随后可以使用 PxrDirt 节点作为 layer mask，在 paint layer 上添加 dust。
