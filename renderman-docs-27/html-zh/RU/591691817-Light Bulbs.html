这个场景使用三种不同类型的 light。

我们有一个 globe light bulb、一个 fairground light bulb，以及 neon signs 本身。

我会在本课中展示这两种 lightbulbs 的制作方式。如果你想进一步了解如何用 Mesh Lights 制作 Neons，有一节完整课程在 这里

游乐场灯泡（Fairground Bulb）

这个 bulb 的设置相当简单。glass shader 已应用到 bulb geometry 上，并且 refraction color 设置为蓝色。稍后会随机化这个 refraction color，为每个 instanced bulbs 增加颜色变化。

在 geometry 内部，有一个很小的 sphere light，radius 设置为 0.002，intensity 和 exposure 都设置为 10。

同样，intensity 和 exposure 这些 light values 稍后会被随机化，用来创建 light bulbs 的闪烁和追逐效果；但目前这是一个很好的起始设置。

球形灯泡设置（Globe Bulb）

globe bulb 比 fairground bulb 稍微复杂一些。

下面是分配给 bulb glass geometry 的整体 shader。

我们来分解一下。

在 shader graph 的上半部分，有一个简单的 triplanar grunge texture setup，叠加在一个 PxrFractal 节点上方，然后用作 specular roughness。这会为玻璃反射增加细微的不完美。

bulb shader graph 的下半部分才是真正产生魔法的地方。

这里有两个 PxrPrimVar 节点。一个名为 Bulb_COL，另一个名为 Bulb_INT

Bulb_COL

这个 primvar 设置并控制 bulb 的基础 glow color。

我们还使用这个 primvar 驱动 glass refraction color

开始为 bulbs 制作动画时，这个 primvar value 将被随机化

Bulb_INT

这个 primvar 设置并控制 bulb glow gain 的初始 intensity。

开始为 bulbs 制作动画时，这个 primvar value 将被随机化

如果我们取 Bulb_COL primvar 的 RGB output，并将它 blend 到一个略微降低饱和度的 Bulb_COL color 上，就可以用一个 pxrfacingratio 节点来驱动 blending。这样就能像下图那样，对 bulb 的 emissive look 进行创意控制。
