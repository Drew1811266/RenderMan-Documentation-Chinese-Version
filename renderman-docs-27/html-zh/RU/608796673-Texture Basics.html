在 CG 中，纹理贴图的使用非常普遍，并且在资产或场景的着色工作流中发挥关键作用。

Loading Textures（纹理加载与贴图读取流程）

在 RenderMan 中，加载纹理贴图主要依靠 PxrTexture 这一节点。虽然还有其他几个节点也能加载纹理，但 PxrTexture 节点是最主要、最适合首先使用的节点。

在继续之前，有一点很重要：RenderMan 需要把你的纹理转换为它的原生纹理格式： .tex

当你把 .png、.jpg、exr、.tif 等文件加载并应用到 shading network 中时，RenderMan 会根据规则自动完成这一步。简单来说，它会替你处理所有转换工作。

Using Painted Texture Maps（使用绘制纹理贴图）

下面拆解几种纹理贴图的常见用途：

使用纹理贴图的一种方法，是把已经展开 UV 的模型导入 Mari 或 Substance Painter 这类纹理绘制软件。

Substance Painter 对纹理贴图使用 PBR Metallic 工作流，因此你会导出若干可直接使用的纹理，例如：

  • Diffuse（漫反射）

  • Metallic（金属度）

  • Roughness（粗糙度）

  • Normal（法线）

  • Height（高度）

这些贴图可以从 Painter 导出后在你的 DCC 中分配；也可以使用面向 Substance Painter 的 RenderMan Preset Browser 插件，一键完成导出、转换为 .tex，并创建材质网络。

预设创建完成后，就可以在你选择的 DCC 中把这个 RenderMan 着色器应用到对象上。非常快速，也非常简单。

在 Mari 中，你可以在两种 shading system 之间选择。

你可以采用类似 Substance Painter 的 PBR 工作流，也可以使用随 RenderMan 提供的 Mari 版 PxrSurface 材质，并直接绘制原生 PxrSurface lobes。

同样，你可以从 Mari 导出绘制好的贴图，并直接在 RenderMan 中使用；不过目前我们还没有在 Mari 中提供 Preset Browser。

如需了解在 Mari 中使用 PxrSurface 材质的更多信息， Peter Aversten（MeshMen） 提供了大量关于这一主题的优秀教程。

Using Layer Masks（使用层遮罩）

使用 layer masks 是设置道具和资产时推荐的工作流，并能带来许多创作优势。其中一个优势是，你可以轻松重新绘制 layer masks，而不必从纹理软件重新导出整个着色器；只需重新导出调整后的 layer mask，并重新加载纹理。

Layer masks 可以在 Painter/Mari 中绘制，也可以通过程序方式生成黑白遮罩，用来决定材质在资产上的显示位置以及如何分层。

Diffuse 等 lobes 可以通过程序方式生成；你可以使用 PxrSurfaceLayer 或 Lama shading workflows，这两种方式都会用遮罩驱动材质各层之间的视觉过渡。
