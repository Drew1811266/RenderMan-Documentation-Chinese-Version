How-to（安装与使用步骤）

这个 resource 设计为与 RenderMan 的 Preset Browser配合使用。

将 folder 中的内容拖入你的 Asset Library > EnvironmentMaps folder。文件按 category 组织，可以在 preset browser 中轻松搜索。

请确保将安装目录中的 RenderManAssetLibrary folder 转移到具有 full writing privileges 的位置。

当你转移 Materials folder 并把 Asset Library 移到其他位置后，重要的是通过 Maya preferences 告诉 Maya 使用新的 path。

或者，你可以在 Maya.env file 中创建新的 environment variable，以确保始终指向正确的 folder：

RMAN_ASSET_LIBRARY = /My/Folder/Example/RenderManAssetLibrary
