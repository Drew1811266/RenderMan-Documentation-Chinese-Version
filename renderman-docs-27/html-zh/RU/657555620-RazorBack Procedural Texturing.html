RazorBack 程序化贴图流程（Texturing Procedural）

为了引入最大 variation，将 ISO maps 和 displacement/utility maps 与 rock、granite maps 结合，并使用 triplanar projection，是非常关键的。

这种 approach 显著增强 detail，并让 Mari node 内创建的 layers 之间能够平滑 transition。

在 combining scanned map projection 和 procedural nodes 的 procedural pass 之后，会进行 final texturing pass，以加入大量 micro-detail variations。

Substance Designer 和 Substance Painter 用于调制 scales 上的 HSV variations，并引入 dead skin 的 subtle details，这些 details 会策略性地放置在 scales 的特定末端。

在整个过程中，会考虑 creature 的 movement 和 environment，并参考 texturing materials 来辅助 decision。

最后一步是增强 scales edges 上特定的 specular highlights，同时增加 albedo detail。mask creation 完成后，重点会回到 Mari 进行 texture publishing，标志着进入 Lookdev stage。
