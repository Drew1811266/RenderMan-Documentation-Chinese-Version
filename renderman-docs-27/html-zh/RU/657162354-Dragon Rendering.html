渲染设置阶段（Rendering）

Houdini 允许用户拥有不同 ROP nodes，这对于即时使用不同 render settings 非常有帮助。在这个 project 中，我创建了一个 preview ROP node，它使用较低 resolution、无 DOF、较小 bucket size，以及更粗的 pixel variance。final rendering ROP 则使用 high resolution，加上 DOF 和更精细的 pixel variance。

Tip 1：较小 image resolution 有助于减少 render time，但请记住你会看到更少 displacement details。

Tip 2：Pixel variance 是最重要的 sampling tool，因为它告诉 RenderMan 应该把 rendering power 集中在哪里。
