Treasure 项目的环境制作

得益于镜头的布局方式，我只使用一根在 Houdini 中程序化构建的柱子，就完成了一个范围可控的工作流，并获得了不错的外观。

我通过柱子上的 Voronoi 断裂来处理几何程序化，并结合点级处理，让边缘产生不规则起伏，从而形成磨损大理石柱的美感。

为了继续发挥程序化的优势，我使用 RenderMan Pattern 节点为柱子着色。通过使用 PxrRoundCube，我能够在 Katana 中实现可平铺的纹理制作技术；这很有用，因为我可以生成可信的纹理，而不必进入绘制应用程序另行制作新纹理。

龙穴承载着一段史诗故事，因此传达年代感至关重要。使用 PxrDirt 是表现磨损效果的好方法，它会在裂缝中加入额外污垢和灰尘堆积。保持柱子几何碎块彼此分离，也有助于以可信方式隔离这种效果，最终让它获得更自然的外观。
