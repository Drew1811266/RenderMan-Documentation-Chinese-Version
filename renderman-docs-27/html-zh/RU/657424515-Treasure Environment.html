Treasure 项目的环境制作（The Environment）

得益于 shot 的 layout 方式，我成功完成了一个 contained workflow，只用一根在 Houdini 中 procedural 构建的 column，就达到不错的 look。

我使用 column 上的 voronoi fractures 处理 geometric proceduralism，并结合 point wrangling，让 edges 产生 wobble，从而形成 worn out marble column 的 aesthetic。

为了继续发挥 procedural 的优势，我使用 RenderMan patterns 为 columns 做 shading。通过使用 PxrRoundCube，我能够在 Katana 中实现 tileable texturing techniques；这很棒，因为我可以生成可信的 textures，而不必进入 painting application 生成新的 textures。

dragon’s lair 有一段 epic story 要讲述，因此传达 aging 至关重要。使用 PxrDirt 是表现 worn out effect 的好方法，它会在 cracks 中加入额外 grime 和 dust accumulation。保持 column geometry pieces 分离，也有助于以可信方式 isolate 这种 effect，最终让它获得更 natural 的 look。
