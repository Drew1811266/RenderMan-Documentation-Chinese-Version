Stirling 项目的色彩管理流程（Color Management）

继续向前之前，我们需要确保 scene 已正确设置。可以在 scene 的 Render Settings 中启用 ACES 设置。如果 RenderMan 已正确安装，它应该会自动从 RenderMan 安装目录获取 OCIO config file；在环境变量术语中，这个目录也称为 RMANTREE。

此外，也要打开 IT viewer 的 Preferences 窗口，确认它同样以 ACES 显示。这样可以在整个 workflow 中保证色彩准确性。在所有 OCIO color profiles 中，适合大多数 CG 工作的正确 color profile 通常是 acesCG。
