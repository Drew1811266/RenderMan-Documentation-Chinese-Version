RazorBack 项目的渲染与输出设置

设置渲染图层和 AOVs 时，关键是为每盏主灯输出一个 LPE（Light Path Expression，光路表达式）。也可以把功能相似的灯光组合在一起，例如轮廓光。

不过，过度分组可能会限制合成中的灵活性。

此外，将漫反射分量与镜面分量分离，可以微调并修正着色和照明之间的任何差异。

在一切都完美校准的理想情况下，也许不需要以这种方式分离 AOVs。不过，当某盏灯光强度过高、导致镜面效果被夸大时，分离 AOVs 就会非常有价值。

这样可以在后期制作中进行有针对性的调整，缓解问题并获得期望的视觉结果。

为了简化降噪流程并方便创建遮罩，建议将 Beauty 通道与 LPEs（光路表达式）和辅助通道分离。

这种分离让每个 AOV 都更容易单独降噪。此外，按材质或对象提取 Cryptomatte 可以快速创建遮罩；即使资产在运动，遮罩也会保持一致。

通过将 Beauty 通道与 LPEs 和辅助通道分离，降噪会更高效，因为每个组成部分都可以独立处理。

此外，使用 Cryptomatte 创建遮罩能提供极高灵活性，可以对场景内的元素进行精确控制，不受资产运动影响。这种方法提升了整体后期制作工作流，让最终渲染结果更顺畅、更精确。
