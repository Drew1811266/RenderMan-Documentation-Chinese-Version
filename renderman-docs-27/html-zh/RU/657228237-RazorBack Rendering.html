RazorBack 项目的渲染与输出设置（Rendering）

设置 render layers 和 AOVs 时，关键是为每个 main light 输出一个 Light Path Expression（LPE）。也可以把功能相似的 lights 组合在一起，例如 rim lights。

不过，过度 grouping 可能会限制 compositing 中的灵活性。

此外，将 diffusers 与 speculars 分离，可以 fine-tune 并修正 shading 和 lighting 之间的任何 discrepancies。

在一切都完美 calibration 的理想情况下，也许不需要以这种方式分离 AOVs。不过，当某个 light 的 intensity 过强、导致 specular effect 被夸大时，分离 AOVs 就会非常有价值。

这样可以在 post-production 中进行有针对性的 adjustments，缓解问题并获得期望的 visual outcomes。

为了简化 denoising process 并方便 mask creation，建议将 beauty pass 与 Light Path Expressions（LPEs）和 utilities 分离。

这种 separation 让每个 AOV 都更容易单独 denoise。此外，按 material 或 object 提取 cryptomatte 可以快速创建 masks；即使 assets 在运动，masks 也会保持一致。

通过将 beauty pass 与 LPEs 和 utilities 分离，denoising 会更高效，因为每个 component 都可以独立处理。

此外，使用 Cryptomatte 创建 masks 提供了极高的灵活性，可以对 scene 内的 elements 进行精确控制，不受 asset movement 影响。这种方法提升了整体 post-production workflow，让 final renders 更顺畅、更精确。
