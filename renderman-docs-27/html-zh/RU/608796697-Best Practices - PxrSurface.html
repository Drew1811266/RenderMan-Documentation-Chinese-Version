理解 PxrSurface 的基础设计。

在 RenderMan 中，用于制作对象、道具和角色的主要材质就是“ PxrSurface ”这一材质。

这是一个可用于各种 shading 任务的单一材质，因此也称为“monolithic”（整体式）材质。材质中的每个部分都代表该材质的一个特定 lobe。

这些 lobes 的一些例子包括：

  • Diffuse（漫反射分量）

  • Specular（镜面反射分量）

  • Rough Specular（粗糙镜面反射分量）

  • Sub-Surface（次表面散射分量）

  • 等等。

每个 lobe 都有自己的一组控制项，用来定义 look，以及它们如何通过 gain 或 weighting 与其他 lobes 组合。

为什么它被称为“lobe”？

你可以根据材质的分布函数来想象它如何反射或透射光线；这些分布通常在 2D 或 3D 中可视化为一个 lobe（用拉长的形状表现锐度、强度和方向）。你可以在 Wikipedia上了解更多。
