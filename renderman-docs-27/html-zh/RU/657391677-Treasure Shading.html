Treasure 宝藏材质的着色流程（Shading Treasure）

gold treasure material 是一个非常简单的 shader。

一个 Voronoi procedural 会驱动 reflective roughness attributes，从而提供它的 textural detail。

这个 pattern setup 使用 large-scale Voronoi noise，并连接到 roughness gain component；这提供了丰富的 textural quality，为 treasure 增添肮脏、古老的感觉。
