RenderMan Learning Paths（RenderMan 学习路径）

无论你是刚接触 RenderMan 的 3D 艺术家，正在提升技能的创作者，还是经验丰富的行业老手，这些学习路径都能提供适合你的内容。

选择一条学习路径，开启一段兼具教育性与创造性的旅程！

着色

如果你想提升着色器、纹理制作和 lookdev 水平，请选择这条路径。

灯光

RenderMan 提供一整套基于物理的解析灯光。这条路径会带你开始学习灯光和布光。

Camera（摄影机）

通过这条学习路径，为你的摄影机增加个性、情绪和深度。

VFX

如果你热衷于 Visual Effects，这条路径就是为你准备的！

渲染

这条学习路径会帮助你踏上成为 Rendering Rockstar 的旅程！
