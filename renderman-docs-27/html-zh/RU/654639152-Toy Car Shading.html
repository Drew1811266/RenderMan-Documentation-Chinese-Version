Shading（着色）

我尽量让 shading process 保持最小化，把在 Mari 中创建的 texture maps 直接连接到 RenderMan 的 PxrSurface shader。除了一个用于微调 roughness response 的 basic remap node，没有构建复杂的 node networks。

这种方法效果很好，因为我在 Mari 的 texturing phase 也使用了基于 PxrSurface 的 setup，这帮助我在 RenderMan 中获得一致结果。
为了确保 physical accuracy，我为每种 material 应用了正确的 IOR 和 Extinction Coefficient values。

由于所有 maps 都是在这些 values 的前提下准备的，因此在 RenderMan 侧不需要进一步调整。
