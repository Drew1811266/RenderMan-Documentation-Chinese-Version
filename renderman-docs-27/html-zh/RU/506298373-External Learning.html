External Learning Resources（外部学习资源）

我们的 RenderMan community 与众不同；在这里你会找到一些实用的 external learning 和 training resources，可以直接开始深入学习！

Pixar Graphics

Pixar Research 在 computer graphics 和 animated films 方面拥有悠久的技术创新与开创性工作历史。

fxphd

fxphd 的朋友们提供了一套出色的 RenderMan courses，由 professionals 设计，也面向 professionals。

The NotePad Show

The Notepad Show 是学习 RenderMan 及其与你喜爱的 DCCs 集成的方方面面的首选去处。

MeshMen Studio

由 Peter Aversten 制作的 VFX、Texturing 和 Look Development tutorials and projects；他的格言是帮助 artists，一次一个 Pixel。
