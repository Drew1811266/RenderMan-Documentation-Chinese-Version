宝藏项目中的深度数据合成流程（Deep Data）

这个 project 也是一次学习 deep rendering 的机会，因为这是我以前的 projects 中从未接触过的一个方面……结果发现处理起来非常有趣。

要在 RfK 中实现这一点，我们只需要把 file type 改为 deep，RenderMan for Katana 就会自动输出 deep EXR format。

Nuke 中的 deep data 让我们能够对非常细微的 detail 做到 pixel perfect compositing。而且在 Nuke 中浏览它也相当有趣。

虽然 Deep compositing 对我来说是一个需要学习的新概念，但我发现它是很吸引人的 concept 和 toolset，尤其适合在 compositing 阶段加入 2D volumetrics 来模拟 distance fog。

最后，这里展示的是我在 NukeX 中的 comp treatment 和 final scene graph。你需要在本地 render frames，但希望整个 project 能让你看到一个高效 pipeline 的有益示例，并从中获得 ideas，复制到自己的 personal projects 中。
