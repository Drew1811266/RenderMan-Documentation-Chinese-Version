Basic Environment（基础环境）

我使用的 main environment light 是一张应用到 PxrDomeLight 上的 ramp texture。这提供了初始的 night sky fill light。

在 Photoshop 中把 ramp 制作为 32-bit EXR file，是完成这件事的最佳方式。你也可以向 ramp 添加额外的 moon 或 sunlight source。如果你选择不在 ramp texture 本身加入强 light source，也可以在 RenderMan 中使用 PxrDistantLight、PxrRectLight 或 PxrDiscLight 来复现 moon 或 sun lighting。

选择实际上取决于你想匹配的 mood、palette 和 lighting references。基于 lighting 或 mood reference 创建 ramp，是制作 initial ramp 的好方法。

UFO Beams（UFO 飞船牵引光束效果设置）

制作 UFO beams 时，有两个 essential factors（关键因素）。

第一个是 light beam 本身；为此我使用了 PxrDiscLight。

对 beams 来说，第二个非常重要的因素是 environmental fog volume。没有这个 volume，你会看到发出的 light，但看不到任何 beams。我在 Houdini 中创建了一个 volume VDB，并把它作为 volume 使用，同时连接了 PxrVolume shader。把 Primary Anisotropy 调整到高于 0 的值，也会改善 beams 的外观。我使用的值是 0.7。

这两个 elements 彼此依赖，才能得到强烈的 UFO beam（实际上任何其他 beams 也是如此）。

在上图中可以看到，我一开始使用了三束 beams（按照我参考的 reference），但在 final image 中，我决定只使用一束。

因此，main UFO beam light 本身是一个 PxrDiscLight。

我想要 sharp beams of “rays”，所以把 light 的 size 做得相对较小。我必须在 light size 上取得平衡：既要让 beams 有足够 sharpness，又要让 beam 足够大，让上方的 UFO 感觉很巨大。

我给 light 添加了一个 IES profile，用来增加额外 shape。

然后我使用 PxrBarnFilter 控制 light 的 cone shape，并使用 PxrRodFilter 阻止 light 向 scene 中 spill 得太远。我只想要一束不会扩散太远的 beam。

Tip：你也可以通过 PxrCookieLightFilter 给 beams 添加 cookie image，以增加更多 visual shape，尤其是在 beams 照射到某些 geometry 上时。

Headlights / Search Lights（车灯 / 搜索灯）

headlights 由三种 lights 组成。

第一种是一个 tiny PxrSphereLight，作为 headlight housing 内部的 bulb source。

第二种是一个 PxrDiscLight，用来创建 main headlight beams。我给 headlight 添加了 IES profile 以获得更多 shaping，并用 PxrBarnDoorFilter 进一步引导 beams。

为了完成 headlight glass 本身的效果，我把它们转换成 intensity 非常低的 Mesh Lights。

有了这两个 main lights 和 mesh light 后，我就可以对 headlights 进行 lookdev，创建我想要的 look。
