每个人都知道黑色和白色在渲染中意味着什么，对吧？

当然！

嗯……以防万一，这里有一份快速指南，说明 RenderMan 在粗糙度、金属度和凹凸中如何处理黑白数值。

材质粗糙度

黑色 = 光亮表面

粗糙度值：1

白色 = 粗糙表面

粗糙度值：0

材质金属度工作流

黑色 = 非金属

粗糙度值：1

白色 = 金属

粗糙度值：0

凹凸贴图

黑色 = 向内凹凸

白色 = 向外凹凸

置换贴图

黑色 = 向内置换

白色 = 向外置换

Presence（Opacity，不透明度）

黑色 = OFF（关闭）

白色 = ON（开启）
