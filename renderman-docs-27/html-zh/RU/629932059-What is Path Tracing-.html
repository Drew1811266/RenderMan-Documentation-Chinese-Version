What is Path Tracing?

Source: https://rmanwiki-27.pixar.com/space/RU/629932059/What+is+Path+Tracing%3F
Downloaded: 2026-06-15T03:49:12+00:00

Path tracing（路径追踪）是一种生成 digital images 的方法，它模拟 light 在 virtual world 中如何与 objects 交互。

通过向 scene 中发射 rays（line segments）并跟踪它们在 objects 之间的 bounce，来追踪 light 的路径。

Path tracing 的名称来自计算 light 从 light source 到 camera 的完整路径。

Light 可能会在 virtual scene 内的许多 objects 之间 bounce。当一条 light ray 命中 surface 时，它会 bounce 并创建新的 light rays。因此，一条 path 可以由多条 rays 组成。

通过收集一条 path 上的所有 rays，可以计算 light source 以及 path 沿途 surfaces 的贡献。这些计算用于生成最终 image。
