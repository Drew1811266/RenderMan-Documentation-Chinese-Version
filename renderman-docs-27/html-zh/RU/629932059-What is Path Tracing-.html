What is Path Tracing?

Source: https://rmanwiki-27.pixar.com/space/RU/629932059/What+is+Path+Tracing%3F
Downloaded: 2026-06-15T03:49:12+00:00

路径追踪（Path tracing）是一种生成数字图像的方法，它模拟光线在虚拟世界中如何与物体交互。

通过向场景中发射射线（线段）并跟踪它们在物体之间的反弹，来追踪光的路径。

路径追踪的名称来自计算光从光源到相机的完整路径。

光线可能会在虚拟场景内的许多物体之间反弹。当一条光线命中表面时，它会反弹并产生新的光线。因此，一条路径可以由多条射线组成。

通过收集一条路径上的所有射线，可以计算光源以及路径沿途各个表面的贡献。这些计算用于生成最终图像。
