Creating a Wooden Floor

Source: https://rmanwiki-27.pixar.com/space/RU/393805842/Creating+a+Wooden+Floor
Downloaded: 2026-06-15T03:49:12+00:00

在本课中，我们将学习如何利用 PxrRandomTextureManifold 的能力，把一组 10 张地板木板纹理随机分配给几何体，让每块木板都有自己独特的随机翻转、方向和位置偏移。

虽然本课展示的是如何重新创建地板木板，但这项技术也可以用于许多其他创作用途，例如墙砖、石块、道路，以及其他需要在多种对象上随机放置纹理的场景。

初始设置

先来看一下 Maya 中的设置。我们有一组独立的地板木板，每块都使用相同的 UV tile。

接着看一下我们要使用的纹理。这些素材由 Texture Supply 的朋友们慷慨提供。在这个示例中，我们使用了他们精美的 5k Brown Ash 40.8x6.8cm 纹理包。

现在开始把这些纹理指定给我们的地板木板。

如果一开始只是在基础 PxrSurface 上为 Diffuse color 添加一个 PxrTexture 节点，可以从下图看到，结果一点也不随机。这不是我们想要的效果：纹理在所有地板木板上的位置随机性不足，而且看起来都非常相似。

这里的技巧是结合使用 PxrMultiTexture 节点来载入一组 10 张纹理，然后用 PxrRandomTextureManifold 对每张纹理和 UV placement 的随机性进行美术指导。

如上面的渲染所示，我们的 10 张地板木板纹理现在已经更随机地分布了。

但我们还没有完成...现在可以继续调整 Randomize 参数，让随机性更进一步。这些设置完全用于塑造你想要的观感，因此这里没有绝对正确或错误的数值。

PxrRandomTextureManifold 内部有几个关键参数

Number of Textures（纹理数量）： 这会决定从 PxrMultiTexture 文件列表中使用多少张纹理

Texture Order（纹理顺序）： 这会控制 PxrMultiTexture 文件列表中的纹理如何被打乱顺序

Random Source（随机来源）： 这会控制 PxrRandomTextureManifold 使用哪个 geometry ID

进一步推进随机化

现在漫反射颜色已经呈现出我们想要的效果，可以进一步提升地板木板着色器的真实感。

让我们看一下最终渲染中使用的最终着色器。这里添加了几个额外的节点

  • 漫反射纹理也被连接到 PxrBumpMixer，再连接到 diffuse bump，用来给漫反射层添加轻微的 bump。

  • 漫反射纹理还被用来驱动 Primary Specular。

它们开始看起来不错了，但颜色仍然都一样。现实中，每块板都会有略微不同的颜色：可能来自制造过程差异，也可能有些随着时间褪色，或者变得更脏、更积灰。

幸运的是，PxrMultiTexture 节点内部还有更多随机化参数。

这里我们正在随机化 10 张纹理的 Hue value。如你所见，每块地板木板现在都有了轻微的色相偏移。

把 Saturation mode 设置为 Subtractive，意味着每张纹理都会得到一个低于其原始色相的随机 Hue value。同样，具体设置成多少是创作选择；这里我们把数值推得稍高一些，以便演示效果。

添加清漆

我们希望给地板木板增加一层额外清漆，这可以通过 PxrSurface 中的 Clear Coat layer 轻松实现。如果使用 Lama，也可以把带色调的 LamaDielectric 节点分层作为基础层，然后调整 PxrLayer 节点上的 Layer Thickness 参数来完成。

这里添加了一个使用默认值的 Clear Coat layer。

通常购买木器清漆时，可以指定它的颜色，这会为未处理的木材提供保护和色彩。我们也可以很容易地复现这一点。

我们只需要增加一些 Layer Thickness（可以想象清漆在木材上涂得有多厚），再使用 Absorption Tint，就能给清漆引入一些颜色。

不错...现在我们已经把一块单调乏味的木材变成了色彩漂亮的上清漆木材。

完成了吗？也许...但我们的清漆粗糙度非常平滑、完美。理想情况下，我们希望给这层清漆加入一些粗糙度，以强化它在受损、磨损和划伤位置的真实感。同样，这是创作决定，因此请参考你的参考图像来调出目标效果。

在这个示例中，我们添加了一张粗糙度贴图，包含擦痕、划痕和细小瑕疵。这里使用 PxrRoundCube 节点，而不是 PxrRandomTileManifold，因为我们不希望这些瑕疵在不同木板之间发生变化。

本例还使用了 HexTiling（而不是 tri-planar）映射，因为它非常适合在大面积上无缝平铺贴图，避免接缝或可见重复。

呼...这节课快结束了，不过在结束前，还要再补充一点。

制作这类示例时，相机角度非常关键。相机越低、掠射角越高，你的透明涂层就会显得越明显，因此可能需要相应地调整它。

进一步优化

  • 尝试使用漫反射贴图（或瑕疵贴图）来驱动 Clear Coat layer 的 Layer Thickness。当你希望刮掉清漆涂层、露出下面的原始木材时，这可以用来增加额外真实感。

  • 给 Clear Coat layer 添加一张 bump 纹理，以加入更多 bump 和瑕疵

  • 为你的漫反射纹理贴图制作一张 Bump2Roughness map。随后可以用它来驱动地板木板的粗糙度、bump 和各向异性。

  • 在地板上散布一些细小灰尘颗粒和毛发曲线，增加一层污垢。

  • 在 Diffuse 或 Clear Coat 上方叠加一张瑕疵贴图，以增加更多损伤。

  • 用地板上的瑕疵来讲故事。它是新的还是旧的？是否曾有椅子被用力拖过地面，留下长长的划痕和凹痕？
