AOVs 和 LPEs

本项目中的 LPEs 是从之前的项目（Rivendell fan art）import 过来的；Katana 的 beauty 优势在于，你可以在项目之间使用 nodes，甚至复用整个 template，也可以在不同 software versions 之间迁移（例如，LPEs group 是从 Katana v3.5 移植过来的，而这个项目使用 Katana v6.0 完成，也尝试过用同一 Katana script 在 v6.5 中运行）。
