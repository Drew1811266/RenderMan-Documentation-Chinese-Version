无论你是刚接触渲染，还是从其他 renderer 转向 RenderMan，这份 cheatsheet 都能帮助你快速熟悉 RenderMan 艺术家在 shading toolkit 中使用的众多技术和创意 nodes。

Shaders（着色器）

操作

RenderMan Node（RenderMan 节点）

用途

简单的 Metallic PBR shader

PxrDisneyBsdf

Walt Disney Animation Studios 制作的所有 CG 动画都使用这个 material model

我想使用 Pixar Animation Studios 使用的 shader

PxrSurface

这是 Pixar Animation Studios 长片动画中使用的同一套 material model。

我想叠加多个 PxrSurfaces

PxrLayerSurface

与 PxrSurface 类似，但允许通过 layer masks 驱动 layering

使用 ILM 的 production layering shaders

LamaSurface

MaterialX Lama 是由 Industrial Light & Magic 开发的先进 material layering system，它引入了模块化构建 material networks 的方法，并包含近期关于能量守恒和色散的 shading R&D 工作。 

我只需要一个 constant material

PxrConstant

一个基础 constant shader（常量着色器）。

用于 volumes、clouds、pyro 和 VDBs 的 shader

PxrVolume

PxrVolume 是一种 material，可用于渲染复杂度差异很大的 volumetric effects

我想使用 Pixar 使用的 hair shader！

PxrMarschnerHair

PxrMarschner 是一种 material，用于渲染真实感 hair、fur 和 fibers。

我想使用 Lama hair shader。

LamaHairChiang

LamaHairChiang 是 Pixar Chiang hair shader 的可分层版本，并在 Pixar production 中使用。

我想给 material 添加 displacement

PxrDisplace

将此 node 添加到 shader output，即可给 material 添加 displacement

RenderMan Node（RenderMan 节点）

用途

加载 texture map（纹理贴图）

PxrTexture

RenderMan 版本的 texture file load node

Triplanar Mapping（三平面映射）

PxrRoundCube + PxrMultiTexture

纹理的 Triplanar mapping

HexTiling

PxrHexTile + PxrMultiTexture

在创建大型无缝地貌时很有用

调整 Texture map 的 tiling 和 offset（平铺与偏移）

PxrManifold2d

这个 node 允许艺术家用 2D 方案放置 patterns，通常依赖艺术家提供的 UVs。

在 RenderMan 中使用 Metallic PBR maps

PxrMetallicWorkflow

允许 Metallic PBR maps 与 RenderMan shaders 一起使用

Texture Correction（纹理校正）工具

操作

RenderMan Node（RenderMan 节点）

用途

将 color map 转成黑白 map，

PxrThreshold

很适合从 diffuse maps 创建 roughness/breakup maps

对 texture maps 做更精细的调整

PxrRemap

可以更细致地控制纹理的 Input、Bias、Gain 和 Output 值

对 texture 做 color correction（颜色校正）

PxrColorGrade

PxrColorCorrect

调整纹理的 saturation、gain、contrast 以及其他元素

调整 texture 的 Hue、Saturation 和 Value

PxrHSL

调整给定 input color 的 hue、saturation 和 value。

使用 curve（曲线）调整 texture（纹理）

PxrSplineMap

使用此 pattern 将 input remap 到 float 和 color 两种 outputs

Bake（烘焙）

PxrBakeTexture

Texture Layering & Blending（纹理分层与混合）

操作

RenderMan Node（RenderMan 节点）

用途

Blend 两个 textures

PxrBlend

允许使用多种 blend modes 将两个 textures 分层并混合在一起

Blend 最多 8 个 textures 的 stack

PxrLayeredBlend

类似 PxrBlend，但最多允许 8 个 textures 分层并分别混合。

Blend textures 及其 masks。

PxrLayeredTexture

这对给 texture result 添加 decals、labels、damage 等内容很有用。

Bump 与 Displacement（凹凸与置换）

操作

RenderMan Node（RenderMan 节点）

用途

向 material network（材质网络）添加 bump

PxrBumpMixer

这个 node 提供了一种现代方式来生成并分层 bump maps

调整 displacement texture values（置换纹理值）

PxrDispTransform

此 pattern 会把 input displacement values 转换到一个自定义 value range，供 PxrDisplace 的 input 使用。

从黑白 bump map 创建 BumpRoughness map

PxrBumpRoughness

使用此 pattern 时，RenderMan 会根据 camera 中 feature 的尺寸，在 bump map 与 roughness value 之间自动切换，

有些 nodes 没有 NG out，但我希望它们也影响 bump

PxrRGBtoNG

有时你可能想为 PxrBumpMixer 创建 bump map input，但最后一个 node 缺少 NG OUT。使用此 node 可将 RGB 转换为 NG

Edge Detection（边缘检测）工具

操作

RenderMan Node（RenderMan 节点）

用途

我想创建 procedural edge masks 和 breakup

PxrCurvature

这是一个实用且直接的 pattern，可创建 edge effects 和 curvature-based results，例如边缘磨损以及与其他 surface 接触的 objects。

我想在 crevices 中加入 dust 和 dirt。

PxrDirt

PxrDirt 会渲染 local geometric occlusion。这对将 shading effects 应用到 nooks 和 crevices 很有用。

我想渲染 Wireframe（线框）

PxrWireframe

这个 pattern 用于把 geometry wireframes 绘制为 pattern assignment。

检测 geometry 的 facing ratio（朝向比例）

PxrFacingRatio

用途很多，包括 procedural edge dirt 以及制作 light bulb emission。

Lights（灯光）

操作

RenderMan Node（RenderMan 节点）

用途

在 RenderMan 中使用 HDRI map

PxrDomeLight

在许多 visual effects 和 animation 工作中，场景主要通过 High Dynamic Range Image（HDRI）照明。

使用动态 Sun / Sky environment

PxrEnvDayLight

当你想为 exterior lighting 快速搭建简单设置时，Daylight simulations 很有帮助。

Rect 或 Area light（矩形或面积光）

PxrRectLight

这个 one-sided area light 模拟矩形 light source。

Disc-shaped Light（圆盘形灯光）

PxrDiscLight

这个 one-sided area light（单面面积光）模拟 disk-shaped light source（圆盘形光源）。

Sphere-shaped Light（球形灯光）类型

PxrSphereLight

这个 area light 模拟点状和球形 light sources，例如灯泡、头灯等。

Cylinder 或 tube-shaped light（圆柱或管状灯光）

PxrCylinderLight

其用途包括照亮 objects，模拟许多建筑中使用的 commercial lighting、linear lights、fluorescent lights 和 light panels。它也可用于制作 lightsabers。

Distant、infinite 或 sun/moon directional light

PxrDistantLight

这个 infinite light 模拟明亮、遥远的 light source，例如太阳或月亮。它会照亮整个 scene，并默认把光线视为 parallel rays。

Portal Light

PxrPortalLight

由于带有很少或没有 direct lighting 的室内照明很难让 renderer 解析，portal lights 会帮助 renderer 将注意力引向外部 light source。

为 shot-specific masks 创建 AOVs

PxrAovLight

允许 lighting artist 输出 AOV (Area of Interest) mask。由于它是一个 "light"，我们也可以使用 light linking 和 light filter(s) 来修改输出 mask。

Light Filters（灯光过滤器）

操作

RenderMan Node（RenderMan 节点）

用途

给 light 添加高级 spotlight 和 barn door 功能

PxrBarnLightFilter

PxrBarnLightFilter 让我们可以创建 physically accurate window barns，并以正确阴影模拟真实片场照明。

给 light 添加 Cookie 或 gobo texture，类似 projector 或模拟窗口 shadows（阴影）

PxrCookieLightFilter

PxrCookieLightFilter 会在 light 前方投射 painted texture。

阻挡所选 light 在 scene 某些区域中的光照

PxrRodLightFilter

PxrRodLightFilter 使用类似 "rod" 的 object 来 block light。

调整 scene 某些区域中所选 light 的 exposure

PxrIntMultiLightFilter

当你想把具有不同 intensity/exposure 的特定 asset(s) 与 scene 其他部分隔离时，这非常有用。
