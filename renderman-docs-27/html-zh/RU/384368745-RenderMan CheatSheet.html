无论你是刚接触渲染，还是从其他渲染器转向 RenderMan，这份速查表都能帮助你快速熟悉 RenderMan 艺术家在着色工具集中使用的众多技术和创意节点。

着色器

操作

RenderMan Node（RenderMan 节点）

用途

简单的金属度 PBR 着色器

PxrDisneyBsdf

Walt Disney Animation Studios 制作的所有 CG 动画都使用这个材质模型

我想使用 Pixar Animation Studios 使用的着色器

PxrSurface

这是 Pixar Animation Studios 长片动画中使用的同一套材质模型。

我想叠加多个 PxrSurfaces

PxrLayerSurface

与 PxrSurface 类似，但允许通过层遮罩驱动分层

使用 ILM 的生产级分层着色器

LamaSurface

MaterialX Lama 是由 Industrial Light & Magic 开发的先进材质分层系统，它引入了模块化构建材质网络的方法，并包含近期关于能量守恒和色散的着色 R&D 工作。 

我只需要一个常量材质

PxrConstant

一个基础常量着色器。

用于体积、云、pyro 和 VDB 的着色器

PxrVolume

PxrVolume 是一种材质，可用于渲染复杂度差异很大的体积效果

我想使用 Pixar 使用的毛发着色器！

PxrMarschnerHair

PxrMarschner 是一种材质，用于渲染真实感毛发、绒毛和纤维。

我想使用 Lama 毛发着色器。

LamaHairChiang

LamaHairChiang 是 Pixar Chiang 毛发着色器的可分层版本，并在 Pixar 生产中使用。

我想给材质添加置换

PxrDisplace

将此节点添加到着色器输出，即可给材质添加置换

RenderMan Node（RenderMan 节点）

用途

加载纹理贴图

PxrTexture

RenderMan 版本的纹理文件加载节点

Triplanar Mapping（三平面映射）

纹理的三平面映射

HexTiling

PxrHexTile + PxrMultiTexture

在创建大型无缝地貌时很有用

PxrManifold2d

这个节点允许艺术家用 2D 方案放置 pattern，通常依赖艺术家提供的 UV。

在 RenderMan 中使用 Metallic PBR 贴图

PxrMetallicWorkflow

允许 Metallic PBR 贴图与 RenderMan 着色器一起使用

纹理校正工具

操作

RenderMan Node（RenderMan 节点）

用途

将彩色贴图转成黑白贴图，

PxrThreshold

很适合从漫反射贴图创建粗糙度/破碎贴图

对纹理贴图做更精细的调整

PxrRemap

可以更细致地控制纹理的 Input、Bias、Gain 和 Output 值

对纹理做颜色校正

PxrColorGrade

PxrColorCorrect

调整纹理的饱和度、增益、对比度以及其他元素

调整纹理的 Hue、Saturation 和 Value

PxrHSL

调整给定输入颜色的色相、饱和度和值。

使用曲线调整纹理

PxrSplineMap

使用此 Pattern 将输入重新映射为 float 和 color 两种输出

烘焙

PxrBakeTexture

将网络中的 pattern 烘焙到纹理，模型需要 UV

纹理分层与混合

操作

RenderMan Node（RenderMan 节点）

用途

混合两个纹理

PxrBlend

允许使用多种混合模式将两个纹理分层并混合在一起

混合最多 8 个纹理的堆栈

PxrLayeredBlend

类似 PxrBlend，但最多允许 8 个纹理分层并分别混合。

混合纹理及其遮罩。

PxrLayeredTexture

这对给纹理结果添加贴花、标签、损伤等内容很有用。

凹凸与置换

操作

RenderMan Node（RenderMan 节点）

用途

向材质网络添加凹凸

PxrBumpMixer

这个节点提供了一种现代方式来生成并分层凹凸贴图

调整置换纹理值

PxrDispTransform

此 pattern 会把输入置换值转换到一个自定义值范围，供 PxrDisplace 的输入使用。

从黑白凹凸贴图创建 BumpRoughness 贴图

PxrBumpRoughness

使用此 Pattern 时，RenderMan 会根据相机中特征的尺寸，在凹凸贴图与粗糙度值之间自动切换，

有些节点没有 NG out，但我希望它们也影响凹凸

PxrRGBtoNG

有时你可能想为 PxrBumpMixer 创建凹凸贴图输入，但最后一个节点缺少 NG OUT。使用此节点可将 RGB 转换为 NG

边缘检测工具

操作

RenderMan Node（RenderMan 节点）

用途

我想创建程序化边缘遮罩和破碎效果

PxrCurvature

这是一个实用且直接的 pattern，可创建边缘效果和基于曲率的结果，例如边缘磨损以及与其他表面接触的对象。

我想在缝隙中加入灰尘和污垢。

PxrDirt

PxrDirt 会渲染局部几何遮蔽。这对将着色效果应用到角落和缝隙很有用。

我想渲染 Wireframe（线框）

PxrWireframe

检测几何体的朝向比例

PxrFacingRatio

用途很多，包括程序化边缘污垢以及制作灯泡发光。

灯光

操作

RenderMan Node（RenderMan 节点）

用途

在 RenderMan 中使用 HDRI 贴图

PxrDomeLight

在许多视觉特效和动画工作中，场景主要通过高动态范围图像（HDRI）照明。

使用动态 Sun / Sky 环境

PxrEnvDayLight

当你想为室外照明快速搭建简单设置时，日光模拟很有帮助。

矩形或面积光

PxrRectLight

这个单面面积光模拟矩形光源。

圆盘形灯光

PxrDiscLight

这个单面面积光模拟圆盘形光源。

球形灯光类型

PxrSphereLight

这个面积光模拟点状和球形光源，例如灯泡、头灯等。

圆柱或管状灯光

PxrCylinderLight

其用途包括照亮对象，模拟许多建筑中使用的商业照明、线形灯、荧光灯和灯板。它也可用于制作光剑。

远处、无限远或太阳/月亮方向光

PxrDistantLight

这个无限远灯光模拟明亮、遥远的光源，例如太阳或月亮。它会照亮整个场景，并默认把光线视为平行光线。

门户灯

PxrPortalLight

由于带有很少或没有直接照明的室内照明很难让渲染器解析，门户灯会帮助渲染器将注意力引向外部光源。

为镜头专用遮罩创建 AOV

PxrAovLight

允许灯光艺术家输出 AOV（Area of Interest）遮罩。由于它本质上是一个灯光，我们也可以使用灯光链接和灯光滤镜来修改输出遮罩。

灯光滤镜

操作

RenderMan Node（RenderMan 节点）

用途

给灯光添加高级聚光灯和遮扉功能

PxrBarnLightFilter

PxrBarnLightFilter 让我们可以创建物理准确的窗口遮扉，并以正确阴影模拟真实片场照明。

给灯光添加 Cookie 或 gobo 纹理，类似投影器或模拟窗口阴影

PxrCookieLightFilter

PxrCookieLightFilter 会在灯光前方投射绘制纹理。

阻挡所选灯光在场景某些区域中的光照

PxrRodLightFilter

PxrRodLightFilter 使用类似 “rod” 的对象来阻挡灯光。

调整场景某些区域中所选灯光的曝光

PxrIntMultiLightFilter

当你想把具有不同强度/曝光的特定资产与场景其他部分隔离时，这非常有用。
