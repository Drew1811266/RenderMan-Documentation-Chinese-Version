合成工作流程（Compositing）

对于 compositing，可以考虑使用 shuffle node 采用 additive reconstruction approach，重建整套 AOVs，并恢复最终 “beauty” image。

reconstruction 完成后，可以使用 utility nodes 增强 light 和 color intensity，以更紧密匹配 references。此外，如果 references 中存在 defocus 和 motion blur，也要确保它们对齐。

用 final color-correction pass 结束 compositing process，使用 glow、vignetting 和 grain 将 asset seamless integrate 到 shot environment 中。Custom-built 自定义构建的 nodes 或来自 Nukepedia 的 nodes 可提供 technical tools，用于 precise adjustments，确保 final result 的 accuracy。

为了紧密匹配 references，请在 spectrum 两端调整 gain 和 gamma，使 grain 与 black-and-white points 对齐。探索 extreme values 可以提供额外 visual 和 technical insights，用于 fine-tune image 并增强 integration。

重要的是要记住，你是在匹配使用不同 assets 的 reference，这可能导致 shaders、specular highlights 和 contrast 出现 variations。例如，scales 会因其 inherent properties 而与 hair 反射 light 的方式不同。

因此，必须相应调整你的 adjustments，以在适应每个 asset 独特 characteristics 的同时，实现 cohesive visual consistency。

使用 node switch 和 W key，在 reference 与你的 rendering 之间进行有效 comparison。仔细检查 reference，根据需要 zooming in and out，并在不同 colorimetry 和 calibration 的 screens 上测试，因为这些 factors 会影响 perception。

通过在 reference 和你的 rendering 之间切换，可以识别需要 adjustment 的 areas，从而实现更接近的 alignment。

关注 details 和 nuances，确保你的 rendering 忠实捕捉 reference image 的 essence 和 characteristics。

这种细致 approach 会提升 final output 的 accuracy 和 quality。
