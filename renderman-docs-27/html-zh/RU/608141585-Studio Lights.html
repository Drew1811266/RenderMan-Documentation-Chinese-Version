Overview（资源概览）

这些 light maps 是在 studio 中拍摄，并编辑为 HDRI textures。

给 lights 添加 texture，是获得非完美 reflections 和 shadows 的好方法。你可以把它们 plug into PxrRectLight 的 color map attribute。Textures 已预转换为 tex format，并以易于 drag-and-drop 的 folder 形式提供给 RenderMan Preset Browser。

Enjoy!（祝你使用愉快！）

  • soft box 完全 neutral、desaturated。原始尺寸为 65 x 65 cm。

  • spot 以 6500K 平衡，并带有轻微 warm variations。原始尺寸直径为 18 cm。

  • kitchen 以 6500K 拍摄，提供 warm color。原始尺寸为 36 x 6 cm。

  • kitchen 使用基于 light source 的 white balance 拍摄。原始尺寸为 36 x 6 cm。

注意，high resolution versions 只应在 closeups 中使用；在大多数 cases 下，low resolution versions 应该已经足够。

How-to（安装与使用步骤）

将内容拖入你的 Asset Library > LightRigs folder。文件按 category 组织，可以在 preset browser 中轻松搜索。

请确保把安装目录中的 RenderManAssetLibrary folder 转移到具有 full writing privileges 的位置。

当你转移 Materials folder 并把 Asset Library 移到其他位置后，必须通过 Maya preferences 告诉 Maya 使用新的 path。

或者，你可以在 Maya.env file 中创建一个新的 environment variable，以确保始终指向正确的 folder：

RMAN_ASSET_LIBRARY = /My/Folder/Example/RenderManAssetLibrary
