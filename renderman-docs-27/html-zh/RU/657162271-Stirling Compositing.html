合成工作流程（Compositing）

这是一段相当长的旅程，但我们终于到达 integration 的最后一步：compositing！

首先，我们要确保 ACES workflow 在 Nuke 中也保持一致。不用担心，这只需要点击几下，Nuke 会为我们把 sRGB footage 转换到正确的 acesCG color space。

要访问 Nuke 中的 project settings，把 mouse hover 到 Node Graph window 上并按 “S”。进入 Color tab，并将 “Nuke Default” setting 切换为 “OCIO”。这会确保 Nuke 自动以正确 color space 打开你的所有 files。

谈到 rotoscoping，我最好的建议是尽量保持 shapes 简单。处理 20 个每个 6 点的小 shapes，要比挣扎于一个过度复杂、很可能让你一天结束时崩溃的 single shape 容易得多。因此，不要犹豫，把 object 拆分为多个 shapes，以获得更好的 control 和 efficiency。

另一个有用技巧是调整每个 shape 的 “Life” input。这允许你在 shape 不再需要时 deactivate 它，避免不必要的 shapes 尴尬地漂浮在 screen 上。

此外，keyframes 越少，问题越少。为了节省时间，可以先做 rough pass，每 5 到 10 frames 设置一个 keyframe（可以用 Shift + arrow keys 按固定 frame 数跳转）。这会帮助你快速识别哪些 frames 需要最多 refinement。
最后，虽然 Bezier curves 是 rotoscoping 的常用工具，但我强烈建议尝试 B-spline shapes。它们通常需要更少 control points，更容易 fine-tune 和 stabilize，尤其是在处理 organic motion 时。

欢迎探索提供的 Nuke scene，以更深入地理解整个 process。你会找到解释 node-by-node process 的 sticky notes。

不过，如果让我根据自己的经验总结 plate matching 时需要注意的关键事项：

  • 极端数值范围（Extreme values）： 任何过暗或过亮的部分都会立刻破坏 illusion。注意 contrast，并确保 smooth integration。

  • Pixel imperfections（像素瑕疵）： 3D renders 从 render engine 直接出来时往往太 sharp。花时间 softening edges，并把它们自然 blend 到 imperfect real-world footage 中。

  • The illusion matters most（幻觉最重要）： 在某些情况下，我甚至会 rotopaint 掉一些 small details，即便它们是真实存在的，因为它们太吸引眼球，可能让 viewer 下意识觉得有什么不对。

归根结底，目标是把 CGI seamless blend 到 live-action 中，同时避免任何可能触发 visual suspicion 的东西。

我个人认为这是一个非常 unforgiving、但也很有 creative excitement 的练习，因为即使最没有经验的 viewer 也能立刻看出什么地方感觉不对。我们的 eyes 天生经过一生训练，用来观察周围世界，因此 integration 中的任何 inconsistency 都会马上变得 noticeable。
