LamaConductor node 用于创建 metal 或 conductive materials。
第一个 example 使用 physical IOR 和 Extinction values，这些数值取自非常实用的 reference website： PhysicallyBased.info

IOR & Extinction（折射率与消光系数参数）

第一个 example 展示如何使用来自下列来源的 IOR 和 Extinction values： PhysicallyBased.info 的 IOR 和 Extinction values，准确重新创建 real-world metals。



Copper Clean（干净铜材质示例）


Iron with a touch of Tail Mix（带少量 Tail Mix 的铁材质示例）



Gold with scratches（带划痕的金材质示例）

发挥 RenderMan 的 BumpRoughness 能力，使用 single map 同时驱动 Roughness、Bump 和 Anisotropy，并且无论 object 离 camera 多远都能保留 details。

Tech Metal（现代电子设备科技金属材质）

第二个 example 复现了一种常见于 modern electronic devices 的 tech metal。
IOR 和 Extinction 都设置为 Aluminum，并通过混合 solid color 与 PxrFractal 添加少量额外的 color variation。
这个 material 的主要 look 由 PxrFractal noise 驱动的 bump 和 roughness 创建；相关 parameters 被调低，用来产生非常细小的 white noise pattern。
