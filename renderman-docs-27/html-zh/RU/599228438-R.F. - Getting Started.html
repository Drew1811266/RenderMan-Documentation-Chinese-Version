 Preset Browser - Part 2（预设浏览器第 2 部分）

 RenderMan 视口

在本课中，你将学习如何在 Maya 中启动并使用 RenderMan 的交互式视口来完成外观开发工作。

 使用 Maya 的 Render View

如果你更喜欢在 Maya 的 RenderView 窗口中工作，这节课会让你启动并开始渲染。
