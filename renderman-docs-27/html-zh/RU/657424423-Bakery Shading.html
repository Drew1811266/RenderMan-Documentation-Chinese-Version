Bakery 场景着色流程

Substance（或 Mari 等其他 3D 绘制软件）的另一个优势是预览能力，它可以让你很好地概览材质，并预先调整贴图。不过，这并不意味着一旦贴图应用到 PxrSurface 上，我们只需要把它们连接到对应通道，就可以认为着色完成了。Substance 中得到的渲染结果和 RenderMan 中得到的渲染结果之间可能存在差异。因此，在细化最终结果时，有时需要重新映射、颜色校正，或用其他方式调整纹理。


复制与 PxrVary

当许多重复元素共享同一个 PxrSurface 时，我们必须找到方法让这些内容产生变化。在这种情况下，最有用的节点之一是 PxrVary。连接到 color base 时，这个节点可以在饱和度、明度和色相上引入变化。这样可以获得更高的真实感，并避免令人畏惧的“完美 3D”效果。例如，对于面包和糕点，这让我们能够为烘焙食品添加微妙差异，因为面包从来不会 100% 相同，它们会因面粉、水、酵母和烘烤时间略有变化。

粗糙度与高光

我还有另一个技巧：在把 Roughness 贴图应用到 EdgeColor 之前，先使用 remap 将它反转。然后可以根据需要调整 remap；在这个案例中，我降低它以保留涂层的光泽。这个技巧可以在粗糙度中提供更多变化，并进一步丰富纹理。

最后，这个布景的一个特殊之处是它的视角：它为整体镜头设计，对象看起来相对较远。正如我们在 UV 章节中已经提到的，我们通过导出小尺寸贴图来优化渲染时间，同时也可以因此避免应用位移贴图。确实，在这个距离下，一个简单的法线贴图已足以有效模拟浮雕感。
