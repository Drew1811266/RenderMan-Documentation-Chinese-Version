着色

为了让这个资产看起来可信，我需要处理许多不同材质，但没有不同几何体可供分配这些材质。所有不同材质都需要合并在一个主着色器下，因此我认为这类资产的最佳方法，是使用 “PxrLayerMixer” 和 “PxrLayer” 节点进行分层着色，以分离这些不同属性，并使用遮罩在材质之间实现平滑过渡。

我为身体的不同区域分别设置了着色器控制，例如花瓣、叶片、脚、眼睛周围区域、腹部、颈部、触角、舌头、嘴等。

花瓣

花瓣的外观开发是更关键的材质之一，因为它既要看起来很薄，也要像花瓣那样透光。最初我尝试借助遮罩通过 SSS 来实现这种外观。

不过，我对这些结果并不满意，因为花瓣开始看起来更像蜡，而且缺少明亮的背光色调。为了解决这个问题，我想到了另一种方法：将使用 Glass parameter 的 Transmission 与 SSS 结合。为了获得额外控制，我使用了 PxrRemap 和 PxrColorCorrect 节点。

叶片

叶片的几何体比花瓣稍厚，因此不像花瓣材质，我不需要使用 Transmission 来获得类似花瓣的背光外观。下面是我仅使用 SSS parameter 得到的结果。

眼睛

我在许多参考中注意到，青蛙和一些其他爬行动物的眼睛有一种金属感。其中一些还有类似金箔的薄片。我觉得这很有趣，并尝试在 Hippie 的眼睛中实现类似外观。

对于虹膜和巩膜，我同时使用 SSS 和 Specular Edge Color。为了获得金属感，我略微提高了 IOR。这给了我半金属外观。虽然眼睛有这种金属光泽，但它仍不是真正的金属。归根结底，它是有机材质，因此我没有只让 Specularity 驱动颜色，而是将其与次表面结合，以便也获得一些来自 SSS 的贡献。

对于金色薄片，我最初只使用 Specular Edge Color，并把 IOR 调到高得多的值，以获得纯金属外观。不过，在光线很少或没有照到的区域中，它会看起来几乎是黑色。因此最终我沿用了眼睛其余部分的相同方法，但保持更高的 IOR，帮助我获得想要的类似金箔的光泽。
