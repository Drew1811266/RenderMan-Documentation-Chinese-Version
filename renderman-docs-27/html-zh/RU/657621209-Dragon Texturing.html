Dragon 角色贴图制作流程（Texturing）

对于 secondary displacement details，我使用了 texturing.xyz textures，并为 dragon 的 hard surface sections 添加了 tileable textures。完成这一步后，我的第一件事是在 RenderMan for Houdini 中 test setup，确保到目前为止的选择是成功的。

我非常喜欢用 pxrDisplacementLayer node 开始 displacement network，因为它让 displacement maps 的 layering 变得非常高效且直接。

之后我回到 Mari，从大面积 colors 开始；随着工作推进，我借助 geometry bakes 加入更细的 details 和 breakups。最后，我用 high resolution tileable textures 和一些 hand painting 完成了整体。

在这个阶段，再进行一轮 RenderMan testing 很有用，它让我能以可管理的 steps 构建 material networks。当我对进度满意后，我制作了 supporting material texture maps，例如 Roughness 和 SSS scale，再加上一些 arbitrary masks，以便后续 look development process 中进行 control。

Tip 1：Start simple，从简单开始，并在查看 references、理解 asset 如何着色的过程中逐步 layer in detail。

Tip 2：借助 geometry bakes，你可以快速把许多部位带入 textures，同时支撑 shape，并保持 procedural。

借助 arbitrary texture masks，我使用 pxrLayeredBlend nodes 为不同 parts 做 color corrections。我还对 Roughness map 做了 grade，以便完全控制不同 values 和 areas，因为自然界中的 animals 往往会用 body 上更硬的 surfaces 进行防御，因此 wear and tear 需要在这个思路下表达。

这一 layer 非常重要，因为它常常把 asset “glues” 在一起，创建具有共同 value 和 design language 的 textural areas。即使对于这只 dragon 这样的 creature，这种 effect 可能很 subtle；在 hard surfaces 上，dirt textures 则可能明显得多。

Tip 1：先制作 displacement，并使用 gray shader 判断 displacement details 和 SSS，以免被 color details 分散注意力。

Tip 2：思考现实中不同 materials 如何相互覆盖，并尝试以 digital 且 non-destructive 的方式重建这一点。
