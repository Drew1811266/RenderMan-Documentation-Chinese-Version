Bakery 烘焙坊场景建模流程（Modeling）

目标是创建一个带有温暖、熟悉氛围的 cartoon decor bakery。
首先，我们需要思考 image 的 composition……我们想在 image 中传达什么？这些 elements 被安排在一起后会如何发挥作用？

这里的 objective 是创建 set 的 overview。因此，framing 展示了整个 environment，让 viewer 能理解空间。viewpoint 的选择强化了这种感觉：camera 放在人眼高度，使 viewer 感觉自己正走进 bakery。

为了设计构成 image 的 elements，我从现实中汲取灵感，使用 bakery 中典型的 objects 和 furniture。这样可以为 viewer 保留熟悉的 landmarks，并生成可信、believable 的 image，也更容易让 viewer 沉浸在这个 world 中。

Tip：一旦 bakery 已经可以被识别出来，就不要犹豫，可以玩转 shapes，创造独特的 decor！

重要的是要记住，bakery 首先必须唤醒我们的甜食欲和食欲！这通过创建丰富的 food 来实现，包括各种 shapes 和 sizes 的 breads、pastries 和 tarts。这些 elements 的丰富变化，以及 shelves 上轻微的 disorder（jars、cups、labels、baskets、frames），会创造出充满生命力的空间，强化 bakery 的 abundance 和 beauty。

Tip：为了减轻 scene，可以考虑移除不可见的 object faces，例如贴着 walls 的那些面。
