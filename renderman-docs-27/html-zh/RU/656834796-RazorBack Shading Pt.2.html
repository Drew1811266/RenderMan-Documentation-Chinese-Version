在第二阶段处理 specular components 时，使用三个不同 lobes 可以最大化 shader 的能力。

第一个 lobe，Primary Specular，由一张 map 控制，用来调节特定 regions 中的 specularity，例如 paws 或 scales 的 curvature。这个同一 lobe 也受 scalar roughness map 影响，该 map 决定 scales 上 specular roughness 的范围。

第二个 lobe，Rough Specular，为整个 model 贡献整体 specular effect，外观明显比第一个 lobe 更 rough。这有助于柔化 scales 和 spikes 上 specular highlights 的扩散。

最后，Clearcoat lobe 会增强 mouth 内 mucous membranes 上的额外 specular highlights。该 lobe 由 ISO mask map 控制，将其限制在需要额外 specular highlights 的指定 areas。

当 displacement 和 specular components 在你的 assets 上正确配置后，最后一步是细化 albedo，以确定 scales 的 color 和 subsurface characteristics。改变 gain 可确保 color、subsurface 和 single-scatter components 的总和接近 1。

对于 mucous membranes，可以使用 single-scatter lobe 来突出 mouth 中的 viscous aspect。

为 scales 和 mucous membranes 保持不同 parameters，可以最大限度控制 shader，并减少在 Mari 和 RenderMan 之间来回切换的需求。

充分利用 PxrSurface shader 上可用的 parameters 至关重要，同时要记住 gain 会约束 parameters。

例如，在 subsurface 中调整 mean free path distance 时，需要根据每个 lobe 上的 gains 调整 shader parameters。

完成这些步骤后，在进入 shot 和 environment settings 之前，要在各种 lighting conditions 下彻底测试 shaders 的 reliability。这可以确保 shaders 在不同 scenarios 中表现一致，并保证 final renders 具有最佳 visual quality。

此时进行一次 higher resolution render 也没有坏处，因为它能帮助我们在理想 conditions 下检查任何 texturing inconsistencies。
