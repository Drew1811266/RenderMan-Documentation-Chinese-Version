Project（项目名称与作品入口）：Creating a French Bakery（创建法式面包店场景）

Created & Written by（创作与撰写人员名单）：Marion Machelard、Emma Siret 和 Céline Soubrane

Edited by:（编辑整理人员信息） Leif Pedersen（编辑：Leif Pedersen）

现在是早上 7 点。黎明的 golden light 覆盖着 shop windows 和 pastry。温热面包的香气充满房间。第一批 croissants 刚从烤箱出炉……要开始美好的一天，当然没有比 French way 更好的方式了！走进这个 immersive scene，让 rustic charm 与传统 French craftsmanship 鲜活起来。

首先，让我们介绍一下这个 project。这个 set 是一部名为 Lèche-Vitrine 的 short film 的一部分，由 Marion Machelard 在我们完成学业后构思。我们的想法是和朋友们一起制作一部 short film，同时各自负责自己的专业岗位。

由于 group work 是我们工作中不可或缺的一部分，共同完成这类 project 总是非常有刺激性；这也是我们希望以这种方式呈现 tutorial 的原因。Céline Soubrane 负责 modelling，Emma Siret 负责 texturing/shading，Marion Machelard 负责 lighting/compositing。

希望你会觉得这份内容很 instructive！

About the Artist（艺术家介绍与创作背景）

Céline Soubrane（艺术家姓名）

在学习 animation 和 VFX 之后，Céline Soubrane 成为一名专注于 environments 的 CG artist。她目前在 Quantic Dream 担任 environment artist，参与 Star Wars Eclipse。

Emma Siret（艺术家姓名）

Emma Siret 毕业于 ESMA，是一名 freelance CG artist，专注于 modeling 和 texturing。她热爱 visual creation，并一直在寻找新的 opportunities！

Marion Machelard（艺术家姓名）

Marion 于 2023 年毕业于 ESMA，随后加入 Red Parrot Studios，在一部长片中担任 lighting artist。她热爱 atmospheres 和 colors，也热爱 storytelling 所带来的各种可能性。
