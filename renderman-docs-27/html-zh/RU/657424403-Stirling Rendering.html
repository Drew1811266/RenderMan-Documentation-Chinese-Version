Stirling 项目的渲染设置流程

现在分层已经设置好，渲染设置中只剩少数事项需要考虑。由于我们正在匹配实拍素材，运动模糊是必须牢记的重要因素。

花时间检查素材，并评估图像的清晰程度。即使帧看起来干净且清晰，也始终要运行测试渲染，确保你的运动模糊与实拍素材中的运动模糊匹配。

我强烈建议至少加入少量运动模糊，因为它有助于在帧之间创建更平滑的插值，增强整体可信度和真实感。运动模糊的量取决于你的素材，所以务必调整到匹配状态。

进入合成之前，我也建议你花时间渲染一个遮蔽通道和 Cryptomattes。这会在合成阶段节省大量时间，让你不必花许多小时做繁琐的转描。可以阅读关于如何设置 Cryptomattes的文档。

我最后一个渲染技巧是关于一些有用的覆盖设置（没错，又是图层覆盖！）。虽然使用 LPE（Light Path Expression，光路表达式）渲染通道会非常强大，但在我们的案例中，没有必要为所有图层都渲染它们。由于反射板主要由镜面表面组成，生成数百个占用大量存储空间的 EXR 文件几乎没有价值。

为了优化工作流，你可以停用 LPE 显示，将某些图层限制为只渲染 Beauty 通道。只要确保每次都仔细检查自己正在调整正确图层的渲染设置；你不会想在周一早上回到办公室检查渲染结果时遇到不愉快的惊喜！
