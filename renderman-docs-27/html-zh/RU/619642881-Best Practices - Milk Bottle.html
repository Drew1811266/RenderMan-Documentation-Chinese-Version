牛奶瓶（Milk Bottle）制作

现在我们可以看看牛奶和瓶子了。

牛奶本身没有使用 textures，后面会单独介绍；因此我们先隔离瓶子，讨论它如何真实地实现“dirty bottle”（脏瓶子）外观。

玻璃具有几项独特特性，使它显得很特别。也正因如此，PxrSurface material 有完整的一节用于重建玻璃及最常见的外观。要理解不完美和脏污玻璃，可以试着拆解图像中发生的事情。不完美玻璃的表面 roughness 会发生变化。与水晶和少数其他表面不同，许多陈旧和磨损的玻璃物体并非完美 reflective 或 transmissive（refractive）。玻璃表面还会增加 fingerprints 和 smudges。你可以使用 masks 生成这些效果，并把它们添加到 diffuse lobe 中，混入 dirt、paint、dust 等额外元素。

更轻松地分层这个效果有两个选项可讨论，另外还有第三个关于 trace depth 的注意事项，会在另一篇文档中介绍：

  • 使用“ PxrLayerSurface ”来分离并重新组合玻璃和污垢的 shading。

  • 正确的 UV layout，可以防止内外表面出现 texture repetition。

  • 合理控制“ ray depth ”，以获得玻璃的正确物理结果。

最好以体积方式建模对象。它们应该有真实厚度，玻璃表面之间的空间会像真实世界中一样工作，也应按你的预期表现。除非确实合理，例如叶片或纸张这类对象，否则应避免使用二维模型。

渲染大多数带液体的玻璃容器时，理解 intersection priority 会很有帮助，可避免 artifacts 和 “air gaps”。技术上，这称为 dielectric interface 或 nested dielectrics。

首先，我们移除上层的污垢和牛奶，检查玻璃层。

在 glass base material 中，diffuse channel 以较低 gain 0.2 加入了一些 dirt。低 gain 结合较暗的 texture，可以确保该效果不会过于 opaque，否则瓶子会显得不像玻璃制成。

在 glass base material 中，diffuse channel 以较低 gain 0.2 加入了一些 dirt。低 gain 结合较暗的 texture，可以确保该效果不会过于 opaque，否则瓶子会看起来不像玻璃制成。

请注意，瓶子的内壁和外壁是分别映射的。fingerprint 通常不应该同时出现在瓶子的外侧 和 内侧。

reflection gain 像托盘和桌子一样使用 texture，用于定义瓶子最 reflective 的区域，或最少磨损和脏污的区域。

refraction color 用来为穿过玻璃的 light transmission 着色或衰减。在这个例子中，为了暗示更多不完美之处，一张 map 被连接到这个 color input。白色区域表示这些位置的玻璃是 clear 且 无色的。

roughness 与 diffuse map 类似，但移除了颜色，并调整了 black point。这意味着 texture 上不应有任何区域是完全黑色，或值为 0, 0, 0。

相反，它被略微提高到 2%，即使最清澈的区域也会带有细微 roughness，这对这张图像来说更真实。
