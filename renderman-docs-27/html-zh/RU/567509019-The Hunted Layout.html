The Hunted Layout

一旦我把每个 prop 的 initial proxy version 都做成 USD files，就可以开始处理 scene 的 layout。

HuntedLayout.mov

Houdini 和 Solaris 非常适合处理单个 set-dressing assets，也适合 scattering 大量 instances。

我实现了这两种 workflows：用于 grass、rocks、general plants 和 background trees 的 Scattering（散布）与 instancing techniques。

我通过 layout node 手动放置 hero objects，例如 powerbox、railroad sign 和 mailbox。我也手动放置了 grass，以及 foreground trees 和 rocks，以确保 scene composition 符合我的目标。当你不太在意 Scattering（散布）的 creative composition 时，Scattering 非常适合 instancing objects。不过很多时候，你仍需要依靠 hand-placing 或移动 scattered objects，来适配 shot 和 story。

借助 R27 中 XPU 的最新 advances，我能够在 Solaris 中使用简单 skydome 运行 interactive viewport，从而获得快速且交互式的 layout session。再加上 interactive denoiser 的力量，你就拥有了一套能轻松吞下 layout 工作的 interactive toolkit。

stagemanager.mov

额外 tip：做 layout 时，如果 assets 已经应用了许多 textures，可以使用 RenderMan 中的 Occlusion Integrator。这会提供非常快速的 occlusion IPR session，而不必把所有 shaders override 成简单的 grey material。

感谢 RenderMan 的 Dylan Sisson 提供这个 tip！
