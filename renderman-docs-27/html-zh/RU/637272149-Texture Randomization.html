Texture Randomization

Downloaded: 2026-06-15T03:49:12+00:00

RenderMan 提供了多个 nodes，可用于随机 texture colouring、file selection，以及随机 orientation/placement。

我们要看的两个 nodes 是 PxrVary 节点和 PxrRandomTileManifold 节点。

在这个示例中，我们有七个 spray cans；目前它们看起来相当乏味。

本课的目标是为每个罐子创建随机颜色的 lids。

我们还设计了七种不同的 labels，希望把它们随机应用到罐身的 labels 上。

Spray Can Colored Lids（喷漆罐彩色盖）

我们先处理 lids，使用一个极其实用的 randomization node：PxrVary。

该 node 接收 base color（或 colored input），并允许 hue、saturation、gamma 以及许多其他 parameters 按 instance 具有随机值。

先把 PxrVary 连接到 PxrSurface 的 Diffuse color，并将 Input Color 设置为明亮的 orange。

这会把所有 lids 都变成同一种 orange color，而这不是我们想要的。

我们希望每个 lid 都有随机颜色。

调整以下 values 后，会得到下面的 image。

Hue

0.58

Saturation

0.81

Luminance

0

Gamma

0.1

这样看起来“还可以”，但我们其实不想让 RenderMan 决定选择哪些 random colors。我们想自己选择它们。

我们希望 art-direct PxrVary 随机选择颜色时使用的 color palette。

实现方式是使用 PxrRamp node，在其中设置 ramp colors。我们还需要把 Ramp Type 设置为 Random Object Color。

注意！这项技术只有在你把 Ramp Type 设置为 Random Object Color 时才会生效。

理解这是一个 random node 很有帮助，因此有时很难 art-direct 哪个 lid 获得哪个 random color。有时，如你所见，PxrVary 会随机给两个 lids 分配相同颜色。

另外，ramp 中 colors 的宽度会决定随机分配和概率，因此要调出你想要的准确 random look 可能有点棘手。

这里的关键词就是 Random！

如果你有 100 个 cans，而且并不在意哪个 lid 是哪种 color，那么 PxrVary 就是解决方案；但如果你需要特定 cans 拥有特定 colored lids，那么为每个 can 定义 PrimVar value（然后在 PxrVary 的 PrimVar attribute 中指定 PrimVar name）才是办法，不过那是另一课的内容。

我们想要 random-colored lids……也确实得到了 random-colored lids。

接下来处理 labels。

Spray Can Labels（喷漆罐标签）

这里有几张 textures，我们想再次随机地分配给每个 label，不太在意哪张配哪个 label。我们要的就是 random！

要实现这一点，我们需要两个 nodes。

第一个是 PxrMultiTexture node，我们会把所有 maps 加载到每个 filename parameter 中。

然后，需要一个 PxrRandomTextureManifold node，通过把 PxrMultiTexture files 分配到 cans 上，来控制它们的 randomness。

像上面的 network 那样连接它们，你就会得到类似这样的结果！

注意：你需要把 Number of Textures parameter 设置为要使用的 file list 中 textures 的数量。

PxrRandomTextureManifold 还有各种可调 parameters，可以添加更多 randomization，例如 random texture orientation、flipping、offset 等等，所以 texture randomization 的数量……几乎没有尽头！

更多用途

这两个 nodes 并不只是用于随机选择 colors 或 diffuse textures。

例如，PxrRandomTextureManifold 也可以用于随机分配和变换 roughness/bump maps、叠加 random textures、变换 random edge breakup masks 等。

PxrVary 可以为 roughness maps 添加 random color/saturation variation，让每个 instance 拥有不同的 roughness values。同样的 technique 也可用于设置 random bump 和 specular values，以及控制 clear coat 上的 random thickness。

这两个 nodes 的可能性没有尽头。

如果你想在所有 instances 上实现完整的 shading randomization，这两个 nodes 是不错的起点！
