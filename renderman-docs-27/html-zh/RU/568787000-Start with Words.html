Start with Words

Author: Eugene Riecansky（作者）

W每当我开始一件新作品，不论是动画还是静帧图像，在我甚至还没有创建任何一个像素之前，我总会先花一点时间把这个想法写下来。

在 Art Challenge 中，artists 会得到一个 theme、对 theme 的简短概述，以及一组 starter models。实际上，这就是我们正在寻找什么的 brief，以及 artist 需要遵循的一组 guidelines。除此之外，artist 可以自由地按自己的方式诠释 theme，而这正是从 words（以及 drawings）开始成为关键第一步的地方。

它能让你作为 artist 花时间思考图像、你想传达的 message 和 story、画面中的 emotion，以及你希望 viewer 体验到的 feeling。

开始发展你的 image 时，可以使用下面这些 topic headers：

故事概要说明（Synopsis）

情绪（Emotion）

场景地点设定（Location）

主角发展设定（Main Character Development）

服装 / 道具（Clothing / Props）

一天中的时间（Time of Day）

天气（Weather）

画面灯光设计（Lighting）

色彩方案（Color Palette）

故事概要说明（Synopsis）

Synopsis 是对你的 image 的 short（简短）而准确的描述。

它可以写成一个段落或几个词，让任何人了解 story 及其 action。它不应过度描述每一个细节动作，而只是对你的 idea 做一个快速的 elevator pitch。

例如，试着把你的 synopsis 读给一个完全不了解你 image 的人听，看看他们怎么说。如果他们理解了，你就有了一个好的开始。如果他们看起来困惑，你可能还需要进一步 refine 你的 synopsis。


对于我的作品 The Hunted，我的 synopsis 是这样写的：

“夜色笼罩大地，UFO-hunting van 行驶在荒野中，寻找并搜寻它最终的奖赏……终于亲眼发现一架 UFO。原本像其他夜晚一样开始的事情突然发生意外转折……UFO hunter 变成了 the hunted，一架看不见的 UFO 从黑暗中出现，用 tractor beam 将他抓走。”

通过写下这段关于 image 的 short（简短）synopsis，我已经对自己想创作的作品做出了一些决定。

我决定一天中的时间是夜晚。

我让 reader 知道他正开车行驶在 wilderness 中，并决定反转这辆 hunting van 的处境。他原本在 hunt UFOs，但他自己反而成了 the hunted。我也决定我们不会看到 UFO 本身，把它的 appearance 和 shape 留给 viewer 的想象。

我在建立 scene，但也留下几个问题让 viewer 自己决定。

这段 synopsis 也帮助我开始思考需要创建哪些 assets。

我知道我需要一辆 van。我知道它会在 wilderness 中，所以需要一些 trees 和 landscape。我也明白 van 是 main character，而 unseen UFO 发出的 tractor beam 同样关键。它让我把思考（和学习）集中到特定方向。我需要 model UFO van、一些 trees 等等，但也需要弄清楚如何以 visually interesting 的方式创建 tractor beam 的 light。

最后，我知道不需要思考（或担心）UFO 的 appearance。我们不会看到它。

所以，我希望你能看出为什么 synopsis 是一个关键的第一步。

接下来我需要回答的一组问题会根据你想创建的 image 而有所不同，它们是：

情绪（Emotion）

你试图向 audience 传达什么 emotion？

作品的 mood 是什么？
你希望他们感到害怕、共情、紧张，还是平静？

场景地点设定（Location）

Action 发生在哪里？

这是 interior shot 还是 exterior shot？

是在 UFO hangar 内，坐落在 busy city 中央，还是一条荒凉的、树木成行的道路上？

主要角色设定（Main Character）

谁或什么是 main character？

角色年龄 / 性别（Character Age / Gender）

他们多大？是 young child 还是 older person？

角色的 age 如何增强整个 story？

角色背景故事（Character Backstory）

他们是谁？在这次 action 发生之前，他们做过什么？

你需要讲述多少 personality/backstory，才能帮助 story？

即使你认为它没什么用，也先写下来。也许它会激发另一个可以加入 image 的 personality trait，一点关于他们过去/当前生活的 hint。

服装 / 道具（Clothing / Props）

Clothing 很重要（如果它是 human 或 alien character）。Clothes 定义了许多关于角色自身和 backstory 的信息。他们是刚下班回家吗？是一个喜欢 Rock n Roll 的孩子吗？是一个梦想成为 archaeologist 的女性吗？

Props 对发展角色也很有帮助。Jewelry 可以透露许多关于角色的信息。Hair style、makeup 也是如此。Boots 还是 fancy shoes？Glasses 可能传达某人 intellectual？Bags 暗示某人来自工作、学校，或有很多东西要携带。Open bag 还是 closed bag？

衣服上的 badges 和 patches 也有助于强化 character。

一天中的时间（Time of Day）

夜晚还是白天？晴天还是阴天？Alien world 还是 Earth？

天气（Weather）

Weather 也能增强 image 的 emotional impact。Rain 会增加 anxiety，而 sun 带来 hope。它是否是两者的混合？头顶 dark clouds，却被充满希望的 sunlight beams 穿破。

画面灯光设计（Lighting）

Lighting 在讲述 story 中始终至关重要，是一个巨大因素。试着用 words 描述你的 image。它明亮且充满希望吗？是一道 dark single beam，还是充满 color 和 brightness 的壮丽 glowing vista？

主要 light source 是什么？还可以使用哪些其他 light sources 来帮助推进 story？

色彩方案设计（Color palette）

同样非常重要。虽然这不是一段关于 color theory 的文字，但为你想追求的 color palette 设定一个起点非常关键。Bright and colorful、充满 hope 和 wonder？Dark green 或 blue night tones？Black and white Noir，或受某部 film 的 cool-toned palette 启发？

正如 Mike Cacciamani 在这篇课程中 另一课中提到的，value 和 hue 扮演着关键角色。

我坚定地相信，light & color 可以成就或毁掉一张 image。强有力的 shading 和 textures 定义整张 image 的真实感。Directed light 和 shadow 引导 viewer 的视线。而 rendering 会让你所有的选择真正活起来。

你也可能会发现自己可以继续为这组问题添加内容。它绝不是你需要问自己的唯一问题，但它是一个很好的 starting point。可以自由加入你自己的问题。

需要说明的是，你在创作过程中并不必须 100% 固守这些决定。

当你创作时，new ideas 总会浮现，但你会发现，在开始制作任何东西之前，至少先列出一些 initial thoughts 是有帮助的。

Scene 会/可能会演化。不要害怕在思考时重写部分内容。我再怎么强调这个 process 的重要性都不为过。
