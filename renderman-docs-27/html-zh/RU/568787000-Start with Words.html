Start with Words

作者：Eugene Riecansky

W每当我开始一件新作品，不论是动画还是静帧图像，在我甚至还没有创建任何一个像素之前，我总会先花一点时间把这个想法写下来。

在 Art Challenge 中，艺术家会得到一个主题、对主题的简短概述，以及一组起始模型。实际上，这就是我们正在寻找什么的创作简报，以及艺术家需要遵循的一组准则。除此之外，艺术家可以自由地按自己的方式诠释主题，而这正是从文字（以及草图）开始成为关键第一步的地方。

它能让你作为艺术家花时间思考图像、你想传达的信息和故事、画面中的情绪，以及你希望观众体验到的感受。

开始发展你的图像时，可以使用下面这些主题标题：

故事概要

情绪

场景地点设定

主角发展设定

服装 / 道具

一天中的时间

天气

画面灯光设计

色彩方案

故事概要

故事概要是对你的图像简短而准确的描述。

它可以写成一个段落或几个词，让任何人了解故事及其动作。它不应过度描述每一个细节动作，而只是对你的想法做一个快速的电梯陈述。

例如，试着把你的故事概要读给一个完全不了解你图像的人听，看看他们怎么说。如果他们理解了，你就有了一个好的开始。如果他们看起来困惑，你可能还需要进一步打磨你的故事概要。


对于我的作品 The Hunted，我的故事概要是这样写的：

“夜色笼罩大地，一辆追踪 UFO 的厢式车行驶在荒野中，寻找并搜寻它最终的奖赏……终于亲眼发现一架 UFO。原本像其他夜晚一样开始的事情突然发生意外转折……UFO 猎人变成了猎物，一架看不见的 UFO 从黑暗中出现，用牵引光束将他抓走。”

通过写下这段关于图像的简短故事概要，我已经对自己想创作的作品做出了一些决定。

我决定一天中的时间是夜晚。

我让读者知道他正开车行驶在荒野中，并决定反转这辆狩猎厢式车的处境。他原本在追踪 UFO，但他自己反而成了猎物。我也决定我们不会看到 UFO 本身，把它的外观和形状留给观众的想象。

我在建立场景，但也留下几个问题让观众自己决定。

这段故事概要也帮助我开始思考需要创建哪些资产。

我知道我需要一辆厢式车。我知道它会在荒野中，所以需要一些树木和景观。我也明白厢式车是主角，而看不见的 UFO 发出的牵引光束同样关键。它让我把思考（和学习）集中到特定方向。我需要为 UFO 厢式车、一些树木等建模，但也需要弄清楚如何以视觉上有趣的方式创建牵引光束的光。

最后，我知道不需要思考（或担心）UFO 的外观。我们不会看到它。

所以，我希望你能看出为什么故事概要是一个关键的第一步。

接下来我需要回答的一组问题会根据你想创建的图像而有所不同，它们是：

情绪

你试图向观众传达什么情绪？

作品的氛围是什么？
你希望他们感到害怕、共情、紧张，还是平静？

场景地点设定

行动发生在哪里？

这是室内镜头还是外景镜头？

是在 UFO 机库内，坐落在繁忙城市中央，还是一条荒凉的、树木成行的道路上？

主要角色设定

谁或什么是主角？

角色年龄 / 性别

他们多大？是年幼的孩子还是年长的人？

角色的年龄如何增强整个故事？

角色背景故事

他们是谁？在这次行动发生之前，他们做过什么？

你需要讲述多少性格/背景故事，才能帮助故事？

即使你认为它没什么用，也先写下来。也许它会激发另一个可以加入图像的性格特征，一点关于他们过去/当前生活的提示。

服装 / 道具

服装很重要（如果它是人类或外星角色）。衣服定义了许多关于角色自身和背景故事的信息。他们是刚下班回家吗？是一个喜欢摇滚乐的孩子吗？是一个梦想成为考古学家的女性吗？

道具对发展角色也很有帮助。首饰可以透露许多关于角色的信息。发型、妆容也是如此。靴子还是精致鞋子？眼镜可能传达某人很有知识气质？包袋暗示某人来自工作、学校，或有很多东西要携带。打开的包还是闭合的包？

衣服上的徽章和补丁也有助于强化角色。

一天中的时间

夜晚还是白天？晴天还是阴天？外星世界还是地球？

天气

天气也能增强图像的情绪冲击。雨会增加焦虑，而阳光带来希望。它是否是两者的混合？头顶乌云，却被充满希望的阳光束穿破。

画面灯光设计

灯光设计在讲述故事中始终至关重要，是一个巨大因素。试着用文字描述你的图像。它明亮且充满希望吗？是一道黑暗的单束光，还是充满色彩和亮度的壮丽发光景象？

主要光源是什么？还可以使用哪些其他光源来帮助推进故事？

色彩方案设计

同样非常重要。虽然这不是一段关于色彩理论的文字，但为你想追求的色彩方案设定一个起点非常关键。明亮且色彩丰富、充满希望和惊奇？深绿色或蓝色夜间色调？黑白黑色电影风格，或受某部影片的冷色调色板启发？

正如 Mike Cacciamani 在这篇课程中 另一课中提到的，明度和色相扮演着关键角色。

我坚定地相信，光与色彩可以成就或毁掉一张图像。强有力的着色和纹理定义整张图像的真实感。有方向的光和阴影引导观众的视线。而渲染会让你所有的选择真正活起来。

你也可能会发现自己可以继续为这组问题添加内容。它绝不是你需要问自己的唯一问题，但它是一个很好的起点。可以自由加入你自己的问题。

需要说明的是，你在创作过程中并不必须 100% 固守这些决定。

当你创作时，新想法总会浮现，但你会发现，在开始制作任何东西之前，至少先列出一些初步想法是有帮助的。

场景会/可能会演化。不要害怕在思考时重写部分内容。我再怎么强调这个过程的重要性都不为过。
