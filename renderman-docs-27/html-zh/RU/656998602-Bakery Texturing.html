贴图制作与 UV

这一部分的难点之一，是元素数量很多，以及如何优化它们以避免浪费太多时间。

第一个关键点是考虑复制。正如 Céline 在建模部分中提到的，很多布景都会复用，因此在复制之前，必须先在基础模型上完成 UV。一些复制后的元素随后会发生变形；要调整这一点，只需选择 UV Shells，并根据变形调整缩放，或者更简单地点击 “Unfold”，正确重绘 UV。

提示：让 UV 与木纹沟槽保持同一方向。这样在贴图制作时，就不必为了调整每块木板上的纹理而创建不同遮罩。

然后是如何排列 UV 的问题。由于元素很多，材质种类也很多，按材质分类似乎并不明智。因此，我们选择按对象或对象组来组织 UV，这样更容易找到对应内容。

使用 UDIM，还是不使用 UDIM

由于最终图像中的大多数对象都很小或距离较远，从分辨率角度看，单张 1024 或 512px 贴图已经绰绰有余。只有绿色木制家具这样的大型部件才需要多个 UDIM 来保持细节。

贴图制作

开始任何纹理工作之前，建立参考并仔细分析它们很重要。这会让你更清楚自己想要的渲染类型，从而更高效。

在我们的案例中，主要参考之一是 Pixar 电影 Luca（幸运的是！）。这部电影有趣之处在于，Pixar 并不是用传统污垢贴图把布景“弄脏”来让它们鲜活起来，而是使用一种图形化审美，强化漂亮、如画村庄中天然存在的变化。这种变化更像是环境中数百年人类选择造成的效果，而不只是衰败。

分析完成后，就可以进入贴图制作。在 Substance 中有许多贴图制作方法，其中最简单的一种是使用填充层，并给它应用基础颜色。然后我们叠加其他带有不同遮罩生成器的填充层来修改它。这提供了很高灵活性，因为颜色和遮罩仍然可以修改，不像直接绘制的图层那样具有破坏性。

通过这些填充层，我们最终生成其他贴图（Normal、Displacement、Roughness），并逐步调整以获得想要的材质。

变化、变化、还是变化...

以木材为例。这里我们试图再现随时间磨损的彩绘木材。因此，我从 Substance 的数据库中取一个木材基础材质开始，并快速调整颜色。随后为油漆应用一个初始填充层，再用其他色调做变化，每次略微改变粗糙度来增加凹凸感（油漆破损 = 更粗糙）。这些填充层被放在一个组中，并应用遮罩来模拟油漆破损，尤其是边缘周围。然后我逐渐添加之前识别出的细节，以丰富纹理。

提示：通常做变化时，我们倾向于只改变明度或饱和度，但也不要犹豫改变色相；这会给纹理增加更有趣的维度！

外观开发阶段的降噪

我大量使用了 RenderMan 机器学习降噪器。当我需要快速迭代时，它给了我很大帮助。

在外观开发期间，项目早期使用它非常关键，因为这有助于减少最终渲染时间，并让我们理解图像在 Denoiser 能清理之前需要收敛到什么程度。

使用 Substance 的优势之一，是你可以创建自己的材质，并直接添加到 smart materials（智能材质，基础材质数据库）中。因此，创建一个好的基础材质（木材、油漆、金属……）很重要；之后你可以选择自己的组，并点击 “Create smart material”。

剩下要做的，就是把它们应用到不同模型上，然后相应地修改滑块。这会节省你大量时间！

我在 Substance 中完成工作后，导出了纹理贴图，最后进入着色阶段！
