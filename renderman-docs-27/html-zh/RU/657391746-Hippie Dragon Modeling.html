Hippie Dragon 角色建模流程（Modeling）


角色基础形体搭建阶段（Blocking）

我一开始在 Maya 中使用 Maya library 里的基础 default Lizard base mesh。下一步是确定 front 和 back legs 的基本 shape 与 position，让它稍微更像 Salamander 和 Komodo Dragon 的 hybrid。对于 face，我尝试获得 Gecko 的 face shape，然后调整 nose 和 cheeks 附近的区域，以得到更 stylized 的 shape。

一旦为 body blocking 出基本 shape，我就用 planes blocking Leafy fins 和 Petal shaped ears。我没有在 Maya 中加入太多 details，因为我把主要 detailing 留给了 ZBrush。在 Maya 中完成 UVs 后，我直接进入 ZBrush sculpting。

角色高低频细节雕刻阶段（Sculpting）

我先在 low poly base 上 sculpting 较大的 details，例如通过增加厚度来 refine Arms 和 Legs、改变 Fingers 的 shape、雕刻 Neck 上垂下的 loose skin、加厚 chest area、削薄 hind legs 附近区域、让 legs 拥有较厚的 thighs 等。接着，我开始添加 Subdiv levels，加入 secondary forms，例如 folds、透过 skin 可见的一些 light muscle details，以及 nostrils 和 eyes 附近的 details。

sculpting 的最后一步是加入更细的 details，例如 thin wrinkles。我没有用 ZBrush，而是选择 Substance Painter 来添加 leathery skin details 和 leaves 上 veins 这样的 micro displacements。对于 leaves 上的 Veins，我使用来自 Color map 的 information 和一些 masks，而不是在 ZBrush 中直接 sculpting。

为了完成 texturing work，我把 model 保持在 T pose，这样大部分工作都可以利用 symmetry；我决定在 texturing 完成后再给 asset pose。
处理 creatures 时，我建议测试 model 在 SSS 方面的响应。我确保只在 material 中应用 SSS parameter，对 base model 从不同 angles 做一些 renders。结果满意后，我才进入 texturing。
