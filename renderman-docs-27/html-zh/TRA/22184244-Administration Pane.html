Administration Pane

Source: https://rmanwiki-27.pixar.com/space/TRA/22184244/Administration+Pane
Downloaded: 2026-06-15T03:49:12+00:00

点击 toolbar 上的 gears icon 可进入 Administration pane。此 pane 分为三个 tab。

 

Admin：tractor engine 的 configuration control。Limit Counters：当前 limit activity 的详细信息。User Connections：提供 connected user 的信息。
