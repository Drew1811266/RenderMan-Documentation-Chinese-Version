Task Graph Pane

Source: https://rmanwiki-27.pixar.com/space/TRA/22184226/Task+Graph+Pane
Downloaded: 2026-06-15T03:49:12+00:00

Task Graph pane 会以图形方式显示所选 job。job tree 中的每个 task node 都表示为一个矩形。box 会根据 status 使用不同颜色：

Green 绿色：活动

Red 红色：错误

蓝色轮廓：已选中

浅灰色：完成

深灰色：等待中

进度条

如果 application 以 RenderMan format 或 Alfred format 响应完成百分比，则会在活动的绿色条上绘制一条浅灰色条，用于表示正在运行 command 的完成百分比。注意，对于包含多个 command 的情况，此条可能会回到零，并循环多次。

日志输出指示器（Output）

如果正在运行的 task 生成了 log output，box 左侧边缘附近会显示一个浅蓝色 box。

预览图像指示器

如果 task definition 包含 preview 或 chaser command，则 task box 会在 node 右侧边缘显示一个放大镜图标。

 
 

任务图特殊功能
 

选择

点击任何 task box，都会让 Task Info pane 显示所选 task 的专属信息。该矩形会以浅蓝色轮廓绘制，表示它是当前选中的 task。
 

上下文菜单

在 task 上右键点击或 Control-click，会在 Task Graph window 中靠近所选 task 的位置弹出 context-sensitive menu。此 menu 可用于查看 task log；在适用时，还会提供一个 entry，允许在特定 task 因错误退出后重启该 task。也可以跳过 task，使 job 即使检测到错误也能继续执行。
 

日志

如果 log 可用，双击会在覆盖 joblist 的新 pane 中显示 task log，而不显示 context menu。
 

折叠作业图

parent node 可以被折叠，以提供其下 child node 的摘要视图。折叠后的 node 用椭圆形表示，而不是方形 node；其颜色代表 child node 的摘要。如果任何 child node 处于 error 状态，该 node 会显示为红色；如果所有 node 都 done，则显示为灰色；如果 node 正在 active，则显示为绿色。
