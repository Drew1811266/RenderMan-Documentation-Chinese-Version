本节介绍 Tractor 的 API 用法和作业脚本语法。

作业脚本
用于编写作业的基于 Tcl 的语法。
Job Author API
用于编写作业的 Python API。
Query API
用于查询和控制作业、任务、命令与 blade 的 Python API。
URL API
用于直接与 Engine 通信的 URL API。
搜索子句语法
Dashboard、tq 和查询 API 中用于搜索作业、任务、命令与 blade 的语法。
