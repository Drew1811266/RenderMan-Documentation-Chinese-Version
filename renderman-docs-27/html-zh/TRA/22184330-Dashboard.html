Tractor Dashboard 是 Tractor 的主要用户界面。它是一个在标准 Web 浏览器中运行的 Web 应用程序。当你将浏览器连接到 tractor-engine 时，会加载 Dashboard 的代码；tractor-engine 的行为类似普通 Web 服务器。

Dashboard 配置大体分为两类。一类是每用户的 Dashboard 偏好设置（每用户配置），它们通过 Dashboard 本身的 Preferences 选项卡设置；这些值存储在 tractor-engine 主机上的每用户区域中。另一类是 辅助应用 配置，通常既需要在每个用户机器上更改 Web 浏览器本身（某些情况下还包括操作系统应用程序注册表），也需要在 Tractor 的辅助配置文件中调整处理器设置。
