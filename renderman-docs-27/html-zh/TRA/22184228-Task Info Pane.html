任务信息窗格

Source: https://rmanwiki-27.pixar.com/space/TRA/22184228/Task+Info+Pane
Downloaded: 2026-06-15T03:49:12+00:00

Task Info 窗格会显示当前在 Task Graph 窗格中选中的任务的专属信息。

该窗格分为两个部分。顶部部分给出任务的全局信息，包括任务 ID 号、标题和当前状态。

底部部分给出该任务的命令信息。如果一个任务内有多个命令，可以使用窗格右侧的方向键显示该任务所有命令的信息。

蓝色的命令属性可编辑，可用于在作业提交后影响作业任务的调度。
 

可编辑字段

  • 命令 - 这是一个以空格分隔的字符串列表，表示已执行命令的参数向量（argv）。如果参数中需要包含空格，可以把该参数放在双引号中，也可以用反斜杠转义空格。
  • 限制标签 - limit tags 字段是一个逗号分隔的字符串列表，表示要关联到该命令的标签。例如，输入 "maya, shake" 会使命令处于激活状态时同时递增 "maya" 和 "shake" 限制计数器。
  • 服务键表达式 - 此服务键表达式是一个字符串，用于描述命令所需的资源。更多信息可在 Job Scripting Operators（作业脚本运算符） 页面中找到。
  • 环境键 - 这是一个逗号分隔的 envkey 名称列表，用于指定添加到启动链的环境处理器。
  • 最小/最大槽位数（Min/Max Slots） - 此项指定调度该命令所需的最小和最大槽位数。
