Task Info Pane

Source: https://rmanwiki-27.pixar.com/space/TRA/22184228/Task+Info+Pane
Downloaded: 2026-06-15T03:49:12+00:00

Task Info pane 会显示当前在 Task Graph pane 中选中的 task 的专属信息。

该 pane 分为两个 section。顶部 section 给出 task 的全局信息，包括 task id number、title 和 current state。

底部 section 给出该 task 的 command 信息。如果一个 task 内有多个 command，可以使用 pane 右侧的 arrow key 显示该 task 所有 command 的信息。

蓝色的 command attribute 可编辑，可用于在 job 提交后影响 job task 的调度。
 

可编辑字段

  • 命令 - 这是一个以空格分隔的字符串列表，表示已执行 command 的 argument vector（argv）。如果 argument 中需要包含空格，可以把该 argument 放在双引号中，也可以用反斜杠转义空格。
  • 限制标签 - limit tags 字段是一个逗号分隔的字符串列表，表示要关联到该 command 的 tag。例如，输入 "maya, shake" 会使 command 处于 active 时同时递增 "maya" 和 "shake" limit counter。
  • 服务键表达式 - 此 service key expression 是一个字符串，用于描述 command 所需的资源。更多信息可在 Job Scripting Operators 页面中找到。
  • 环境键 - 这是一个逗号分隔的 envkey 名称列表，用于指定添加到 launch chain 的 environment handler。
  • 最小/最大 Slot 数（Min/Max） - 此项指定调度该 command 所需的最小和最大 slot 数。
