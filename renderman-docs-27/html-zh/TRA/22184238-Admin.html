Admin 选项卡提供了几个用于管理 Engine 的功能。

点击日志图标会在新窗口中显示 Engine 日志文件的最后几千个字符。之后可以刷新此窗口，以获取当前日志末尾。用户可以编辑 URL 末尾的数字，以更改检索的日志文件数量。

配置部分有五个按钮，可提示 Engine 重新加载各种配置文件。用户必须具有管理员权限才能使用此功能。

派发部分有一个开关用于控制每个派发层级，也有一个主开关用于控制所有派发。用户必须具有管理员权限才能使用此功能。派发暂停后，所有 Dashboard 的工具栏中间会显示一条消息，让用户了解系统状态。暂停派发只会停止新任务的派发，不会终止正在运行的命令。
