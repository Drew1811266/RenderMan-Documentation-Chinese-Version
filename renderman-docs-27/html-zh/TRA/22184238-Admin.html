Admin tab 提供了几个用于管理 Engine 的功能。

点击 log icon 会在新窗口中显示 engine log file 的最后几千个字符。之后可以刷新此窗口，以获取当前 log 末尾。用户可以编辑 URL 末尾的数字，以更改检索的 log file 数量。

configuration section 有五个 button，可提示 engine reload 各种 configuration file。用户必须具有 administrator right 才能使用此功能。

dispatching section 有一个 switch 用于控制每个 dispatching tier，也有一个 master switch 用于控制所有 dispatching。用户必须具有 administrator right 才能使用此功能。dispatching 暂停后，所有 Dashboard 的 toolbar 中间会显示一条 message，让用户了解系统状态。暂停 dispatching 只会停止新 task 的 dispatching，不会终止正在运行的 command。
