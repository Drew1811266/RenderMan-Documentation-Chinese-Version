Query Pane

Downloaded: 2026-06-15T03:49:12+00:00

Query Pane 是用于搜索和显示 Tractor 实体的通用工具。
 

布局

页眉由实体下拉菜单、刷新按钮、搜索子句文本输入框和书签下拉菜单组成。面板的其余部分用于显示匹配的实体。
 

用法

使用搜索页面通常包含三个步骤。

首先，如果所需实体尚未选中，则从下拉菜单中选择它。可选实体包括 job、task、command、invocation 和 blade。 在搜索中包含归档项 复选框可以按需选中，以便同时显示匹配的已删除 job。

其次，从同一个下拉菜单中选择所需属性。鼠标悬停在属性上时会显示其描述。对于某些实体，可以选择关联实体的属性。例如，搜索 task 时可以选择 job owner。可以通过 shift-select 来切换某个实体的所有属性。如果没有为要查询的实体选择任何属性，则会检索全部属性。

最后，输入搜索子句。搜索子句必须遵循 Tractor 搜索子句语法。如果没有输入搜索子句，则返回所有实体。

可以通过按刷新按钮，或在搜索子句文本输入框中按 Enter 来执行搜索。

查询执行后，实体和属性会显示在面板主体的行和列中。选择某个属性的列标题，可按该属性对实体排序。拖动标题可以重新排列列，拖动标题之间的分隔条可以调整列大小。
 

书签

可以选择书签图标来显示下拉菜单，用于创建新书签或执行已有书签。书签会保存搜索字符串、实体、属性、列顺序和列宽。书签有名称，并会出现在以相关实体命名的分区下。

要添加书签，请按带加号的书签图标。随后会出现提示，要求输入书签标题，该标题会用于在菜单中表示此书签。

要移除书签，请按带减号的书签图标。该图标会变为红色，表示选择某个书签会将其移除。
 

到 Job 和 Task 的超链接

双击某个 job 会在 Job 面板中显示该 job 及其图表。Dashboard 必须能看到 jid 这个已显示属性，才能推断要显示哪个 job。

双击 task、command 或 invocation，也会在 Job 面板中显示该 job，并同时选中相关 task。Dashboard 必须能看到 jid 和 tid 这两个已显示属性，才能推断要显示哪个 job 和 task。
 

截断的结果集

注意，为避免 Dashboard 和 Engine 出现性能问题，任何搜索返回的实体数量都有一个站点可配置的上限，因为在大型数据集中，不受限制的查询结果可能包含数百万行。
