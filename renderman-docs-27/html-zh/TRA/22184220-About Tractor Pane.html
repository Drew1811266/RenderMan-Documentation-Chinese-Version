About Tractor Pane

Source: https://rmanwiki-27.pixar.com/space/TRA/22184220/About+Tractor+Pane
Downloaded: 2026-06-15T03:49:12+00:00

点击 toolbar 左侧的 tractor 图标会显示 About Tractor pane。

Engine Version 显示 Tractor version、build date、build ID 和 platform。与 Tractor Support 沟通以确认正在使用的具体软件版本时，这会很有用。

Dashboard Version 显示 Dashboard 的 Tractor version、build date 和 build ID；对于 custom installation，这些值可能不同于 Engine 的值。

Scheduler Mode 指示用于决定下一个应调度 command 的整体 rule system。

Thread Counts 显示专用于各种角色的 engine process thread 的数量和类型，例如处理传入 blade request、主动联系 blade（例如终止 active task），以及服务 Dashboard 和其他 client request。

Configs Last Reloaded 显示 任意 config file 最近一次被重新加载的时间和日期。

Engine Uptime 显示 engine 已运行的时长。

# Licenses 显示 Engine 消耗的 license 数量。这代表最大 concurrent active task 数。无论 active task 是否正在运行，Engine 都会消耗所显示数量的 license。对于运行多个 Engine 的 studio，记住这一点很重要。

# Active Commands 显示当前在所有 blade 上运行的 command 总数。
