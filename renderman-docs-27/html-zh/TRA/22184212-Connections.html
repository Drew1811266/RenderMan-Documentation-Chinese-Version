connections tab 提供已登录到 engine 的 users 或 scripts 的信息。提供的信息包括：

  • Login - 通过 UI 或 script 登录的 user name。

  • Host - script 或 user 登录来源 host 的 IP address。

  • Buffer - 等待发送给 user 的 subscription buffers 数量。

  • Notified - 自 user 或 script 上次连接并请求 mbox data 以来经过的 seconds 数。

    Buffer 和 Notified values 可用于帮助识别已停止请求 subscription data 的 UI 或 scripts。这可能表示存在问题。

  • Session - user 登录 engine 时返回的 session ID。当 user 或 script 登录 engine 时，会在 ${TractorDataDirectory}/users/USERNAME/ 中创建 SESSIONID directory。此 tab 中的 Session value 对应 user 登录时创建的 directory。
