对每个 Tractor 应用程序的详细讨论。

tractor-engine
用于手动启动 Engine 的命令行工具。
tractor-blade
用于手动启动 blade 的命令行工具。
Dashboard
用于查询和控制作业与 blade 的基于浏览器的 GUI。
tractor-spool
用于提交作业的命令行工具。
tq
用于查询和控制作业与 blade 的强大命令行工具。
NIMBY
用于管理本地主机参与调度情况的桌面 GUI。
tractor-dbctl
用于管理后端数据库的命令行工具。
