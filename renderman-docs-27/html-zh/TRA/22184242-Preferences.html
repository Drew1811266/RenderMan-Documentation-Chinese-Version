Preferences Panel

点击 username 后激活的 drop down menu 中的 "Preferences"，即可显示 dashboard preferences pane。

  • 默认 Job Graph 展开方式

    job graph 现在支持一种新的 display mode，可以以 collapsed format 显示 job graph。job graph 的 preference 定义 task graph 首次显示时的 initial display mode。两个选项分别是完全展开所有 task，或将所有 task 折叠到 job node。
     

  • Jobs 页面布局

    选择 jobs page 的 layout scheme。当前有两种 layout：一种让 job list 使用 browser window 的整个宽度；另一种把 task graph 放在 job list 右侧。
     

  • Job List 刷新间隔

    Job List 支持从 engine 自动 reload job data。reload interval 定义两次自动 reload job data 之间的时间（以秒为单位）。使用 filter 时，job reload 有助于让显示与 filter 更同步。
     

  • blade 列表刷新间隔

    定义 server list 从 engine 自动刷新的时间间隔（以秒为单位）。
     

  • 在 Job Graph 中显示 Log 指示器

    控制 job graph 中 log indicator 的可见性显示。
     

  • 在 Job Graph 中显示 Preview 指示器

    控制 job graph 中 preview indicator 的可见性显示。
     

  • 滚动条宽度

    控制 scrollbar 的宽度。
     

  • 确认重启

    控制重启 job 时是否显示 confirmation dialog。
     

  • 确认重试活动 Task

    控制重启 actively running task 时是否显示 confirmation dialog。
     

  • 确认重试所有错误

    控制请求对所选 job 中的 Retry All error task 执行操作时是否显示 confirmation dialog。
     

  • 确认跳过所有错误

    控制请求对所选 job 中的 Skip All error task 执行操作时是否显示 confirmation dialog。
     

  • 确认编辑

    控制编辑 job 时是否显示 confirmation dialog。
     

  • 确认中断

    控制中断 job 时是否显示 confirmation dialog。
     

  • 确认删除

    控制删除 job 时是否显示 confirmation dialog。
     

  • 确认 Command 编辑

    控制编辑 command attribute 时是否显示 confirmation dialog。
