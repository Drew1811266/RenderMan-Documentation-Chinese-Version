偏好设置面板

点击用户名打开下拉菜单，然后选择 "Preferences"，即可显示 Dashboard 的偏好设置面板。

  • 默认 Job Graph 展开方式

    Job Graph 现在支持一种新的显示模式，可将作业图以折叠格式显示。Job Graph 偏好设置用于定义 Task Graph 首次显示时的初始显示模式。两个选项分别是完全展开所有任务，或将所有任务折叠到作业节点。
     

  • Jobs 页面布局

    选择 Jobs 页面的布局方案。目前有两种布局：一种让作业列表使用浏览器窗口的整个宽度；另一种把 Task Graph 放在作业列表右侧。
     

  • Job List 刷新间隔

    Job List 支持从 Engine 自动重新加载作业数据。刷新间隔定义自动重新加载作业数据之间的时间（以秒为单位）。使用过滤器时，作业重新加载有助于让显示内容与过滤器保持同步。
     

  • Blade 列表刷新间隔

    定义服务器列表从 Engine 自动刷新的时间间隔（以秒为单位）。
     

  • 在 Job Graph 中显示日志指示器

    控制 Job Graph 中日志指示器的显示。
     

  • 在 Job Graph 中显示预览指示器

    控制 Job Graph 中预览指示器的显示。
     

  • 滚动条宽度

    控制滚动条的宽度。
     

  • 确认重启

    控制重启作业时是否显示确认对话框。
     

  • 确认重试活动任务

    控制重启正在运行的任务时是否显示确认对话框。
     

  • 确认重试所有错误

    控制请求对所选作业中的所有错误任务执行 Retry All 操作时，是否显示确认对话框。
     

  • 确认跳过所有错误

    控制请求对所选作业中的所有错误任务执行 Skip All 操作时，是否显示确认对话框。
     

  • 确认编辑

    控制编辑作业时是否显示确认对话框。
     

  • 确认中断

    控制中断作业时是否显示确认对话框。
     

  • 确认删除

    控制删除作业时是否显示确认对话框。
     

  • 确认命令编辑

    控制编辑命令属性时是否显示确认对话框。
