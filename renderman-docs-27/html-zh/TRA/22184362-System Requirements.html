Tractor Engine

中央作业队列，只安装在 一台 host 上。
C++、Python、Postgresql（bundled，随附）
兼容的操作系统（Compatible Operating Systems）：Linux 64-bit (gcc4.4)、macOS (12.x+)

Tractor Blade

Task execution server（任务执行服务器），安装在 render farm hosts 上。
Stand-alone pure Python package，兼容 py2.7 和 py3.6+
仅使用 stock Python library modules。
使用本地 site-installed Python（或 venv）运行，可作为 zipapp 或 pip installed wheel。
也包含在使用 bundled python 的完整 Tractor installer 中。
兼容的操作系统（Compatible Operating Systems）：Linux-64、macOS 12.x+ 和 Windows 7+。

Tractor Dashboard

基于 web-browser 的 user interface，用于 job 和 blade 信息与控制。
纯 JavaScript，由 tractor-engine 提供服务。不需要 platform-specific plugins。
兼容的 Web Browsers（Web 浏览器）：Chrome、Firefox、Safari 和 Edge。 

Tractor Job Submission and Query

用户 workstations 上的高级 command line tools 和 scripting APIs。
用于 Job Authoring、Submission 和 Query 的 scripting APIs，可下载 wheels 并通过 pip install 安装到本地 Python 2.7 或 3.6+。
Command-line submission 和 alternate API installer 可下载为面向 Python 2.7 或 3.6+ 的 stand-alone zipapp。
仅使用 stock Python library modules。
也包含在使用 bundled python 的完整 Tractor installer 中。
兼容的操作系统（Compatible Operating Systems）：Linux-64、macOS 12.x+ 和 Windows 7+。

RenderMan DCC Integrations

面向受支持 RenderMan DCC integrations 的 Job Authoring and Submission。
Job Authoring and Submission Python3 API modules 内置为
RenderMan 对 Houdini、Katana、Maya 和 Blender 集成的一部分。
