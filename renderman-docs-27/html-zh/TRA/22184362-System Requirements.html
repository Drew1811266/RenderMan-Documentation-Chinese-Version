Tractor Engine

中央作业队列，只安装在 一台 主机上。
C++、Python、PostgreSQL（随附）
兼容操作系统：Linux 64 位（gcc4.4）、macOS（12.x+）

Tractor Blade

任务执行服务器，安装在渲染农场主机上。
独立纯 Python 包，兼容 py2.7 和 py3.6+
仅使用标准 Python 库模块。
使用本地站点安装的 Python（或 venv）运行，可作为 zipapp 或通过 pip 安装的 wheel 运行。
也包含在使用随附 Python 的完整 Tractor 安装程序中。
兼容操作系统：Linux-64、macOS 12.x+ 和 Windows 7+。

Tractor Dashboard

基于 Web 浏览器的用户界面，用于作业和 Blade 信息与控制。
纯 JavaScript，由 tractor-engine 提供服务。不需要平台专用插件。
兼容 Web 浏览器：Chrome、Firefox、Safari 和 Edge。 

Tractor Job Submission and Query（作业提交与查询）

用户工作站上的高级命令行工具和脚本 API。
用于作业编写、提交和查询的脚本 API，可下载 wheel 并通过 pip install 安装到本地 Python 2.7 或 3.6+。
命令行提交和备用 API 安装程序可下载为面向 Python 2.7 或 3.6+ 的独立 zipapp。
仅使用标准 Python 库模块。
也包含在使用随附 Python 的完整 Tractor 安装程序中。
兼容操作系统：Linux-64、macOS 12.x+ 和 Windows 7+。

RenderMan DCC Integrations（DCC 集成）

面向受支持 RenderMan DCC 集成的作业编写与提交。
作业编写与提交 Python 3 API 模块内置为
RenderMan 对 Houdini、Katana、Maya 和 Blender 集成的一部分。
