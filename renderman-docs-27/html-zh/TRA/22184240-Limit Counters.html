Limit Counters

Downloaded: 2026-06-15T03:49:12+00:00

Limit Counters tab 显示 engine 中当前生效的 limit count 统计。

 

counters tab 可限制为只显示与正式 limit 相关的 count，这些 limit 在 limits.config 文件中定义。

Site 值是 limits file 中定义的 site-wide 或 global limit。在上方示例中可以看到定义了 5 个 global limit：limit10 == 10、limit15 == 15、limit5 == 5、limittest==50，并且 prman 当前不受限制。在这张图中，这些 limit 当前都未生效。

 

取消选中 Show only limits in config file checkbox 会显示所有当前 limit。除上述 config-defined limit 外，还有 engine 动态构建的 limit。这些 limit 来自：

  • 正在运行的 command 名称，例如 prman
  • 用户名称，例如 rdavis
  • service key，例如 pixarRender 或 RMSRibGen
  • job script 中定义的任何 tag，例如 intensive 或 Prod 6
