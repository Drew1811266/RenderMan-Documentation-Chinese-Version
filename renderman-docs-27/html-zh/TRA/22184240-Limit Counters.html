Limit Counters 标签页显示引擎中当前生效的限制计数统计。

 

该计数器标签页可以限制为只显示与正式限制相关的计数；这些限制在 limits.config 文件中定义。

Site 值是 limits 文件中定义的站点范围限制或全局限制。在上方示例中可以看到定义了 5 个全局限制：limit10 == 10、limit15 == 15、limit5 == 5、limittest==50，并且 prman 当前不受限制。在这张图中，这些限制当前都未生效。

 

取消选中 Show only limits in config file 复选框会显示所有当前限制。除上述配置定义的限制外，还有引擎动态构建的限制。这些限制来自：

  • 正在运行的命令名称，例如 prman
  • 用户名称，例如 rdavis
  • 服务 key，例如 pixarRender 或 RMSRibGen
  • 作业脚本中定义的任何标签，例如 intensive 或 Prod 6
