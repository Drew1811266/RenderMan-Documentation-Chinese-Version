Query API 变更

由于底层数据库 schema 发生变化， invocations() 函数现在会在查询 API 中以键 bladeid 返回 blade ID，而不是以键 blade 返回 blade 名称。需要 blade 名称的脚本可以在 columns 关键字参数中指定 Blade.name。

>>> invos = tq.invocations(search_clause, columns=["Blade.name"]) >>> print invos[0]["Blade.name"]
