Site Status Filter

Source: https://rmanwiki-27.pixar.com/space/TRA/22184288/Site+Status+Filter
Downloaded: 2026-06-15T03:49:12+00:00

概览

站点可以通过 TractorSiteStatusFilter.py 中定义的类扩展 blade 的功能。自定义内容包括：

  • 添加用于判断 blade 是否已准备好请求更多工作的指标，

  • 响应命令启动、启动失败、完成和错误等事件，以及

  • 响应命令输出中的模式，例如抑制冗长输出，或由于发出警告而提前终止命令。
     

安装

TractorSiteStatusFilter.py 必须安装在 blade 可见的目录中，理想情况下是 blade.config 中由 SiteModulesPath 定义的目录。

首先，从 Tractor 安装目录将 lib/python2.7/site-packages/tractor/apps/blade/TractorSiteStatusFilter.py 复制到 SiteModulesPath 目录。

请注意，需要重新加载 blade.config，blade 才能反映对 SiteModulesPath 或 TractorSiteStatusFilter.py 的更改。
 

场景：新指标

可以通过覆盖 FilterBasicState() 和 TestBasicState() 方法，使用新的指标来判断 blade 执行工作的能力。在下面的示例中， www.example.com 可能表示许可证服务器等资源；在 blade 考虑请求工作之前，需要能够访问该资源。

def FilterBasicState(self, stateDict, now):
    if stateDict is not None:
        rcode = os.system("ping -q -t 1 -c 1 www.example.com >& /dev/null")
        stateDict["server_reachable"] = (rcode == 0)
    self.super.FilterBasicState(stateDict, now)

def TestBasicState(self, stateDict, now):
    if stateDict is not None and not stateDict.get("server_reachable"):
        return False
    return self.super.TestBasicState(stateDict, now)

场景：记录错误

可以通过覆盖 SubprocessEnded() 来维护一份单独的出错进程日志。

def SubprocessEnded(self, cmd):
    if cmd.exitcode:
        with open("/tmp/errors.log", "a") as f:
            f.write("process %d errored with exit code %d\n" % (cmd.pid, cmd.exitcode))
    self.super.SubprocessEnded(cmd)

场景：提前终止进程

blade 会针对进程输出的每一行调用 FilterSubprocessOutputLine()。

在此示例中，任何发出以 R10007 开头的行的 prman 进程都会立即终止。可以修改该逻辑以检查其他应用程序的输出、查找其他模式并执行其他操作。请注意，未匹配的情况应交由超类的 FilterSubprocessOutputLine() 处理。

def FilterSubprocessOutputLine (self, cmd, textline):
    '''
    Return a 2-tuple giving a disposition code and a value.
    The pass-through result would be:  (self.TR_LOG_TEXT_EMIT, textline)
    '''
    if "prman" == cmd.app and textline.startswith("R10007"):
        # example:  immediately fail on prman warning number "R10007"
        return (self.TR_LOG_FATAL_CODE, 10007, textline)
    else:
        # otherwise defer to the usual built-in filters
        return self.super.FilterSubprocessOutputLine( cmd, textline )
