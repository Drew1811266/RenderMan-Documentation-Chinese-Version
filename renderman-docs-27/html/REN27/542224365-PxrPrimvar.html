PxrPrimvar

Source: https://rmanwiki-27.pixar.com/space/REN27/542224365/PxrPrimvar
Downloaded: 2026-06-15T03:49:12+00:00

Allows GPrim primitive variables (primvars) to be delivered through pattern graphs.

Input Parameters

Variable

Define the style of the variable

Variable Name

The name of the primitive variable.

Variable Type

The type of the primitive variable:

  • float
  • float2
  • color
  • point
  • vector
  • normal

Default Float

If there is an error when retrieving a variable of type 'float', this is the value that will be used

Verbosity

Diagnostic output levels for scene debugging

Output Parameters

result

The result as a color.
