PxrGamma

Source: https://rmanwiki-27.pixar.com/space/REN27/542223657/PxrGamma
Downloaded: 2026-06-15T03:49:12+00:00

Adjust the gamma of the input color by the given amount. Gamma affects midtones, pure white or black input colors are unaffected.

Input Parameters

Input Color

The color to have exposure adjustment. Plug a color pattern here.

Gamma

The amount of gamma to adjust the inputRGB.

Output Parameters

resultRGB

The clamped color result.

resultR

The R channel from the resultRGB output.

resultG

The G channel from the resultRGB output.

resultB

The B channel from the resultRGB output.
