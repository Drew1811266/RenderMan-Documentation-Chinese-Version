PxrChecker

Source: https://rmanwiki-27.pixar.com/space/REN27/542222962/PxrChecker
Downloaded: 2026-06-15T03:49:12+00:00

PxrChecker computes a simple checkerboard pattern.

Parameters

Color A and Color B

The two colors of the checkerboard squares.

Dimensions

The number of dimensions in which to compute the checker pattern

Manifold

The manifold over which to apply the noise (The default is ST).

You can connect a 3D or 2D manifold.
