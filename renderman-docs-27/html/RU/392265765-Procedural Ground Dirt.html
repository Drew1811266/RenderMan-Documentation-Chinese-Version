Procedural Ground Dirt

Source: https://rmanwiki-27.pixar.com/space/RU/392265765/Procedural+Ground+Dirt
Downloaded: 2026-06-15T03:49:12+00:00

Often, when a prop is left on the ground or a shelf or has a general buildup of directional grunge, the lower parts of the prop become dirtier than the top parts.

Imagine a fire hydrant that has been on a street corner for many years, a box left on the dirty floor, or, in this example, a display case that went on a messy shelf for a long time and has started to get dirtier on the bottom than on the top.

Of course, you can use Mari or Substance Painter to paint a dirt ground mask, but in this example, we want to look at how we can achieve this same look procedurally using RenderMan’s shading toolkit.

The advantage of doing this effect procedurally is that if we have a number of these trays scattered along a bench, on the floor, or on a display shelf, we don’t have to paint individual masks for each tray instance and get shading variation for each tray.

Let’s have a look at how this can be achieved in Blender. Of course, this effect can be equally achieved in other DCCs, but we’ll use Blender in this example.

Goal

We aim to create a b/w gradient that starts at the bottom of the prop and travels upwards.

We will then use this as a mask for layering in our dirt diffuse layer.

Steps

The shader has been simplified to demonstrate the effect by using a PxrConstant node, so we can easily preview the mask we are trying to create.

The key to achieving this effect lies in a few nodes. The first is a PxrRamp, and the second uses a PxrManifold3D node.

  • If we attach a PxrRamp to the EmitColor of our PxrConstant, this is what we get.

It’s not very useful or impressive at the moment, but wait…

The next step is to create a PxrManifold3D node and connect it to the manifold input of the PxrRamp node we just created.

  • Next, in the PxrRamp node, change the Ramp Type to T Ramp.

For the effect to be beneficial, we need to create a coordinate system in Blender so we can control the gradient's direction.

  • Create an Empty-> Plain Axis

  • Give the Plain Axis node a useful name, like GroundDirt

  • Under RenderMan Geometry, turn on Export as CoordSys

Note : In Maya, a coordinate system can be created using a 3D Placement node

Once done, we need to tell the PxrManifold3D node that we want to use the Axis controller as our coordinate system.

  • In the PxrManifold3D node under Coordinate System, enter the name of your newly created Axis Node. In this case, it’s called GroundDirt

Our gradient is now looking much more useful.

We can control the effect by adjusting the ramp knots to set where we want the dirt to start and end on our prop.

We can further refine the placement by moving the GroundDirt Axis up and down.

GroundDirt_MoveAxis.mp4

We now have a ground-dirt mask that is much more useful … but we can push it even further by breaking the ramp's smoothness with a dirt texture map.

There are a number of different ways we can use this texture map to breakup our ramp, but one such way is to use a PxrBlend node to blend the dirt spots texture above with our ground dirt ramp.

As we are trying to be as procedural as possible, we want to use a PxrRoundCube & PxrMultiTexture setup to tri-planar our dirt spot texture onto our prop. This approach allows us to randomly offset, rotate, and tile our dirt spots onto other instances.

  • Using the PxrBlend node, we then blend the tri-planar dirt spot texture over our ground dirt ramp using a Multiply ink mode. You can also play with different ink modes to give you different results.

  • If we need the effect of the mask to be more pronounced, we can adjust the mask, again, in a number of ways, but in this example, we’ll use the PxrSplineMap node and curves to adjust the mask.

Nice!

Now, we can use this mask as a layer mix for a procedural Lama or PxrSurfaceLayer shading setup.

Note, once we start to use this mask in our actual shader you might find you need to further adjust the brightness and contrast of the mask to dial in the look you are going for.

Further Practice

  1. Try using other textures or a PxrFractal node to further break up the ground dirt gradient

  2. Use a PxrDirt node to add crevice dirt into the mask (as the above setup demonstrates)

  3. Use a PxrDirt node to remove crevice dirt from the interior of objects that wouldn’t always be affected by ground dirt.
