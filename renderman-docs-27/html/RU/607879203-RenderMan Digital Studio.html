Project: RenderMan Digital Studio

Created by Pixar Artist: Eugene Riecansky

Features: ACES, USD, PxrSurface

The scene comes with a variety of studio coves. Plus, a fully stocked asset library with a wide range of studio-inspired assets and kits, kindly supplied to us by RenderMan's in-house rental company, Point Reyes Electric, complete with a variety of model and shading variants.

You can also use the assets in your own projects, rather than having to use the provided studio scene

All ready for you to create your studio environments in XPU!

Maya

The asset pack is located in Maya’s Content Browser.

To install the assets, open the Content Browser, select the Favourites tab, click Add New, and select the Assets folder in the Maya download. All the Studio assets will now load into the content browser.

From here, you can drag and drop the assets into the scene.

To ensure all textures are loaded correctly, ensure that your project is set correctly.

Solaris / USD / Katana

To use in Solaris, add the Assets folder in the downloads to your Asset gallery, then use them with the layout node. Or you can just load them individually using a file or assetreference node.

If you want to use the assets in Katana, you can load them in via a USDIN node and create your studio setups.

Blender

To use in Blender, just add the Assets folder to your file path structure in the Blender Preferences, and you can then see all the props in the Asset Library panel.
