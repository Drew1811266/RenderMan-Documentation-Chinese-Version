PxrDirt & PxrCurvature

Two essential and commonly used geometry patterns in procedural shading are Ambient Occlusion and Curvature. RenderMan can render both of these features, which can be used either in a procedural shading network or baked out and re-read by a PxrTexture node.

PxrDirt (Ambient Occlusion)

The PxrDirt node renders local geometric occlusion.

This is useful for applying shading effects, such as dirt, rust, and dust, into the nooks and crevices of your model.

PxrCurvature

The PxrCurvature node is a practical, straightforward way to create edge effects and curvature-based results, such as edge wear, edge breakup, and edge variation on plastics and objects in contact with other surfaces.

One practical example of these two nodes is using a PxrCurvature node as a layer mask between two layers, such as a base metal and a paint layer on top, as seen in the Midnight Rebel scene.

The PxrDirt node can then be used as a layer mask to add dust to the paint layer.
