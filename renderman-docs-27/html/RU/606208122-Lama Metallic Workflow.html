Lama Metallic Workflow

Downloaded: 2026-06-15T03:49:12+00:00

Whether you’re using PBR maps from an online material library or texturing software like Substance Painter. RenderMan ships with a handy node called PxrMetallicWorkflow, which can be easily used with your PBR maps and a Lama workflow.

This lesson will guide you through the process.

The maps used in this lesson were kindly provided by our rendering friends at Greyscalegorilla, and the example used is Copper Corroded Heavy 01

As the Eagle model doesn’t have any UVs, we’re using a triplanar approach to mapping the textures onto the model via the PxrMultiTexture node

If your model has UVs, you can use a PxrTexture node instead of the PxrMultiTexture node.

Note: Currently, exporting a Lama material network from Substance Painter requires updating the normal connections. You can easily do this by selecting Update Lama Nodes from the RenderMan menu in your DCC
