PxrSurface - Metallic Workflow

Source: https://rmanwiki-27.pixar.com/space/RU/619741326/PxrSurface+-+Metallic+Workflow
Downloaded: 2026-06-15T03:49:12+00:00

While we don’t use a PBR Metallic workflow at Pixar, it remains a fairly typical workflow among artists, whether you’re using PBR maps from an online material library or texturing software like Substance Painter. Luckily, RenderMan ships with a handy node called PxrMetallicWorkflow, which can be easily used with your PBR maps and a PxrSurface.

This lesson will guide you through the process.

The maps used in this lesson were kindly provided by our rendering friends at Greyscalegorilla, and the example used is Copper Corroded Heavy 01

As the Eagle model doesn’t have any UVs, we’re using a triplanar approach to mapping the textures onto the model via the PxrMultiTexture node

If your model has UVs, you can use a PxrTexture node instead of the PxrMultiTexture node.

The setup is as follows:

  1. Create a PxrSurface

  2. Create a PxrMetallicWorkflow node

  3. Connect the resultdiffuse from the PxrMetallicWorkflow to the PxrSurface diffusecolor

  4. Use a PxrTexture for the diffuse map (or in this case a PxrMultiTexture node)

  5. Connect the diffuse texture to the basecolor input of the PxrMetallicWorkflow node.

  6. Load the Metallic texture map.

  7. Connect the resultr from the metallic map into the metallic input of the PxrMetallicWorkflow node.

  8. Connect the resultSpecularEdgeRGB from the PxrMetallicWorkflow into the specularEdgeColor of the PxrSurface.

  9. Connect the resultSpecularFaceRGB from the PxrMetallicWorkflow into the specularFaceColor of the PxrSurface.

  10. Load a Roughness map.

  11. Connect the resultr from the roughness map into the specularRoughness input of the PxrSurface

  12. Load your Normal map using a PxrNormalMap node.

  13. Connect the resultN from the PxrNormalMap node into the bumpNormal input of the PxrSurface.

  14. Hit Render and bathe in the PBR glory!

Further Refinements

  • In the example above, I also used a Houdini Dust HDA called EasyDust from Timothee Maron

  • Use a PxrRemap, PxrThreshold, or PxrSplineMap node to correct the roughness map to your lookdev needs
