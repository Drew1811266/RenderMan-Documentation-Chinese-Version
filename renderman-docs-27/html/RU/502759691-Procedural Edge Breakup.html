Procedural Edge Breakup

Source: https://rmanwiki-27.pixar.com/space/RU/502759691/Procedural+Edge+Breakup
Downloaded: 2026-06-15T03:49:12+00:00

Adding edge breakup to props is a common practice, often achieved by painting masks in texturing software like Adobe Painter or The Foundry’s Mari.

At Pixar, however, we tend to take a procedural approach to edge breakup.

This allows retaining creative control, such as quickly reacting to the director's notes without repainting and exporting edge masks, and creating shading variants on props, from a new-looking prop to various stages of wear and damage.

Below is a quick guide to procedural edge masks.

Step One

Let’s start by adding a PxrCurvature node with the default settings to our geometry. In this example, we’re going to use the RenderMan University shield.

As you can see from the first render, it’s not very impressive … but worry not, in a few simple steps we’ll vastly improve the look.

Step Two

The next step is to adjust some of the default settings.

The first thing we want to do is make the curvature brighter, so increasing the Output Gain from 1 to 10 immediately produces a stronger signal.

Increasing the number of samples from 4 to 8 (or 16) will reduce some of the grain in the curvature, but will slightly increase the render time.

Next, we'll load a map to start breaking up the curvature. In this example, I’ve used grunge_04.tex from our Grunge Texture Vol 1 Pack, and mapped using a PxrRoundCube+PxrMultiTexture (Triplanar) approach.

To add a bit more contrast to the grunge map, a PxrThreshold node is used to achieve this effect.

Using a PxrBlend node, we can subtract the grunge map from the curvature by setting the mode to Subtract.

Step Three

Now we’re starting to get somewhere useful for use as a mask between two layers or to add bump/displacement to the edges of the shield.

The last step in this simple example is to increase the brightness of our result, and here we’re using a PxrSplineMap node.

Great, now we have a mask that’s more useful for our purposes, but it’s worth noting that this example is just a straightforward approach.

Refine Details

To increase the realism, you could try subtracting the curvature with multiple grunge maps, adding more detail, and breaking up your mask.
