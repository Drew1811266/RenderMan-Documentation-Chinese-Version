In rendering, you’ll use texture maps of all kinds, constantly.

This set of lessons will guide you through how RenderMan handles textures, how to add randomization to those textures, and if you’re painting your textures in Mari or Substance Painter, you’ll learn how to correctly export and import those into RenderMan so you get visual parity when rendering.
