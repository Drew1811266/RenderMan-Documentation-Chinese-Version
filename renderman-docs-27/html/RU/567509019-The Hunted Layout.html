Once I had each initial proxy version of the props as USD files, I was able to begin working on the scene's layout.

HuntedLayout.mov

Houdini and Solaris are perfect for both single set-dressing assets and scattering large numbers of instances.

I implemented both workflows. Scattering and instancing techniques for the grass, rocks, general plants, and background trees.

I hand-placed hero objects via the layout node, such as the powerbox, railroad sign, and mailbox. I also hand-placed grass, along with foreground trees and rocks, to ensure the scene's composition was as I wanted it. Scattering is perfect for instancing objects when you don’t mind too much about the creative composition of that scattering. However, there are many times when you’ll need to rely on hand-placing or moving scattered objects to suit the shot and story.

Using the latest advances in XPU in R27, I was able to run an interactive viewport in Solaris with a simple skydome to allow me to get a fast and interactive layout session. Add in the power of the interactive denoiser, and you have an interactive toolkit that eats layout for breakfast.

stagemanager.mov

One extra tip is when doing layout, and you already have many textures applied to the assets, use the Occlusion Integrator in RenderMan. This will give you a very fast occlusion IPR session without having to override all your shaders to a simple grey material.
