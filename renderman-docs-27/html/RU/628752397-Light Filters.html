Light Filters

Downloaded: 2026-06-15T03:49:12+00:00

Light Filters are used to craft and refine your lighting by applying filters to those lights.

In the real world, this is often achieved by using gobos, gels, blockers, and reflectors to alter the color, quality, and amount of light in a scene.

RenderMan's light filters are a powerful and flexible way to achieve the same (and more) when lighting your scenes. 

Below is an overview of the light filters you can use to take your lighting to the next level.

Barn Door Light Filter

The PxrBarnLightFilter allows us to create physically accurate window barns to simulate the real set lighting with correct shadowing. Its other usage includes controlling bounce lights in a large scene. In addition to the physical mode, it also provides an analytic mode.

Cookie Light Filter

The PxrCookieLightFilter projects a painted texture in front of the light, often used on-set to fake windows, trees, and projections.

Rod Light Filter

The PxrRodLightFilter uses a "rod" like object to block light. The blocker can be shaped into an irregular shape and is then placed next to the object where you want to block the light. This is similar to the above PxrBlockerLightFilter, but with more controls and flexibility.

Intensity Light Filter

The PxrIntMultLightFilter is a light filter that allows you to multiply the intensity/exposure of the light. This is very useful when you want to isolate a particular asset(s) from the rest of the scene that has different intensity/exposure. This is via linking the objects to the PxrIntMultLightFilter.

Ramp Light Filter

The PxrRampLightFilter uses a ramp to control the light. This allows more complex control of color, intensity, and more.
