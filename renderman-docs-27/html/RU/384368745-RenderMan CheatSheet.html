RenderMan CheatSheet

Whether you’re new to rendering or coming to RenderMan from another renderer, this cheatsheet will help you get up to speed with the many technical and creative nodes that RenderMan artists have in their shading toolkit.

Shaders

Action

RenderMan Node

Uses

Simple Metallic PBR shader

PxrDisneyBsdf

All CG animations made at Walt Disney Animation Studios use this material model

I want to use the shader used at Pixar Animation Studios

PxrSurface

This is the same material model used in Pixar Animation Studios' feature animation.

I want to layer up multiple PxrSurfaces

PxrLayerSurface

Same as the PxrSurface, but allows for layering driven by layer masks

Use ILMs production layering shaders

LamaSurface

MaterialX Lama is a state of the art material layering system developed at Industrial Light & Magic that introduces a modular approach to building material networks and includes recent shading R&D work on energy conservation and dispersion. 

I just need a constant material

PxrConstant

A basic constant shader.

A shader to apply to volumes, clouds, pyro, and VDBs

PxrVolume

PxrVolume is a material that can be used to render volumetric effects which can vary wildly in complexity

I want to use a hair shader used by Pixar!

PxrMarschnerHair

PxrMarschner is a material designed to render realistic hair, fur, and fibers.

I want to use a Lama hair shader.

LamaHairChiang

LamaHairChiang is a layerable version of the Pixar Chiang hair shader used in production at Pixar.

I want to add displacement to my material

PxrDisplace

Add this node to the shader output to add displacement to your material

Bump & Displacement

Action

RenderMan Node

Uses

Add a bump to the material network

PxrBumpMixer

This node provides a modern way to generate and layer bump maps

Adjust displacement texture values

PxrDispTransform

This pattern transforms the input displacement values to a customized range of values for the input to PxrDisplace.

Create a BumpRoughness map from a b/w bump map

PxrBumpRoughness

With this pattern, RenderMan automatically switches from a bump map to a roughness value depending on the size of a feature in the camera,

Some nodes don’t have NG out, but I want them to contribute to the bump

PxrRGBtoNG

There might be times when you want to create a bump map input to the PxrBumpMixer, but the last node is missing NG OUT. Use this node to convert RGB to NG

Edge Detection

Action

RenderMan Node

Uses

I want to create procedural edge masks and breakup

PxrCurvature

This is a practical and straightforward pattern for creating edge effects and curvature-based results like wear on edges and objects in contact with other surfaces.

I want to add dust and dirt into the crevices.

PxrDirt

PxrDirt renders local geometric occlusion. This is useful for applying shading effects to nooks and crevices.

I want Wireframe renders

PxrWireframe

This pattern is meant to draw geometry wireframes as a pattern assignment.

Detect the facing ratio of geometry

PxrFacingRatio

Useful for all sorts, including procedural edge dirt and crafting light bulb emission.

Lights

Action

RenderMan Node

Uses

Use a HDRI map with RenderMan

PxrDomeLight

In much of visual effects and animation work, scenes are primarily lit through a High Dynamic Range Image or HDRI.

Use a dynamic Sun / Sky environment

PxrEnvDayLight

Daylight simulations are helpful when you want a quick and easy setup for exterior lighting.

Rect or Area light

PxrRectLight

This one-sided area light simulates a rectangular-shaped light source.

Disc-shaped Light

PxrDiscLight

This one-sided area light simulates a disk-shaped light source.

Sphere-shaped Light

PxrSphereLight

This area light simulates point and spherical-shaped light sources, such as light bulbs, headlamps, and more.

Cylinder or tube-shaped light

PxrCylinderLight

Its usage includes illuminating objects, simulating commercial lighting used in many buildings, linear lights, fluorescent lights, and light panels. This is also used for making lightsabers.

Distant, infinite, or sun/moon directional light

PxrDistantLight

This infinite light simulates a bright, distant light source, like the sun or the moon. This light bathes the entire scene and treats the light as if the rays are parallel by default.

Portal Light

PxrPortalLight

Since lighting interiors with little or no direct lighting is difficult for a renderer to resolve, portal lights help direct the renderer's attention to the exterior source of light.

Create AOVs for shot-specific masks

PxrAovLight

Allows a lighting artist to output an AOV (Area of Interest) mask. Since it is a "light", we can use light linking as well as light filter(s) to modify the output mask.

Light Filters

Action

RenderMan Node

Uses

Add advanced spotlight and barn door features to a light

PxrBarnLightFilter

The PxrBarnLightFilter enables us to create physically accurate window barns that simulate real set lighting with correct shadowing.

Add a Cookie or gobo texture to the light, like in a projector or pretend window shadows

PxrCookieLightFilter

PxrCookieLightFilter projects a painted texture in front of the light.

Block light from the selected light in certain areas of your scene

PxrRodLightFilter

PxrRodLightFilter uses a "rod" like object to block light.

Adjust the exposure of selected light in certain areas of your scene

PxrIntMultiLightFilter

This is very useful when you want to isolate a particular asset(s) from the rest of the scene that has different intensity/exposure.
