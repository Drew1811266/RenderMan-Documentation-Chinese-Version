Analytic Lights

RenderMan includes a set of built-in basic geometric lights. These specialized shapes - DomeLight, DistantLight, RectLight, DiscLight, and SphereLight (particularly the latter two) - can provide superior sampling behavior, compared with arbitrary geometry.

Mesh Lights

These are great for using arbitrary shapes in lighting. These consume more memory since they are geometry and complex shapes, which may increase noise. These can make use of visibility like any other object: camera, shadow/transmission, and indirect visibility.

Emissive Surfaces

These typically use a constant bxdf or "glow" parameter to indirectly light a scene. This is inefficient and should usually be avoided, but it may be helpful for texture-mapping "lights" onto futuristic objects and panels with low emission values.

If you want to do this efficiently, try using a textured mesh light
