General information about integrators in RenderMan:Integrators

To add an integrator simply add one of the dedicated LOP node to create the corresponding USD integrator primitive.

The previous ways (either via the dedicated tab on the Render Settings LOP or via the PxrIntegrator LOP) to add an integrator, while still supported, are deprecated.

Then assign it to the render settings primitive:

  • Houdini 20.5 and below: use a PxrAssign LOP (or the PxrAssignIntegrator alias) to assign an integrator primitive to a render settings primitive. The render settings primitive must exist before assignment.

  • Houdini 21.0 and above: while the PxrAssign LOP workflow is still valid, it is also possible to assign directly an integrator on the Render Settings LOP.

If the assignment was properly done, in the Scene Graph, on the Render Settings primitive, the integrator primitive should be listed on the ri:integrator attribute in H21 and above or, on the outputs:ri:integrator attribute for H20.5 and below.
