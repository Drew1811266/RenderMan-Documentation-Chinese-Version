The following workflow is for H21.0 and above

Since it is difficult for a renderer to resolve interiors with little or no direct lighting, portal lights help direct the renderer's attention to the exterior source of light.

To set up Portal Lights in Solaris requires the portallight LOP to be in the Solaris network before the domelight it will be attached to.
