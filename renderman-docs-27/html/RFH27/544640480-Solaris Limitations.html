Known Limitations

Geometry

  • NURBS and Implicits not supported

Shading

  • For best performance, you should manually pre-convert your textures via txmake or use the Texture Manager

  • The dynamic and uniform array node is not expected to work in Solaris

Image Tool

  • Rendering to IT with ACES currently needs the env variable IT_OCIOV1 to be set to your ACES config.ocio

  • Crop tool and image selector is not supported

OSX

  • M1 on OSX is not currently supported. 
    You can run via Rosetta, but be warned, performance won't be as good until native support is added.
