You can find out all about Light Filters in the Renderman documentation.

Adding Light Filter

To add a light filter, in your OBJ pane, select the light filter in the TAB menu.  

Parent the light filter OBJ to the light OBJ if you want the filter to inherit the transformation from the light.

We can translate the light filter and scale it.  It is best to translate the light filter in Z away from the light to avoid seeing the edge of the light filter.

Set Light Filter

Drag your light filter VOP to the Light Filter shaders of your light.
