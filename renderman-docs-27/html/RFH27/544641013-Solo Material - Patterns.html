Solo Material / Patterns

Source: https://rmanwiki-27.pixar.com/space/RFH27/544641013/Solo+Material+%2F+Patterns
Downloaded: 2026-06-15T03:49:12+00:00


Artists can solo shading nodes and sections of material networks using the Visualizer node, to aid in lookdev and artistic control. 

This is especially useful when dialing in the look of a layer mask, adjusting a texture map, or getting those fingerprint grunge maps looking just right on toy ice cream trucks!

To solo a node or incoming shading network, simply select the node you wish to solo and press the X key. 
Doing so, will insert a visualize node and automatically connect the output of the selected node to the input of the visualize node. 

To 'un-solo' a node, cut the input to the visualize node and continue working on the lookdev.  

Note: Due to some limitations within Solaris, there are some use cases where the visualizer node doesn't work, but for the most part, you can enjoy a powerful and creative lookdev workflow!
