Interactive Denoiser in Solaris

Downloaded: 2026-06-15T03:49:12+00:00

Please note that enabling the interactive denoiser on the RenderMan Render Settings LOP will automatically add the requested AOVs for denoising if those are not already present.

Cheap First Pass
If checked, the first pass will use a cheaper (slightly faster but lower quality) heuristic.

This can be useful if rendering something that is converging very quickly and you want to prioritize getting a denoised result faster.

Min Samples
The minimum number of average samples per bucket before the interactive denoiser runs for the first time.  Changing this preference requires the render to be restarted for this option to be respected

Interval
The time interval in between denoise runs (in seconds)

Note : To enhance interactivity when working on asset lookdev, try increasing the min samples to 8-10 and reduce back down to 2 - 4 when working on a larger scene.
