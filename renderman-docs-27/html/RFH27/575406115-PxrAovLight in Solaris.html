PxrAovLight in Solaris

Source: https://rmanwiki-27.pixar.com/space/RFH27/575406115/PxrAovLight+in+Solaris
Downloaded: 2026-06-15T03:49:12+00:00

There is a dedicated PxrAovLight LOP. It is up to the user to create the Render Var / AOV used by the PxrAovLight. To do so, simply use a Render Var LOP and place it on the first input of the RenderMan Render Settings LOP (similar info at: Setting up LPEs in Solaris ).

It is possible to use light filters (Light Filters in Solaris) and light linking (Light Linking in Solaris) on the PxrAovLight.

More information about PxrAovLight usage: PxrAOVLight in Maya
