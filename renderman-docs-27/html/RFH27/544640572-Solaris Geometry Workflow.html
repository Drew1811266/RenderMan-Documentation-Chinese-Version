Solaris Geometry Workflow

Downloaded: 2026-06-15T03:49:12+00:00

To allow RenderMan to control your geometry settings, these can be found in the RenderMan tab inside the RenderGeometrySettings LOP

(RfH needs to be installed)

Choose Set or Create or Set if Exists to create or change an opinion on the geometry attribute or choose Block to disable previous opinions on the attribute.   All available parameters are listed but none are set by default.

These are the same settings that can be found in the RenderMan spare parameter tab on object nodes.
