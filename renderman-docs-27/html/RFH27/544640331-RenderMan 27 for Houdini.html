RenderMan for Houdini (RfH) is an integrated interface to RenderMan inside Houdini. 

Installation

RenderMan for Houdini requires the installation of RenderMan Pro Server as well as the RfH plugin in order to function correctly. The RenderMan installer does this automatically.


To complete the setup you should follow the directions for Installation of RenderMan for Houdini.
