RenderMan Render Settings

In USD, the expectation is to describe AOVs using Render Var primitives, attach those to a Render Product primitive where the output image filepath as well as the display driver is also set. The Render Product primitive must then be attached to a Render Settings primitive. Note that the Render Settings primitive can also be attached an integrator or, display and sample filters. For getting starting quickly, Pixar provides the RenderMan Render Settings LOP node that does all the primitive creations and connections mentioned above. The RenderMan Render Settings LOP internally uses the RenderMan Render Vars LOP which quickly define the most common RenderMan AOVs. The RenderMan Render Vars LOP can also be used on its own if one decides to set the Render Product and Render Settings primitives using directly the SideFx's LOP nodes instead of the RenderMan Render Settings LOP.

Image Settings

Render settings for the output image, such as resolution and which camera to render from.

Render Settings
