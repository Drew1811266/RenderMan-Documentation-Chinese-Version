The RenderMan Digital Studio ships with a number of files and these mimic a small pipeline workflow.


RenderManStudio_Assembly_v01.hip
This scene lays out all the assets in an easy-to-see formation.  You can explore the shading and model variants that are in some of the assets (FlightCases, SpotLights, HardCases, PanelLights)

If you simply want to jump in and play with the studio setup, you don't have to worry about this file, but this stage and file are included for your convenience.


RenderManStudio_Layout_v01.hip
This is where the scene layout happens.  Re-arrange the entire scene as you wish and add some of your own USD assets!


RenderManStudio_Lighting_v01.hip
Taking the USD file from the layout stage, this is where you can play around with the lighting we have provided and/or add your own to create bespoke film studio scenes that fit your creative needs.
