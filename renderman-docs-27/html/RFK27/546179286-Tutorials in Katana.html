The RenderMan for Katana package contains a collection of example scenes which demonstrate a number of rendering workflows, such as:

  • Baking

  • Bokeh effect

  • Denoise

  • Holdouts

  • Instancing

  • Layered Materials

  • AOVs

  • Mesh Lights

  • OSL Shading

  • Texture Projection

  • Volumes

These scenes are found in the Examples directory:

$KATANA_RESOURCES/Examples/katana_files
