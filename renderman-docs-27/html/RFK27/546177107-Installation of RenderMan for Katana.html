Katana Support

See Installation and Licensing for the list of Katana versions that are supported.

Installation and Licensing

RenderMan for Katana (RfK) can either be installed in one of two ways:

  1. Use the RenderMan Installer which manages the downloading, installation, and licensing of RenderMan Pro Server (the actual renderer) along with RenderMan for Katana. 

  2. Download the RenderMan for Katana and RenderMan Pro Server packages from the forum Product Downloads page. The RPMs can be installed by hand.

See the RenderMan Installation and Licensing documentation for a more detailed explanation.

If you have 3Delight installed, you'll need to set your DEFAULT_RENDERER environment variable to prman or change the renderer parameter in RenderSettings to prman.
