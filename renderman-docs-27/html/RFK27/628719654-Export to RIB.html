Export to RIB

Source: https://rmanwiki-27.pixar.com/space/RFK27/628719654/Export+to+RIB
Downloaded: 2026-06-15T03:49:12+00:00

You can export some or all of the Katana scene to a RenderMan RIB file from within the Node Graph pane. Right-click any node to open the context menu, select the Debug menu, then choose “Open prman .rib Output in External Editor.” RfK builds the scene up to that node, writes it to a RIB file, and Katana opens it in the external text editor set in your preferences.

This Debug RIB Output will match the disk render version of the scene.

To check a scene's state for live renders, there is a PrmanGlobalStatements option that will write out a RIB before any render. The option is in the plugin section under “Debug”: plugin->Debug->debugRibFile.
