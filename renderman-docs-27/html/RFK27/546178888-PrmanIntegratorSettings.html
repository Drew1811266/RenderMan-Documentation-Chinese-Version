PrmanIntegratorSettings

Source: https://rmanwiki-27.pixar.com/space/RFK27/546178888/PrmanIntegratorSettings
Downloaded: 2026-06-15T03:49:12+00:00

This node allows the user to select the integrator needed for rendering their scene. Once chosen, it populates the UI with the relevant controls for the chosen integrator. The RenderMan documentation contains the explanation of controls for the Integrators as well as their best usage scenario.
