This section covers the usage of Tractor's APIs and job scripting syntax.

Job Scripting
The Tcl-based syntax for authoring jobs.
Job Author API
The Python API for authoring jobs.
Query API
The Python API for query and control of jobs, tasks, commands, and blades.
URL API
The URL API for communicating directly with the engine.
Search Clause Syntax
The syntax used in the Dashboard, tq, and query API for searching for jobs, tasks, commands, and blades.
