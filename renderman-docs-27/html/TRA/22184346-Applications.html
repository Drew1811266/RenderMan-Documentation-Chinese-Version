Detailed discussion of each Tractor application.

tractor-engine
The command-line tool for manually starting the engine.
tractor-blade
The command-line tool for manually starting a blade.
Dashboard
The browser-based GUI for query and control of jobs and blades.
tractor-spool
The command-line tool for submitting a job.
tq
The powerful command-line tool for query and control of jobs and blades.
NIMBY
The desktop GUI for managing the local host's participation in scheduling.
tractor-dbctl
The command-line tool for managing the backend database.
