Limit Counters

Source: https://rmanwiki-27.pixar.com/space/TRA/22184240/Limit+Counters
Downloaded: 2026-06-15T03:49:12+00:00

The Limit Counters tab displays the the tally of the current limit counts in effect in the engine.

 

The counters tab can be limited to only displaying counts associated with formal limits, as defined in the limits.config file.

The Site value is the site-wide or global limit as defined in the limits file. You'll see in the example above there are 5 defined global limits: limit10 == 10, limit15 == 15, limit5 == 5, limittest==50, and prman is currently unlimited. In this picture none of these limits are currently in effect.

 

Clicking off the Show only limits in config file checkbox will present ALL the current limits. Besides the config-defined limits, as mentioned above, there are limits that are constructed by the engine, on-the-fly. These limits come from:

  • The name of the running command, i.e. prman
  • The name of the user, such as rdavis
  • The service key, i.e. pixarRender or RMSRibGen
  • Any tag defined in the job script, like intensive or Prod 6
