RenderMan 27 for Blender

RenderMan for Blender (RfB) provides easy access to RenderMan features via Blender.

The addon also connects Blender with RenderMan Pro Server in new ways, leading to fast interactive results without the need of writing RIB. The RIB-out functionality continues to support existing pipelines and farm rendering.

RfB provides truly seamless access to RenderMan's speed, power, and stability for Blender users, who can maintain a simple, Blender-centric workflow, or take advantage of the plugins' flexibility to create an optimal, customized pipeline.

Installation

RenderMan for Blender requires the installation of RenderMan Pro Server as well as the RfB plugin in order to function correctly. See the Installation instructions for more information.
