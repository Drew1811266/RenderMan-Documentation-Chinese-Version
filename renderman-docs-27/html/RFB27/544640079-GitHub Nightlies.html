RenderMan for Blender is constantly receiving updates and bug fixes, so please refer to the RenderMan for Blender GitHub page for the latest nightlies version. 
Please note that releases from the GitHub page are not always fully tested versions, so use them at your own risk!

The GitHub page can be found here :
https://github.com/prman-pixar/RenderManForBlender

To install, please use the usual method of installing RenderMan for Blender.

  1. Download the RenderManforBlender.zip file
  2. Start Blender and from the File → User Preferences → Add-ons install the downloaded .zip file.
  3. Change your renderer to RenderMan and enjoy!
