Particles

Source: https://rmanwiki-27.pixar.com/space/RFB27/544638169/Particles
Downloaded: 2026-06-15T03:49:12+00:00

The point or particle primitive is intended to be used for tiny objects such as sparks, snowflakes, or raindrops. They are optimized for the case where the intended particle takes up only a couple of pixels on the screen.

RfB supports Blender’s particle systems, including instancing, a very common use for particles.

For the Halo type, we render the particle system as RiPoints.

Note, motion blur is done via the velocity vector.

Visibility

There are also many times that you want to render the particles without rendering the emitter geometry. The best way to achieve this with RfB is to turn off Camera Visibility (there are also times when you’ll want to turn off Indirect & Transmission Visibility too)
